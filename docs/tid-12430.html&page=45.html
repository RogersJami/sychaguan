spacejam 发表于 2015-5-24 06:37:21

简单几句讲完：
1.战略层面最简单的理由就是被低估
浪潮、中科、南威是基于服务器需求的猛烈增长，简单把2012年股价当作底价看
四维图新是因为数字地图的需求还有大把
煤炭是因为煤价低谷和替代能源
新疆喀什是因为一带一路需要走巴基斯坦和瓜达尔港
中航飞机是大运需求

这些后面有巨大空间的，只要公司不出问题，只要诸位高僧法号能持，多个20%成本买入也差不了多少，收益只是多少。

这是估值，基本面。

2.战术层面
重要的是放水，随着海量资产进入股市交易，
只要需要的货币还在源源不断供应，牛市一时不会结束，
葡萄提过的1%降准后的十倍放水规模，
哪怕一倍留给这次牛市，也足以保持牛市一段时间，这是信息优势比不了。

3.战役层面
这是最无聊的，也是吹牛的最喜欢的每天10%收益之类，
其实戳破就是一层纸，
每次庄家拉升完毕前后，吸筹的时候，
由于庄家稳定吸筹需要，
股价会稳定在一个区间运行方便庄家低买高卖，
这个区间里如果跟庄家同步操作，很容易短期每天几个点收益。

反映在数据上就是股票在不同价格的成交量不是联续的，
而是一段一段的台阶式价格，比如中航飞机：
10-11，14.5-16，18-19.5，20-21.5，24-27.5，30-35.5
明显的高成交量，用matlab很容易看出来。
而这一个个成交量高峰就成了压力位和阻力位，
当然，也可以继续细化来看更短期。

日线看不出庄家吸筹的区间就看周线，再看不出来就看月线。继续细化往分钟线看就是高频交易。

技术分析就这么简单，所谓的高手一般就靠这个，说穿了就是一层纸的事。

spacejam 发表于 2015-5-24 15:28:56

其实如果更细致观察每个平台，还会有更多短线特征：

比如每次拉升完毕后会开始下跌，
这个过程中的高点和低点基本就是这个价格区间的高点和低点。
为什么？
原因很简单，
要给这个区间买入股票的人一个幻觉，
还可以有更高的价和更低的价。

比如日线级别的拉升一般不会超过13个交易日就要进行，
周线级别的拉升不会超过三个月，
原因也很简单，
老板要考核操盘手的业绩，
老板每个季度又要还款给银行满足存贷比。

其实庄家相对散户条件很不利，
资金是贷款的有利息，
又有时间压力，
又要花钱买通媒体造舆论，
又要花钱买通上市公司配合发布坏消息吸筹和好消息拉升，
唯一优势就可以可以集中资金和信息造成短期价格畸形。

看一只股票看多了还能看出来更多东西，
比如飞机在三十几的时候，
短期震荡拉升幅度基本是1.3-1.5之间，
平均1.4，大约就是葡萄提到的4%这个幅度。

庄家都是人，是人就有很难改掉的习惯。

一些补充

最近两周操作，每天机动资金收益百分之7或8。大体操作决策过程如下

1.抵押重仓股

2.抵押自己加存量机动资金以及其他渠道筹措资金做机动短线

3.大胆满仓原因3点

1)希腊债短期过关，因为到期债务不到60亿欧元且默克尔在红场阅兵第二天飞抵莫斯科后，传闻普京出资给希腊债务过度

2)隔夜拆解利率持续出现新低

3)作为标杆的暴风科技开板后回调，马上持续走高

然后集中精力最神创，不问价格只看图形。图形出现自己参考组合标准突破，果断加。每次选三个标的，最近两周一般选的标的买入后第二天不涨视同假突破就走，每次资金配置1:1:1，这样每次操作亏损多在百分之1.2到百分之2.7之间。但是起来就是百分之7到八甚至打板。第二天早上趁高开百分之4甚至百分之7的当口观察，回调破短均线就走人。三个中，每次总有一个到两个中。不过上周最后几天，遇到过买什么，什么停牌的问题。所以，收益率急速下降。这几天筹措新资金。还有我做短线都是平时积累的标的，及时一时被套，也是因为产业方向对头或者估值有稳定套利区间的。跪一时，能扛得住。前天我老师给我看神创某股，过百价格，没有任何利好每天几手成交，你图形再好，我绝对不进。

而短线上，今天做什么股，是昨天前天做准备。本周做什么板块下周做什么板块，本月做什么板块下月做什么板块团队讨论。相关标的来源是，去年利用理论做密集操作试错过程中积累的标的。(去年出现过三个涨停的股票，有2200个股票图形记在自己脑子里，只要有相关板块热点出现，做什么大致有方向)还有就是次新股中，具有国家政策扶植方向的市场独占股，用孤舟话说买就买市场唯一。比如去年看中的迦南科技，开板后盘整一个多月，起来就是药不能停。当然信息优势有的，优势就是知道哪些地方的三甲医院直接接到文件说必须采购国产医疗设备，去年的信息。

另，市场及时在今天还有价值远远 被低估的洼地。比如，今天和人说起某个股票，到上周末收盘市值不足八十亿，固定资产价格以边际效益折半来计算也高于1300亿。去年到现在就是死活不涨，从估值角度价格低到可笑可。还有现在能出现月线多方炮的股票，犹豫个啥。

功夫在平时。其他身体舒服的时候给你补充。

spacejam 发表于 2015-5-24 19:36:31

那么为什么庄家要制造一个价格区间呢？

原因就是这样操纵的成本最小。

首先要选择一个没有大量套牢的区间，
比如区间在31-35，
这段时间买入人的平均成本可能是33，
当价格接近34.5，庄家集中抛出一半，
股价自然就向33回归，
当接近33，庄家可以选择涨或者跌。

就好像单摆，
庄家通过在平衡位置和极限位置集中发力，控制摆向左边或者右边。

没做过庄或者操盘，
但把自己带入庄家角色后觉得逻辑上应该如此。
