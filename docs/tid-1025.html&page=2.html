综上，我认为我国房价上升是多重因素的复合体。我们要看到其中的投机因素，不合理因素。也要看到其内在因素 ...

多重因素没错，但要分清主次。
成本、需求等因素是涨的基础，但涨幅远低于房价涨幅，支持不了房价这种涨法。
最主要的还是炒作投机、传销，这些就像杠杆，把本来一年一成的上涨变成一年五成，直接挖空房地产发展潜力。
炒作本质就是两头吃，吃完卖家吃买家，把行业利润和发展潜力变成炒家私利。炒作能加快产销速度，但也会带来动荡。
房地产周期长、需求比较固定，一次动荡就要很久才能恢复，炒作明显坏处大于好处。不控制炒房，就是谋杀房地产。
