新型城镇化不是遍地开花，是有序，切入点是区域一体化。在一个区域内，资源向大城市、重点城市周边的中小城市倾斜。所以有中小城市入户政策开发，有条件、有能力的农民工市民化；教育、医疗等公共服务机构郊区化；区域内城市交通一体化等等。

在区域内实现新型城镇化，也需要一个过程，在这个过程内，个人按喜好、定位、能力等实际情况，找到安居和工作场所。如果把这种结果说成人的高下分层、或者歧视，也不出格，毕竟这个世界上还没有大同世界。

2、资源集中于中心城市这种模式已经到头了。城市需要缓冲区、功能疏散区、功能备份区。就像纽约需要新泽西一样。而且纽约新泽西的关系也是有缺陷的，从年前曼哈顿内涝就能看出。
也就是说我国沿海发达城市这种静态的阻止进入的模式是不符合未来城市发展需要的。30年前能进上海，30年后为什么不能退出上海？
而且对沿海发达城市的老年人来讲，还有一个致命的威胁，那就是城市污染。“洛杉矶光化学烟雾事件”等等前车之鉴不远，受伤受害的以老年人为主。

培训技术不能脱离生产，不看好学校，还 ...

社会上的职业技能培训，只是给予单个人一个进入工业化的入口。
以中国人家庭传承的特点，教会一个人某种技能，在社会上站住脚了，那么很快就能带动亲戚朋友老乡参与进来，而且家族传承也很普遍。这带来的就不仅仅是经济效益来。社会效益十分可观。

国家在这方面做的非常不够。一是培训成本大部分个人承担，二是培训没有系统没有组织没有反馈没有与时俱进，效果相当零散，培训中培训后都没有交流协同提高的可能。三是距离职业化要求很远，比如对操作规程安全规程非常不注意。就算是职业技术学校出来的，也要在工作中带较长时间。四是现在的职业培训系统能力缺失，可以说烂到骨子里了，职业技术学校没有能力完成这个任务。个人觉得走出这个困境需要打烂重建。
