1.减少投资额和分散项目地,减少每一次损失的数量.不能够大量央企糜集在一两个国家搞开发.这样目标太大.多搞蚂蚁搬家,哪怕这损失经济效率.经济效率是一个相对概念,不是最好的就有最有效率,而是尽快完成投资-回报循环就是最有效率的.

2.加深利益捆绑,提高每次翻盘操作的成本.有利益捆绑的朋友才是牢靠的,所以,永远站在这样的朋友这边,而不期望所有人都成为自己的朋友.
