我不吃青椒 发表于 2013-11-4 12:18:08

       从另外一个角度看，普世价值解决的是工业难以呈现出爆发性增长后，西方世界需要跳出社会化生产的语境，必须有一种新的理论来阐述这样的一个事实：没有活跃的工业和科技活动为有志青年提供舞台，社会组织形式的重构不可避免。尤其当这种情况出现在一个标榜上帝子民的国家时。他的统治者们必须找到自身执政的合法性。这就是为什么当用普世价值攻击其他国家，总是要去质疑被攻击国家政府的合法性。当然我们都知道这理论死道友不死贫道，所以中国每次反驳这种论调的时候就摆事实。以揭露那些西方国家的丧心病狂。但这也造成了一个有趣的现象，就是各列强争相比“烂”，大有脏唐臭汉之势。无奈啊，人类的科技发展遇到了前所未有的瓶颈。
       理论是没有好坏之分的。从普世价值的喧嚣一时，我们也得以看到当初社会主义运动时所提出的价值观，资本家们重新包装一下也能拿来用。但是社会主义运动解决不了的问题，资本家们一样没有好办法。以至于99%运动的兴起。虽然没有组织目标和完善的组织结构。但是99%几乎从根本上瓦解了自第一次工业革命以来所有的“信”。尽管人们还不知道要信什么，但是他们已经开始“不信”什么了。
       预测一下，当概率论与数理统计进入科学、工程、金融等各个方面的时候，其进入哲学发展，并使之成为主流已不可避免。最终人们凭借其所提供的更广泛的动员能力，而闯过“复杂性”这个魔鬼设置的难关。最终推动下一轮产业革命。

我也来凑凑热闹，胡说几句：

1、不管是目前所提的普世价值也好，还是宗教理念也好。从某种联系来讲，普世价值就是一种宗教理念，宗教就是一种普世价值。二者是有一定的内在等同性的。都是一个信仰的方向，一份理念的表述，一种精神的追求。目的都在于有意识的引导人民群众接受这种价值观，并改造人民群众，从而得到人民群众（或者说是教徒）的精神上和物质上的支持。

2、物质基础决定上层建筑，也就决定了宗教、哲学、信仰和价值观。基于西方资本主义国家所提倡的普世价值本质上是没错的，但西方在推行的过程中，附带了太多的别有用心。他们隐瞒了其中的缺陷，却夸大了普世价值的作用，把它当成了万能钥匙。一句话，西方国家根本就不是为了让你自由平等而来。

3、我们到底要不要普世价值？这个问题毋庸置疑，我们肯定要的，一切美好的事物都是我们追求的目标。但普世价值要结合我们的国情。在中国今后的发展过程中，估计普世价值会以其他的形象和表述出现，比如带有“中国特色社会主义”之类的字眼。

4、我们最大最好的普世价值，我认为就是毛主席所说的：为人民服务。
