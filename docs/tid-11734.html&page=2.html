zilewang 发表于 2014-10-11 14:16:13

统计上的涨跌和个人观感上的涨跌，是不一样的。

个人观感，一般来自于：一，开发商促销手段，高开低走，或者故意饥饿营销，捂盘惜售；二，数据来源不完整，只局限于自己所关注的小区域；三，开发商也有经营好赚钱的，也有经营不好亏本的。

统计上的数据来源于整体，新房来源于签定新房合同时向房地局申报的价格，这是真实的市场价格。国家统计的新房价格是非常靠谱的，如果排除后期的数字修正的话。但二手房价格，目前的统计方法落后，基本可以忽视。
