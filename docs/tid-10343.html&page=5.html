世界上最遥远的距离（改编版本）

世界上最遥远的距离，讹传是伟大的印度诗人，敬爱的泰戈尔爷爷的原作，经多方求证无此事。这首诗网上流传的版本多样，但综观皆有拼凑拖沓，文气不贯的毛病。因此，本闲人闲极无聊，发挥特长，整理成章，以飨读者。

世界上最遥远的距离，

不是从生到死的距离，

而是我天天和你在一起，

你却不知道我爱你；

世界上最遥远的距离，

不是我天天和你在一起，

你却不知道我爱你；

而是你明明知道我爱你，

却对我满怀冷漠和厌弃；

世界上最遥远的距离，

是树枝和树枝的距离，

虽然同根生长，

却无法紧紧相抱相依；

世界上最遥远的距离，

是鸟和鱼的距离，

一个在天空中幸福地翱翔，

一个却在水下默默地哭泣。
