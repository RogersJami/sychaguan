现象-圣人的教科书                     一九合十   http://blog.sina.com.cn/u/1779146080

圣人是有教科书的，现象就是圣人的教科书。

我们现在说“现象”这个词的时候，脑子里面只有一个意思。
其实“现象”这个词，不简单，它是两个意思。
“现”，是显现，就是正在显露出来的意思，现，是“正在出现”。
“象”，是图像，景象，是已经出现的那个玩意。
完整的表述“现象”这个过程，它是这样的：
已经出现的那个“象”，它是动态的，它是由始终正在进行的那个“现”不断勾画出来的。
这个画面，举例来说，比如写字，你用毛笔写字，你去体会这个过程的变化，你正在写的那个笔端，拖出来的那个慢慢出现着墨的过程，这个叫做“现”。你写完留下的那些个笔墨，叫做“象”。

我说这个，可能就有人不明白，以为没有意义。
那换一个很值钱的玩意来说：
股票的走势，期货的走势，正在运动的那个、正在走的那个，叫做“现”。已经走出来的那些，叫做“象”。
象有象的意义，现有现的意义。
把“现与象”这里面的道理悟透了，天下任何事情都不那么麻烦，都是很好玩的了。

捭阖也是一样的道理。
所以，这些高深的学问，就放在鬼谷子兵法里面。
鬼谷子，表面上看，都是讲游说的阴谋诡计，其实不是的。
大争的世道，圣人没有大争的本事那不是天下坏人得好处了吗？所以，这些本身首先肯定是圣人通晓的。并且是有教科书的。

真传一句话，假传万卷书。
事有象，现有比。
这句话，就是圣人的教科书。

因为这个文明，是依赖于我们的子孙后代。由他们生生不息的干出来的。

我这里讲的，就是在这个恢弘的时代，我们的后代要怎么玩的技术活原则的问题。

之所以能够干、可以干、值得干，是因为天运、地运都到了，人也得跟上，要干才能出结果的。

动物本能的复兴，是工业革命的事情，是央格鲁沙克逊人的事情，是英国人和现在美国人干的事情，而不是我们要干的事情。

（不懂“中庸”，去看我写的易经文章。再说一遍，中庸的庸，不是庸俗的庸，而是“广备而致用”。就是说药柜里面药材很多，这叫“广备”。依据毛病而给不同的药，这个叫做“致用”。所以“中庸”不要搞错意思了。）
