a6611990 发表于 2016-10-17 10:04:57

先多谢各位茶友的回复！

我在2015年11月和泉哥结束了我们的激情、冲动创业后。终于认识到我们这类商业训练以及接触层面较低的人是不可能，能够孤注一掷就可以成就大事业的。决定了我们要慢慢一步一步的走，不能放弃原来的收入。基于我对之前激情创业的反思结果：传统、积累、力所能及这几个硬性要求。他先做他原来做的广告装修店，我则在16年的4月找到了一个450平方左右的铺面开我所熟悉的小超市，以及做洗发水。小超市已经新开张几个月了，洗发水产品设计已经做好了但还没有开始卖。如果我的洗发水能够成功达到年纯利润百万以上，到时再来和众多茶友分享我的产品设计、推广的经验以作抛砖引玉之用。

重新开了一家比之前大几倍的超市我又有了一点新的理解与认识。现在写出来帮助自己记忆也与众茶友共同讨论。

超市所能提供给顾客的最本质的产品是什么呢？而顾客所想要的又是什么样的一个店呢？基于这样的思考我认为超市最底层的代码是以几点：
1.信任。他信任你这样的大店、装修好的店。所以你不能让他失望，所以你的产品质量一定要好，一定要对得起你的价格。

2.服务。顾客在购买你的产品的时候，心理上天然的认为价格包含了你的服务。所以你的服务要好。小妹服务细节要做好。产品要有足够好的售后。不能说卖出去就算了。要有售后。

3.方便。超市产品的品类要尽可能的多和全，品类下的品种也要多，并且要各个价位尽可能的覆盖到，或针对店面的主要消费群体的收入做到尽可能的迎合。性价比要好。如果顾客不太乎价格，就同等价位质量要比别人的好。如果顾客在乎价格，那就同等质量下价格比别人便宜。

4.利用人性的弱点的。人天生就喜欢占便宜。买东西更是喜欢便宜，可以把一部份商品比如米、蛋、面之类的平本或亏一点做为引流商品。也要搞一点价格本来虚高的商品比如爆米花平本卖，增加顾客对店的好感。
