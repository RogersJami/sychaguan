其实我觉得为人不能太刻意。

自己本身不是那八面玲珑的性格，非要学着去做，那就是邯郸学步，自己难受别人也尴尬。这个和工科生没有关系。

其实你自己也说了要真诚，这个就是最重要的。将心比心出自本心的关心，和察言观色出于逢迎的关心，稍微有点社会经验的人都能感觉到。

举一个不太恰当的例子：
我也是偏内向的人，06年刚毕业是做了销售工作，从家人到同学没有一个看好的。刚刚工作时，也对那些性格外向自来熟的同事颇为羡慕，恨自己不能在最短的时间内和客户打成一片谈天说地，迅速拿下订单。为此也看了不少关于销售、口才的书，也曾试着模仿那些我所羡慕的人。可是实际效果很糟糕，一段时间自己都要迷失了，有时候也有你说的那种“唯唯诺诺”。

后来，痛定思痛，仔细梳理了自己的性格、优点和缺点，决定在为人处事上还是要坚持自己的风格，不刻意逢迎和吹捧。结果自己轻松多了，有一说一，实事求是，不再患得患失、唯唯诺诺。因为你就你自己，永远也不是别人。到现在工作6年了，基本上和绝大多数同事和领导关系融洽。你的工作能力是靠工作成绩来表现的，不是看你在酒桌上有没有谈资，也不是看你是不是八面玲珑。

至于你说的经验或者书籍，一来那些不过是别人的经验，不一定适合你；二来，这些总归只是一些技术、技巧方面的东西。不是说这些不重要，而是说如果你自己没有认清自己，这些东西只会让你迷失自我——就像我刚开始工作一样。
--------------------

至于你说的第3，4点，我想这实际上还是设身处地的站在别人的角度想一想就明白了；第5点，实际上只是你对自己还是不自信而已——不要把自己的自身发展寄托在领导身上，他只不过比你早入职几年而已。你想混出一点名堂，永远都是看你自己的能力在什么高度，还有就是运气。

我想对于销售工作，没有做过的人往往会认为成功的销售人员应该就是能说会道，八面玲珑这样的。我认识的两个挺成功的老销售，就是偏内向的，其中一个甚至看起来有些木讷。

我自己的理解，销售并不是这样，我感觉在同等条件下，销售的好不不好其实和外向和内向的关系不是太大，产品能不能销售出去归根结底比拼的还是质量、价格和服务。当然，我做过的是比较简单的外贸销售（并且是食品添加剂原料类的销售），没有经历过和竞争对手坐在一起竞标那样的场景，所以可能不具有代表性也很难举出什么精彩的案例。

在我看来，单纯的能和客户迅速打成一片，善于自来熟只是其中一方面。更多的，销售是一个概率问题，比如我会广泛了解潜在的客户，列出重点客户并投入主要精力，同时不放弃任何一个可能次要客户。网撒的广了，总会有些收成的。销售有时候也似乎有些运气成分，但总的来说，还是功夫在日常工作的每一点一滴中。

当然，我也会精心准备每一次和客户的见面，也会猜测客户的真实心里，引导客户说出他真实的需求等等，这些都是技巧性的东西罢了。最关键的是，找到并形成适合你自己的工作风格，并持之以恒的坚持下去，只要这样做了，肯定能成功。对于一个销售人员来说，想客户所想，急客户所急，实实在在帮客户解决问题应该是他必备的素质，长期坚持下去想不成功都难；你看所有成功的销售都是做长久的业务，一锤子买卖那是骗子。

再一点，并不是所有人都喜欢夸夸其谈，比如我就喜欢话不多但是能说到点子上的人。负责采购的也都是普通人，性格也是一人一个样，你怎么就知道他就不喜欢和你这样风格的人打交道？比如我现在做采购就在同等情况下有我自己的倾向。客户需要研究，但是刻意的逢迎可能会适得其反。

实际上，能和客户打成一片，吃喝玩乐、无话不谈的那种销售，我是真的很佩服，因为要付出的可能更多，没有人随随便便成功。你想想如果换成是你，你能吃得消么？反正我是不行，所以还是找找自己的强项，发挥自己的优势就行了。

因此，实际上说的还是做事情要实实在在，不管对自己还是对别人，邯郸学步要不得。
