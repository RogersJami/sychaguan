liziang 发表于 2012-6-18 22:02:54

国内超精密加工设备国家还是很重视的，作为重大专项每年投不少钱下来。
我博士期间做的课题是和超精密加工技术相关，所在的研究所也是国内搞超精密加工设备的山头之一。
现在国内超精密加工设备和国外差距还挺大的，还是不停的追赶，被国外打压的也挺惨。
首先国产核心部件性能和国外差距巨大，如电主轴，直线电机，数控转台等等，最高精度的都是禁运品。我们做课题，很多时候想爱国下，用国产件也没办法，精度达不到，只能要么用国外的，要么自己动手。
其次产业化困难，虽然这个应该不是我们研究所的活，研制的机床基本都是单件或小批量，高精度的机床多数都是定向研制，做完直接运走，很多我们很好的技术没法转化到工厂。
不过国家投钱下来还是物有所值的，所里研制的超精密设备起码解决两个问题：
一个是很多核心部件，国外不可能卖你设备帮你造的，还是得靠自己研制的设备，国防项目中很多
二是虽然研制设备没有经济效益，但是有社会效益，举个例子，某厂为新产品需要上一批五轴机床，找老外报价，一律5百万左右，嫌贵？对不起，都这个价，除了我们几家你没地儿买去。找到我们所，先做一台试试看吧，花了两百万做了一台，一用没问题，再做六台！老外同类型机床立刻降价50%。
