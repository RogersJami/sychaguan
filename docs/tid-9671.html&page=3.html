新中国从建国起对旧社会下的军公教人员采取全部包下来的政策，也就是由财政供养，这个你可以去查
当时只要有文化的人尤其是中小知识分子在新社会和新政权下的物质待遇是很优厚的，代价是要经常搞思想政治学习，转变立场。知识分子(其实主要是那些高级知识分子)是地位下降不代表生活待遇下降。而国民政府时期，虽然教授地位高待遇优厚，但是很多青年学生经常是毕业就等于失业，所以眼睛不能光盯着上层知识分子那一小拨人。
将有文化的人由财政全部包下来的政策保证了社会中层的稳定有饭吃，连同土改一起就保证了社会的高度稳定，后来怎么折腾也没动摇这个基本结构。当年还有靠一首诗进作协的，作协是事业单位，那是由财政发工资的。

以上有我家的亲身经历为证。我爷爷兄弟二人都读过小学，那时乡下小学就算知识分子了。我爷爷年纪大点算有解放前参加过三青团的污点，所以只能做乡村老师。小爷爷正好解放时成人，于是作为党的基层财贸干部最后做到县里的区长离休。那时的教师队伍是土工选拔干部的重要来源。识文断字的人到那都吃香。
毛时代实际上要求的是“又红又专“，而不是知识越多越反动。说白了就是要求有文化的要效忠共产党的领导，这其实是历代开国王朝必有的手段而已。
共产党不是不要官僚，只不过他要的是技术官僚，而不是明清那种除了空谈心性服从听话外一无所长的官僚。
共产党相比国民党能保证稳定的关键在于土工从苏联手里传承了一套现代财政体系，能够支撑财政供养更多的人员，即使这样党内的财政开支一直很庞大的。所以大生产运动曾经起过很大的作用，就是在于它减轻了财政负担。
