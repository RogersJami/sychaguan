原 创: 新 墙 旧 事—— ( 一 ) : 小 码 头

本帖最后由 老光 于 2013-3-28 12:45 编辑


        去年到北京城里过大年，与儿子团聚时刻，便与家住株洲的老弟通过电话。
        自上世纪末的1999年从株洲开罢餐馆回到桃源之后，与老弟已有十多个年头未曾相聚。此次，老弟再三再四要求俺，返程时刻，一定得落一落株洲，与我们相聚个几天。
       一母同胞的兄弟之情，本就血浓于水。老弟夫妻俩的这一份情意，无论如何那也是不可拂却的盛情嘛。于是，三月二十四日，乘飞机到达长沙，还未出得候机楼的大门，便见老弟父子俩早已守候在此。
       吃罢过晚饭，相互间说起这些年各自的情况。言谈中，得知俺学会了电脑，并开始学着写一点东西，又时常在网上驰骋，不免勾起老弟的羡慕情怀。老弟谈起他的一位好友，也是老年开始学电脑，将自己平日内的所见所闻，凭记忆写下一些作文类的东东，却未曾发表过一篇，并说写得相当的有文采，有很强的可读性。不由引发起俺的兴趣，跟着从侄子的电脑中下载过来。仔细观看后，感觉文章的确写得蛮不错。可惜的是深藏闺中，未予发表，似觉得可了惜。
       于是不揣冒昧，特地拿将过来，略作些文字上的修饰，代朋友发表在此，以飨各位。

       以下便是高姓朋友所写的文字：


       新墙是个小地方，在湖南省的地图上，充其量也就是颗砂粒子大小，隶属岳阳县管辖，离岳阳新县城荣家湾有十二公里远近，故此，这新墙又有个“十二公里”的别称。

       新墙镇的最高权力机关—镇党委和镇政府早已搬迁到靠近107国道边上的繁华地段，权力机关一搬迁，将一个破旧的新墙老街遗弃在离国道还有三公里的河沿上。
       新墙镇老街靠近新墙河，新墙河潺潺流水直入八百里洞庭，洞庭湖又连接着万里长江，长江日夜奔腾不息灌入浩淼无际的东海。因此上，这新墙镇也算是个通江达海，见过大世面的小码头，可千万不要小觑了它。想当年，水运繁华鼎盛年代里，新墙老码头因其热闹与喧嚣也曾享有过小‘汉口’的美誉，老一辈的一提起昔日的辉煌，至今还啧啧不已，唾沫子横飞。
       时过境迁,今非昔比，特别是九八年的那一场大洪水，把个好端端的小‘汉口’荡涤得一塌糊涂。从此，往日的热闹与繁华不再，留在河沿上的只剩下一些断壁残垣，碎砖瓦砾与杂草丛生。
       镇上的年轻哥哥妹子们，胆大的去了那北京、上海、深圳等大都市。麻起胆子的也去了省城长沙或是湘北重镇岳阳市。娃娃们削尖脑壳、钻山打洞，霸蛮想融入另一种不属于自己的生活，远走他乡，撇下些老弱病残，留守妇孺无可奈何地撑守着这片荒芜的土地，怀揣着一份故土难舍的执着，仍旧面朝黄土背朝青天。
       03年，原外贸公司的同事彭强一时兴起，跑到新墙镇上投资，和些当地人打群结伙求财开办厂子。可蜜月期不到一年，合伙人就集体撂了挑子，生产经营三天两头出状况，搞得彭强伤透了脑筋。 他自己还有一份主业在长沙需要亲自打理，故此不得不四处找人替他来收拾这付烂摊子。
       此时，恰逢我关闭了国外的分公司，打道回国谋求发展，回湖南却面临着从外贸公司下岗的窘境。见此景彭强喜出望外，三顾茅庐，好言相邀。经不住他一顿子的蛊惑与引诱，上了‘贼船’走错了路，从泰国曼谷这个国际大都市径直跑到穷乡僻壤的新墙，在老街码头一呆三年，和一帮子农民企业家厮混了千多个日日夜夜。三年时间，算起来不长也不短，可生活、工作与环境的突变，却有着天壤之别，与之打交道的人与事更是具有农民意识和地方特色，真正是春夏秋冬，风霜雨雪，甜辣苦酸， 五味俱陈。
