转一段湾湾 YST的评论作一个参考，这里论述的对手是小日本，没包括美军，如果美国为了钓鱼岛而跟中国开打，那就是掀桌子反脸的时刻，那就是另一回事。

中日军事对垒的态势 甲. 野田首相的误判情势 面对中国庞大的反日浪潮和经济抵制，09月20日，野田首相承认误判情势，钓鱼岛国有化产生的演变远超出他的想像。野田之所以误判情势是因为他误判中国大陆的军事实力和战争意志。野田的自大与自信是自以为日本的海空军对解放军具有优势，其实是根本错误的。

日本的海军从表面看的确优于解放军的海军，这主要表现在大型水面作战船隻上。超过五千吨的战舰，解放军只有28艘，日本海上自卫队超过五十艘，这是最看得见的优势，但也是肤浅和最令人产生误解的数字。现代海军作战不比二战时期，更不比甲午海战，军舰摆开阵势互轰。现代海战比的是系统对抗，主要靠信息、导弹和空中力量。 信息化非常耗钱，这是为什麽中国军队在信息化方面一直非常落后。但是中国稍微富裕一点后就急起直追，尤其最近的十年真是进步神速，因为钱多了，最明显的例子就是「北斗卫星导航系统」。今天中国军事的信息化已经处于世界一流的地位，和美国相较即使在某些地方落后也没有代差，有些地方甚至领先世界，譬如量子通讯。中国的预警机也绝不比美国逊色，这些都是具体的例子。有了钱信息化就好办，中国人聪明，只要是跟数学有关的中国赶超先进国家绝不成问题。

日本的空军完全不具优势，无论数量和质量都不如解放军，在钓鱼岛海域的上空日本更是处于地理上的劣势。
军舰最怕飞机、次怕导弹。飞机不用说了，日本的一线战机都已老旧，大部分的F-15J（日本空军主力）并没有发射中程空对空导弹的能力，完全不是中共新型战机J-10与J-11的对手。这就是为什麽日本买F-22不成，眼巴巴地等F-35。眼看中国新型隐形战机J-20、J-31不断出现，没有研发战机能力的日本只能乾着急。

再谈导弹，这是中国的强项，世界一流，某些地方领先。中国大陆非常重视反舰导弹，中国在这方面的研发领先美国，服役的型号繁多，YST个人知道的就有十几个。现代的反舰导弹射程都相当远，譬如大陆的「鹰击」系列大部分的射程都超过一百公里，其中「鹰击-62」的岸射型射程超过四百公里，见下图。「鹰击-62」还有舰射型和空射型，它们的射程更远，空射型的有效距离超过550公里。大陆还有一种高度机密的反舰导弹，「鹰击-12」，射程超过六百公里，不但是高超音速而且雷达特徵很小，属于隐形导弹，它的导引设备特别先进，是多种感应器构成的复合制导，除此之外，据说弹头装药超过一吨，专门对付大型军舰。

这些反舰导弹对过往船隻形成极大的威胁。估计大陆沿岸装备的反舰导弹大大小小大约一万枚，战争时期黄海、东海与台海基本上是死亡之海。大陆预警机在安全空域飞行执行任务，它是战区的大脑，所有岸射、舰射、空射和潜射的反舰导弹都在预警机的统一指挥下进行协同攻击（coordinated attack），这才是高科技的海战。想想看，日本大型军舰多有中国的导弹多吗？这根本不是重点。

日本夸耀它的大型战舰多、反潜飞机多是非常误导的，打起仗来根本不是日本人想的这麽一回事，没有空中优势的日本将一败涂地。
