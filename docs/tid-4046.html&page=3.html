房产升值这个议题。现在说不会涨或者说不会跌都太过武断。我是学数学滴，前端时间根据上海市银行存款和民间公开的基尼系数，估计出上海存款在50w，年收入20w以上的家庭有100w户左右；同期上海土地供应为1200公顷。可以看得出来：以10年按揭计算，只要这批人有决心买房，供需基本是平衡的。更何况有外地人不断涌入。
也就是说上海的土地供应基本职能满足最有钱的那部分人，房价也肯定是由这部分人决定的——除非改变现有的住房政策和开发模式。

北上广以外其他城市，很难满足这个条件，刚需不刚，很虚。
