这里补充一个话题，去年因为和阿里的朋友接触后知道部分信息后推算得出的结论是，从未去年开始计算起，仅仅服务器领域扩容就高达100倍我自己计算后很惊讶。在和一些朋友就大数据治理方向闹掰扯前，我和他们证实了一件事，这个一百倍的扩容很可能是保守的，这个信息来自工信部。这件事我写在去年茶馆的回复了，并因此推导出两件事，1.以大数据中心的供电需求计算判断核电整体启动；2以大数据组织形式推导过亿人口都市圈。

这里介绍一个简单公式，在市政学一个基本公式是，当都市人口翻翻对应的基建需求是三倍到四倍增长，在这个公式应用在汽车增量与公路交通建设之间的矛盾能很充分说明问题。而另一个涉及数字化组织的边际效益是，城市规模每翻翻，城市副中心增加百分之50。这样的推导，可以解释去年整体牛市从传统领域到新兴领域的普涨基础。只是，这轮牛市来的太快，脱离的国力基础本身的运行规律与经济转型本身的政策逻辑与市场逻辑，所以我们必须回归慢牛。
