难道说，地产经济模式要一直搞下去了？ [ nova2000 ] 于:2013-03-02 07:15:27 复:3851721

城镇逐渐向一线城市拉齐，在那之前一线城市的地产价格都会一直维持在高位

在城市高地价下被淘汰的产业（重资产类企业，不是轻资产的服务类）流向城镇

于是在城镇落户，吸收当地劳动力形成新的产业，然后扩散城镇化

但是这样带来的问题是，无论投资什么最终都会流向地产行业

也许这将能够在很长一段时间内吸收制造业挤出的人群以保证就业的稳定

一句话，我们没有退回到1998年，我们退回到的实际是2001年~~
