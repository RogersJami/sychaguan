新疆鄯善县严重暴力恐怖袭击案告破
   新华网乌鲁木齐６月３０日电（记者毛咏）新疆警方发布消息：新疆鄯善县严重暴力恐怖袭击案于６月３０日告破。

　　现已查明，今年２月以来，艾合买提尼牙孜·斯迪克纠集多人进行宗教极端活动，收听煽动暴力恐怖活动的音频资料，形成了１７人的暴力恐怖活动团伙。６月中旬以来，该团伙筹措资金，购买刀具、汽油等作案工具，多次踩点。

　　６月２５日，公安机关根据线索抓获该团伙成员１人。艾某因害怕罪行败露，２６日晨，伙同１５名暴徒，先后袭击鄯善县鲁克沁镇派出所、巡警中队、镇政府、建筑工地、个体商店和美容美发厅，烧毁车辆，持刀疯狂砍杀公安民警和无辜群众，造成２４人死亡，其中维吾尔族１６人、汉族８人，包括２名女性。另有２１人受伤。处置过程中，当场击毙暴徒１１人（注：已经没二话了。），击伤并抓获４人，缴获刀具、汽油桶等作案工具若干。

　　经群众举报（注：艾力不可能藏到汉人堆里去，维族人也不是所有都赞同），最后一名逃犯伊布拉音·艾力于３０日被抓获。
