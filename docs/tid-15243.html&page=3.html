前几天身体不适，今天接着码。

工业革命和市场的结合产生了市场经济制度，由于工业化的社会分工能显著提升效率，物质被大量制造了出来，国家和民众变得更富，造富示范效应掀起了全球市场经济运动，一国或主动拥抱市场经济制度加入这场运动，或因为先发国家的殖民而被动卷入。市场经济每到一地，就对该地的农业-农村社会进行拆迁，取而代之的是高度流动的工业化-城市化社会，形成一个集态度、信仰、制度于一身的复合体和一个寓经济和技术于其中的大网络。

态度——财富资本化

在农业-农村社会，财富积累完成之后，会立刻被用来消费、炫耀、送礼等，以免毁于战争或被权力洗劫。在工业化-城市化社会，财富被当作为利润而再投资的资本，结果是永不满足无极限，永远有额外的利润在等待攫取，并伴生了物品货币化的意识形态，几乎一切物品都被赋予货币价值，几乎一切物品都可以用金钱买卖。

信仰——个人原子化，极度信仰个人自由

工厂化生产取代家庭生产，成为最主要的生产方式，劳动力受雇佣进入工厂谋生，首先在经济上从家庭中脱离出来，个人因为生病、事故、年老、失业等陷入贫困，亲属团体不再是“保险公司”，国家福利制度承担救助和保障功能。

随着个人的生存与家庭不再密不可分，个人通过各种契约关系与他人发生联系，成为经济—政治-法律综合体，相应地，原先的习俗、价值观、生活方式发生了改变，个人仅凭自身获得（或者说被赋予）各种权利和义务，任何团体都不再高于一切，家庭不再是社会的基本组织单位，被削弱到仅仅帮助子女学会某些技巧，使子女融入和参与到社会中去，家庭规模日益缩小，夫妻成为唯一的联合单位。

最终，个人成为社会的基本单位，个人被“解放”了，自由了。

制度——各领域相互分立相互制约

个人首要的忠诚对象不再是家庭等基于血缘或地域的团体，而是国家，个人和国家并不能直接发生联系，所以必须让一些其他团体成长起来，作为沟通个人和国家的载体，而且这些团体必须是开放的、流动的，个人可以自由进入和退出。

这些团体划分了不同的领域，经济领域的公司、政治领域的政党、宗教领域的派别、社会领域的协会，，个人串联着相互分立的这些领域，比如美国的随便某个Ｍｒ．，他在Ｇ公司上班，共和党党员，信奉基督教，加入美国全国步枪协会，并且每一个领域都被其他领域制约，国家－社会得以正常运转，这就是西方所说的公民社会，自由、民主、平等普及，福山认为历史在公民社会终结，没有任何其他模式更优。

但经济、政治、宗教等领域的相互分立只在理论上成立，实际表现中却是相互渗透相互勾结，原子化的个人根本没有力量对抗组织化的团体相互勾结，个人还是被奴役着，除非成为某个团体中的代言人。

回过头来说大数据。

大数据是互联网发展出来的技术，互联网基于点对点的传播方式，促使信息快速扩散，每一个点得到信息，立即传播给临近的第二个点，第二个点再传播给第三个点，以此类推。似乎每一个点都是不可或缺的，但是你这个点不传播，自有其他的点传播，你这个点就没有存在的必要，所以每一个点又是完全可以替代的。以围棋类比，每一个点是一颗棋子，既是可以舍弃的，又是必须保存的，就看是否能把多颗棋子做成两个眼成活、围成空，决定权在棋手。

工业革命以来的科技发展过程完全取决于政府、公司和消费者三者的商业选择，他们决定了研究和商业化哪种技术以及如何去运用它，政府和公司占据其中的主导和主动作用，为消费者提供多种选择，消费者选择其中的某一种，所以在某种程度上是被动的，是被塑造的。

政府对应权力，公司对应资本，消费者对应个人，在市场经济制度下，权力和资本的结合已经奴役了个人。大数据结合市场经济制度，分离各领域将前所未有地高效，如果被用于统治个人，大多数人的存在将失去社会价值，等同于行尸走肉。
