12. 决定

就在LD拒掉offer的第二天一大早，接到面试经理电话，问她为什么不接offer，是因为钱吗？并且表示，虽然给她的offer已经比公司目前的老合同工都高，他还可以想办法。如果当初的offer直接给到位，我家LD就接了。现在中间出了这么个插曲，她又是个不喜欢绕弯子的人，于是就实话实说：就在跟您面试前，甲方老大找我谈过，正在努力把我变成他们的正式员工。在业主公司工作，做正式员工更稳定，我想了想还是等那边吧。

面试经理向她解释，他们的项目计划很长远，工作稳定性没有问题，而且他们会想尽一切办法留住表现好的合同工。并问她除此(合同工)之外，还有什么他可以做的(可以让LD改变主意)。LD就跟他说，合同工风险还是大些，她不愿承担这些风险......谈到最后，面试经理下了决心，“如果我们把这个职位改成正式员工，你会不会认真考虑？”LD回答说，只要工资水平能达到她这个级别的官方统计平均值，她就接。面试经理说，“好，我去跟HR谈，你等我电话。”

LD马上给我打电话说这个事。我听到以后，琢磨琢磨，大致明白了其中的关节：

重点项目不像生产项目，投资受资本市场和全球经济气候影响较大。一般说来，正式员工能满足基本的项目需求就行，如果项目太多，就用合同工填。由于税务原因，有些老游击队员也喜欢做合同工。现在有项目，经理想多个正式员工(head count)，还是做得到的。因为过上几年，即便没人辞职，也会有人被调到别的部门，坑自然就腾出来了。

果然，面试经理把这事搞定了。正式员工面试程序要复杂、HR要介入。又经过一番折腾，LD接到一个新的offer。这个offer非常好，与之相比，原来给的合同工时薪就太少了。估计是考虑到并且容忍合同工在工时上的水分。

没什么说的，LD把这个offer接了。接之前，我建议LD，给甲方/乙方4个周的notice----正常情况下，2周就行。不过LD手上的东西比较多，需要的时间长些。如果用不了4周，就趁机休个假。

15. 细说说R经理

大师兄提到， R经理连有提携之恩的A副总都不袅 我觉得倒不至于。

16. 接着说R经理

我家LD跟R经理辞职，礼数有所不周。正常情况，应当先找R经理谈一下，说些客套话，然后再交辞职。直接一封辞职信寄过去，从公事公办的角度没有问题，同时也摆明了以前没有私交，以后也不准备打交道。

R经理的表现更加糟糕。作为分公司经理，他在分公司代表的是乙方这个组织而不是个人。手下有人辞职，别说是一个受甲方重视的核心骨干，就是一个普通的员工辞职，会办事的领导也会抽空上门客套两句。

当年我从乙方辞职，L副总闻讯立即约我面谈，虽然这种情况下大都是尘埃落定，谈也改变不了什么，但是姿态还是要的。这是作为领导的职业素质，就像空姐的微笑一样。后来跟LD参加乙方公司的圣诞晚会，碰到L副总，L副总迎上前搂着我的肩膀说，“哥们，我出多少钱才能把你雇回来呀？”这种客套话当然不能当真，但是听着还是蛮顺耳的。

A副总在的时候，每逢有老人离开，都会安排人买蛋糕，找个时间把大家召集起来客套两句。Costco的大蛋糕一个才20块，买两个也就40块。A副总看重的人离开，他都会请顿午饭。

R经理上任后，分公司的乙方组织就消失了。没有人代表公司做一些面子上最基本的事情。

我在本文第一节，首先写老秘的故事，就是想交代一下R经理的行为特点。就老秘被裁那件事而言，于公，R经理作为一把手有责任稳定军心；于私，R经理和老秘是20多年的同事。无论如何，R经理都应当让老秘以一种体面的方式离开。

我的好友T经理曾经跟我吐槽：分公司成立20周年的时候，举行了一个庆祝活动(我家LD应当参加了这个活动)，邀请了甲乙方现任及退休的很多相关领导。领导们都是西装革履、甚至领结燕尾服，唯独分公司现任R经理，穿着皱巴巴的挽着袖子的衬衣、皱巴巴的西裤、脏兮兮的鞋，讲话的时候，站没站相，说话不成句，说几句话咧一下嘴。T经理说，他不知在场的乙方高层怎么想，反正他觉得挺丢人的。

我觉得，R经理从没把自己当经理。
