记得罗素说过知识是通过对材料的思维活得的意识（大意），IT信息爆炸大大丰富了人们的材料，但没丰富人们的 ...

你引用的罗素，我很赞同。记得以前看《怎样解题》中谈论到如何解决复杂甚至难解的问题，他给出两个方案，一.感觉，二.先不解决。

西方许多哲人所说的感觉实际上中国人下棋的时候所说的棋感，只要经验足够，你可以以不到一秒的时间找到正确的点和方向。但是网络的问题则是许多人没有这样的积累，人云亦云自然无法避免。

《怎样解题》中所谈论到的那种及其难以分辨的问题，有的时候最好的解决方案却是“先不解决”，因为你站在原地要么事实的真相会逼近你，要么你的经验与知识会在不经意间触动某根心弦。

可是网络中的人大多如同《浅薄》一书所说，不过是某种超级生命体，如同蚂蚁群一样。以前我还存有质疑，后来在网络上发过一些帖子做实验，得到的结论刚好与它的预言一致——无意识。
