按我的理解简单分析一下数据
1.租赁 : 37.2%，自有住房 :  56.9%，其它 : 5.9%，租赁比例还是很高
2.自建住房比例很高 5.6%，跟选取的抽样地区有关，如城郊结合部，城中村
3.经济适用房比例很高 6.1%，跟选取的抽样地区有关，有意体现优越性？
4.购买原公有住房比例最高 21.5%，老房子还是很多
5.购买商品房 20.2%，比想象中要低
6.购买二手房 3.5%，跟实际生活经验比要低，跟选取的抽样地区有关，新小区比较多？
7.疑问，这里的家庭如何定义，如果3个人合租1户住房，统计的时候算1户家庭还是3户？
8.结论，数据有一定参考意义，细节值得推敲
