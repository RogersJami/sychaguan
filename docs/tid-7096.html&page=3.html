师兄似乎太小视刘备了点。刘备也是一方老大，用人上当然不会只亲信自己兄弟的，否则也难以成事。最好的反例就是半路来降的魏延，刘备得成都后需要人镇守汉中，众论以为必在张飞，飞亦以心自许。但刘备却拔延为督汉中镇远将军，领汉中太守，一军尽惊。这说明刘备绝不会是任人唯亲的。

赵云的问题，还是他自己的理想和抱负跟刘备不对路，更接近于诸葛亮一点。刘备在关、张死后欲打东吴，赵云劝刘备“国贼是曹操，非孙权也，且先灭魏，则吴自服。操身虽毙，子丕篡盗，当因众心，早图关中，居河、渭上流以讨凶逆，关东义士必裹粮策马以迎王师。不应置魏，先与吴战；兵势一交，不得卒解。”可见一斑。

而且刘备夺了益州之后，想着打了一辈子仗，该享乐一下了，想以成都中屋舍及城外园地桑田分赐诸将。赵云却驳之曰：“霍去病以匈奴未灭，无用家为，令国贼非但匈奴，未可求安也。须天下都定，各反桑梓，归耕本土，乃其宜耳。益州人民，初罹兵革，田宅皆可归还，今安居复业，然后可役调，得其欢心。”明显是没有揣摩领会领导意图，不受重用也是情理中的事。
