而且反23条算啥，人家还反国教。还有所谓2017普选，香港反对派也是混淆视听，说要符合“国际标准”的普选， 但在《基本法》里明确规定了特首提名机制。TG不是自己给自己添堵吗？对台湾再不吸取教训，那也是自找苦吃了。
