妹妹这样认为是可以的，因为有许多人总是试图抓住一切的波动。
ETF是一种指数基金，不是黄金交易所交易基金，它的走势与实际指数的走势相同，比如510300就与沪深300的走势一样（510050，510180分别对应上证50和上证180），159915就与创业板的走势一样（159901，159902，159903分别对应深100，中小板，深圳成份），这几个品种的交易量都比较大，当然这个就木有找黑马，骑黑马的乐趣了{:soso_e112:}而且收益肯定要比你找到黑马低许多，甚至于很多时候还比不上你随机选择一个票票{:soso_e127:}但是优点就系比较稳健（万一木有骑到黑马，而系一个原地不动或者还倒退几步的票票，介个要好得多），怎么讲呢？凡事有一利必有一弊，你选择了稳健就必然与暴利失之交臂，但系也避免了风险（当然系统性风险还系无法避免，比如5.30“半夜鸡叫”神马滴）。
