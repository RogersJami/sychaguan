presario2200 发表于 2012-6-24 12:54:29

写得有点意思，但也没啥新意。年轻人这类文章看看即可。不能当真。这类“成功回忆录”颇像股评，基本都是先放枪后画靶，而且只会把好的一面展现给你看，台面下的事情丁点不讲——往往这时最关键的。就像忙总说过的——少看那些成功的东西，你知道他家里是干什么的？马云也说过类似的话，别人成功你别管，要多看看别人怎么失败的。

从某种角度看，我觉得作者就是个被体制异化了的人。工作20多年，又做过领导，想必40岁是有了，很多东西该到了看开的年纪了，但是我瞅着多少有点帝王将相的思想不说，居然还把等级啊、待遇啊、以及那些“享受”上的事情拿出来说。这都什么日头了，早过了坐软卧还得出示副处级身份证明的年代了。飞机、高包、土特产。。。我“享受”这些待遇的时候才25，行则奥迪住则五星，土特产啥的我从来不往家里拿，就地转手送给司机。比他强多了，没什么值得说的。

说到这儿想起来这两年很热的官场小说《侯卫东官场笔记》，我最近刚看了一本，别的没啥，就觉得这作者也是个满脑子帝王将相。张口官场闭口百姓，不知道的以为是上帝在写回忆录。其实作者也就是个重庆郊县的副局长。真不知道他这么多“官场智慧”都用到哪里去了。三哥没犯事儿的时候讲公职人员的阅读太“庸俗”，要多读经典。不知道三哥是不是也看过本帖以及各种官场小说，但本贴以及《侯卫东》必属此类无疑。

中国的未来，如若官僚都是这种见识、这种三观，前景堪忧。

对于刚工作的年轻人，我觉得其他的东西不用多想，想多了白想，而且人会变得猥琐。有些事情是学不来的，比如待人接物、言谈举止、人格魅力还有组织协调能力等等，相当一部分是天赋。记住三个字我看就行——勤、勤、勤。数十年如一日地勤。人心都是肉长的，我没见过哪个一辈子“勤”的人被周围的人或者组织亏欠一辈子，人心都是肉长的。勤还能补拙。比如说，口才不好怎么办？在任何可能有发言机会的场合，都预先写稿子，反复背诵。如此虽不中亦不远矣。

人际关系要想处好，有几条通用的原则是需要记住的——1、不和人争辩，如果不是要命的东西，即使知道人家讲错了也不要说破 2、做好的听众，人人都喜欢和愿意倾听自己讲话的人相处 3、大方点——谁不喜欢大方的人？
