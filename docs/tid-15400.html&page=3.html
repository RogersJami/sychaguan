我觉得任何国家的衰落或倒掉都是因为利益分配模式出了问题。

分配模式有两种，一是国内的利益分配模式，二是国际范围内的利益分配模式。这两个模式又是相互关联和影响的，尤其是在科技发达交通便捷之后，包括互联网引发的便捷。

大的方面讲，利益分配模式是由社会组织模式决定的，而社会组织模式又是由生产力发展阶段决定的。

历史上遍布着为了维护利益分配模式的格局，用各种手段（包括打着发展的旗号，给大众描绘美好前景）去绑架该国家整体最优决策的小算盘。遇到要决策的事情，这些利益者首先想到的是该决策是否会打破现在对自己有利的分配格局。

而后发至上的国家在上升期，分配模式基本都是处于较优的状态。

数字化社会对人、生产、国家制度等结构带来各种冲击。
换句话说，一种新的生产方式会对各种旧的结构模式带来必然冲击。旧的机构模式也会被新生产方式推动的优化、改变、甚至崩溃。直到被打磨到适应这个生产方式得以生存。

打磨的路程如何行进不得而知，但能推导的就是这个过程和历史常态没啥区别。拿国家体制来说，有新生的、有灭亡的，有蒸蒸日上的，有日暮西山的，还是像烙烧饼一样走这个历史大循环。

总结起来就是，数字社会的激荡发展（高潮与低潮）与人类现有的各种模式、结构、制度的反复变动，是并行前进的。
