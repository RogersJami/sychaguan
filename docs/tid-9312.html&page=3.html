小地方的感觉：

现在很多小地方依然是GDP挂帅。建开发区一是可以把土地使用性质转变成城乡建设用地，这样便于开展建设；二是可以把开发区从乡镇的基层选举变成开发区的直接任命，便于管理；三是可以拉动项目投资，容易出政绩；四是方便国家或上级对口帮扶资金或其他专项资金的申请；
