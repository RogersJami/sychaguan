每一个中国人都应该记得今天这个日子~~
这曾经是一个时代的结束,也是另一个时代的开始,但是当我们再回头,我们竟然发现,那个曾经的时代是多么得让我们难以割舍.
一个伟人,应该是有一个什么样的标准?这个标准我是知道的,然而却是不可说的,只有深深去挖,细细去品才会最深刻地感受到什么是伟大.
没有必要争论什么,只是觉得某些人士对于伟人的苛刻恰恰是他们最深刻的自卑,他们的心理底蕴是那样的简单:既然自己不成功,任何别人都不能是成功的.
一个羸弱到人尽可欺的民族,突然变成一只屹立世界东方的铁拳,你可以说这是多么的不正常,但是你却不能否认这是多么的伟大.当我们曾经是铁拳,至少我们知道,我们可以成为铁拳.


有人曾经说过,毛主席的功过是7/3开的,至少我并不认为如此.秦始皇的焚书坑儒奠定了中华一统的基础,如果没有秦始皇的过,中国可能还像欧洲一样分而治之.当然更大的可能是另一种,当我们作为一群分裂的小国,我们面对列强也许早就没有了等待的权利,而是像非洲南美和印度一样彻底沦陷.也正因为如此,我们又凭什么带着有色眼镜去看待文化大革命.伟人的远见卓识是一般人所不能理解的,也许我们不能理解,但是我们至少应该信任.
看看我们现实的社会,我们又能说些什么呢.当我们已经走出了文化大革命,我们又得到了什么?或许正像许多学者说的一样,文化大革命恰恰是还需要继续进行的.

别不多说了,深刻缅怀中~~
