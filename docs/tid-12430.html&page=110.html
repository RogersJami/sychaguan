所有人都在盼牛市，但现在我领悟了，牛市根本不是什么好事，而 ...

洋自干五罗斯义很早前在微博分析过，一到牛市，就是经济有问题、相对困难的时候，一到熊市，就是比较顺利的时候。所以00年代中前期，中国经济没什么大问题，股市熊得一塌糊涂。
由此可以推论，中国股市的逻辑和其他有本质不同，中国股市是一个大零和逻辑。我认为其内核是操盘者本人是将资本市场当做一个零时工具，而非长期工具。需要就用，用完就扔。什么时候需要呢？经济下行，需要筹钱了，总不能无限印钞票，于是来割韭菜了。股票上涨，赚的钱就是最后顶点接盘然后套牢的人亏的钱。套牢的，肯定不是游戏启动者。而参与者缺乏这种认识，反而认为资本市场是一个长期工具，于是始终不得要领。这与美股的逻辑是相反的。美股尽管有无数投机工具，貌似参与者个个都是花街狼只求咬一嘴发达，但整体市场是作为长期工具存在。
