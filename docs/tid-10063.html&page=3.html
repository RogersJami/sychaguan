gwa 发表于 2014-3-28 07:44:46

我也借这个话题来发挥一下：

一棵大树，外表光鲜的是粗大的躯干和茂密的绿叶，但实际上真正的实家伙却是深埋在地下的盘根错节。
如果你已经是或准备成为那些真正“被提拔”了的人，那么你必须需要呈现这样的“主根”特征：
1。有往深处和未开发处穿越的能力。（经过磨难的锻炼和考验，实实在在的工作风格），能开发自己的地盘。
2。有吸附副根和微根的能力。（掌控和发展），双向地输送和吸收正能量。结果就是：在那个巨大的地下世界里建立了自己的独特地盘（根据地）。
3。 最重要的一点是，牢记自己的“根”的本份：除了在充分的理由和必要的基础上，选择自行脱离主干外，永远与其他的“枝枝叶叶”们悉息相关，命运相连。
4。有些根是主干的亲生子，也有些根是后来再生的，或者是半道加入的。但是机会对每个根来说，最后都是平等的。

还想特别写几句话：当“根”们一旦被剥离泥土，暴于地上，再高的参天大树也就寿终正寝了。现下的那些“剥根断枝”大法，就充分地呈现了当局者的政治远见和斗争艺术。
因此我们这些小“根”们，除了研讨“上升之术”外，要牢记会辨道辨人，学习审时度势的本领。

说两点吧，都跟自己有关系，不一定对。

我做第一个工作。 你说的那几点，勤奋，参与感，人品，情商，算是都做到了吧。穿透力和得分两个，因为是新手，没什么可说。  在我那个组里，我只是个中等吧，或许偏下些。组面所有都比我强的人大有人在。

结果如何呢？ 大大领导派了个人来取代现领导。空降者慢慢招人积聚实力。终于等时机一成熟，把原来那些人，所有的，强的弱的全开了。我懵懵中误打误撞，先走了一步。而那些人在环境最差的时候丢了工作，叫天天不应的绝望啊。

要是没运气，楼主这些东西也是白搭。这是第一点。第二点就是，工作和理想，得找个协调方式。

如果你一头想往上走，楼主的建议应该不错。在我看来，好处是有成就感，生活待遇不错。坏处是得跟好的坏的人打交道，得忍着，得不断让自己适应甚至讨好别人。对处理很多鸡毛蒜皮，得琢磨别人，得勾心斗角，得玩政治。

如果你更注重技术，那仕途就看开些。找个适应自己的地方最好。当然，将来前途平淡。坏处也是不少的。

说到底，心安最好。如果一个人主动地选择了他的路，并接受这之后的所有好坏，就足够了。
