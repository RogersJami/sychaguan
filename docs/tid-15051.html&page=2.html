笨锄儿 发表于 2016-1-4 21:39:05

2015年初，法国查理周刊恐怖袭击就登上了世界新闻头条。11月13日，巴黎再次发生恐怖袭击事件。一头一尾，不知道是不是巧合。
这一年中国也不太平。自09年发生新疆暴恐事件发生之后，貌似民族问题引发的恐怖袭击在中国人民眼中就再没消失过。国内今年虽然没有发生过这类事件，但新闻各类反恐行动报道明显曾多。在加上天津爆炸事件、“东方之星”沉船事件，以及前一阵子的深圳塌方事件，我想很多人都和我想的一样，我们周边到处都有摄像头，但感觉上却越发不安全了。
今年8月去包头响沙湾，高速公路不足一公里的地方车辆寸步难行。到了跟前才知道，两百公里以外的鄂尔多斯开民族运动会，过往车辆所有人员全部下车检查证件。其时两名警察荷枪站立，粗声大气，气氛多少紧张。民族问题，什么时候都是个紧张。
