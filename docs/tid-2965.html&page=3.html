       书中有条暗线，蛮有意思：就是皖南事变，边区的生产运动到延安整风这段，这几件事时间上互相有重叠，但书中是分成几集（几个专题）来单独写的。
       1、皖南事变之前，遵义会议后，党内服了毛主席的军事才能；
       2、国民党的摩擦主义在皖南事变达到高潮，书中说，经过这事（处理皖南事变），党内更加团结，更加佩服毛主席高超的领导才能，这块是政治和外交；
       3、然后下一章写的是陕甘宁边区的政权建设和经济财政问题，我觉得这章的重要看点是财经问题的解决，让大家服了主席的经济管理能力；
       4、接着就是整风了......
       讲到陕甘宁边区财经问题上的主席怎么解决内部分歧的（分歧部分的观点估计也有亲苏秀才的一份，毕竟这属于“文治”，正统马列们最后自信防线。）。本章的最后，作者大段引用了当时贺龙的报告，这是一个彻底本土化的标志人物，报告上大赞主席的政策，甚至有光读《资本论》，不一定能解决中国实际问题之类话。

耐心的背后是高度的责任感

本帖最后由 zilewang 于 2014-6-4 20:21 编辑

终于搞了一套纸质《毛泽东传》，看着纸质书就是舒服。
长征途中，毛主席一直在不停地做着关键人物的工作，张闻天，博古，周恩来，以及一些中下层军官。
这只是罗列在书本上的一段文字，而真实的背景是，毛主席自“一边站”之后，长期得不到重用，意见得不到采纳，苦恼经营的根据地沦陷，曾经兴盛的红军，如今成丧家犬。
众人皆醉，唯我独醒的赶脚，对于一个才华横溢，抱负不得伸展的大丈夫，是一种何等的折磨。屈原不是有离骚问世吗？
做为普通人，可能无法真正理解这种痛苦，但对照我们工作中的境况，自会有一番体会。工作中，你有过欲展抱负无门的时候吗？你有过不断向领导反映意见，却不断否定，还受贬冷宫的赶脚吗？
我相信，在我们的工作中，我们肯定都遇到过。曾经，我遇到一个神猪一样的领导，无论你如何苦谏忠言，得到的都是失望加怀疑！我曾经的做法是，妈的，此地不留爷，自有留爷处，老子不干了。沉默代替了进言，磨洋工，混点成了每天的工作。一度彷徨迷茫，是自己的能力不行吗？还是这个领导能力差火？曾经困惑了我好长时间，最后以一种慢性自杀的方式结束一份长达六年的工作。
也许某些人，不会如我这般极端，无知，但是，生活中，我们曾经坚持不懈地向团队贡献自己的才智，进言苦谏，在不得赏识、冷落之后，常常以一种磨洋工或者软对抗的方式，诉说着这个团队的不满。撂挑子，或者孤芳自赏，成为了我们证明自我价值的方式。
我也见过一个因猪头小队长对自己有知遇这恩，工作任劳任怨，苦苦为小队长支撑着摇摇欲坠的局面。他也曾经以此为荣：看，关键时刻，还是要靠我这样忠心耿耿的嫡系，你们这些家伙，过河拆桥。其实，在最后猪头小队长耍了他一次之后，他也心灰意冷，给自己背上了道德的包袱。
现在看来，这样的做法，仍然显得小。
看过毛主席遵义会议之前的所做所为，我是真心的知道自己的不成器在哪里！这种耐心，以及耐心背后的高度责任感，真正做到了，不避斧钺，不离不弃，这是真正的大公无私。
主席和那样站在河边指点江山的书生不同，是真正做事的人，真正有责任心的人。这就是自强不息，而不是我们嘴上说的“自强不息”，不是书生意气的酸文假醋，是拥有责任的“自强不息”。
工作中，生活中，我有足够的耐心吗？我的耐心被消耗掉了吗？忙总谈到自己每天坚持给夫人一封信，24年坚持下来，我曾疑问，这我还坚持不下来！忙总回答很简明：你每天都要坚持吃饭喝水一样简单。

zilewang 发表于 2015-11-1 09:49:42

RE: 913事件的当代版

最近再看《毛泽东传》，以前有些朦朦胧的地方，似乎慢慢明白其中一点意思了。不过，大部分还是知其然，不知其所以然， 甚至也就是当个故事娱乐一下，还没上升到大牛那种境界，可以将《毛泽东传》当成方法论和工作指南来认识。

不过，刚好看到913事件的经过，联想忙总曾与人讨论说，原四野老人一直在为林越翻案，并且以林彪算无遗策的谨慎个性，为什么在末期的斗争中表现如此低能，还被亲信小人绑架，这是迷思之一。
印象深刻者，莫过于周警觉到一架直升机的异动，并不断追查下落，步步进逼，传记中只交待了最后的结果，林彪事觉败露，仓皇出逃。

瞬间的印象是联想到几年前的护士长，似乎也有异曲同工的地方。背后有一股力量不断给予护士长压力，调查其弊案，最后不得不铤而走险。

一切看似荒唐的表象，似乎背后都有着不那么简单的复杂博弈与纠结。这里面的矛盾，到底是毛与林的矛盾是主线，还是林与其他什么人有更深的矛盾决定着林的行为，迷！
