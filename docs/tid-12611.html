雾霾（大气气溶胶）

雾霾其实也称为大气气溶胶，或者是气溶胶的一部分。 其中的界限并不明确。

从 三个方面讨论 雾霾

1. 雾霾的来源
2. 雾霾在空其中的存在
2. 清除

空气中的非分子粒子统称为气溶胶。

气溶胶的来源主要是
海盐 风吹起海浪，空中的海水，水分蒸发，剩下的盐分被吹到空其中。每年的产生量
为：50-200亿吨。
气相转化来的粒子，也就是 云，雨，雾。 每年产生量为35-208亿吨。
沙尘，也是伴随剧烈天气过程被吹到空其中， 每年产生两为0.7-180亿吨。
其他的一些来源量都比较小， 主要有： 生物气溶胶，烟尘，火山灰，人工气溶胶。

颗粒物的粒子半径在1微米以下的粒子，可以在大气中长期存在。 1-10微米的粒子存
在的时间时间尺度在1-10天。 超过10微米的颗粒物很快就能在发生源附近沉降。

海盐，沙尘，云雾等气溶胶的产生量大，通常粒子半径也比较大。 粒子半径比较小
（小于1微米）的部分，可以在整个大气中长期存在，并且混合的非常均匀。 比如，
小颗粒的海盐气溶胶在大陆和海洋上空的分布非常相似，说明存在有效的长距离传
播。

生物气溶胶，烟尘，火山灰，人工气溶胶。 这些气溶胶量不大，但通常粒子半径比较
小。 火山灰可以进入高层，甚至平流层，也可以长期存在。 生物气溶胶也就是花
粉，病毒，细菌，真菌孢子。 最高可以占到气溶胶总数的30%。

人工气溶胶可以占到气溶胶总数的10%。 每年的排放量比工业化前（1860年为标准）
增加2-10倍左右。 二氧化硫增加增加最多，从2.4万吨增加到73万吨。

人为排放的气溶胶，主要是通过燃烧过程产生。 包括火力发电厂，汽车燃烧，家庭做
饭的燃烧等等。

再说气溶胶的清除，主要有三种方式。 湿清除，随下雨的过程回到地表（海表）效率
比较高。重力沉降，这个效率比较低，表较慢。 还有就是动力， 大风一吹，都扩散
了。

好了， 再说说城市里常见的“雾霾”。 那种灰不溜秋，秋天里常见的雾霾天。 主要成
分还是液态水粒子。 这个浓度大，照相很能出效果，比如 央视大裤衩不见了的照
片。但这还是大自然的杰作。 雾这个东西，工业化社会之前就存在。

“都不能开窗，开创就是一层灰”。 这个主要是大粒子。 一般在当地产生，当地沉
降。

“马路上的一股味”。 主要是汽车燃烧的尾气。


大气中气溶胶天然存在， 人类活动也不可能不排放。上面说了， 认为活动主要是通
过各种燃烧产生气溶胶。 控制排放，必然伴随找工厂关门。 电厂肯定不能关门， 车
可以少开。 关键是汽车工业那么长的产业链养活了那么多人 ，当真关的起么？

不要说关掉汽车工厂。 做个调查证明汽车是污染元凶都没这个胆子。 技术上做这个
调查很容易的。 每年10-1 春节放假期间汽车使用量下降。 找个相同天气条件下的
时段， 对比下PM2.5 就出来了。

所以，雾霾问题，真的是政治问题。
