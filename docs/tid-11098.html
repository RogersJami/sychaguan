自制琥珀桃仁

本帖最后由 思炎 于 2014-6-21 10:36 编辑

核桃自古以来就是中国人的滋补佳品，最为人所熟知的是吃了健脑，核桃有脑黄金的称号。

核桃可生食、炒食，也可榨油、配制糕点、糖果，或作为零食。
核桃不仅味美，且具有很高的营养价值，《本草纲目》记述，核桃仁可“补气养血，润燥化痰”。中医认为核桃性温、味甘、无毒，有健胃、补血、润肺、养神等功效，经常吃核桃既能健身，又能抗衰老，核桃也被誉为“万岁子”、“长寿果”。

现代研究认为，核桃中的磷脂、不饱和脂肪酸对脑神经有良好保健作用，而且具有预防动脉硬化的功效; 核桃有防止细胞老化的功效，能有效地改善记忆力、延缓衰老并润泽肌肤；对慢性气管炎和哮喘病疗效极佳。核桃树叶中含有抗生物质，因此也有杀菌的功效，能够缓解体内的炎症，防止心脏病、癌症和关节炎。

英国的研究发现每星期吃2~3次核桃有预防糖尿病的作用，减少两到三成的风险。
美国饮食协会建议人们每周最好吃两到三次的核桃，尤其是中老年人，其中所含的氨基酸、油酸以及抗氧化物质等对于保护心血管、预防冠心病、中风、老年痴呆等都有明显的功效。

核桃好处虽然多，但也不宜一次食用过多，以免影响消化，上火。 一般来说，可每天服用大约20~40克左右，即为4~5个整核桃。


在国外超市只能买到生核桃仁，据说生吃营养最好，但生核桃仁表面褐色薄皮带苦涩味，总感觉有点不舒服。 N次买来，边吃边想念国内的零食- 琥珀桃仁，不怕不识货，就怕货比货，所以每次都是吃了一些，叹息一声，束之高阁。 再想到时，拿出来已不新鲜，扔掉。

真是，食之无味，弃之可惜。 这次决定不再浪费，自己动手制作琥珀桃仁。

第一次试做就旗开得胜，而且做法很简单，几分钟就搞定，特来与茶友分享。

我这里用了250g 核桃约 3.55 英镑， 在国内的价格更便宜些。 自己制作的好处是更安全卫生，而且可以自己掌握口味，甜度（放多或放少冰糖）。

可作为招待亲朋好友的冷盘，下午茶点，或做下酒小食。 老少皆宜，特别是小朋友会喜欢吃。


做法：
1. 将核桃仁放水里煮熟后捞出，沥干水分待用 （不用剥皮）
2. 热锅，锅中加入少量清水，放入几粒冰糖（我放了5小粒）和一勺蜂蜜，用小火熬出稍粘稠的小泡
3. 将核桃仁放入锅中，拌匀，使核桃仁均匀裹上一层糖汁，待收干一点后盛出。
4. 锅里放适量油，再放入核桃仁小火慢炸致酥脆。 捞出装盘即可。

* 因有糖的分隔，所以桃仁并不吸油。也可捞出后，立即撒上白芝麻拌匀。
