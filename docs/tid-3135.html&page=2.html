http://www.dfdaily.com/html/8757/2012/9/11/859910.shtml
　　中新网9月11日电 日本政府不顾中方一再严正交涉，11日宣布“购买”钓鱼岛并将与所谓“所有者”签署合同。据日本新闻网报道，日本政府11日上午举行内阁会议，最终通过了从2012年度国家预算中支出20.5亿日元“购买”钓鱼岛本岛、北小岛、南小岛的决议。

　　报道指，日本首相野田佳彦依据这一决定，向财务大臣安住淳作出汇款的指示。

　　此外，日本政府预定于11日下午，派出内阁官房副长官与钓鱼岛所谓“所有者”正式签署“合同”。此后，这三个岛屿即将实现“国有化”。
