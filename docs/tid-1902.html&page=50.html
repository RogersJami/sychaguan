初期应该有引进一些外援吧，毕竟人家玩了几十年。

主要是两块：一是造船，目前陷入寒冬，很多过剩产能会被淘汰，皮之不存，毛之焉附，忙总说技术工人无法储备，没有单子吃饭就是问题；二是海工，这一块接外单我们缺乏经验和技术实力，现在初级阶段，内单基本上要靠三桶油，特别是中海油（其它两个短期内难成气候）。我的看法，国家在这一块谨慎有余，进取不足，胆子太小，步子太慢。当然，进深海有风险，等中海油五型六船深海舰队站稳了再发力是可以理解的，不过没有足够的项目拉动，产业链起不来。同样的情况在航母项目上也存在，是冒风险一步到位，还是求稳妥按部就班，不完全是科学，也需要胆略。有了项目拉动，资金保障，人有活干，技术才能突破。靠外援是一个阶段，核心技术核心人才你肯定引进不来，好走路快走完了，以后关键还是要靠自己，陆、海、空、天都如此，其中“海”这一环最薄弱，人才技术不足只是表象，不下决心砸钱，不靠大项目拉动，就算有人才有技术也没用。搞企业的，讲市场拉动；搞科研的，要项目带动。市场变化快，技术进步快，都需要在战争中学习战争。

武当七瞎 发表于 2012-5-15 00:01 static/image/common/back.gif
海里面是怎么毒鱼的？这可不是小池塘，想象不出来。

所谓毒是食物链累计下来的，越往高端，累积的毒素越多，之所以叫毒，就是对你身体没好处，而你又很难排泄出去的东西，所以初级动物有没有毒主要看环境是否污染，而食物链顶级动物往往跟环境没多大关系，它吃了大量动物，结果毒物没被处理干净，集中在体内了。
处理污水也有用动物干的，就是利用动物的这个作用。
毒鱼大多是吃活鱼的物种，相反食物链初级动物，比如虾和螃蟹往往没事。早期地球是充满有毒气体，水里也应该一样的，都是火山啥东西搞的，经过历代植物动物们的努力，把有毒物质沿着食物链往上送，大型动物们相当于毒物的矿了，终于给搞干净了。
从这个角度看，如果把人当食物的话，最环保的应该是印度人，这些家伙吃素，累计的动物毒素少，西方白皮猪次之，吃牛肉，牛吃草，算二级，我帝国人最差，啥都吃，累计的毒素不可小视。
