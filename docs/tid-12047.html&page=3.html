现在才有机会看到这篇帖子，很遗憾没有参与前面的讨论。

关于德，整体上偏向于忙总的看法。做事情本来很难，好多时候感觉下三滥的手段不会少用，这里也不好用德来评价，以目的为导向要容易的多。德有时很虚，不妨多做些事展示德（尽管有些阴）。但是整体上要内心足够强大，并且少些私心。

前面您提到的银行考察两家冲上市民企的故事，银行那边估计有更多的考量，不知道是不是这样？

宣传凝聚人心时效性比较重要，因为这一块儿出问题还是执行上的问题（宣传好了可以更多的减少失误），团伙做事，考虑到个人的利益，还是能表现出所有优秀团队的素质。

好多公司的一线管理者只知KPI（感觉碰到的都是这样的），竭泽尔渔，很大情况在于中下层和高层的利益与出发点都不同。一线管理者能清楚了解上层的意图就很重要，如果能在管理方式和流程上解决这个问题就完美了！
