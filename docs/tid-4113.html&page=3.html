太祖毛爷曰：李恪英物，李治朽物。以唐宗之明，焉能不辨？然终听长孙无忌之言，可谓英明一世懵懂一时。

跟着燕大学习，恰好小时候看《今古传奇》，有一篇太宗立储，有一位仙风道骨的老人站在标题上惆怅，就来摘几个点，胡诌几句，还请大家海涵不妥之处。太宗比无忌高几个数量级，一个顺手、听话的人，用着用着就枝繁叶茂，再砍似乎就砍住自己的肉，像和珅的的处理方式似乎更便易。
具体到立李恪还是李治的时间点，李恪只有24岁，太宗玄武门27岁：
1、年龄差别不大，在太宗死后姑且认为没有改变策略，变得骄奢淫逸，太宗恐怕看出这点，似乎可以认为“李恪的文韬武略更像才艺”；
2、手下的良臣猛将有几个数量级的区别？
3、当权的良臣猛将都在占地、美女、生孩，拥护李恪和无忌对砍是要划拉一下算盘，是雪中送炭还是锦上添花？失败可是株连九族。
　综上，太宗并不认为自己死后，李恪有哪怕一丝一毫的胜算，自己倒是可以痛下杀手帮李恪，可是（李恪的文韬武略更像才艺），开创自己该做的都做了，本来下一代就是守成，也许太宗认为自己100分，李恪和李治只是4分和2分的区别，选李恪有李唐灭亡的危险，李治没有。
