cheng 发表于 2014-7-23 16:35:31

印度的癌症药物的确有着很大的价格优势，我上次去印度旅游时，朋友托我从印度带治疗癌症的药物回来，因为在国内买一个英国生产的抗癌药物，买2瓶的价格，我在印度可以买10瓶了。因为我第一次去印度，且还是买关系到别人性命问题的药品，所以很是害怕买到假货，在斋普尔看电影时碰到一个医生家庭，问他们要到哪里可以买到正宗的药品，是不是只有到医院买才不会买到假货？他们很是诧异，他们说在医院附近都有卖药的地方，只要去买就可以了。我们到了新德里打tutu车去到一家医院，医院里的人让我们到外面去买药，走到一片很破的房子，就是像灾区搭建的简易房，那就是他们的药房，当时我都有点绝望了，不知怎样才能找到个正规的地方去买药。谁知就有许多印度人从医院中出来后，就到这里拿着医生的处方买药。我同学说，也许这就是印度的正规药房。随后在外面观察了一会，好像觉得不会有假药，便上前去打听，店主一听说我要买10盒药，觉得我是个大客户，因为没有那么多现货，他立刻请我和我同学坐在他们很简陋的药店里，并请人去买奶茶来招待我们，他马上打电话请人送货来店里。整个过程不知为什么我觉得很是放心，并无担心上当受骗的感觉。且店主因为我们购买的数量，还主动给了我折扣（我当时根本就没有想到要讨价还价，有可能是在中国买药从没打过折吧），顺利买好药。

我们晚上到新德里机场返回中国时，在机场也发现了在售卖同样的抗癌药物，且有很大的广告牌，说购买两盒以上可以打九折，而我购买的折扣大概在8折左右，可见在印度市场上的价格做得还是很规范，并没有因为是外国人或不了解市场等乱开价。这个让我对他们的印象还是很好的。

当然每个国家都有阳光与阴暗面，我们在瓦拉纳西旅游是，乘坐的tutu司机就因为我们不答应继续包车游玩，而故意把我们扔到一个我们都不知道是哪里的地方，还好是大白天，只是给我们造成了一些麻烦，而不是什么安全问题。

当然，工作中也和许多印度同事打交道，总体发现如果印度同事要你做的事情，他会追得很紧，但如果你让他做的事情，就比较难得到。不过，我觉得也许是因为我追得不紧的原因。而印度同事追我追得紧，也许就是印度的国情造成的，如果不追，一件事情就无法完成，所以即使针对非印度同事，他们也会用同样的方式来处理，中国人会觉得吃不消。而中国人不喜欢紧紧地问别人追要结果，追得不紧，所以也就很难得到结果。

上面罗罗嗦嗦地讲了我的一些观察，大家见仁见智的理解吧。我总体认为受过高等教育的印度人都是很聪明的人，他们观察和理解别人的能力都很强，而且印度人通常表达能力较中国人强，且语言方面（英语）比中国人强，但在做具体事情时，好像不紧紧跟在身后追讨，比较难有结果。
