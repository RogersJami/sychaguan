正宗的桂林米粉是马肉米粉，和南宁老友面、柳州螺丝粉并称。但一直对桂林米粉和老友面不感冒，不知道马肉 ...

马肉米粉只是桂林米粉的一个分支，只不过从名称上看最有噱头所以最出名罢了。桂林米粉的特点在于各家店不同的卤水，好卤水才是桂林米粉的化龙点睛之笔，在桂林中山路正阳步行街后面有一家米粉名店取的名字就叫“卤神”，可见卤水对于桂林米粉的重要性。卤制好的马肉肌红肉粗有嚼劲，可以片得很薄，用在米粉里作盖面菜即养眼又美味，关键还可以省料。正是“从南京到北京，买的没有卖的精”，桂林米粉过去到现在作为一种街头小吃，用马肉正是其理。当然现代马肉很少有得卖，但桂林米粉里的配菜还不是少不了一味卤牛肉也是同理。
