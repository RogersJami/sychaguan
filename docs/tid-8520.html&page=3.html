所谓的自主意识，全是从上面3步中不断复杂叠加而来。而这三部貌似技术上在电脑上编程实现并无难度。在巨量信息和超强处理的叠加作用下，这个储存和处理信息的后台最后会不会变成终结者天网？

葡萄所提及的大数据量级下的条码相关应用模式碰巧本人几年前曾经接触过数次，其中一次就是这个药品条码。遗憾的是当时对大数据这个概念还不能真正的理解其价值，不然也许当时做出的判断就会更加坚定——而也许有些事情就这么错过了。
“一物一码”与“一类一码”最大的区别不仅仅是数量级的提升（就举二锅头为例，一类的二锅头的一维码都是一样的，但一年的出货量可能上了百万千万瓶，也就是“一瓶一码”的模式下数量级提升了千万倍），更重要的是“一物一码”的模型下将此“一物”也就是这一瓶二锅头的全生命周期数据都实现了收集和管理（如果没有这个“一码”，也就不存在某一条数据了）。
您提及的药监管理，其实只是一个小领域——或者说不得已缩水之后的领域。在最开始的时候，这个条码管理是针对于更加广阔的领域的，只不过遇到了各种问题之后，缩到了药监领域并且得到了切实的落地，当然这已经是功德无量的一件事情了。至于给您开发出来的这个借口，在下认为技术原因的可能性并不是那么的大。

作为一个CS专业的学生，来从技术和科幻的角度歪歪楼

Big Data的概念与思想实际上并不是这几年才有的。数十年来CS研究者一直渴望着对海量数据的处理、挖掘能力。而更让无数CS研究者、思想家、科幻爱好者着迷的梦想，是人工智能。而这个梦想随着硬件本身性能的增长，分布式计算架构（如MapReduce, Hadoop之类），以及Machine Learning领域的不断进步（从SVM到现在的热点Deep Learning），已经不是那么遥不可及。

葡萄提到阿里云的计算能力，BAT三巨头在国内是在大数据计算架构和机器学习（主要是Recommendation System，Image Search）上走的最远的，虽然远不能和Amazon, Google这样的巨头相提并论。无论国外还是国内，对分布式架构的研究都是工业界引领的，如Google的MapReduce和BigTable，这个动力来自于业务规模的爆炸性增长。

人类技术的发展，不是线性的，而是爆炸性的。谁也无法预测三十年后，科技会发展到如何的地步。而科技的发展，也导致着生产力和生产关系的巨大变化。例如《黑客帝国》，就是对未来人类文明的宏观展望。

在我的理念中，人类未来的一个重要问题，是我们的文明形态，甚至生命形态是怎样的。打个不完全切合的比方，人类文明是否有可能演变成《StarCraft》中Zerg，或Protoss的文明形态？
探讨这类问题，借助已有的影视作品是比较方便的。以或许是世界上最著名的动漫——EVA为例。在EVA中，我们看到了目前无法理解的科技，如初号机、使徒，表面上是无坚不摧的变形金刚式的机器人，但实质上是和人类同源的巨型生命体。它们装载了S2机关，从狄拉克之海中获取无穷无尽的能量。如果借用犹太教的说法，使徒获得了生命之果，而人类获得了智慧之果。

而人类获得智慧之果的代价，类似于圣经中巴比伦塔的记载一样，就是人类从此失去了心灵上的联系。人类之间沟通的障碍，思想上的相互猜疑，也就是EVA中的能扭曲空间的AT Field。EVA的最后，掌控着人类命运的组织SEELE的最终目标，就是让人类的心灵和肉体完全融合在一起，形成一颗生命之树，也就是“人类补完计划”。人类在思维和肉体上得到了补完，代价则是失去了个体自由意志，人类作为一个整体到达了下一阶段的生命形态。在EVA的框架下，修改人类记忆，人与人思想上的直接交流，都不是不可能。而当人类被补完后，这些目标也就失去了意义。

除了EVA，StarCraft的背景也同样宏大。其中的Zerg和Protoss都是Xel'Naga 创造出来的生命。其中，Zerg是纯粹的肉体至上的种族，它们不断吞噬着别的生命，在自然界的恶劣环境中不断进化，获得了在星际飞行的能力。值得注意的是，无论种群的规模在星系中如何扩张，它们始终只有一个自由意志，就是Overmind。也就是说，整个Zerg种群实际上只有一个生物，就是Overmind。

或许有茶友看过江南的《上海堡垒》？里面的Delta文明实际上就是以Zerg为蓝本。里面有个描述是很有意思的，形象地展示了这类生命形态的强大之处。Delta文明向地球上投放的捕食者（实际上相当于母体分裂出的微不足道的一块组织），为了获取人类计算机中存储的Alpha文明遗留下来的技术，迅速进化出了利用触手读取硬盘的能力。

这类文明的恐怖之处，在于意志上的绝对的服从，和肉体上的不断进化。实际上，我觉得人类文明和这类文明有许多共同点。人类虽然每个个体看似有着独立的意志，但群体又是无意识的，会被洗脑，会被宗教、信仰等诸多工具控制。从人类这个种群上看，我们不过只有几种意志罢了。而个体的创造力，又保证了人类文明可能的进化，类似于Zerg个体的进化能力。只是，我们的这种文明形态，与Zerg相比还是太初级了，效率过低，微观上意志传达不够有效率，进化能力也偏弱。（当然与蚁群相比，我们是迈出了一大步）

一开始，我是想把机械文明和Zerg，人类这种有机文明分成两类。但转念一想，它们可能殊途同归。机械文明的终点，不会是《黑客帝国》或《变形金刚》中描述的场景，很可能是进化成硅基生命，类似于《孢子》游戏中的Grox文明，它们从物质上同化一切不属于自己的存在，不断扩张。有意思的是，《变形金刚》的动画续集《Beast War》中，变形金刚也最终与有机生命体融合，开启了新的纪元。

或许，应该把生命形态分成物质形态生命和能量形态的生命，就像Zerg和Protoss？不过这个就太玄乎了~~

随便写写，希望和大家一起讨论。

之前问过3D打印能否解决质量管理问题，因为质量控制做不好分散生产就是沙地上的高楼。现在看来大数据并不能把生产结构抹平啊，有可能抹平的是管理结构。不过当年电子政务推不动也不是技术障碍造成的吧。
拉开说个题外话，牛顿三定律的伟大，我认为就是找出容易测量和操作的，对足够多现象产生影响的物理变量，有了这些基本描述概念，物理学家才能分工合作建立理论体系并指导技术进步。当然单个变量是有局限的，力的单位本身不能直接描述电磁强度，各种量纲组合，才能描述现实世界。
现在，大数据技术本身只是提供了新时代的媒体技术，但是只有找到合适的变量定义，有意义的描述方法，这个媒体才能连接思维和物质世界。
现在我们能给每一个出产商品提供GUID，那么假如能解决录入和追踪，那么可能做到对整个社会经济运行建模，观察足够模型在发现基本规律之后，才能找出有意义的变量，然后尝试针对性调整。
