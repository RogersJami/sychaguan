附近小区居民不满。

　　“还我呼吸，还我生存，还我健康”“损民利己排毒气，不把生命当回事”。昨天9时许，龙华街道第四工业区信隆实业有限公司门口，数十居民因不堪忍受工厂常年的废气粉尘排放，排成排站在工厂门口，向厂方讨说法。
...
“前段时间我们小区业主投诉，昨天区环保局约我们座谈，双方不是第一次谈了，谈了很多次了，但没有效果。昨天深夜12时以后还在排废气，一直排到今天早上。所以我们现在通过非正常的渠道，来引起社会的注意。今天我们在这里站了一个多小时了，厂里的领导有没有出来一个？根本没有诚意。”

　　尾声：又一次协议达成

　　昨天17时30分，业主方面给记者发来一份由几方达成的谈判协议。在这份由龙泽榕园业主代表、信隆实业、政府相关部门共同签名的协议上，作出了如下承诺：信隆实业的烟囱要往下弯排，以不超过厂区的铁墙高度，并往厂内排放的气体要达标，符合环保标准（工期一个月）并对粉尘和噪音问题做了相关协商。

　　居民疑问：

　　呼吸都有困难，为啥监测合格？

　　...
　　记者感受：

　　待两分钟就咳嗽不止

　　在两名小区业主带领下，记者来到龙泽榕园小区8楼楼顶现场感受一番，记者待了两分钟就咳嗽不止。

　　龙泽榕树园位于信隆实业的正北侧，中间仅一墙之隔，之间的围墙有铁板加高相隔。站在8楼楼顶非常清楚地可看到附近的锦绣花园的围墙铁挡板——靠近厂里这面锈迹斑斑，靠近小区这面则还保留着原来的颜色。

　　“这就是他们排放的气体腐蚀的结果。”这位业主告诉记者。在小区正南方，记者看到一个正在向外排放气体的烟囱，时而排放黄烟，时而排放白烟，一阵微风吹来，一股让人恶心的气味扑鼻而来。让记者直有咳嗽的冲动，继而有头晕的感觉。记者从8楼下来后，那股让人恶心头晕的气味才有明显的减弱。

　　“这还是最轻的，今天有人来，他们不能排放太多；平时到了晚上，这里的气味简直让人无法忍受。大人小孩一起在家里咳嗽，窗子常年不能开，就这样，恶心的气味还是无孔不入。不少家庭的小孩已经出现了不适症状。”

　　“除了废气排放，这里的粉尘和噪音也让人无法忍受。白天还好，晚上的噪音简直让人无法入睡。”

　　这么恶劣的环境，为什么还要在这里购房置业呢？
