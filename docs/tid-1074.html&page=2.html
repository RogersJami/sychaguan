台湾的政治环境，从李登辉时代开始冻省，就进入了迷惘和进退失据的时代。

所谓冻省，就是李登辉裁撤了台湾省在“中华民国”体系内的行政结构，当初极力支持李登辉的省长宋楚瑜被边缘化，迈开了国民党直接分裂的第一步。

而此前，在台湾政治体系内，对复兴中华文明的力量，还是期待政治改革的风潮，把万年国代吹走，换上清新的政治力量，也期待新一代的政治人物，如宋楚瑜，马英九，焦仁和等等新星能梯队完成中华文化的复兴的。

当时台湾的有识之士，在宋楚瑜和李登辉同流合污之后，就开始对国民党失望，但是民间已经国民党中层还是对体制抱有希望的，但是李登辉冻省，不但从体系上废除了中华民国的法统秩序，也直接将铲除了宋楚瑜势力，同时彻底断送了体制对复兴中华民族文明的阶层的召唤，最直接的结果，就是把原来中华民国的国民，变成了台湾岛的岛民。
