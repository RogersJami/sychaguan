说到足球，我和欧洲足球圈子的人稍微接触过，简单说说。
从运动项目本身来讲，足球和橄榄球是所有集体项目中，最像“战争”的两个项目，趣味性和观赏性很高。
欧洲是足球中心，那里普通大众对足球的兴趣是几代人传下来的，这一点其它地区无法相比。但足球这个行业阴暗面很多，黑帮控制、赌博、药物、近亲繁殖等等，数不胜数，国际足联，欧足联、亚洲足联等等，腐败和官僚简直不可想象。举个例子，90年代，意甲联赛所有重量级别球员的转会，都必须经过尤文图斯老板拉涅利家族的点头。
楼主提到586会重视足球，我想首要原因在于这项运动可以作为社会情绪宣泄通道的最佳选择，毕竟城市化，人们大多数都很压抑（不展开了），需要有宣泄的通道。
中国人是适合踢足球的（相比篮球）。现在的问题大家都知道，确实需要浴火重生，其实很简单，全面引进欧洲的培训体系，包括青少年。

看了您的帖子感觉中国足球还是一个分配利益问题

青少年足球得不到应该有的资源分配


任何行业都是挣钱分钱把戏，足球大方向上是国家和俱乐部责任划分的问题，是花钱出现的问题还不是分钱出现的问题，以前计划体制下，培训球员责任全部由国家承担，队伍属于国家，比赛跟国有企业一样，可以说质量惨不忍睹，球员待遇很低，社会地位不高，身体对抗能力非常糟糕。但是即便如此，同样计划体制下的北朝鲜依然很早就在世界杯上打入8强（干掉过意大利），被淘汰也是领先几个球的情况下翻船的，可以想像对手做裁判工作应该不小。

改开后如果把球员看作国有资产的化，俱乐部立刻成了国有资产的收割者。加上高额奖金的刺激，几年后引发了国内大规模足球热潮，当时足协冒出过不少口号，给人感觉都快拿世界杯了，连亚洲都挡不住中国男足了，结果被连续打脸，直到今天。这不过是计划体制培训球员和市场经济运作俱乐部两个波峰叠加出现的假象而已，马上到来的就是低谷，随着国有训练体系的瓦解，足球成了俱乐部的裸奔，自然后续乏力。

分钱机制现在成了忽悠投资人的陷阱，很多人被忽悠进去（包括赵本山），结果颗粒无收，当初的霸主辽宁队，一度固定资产只有一部电话。俱乐部成了击鼓传花，跟股市一样，国外教练一概水土不服，实际上很多国内教练成了幕后交易的白手套。几年不到，足球成了忙总说的赌博工具。
