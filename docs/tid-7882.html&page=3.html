古代的部落战争，胜者杀掉败者很正常，没有余粮的话就连奴隶都不会留。党同伐异才是人类有了组织行为后就一直在做的事，而普世价值实际是在充当党同伐异的手段，只是高下有别。
前面有提到养狗的事，基本上几个问题可以看到不同人的立场。比如虐待狗的问题，养狗伤人的问题，吃狗肉的问题。

实际上如果全球的生物有能力投票选择最可恶的生物的时候，人类是不是会以绝对高票当选？
而人类如果遇到科技水平远高于地球的外星文明，会怎么样？
当力量差距过大的时候，没有平等可言；当生存空间狭小的时候，没有自由可言。
归根到底，人不能摆脱自身和环境的限制侈谈“普世价值”。毕竟人类可不是神马“Fiat Lux”的大能。

在国际博弈中，普世价值的功能其实是有区分的：目的和手段

在中东北非的阿拉伯之春中，对于上街的人来说是目的，这真是要追求的目的。而对于国外干预势力来说，就是个手段，颠覆现政权、进行社会解构的手段。从建是有成本的，国外的干预势力凭什么要去付从建的成本。现实结果显示是没有从建，民生比解构前更加糟糕。也就是说他们提出的目标其实就是战略欺骗，为了自己国家谋利，通过思想指导行为的规则，把别人往沟里引，自己站在岸上往下扔石头，岸边林子里的果子就他自己吃。

在国家间的博弈中，如叙利亚的博弈中，国家间的价值观其实就是其中的影响因素之一，而且还是拿来往脸上贴金的，外交场合拿来装裱自己高尚的。如果站在美、俄、中、欧洲、沙特等的立场考虑其言论都是有道理的--选择性前提条件下的逻辑自洽，没谁没道理。可事情还是在不停的变化，对某方有利或对某方不利，说明起作用的不仅仅是各说各的道理而是还有其他，都有理就相互对冲了道理这个因素。起决定作用的还是实力的方向性上的对比和利益的归属---秀才遇见兵有理说不清，拳头大的是爷。

价值观或者理解成问题解决方案中的核心因素之一，关键是可执行不。这个得好好权衡。
