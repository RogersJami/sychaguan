这里我需要厘清一个中国社会分析的现状。大体上，还是主席在十大关系中阐述的问题与矛盾。这里我不是说， ...

雄文啊雄文，佩服
只要人类社会，不管中外，只要存在以私有财产为基础的政治经济分配存在，并且社会存续本身依旧基于人与人之间的血系与血缘关系为纽带的组织形式为社会基本单元。以攫取私人或者利益小团体，并通过血系纽带为传承的固化倾向就不可更替。

只要人性如此，那么社会就终究会陷入困境，并需要重组。
以前我在读十三经的时候，总觉得儒家如此重视道德，将道德完善当做社会发展的目的是一叶障目，我总认为社会发展的目的还有很多，比如法律治理下的秩序，生产力的发展等等。
后来想的多了发现，只要这个社会绝大多数人的道德水平没有出现极大的提高，那么所有的完美社会设计（比如共产主义社会）都是不可能的，人性将导致我们目前所深深厌恶的所有问题必然出现，比如贫富差距。如果我有一笔很大数量的合法收入，我想不想传给我的子嗣呢？当然想。但所有的这些看似合情合理的想法，将慢慢积累一个逐渐不平等的社会，直到这个社会自己崩溃。
我现在非常同意儒家的一个看法：每个人在道德上的提高是非常重要的，人人都成为尧舜，社会才能长盛不衰。
但，这是不可能的。
