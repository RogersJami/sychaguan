美国飞行员回国后在媒体前控诉，飞机一落地就被PLA四面包围， ...
      所谓应急操作程序是指销毁绝密的密码。以前的谍战片中密码都是用一个本子，现代科技发达以后密码本已经取消了（因为很庞大，人脑绝对不可能全部记下来。如果人脑能记下来的密码，那么很容易被现代计算机快速破译），而是采用电子芯片的形式保存密码。在紧急情况下，这个东西是必须要销毁的。你的命可以没有，这个东西不能落到别人手里，这个别人是指除了自家以外的其他任何人，不管是是敌是友。
       此类电子侦察飞机上都配备利斧，在舱门打开前，他们先是拔出密码本芯片，要用榔头敲碎。其他的一些关键设施就用斧头劈，能劈烂多少就劈烂多少。我们的兵在外面啊，命令他们限时下机接受投降，要不然他们还要继续破坏设备。就因为处置迅速，所以给我们留下了很多宝贝。
       关于军事保密通信方面我再多说一些。基于现代计算机技术的高度发达，密码的算法已经非常复杂，密钥的长度也非常长。但是即使是一些看上去非常厉害的密码体系，如果遇到一些数学天才和强大的计算能力，再加上日常的监听记录分析积累，你这个密码是完全可能被破译的，比如山东大学的一个女教授就破译了MD5密码体系。比如现在互联网常用的加密体系，如果你只用128位的密钥，那么黑客可能利用控制的一些计算机，利用最简单的穷举法，大概用30台电脑（具体数字可能不是非常精确）花一周左右时间就可以破译。现代数学理论证明，没有一套密码体系是不可破译的（理论上除了将来的量子通信技术不可破译，因为量子技术有防侦听能力）。所以现代国家为了防止泄密，要定期更换密码，要根据不同的使用途径决定不同等级的密码体系。几套密码体系可能存在于若干个（也可能只有一个）电子芯片上，按照一定的周期轮换体系（或者按照上级命令随时更换），以确保密码不被破译。
       虽然现在破译即时密码很难，但是以往用过的密码被破译的案例是发生过的。即使破译别国以往的密码，那也是非常非常有价值的一件事情，因为你会根据以往的电侦记录知道很多以前发生的事情。如果即时密码被破译，那将是一场灾难。如果不能及时知道密码已经被破译，那就是一场噩梦。如果及时知道密码被别人掌握，那也会花费大把的银子以及泄露少量通信内容。别人掌握你的密码，一种途径是别人技术比你牛，运气比你好，人家把你秒杀了。另外一种途径是出了奸细，准备把看家宝拿去换钱的，你还别不信，真有这种人，被灭了。
