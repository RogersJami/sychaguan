刘志军3年办了30年的事，留下的债务却仅占中央+地方政府负债总额的1/20。高铁彻底改变了中国的城际交通模式，对中国社会和经济的重大贡献是语言和统计数字无法形容的。

高铁大部分线路财务已有利润（覆盖利息和折旧无压力）。而地方政府的许多项目却不仅覆盖利息和折旧无指望，连运用费用都要财政倒贴。两者相比，判若云泥。将来一旦地方债务东窗事发，华尔街肯定要求中国政府出售高铁而不是地方那些面子工程。

重庆的债务情况我不清楚。2011年审计署搞了一次全国性的地方融资平台审计，全国十万多亿，重庆两千多亿，约占全国的1/50，直接欠债情况并不突出，但当时重庆率先搞了一批BT项目，尤其是保障房和基础设施。还有一些其它花招，虽然金额不大，但是很招眼的，这和薄敢为人先的风格很紊和。

但现在，薄被拿下已有一年多了。这段时间恰逢地方政府大干快上、疯狂失控、债务指数攀升的阶段，而重庆前后继任的两届领导，德江只是过渡一下，挣财书记在很认真地核查薄时代的政府债务（含BT项目），自然不会立马就开始大搞特搞。

这样重庆已由昔日的“激进”变为今日的保守。今日重庆财政毫无疑问应属于最健全的省份之一。典型的黄奇帆风格的估计，就是40万亩储备土地，现在才干掉20万亩，剩余20万亩，每亩赚200万差价，还有4000亿的潜力。有几个兄弟省份有这种“潜力”呢？

历史没有假设；如果我们偏偏假设薄督在2012年疯狂地这4000亿全部干掉，这个金额也仅占中国2012完成的37万亿固定资产投资的1%强，再让黄市长玩上1倍杠杆，也才占2%，与兄弟省份比，又是小巫见大巫了。兄弟省份有多疯狂，可以想象。

其实，为老百姓做些实事，替TG收买些人心，顺便建几个巡警平台，再唱唱红歌什么的，真的花不了几个钱。中国的老百姓太善良，太容易打发了。但是即便如此，重庆已经让很多人无法忍受了。

我并不为薄可惜。但是尊驾所言，和事实相差太远。

美国西北大学中国地方债务问题专家史宗瀚(Victor Shih)表示，重庆地方政府、国有企业、国 ...
这个史宗瀚还真屎。把所有的国企债务都计入政府债务中，这显然不顾中国的基本现实。在统计地方政府债务的时候，应该把那些以公司形式执行公共财政职能（比如教育、医疗、基建、环保、水处理、公园绿地、保障房、新增土地储备）的债务并入地方政府。

那些竞争性行业的国企，尤其政策允许破产的国企，不应计入地方债务。比如某国有水运企业买了一条新船，难道造这个船的贷款要由地方政府还？再比如，某国有企业贷款进了一批原材料，这个流动资金贷款也是地方政府债务？

再者国有开发商债务难道与国企债务不重复计算么？

第三，按史专家的口径，中国全国的债务肯定超过重庆的100%。全国光银行贷款余额就有67万亿，还有30-60万亿的债券、票据、信托贷款、委托贷款、地下钱庄，这加在一起等于GDP 200%了。何来全国22%？这是赤裸裸蔑视我五大国有银行加上若干股份银行的贷款能力啊。

最后，现在几家主要投行计算中国影子银行的方法全是有重大缺陷的，甚至就是错的（比如高盛，比如瑞银），一面重复计算（比如把银行理财和信托贷款重复计算），另外一方面又公然漏了BT等融资方式。既然方法都不对，结果更不可能准。

这个史专家连投行都赶不上，根本就是概念乱成一团浆糊，是一个低档混混。
