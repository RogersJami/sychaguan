读完第一卷的一点感想：
有一些看不懂的地方，就是那些统计表格，看着头大。还有就是台湾的惯用语言有的让理解导向产生模糊。但是对看懂全文还影响不大。

文中，流动空间概念其实指的就是虚拟网络中由于连接而在虚拟网络中生成的一个个虚拟场景。这也就是书中流动空间概念被定义为什么什么的实践的理论基础。

无时间之时间的概念指的是人类在虚拟网络中真实的时间被碎片化，因为一直以来人类的各种非生理活动都是有规律的，所以人类头脑中就被刻上了时间规律。未来随着人类的劳动越来越趋于网络上，这样劳动的连贯性就降低了，和从前脑子里的时间概念和规律就不一样了。于是除了生理时间，好像其他时候的时间就没感觉了。
话说回来，这状况看似人类劳动时间见少了，其实人类的头脑一直在断断续续的劳动着，思考着。当然是为工作。
