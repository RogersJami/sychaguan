sandmaning 发表于 2015-4-29 12:02 static/image/common/back.gif
已入部分中航军B，目前是否可以卖出部分飞机股票，转入基金？？
不一定同步啊，它的基金仓里没有中航飞机这个股票，是中航系其他股票（个人感到很奇怪，为何不加飞机）。而且它的仓貌似还没怎么建好，价格走势怪怪的，没体现杠杆，你可以比较其他军工b的基金。

如果是新手的话，不建议买。作为新手以前我买过某只基金，几天涨停又几天跌停，逃都逃不出来。有很多人就被套了几个月。

什么叫杠杆，打个比方，比如动力跌3个点，它有可能跌停；反之，相关股票涨几个点，它就可能涨停，所以说它风险很大，收益也很大。

具体你可以问问楼主，《爆仓记》里面有介绍的。
