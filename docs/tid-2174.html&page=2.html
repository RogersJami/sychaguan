                                                                中东动乱国家经济恢复任重道远

        去年年初，一些中东国家相继爆发严重的政局动荡、社会动乱乃至战乱，至今已有一年多时间。虽然这些国家动乱后的结局各有不同，但动乱对这些国家的经济都造成了严重破坏，复苏之途任重道远。
　　相关数据显示，去年埃及经济增速仅为1.8％，比2010年5.1％的增速锐减了3.3个百分点。不过，埃及的情况在几个动乱国家中还算是好的。也门因动乱和战乱去年经济估计下降了10.5％，与前年增长7.7％形成鲜明对比。突尼斯动乱发生最早也结束较快，去年经济下降0.8％，而前年则为增长3.1％。利比亚的动乱和战乱对石油生产造成严重破坏，去年经济下降了61％。叙利亚则因战乱持续不断，导致国际机构无法对其经济做出评估和预测。
　　从总体情况看，中东石油进口国遭受打击最严重的经济领域是旅游业和外来投资。当前，它们普遍面临的经济困难是失业率居高不下、外汇储备急剧减少，通货膨胀不断升高。而它们普遍采用扩大政府支出、尤其是增加各类补贴以安抚民心的做法，导致财政赤字大幅增加，急需外来资金的支持。
　　根据国际货币基金组织的估计，包括埃及、突尼斯等中东地区石油进口国今年大约需要470多亿美元的外来资金支持以满足经济复苏需求，明年仍需450多亿美元的外来资金支持。其中，埃及本财年和下两个财年的外来资金援助需求分别高达104亿、128亿和108亿美元。突尼斯今明两年的外来资金援助需求均超过50亿美元。如何获取这些巨额资金援助，对它们来说是一大难题。
　　从近期看，中东动乱国家经济前景均不乐观。根据国际货币基金组织的预测，埃及和突尼斯今年的预测增速分别为1.5％和2.2％，明年的增速分别为3.3％和3.5％。也门经济今年仍将下降0.9％，明年可望增长2.9％。利比亚今年经济增长预期高达76％，但去年严重战乱对总体经济造成的破坏不容低估，预计明年增速仅为2.1％。
　　经济分析人士普遍认为，中东发生动乱国家的经济稳步复苏的先决条件是政局平稳、社会稳定。只有政局和社会长期稳定，才能使外国旅游者和投资者恢复对这些国家的信心，使至关重要的外汇收入和外来投资重新回归，进而刺激经济复苏。

                                                          巴基斯坦称希望早日加入上合组织

    巴基斯坦总统阿西夫·阿里·扎尔达里将于下周访问中国，并出席在北京举行的上海合作组织峰会。巴基斯坦驻华大使马苏德·汗在1日召开的新闻发布会上表示，两国领导人届时将进行双边会晤，希望进一步加强巴中经贸、能源领域的合作。此外，在上合组织北京峰会框架下，巴基斯坦希望早日成为上合组织成员国。
　　马苏德·汗介绍说，巴基斯坦和中国是全天候的战略伙伴，两国高层互访频繁。扎尔达里自2008年就任总统后，多次访问中国，此次参加上合组织北京峰会将是他第9次访华。总理优素福？拉扎？吉拉尼2008年上台后也先后5次访华。这些足以说明巴中友好关系和最可信赖的朋友情谊在不断深化发展。
　　与此同时，两国也正致力于密切经济合作，发展公共外交和文化交流。鉴于两国之间巨大的发展潜力，马苏德？汗希望通过此次双方领导人会晤，在原有的合作成果上能更深一步拓宽合作领域。如能源将是两国未来合作的优先选项，基础设施和农业等方面也将展开密切合作。而巴基斯坦与中国新疆等地的边贸合作也将进入一个新的发展阶段。
　　除此之外，两国文化交流也在逐渐升温。“中文在当地很受欢迎，学中文已经成为一种时尚。”马苏德·汗坦言，巴基斯坦欢迎中方来设立更多的孔子学院，以满足巴国内不断升温的中文需求。
　　出席上合组织北京峰会期间，扎尔达里还将与上合组织其它成员国及观察员国领导人进行会谈。马苏德·汗表示，自巴基斯坦2005年成为上合组织观察员国以来，积极参加该组织各种会议和活动，并共同致力打击“三股势力”(恐怖主义、分裂主义和极端主义)，为维护地区安全和发展做出了积极贡献。目前，巴基斯坦已经递交申请书，希望早日成为上合组织的正式成员。

                                                外媒:默克尔希望西班牙接受欧洲金融稳定工具救助

     据德国《明镜》周刊网络版2日报道，德国总理默克尔预计，西班牙已无力单独化解本国银行业危机，因此她希望西班牙接受欧洲金融稳定工具（ＥＦＳＦ）的救助。
　　报道说，上周默克尔与财政部长沃尔夫冈·朔伊布勒商定了这一共同方针。德国政府希望西班牙接受救助，以便抑制因希腊可能退出欧元区所带来的传染风险。
　　上周三，朔伊布勒已向西班牙经济与竞争力大臣路易斯·德金多斯施压。他提出，为稳固西班牙银行业，西班牙必须向欧洲金融稳定工具借款，否则西班牙只能用高利率从金融市场借钱。对朔伊布勒的要求，德金多斯表示拒绝。他说，西班牙能够筹集到资金。
　　报道说，德国政府专家预计，西班牙银行业至少需要注资500亿欧元（约合620亿美元）到900亿欧元（约合1116亿美元）。因房地产信贷无法收回，西班牙银行业目前出现巨大的资金“窟窿”。专家担心，如果没有欧元区其他国家资金支持，西班牙政府将难以应对当前困难。
