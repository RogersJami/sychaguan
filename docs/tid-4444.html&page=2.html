过长江之后，带出来的200多个家乡子弟只剩下8个，外公转到地方，建国后不久裸退，当着上司的面撕了TG证， ...

嗯。。。您家祖上的经历，正是我个人理解的广义“大家”。

最近几年一直在注意，发现“出色”、“成功”的人，经常有个现象就是“隔代遗传”。盖因前80年，大历史造成的动荡，往往打断家庭的“经济”、“地位”意义的继承。所以父母一辈，可能是普通工人、农民，但是意识品质，往往仍然可以传递下来。结果就是，改开之后大环境好转机会增多，长期来看正是这些意识品质，可以让后代重新出头，跨越前一代的经济上的“寒门”阶级。
反过来，经济、道德、意识等方面的“老百姓”，往往在前几辈，同样也是普通人，缺少可圈可点之处。或者说家族在意识形态上，没有什么特别过人的传承。

从个人注意到的例子来看，不管是乡村还是城市，家庭（家族）近代祖上出色过的（旧时的一村之长、大小地主，往往是当地最出色的代表了），统计上来看，后代（现在30、40岁）如不特别出色，过的至少也不会多苦。这个现象正是三观、意识传承的结果。至于经济上的继承、直接人脉的多寡，在我看来远不如三观、意识的影响之深远。忙总的家族传承就不说了呵呵。

再推广一点来说，最近两三年，我和老婆一直在自我分析我们身上各种优缺点，发现都能从父母的三观、意识、性格、教育方法上找到直接原因。而父母们又是从祖辈的人生经历中，形成他们的特点。祖上的区别，传递到我们这一代身上，就形成了诸如学习是否自觉努力、是否自立自强、是否擅长利用人事资源、是否利己利他等等的区别。由于仅仅对我和老婆两个家庭的了解是足够深入的，所以现在还没有什么像样的取样，不能很有把握形成广义结论，呵呵。

但是，就我们俩的思考，感觉从统计意义来说，长期来看没有什么不是从祖、父辈身上继承发扬的。而短期（10年以内）的投机、机遇、努力等等，往往只能是在祖上决定的“轴线”上下浮动。所以“阶级决定论”、门当户对、传说中的“投行招聘问父母”，我们是越来越觉得有道理了。
