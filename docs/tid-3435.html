回答这一问题并非易事，需要具备翔实可靠的资料、宏观经济把握能力、对市场的理解，以及坦率和表达能力。以上四点，缺一不可。请让我对此作一次尝试，希望能以此回答大家心里的疑惑。我们先从几个常见的、浅层的问题入手。

juyern 发表于 2012-10-18 15:41:05

juyern 发表于 2012-10-18 15:41:21

一般地理解，只有市场双方的意见不同，才能出现交易；价格上涨，伴随成交量放大，不妨解释为，双方分歧较大，多方力量在这一过程中占优势；各自的力量在成交量和价格的变动中释放，从而获得短暂而不稳定的平衡。仅从这点看，无法获得后续走势的有效提示。

juyern 发表于 2012-10-18 15:41:44

不过据以往经验，接下来马上出现较大幅下跌的可能性不大。只有在出现“虚假价格”之后，才容易出现价格下跌，而且幅度较大。常见的“虚假价格”，表现为连续无量上涨，成交量很少，持续时间较长。目前，我国房价，还是铜仁房价，均未出现明显的这种迹象。

juyern 发表于 2012-10-18 15:43:35

国家总理温家宝曾多次在公开场合表态，落实和完善房地产市场调控政策，坚决遏制部分城市房价过快上涨势头。这句话是很值得琢磨的，政府官员技能都很多，涉及面广，不仅仅是表演。

juyern 发表于 2012-10-18 15:43:50

任何一个在市场里摸爬滚打过几年的人都知道，政府是市场经济最大的参与者，领会政府的意图是投机的入门必修课。“遏制房价过快上涨”，言下之意，并非要让房价快速下跌，而是打压快速上涨这一部分，不允许能量短期内迅速释放，维持楼价缓慢、稳定、有序上涨，对大多数人都有利。很多人误解了政府的本意。

juyern 发表于 2012-10-18 15:44:07

我既不懂经济，也不懂金融。个人以为，资产证券化，并且吹大成泡沫，然后破裂，这就是经济发展映射在资本市场的真实过程。泡沫，就是经济成长付出的代价。房地产作为资产的一种，证券化和泡沫化都是大势所趋。问题在于，这个泡沫何时破裂，是否已处于临界点？这是一个棘手的难题。

juyern 发表于 2012-10-18 15:44:39

记得一个初中数学题，概意是每小时往池里注水1立方米，池子有点漏，每小时可能流走0.3立方米水，然后注水一小时要休息二十分钟，因为有人来接水煮饭，问多长时间水池能装满水。这个非常形象，可以和资本市场做类比思考。

juyern 发表于 2012-10-18 15:44:56

大树怎么高，也不可能长齐天。泡沫虽然五彩缤纷，但是终究有破裂的那一天，这是规律。对付泡沫的办法是，用一个新的更大的泡沫。这就像一个水桶漏水得厉害，只能找一个比它更大的桶来装住。但是，在没有找到一个更大的桶之前，最明智的做法就是减缓水流失的速度。国家现在干的，大概就是类似的事。

juyern 发表于 2012-10-18 15:45:14

本帖最后由 juyern 于 2012-10-18 17:18 编辑

即便泡沫破裂，也只是一种假象，因为真实的生产能力还在。爆裂的只是资产价格，对生产能力影响不是很大，甚至还是一种清除病毒并注入活力的特效针剂。资产价格只是现实经济映射的影像，其重要性被过高地估计了。

juyern 发表于 2012-10-18 15:56:30

本帖最后由 juyern 于 2012-10-18 17:19 编辑

单纯的市场交易，比较容易理解，如果里面掺杂几伙实力较强的投机者和操纵者，就不是那么容易看懂了，价格和成交量的波动都可能成倍地放大。再加上政府呢，这就复杂得不是普通人能完全看明白的了。一国之内，再强大的商人，在政府面前都脆弱得不堪一击。只要你有影响力，最后都难免成为国家手里的工具，银行如此，股市如此，房地产亦如此。

juyern 发表于 2012-10-18 16:23:50

如果把房地产看作一只股票，政府就是高度控盘的庄。如果股价急剧拉升，跟风者稀少，最后就只能从高位往下一路散发了，这是急功近利，在没找到下一个目标之前绝不能这么干。

juyern 发表于 2012-10-18 16:24:07

一种的方式是，宽幅震荡，散发一部分筹码，吸引新的对手盘。这里有两个问题，1.对手需要一定的实力，而且有吸纳时间；2.剧烈波动过程中系统的稳定性不易把控，难度较高。因此，这一方案被否决。
二是，小幅上涨，缓慢上行。这样给人造成一种趋势的错觉，引起新的跟风者进入；而且比较稳定，操作简便。问题在于，空间过于狭小，把自己逐步往最高处逼，最后还得靠从上往下抛压散发，杀伤力巨大，最后骂名远扬。

juyern 发表于 2012-10-18 16:24:20

本帖最后由 juyern 于 2012-10-18 17:24 编辑

解决的方案是，窄幅波动，缓慢上行。这样兼取二者之长，且具有可行性。基于此，形成了对今后房价的看法，短期内上涨下跌难以预测，但是波动幅度有限，中期而言，还有上涨空间；当然，最后是泡沫都难免爆裂的结局；再开始下一轮的循环。
