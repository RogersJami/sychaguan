路透社称：美联储推更多货币刺激举措可能性加大


          据路透中文网6月13日刊文称，在6月19日至20日召开政策会议之际，美联储再次发现自己处在艰难的十字路口。欧洲银行业危机愈演愈烈，美国就业市场萎靡不振，美联储似乎越来越有可能推出更多货币刺激举措——尽管政治上面临阻力，内部官员对此保留意见，且担心采取举措是否会奏效。
      文章说，考虑到美国经济前景依然疲弱，但并未陷入衰退，美联储可能选择相对比较容易实现目标的延长“卖短买长”操作，此举旨在通过卖出短期债券，买入长期债券，推低长期借贷成本。美联储可能采取的另一个相对成本较低的工具是，将可能加息的官方指导时间进一步推迟。亚特兰大联储银行行长洛克哈特认为，进一步提供货币支持举措的门槛仍然很高。他表示，“在某些方面，市场承担了美联储压低长期利率的工作。”
     但美联储内外越来越多的分析师表示，美联储不能坐视美国失业率仍高达8.2%，而且几乎没有下降的迹象而无动于衷。他们主张延长扭转操作，若不然就通过量化宽松(QE)进一步扩大美联储的资产负债表。前圣路易斯联邦储备银行经济学家迈克尔·迪克尔说，“美联储传递出来的讯号是，他们不会任由经济停滞不前。”

赞比亚央行：寻求与中国开展全面金融合作


        据新华社报道，赞比亚央行第一副行长布瓦利亚·南杜13日在接受新华社记者专访时表示，赞比亚愿意与中国全面开展金融服务，希望中国银行赞比亚分行为当地企业提供更多金融支持，并在赞比亚经济发展中发挥更大作用。
        南杜说，赞比亚央行一直以来都在寻找能为当地企业提供全方位金融服务的合作伙伴。中国银行在这方面有着突出的优势和势力，双方的合作将把这些优势转化为对当地经济发展的强大推动力。
        他指出，赞比亚采矿等行业的融资主要来自海外市场，而这形成了高昂的利息成本，不利于行业可持续发展。
        南杜将于16日率赞比亚央行代表团赴中国访问，并于中国相关金融部门以及中国银行总行就如何推动赞比亚与中国在金融方面的合作进行为期一周的考察研讨。
        同时，双方还将就提高驻赞比亚外资银行资本金的情况，以及用人民币取代美元作为赞中之间贸易结算货币的议题进行商讨。
        南杜解释说，赞比亚央行要求驻赞外资商业银行提高注册资本金至1亿美元是出于增强赞比亚金融业抵御财务风险、增加贷款流动性的考虑。
        他说，对于用人民币取代美元作为两国贸易结算货币的提议则是出于减少不必要的金融成本，更好促进中赞贸易交往的需要。
        南杜表示，赞比亚希望成为继尼日利亚之后第二个使用人民币作为国际贸易结算货币的非洲国家。
       他还希望中国银行赞比亚分行在服务驻赞中国公司和华人华侨的同时，发展更多的当地客户。他说赞比亚非常欢迎中国在赞设立像中国银行这样的金融机构，通过全面金融合作，双方经济发展的前景将会更加广阔。
