技术标准不同就不能出口？拿到订单按照外方的标准做不就OK？

目前欧盟的所谓绿色标准我们国内没有一家企业能够达到。其实轮胎就是例子，我们自己企业没法进入欧盟，只有米其林之类企业可以。这是一种非关税贸易壁垒。

欧盟在建筑领域的能耗指标和环保指标非常变态，如果中国企业按照那种标准生产，按照国内价格销售，企业都要亏损，出口也就没有任何价格优势或成本优势，加上物流成本，更贵。

所以国内出口欧盟的家装建材全部是代工，贴牌生产，主要是认证麻烦。而结构件无法代工，因为涉及安全问题，没人敢贴牌。

中国家装模式是装修公司包工不包料，装修材料（哪怕是螺丝）都要业主自己采购。有的地方顶多包点水泥 ...

现在装修模式是全包、半包和清包三种模式，全包就是人工材料全由装修公司搞定，当然业主出的米米要多，但也很省心；半包就是装修公司出人工、水管、电线、油漆、板材，因为这些基本都是品牌标准化，剩下的材料由业主自己去选择和购买；清包就是找装修游击队，或装修公司里的接私活。
就我观察，半包较流行。
