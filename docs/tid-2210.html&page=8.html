真正北京吃的东西，烤鸭和涮羊肉还行，其他都是不堪入口的，尤其是什么小肠陈，卤煮火烧之类。
内脏本来就是原来下等人吃的东西，为了压膻，放了很多佐料的。重庆的火锅也是一样的吧，材料用的是没人吃的部位吧。根本不是美食。就是给重体力劳动人民补充体力补充蛋白质的。

我还记得我第一次在天津吃羊汤的时候，差点呕出来。天津的羊杂羊汤和大连的不一样，大连的是，汤多羊杂少而且羊杂切得碎，汤是乳白色的，撒上胡椒面，膻气能压住。而天津那才叫羊杂，羊肺羊肝羊新煮好切片，然后加上少量羊肚羊肠，摆好了，谁要吃自己点一碗（可以选自己喜欢的比如肝多点儿，肺多点儿），然后摊主撒上葱花，香菜，给你浇上汤，汤也不是大连那样奶白色的，而是淡淡的颜色，汤随便添，自己加辣椒面,。一碗里面大半部分都是羊杂，另外还可以单点羊眼羊脑子。
第一口太膻了，受不了啊，放多少辣椒面都压不住，不过现在好多了，耐受度提高了。
但是天津的同学告诉我，天津这是真羊汤，你们大连那个是假的羊汤，就是羊骨头羊杂熬的汤，有没有羊都不一定。
