毁灭你与你何干 发表于 2012-4-26 04:38:58

楼主好，因为尊重楼主的性格，忙虽然可能帮不上，权且提供一个另类的看问题的方法。楼主貌似缺少一种把握问题的整体感。首先应该从自身的角度出发，设定一个整体的主要目标。那么达成这个主要目标的每一步都会成为优先需要达成的次优目标，而伴随这些次优目标而产生的和与主要目标木有关联的，就是次要目标。而这个局有多大，主要目标便可以人为的随时设置。好像说的挺乱的（挠头）。打个比方说，家里要添小孩，那么稳定的收入就是主要目标，而稳定的收入需要稳定的工作环境，和自身积极的工作态度。那么稳定的工作环境需要和上，中，下各方各面搞好关系，积极的工作态度需要自己在能力上和业务上更进一步。而在工作中产生的问题，比如和上司的矛盾等等，不应妨碍到自己主要目标的达成。当然了，假如对人民和国家负责也是你的主要目标，遇到很多不好的事情不能忍，那么就这个时候就需要在这2个主要目标里做权衡了。毕竟，鱼和熊掌不可兼得啊。打了这么多字，其实一二句话也可以说清楚，莫怪啊。
