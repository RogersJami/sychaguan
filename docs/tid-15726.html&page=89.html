朱民：美股已经进入高风险阶段
　　清华大学国家金融研究院院长、国际货币基金组织(IMF)前副总裁朱民3日在杭州市人民政府主办的“2017杭州湾论坛”上表示，目前美国股市高估，美国股市进入高风险的阶段，这是不争的事实。



　　朱民认为，未来美股面临巨大不确定性。美国股市下一步走势变化如何，主要取决于公司利润、美元走势、特朗普税改和债务上限等财政政策的变化。



　　在他看来，在流动性方面，目前美国短期利率水平已经出现拐点，利率正在上升。



　　“美联储加息几次不重要，但利率总体水平在上升，全球流动性紧缩的拐点已经到来。这是根本格局，不会改变。”他说。
