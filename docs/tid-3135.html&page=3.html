1953：社会主义改造、抗美援朝胜利、三反五反、高饶事件。（其实这个三反五反和高饶事件的权力斗争，隐含着了一个主题文革）
1965：文革开始了
1977：四人帮下台、小平重回台上、改开的开始（到1989年为止，这10年的改革是激烈的）
1989：学潮动乱 国际禁运 国际上原苏联要不行了，从此改革速度下降
2001：中美媾和，中国加入世贸，中国输给美国实物支持美国（美国借911，以师出有名，个人很怀疑美国控制者是知道有人袭击的，但故意放任911发生，以师出有名。不然去媾和中国让中国发展干嘛，这成本太大了。你看，中东后期，打伊朗就师出没名，中俄欧都不支持，搞到现在都没开打，并且拖坏了美国）打中东压欧元（国际前提，此前99年还是00年欧元诞生），而美国给中国美纸并允许中国用美纸全世界买资源，多余的美纸回去支援美国。这个过程一直持续到现在，正因为加入世贸，才中国这10年的和谐大发展，国力大幅增加。
2013：？？？ 以后12年怎么走？ 我估计要有战争，涉及中国参与的战争。伴随战争，人民币真正走出去，从此世界形成三大吸血货币美纸、欧元、人民币。世界连一个美纸（2亿人口）吸血都承受不了，何况三个？所以只能通过战争来决出一个能让世界接受的方案来。

静等高人指点
从玄学角度来看，共和国成立于1949年10月1日，其干支为己丑、癸酉、甲子。
此6字干支为财官印俱强，也就表明此国大不简单，所财官印三奇俱全。如此时辰按下午3点前计算则是辛未时（毛泽东宣布成立的具体时间不知道）（此时三奇俱透在干），那么月令酉官透在时辰，也就是说中期共和国是事实的最强国，然后后期共和国是要做老大盟主的。如果按三点后则是壬申时。那么，官杀星不透干，月官时杀藏地支，那么共和国只能做事实的最强国，而且从中期就开始了，但没有能力做盟主，后期共和国可能有内祸发生。
6字年支丑，月支酉。而巳+酉+丑汇聚一起，就要发生巨大的“物理化学”反应，有如爆炸、核变等等，用玄学术语说就是“巳酉丑三合西方金”。三字聚到一起，三字就变成力量非常强大的五行金。而金就是共和国的官杀，官杀一旦太强大太过了，就会变成压力、障碍、战争（对个人八字而言，就是表达压力大、被伤害、被杀、被制裁、生病、死忙、各种祸。。）。所以明年国内必然压力重重，压力一大，过临界点，必然会发生变革。

个人认为可以按60年一柱来看（只是个人看法，但比较符合历史上大一统朝廷的诞生终结周期，4*60=240年，大多数大一统朝廷都是在行180-240年这个周期时完蛋，这个时期恰好是按60年一柱的时柱运），以此看,则：己丑一柱为1949-2009，癸酉一柱为2009-2069，甲子一柱为2069-2129.时柱则是2129-2189.

如果是辛未时，则行到未时（最后30年），共和国可能要变国体了，也许是地球上没有国家这个东西了，真正的地球村。因为未为木的墓库，而共和国是甲木日诞生，当到未时，共和国入墓终结。

大家拟目以待吧。
