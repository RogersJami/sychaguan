请教忙总，以兄台所见，这个沉沦的起点在哪里，何以会导致这次沉沦呢？

我认为沉沦的起点就是满清殖民政权的建立，现在出于政治考虑刻意掩盖这段历史的真相，然而任何人也不能欺骗所有人于永远。
香港在英国人统治期间，英国人只会为自己利益考虑，不会考虑他们治下的中国人的切身和长远利益，同样，满清政权只会为自己利益考虑，不会考虑他们治下的中国人的切身和长远利益。
英国人好歹为香港中国人带来了西方的商业文明，可以作为中华文明的有益补充，满人作为一个野蛮愚昧的族群，带给中国的只有破坏与蒙昧，毫无进步意义。
满清二百四十年，就是香港沦为英国殖民地九十年的放大版，不同的是，汉人推翻了满清政府，顺带把东北（相当于英国人的英伦三岛，当然东北本来也是汉人的领土，但在清代汉人是不准进入东北的，那是满人为自己准备的退路）、蒙藏新疆（相当于英帝国的印度南非澳大利亚，这些地方清以前的汉人王朝也实行过程度不等的统治，但在清代汉人是不能进入的，就象香港人不能自由进出印度南非和澳大利亚一样。）一古脑收了回来，而且让前殖民者无处可逃，不得不冒充汉姓混迹于中国人之中。
明末时徐光启就翻译了《几何原本》，而且其赶超的决心与意志丝毫不亚于现在的中国赶超美国，崇祯皇帝批准徐光启翻译西方科技著作，是立下了“超胜”西方的宗旨的，可以想像，如果明朝不亡，或者继明而起的是一个汉人政权，中国根本不会等到十九世纪末期才出来一个睁着眼睛看西方的魏源。
我们今天仍然在为满清压制中国二百四十年的罪孽买单。
