原 创： 烧制红烧肉 ( 下 )



   

    另外的一种弄法呢，是“烧制红烧肉”，基本上也是传统的搞法。
    从街上买回来的肉，模样儿要跟那“炒制红烧肉”一个样，也是连精带肥的都得要有。
    要晓得，这“红烧肉”之所以味道好，吃起来香喷喷，肉嘎嘎，肥而不腻，靠的就是这连精带肥的猪肉两下里相结合。若尽是选的一砣肥肉呢，又显得太腻哒；倘若没得肥肉打伴噻，尽是些精嘎嘎的瘦肉嘛，那肉吃起来神像似嚼槟榔的，嚼得满嘴巴直刹，腮帮子直作痛！牙齿缝里头塞满哒，没得点好味道！所以说，选好料是蛮要紧的一件事情。最好是割一砣猪身上的‘腰板子’肉。‘腰板子’俗称‘五花肉’，这肉肥瘦相当，即有精肉，也有肥肉。
    大块的肉买哈回来，先用刀解成小块子，一者是容易煮熟透，再者是锅里头好盛装下。然后，把切好的肉块，一齐给塞进锅里头，开起大火，一顿子将它煮熟透。测试的方法，简便易行，拿一根筷子，从里到外，不费力气一戳就穿便为好。
    跟着，把煮好的肉块捞哈起来，让其冷却过性。
    然后，或切砣，或切块，总之一条，块头莫弄得太细小，原因与前头讲的同为一理。
    接下来，点着火，将锅儿搁在灶火上面，直烧得冒出青烟，便放入几勺子油。等到油烧滚，往油中放入两勺子白糖或是冰糖。待到糖溶化，慢慢变成深黄颜色时（此时叫作：斑汁），再把切好的肉块倒入锅中，就着斑汁翻来覆去一顿炒，直炒得所有肉块均沾染上深黄色的斑汁方可。此刻，可随即放入盐，味精，花椒，桂皮，草果，香叶，生姜，八角等等酌料进去。跟着用口炖锅或是砂锅，将肉块与酌料，统统盛放进去，再放水进去，以淹至平面为准。搁在灶火上，开起小火慢慢地炖煮。
    熬上个把小时后，可揭开锅盖尝尝味道，若是火色还未到，请继续熬煮莫要歇火。直熬得合乎你的味口时，方可罢手。至此，烧制红烧肉便成功。
