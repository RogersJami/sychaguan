zhemingjun 发表于 2015-2-26 11:54:53

从小在农村，现在虽然在为了孩子读书，在县城买了房，还是没有和农村脱离干系。
就像上面有朋友说的，农村不是你说的那么好，也不是你想的那么坏。
不过这个和是不是农村没有关系，自古以来，锦上添花的多，雪中送炭的少。
小时候家里穷，一年种的粮食不够吃（家里因为超生，被扣了一个人的田地），去找别人借一斤米，别人也会担心你还得起不。这几年，我家这个贫困户也慢慢好起来了，甚至比很多人还要好，大家都来奉承。我觉得这个是正常的。我会记住别人对我的好，能回报的时候就回报一下。别人对我的不好，很多时候也是一笑置之，谁没有那样做过的时候？
平时，回到家里，能对别人好的，也就是看着别家需要帮忙的时候，去做点力所能及的事情。毕竟，现在也不是有哪家吃不上饭。至于那些给我说要我带他的孩子出来工作的，我带过一个，从此畏惧。实在那些至亲的，都是打哈哈。这事儿，真不好做。小孩儿要是努力的，不用带，他自己也会努力出息。不努力的，那是烂泥不上墙。
任何社会，任何环境，都有懂得感恩的人，白眼狼也不少。能否换位思考是关键。见人说人话，见鬼说鬼话在社会上是基本功。我还记得曾有评论《鹿鼎记》的文章，韦小宝对自己老婆施恩，都不是无故的，一定会找一些由头，绝对不平白无故的。既不让受恩的人难堪，又会记住自己的恩情，所以他能到那么高的位置。从做人来说，韦小宝实在是楷模。

以前人口几乎没有流动性，住一个村里，在同一块地里刨食，农闲之后没什么事可做，走动得多，所以知根知底，是个熟人社会。进城进厂之后，各忙各的，一年见不了多少次面，见了面也不知道说些什么，相互之间只是认识，成了半熟人社会，有些变化更强化这种现象。

以前河道清淤是全村一起干，分段包干到户，家家上阵，现在请专业工程队了；以前盖房子是砖混结构，你家盖房子，我来帮工，我家盖房子，你来帮工，现在都包给工程队了。

物质丰富了，业余生活也更多样，各玩各的，不再会聚到哪家一起看电视。

随着跨地区婚姻的增加，加上出生率低，亲戚关系更简单了。
