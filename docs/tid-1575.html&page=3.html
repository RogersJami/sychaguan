不用担心，在茶馆大家就是闲聊讨论一下

就我接触过的项目来说有国外在中国推广的牙膏新品。前期推广采用的是直复营销，先做销售，亲近终端消费者，通过销售，能够了解当地市场对产品的感受。同时利用促销活动尽可能多的收集用户信息之后再进行相关的市场推广手段调整。

推广牙膏的手段据目前有新意的不多，一般是“用牙科医生的建议”这样的广告（营造出一种credibility）， 有的牙膏公司“舍得孩子套得住狼”， 在新产品上市前（或同时），会采用在超市和合适零售店摆放试用装（小包装那种）的方式，由人任意拿。

当然如果是儿童牙膏，就趋向于明细化，也更有创意些

不过，从商业的战略角度来看，新航的空姐制服也是将“品牌符号化”策略方式。

1972年，马星航空公司（MSA）分成两家新的航空公司——新加坡航空公司和马来西亚航空公司（Malaysia Airlines）。

对于从马星航空公司分离出来的新加坡航空来说，之前马星航空女空乘所穿的南洋特色本土服装是其所继承下来的最引人注目的资产。蔡建华介绍，新航制服是法国着名女装设计师皮耶·巴尔曼（Pierre Balmain）于1968年为航空公司量身定做的。它沿袭了马来沙笼柯芭雅服饰的传统线条，并在领口处做了一些细微改动，而且在褶边、袖口、领口和前襟处使用了镶边材料。一些主要设计元素——宝蓝背景蜡染彩花设计、制服质地和重量都是巴尔曼设计过程中的灵动之笔。不久，这项杂糅了法国人时尚天资的独特创意很快成为旅客眼中马星航空公司飞行服务独有的商业标志。


　　1974年，独立运作的新航邀请巴尔曼女士对制服进行了设计修改，之前比较板正的领口在修改后更加圆润，符合亚洲人柔和的面部线条，展现出独特的东南亚风情。自那时起，新航空姐被人们称做穿着沙笼柯芭雅服饰的“新加坡女孩”（“Singapore Girl”）。她们穿着同一制服将近40周年——可以断定，新航是世界上唯一一家可以将此录入《吉尼斯世界纪录大全》的航空公司。
