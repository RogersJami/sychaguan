太极能杀人？？头回听说，能侃侃不

说一点太极拳的事情，网上喝茶打屁，大家不必较真。

我哥上小学时候，有幸跟一太极老师学过2年，后来就没有再练，一直到最近几年，对这个事情又有了兴趣，捡起来练了几年，具体的我也没有问过他。（我哥今年47了）。

08年5月我回老家，中午吃饭时，村里几个邻居在家里聊天，其中一个年轻人，20多岁，当时号称一霸，比较喜欢打架，快1.9米的个头，真的是膀大腰圆，据说三里五村是没人敢惹的角色。聊天时候说到太极拳，这小子非要和我哥试试，我哥推辞再三，我突然也很感兴趣，想知道太极拳到底能不能实战，加上这小子是村里沾点亲戚的，我们都是他叔辈，也就说让我哥和他试试。（为此后悔了很久）
  当时我看到的就是：这小子一伸手打向我哥，被我哥两个手掌顺着他胳膊一捋，直接侧向撞到屋墙上，倒地，没了动静，吓的众人一番掐人中，一边打120，折腾半天，总算缓过气来，服了！还有就是后来在家里躺了大概半个多月。 我前后给人家赔了5万快钱{:4_269:}，
   后来，我问我哥，我哥说当时搭住他的胳膊的时候，就感觉他很“轻”，所以也就用了2成力，又怕他碰到头，扭了一下才把他“发”出去，没想到会是这样，后怕的不行，我哥与人为善，从未和人动过手。
   这件事情后，结论如下：
   1、真正的武术，就是用来打的！这个不必怀疑。
   2、现在的武术，为啥中看不中用？因为都经过无害化处理了，公园里练得，很多效果和广播体操差不多。
   3、功利化的时代，很多师傅根本不会传真功夫的。
   4、真正有功夫的人，轻易也不敢试，真试出问题来，轻则赔钱，重则判刑，谁敢轻易动手。
   5、我哥总结的：学真功夫，两个条件：一是师傅肯教，学到真东西了；二是自己要有悟性。（内家拳的很多口诀，必须在练得过程中，悟！）

   附加说明一点：本人不会任何武功，本人对武功的真谛也是一窍不通。
   大家看个热闹，不必深究。
