不困难。富士康来报名的排队很长很长，而且很多河南人是以富士康来豫为荣的。

至于有些人不打算就业，其实也就是观望而已。反正我对河南农村人的感觉是，人虽然穷，但是对尊重要求比较高，比较胆小怕事，所以在广东受了欺负，都想跑回来在家门口就业。另外他们的年龄，也不可能有啥自强不息的精神了。就是想安心就个业，给小孩儿攒点钱罢了。

不排除官员故意造势宣传，肯定有，但说到底，富士康那种只要努力加班就能拿到3000左右，而且有保险，无污染的企业，还是受河南欢迎的。河南政府这边的意图是：解决就业，然后带动周边就业，适当增加租金和税收；让多一些河南人受点工业化的影响，否则农村人永远小农，下辈子也小农；吸引一些现代化的配套，因为河南电子基础差，比四川都差得多，当年没布局；增加经济外向性，增加出口额，让河南感受一下国际化是怎么回事儿，能前进一点是一点。

发展情况还是差异太大了，河南的过剩人口太多，虽然和四川相比同是人口大省，但河南水资源差一些，所以河南农民生活不如四川农民。另外河南农民已经被隔三岔五的大祸驯服了，比较逆来顺受，虽然体格高大一些，但同一个工地上，四川人往往敢上高架子玩命儿，河南农民一般不敢，所以工资拿的也不太多。

不过近年来河南官场造假GDP的本事已经落后了，因为中部地区看东部有优势，西部有政策，自己啥都没有，于是喜欢吆喝，开始PK，一PK，扯淡造假都瞎指挥乱规划都来了，湖北湖南已经超越河南。
