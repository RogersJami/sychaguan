请教一个追债的问题

本帖最后由 流江河 于 2012-3-6 05:47 编辑

向大家请教一个追债的问题：我一个朋友，在广东小本经营，他老乡（朋友老家同县人）跟我朋友借钱，未许。他老乡又提议跟朋友一起搭伙做柴油生意，朋友同意了。朋友买了柴油（20万元人民币），卖给老乡联系的工地（在他老家地级市），工地把钱打给老乡的另一朋友甲。老乡久久不跟朋友结帐，躲着朋友都一年了。朋友跟县公安局报案，公安局不予立案。据说那位老乡的朋友有人在公安局（具体职务忘了，不是局长之类）。

以前看过忙总帖子里面有为公司追债的事情，好像有什么追债公司？是怎么操作的，一般佣金是怎样的？但朋友这个是私人欠款，且金额比较小。希望大家为朋友指点迷津，直接回复或站内信箱都可以。谢谢！

注：朋友是川东人氏，朋友和他的老乡都是小县城，离重庆比较近，我在网上看到重庆有一些追债公司，不知是否靠谱。

wxmang 发表于 2012-3-6 11:48:57

这个事情恐怕够呛，只能自己以后注意了，花钱买教训。如果是黑社会性质收债公司，至少见面分一半，如果闹出伤亡事故，还得掏医药费。

如果是走司法程序，100万以下都基本不立案，因为忙不过来，太多了（当然有熟人另说）。要想立案，除非出现刑事‘案件。可是20万值得闹成刑事案件吗？

自己慢慢追债吧。

在磨刀的胖猫 发表于 2012-3-6 15:00:48

这样的民事纠纷，公安局报案一般是没用的，有没有人在公安局都差不多
法院目前在走大调解程序，就是立案前先调解，就是缓解审判鸭梨，案子多，人手少，资源紧张，像你说的这种小案子，实在太多了，很多说合说合就算了，不予立案
如果确实有证据，有对方财产线索，又认识比较靠谱的律师，可以试试司法程序
有证据指对借款事实留有双方确认的证据，借据什么的，最好还有其它旁证佐证
对方财产线索指对方的资产，房子汽车啥的，你能确认的，否则很可能赢了官司收不了钱
靠谱的律师就是那种法院关系不错的律师，至少立案这一关过得了，法院门口摆摊的还是尽量不去，找那种大的事务所
