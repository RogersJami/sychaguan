个人观后感：

看看李家代表的香港四大富豪们对香港市民的抽血，再联系台湾十大家族对台湾社会的整体控制，就能感受到这两个完全实行资本主义的地区对中下层民众的抽血多么厉害，而所有的罪恶却被巧妙隐藏，民众的愤怒被完全转移到香港政府、台湾当局及不相关的大陆中央政府身上。

台湾地区完全是自由选举，可是不管是国民党还是民进党，上台没几年都要挨骂，可是那些隐藏在背后和控制社会资源的富豪家族们，却没有一个被骂的，而且一旦有人公开吗，就要官司缠身，不搞得你倾家荡产不罢休，谁敢骂？

香港马上就要普选了，难道普选就能解决香港的发展问题？

看看台湾的吊样就知道了，台湾搞民主选举多年了，经济发展停滞的问题照样无解。
反而因为政治恶斗，消耗了很多本可以利用的资源，比如搭上大陆发展的顺风车，韩国就利用这个机会每年对大陆巨额顺差，本来这个顺风车完全可以给台湾地区，都是一家人嘛，可台湾人自己宁可搞分离，也不要这个大蛋糕！

香港普选之后，只会麻烦越来越多，政治恶斗越来越多，而真正惠及香港民生的政策却会因为政治因素而无法施行。
可以预见，很多香港人的下半生只会交给马路了。

和历史对比，这两个地区，和明朝末年江南地区何其相似？
明朝末年，江南地区的所有社会资源（以土地为代表）完全被大地主士绅阶级控制，中下层民众完全被抽血，而愤怒却被士绅阶层巧妙的转移到明末朝廷身上，总之朝廷无能，所以要征收各种粮饷，而士绅阶层根本不纳税，又可以借机吞并土地，囤积各种社会资源。

香港和台湾地区，已经没救了！
他们已经被资本完全控制了，从头到脚的控制了，而这个背后的资本就是香港的四大家族和台湾的十大家族富豪们。
而前台的政客们已经成为富豪们的玩物和替罪羊。

就如这位香港立法委员所讲的：嗟尔香港，气数尽矣！
这句话，也完全可以送给台湾人民。

希望大陆不要走香港和台湾的邪路，一定要坚持公有制主体不动摇，社会主义道路不动摇，人民民主专政不动摇。
