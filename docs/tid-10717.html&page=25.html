我来讲个真实案例

8月份的时候深圳科技园南区开了一个盘，房子很不错，但开发商很一般，因为是大户型，单套要一千多万，卖的很惨淡。

我当时去看，觉得以实际得房来算单价已经低过周边二手房，交通方便又有深圳的顶级学位加持，一套一千万左右的话真的可以考虑，遂认真砍了一下价格，还算满意（未下定）。

回去想想换房成本实在是太高（买卖+贷款），就推荐给了一个朋友，她看完觉得也不错，但还是担心换房的问题，就没怎么积极跟进。9月底房贷付清二套按首套的政策一出台，她又去售楼处，发现价格立马涨了80万，又犹豫了两天，房子没有了～

其实如果当时她看完房后先交个50-100万的定金，并跟开发商谈好后续付款计划的话，现在坐在20平米大露台上看海的，就是她了～
