这个葡萄所说的“垂直垄断模式”是指什么模式丫，百度不到，恳请大牛们解释下。

垂直垄断是我们小圈子自己的词汇，意思就是在某个产业链环节的垄断，比如微软比如英特尔在自己领域是绝对的霸主但是美国政府不允许他们的垄断针对it产业链条上下游通吃。而横向垄断就是日韩垄断财阀的模式。每个财团基本可以形成自给自足的经济自循环，不仅可以脱离国家与政府的控制，甚至可以凌驾于国家与政府之上。不过三星和现代，他们的股份百分之六十掌握在以美国财团为主的外资股东手里，说他们之余韩国的横向垄断并不是太合适。日本的三菱，住友以及三井这样的财团更贴切。

白沙 发表于 2013-10-16 10:02:22

一堆人绕来绕去把问题绕复杂了。
所谓的土地流转问题，其核心内容只有两点，一是大量农民的分流问题，二是粮食安全问题。其中粮食安全问题又可细分为农产品生产数量的问题和粮食的价格问题，也就是说我们既要保证生产足够多的粮食确保主动权在我，还要保证这些粮食能够以老百姓可以承受的价格在市场上交易。
所有与实现上面两个目标相背离的制度与利益都必须搬开，至于今后土地的经营模式，只要符合上面两条，都可以。

白沙 发表于 2013-10-16 10:10:19

       从国家层面来说，就是要实现两个目标，一是普遍增加农业及从农业分流出来的人员的生活水平（提高收入），二是要提高粮食的单产和总产水平。农业的集约化经营是必然出路，但也别指望今后靠农业发大财，试图垄断粮食来谋取额外收益的行径想都不要想。理想化的目标是让农业从业人员总体收入达到目前西方发达国家的水准。

白沙 发表于 2013-10-16 10:32:35

       为什么土地改革要延后？这里面还有一个重要的因素，就是国家的配套条件还没有准备好，早推出要出乱子的。因为土地它目前还是大部分进城农民工最后退守的依据，一旦把这个缓冲地带剥离，而配套措施没到位，政府的压力就非常大。这些需要准备好的配套措施有以下几个主要方面：
       一、解决就业问题，这个最关键，一个没有相对固定收入的人即使他在城市有房子也是没办法在城市立足的。而目前我们的就业形势很严峻，一方面传统产业产能大量过剩，本身存在巨大的减员需求；另一方面市场垄断广泛存在，新生企业创业艰难，吸纳就业能力有限。目前的对策是破垄断活市场增就业。
       二、解决农民进城的制度壁垒。这主要包括社保、医疗、子女教育及保障房申请等附着在城市户籍上的福利，打破户籍壁垒就是实现这些福利的无差别全覆盖。目前在这个问题上的主要障碍是两个，一是地方财政的支付压力，二是地区差异导致的转移障碍。消除这两个障碍的主要思路其实大家在前期的讨论中已经有一些影子了，比如移交部分地方事务给中央政府，这样的好处是利于全国统筹，且不会给地方坐大。一旦社保医保等实现全国统筹以后，这些福利的地区转移将不再是问题。目前需要做的具体工作是消除地方社保账户（尤其是中西部不发达地区）的空帐问题，二是要调整一些制度设计，对地区差异导致的不平衡做适当调整，不能鼓励懒人，打击发展经济的积极性。
