原来在一个乡下的医院看过这种情况，一个剖腹产，一般要上十几个人，严重的时候药房的人都要上。很多出现这种现象的医院都是中小型医院，他们备的人手不够，一般二级甲等以上的医院，手术室都是有专门的几个人的。原来我们县医院手术室就一般有上手术台的护士，也有管手术室的护士，负责手术室的卫生打扫和消毒，各项器械摆放，也要去给医生递器械，等等。或者是一个科有几个人是专门上手术台的，这个是做的相对比较正规的。

国内的很大的问题就是护士的编制被大大压缩，国内一个医生对应有时候不足一个护士。国外一般是一个医生几个护士的配比的。国内一般就是实习医生跟着上手术台，实习生帮着护士干活。实习生当然应该跟着做事情。不过有时候护士编制的确被压缩的太狠了。

国内 的护士劳动强度确实太大太大了。医生也是。说是8小时工作制，其实从上班开始就一直在忙，下了班还要再收拾收拾。医生更麻烦的是，随时都可能被自己管辖的病人给抓过去。就像我说那个医疗事故，其实管床的医生头天上了一天的手术台，晚上还忙了半夜，第二天就该去休息，但是偏偏就第二天出了事情。归根结底，问题还在医院身上，但是病人最终会觉得你医生护士没有做好。

现在西方对时间的认识，在弦论出来后，有新的突破，越来越与我们祖宗的概念相关了，时间是认为定义的向量 ...

我不是学物理的，也不懂弦论什么的，不过我自己偶尔也思考过时间的本质。我自己认为时间本质上是运动的一种表现形式，因为地球从诞生那天就不停地在运动，我们所定义的时间不过是以地球为参照所定义的运动间隔，这确实是人为定义出来的一种向量。人的意识会灭亡，时间也会消失，而运动将永远持续下去。所谓的黑洞不是时间的消失点，而是运动的静止点，也许是质量的聚集造成运动的丧失吧。所以时间的本质是运动，空间的本质也是运动的到达界。以上都是我没事时的胡思乱想，聊博一笑。
