反腐永远在路上，没有一个完美的制度在所有的时间把把所有的人管好。有人、有利益、有差别的地方就有腐败，永远会有人找出制度的漏洞，为自己谋取利益，不腐败是不可能的。
我觉得，反腐败的理想状态是把腐败行为控制在一定程度之内，使它不阻碍社会的发展，使社会的各个阶层（阶级）的矛盾不要那么激化，
制度也要与时俱进，不断的改进，完善，改进完善的动力就是社会各个受到腐败损害的阶层的抗议（反抗），统治阶层的不断的调整。
