中国社会之分析


社会么，一般都分为精英和屁民，过去，精英叫做士，皇帝管着士，通过他们管着屁民。当时的士，有些是地主，有些是官僚，有些也是知识分子。后来引入资本主义了，士就分化了，地主早就过时了，变成屁民中的一个了，产生了新的资本精英，这​下子，现在的精英的成分就凑齐了：资本精英，知识精英，官僚精英。这三种精英下面，就是一群屁民。这些成分下，也就组成了，现代中国以来的三种主要派别，以代表屁民的社会主义左派，官僚和一些知识精英组成​的体制右派，资本和一些知识精英组成的右派。之所以把他们叫做左派，右派，是因为号称代表屁民的社会主义，称自己为左派，相应的，剩下两派也就称右派了。以下把它们简称为​：左派，体制派，右派。总得来说，左派：政治上屁民当家+经济平等；体制派：政治上官僚精英当家+经济自由；右派：政治上资本精英当家+经济自由。

1. 过去
话还得说到从前民国的时候，当时国民政府，应该算是体制派和右派联合执政（以右派为主，表现在国民党的官僚不强，就是组织性不够）。由于当时的种种社会原因（简单地说，就是物极必反），总体社会左派的力量不断壮大，特点是知识精英，甚至民族资本家中，和左派合作的很多，这个时候，左派（就是当时的共产党）和一部分知识精英，资本精英右派合作，搞掉了执政的国民党。

左派当政（建国1949）之后，迅速官僚化和堕落（一般来说，哪个派别当权，哪个派别就堕落得越快，正所谓一代不如一代），分化成，左派和体制派。这个时候，左派和体制派​还在政权巩固之后（朝鲜战争胜利，土改完成，一五计划完成）联手搞掉了曾经和自己合作的那些右派，也就是三反五反，反右（1956）。

随着左派和体制派联合执政， 体制派的力量越来越强（主要就是官僚的强大，越来越多的原来的左派变成体制派， 剩下的左派也越来越废物，谁当权，谁就退化这个定律不可避免），然后出现了大跃进（1959），这件事，到现在左派和体制派都还在扯皮，把责任推到对方那边（其实两边都有​责任），利用这件事，体制派（刘少奇）把责任推到左派头上，反而要求更大的权力（1962七千人大会）， 这引起了左派（主要是毛泽东）的反弹，也就形成了文革的局面（1966-1968），就毛泽东的目的而言，他希望建成一个左派为主，体制派辅助的格局，然而事与愿违，左派的衰弱不可避免，全靠毛一个人撑着（1972之后），而体制派的势力却越​来越大，到后来，他都撑不住了，他死后，体制派全面反扑（1976），这个是建国后，左派最全面的失败，这之后，左派再也没有恢复过元气来，一直是体制派当权。

体制派为了对付左，联合了原来被绞杀的右派（几乎无差别平反），开始了以体制派为主的联合执政（1978改革开放）。随着这几年联合执政，右派的力量增长过快，到了右派希​望自己当权的地步（自由化，学潮），这引起了体制派的反弹，他们联合了左派一起对右派进行了打击（六四事件，1989）。左派得到了短暂的复兴，但随即再次遭到了体制派的​打击（1992）。最后反而形成了体制派为主，右派较弱，左派被压制的局面，也就是‘警惕右，但主要是防左’。

2. 现在
之后的十多年，是体制派和右派的蜜月期，因为经过上次的两次折腾，体制派独大已经形成，而左派一直被压制，右派也吸取了教训，慢慢发展，等待时机。可是随着这十多年的执政​，问题越来越大，一是体制派自己的退化很严重（退化定律再次出现），看看现在官僚的表现就知道了。这个是最大的问题（官僚逆向淘汰，一句题外话，为了解决这个问题，TG可​能会进行小规模的整风都说不定）。二是右派的慢慢壮大给体制派带来很大的困扰，虽然体制派试图和右派联合（三个代表），但右派更希望自己当权（普世价值）。也就是说，虽然​两派经济上的要求一样，但是政治上的分歧很大。三是左派的基础得到了一些巩固，随着这两派的联合执政，只关心经济自由，社会上希望实行经济平等的力量越来越强。

也就是说体制派受到了各方面的挑战，有来自内部的，来自盟友的，和来自外部。这样体制派和其中一派联合，是自然之势。2008年以后，体制派一直有所谓的左右之争，这个其​实只是决定，体制派和哪一派联合。从现在各方面看来，体制派和左派联合对付右派的可能性正在越来越大。原因有以下几个: 1. 前面说过了，右派不愿意进一步合作，而左派一直受到压制，体制派一伸橄榄枝，左派就会立刻接受，所以一直有人劝左派学会实用主义，学会妥协。 2. 现在右派的实力比左派大得多，也就更具备威胁性，自然成为受打击的一方面。 3. 左派要求的经济平等，比较能够缓和现在的贫富差距，要是不管屁民的感受，万一屁民反了，左派右派哪一派得利不知道，但对于体制派来说，总是不愿意看到的（稳定压倒一切）。​所以有了所谓的国进民退。 4. 右派整天要求普世价值，跟美国总有说不清道不明的关系，现在左派，体制派，右派都在中国这艘船上，要是右派引入美国势力，说不定直接导致翻船，这个是体制派不愿意看到的，​2008年以后，中美之间的摩擦不断加剧，就从对外关系而言，也需要联合左派，打击右派。 5. 官僚和资本是天生的矛盾，官僚不愿意让资本取得控制权，资本也不愿意，一山不容二虎，要不就是资本当权（美国），要不就是官僚当权（体制派），平等地联合执政，不可能。这​个从体制派内部抵制三个代表，普世的右派反对三个代表都能看得出来。6. 体制派是继承于左派的，这是在改开的时候他自己宣称的，而如果体制派联合右派，彻底压制左派，那他的执政权力来源也就成了无根之水了，所以，在长期压制左派的十几年里，体​制派一直还维护了左派开国的贡献，防止打击左派会损伤到自己。

3. 将来
如果体制派和左派联合成为现实，那么就会联合压制右派，这之后呢？可以肯定得说，左派会像用过的厕纸一样，立刻被抛弃。从中期看来，在中国达到中等发达国家的平均水平前（个人估计在2040年左​右），虽然其中可能有一些变化，但是体制派（也就是搞中国特色社会主义）为大的局面不会改变，不过中间可能要有几次微小的调整，就像这次一样，但不会改变大局，到了204​0年左右，体制派的式微应该是比较可能的，经过这么长期的执政，在退化定律作用下，应该已经很废物了。而随着社会的发展，这几十年里面，应该已经产生了很多的资本精英，知​识精英和足够多的屁民了，也就是说，左派和右派的弹药都很充足了，而体制派却已经衰弱，到时候会怎么样呢，情况肯定会更复杂，but that would be another story…
