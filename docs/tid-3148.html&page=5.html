【原创】抗战国军阵亡旅及以上军官246人名单 [ 苍野 ] 于:2009-11-02 04:59:20 主题帖
抗战中，国民党军阵亡了多少将领，“据说”何应钦提供过206 人的数字。网上很多人以此为据，愤愤不平地指责只阵亡了左权和彭雪枫两名将领的中共武装如何消极抗日。但有趣的是，这些指责的帖子，为了增强观感，很多都附上国军阵亡将领的名单；但这些名单如果详加清点，就会发现和国军一贯的统计一样，混乱矛盾，一笔糊涂账。

例如，这个百度知道里提供了两个版本，其中一个七十多，一个一百四十人。

七十多人的版本，是蓄意剔除了所有的旅、团级指挥官的结果；干嘛这么做，我看和共军阵亡军官大有关系，奥妙究竟何在，我后文另讲。

这个版本的问题是：

1.和何老总的两百人数字差距极大，无法弥补——别说不算阵亡的团长追晋少将的和旅级少将，此版本一下子把1937～1938年阵亡的蔡炳炎、陈德馨、陈钟书、高致嵩、官惠民、黄梅兴、姜玉贞、庞汉桢、秦霖、吴继光、郑廷珍、赵锡章、朱炎晖13位旅长追晋中将尽数扫地出门，怎么可能补得上？

2.算师不算旅，是对抗战中前期阵亡的国军旅级指挥官的极大不尊重！抗战初期，国军的主力师编制为师-旅-团，旅长阶级直接就是少将。到抗战中后期，主力部队基本取消了旅一级编制，多数旅改编为师；长官阶级不变。就所辖人数而说，抗战中后期国军的师多数编制不充实且存在严重吃空，实际统辖人数未必就及得上前期的旅；此版本计入了后期的师级将领而不算前期，尤其是头两年的旅级将领，实属赤裸裸的污蔑啊:)

一百四十人的版本算入了旅长团长，但它的问题是，重名的太多：戴民权、范廷兰、付忠贵、高道先、韩炳宸、黄启东、江春炎、金崇印、林英灿、徐积章、张庆澍、周鼎铭、朱实夫等13人都计入了两次。把这个因素剔除，就只剩 127人——和何老总的数字还是相差很大。

不过涅，这张名为《国民党抗日牺牲表（绝无他意）》的名单里，怎么出现了左权、彭雪枫、田守尧、包森、杨靖宇的名字？哦，此名单中有11个八路，10个新四军，24个东北抗联干部^_^ 介个，虽然第二次国共合作八爷四哥抗联名义上都归蒋委员长节制，但这些共产党员此时可并非国民党员！要说这个版本要显示国府兼容并包的气势把，共军阵亡的30个军分区司令、副司令它又只算了9个，唉。

在抗战中，国民党军队进行了英勇的抗战，在正面战场起了主要作用，成千上万的将士殉国，其中包括许多将军。共产党所领导的八路军、新四军在敌后同样进行了英勇的抗战，也有很多将士殉国，但是在殉国的高级将领中，只有左权为人们所知。所以后来有人指责共产党抗战作用不大时，就会说共产党只死了一个左权。共产党在抗战中所起的作用虽然不象历史书上所宣称的那么大，但也是很重要的，八路军、新四军所牺牲的高级将领，也不只一个左权。

这个问题，涉及到一个编制问题。红军一开始整编为国民革命军第八路军，后来按投入战斗的序列称为第十八集团军，是一个三师六旅编制，只有大约四万五千人，他们的将领并没有国民政府的军衔。由于当时共产党与国民政府只是一个合作关系，只是名义上接受国民政府的番号，实际上八路军在敌后的扩充完全是自主的。在敌后的战斗中，八路军的发展非常迅速，部队迅速扩充，成立了许多旅一级的部队，建立了大片根据地，划分了许多军区以及下属军分区。以山东根据地为例，在一九四零年，一一五师对在山东各地的主力部队实行统一编制，共编为七个教导旅，共七万人。这只是一个地区，而且还是在抗战中期。所以，八路军已经不能看成仅仅是一个集团军的兵力。新四军的情况与八路军类似，他们最初被编为四个支队，支队相当于旅，支队下辖团。经过敌后抗战的发展，在“皖南事变”后，新四军重建军部，主力部队被统一编制为七个师，第六师编有两个旅，第七师数量较少，仅有一个旅和一个独立团，其余各师均有三个旅编制，全军共九万人，这也是在抗战中期。国民党军队的编制中，旅长及旅长以上的军官都是将军，个别主力部队的团长也授将军衔。以此类推，八路军的旅长以上和相当于旅长以上的军事干部也应该是授予将军的军衔。而八路军与新四军这一级别的高级干部，在抗战中牺牲的不在少数，而作为八路军总部担负重要指挥作战责任的副参谋长左权，是这些殉国将领中级别最高的，和国民党方面阵亡的最高级将领张自忠差不多。

共产党阵亡将领虽然在数量上较国民党阵亡将领要少一些，但不是那种指责所说的，仅仅只有一个左权。由于种种原因，现在国内只注重对解放战争的纪念，忽视了抗战中流血牺牲的将士，所以共产党抗战中阵亡的将领并不很为人所知，他们中的许多人有着卓越的才能，如果在共和国成立以后，也会成为灿烂的将星，但他们却在民族解放战争中陨落了，他们也是值得人们纪念的。
