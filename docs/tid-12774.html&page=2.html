我知道的几个 ...

政府对于房市和股市的态度，忙总有过相当精辟的论述，
股市，本质是把老百姓前半生的积蓄掏空
房市，本质是把老百姓后半生的积蓄掏空
在掏的过程中一般遵循以下三条原则
1.资本平衡。这个是说，一般情况下政府的支出是大于收入的，于是，这个缺口就要由房市和股市来补
2.长短期平衡，这个是说，政府对于这比钱要尽可能有统筹，不能今年大鱼大肉，明年窝头咸菜，对房市股市资本的提取要有限制
3.尽可能让市场规律其作用，这个是说吃相不能太难看，要注意公信力，和政府的形象
以上三条原则排序要分先后。
