中国自古有“精卫填海” 荡气回肠的故事。 2000多年前的汉代，唐、宋、元、明、清等史料中也可以看到有关围填海的记载，当时主要用于防灾、农垦或晒盐。

建国初期50年代围海晒盐，，形成了沿海地区四大盐场，比方长芦盐场、海南莺歌海盐场。
60年代中期至70年代的围海造田，形成了大量的农业土地，为我国的粮食生产和经济建设作出了重要贡献。

南海填海对于中国来说技术不是问题， 咳咳，现在钱也不是问题 :P

烟台填海工程
深圳填海工程
