一周新闻回顾（7.14—7.20）

本帖最后由 dada2020 于 2014-7-21 11:18 编辑

7.14——7.20一、习近平出席金砖国家峰会，会见金砖、拉美多国领导人：1.        建立金砖国家开发银行（总部上海）和应急储备安排：中方将正式实施100亿美元中拉基础设施专项贷款， 并在这一基础上将专项贷款额度增至200亿美元。中方还将向拉美和加勒比国家提供100亿美元的优惠性质贷款，全面启动中拉合作基金并承诺出资50亿美元。中方将正式实施5000万美元的中拉农业合作专项资金。————重点在基建方面，中国特色的双赢之道，比如：中国、巴西、秘鲁宣布建立连接大西洋和太平洋的两洋铁路；阿根廷基什内尔和塞佩尼克水电站、贝尔格拉诺货运铁路改造等。农业合作似乎也是新亮点？习总还特别参观了阿根廷的一个大农庄）
2.        建立中国—拉共体论坛，尽早在北京举行论坛首届部长级会议————另外，中非合作论坛部长级会议2015年在南非举行。。习总称，欢迎印方作为创始成员加入亚洲基础设施投资银行。。————亚非拉美大团结万岁，哈哈。话说中国在美帝后院搞这么大动作，估计奥巴马有点坐不住吧？习总访拉期间还和奥巴马通电话，奥巴马表示：第六轮美中战略与经济对话和第五轮美中人文交流高层磋商富有成效、期待今年11月再次访问北京并出席亚太经合组织领导人非正式会议。。————话说，这个美中战略对话，还是从全球金融危机那阵开始的吧？看这个成立的时间点，应该还是很重要的，不过好像也比较低调，不了解。。
3.        中国与智利：建设好中智自由贸易区、智方愿意同中国加强在联合国、南极事务中的合作————这次出访还提到中哥（哥斯达黎加）自贸区。另外，与北欧国家开展北极事务合作倒是在新闻里听过好几次，南极事务合作好像很少提到啊？估计楼主最近太懒了，没听到。。
4.        中阿元首共同关注阿根廷与美国“秃鹫基金”的债务纠纷————秃鹫基金，“趁你病，要你命”，偏好投资行情看跌的公司、违约或即将违约的高收益债券及主权债券，低价买入，要求高价赔付。阿根廷不幸中招，深陷主权债务危机，美国最高法院判定，阿根廷必须向秃鹫基金先行支付15亿美元的本息，否则可依申请没收阿根廷政府的海外资产。背负150亿美元巨额债务的阿根廷，其外汇储备金仅剩280亿美元，总统措辞强硬拒绝支付。————习总这时来访，算是阿根廷“救命稻草”吧？中国援助主权债务违约的国家，阿根廷并不是第一个，比如经常提到的希腊；但是对始作俑者美国“秃鹫基金”这么指名道姓，很少见啊！！！为什么捏？{:soso_e107:}
二、经济运行方面：1.        上半年主要经济指标公布：广义货币M2同比增14.7%，全国税收收入增长8.5%，GDP同比增长7.4%；李总表示，保持今年增速在7.5%，cpi3.5%。
2.        继续放开民间投资：山东规范民间融资为实体经济服务————组建专门的金融监管部门，批准成立民间融资公司，年利率16%———这个值得关注呀。还好这只是地方公司，如果支付宝这么高利率，或者一半，估计银行就要死掉一半了。。上半年我国社会融资规模10.57万亿元，同比多4146亿元，为历史最高水平，只是不知钱投去哪儿了。。。放开民间投资的门槛，比如铁路基础设施建设、医疗卫生等领域，吸引利用民间资本，这也是李相的主要思路之一。此外，国资委启动改革试点，探索以管资本为主加强国有资产监管、探索多种所有制经济交叉持股、建立职业经理人制度，民间资本又多了个“国企”的去处。
3.        中国超过日本，成为全球第一大工业机器人市场。2013年中国市场工业机器人销售总量近37000台，提到国内最大的机器人企业沈阳新松。
4.        张高丽福建调研“一带一路”建设———没看出什么特别的地方

三、农业方面：1.        黑龙江发展大农业：以合作社推动土地规模经营、土地经营权抵押贷款、绿色食品认证等
2.        夏粮丰收、单产提高
四、中央将展开2014年第二轮巡视：对广西、上海、青海、西藏、浙江、河北、陕西、黑龙江、四川、江苏等10个省区市开展常规巡视，同时对国家体育总局、中国科学院、一汽集团开展专项巡视。————长三角、江浙沪！魔都是直辖市里最后一个被巡视的吧？居然还有体育总局！分明就是国足惹的祸！
五、国际方面：1.       珍爱生命，远离马航！！愿逝者安息、生者平安！——俄罗斯和乌克兰互相指控对方击落马航客机，此前几天，俄罗斯自己家里也不太平，还是受伤者呢：俄称乌炮弹落入俄境内造成俄国公民一死两伤，乌否认；之后莫斯科地铁出轨，19死161伤；再后来，美欧宣布扩大对俄制裁；再再后来，就是马航客机事件了。。
2.       我国自主研制的ARJ21新支线飞机以及C919大型客机亮相英国范堡罗航展————国产大飞机？？怪不得前段时间李相访问非洲，还力推铁路、航空合作
