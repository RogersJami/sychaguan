无限忙的徒弟 发表于 2012-9-28 14:34:32

1. 拍摄的团队对工程知识比较欠缺，想表达的东西太多，结果啥都没太搞清楚。
2. 陈晓卿是搞小资生活的美食节目的，可能不适合搞这种科学工程的节目，或者是应该有更加适合的人。
3. 画面有些乱，净搞什么镜头伸缩，镜头拉来拉去，居然还老出现几个镜头同时出现，结果啥都没看清，还眼花缭乱的。猜想是不是 摄影师拿了新设备要show一下。
4. 解说词很雷人，为了突出“超级”出现了很多客观工程节目不适合出现的词语：“最。。。最。。。最。。。，后果将功亏一篑，面临巨大的困难，困难重重，极高的要求，极大的挑战，极其。。。。”听着觉得写解说词的人估计没花什么时间弄懂这几个工程上，解说词听上去不像中文稿，像是模仿外文翻译稿。
5. 一些重点的突破需要更加深入浅出的解释。
总的来说，这个节目的idea很好，也对如何向人们有效有趣客观的传达科学技术工程知识提出了挑战，不知道下一部会不会有改进。

说实话，觉得拍的很不咋地，把很好的题材拍的乱七八糟。
北京地铁那一集，太多东西揉在了一起，很多不同段落衔接的匪夷所思。正说着9号线施工难度的，突然来个老1号线建设者的煽情流泪，不知道导演咋想的，好像连台词都没有串起来。看完这集，就记得起堵堵堵了。
港珠澳大桥那集还不错，不过不搭边的煽情多了点，那种最。。。必须。。。再加几个工程师摆拍的段落太多了。拍了很多普通市民的生活，是想表现人文关怀吗；选了很多工程项目里的小角色，连他们怎么开车怎么走路怎么在办公室里唠嗑都拍了，可就是没表现出来怎样用智慧解决工程难题。
所谓超级工程，亮点应该是国家实力、工程智慧、劳动结晶。难道就不能以一名科学家的角度、用向公众做科普的态度，说清楚项目意义、难点、解决办法、创新性这几个最基本的问题吗。如果想煽情，请多拍拍从普通工人到技术总管面临从没遇到过的难题时，怎样迎难而上吧。
