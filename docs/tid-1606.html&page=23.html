60卡是什么意思呢？


简单来讲，人消耗热量由两部分组成，新陈代谢，运动。其中新陈代谢远大于一般性运动。一个都市上班族如果没有大量运动，新陈代谢所消耗热量不会有大的变化。一天也就是2000-3000大卡（数据不一定正确）

运动的目的是提高身体新陈代谢的水平，让那个相对固定的主要消耗增大。减肥当然不单纯是靠运动消耗的那几百一千卡。

美国天然气价格创十年新低

    据报道，周三美国天然气价格十余年来首次跌破2美元/千立方英尺，而前不久市场还认为供应短缺。
      
       　　美国的天然气供应迅猛增长，分析师们担心到今秋地下储备设施将储满，造成价格进一步下跌。周三天然气期货价格跌至1.984美元/千立方英尺，为2002年1月28日以来的最低价，当时的价格为1.91美元。价格降至1.75美元将创1999年3月23日以来的最低水平。
      
       　　天然气价格下降是家庭和企业的福音，也将促使电价的下降，因为美国电力的1/4来自天然气。从去年10月到今年3月，家庭平均消费天然气868美元，同比上一年冬季下跌17%。同期户均汽油消费为1940美元，增长7%。
      
       　　美国天然气产量、储量巨大，钻探商、经济学家、决策者和用户都在考虑如何应对。去年美国日均生产天然气630亿立方英尺，与2006年相比增长24%，而同期消费量的增速还不到产量增幅的一半。

       1千立方英尺=28.3168立方米
       美国天然气期货价格是1.984美元/千立方英尺=12.49285元人民币/28.3168立方米=0.441182元/立方米
