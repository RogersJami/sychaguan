文宣有两种搞法：
（1）自说自话
（2）用对方习惯的表达形式传递自己的思维

第一种模式，需要强大的物理实力为后盾，比如美国目前的搞法。我的标准就是普世真理，你们照抄照搬就行了。同样一套美国说辞，从波多黎各嘴里说出来，就只能贻笑大方了。不过中国梦有点自说自话的意思，不停的说，不停的重复，慢慢有人会接受。
第二种模式，在自身实力不够强的情况下，在话语权没有那么大的情况下，是适用的。

有一个实例，又一次和一个交流非常多的同事开车去德国，大概要走六七个小时，路上吹牛打屁是必须的了，他说起“我无法理解你们中国人和日本人的纠结，两个邻居打打闹闹有什么意思呢？”。另外网上能看到的MTV（BRUNO--Dove of Peace， http://v.youku.com/v_show/id_XMTQyNTA0ODQ0.html），传递出一个信息，西方人不关心我们东亚发生的事情，西方政客精英们则喜闻乐见东北亚的争吵。对他们来说，远东发生的事情就是邻居间的不睦。这当然是政客和媒体刻意引导的结果。我国最近就安倍参拜的事情的应对就好了很多，一是大张旗鼓的把那些日本战犯和希特勒比较，另外就是中日外交机构在英国围绕安倍参拜做的“伏地魔”舆论战争。
我们应该非常正式的在国际组织比如联合国再推进一次“东京审判”，是对非正义的一次审判，是对反人类罪行的一次审判。雇佣大量西方人，用西方社会接受的方式告诉二战日本人对中国人、朝鲜人、东南亚人、美国人、英国人做了什么，让他们自己得出结论，中国和日本的斗争，韩国和日本的斗争，是正义和非正义的斗争，是为了人类未来不受法西斯奴役的斗争。把日本被审判的二战战犯和逃脱审判的二战战犯都定在耻辱柱上。
