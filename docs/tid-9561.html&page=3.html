俺从另外一个角度来谈谈宋朝军事斗争失败的原因

汉朝军力之所以强盛，使用的武器比匈奴优越是一个重要的原因，汉军用弩（具有射程远精度高的优势，汉代的弩已经有现代意义上的准心等装置了），不是有一种说法“一个汉军可以对付5个匈奴兵吗”，在肉搏战的时代，如果武器不好，恐怕是不可能滴。在大规模养马提高了军队的机动性后，汉朝的军事实力对匈奴就具有比较大的优势了。否则也不会有陈汤的“犯我强汉者，虽远必诛”这样的豪言壮语了。而到了三国，三国之间除了互相厮杀，都还派兵进剿周边地区，整个三国时代，也木有记载神马匈奴或其他周边地区的军事威胁，这个也从侧面反映了当时汉族在武器设计，生产上对周边民族的巨大优势，只是这种优势在晋代慢慢地被削弱了，特别是五胡乱华之后。到了唐代，幸亏开国的李世民及其部将具有丰富的军事斗争经验，凭借着这种经验，虽然武器相比周边地区已经木有什么优势了，还是取得了唐初军事斗争的不断胜利，但是到了唐朝中叶，形势逆转，不仅在传统的机动性上输于对手，武器也变差了（著名的大马士革钢刀就是产生于这一时期，号称冷兵器时代最好的刀）。谋略的作用必须建立在大致相当的物质基础之上，想想宋朝的步兵在平原上面对辽，金的骑兵冲击，武器还不如人家，打赢那真得是很难的，而且，人家和你打交道那么久，你的那些计策别人多多少少也猜得出一点的。
