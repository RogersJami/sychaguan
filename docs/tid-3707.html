生活碎事之女儿趣事

我的宝贝女儿已经三个月大了，呵呵。关于他的一些事挺好玩的。和大家分享一下：

1.刚出生的第三天，护士将宝贝抱进护理室抽血，规定家属不能进，所以我和LD就在外面透着玻璃看。护手要在小家伙的脚指头上扎口吸血。由于害怕，她脚乱蹬，护士加大手劲，她哭了而且哭的很厉害。抽完血，护手将他放到推车里，听她哭声慢慢小了，一会就不哭了。过了几分钟，护士把她抱出来，交给我。那小眼睛先一看，一惊，然后很委屈的哭起来。宝贝认人了。

2. 有一次我给宝贝把屎，只听她扑的放了个响屁，过了三秒钟，她竟被自己的屁声吓哭了。


3. 刚开始两个月，宝贝要是拉尿了，就哭的很厉害，让你给她换干爽的尿布；第三个月开始，宝贝拉尿了，就一脸的坏笑，你一看就知道又拉尿了。


4. 我抱宝贝下楼晒太阳，旁边一个妈妈抱了个比我家宝贝大20天的宝贝，不过她家宝贝长的很壮实。我就伸个手指到那个宝贝的手掌里。那个宝贝拉着不放，劲还挺大的。我正在逗那个宝贝玩，我家宝贝就不知为什么哭起来。我赶紧收回手要抱紧她哄一下，她就不哭了。那个宝贝的妈妈说你家孩子还吃醋呢。LD看在眼里，说：这么小的孩子还学会吃醋了，没人教啊。我嘿嘿一笑：还不是遗传的。
