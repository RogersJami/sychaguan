陈运文说，数据被喻为“石油”，目前无论是政府还是大型的互联网企业都掌握自己的“原油”，在“提炼”过程中，更好地发挥和挖掘数据的潜在价值，创造更好的社会和经济效益。但哪些数据是应该可以共享、交易，还应该有所界定。
