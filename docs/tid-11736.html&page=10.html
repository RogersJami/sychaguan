那调查一下，你能接受这个出版物吗：把忙总读《易经》中的“牵强附会”中的很多部分（涉及到政治、党史等敏感问题的全部篇章）都删去，主要保留跟管理相关的，并且不补充其他新内容（主要是忙总没精力补写）。

因为，按照忙总本人和出版社编辑达成的共识：当忙总猴年马月有空了，把牵强附会部分都换成跟企业管理相关的案例，然后再出版。这个才是做私房菜的出版，不然，按照第一段的方案，是快餐做法。

所以，问题就是：你是要吃快餐呢，还是吃私房菜？;P还是：先吃快餐，以后再吃私房菜。（意味着做两回）
