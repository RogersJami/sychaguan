四，月上小山坡，人约黄昏后。

      我把她要回家的消息告知父母，老两口都很开心，觉得他们的一件心事有希望了。在吃晚饭时对这件事有过讨论，觉得还是要去摸摸底，不能像以前没摸底差点出问题。第二天父亲去她的村里打听。没多长时间父亲就回来，啥也没打听到。是这样：她一毕业就长年在北京，同村的根本就没有人知道她的近况。看来只能是我自己来试探了。

     时间过的很快，她是坐班车大清早到家，先在她的外婆家待着，约好晚饭后和我见面。这一天的时间过的特别的慢，晚饭吃的特别的早。放下筷子就和她联系，吃好了来接她。过了半个小时她叫我去接。当时的心态很好，感觉就像两个老熟人见面一样，所以没有紧张。再加上到她外婆家还要见她的外婆，父亲，表弟，表妹，不由的挺直了胸膛。来到外婆家见到了未来的老丈人，我连忙递了根烟，并和外婆和表弟妹们打了招呼就载着她去目的地，一个在山脚的冷饮摊。见面时是快7月中旬天气很热，但在那冷饮摊就很凉爽，有山风带来潮湿的空气特别凉爽。

        去的路上我在家门口停了下，美其名曰是拿矿泉水，其实就是让父母和小姑看下，有可能是未来的儿媳是啥样。我偷偷的看了下老妈，已是笑的合不拢嘴，眼里有异样的光彩。当然我老妈的满意度的要求是很低的，之前我说过只要是蹲着撒尿的老妈都会同意。我又看了下小姑，小姑给我竖起大拇指还点点头，算是认可了吧！


      到了目的地后，相对而坐，我点了瓜子，熟花生，炒螺蛳，还有两杯绿豆汤。这时我才仔细看她，身材有点偏瘦，眼睛很大，鸭蛋脸型看上去比较富态，特别是耳垂很大。我坐下来的第一句话就是，终于见到现实中的你了，很高兴认识你。下面聊啥不重要，重要的是我们坐在一起可以生动形象的看着对方。不一会螺蛳上桌，她说好久没吃过家乡螺蛳，尝了一只，大赞。就嘬个不停了，边吃边跟我说，你要不快吃就没了。我挺喜欢这样的，大大咧咧不做作。

     我笑着看她吃，没一会就吃了一大半，突然没有"嘬嘬"声。只见她凝视前方，我自我感觉良好的以为成了“花样美男”，被一个女孩凝视是很开心的。我也笑着看她，过了会她回过神对我说：哦，你后面的景色好美啊。{:4_267:}   还好前面我没问，我有那么好看吗？{:4_235:}

      回头看了看，也被迷住了，不远处有座小山，山的后面是县城。县城灯光把天空照的通红，一轮圆圆的红月慢慢的爬上山头，月亮边的云在快速的向北方移动，偶尔还有几只鸟在飞往回巢的路上，这是一幅动画感十足的画面。这时清风徐徐，再看看对面的她加上这样的景色，想不恋爱都难。这样的场景永远定格在脑海中。以后我俩经过那个冷饮摊我们总会相视一笑。

        很早就把她送回家，毕竟坐了一天的车很累。同时约好改天再见。

五，进展是快的，测试是过分的。


   接下来的两天都是晚上见面，天南海北的聊。可能她在北京呆的时间长，虽说的是家乡话，但加上儿化音后，特别的好听。所以也很喜欢和她聊天。

   到了第四天晚上见面已经很熟了，几乎无话不聊。我俩吃完晚饭，就一起去县城的一河边公园见面，那天晚上天气很闷热，乌云时不时的遮下月亮。走到无人处，不自觉的就牵了她的手。现在还记得当时我感觉跟做贼一样，常常四处看下有没有熟人。其实是心虚，黑灯瞎火的那里会认出来。这天晚上聊了很多，各自阐述了人生观价值观，以及对未来两人世界如何相处交换了意见。她说的一句话印象特别深刻，她说过两人在一起不想过死板的生活，当时对这句话很认同。在以后的生活中，终于领会了这句话的意思。

      在回去的路上经过一盏路灯下，我放慢了脚步，跟在她后面走，上上下下仔细打量一番。

她回头看了下我。

你怎么啦？

我在想一个根本的问题。

什么根本的问题？

我喜欢的是前凸后翘。

她回过身弯着个头，看了会月亮。
回答我：没事啊，去趟韩国不就都有啦！{:4_280:}
（至今我还记得她狡黠的目光）

测试通过，回答的落落大方我很满意，重要的是这样的问题她还不生气，说明以后生气时好哄，不会小家子气。

在路上我约她明天一起去龙穿峡玩下。晚上我想了很多，觉得就是她了。
