foursea 发表于 2017-1-28 13:12:07

这是关于机器人使用的的一个评论。看后才知道工程师的重要性。

客户买这类设备主要是几方面原因：1，加工大批量精度需求比较高的产品，2，大量比较复杂的产品，通常需要几道工序和几台设备，用机器人设备一台就够了。3，安全生产需要，尤其是加工一些比较窄的工件，（手工加工容易受伤）。４，加工很重的工件，人工工作强度太大，太慢。所以，机器人并不是为了代替工人，更多是补充人工的不足。因为机器人需要有经验编程工程师和高级操纵技工，和更细致的维修和维护。通常零件和服务都很贵。

所以，要发挥机器人设备的长处，１）需要一定数量的重复加工产品，２）有聪明工程师，细心的技工和有经验的维修人员。如果产量很小，再专门编程，ＳＥＴＵＰ，就不划算了。如果技工很笨或粗心，就会人为造成很多问题。由于设备精度要求很高，调制不对或机械方面出问题，产品质量就无法保证。很多时候好的技工和工程师很不好找。尤其是在南部和西岸，工厂里都是老黑和墨西哥人，英文都是大问题，教起来太难了，也不知道他们学了多少。而且西岸人员流动很快，每次过去都是新面孔，也不知从上一个人那里学了多少就开始操纵搞精设备，很吓人的。所以我们机器人客户主要在中西部和中部小镇上，很大的原因是那里工人选择少，没地儿跳槽，干得比较久：－）
