真正要实施一项政策的话，会使用精确的语言，“套”这样的模糊概念只会在限购这样的政策中使用，要征税的话是不能用这个概念的，会代之以“平米”。国家会认定25周岁以上公民具备独立固定资产登记资格，假定个人名义下限定30平米免征，那么一个家庭夫妇俩带一个小孩免征面积90平米，考虑你今后可能生二胎，最多给你120平米的免征面积。
现在假定来一个假离婚的案例，夫妇俩离婚分割房产和小孩抚养权，一人带一个小孩分一套房产，对不起你的免征面积只有60平米，超出部分请自觉纳税！
