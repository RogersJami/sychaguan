纯贸易型的企业还好，因为占用资金比较少，只要不给国外客户放太多帐，国内工厂那里晚几天付账，在税务局有业务关系快点退税，所以，只要没有大的风险，基本上还凑合。

但是国内工厂就惨了，如果是民企工厂，从银行贷款很难，搞生产买材料要当场付款，从贸易公司那里拿钱最快也得60天吧（从接单算起），所以基本上工厂都得自己垫钱（没有本金只好到处借贷，民间借贷因此有了一定市场），另外工厂工人队伍不好带---有技术和水平的要用高工资留住，没技术没水平的不好管理，再加上税负一直没有得到减免（虽然相爷多次说要减免），还有房租水电都在涨，所以算来算去亏本的可能性很大。

象你们的公司，因为是国资委下属的，所以享有一些进出口方面的特权，比如以前的一些出口批文你们肯定能好拿，进口方面也肯定有政策支持。还有从银行融资也很方便，都是国企嘛。再加上贸易公司本质就是皮包公司没太大成本，还有国家的政策性支持，想不发都难，只要不好大的风险，基本衣食无忧吧。

譬如我公司，一个原来省属贸易公司，后来逐步改制为民企，靠着以前在银行积攒的口碑，有良好的信贷额度，不过后来一不小心搞实业，弄了个船厂，风光了两三年，没想到08年后船运市场大跌，很多船东都毁约不要船了，结果导致砸在手里近6条船，占压资金好几个亿，现在只能从银行拆东墙补西墙，甚至都借过高利贷来维持资金链，现在都后悔死了，目前严格来说早资不抵债了，但只要银行不彻底追债，那目前就是资产优良啊，一追债搞不好就死翘翘。

哎，现在来看，如果有渠道和本事的话搞个皮包公司远比搞实业要好啊，赚钱的速度和风险都比实业好，搞实业风险还是比较大啊，皮包公司负担小。

国内现在这个环境对于实业民企来说还是很差，不走歪道和偏门，生存就很不容易啊，想做大做强一靠本事，二要去抱大腿，否则就算你搞好企业了，也要担心被别人惦记上啊（例如太子奶事件）。

个人意见：

血汗工厂在过去十来年的贡献就是：换来了国家所需的外汇，解决了大批农民工的就业，增加了政府税收

为了以上目的，国家采用了以下手段：
固定美元-人民币汇率政策（8.2及以上的汇率可是维持了好多年啊）
高出口退税（前些年13%的退税都算是比较低的退税率了，现在13%就是高退税率了）
以及对血汗工厂对员工的盘剥的不闻不问（以前政府根本不管企业员工的保险和工资是否按时发放）

至于为什么过去十来年国企没有成为出口创汇的主力，也没有成为解决就业的主体，个人认为原因有以下几点：
1.附加值低，利润低的产品国企根本搞不过血汗工厂，国企员工牛得很，你想血汗压榨他根本不可能。
2.前些年国企还在升级，现代企业制度正在制定和完善中，同时没有能力做大批量有竞争能力的中端产品的出口
3.国企改制的一个内容就是精简人员，不可能成为解决就业的主体，所以要靠血汗小工厂。

为什么现在血汗工厂要倒闭：
1.金融危机和经济危机导致西方市场萎缩，如忙总所说，我们的低端产品生产能力过剩，要去产能化。
2.国内人口结构发生变化，那个什么拐点要来了.
3.国家外汇够用了，不需要血汗工厂的外汇贡献了。
4.国企比以前强大多了，都能出去搏一搏了，可以成为出口的主力了。

总之，就如忙总以前说过的那样（忙总原文恕我不转达了，就在本站滞涨贴里有），血汗工厂如同被剪刀差盘剥的农民，如同被下岗的国企工人，如同被房子，学费，孩子所奴役的“X奴”们一样，都是TG当家治国所要用到的道具或龙套，属于你的表演时刻过去后，你就是要被踢到角落的，龙套就要有龙套的觉悟。
而这样的场景在TG的发家过程中早已不新鲜，例如：革命时期给农民打土豪分田地来拉队伍，农民兄弟那是香饽饽一个----解放后土地收归国有，用剪刀差盘剥你到现在----工人兄弟成为老大哥，又一个香饽饽-----国企改革搞下岗，老大哥从头再来，等等，更别提解放后小资产阶级从公私合营到一无所有了。
别了，血汗工厂，多看看星爷的《喜剧之王》吧，你个死跑龙套的。
