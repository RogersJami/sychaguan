这个您放心，蜂巢的建造得非常的合理，共有三层，最上一层是贮藏成熟蜂蜜的部分，第二层是贮藏孵化蜂蛹用的蜂蜜，第三层就是蜂蛹所在的巢室。我们采割的只是最上一层那一部份，保留第二层和第三层，这样就不会毁灭蜂群了。只要有经验和有善心的蜂农，是会保留下第二层和第三层的。
