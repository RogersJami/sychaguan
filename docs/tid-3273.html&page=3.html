“损卦两个原则，对上位者来讲，要诚信待民，损下益上要适可而止，不能竭泽而渔，更不能榨取诈骗，巧取豪夺；对下位者来讲，要心甘情愿，诚心诚意奉献，同时要量力而行，切不可勉强而行，更要找准时机，在上位者急需奉献时采取雪中送炭。”

1。百姓的“心诚志愿”奉献的前提，是来自他们相信自己的奉献，会交到值得“信靠”的“领导”的手里，而运营的结果会反过来为自己“祈福”。现在的危机是，如何重建这样的“信任感”？ 要靠行动，而不是靠语言来奏效！
2。现在的那些“锦上添花”的作为，是扩大贫富差距，助长“不劳而获”，引发“资金外流”等等的源头。因此，“雪中送炭”的关键，是看清真正的“需求”对象，组织和依靠可靠的力量，把“炭火”送进百姓的心头。上下同心，共度“寒冬”。
