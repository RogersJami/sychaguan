据《联合早报》5月25日报道，法国总统奥朗德在欧盟峰会上提出发行欧元区共同债券的建议，遭到德国坚决反对，但奥朗德24日仍旧加紧推动，并设法说服富国同穷国一起分担债务。

　　德国解决欧债危机的对策是坚持成员国严格遵守预算规范，实行紧缩措施，默克尔认为欧元共同债券并不是好方法，同时也无助于促进经济增长，她坚称应通过严守紧缩政策对抗经济危机。

　　奥朗德却力主实行刺激增长，平衡紧缩的政策。奥朗德表示，希望看到欧元债券“写入欧盟未来的议程”。他支持发行欧元区共同债券，让各成员国能在货币“市场上较易取得融资的管道”。各国政府可以融资投资，这不仅可以在短期内创造新工作，往后也可巩固经济持续增长，同时协助必须付出高昂借贷成本的国家，如西班牙和意大利。

　　欧债危机迟迟无法解决，成员国之间开始对德国主导的紧缩计划发出不同的声音，会议上有相当多成员国赞成改弦易策，据意大利总理蒙蒂透露，支持奥朗德建议的占“多数”，蒙蒂指出，就连不属于欧元区成员的英国也对发行欧元债券给予支持。

　　不过，默克尔却毫不妥协，支持德国的数量有限但大多财政情况良好，如荷兰、芬兰、瑞典。默克尔表示，与会国对欧元债券议题进行了“平衡”的讨论，好些与会人士都质疑统一欧元区利率的益处。柏林担心欧元债券将让德国纳税人长期为欧元区较弱经济体的公共财政背书。德国经济部长罗斯勒也指出，欧元债券将让那些预算政策不健全的国家坐享其成，而致力维持健全预算政策的国家则受到处罚。
