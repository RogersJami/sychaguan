中国的银行还真不是能放高利率就不会低放。比如首套房贷，比如国有企业贷款，这些在很长一段时间里都是利率低，银行一样拼命放的。毕竟国有银行，还要考虑国家政策扶持方向和国家利益。

不过嘛，由于资金成本的不断提高，现在股份制银行已经基本不放神马没钱赚甚至赔钱的房贷、国企贷款了。他们的主攻方向就是小微贷，吃原来高利贷的那部分肉。商业银行小微贷，随随便便就是15%左右实际利率（把神马咨询费、什么存贷挂钩之类的变通手法算上）。

至于利率很高但国家又明令限制的房地产开发类贷款，就走表外，理财、信托产品。万科的信托，到客户年息都有10%，万科估计得支付12%以上利率。至于小开发商的信托，支付20%利率也不奇怪。

如果利率这块国家真的放手市场化，那么贷款必然大量投向房地产、矿业等高收益领域，而从实体经济中快速撤离。谁让前者给得起高息呢？

至于现在货币市场利率高的原因，到底有哪些因素，哪个因素是主要的哪个是次要的，我同意你说的央行紧缩是主要因素。但反过来我相信你也不得不承认，余额宝等炒钱空转，是抬高货币市场利率的一个重要因素。同时，你也得承认中国现在利率并没有完全市场化，余额宝等主要利用监管双轨制获利。

回雷雷兄：

       实体经济的确非常困难，但实体的困难不能归因于银行，因他们自己经营的问题，或者说被zf忽悠了。当年大放水，利率奇低，银行都求着企业贷款。好多企业忘记了利率不可能永远那么低，忙着扩大规模，跨行扩张，结果还没等挣钱，资金成本高企了。还有些企业更精明，用其它项目贷的钱都放在房地产上了，涨的时候挣得很爽，跌时时候就没钱还了，大喊要资金链断裂。温州最为明显，现在哭着骂银行骗他，当年挣钱的时候怎么不骂呢，无非是当年房地产涨了，现在跌了没钱还而已，说白了也是个房闹。
       银行从来不是雪中送炭的主，只会做锦上添花和落井下石的事。听起来很不负责任，但仔细想想银行也只能这样，挣钱的项目我投，明知赔钱还投钱是不可能的，对可能挣钱，但有风险的，那就提高利息以覆盖成本。
      兄所说的房贷，银行低利率放并不是出于好心，而是因为房贷是最安全的项目，不会赔钱。国企前些年放的利率也很低，一是因为当时放水银行求着人家借钱，二是担保好资金安全，自然利率低。但即便那时，赔钱的绝对不会干，在曹妃甸工行公然与河北LD叫板，省长放了狠话，结果还不是不了了之。再倒回来看，当时大放水时，国家政策扶持方向和国家利益很明确，银行怎么干的，国企不要钱银行硬往里塞，中小企业喝着多少，也就是意思意思罢了，原因很简单，让银行赔钱担风险，他不会干，逼他干他就磨洋工对付。
     现在房贷少了，资金成本高是一个原因，更主要是额度问题，政策不让往里放。银行的客户经理现在很纠结，很焦虑，工作压力空前大。做，全是摇摇欲坠的企业，不做，还有指标考核，进退维谷。要是让做房贷，他们巴不得呢。
至于股份制银行跟高利贷抢小微贷，这不正是我们想要的结果吗，我们应该鼓掌欢迎，总比让他们去借高利贷强。那些借到小微贷的企业倒是应该谢谢各种宝，至少不用担心被高利贷卸胳膊杀家人了。
     兄所说的“利率这块国家真的放手市场化，那么贷款必然大量投向房地产、矿业等高收益领域，而从实体经济中快速撤离。谁让前者给得起高息呢？”，我有点不同意见。兄这句话的前提是房地产，矿业给得起高息，这个前提是不稳定的，现在给得起我承认，但万科12%的资金成本，还有20%利率的同志，他们能扛多久呢，从理论上我不会推算能扛多长时间，但从现在逐渐增多降价的市场来看，有些人扛不住了，将来会有更多的房地产企业扛不住。兄所讲述目前的情况我完全同意，但我的结论与兄相反，先死的肯定是高负债行业，对房地产矿业，兄看到的是高收益，我看到的是高收益背后的高负债。
      信用紧缩是最可怕的事，到那时即使国家放水也没用。现在shibor这么低，说明银行有钱，水很足。但实体利率却是那么的高，万科都要12%利率才能借到钱，其它更不用说了。两者结合起来，我想来想去只有一个可能―――信用危机，银行有钱不敢放出去。08年宝宝大喊信心比黄金宝贵，不是没有原因的。那次宝宝喊成功了，现在木子会不会喊呢？当年用黄金买信心的同学现在还在高负债的泥潭里挣扎，这次即使喊，又还有多少韭菜呢。我想经过这么多轮收割还能剩下的韭菜大概也是铁韭菜吧，能不能割着就看刀是不是钻石硬度了。
      我这么悲观不知道会不会有人把我这自干五当作美分党。愿有兄弟说服我，给我危机解除的信心。
      远期的信心我还是有的，我不认为经济危机是坏事，这只是经济过程的一个调整，就象每过一段时间家里就要大扫除一样，非得糊弄到过年一起打扫，到时估计家里都没地方插脚了。因为前任混着过不打扫，这一次危机是针对前二十几年的大调整，我这草民还是龟缩自保吧。
