这次股灾，个人意见，是强总激进转型引发，中国事情历来经济问题政治解决，那么强总将会面对挑战。类似于大跃进开始阶段，彭总庐山会议挑战胡服，然后老大为了政治格局平衡，强拿彭总，随后三面红旗全面发展。
从现在角度看，当年大跃进也是迎接国内重工业化进程，步子大了，军队力量出来挑战行政系统时候，为了大局重新整肃后，步子更大，行政系统更加狂热，终于导致后面的马鞍形，然后就是七千人大会，一直到文革。
这个角度看，我们要警惕。
