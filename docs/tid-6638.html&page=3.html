貌似正式的说法是：
一，粮食不够吃。这问题一直困扰项羽到乌江。个人觉得，这可能是最根本的原因，并且符合项羽一贯的行事风格。
二，入关前，怕控制不住20万秦降卒。
还有不正式的第三，对秦王朝的恨，如滔滔江水，一发不可收拾。貌似这个也靠谱，看项羽进咸阳，一把大火，将秦人的荣耀全毁了。
