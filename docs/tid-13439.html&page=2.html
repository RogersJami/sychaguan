乌鸦 发表于 2015-8-25 10:28:49

不能这么轻易判断。

如果仅凭这幅走势图看（不看具体类型），这是一幅典型庄股走势，这可以看成交量的走势。这点就跟看国债图一样。所以，不能轻易下结论，因为庄股都是走势可控的。其实我们股市的老庄股，比如德隆系的湘火炬、合金投资等等，图形也是如此。后来他们的下跌是因为被戳破。而从一贯走势是无法得出戳破的暴跌点的。

对于目前我们的股市走势，从5日均量线看现在是5浪下跌的末端。（从6月初开始量的最高点开始）。但现在不到抄底时间，不过快了。

股市下跌，就像我们手里的乒乓球落在地上，开始弹起的振幅大，运行频率小，到最后频率增强、振幅缩小。也就是僵硬期。




乌鸦 发表于 2015-8-25 10:43:37

在市场的跌势末端，指数没有太大的跌幅了，而很多个股在这不大的下跌空间中，他们的跌幅是很大的。
原理就像我们起跑时，必须向后撤一步拉开起跑姿势一个道理。

所以，很多在这个下跌空间抄底进入的资金，账面损失会很大。而绝大多数人无法坚持拿到个股起跑的时候而出局。结果没了头寸、慌张开仓。（太多人开始通过抱怨市场或自己的运气，来安慰自己）

所以抄底需要找个好点位。
