千万别上当，现在土地价格一年一张，如果签长期流转，期间一旦出了问了很难解决。比如土地价格大幅度上升、比如土地被破坏无法耕种、比如期间土地被征用地上附着物的赔偿归建设、出资者影响补偿这一款等等等。如果要签，要确定：1、土地使用用途；2、合同提前终止条款，且你家不承担其他损失；3、重大政策调整或土地价格大幅上升后的你家的重新议价权。如果无法满足，就只答应签机动地承包合同，一年一发包，现在的土地受让方同等条件有优先权。
