供给侧改革，不要用行政力量去强行关停并转。要用市场的力量，关键点在于：大力反不正当竞争，建立良好的竞争秩序。

哪些算不正当竞争：
垄断：垄断价格和垄断市场
共谋：几个寡头协商操纵价格、市场
倾销：一切低于成本价的销售，一切所谓“烧钱”亏本“抢市场的行为（比如滴滴补贴司机，亏钱的外送业务等等）
独占：利用特殊的关系等不正当手段，阻止竞争者进入。比如铁路只供XXX水。一切政府机关行政事业单位的采购一律要公开，招标。特别是利益相关方不得独占，比如药厂托管了医院的药房，只买自己的药。

这些不正当竞争行为大家都习以为常，其实应该重罚。 一个良性的市场竞争环境下，差的企业自然淘汰了。

政府的角色是市场秩序的维护者，是对违法行为的惩罚者。

供给侧是个新瓶子，装的还是旧酒，几年前叫稳增长调结构去产能，再往前叫剥离不良资产。

供给的是财政、税收、货币、民生组合政策，具体路径和节点不清楚，大方向这样用：

制造业去产能，集中在煤炭、钢铁、水泥等行业。一，关停并转僵死企业，财政补贴一部分养老、医疗，个人也多交一点，维稳；二，续命休克企业，减免税收，取消行政收费，缓收养老、医疗公积金，新的低利率贷款置换旧的高利率贷款，或许还会下拨技改资金。

房地产去库存，房子暂时卖不出去，政府承租下来当作廉租房，或者买下来棚户区改造，局部刚起点小波澜，没见动作有多大，小动作也说不上。

产业升级、新兴产业砸钱，国内自己烧钱叫独立自主开发，国外撒钱并购叫充分利用国际合作。

双创是说暂时工作岗位没这么多，自己想办法折腾去。央企国企为提高效率甩出一些外包业务，人脉和技术在手的可能有点搞头。

钱从这几个地方来：

开源，一，出口高端产品，武器暗暗地卖，高铁明明地推销，核电站缓缓地建，飞机慢慢地造，这类新闻经常见；二，提高政府赤字额度，中央经济工作会议已经吹过风，看3月两会时人大是否批；三，股市抽血，去年上半年收割了一次。

节流，巡视组审计和纪委抓老虎配套，一，压缩政府开支，公车改革进行中，监管地方政府负债表，财政体制改革吹了2年风，看3月两会；二，压缩央企国企开支，身处其中的茶友们有体会。

总体一个字，熬，然后待变。
