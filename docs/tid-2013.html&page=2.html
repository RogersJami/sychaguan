是这样的，我示范一遍，当你登陆poco后，请（找到并）选择进入空间管理后台—相册

如图

http://img226.poco.cn/mypoco/myphoto/20140114/13/64772759201401141329254273364620272_002.jpg



然后，点击你要贴到论坛帖子里的图片所在相册，点开相册后，图片可见如图

http://img226.poco.cn/mypoco/myphoto/20140114/13/64772759201401141329254273364620272_001.jpg


再点击你要帖到论坛的图片，比如我这次点击第一张图。
图片打开，在右手边有选项，最容易的方法是点击：3 贴入论坛。 点击后，你会看到提示，告诉你在论坛贴里只需CTRL+paste就可完成贴图。

http://img226.poco.cn/mypoco/myphoto/20140114/13/64772759201401141329254273364620272_000.jpg
