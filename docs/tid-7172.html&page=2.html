广州仁者 发表于 2014-11-13 08:39:37

信息化高速发展下，相对传统商业而言，电商无疑是革命性的模式，洗牌正在进行中。

消费者和有实力的品牌生产厂家是直接获益者，传统流通渠道受影响最大。

餐饮、娱乐消费行业必须转向O2O模式才有竞争力。

暂时受冲击最小的是生鲜肉菜市场，但随着物流冷链的完善和社区电商的发展，很快也会开始受冲击的。

当前对厂家而言，主要的价格冲击来自两个方面：

1、假货的冲击：这个不多解释了，现在国家相关法规也在完善，各电商平台也在重视这个。
2、对传统渠道的冲击：同一件服装，专卖店850，天猫店350，这样的冲击本身是有些厂家自身缺乏货品类目管理所致。

说说自己对电商和实体的感受：
最近买了两次服装，一次给自己买正装，一次给妻子买衣服。
自己买正装，买了过后，去天猫旗舰店看价格，比自己买的衣服要便宜大约三分之一，但是，从吊牌的表示来看，一个是含羊毛91%，天猫的是含羊毛30%多。
给妻子买的衣服哥弟，实体店九折，天猫价格便宜一半，也是含羊毛的量不一样……
作为品牌来说，今后肯定会实行分开的，能在实体店买得到的款式质地，网上应该买不到。网上买得到的款式质地，实体店也不会销售，形成差异化。就像家电如此趋同的，都会在型号上下功夫。
网上消费，日化类什么的，相对来说比较有优势。
另外，今年双11，也特意看了一下价格，网上的价格很多是提高打折。附近的百货也是如此。
