这个问题很有意思，我总结其中的困惑源于四点：
1、价值取向不同。去汉城看看，城市寸土寸金的地段竟然很多破烂民房，人家土地是私有的，人家对这块土地有感情，你怎么着有钱，人家不愿意你怎么打官司都没用；
2、农民不断发现钞票是纸张，价值及其不确定，而土地价值至少永远实用，养育了人口几千年，所以拿政府补贴那点钱是贪图眼前；
3、农民发现，今后中国有钱人是住乡下的，城里公寓其实是未来贫民窟（最终要跟国际接轨的），未来只要农村的基础设施如自来水、电力、卫生、交通等跟城市接轨了，农村民房就是别墅；
4、农民尤其受教育很少的农民对城镇工作、生活以及今后劳保缺乏信心，谁知道10年后怎么样呢。在自己村子里，老了至少有邻居交往，还可以自己种点蔬菜、水果和粮食，生活仍然是充实的。
