craftsman 发表于 2012-11-25 21:37:55

EPT的学习还没完我们就开始联机调试我第一次从无到有无样机设计的机床，还是当年难得一见的CNC机床。X研和他的人马在招待所住下，由于有两个全闭环的圆坐标，还用到了高精度的圆光栅，所以来的人除了计算所还有光机所的人，那几天晚上总在招待所陪他们吃饭，听那人将一些蒋筑英的事情，那是当年的风云人物。

这次联调让人头疼的事情太多，每个环节都要做大量的工作，有的地方还改硬件。在反馈环节，首先是光栅信号的李萨如圆的调试，然后看倍频到多少时信噪比的增益最大... ...到了伺服这边事情更多，增益小跟踪误差大，增益大又会自激，加前馈... ...相比起来计算机这边的麻烦最少，但是软件上也为上面的各个环节提供了一些帮助。能在初期认识CNC的时候有这样的机会，看到一套自制的CNC系统对认识数控技术有极大帮助。我们的总工L总在会上说，“这次就是出不来产品，我们也要出人才。NC技术对我们太重要了”。

后了，后来也介入过国内一个系统的开发工作，处于爱好收集了好几套系统的源代码。几十年后我认为PLC，CNC系统都是有可能出现国产大户的，至于至今没有有建树的国内企业很可能与这些年社会的急剧转型有关，在急剧变化的不确定中没有人在愿意做长线的投入了。不过我多年来认为CNC系统今后作为一种商品会消失，今后的机械控制系统一定是PC机（工业级的）+软件+运动控制卡+伺服驱动器+伺服电机+编码反馈系统组成。其中控制软件与运动控制卡可成为一个行业，电机与伺服驱动会成为一个行业，伺服驱动最终会集成到电机里。用户买自己需要的软件及运动控制卡，根据选购的伺服与电机，在PC上做一次硬件配置和界面组态就可以了。其实现在的运动控制卡等已经出现，想在就是没有人做一个功能集成的软件，我也联系过国内的龙头院系，他们以为路子可行但这样的软件很难在现有的商业环境下找到盈利模式。但我认为这一天最终会到来，机械制造商根据需要买不同轴数的卡，根据选用电机及其他执行元件不同在PC上配置一次硬件，在根据下端客户要求组态出一个界面就可以了，制造商将使用说明书、工艺手册、诊断手册、网络远程服务全集成在这台工业PC上。

请教不敢。
3D打印几年前见过，一个透明容器装满一种遇到一定强度的激光就会固化的透明树脂，从不同方向多个位置的激光源向容器内的一个点发射低于树脂固化阈值的激光，当激光在该点汇聚超过树脂固化的阈值该点就被固化，精确移动汇聚点就在该容器内生长出一个固体模型。这很像伽马刀的手术原理。
至于有人说打印飞机大炮，现阶段还不行，我是说你真用它打印出来的肯定打不过传统工艺造的。但今后难说，航发的关键件使用可控晶体生长技术做的，原子核可裂变也可聚变，如果真能按设计受控裂变、聚变，晶体也能受控，那就似乎可行了。
不过今天我依然觉打印飞机得很科幻。记得儿童时读过一本苏联的儿童科幻读物，说我们长大了就会发现当时的一个行业：发电、配送电行业消失了，因为每家每户的门口都有一个插头，插着一个核电池，工厂门口插的那个核电池不过大一些。嗨，今天也还在发配电，并没有满城尽插核电池。但谁敢说这种未来是完全不可能的？

craftsman 发表于 2013-1-8 21:24:10

东芝事件后，开了很多会，听军代表报告，到XX院听讲座，我听的专题是“舰船变速器中的齿轮”，听得我热血沸腾，想起七十年代初，家长带我看了几部“内部影片”，什么“啊，海军”、“中途岛大海战”、“军阀”、“火山禁地”等，家父曾对我说过“以后的国家民族间的竞争主要是他们的工程师的竞争”。
回来汇报完了，老总就要我有要出去进修。一年半啊，在XX工学院，题目大得很“计算机在机械工程中的应用”。当时进修如果是教育科派出就没有补助，总工派出按出差补助，住在研究生楼。缴费报销，补助归己，没天理呀，一个月几十块工资，竟拿好几百补助。
当年上机的计算机叫VAX II，不是个人计算机的操作系统，我们的主要学习内容是CAD、CAM、CNC也有一些涉及。有一天在机房里，一个fortran程序怎么也调不过，情急中叫机房实验员帮我看看，他说“计算机是不会出错的，你检查你的程序。”邻座的一个高个老师站起来用广东口音说“你的话是真理，就是对不起你拿得工资”。他帮我指出了错误，后来我知道他叫钟毅芳，教授。
校庆的晚会上一个电机系的硕研唱了一首歌“你从哪里来，我的朋友”，我等惊为天人，第二天同班的洛拖所的室友找了个在该系做讲师的同学，就去看望那个女生，大家都说台下不化妆的她比台上漂亮多了，她大方的和我们每个人握了握手，我开玩笑的对她说“人不捯饬不好看，捯饬完了你再看----还不如不捯饬呢”。回来的路上我等已婚青年各自表达了心中的“怅惘”。
由于是各教研室的大牌上课，这次进修收获极大，同时也把我的所有爱好深深的羁绊在这个专业的天地里。
回到工厂开始搞CAD系统，一年中有几个月待在中关村，当时的中关村可不是今天的样子，“四通”、“科海”、“大恒”... ...你经常能见到求伯君一类的人物。那时中关村有一份自办的报纸“民企之光”，上面登载好多大牛的传记：2.13系统的吴小军、大恒的三剑客、西山打印系统... ...我每次从北京回来都背回工厂，大家十分珍视。
当时需要决策的事情是：用工作站、还是用PC机？那时的工作站有阿波罗、还有sun sparc station，最后还是确定在PC机上搞，买了两台IBM XT和几台AST XT，找个在大院工作的亲戚拷了一张五寸盘的autoCAD 2.6就开始了CAD起步工作，老工程师说，将图板上的工作搬到屏幕上还不如在图板上做。确实，如果将会图工作放到屏幕上是很可笑的，当时用的一种绘图工具叫数字化仪，用逻辑笔在上面点击菜单，绘图。为了提高绘图效率，我们准备买一套在autoCAD上二次开发的软件，等销售把试用的软件装了，我们少有的一致认为太不值，全部代码都是autolisp编写，标准件均为形或块做的。从那天起，我带了几个人开始自己用highC（一种在DOS下的32为C语言）开发autoCAD下的机械CAD系统。首先是汉化autoCAD的菜单、对话框，命令行与提示栏的汉化最为困难，要将点阵字库驻留内存，在由开发的执行代码写到屏幕上，找到不同分辨率屏幕上的内存的帧页面地址很困难。当我把这一模块代码调通时，机房里的人都围了过来，我也狂妄的喝道“纸笔伺候！”一个小伙子真拿了一张零号图纸和一只毛笔来了，只是没有墨水，硬是把绘图仪用的墨水倒了一些，我不知羞耻的写下了“合金钢不坚，中子弹何难。科学有险阻，苦战能过关。”刚写完这几句，一刚分来的小姑娘叫到“好湿，快把领导的黑宝贴到墙上”。我慌忙制止，加上“录叶帅诗”几个字。大家你一言我一语的议论着，中有一人说“把DOS的内存映像搞清楚，你心里一定很有成就感吧？”我说“搞清楚内存驻留，DOS内存分配就好像是无忌惮的偷窥微软的最私密隐私一样过瘾，此番快感无法言传。”再看周围的女同事纷纷用手捏住了衣领领口处。

关于CAD的定义有过很多，我以为最好的定义是北理工的女博士副校长说的“CAD是将工作变为游戏的技术”。
我们的机械CAD系统将设计计算、出图集成起来，这是当年autoCAD没有的功能，齿轮、凸轮、轴、皮带传动、链轮传动... ...在自己开发的模块内完成设计计算与校核后直接生成DWG文件，我们把人家DWG（不是DXF）文件格式也搞到了。这些图形文件在autoCAD的平台上做工程化编辑。
当年有个叫中科健的公司，曾和我们联系想让我们搞CAD与CNC的团队到深圳去，要离开我们的企业下海，这不是我的选项，离开哪些不断给我们找麻烦的机器，听不见他们的轰鸣，那还是生活吗？
有一天母亲叫我回家见一个人，见面才知道是家母的中学同学，XX银行的行长。“你妈说她儿子会搞计算机，我还想她自己也不知道计算机就没当真，后来在科委见到人才库里你的名字才了解到你还得过奖。这样吧，你调到我们行来，我们马上要上信息系统，非常需要人，我和你妈像姐妹一样，你来一年后保证升到正县级。”
“阿姨，我喜欢机械专业。我不喜欢银行的工作。”
从那以后，地面上的大小关乎计算机的事，纷沓而至。
一天，同事（入行不久的小姑娘）站在机房门外，不愿换拖鞋就对我大声说“X头，有人找你，说是找小X，我说那是我们所长。”我赶快探头一看是科委主任，那时部属企业与地方联系不多，小丫头根本不知道外面来的也是领导。主任是本地人，一所工科大学老师，文革后想法调回了老家。
“小X，喔！X所长，走走，下楼上车，车上边走边谈，我已经给你们L总说好了，你来帮个忙。”老主任着急的说。
“老前辈你要再敢叫X所长，就是乱伦。我告你为老不尊！”
“哈，哈... ...当领导了你还是正经不起来，我看你的进步有限了。”
路上得知，做炼钢电极的碳素厂进口生产线上的计算机坏了，生产线停了。这还了得，时间就是能耗。到现场一看，还不是PLC，一个CRT上有光标在闪启动不了。
仔细问了值班人员，这台显示屏断电的后果是什么？你们断过没有？得到答案后我拆开了控制柜，发现起就是一台PC机，只是没插键盘。
拿一个键盘来，进入BIOS设置看到硬盘式好的，可能是引导区数据坏了。在他们厂的办公计算机上将问题盘拷贝的办公计算机上后用办公计算机顶岗生产线经正常了，那个年代没加密呀。
带回硬盘，发现可能是非正常退出把引导数据搞坏了，数据映像备份，硬盘搞好了，顺便看了看里面的文件我笑了：可执行文件竟是basic，还不是编译后的*.exe文件，打开文件读一遍，基本懂了这台机器是干什么的。将硬盘的DOS换成CCDOS，basic的显示文本全写成汉字。
硬盘送回去，该厂领导大惊，说这是今年的技改项目，定要报上去。我们研究所收入了第一笔令我们感觉天文数字的小金库启动资金。

还是国企靠谱，您说的这种行业内集训，私企怎么都是不可能的。行业内集训不仅交流信息，还能为参与者创造出好的学习条件，为学习者指明方向和边界，最快速度收到时效。
作为对立面，私企里面人心涣散，一个人想学习，可是真困难。

那一年，雷军还在武汉；那一年，PC机还用的是DOS；那一年，和大牛倒软件人家军用挎包是一挎包硬盘，我们是一挎包三寸、五寸盘；那一年，我也迷上了解密、病毒这些傍门左道。
科长，此时的设计、工艺、情报资料等单位合并成为研究所，也因年纪大不当科长了，我们依旧以“欺负”她取乐。她们一代人学的是俄语，用计算机本来就费劲，好在我可以用最为粗俗的语言给她说个明白。但经常用着用着又糊涂了，DOS命令“dir”经常敲成“die”，然后下行就会出现那个全世界人最熟悉的提示“坏命令或是文件找不到”。一天我在她用的机器上运行了一个我用汇编写的驻留程序，die执行，弹出一个闪烁汉字提示框“die拼音是：爹；英语是：死；你个哈货，要爹还是要死！”搞了恶作剧我也忘了，直到中午下班科长把我拉到没人地方，紧张的对我说，她的计算机骂她了，还是用湖南话骂的，我一听心里大笑还是严肃的对她说“不要紧，下午你教我几句更恶毒的湖南骂人话，我定要帮你骂回来。”
下午到机房说给小字辈听了，皆大乐。邻座的工程师笑得直不起腰对我说“上午就觉得奇怪，科长的脸无端的红了好几次，还有一次差点伸手打了显示器。”
科长来了，没有教我骂人的湖南话，倒是用最恶毒的湖南骂人话把我好好骂了一通。

机械专业的我惭愧地飘过，现在学校这个专业的学习基本上还是几十年前的老一套，制图画图，专业已经脱离生产实践很多了。

“几十年后我认为PLC，CNC系统都是有可能出现国产大户的，至于至今没有有建树的国内企业很可能与这些年社会的急剧转型有关，在急剧变化的不确定中没有人在愿意做长线的投入了。不过我多年来认为CNC系统今后作为一种商品会消失，今后的机械控制系统一定是PC机（工业级的）+软件+运动控制卡+伺服驱动器+伺服电机+编码反馈系统组成。其中控制软件与运动控制卡可成为一个行业，电机与伺服驱动会成为一个行业，伺服驱动最终会集成到电机里。用户买自己需要的软件及运动控制卡，根据选购的伺服与电机，在PC上做一次硬件配置和界面组态就可以了。其实现在的运动控制卡等已经出现，想在就是没有人做一个功能集成的软件，我也联系过国内的龙头院系，他们以为路子可行但这样的软件很难在现有的商业环境下找到盈利模式。但我认为这一天最终会到来，机械制造商根据需要买不同轴数的卡，根据选用电机及其他执行元件不同在PC上配置一次硬件，在根据下端客户要求组态出一个界面就可以了，制造商将使用说明书、工艺手册、诊断手册、网络远程服务全集成在这台工业PC上。”

我从业的那个国有企业的第一任党委书记（八十年代以前，党委书记才是一把手）是一个烈士遗孀，可惜我来到那个企业之前她已经调回北京了，我到那个建于六十年代末的三线工厂，没能见到她本人，听过很多关于她的故事。八十年代，我为单位的CAD项目在北京到过她的独子的家里，在墙上见过这位女性，清秀的叫人敢欺负她，完全没有人们口传的彪悍之风、令人生畏的犀利眼神。由于她和我父亲同在中原的新四军，父亲也有一位年轻的女性政委，我对她有一种天然的亲近。
美丽的女人，关于她的故事很多,生产调度会上人们时常听她如此训人“一共十四种关键件，有九种你不知道在什么工序上，我来告诉你：主轴在渗碳炉边地上，一共十六根，就这样堆在那里。到了磨序，你就可以来告诉我，主轴都变形了，是热处理变形，反正渗碳炉不会说话，没人会知道你的渎职行为。国家的工资不是用来养活你这样祸国殃民的寄生虫的！... ...”
那个年代的企业领导，多数是这样的人。等我这个晚辈来时，党委书记是一个山东的八路军，每个月底、季度末。要交产品入库（计划经济，入了库才能上报产值）时，他会拖着肥胖的身体爬上天车，坐在驾驶室里。从上往下看得清楚，谁要停下来，他在上面立刻大骂。
天车上的“椅背”也就是个大的法兰盘，把他老人家的短袖圆领白汗衫（当年国标夏装，现在看不到了）磨了十分规则的五个洞：中间一个大的，四周均布四个小的。通宵加班后，回到干打垒的集体宿舍的大通铺上，睡不着了。宿舍里有一个郭德纲的老乡，铸造车间造型工，十分会讲故事、笑话。大宿舍里有人在洗澡、有人在打鼾、有人在洗衣。老书记睡不着非要那个造型工讲个故事，周围的人也起哄“来一段！”，那造型工可能累了、也可能看见大家要求故意端着，就是不讲。书记急了，山东人学者天津口音“操你妈妈，不讲是吧?不讲是吧？你老婆调来的事没门啊！”周围的人也群情激愤“再你妈妈不讲，把bikei的被子包上大石头给拽鱼塘切”。那些嘎小子说了就要做的呀，那造型工忙学者样板戏的反面角色认耸的腔调“长官息怒，长官息怒！我讲不奏完啦吗？不过介一条咱搁头里啊，讲完啦咱大伙睡觉，别他妈没玩没了！”大家一致怒吼“就是就是！”
书记也发话了“谁要多一句嘴，大伙操他妈妈！”，前半句书记用的山东家乡话，后半句人家也是标准天津话。
造型工开口讲故事了。宿舍，其实还不如今天民工的活动板房，但那时没有任何娱乐活动，讲故事时四周的人都静下来了。听着故事：
“有一个山东老客，到南方来了。他一看，好嘛！介地方介嘛多河呀，介河里肯定有鱼呀！那bikei山东老客嘴馋啦，想吃鱼，奏拿一杆，拴上线，弄两蚂蚱拴线头里，往河里一拽。人蹲那钓鱼哪！小半天过去啦，那老客皮都晒秃噜啦，还他妈没钓上来一条鱼叻。
介老客心里嘀咕：怎么介嘛大的河，就没一条鱼叻？
河里的鱼呀，也在嘀咕：介你妈河边怎么就多出一个长白山嗫。吓得人家鱼呀都不敢吃食了。
介天下来老客吃不住了，在河边破口大骂‘介你妈介条河里都是嘛玩意儿！操你妈妈... ...'他奏没玩没了一直骂道太阳落，月亮起。他介一骂，河里有一路过的王八听不过去了，非要爬上去和那个老客理论理论，介鱼呀就拦这介王八，‘王八大哥，您呐别生气，银家骂我们叻，您呐赶路走您的吧’，那王八不干了，介王八的义气重啊！脾气大啦‘您钓不着鱼你不能把介一条河都骂啦！介是咱的母亲河呀，你个bikei的你张嘴奏骂，您是嘛玩意儿’
老客听不见河里的王八话，转身就走了，介王八一看急了，一下就蹦到那老客的背上去了，到今儿那王八还在那老客儿的背上扒着叻”。
众人看着老书记背上的那几个破洞，大笑起来，唯独老书记不知为何，疑惑的看看大家嘴里喃喃的“说好了的，大家睡觉睡觉。”
