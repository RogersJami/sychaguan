学习了一段时间，怎么感觉学不进去了呢

学习一定要集中精力时间，最快速度学完一门，心里有个谱之后再来第二遍，第三遍，不要卡在一个地方啃啊啃，因为一门课知识点很多，卡在一个地方没进度是完全没必要的。甚至不重要的章节可以完全放弃，比如我看票据法，始终理解不能，干脆放弃掉了，占分比也不多。但重要章节一定要搞懂，有时候还要背诵，做题了解出题角度。

卡在一个地方有百害而无一利。教材多读几遍，有时候前后对照起来，有些问题自然在潜意识里面就联系起来或者解决掉了。再就是多学几遍，然后对照着各种资讯上面的新闻，当能将所学的内容与现实结合起来，并感觉到对现实的认识比以前有很大的进步，这样就形成一个学以致用的良性循环，于是越学越有劲。

再者，每个人的学习方法不一样。我属于比较笨的，学习方法就是 认真做笔记，每本教材做3遍笔记，知识点和知识体系慢慢就建立起来了，之后就是做题，运用知识。

我想对多数人来说，考CPA都需要一个记忆的过程，都要熬过枯燥的阶段，但这个阶段是基础，会在以后发挥作用的。

比如审计一向是CPA的难考科目，但我在认真学审计的过程中，不知不觉接触了审计的思路、思维方式、操作方法，即如何在信息中抽象，去粗取精并形成比较靠谱的结论。这个教材是很枯燥的，包括很多表述都和一般的科目不同，要理解概念和逻辑关系，但是通读若干遍，会有恍然大悟的感觉。

牯岭四 发表于 2016-6-30 18:11:05

专业阶段考了四年，其中后两年专门在考战略的废柴出来冒个泡。

专业阶段学六门课程的时候，除了战略和财务管理之外，其他科目通过阅读教材的方式来入门，都不太合适，枯燥，容易犯困。网校上下载些音频，加个速，作为入门引导，相对容易。

CPA涉及的知识比较多，系统性谈不上，说是大杂烩倒是很合适。即使考试考不过，整体学完还是能对企业经营的大多数方面有个粗浅的了解，真遇到个问题需要深入研究，至少知道上哪去找资料，比学校里的大部分理论课程更贴近实际。

当然，如果以前完全没接触过财经方面的知识，学这个一般是从剥层皮的感觉开始。我以前是学生物的，剥皮的印象挺深刻。

这东西说难确实谈不上，尤其是改成机考以后，专业阶段难度较纸考时代已经大大降低。

有志于完全考过的，也就是自己跟自己磨，拼的就是个韧性。

从投入（主要是时间）产出（直接经济利益回报）的角度上来讲，这个考试性价比极低。从学以致用的角度来讲，实用性还是不错的。

正在学造价，又在重温剥皮的感觉。
