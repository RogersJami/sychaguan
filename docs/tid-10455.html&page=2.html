响应馆主，来个家常小焯（清水煮），用的是西兰花幼苗和紫皮芦笋，那紫笋真甜，反正是跟紫物飙上了。

西兰花芦笋炒肉片的做法
   
1.猪里脊（牛肉）切片，用适量料酒，酱油，糖（少量，根据个人口味，可省）和生粉抓匀后腌10分钟
2.芦笋切成段
3.西兰花切小块，放入滚水中煮2分钟，捞出冲冷水后，沥干，待用
4.锅里放油，把肉片放入炒至变色捞出沥油
5.锅中留底油爆香蒜片，放入西兰花，芦笋翻炒均匀后，加入肉片，放盐和鸡粉（可省）调味，即可。 讲究的话可用生粉水勾薄芡，最后淋上香油。
