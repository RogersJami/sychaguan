3bie 发表于 2013-2-21 22:02:13

时间过得真快，读忙总的文字已经进入第四个年头了，到今天终于感觉到自己有进步了。从忙总的文字中，我经常会被他一些看似矛盾的话语所困惑，例如忙总经常告诫我等后生，人要有理想，要有激情，要自强不息，要坚韧不拔，但又告诉我们不要理想主义，要从实际出发，要看菜下碟。忙总经常表现出自己的理想主义，但又一遍遍地说自己是实用主义的。刚开始读到这些文字，如果没有社会经验，人就容易迷惑，到底该实用主义还是理想主义。到今天，有了几年的社会历练之后，我终于知道，主义不主义都不是问题，关键是知道自己要什么，然后去做。忙总以他一贯直白的文字告诉我们：纯粹的理想主义者在政治这个漩涡中都活不长，纯粹的实用主义者，也难以长期凝聚人心，保持斗志。真正的政治家都是理想主义与实用主义的结合者。这种政治家就是真正的高手。也许我一辈子都成不了高手，但我要用忙总曾说过的话勉励自己：取乎其上，心向往之！

我看《毛泽东传》的体会就是毛主席是一个把理想主义与实用主义结合到完美的人。


光有理想主义，没有实用主义，就不能做事情，只能空对空吐口水，意淫自己的理想多么伟大，最后成为羡慕嫉妒恨一派。光有实用主义，没有理想主义，就是一个争权夺利工具，没有目标，最后成为行尸走肉。


我们现在自以为有理想的，实际上并没有理想，因为只有在现实失败面前经过考验，甚至生死考验，还能坚持的才叫理想。现在很多人的理想其实是口水。看看十多年前哪些所谓极有理想的所谓民主人士，自由战士，人民的大救星都在干什么，除了为了美国人的三瓜两枣狗骨头而互相抹黑，揭短，泼粪外，就是坑蒙拐骗，男盗女娼。


我们现在自以为实用主义的，实际上差得远，因为没有能力做事情，拿不起放不下，本事一点没有，就会吹牛。


还是老老实实修炼自己，做事情，对自己，对国家都更有用处。
