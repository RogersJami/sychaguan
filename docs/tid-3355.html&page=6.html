听风看雨通三统 发表于 2012-10-8 22:29:55

关于共同富裕的问题。坦白讲，近年来，对于小平论述的社会主义的两个方面，第一个方面，即解放和发展生产力，是落实得比较得力的；而对于第二个方面，即消除两极分化，实现共同富裕，则有不同认识。最主要的分歧，其实不是是否应该实现共同富裕，而是是否应该在现阶段就把共同富裕，提到和改革开放、发展生产力同样重要的位置。对这个问题，主要有三种意见。

第一种意见认为，中国如果能在经济发展过程中自动实现“帕累托改进”，即各个群体的利益都能自动增长，就是自然实现了共同富裕。因此，只要能保持经济发展势头，就是自然实现了共同富裕。因此，没有必要单独提出共同富裕，冲淡实践中的经济建设的中心地位；

第二种意见认为，如果任由10%的人群控制70%以上的社会财富的趋势继续发展，这少部分人，就会利用自己掌握的经济利益，在掌握决策权和话语权的群体中，收买自己的代表，最终彻底固化两极分化的利益格局和发展趋向。那时，将会出现社会彻底分化，甚至爆发社会革命的风险。因此，必须在当前就把共同富裕提到中心工作的地位，以共同富裕的实践，推动经济建设的健康发展；

第三种意见认为，既不能指望自动实现“帕累托改进”，把这个当做“共同富裕”。这是一种敷衍的态度；也不宜以强势的政府行为，全面重新分配财富，人为实现共同富裕。这会导致社会经济发展活力丧失，影响中国把握经济跨越式发展的历史机遇期。正确的做法是：以共产党为运作资源的领导核心，重建党的社会基础。在这个过程中，争取调动尽可能多的资源，服务于民生改善，以及以就业为民生之本的产业政策导向。坚持和推进这一条，其实就可以保证党的核心领导地位，以及这种核心领导地位，更多服务于社会资源配置活动的“重心下移”。这才是真正稳妥且符合国情的实现共同富裕的思路。

目前看，中央确定的实现共同富裕的思路，应该是第三种。具体说，就是不去刻意强调共同富裕，但坚持共同富裕不动摇。而具体实现共同富裕，则是落实在民生改善上。而且，这种改善，是和资源配置的比例，以及党的社会基础重建和巩固的过程作为保障的。也就是说，党要保持自己的领导核心的地位，就必须重建和加强自己的基层社会基础。而要重建党的基层社会基础，则必须落实在促进就业，改善民生上面。这既是以经济建设为中心，也是在实实在在地改善民生，更是在加强党的执政基础，真正巩固党的核心领导地位。
