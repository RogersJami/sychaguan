干蒸烧卖

http://images.dayoo.com/guangzhou/64375/64380/img/attachement/jpg/site1/20090116/001372af5a290adae4fc0f.jpg



广东小吃。在２０世纪３０年代，干蒸烧卖已风靡广东各地，近２０年来，又传遍广西的大中城市，成为岭南茶楼、酒家茶市必备之品。色鲜、味美、质地爽润。用鸡蛋液、水和面作皮，猪肉、虾肉、冬菇作馅，捏成石榴花形，蒸制而成。其色皮淡黄，蟹黄鲜艳，皮软、肉爽，稍含汁液，鲜美而香。没有哗众取宠的外表，也不是移山填海的份量，却注重精致讲究的填充，是广州人喜欢的茶楼传统早点之一，是一种以烫面为皮带馅上笼蒸熟的面食，主要有猪肉和牛肉两种。现在有配以糯米、虾肉、冬菇、马蹄粒、笋粒等爽口配料，使其更加鲜香爽口，肥美不膻。 干蒸是广东的一种点心，是烧卖这种中国土产点心的地方衍生品，虽然烧卖的历史与广州太大的牵连，但它偏偏是广州人早茶最喜欢吃的早点之一。 其中牛肉烧卖的历史有七八十年之久。现在广州也有的高档茶楼在烧卖里面用上鲜虾肉，这种新式的烧卖做法，味道极其鲜美，口感爽滑！所以，在广州喝早茶的时候，记得叫上一笼的干蒸烧卖，品尝一下这历史悠久的美味。

广州喝早茶的流程:



　　1、找到位置，



　　2、叫服务员开卡，冲茶、摆杯筷、品茗



　　3、写卡点要自己钟情的茶点，坐在座位上等服务员推点心来，也可以直接拿卡去前台端，然后就是开始品尝咯.



　　4.因为喝早茶一般都不会一个人去，若你是主人的话要经常要帮其他人斟茶，茶不要太满。若其他人帮你斟茶你要用右手点击一下桌面以表谢意。若茶壶无水把茶壶盖翻开，提示服务员续茶水。



　　5..叫服务员埋单，开发票
