花差花差农民 发表于 6 天前

首先感谢葡萄三万二千字的大礼！十多年了，从葡萄的文章中学到了非常多，最重要的是学会了如何看历史。不敢说完全看懂了葡萄的所有意思，这里写个小读后感跟大家分享。
我从小也可以算一个历史爱好者，但是没有葡萄的引导，我最多也就是个爱好者，不能窥得政治经济学的真髓。现在的外面的历史书，已经大多数沦为明朝那些事，晓松杂谈，逻辑思维等骗钱的东西。像葡萄这样真正在教人看懂历史的，几乎没有。
可以看得出来，葡萄在这篇写的很用心。历史的发展的主线到底是靠什么，葡萄给出的体系的答案。明朝末年并不乏有能力的仁人志士，但无力回天。清朝可以说结了所有的前朝经验，但面对三千年未有之变局，旧的体系无法应对。中国有如此众多的人口和早熟的文明，我想任何时候都并不缺乏人才，即使天不生仲尼，也会有叔尼季尼的。
现在的体系又到了变革的节点，物联网、5G、AI、区块链，这些将成为改变社会治理方式的工具-因为这些能让一切信息的汇总、追溯不通过各级官僚系统就可传递到政策制定者手中。所以习总要亲自担任中央全面深化改革领导小组、中央网络安全和信息化领导小组的组长。但是，这个未来将向何处去，是1984？Matrix？还是小灵通漫游未来？这就要看社会大众的了。这两年尤其关键，能不能闯过去，可以说谁都没有把握。
幸运的是这次中国所处的位置，可以说是领跑者之一。真正的对手也只有一个。留给欧洲的时间不多了。终点是星辰大海，看这次谁能笑到最后。
美国是一个伟大的国家，在于他能变化，愿意变化。床破也许会成为今后历史上着重书写的总统。
现在的网络变成小圈子的情况越来越严重。前几天刚主动退出了一个微信群。那个群基本都是炒股的，整天关心的事情就是哪个股票明天会涨停。话题永远只有骂证监会，骂ZF，传播翻墙来的各类小道消息，希望社会制度赶快变天，总之谁碍着他们发财就不行。我看着也烦，干脆退了。
可想而知，葡萄的小规模社会学实验，和伟人的那次大规模实验一样，都失败了。不管你用的是什么手段，什么方法，永远不变的只有人性。在物质没有极大丰富（或者精神极大丰富？）之前，企图利用家族传承/外部势力入侵/各类信息不对称等机会去占据社会优势地位永远是大多数人的第一选择。所以，所有的革新和变化，都不可避免地有会反复，甚至失败。
很惭愧的是，最近几年，步入中年，各方面谈不好，但也并不顺利。现在是人懒，时间碎片化，脑子也不行了，尤其提不起精神来看书学习。葡萄的文章给了我一次警钟：这样下去恐怕要被开除出数据社会了。我得振作起来，抓紧时间学习新知识了。
很多人说葡萄的文字比较难以看懂，确实也是如此。虽然这篇文章有3万多字，但是仍然是高度浓缩的，需要有比较强的知识储备。另外除了有众多的知识点和发卡弯外，葡萄的行文方式似乎也比较英文化，独立句式和复合修饰语，各类从句都比较多，可能有些人读不顺。鄙人不才，昨天又通读了一遍之后，改正了原文手误的错别字，给长句尽可能加上了逗号、顿号，以利阅读。如果哪位需要这个版本，请发邮件到baichuantao@foxmail.com。我将发送给您。
最后，谢谢思炎这些年来为大家无偿的服务！希望大家有缘再见。
