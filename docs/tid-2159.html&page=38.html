可能跟长期吃殖民地有关，西班牙当年从南美洲可是抢了不少财富的。

欧洲人变懒，跟殖民地没关系，源头是英国工党在二战结束后开始搞的所谓“福利社会”及“人文主义”，由此诞生了一系列的政策怪胎

福利社会最大的作用是为工党拉来了相当多的选票，缺点就是为政府背上了一个沉重的负担，而且由于选举制度的存在，尽管人人都知道这样的后果必然是导致国家财政破产，但没人敢主动提出削减社会福利，尤其是不恰当的福利，因为这些福利照顾到的选民意味着可观的选票

欧洲反移民的极右翼注意到了这个倾向，所以才会主张限制移民，排斥移民。这些移民往往来自不发达国家或前亚非拉殖民地，除了从事低端体力劳动外，基本无法为欧洲的经济做出更大的贡献，但出于“人道主义”，这些移民却又可以享受政府的福利政策，其中生育是最简单、获利最多的福利劫取方式。他们的选票，就是他们与政治家讨价还价的最好武器，而且现在的形势表明这些低端移民已经可以影响一些国家的政策倾向了

总的来说，用个时髦的话来讲，就是“体制问题”，在B 希特勒出现前，是无解的。
