看电视里面，遵义会议上，王稼祥躺在病床上出席会议支持毛主席！
请问是什么电视剧？
-----------

王稼祥在1933年4月遇空袭被炸穿了肠子，在没有麻药的情况下历经了八小时手术.但因条件有限，弹片无法取出,在长征时他的肠子流脓，甚至爬出蛔虫，仍以常人难以想象的毅力坚持下来。

另外让我敬佩他的是： 他年轻时留学苏联五年，英语、俄语成绩都非常好，但他不像博古王明等留洋的人，他不依照西方或照苏联人的观念来思考中国的问题，而是始终从中国人的立场与利益考虑国际问题。

不难怪，他与主席在一见面就惺惺相惜，然后就一直支持主席，对挽救了红军与中国革命有巨大功劳。
他是第一个与毛直言“轰李德下台”的，也是首次提出 “毛泽东思想”概念的（在1943年7月8日，延安《解放日报》）

他是新中国首任驻苏联大使。
点将他而不选派其他留苏精英（毛主席曾曰：那些“对中国考虑少，对别人的事却操心得太多了”的精英......），可见主席与总理对他的信任，也可见新中国是要走自己的路的。

-----

历史阶段不同，现在的精英比老一辈的条件好太多了，可也出现了不少群魔乱舞没有节操，哗众取宠的一帮公知，在战时讲得难听的话是汉奸。 而王稼祥一个真诚的，为中国利益出发着想的前辈，他的节操是值得现在所谓“爱国公知文人”学习的。
