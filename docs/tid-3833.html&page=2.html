马甲屁特 发表于 2013-2-28 17:31:23

帮lz补充一点，几年前写的
随着互联网发展，各种学习资源越来越丰富，但信息的过多和过少一样会造成使用的低效率，如何高效的获取资源成为必备的技能，下面简单汇总一下常规方法，希望能对大家有所帮助。

1、资源分类与简述
1.1 电子书：大体分为扫描和文本两大类型。
1.1.1 扫描电子书
超星图书馆：在早期，为了压缩文件体积以便低速网络传输，他们扫描图书的分辨率设置的都非常低，有部分文件到手后需要处理。pdg阅读软件对新pdg文件兼容不好，一些新扫描的彩色pdg不能被老软件正常显示。不过好在pdg很容易通过虚拟打印的方式转化为pdf文件；
大学、城市图书馆的电子书库：大部分扫描质量较佳，新书较多，有些偏门的也能找到，这类电子书的集散地是人大经济论坛，但那里经常要积分才能下载。如果不急着用可以等一段时间，verycd的资料版总会有人转载的；
发烧友个人制作，这个多在一些专门的论坛，比如有网友拆了大型字典后扫描上传，精细程度远高于前面两类，可与而不可求。但资料类型多限制在字典或较冷辟的；
讲义、试卷、内部教材等，多出现在考研论坛，大多需要用钱购买，对普通读者意义不大；

1.1.2 文本电子书
直接录入，在早期多采用这种方式，比如九十年代初磁盘装的唐诗、《老子》等经典。另外有专业人士在做古籍录入整理工作，这部分书籍质量都还好
OCR扫描，现在的文字识别软件可是讲扫描文件直接转化为文字，但中文或版式复杂的图书识别率不高。一般机器识别后还要人工校对。虽然也有网友精排精校，但总的看出错率较大
出版社提供的电子书，多用来作样刊——只提供部分章节，大多质量不错，想看全本需要交钱。
网友收集的精华本，一般是重要写手作品汇总，质量还好。

综合的看，两大类电子书互有利弊。扫描版的优点是原汁原味，没有转换过程，不会出错。但有些扫描质量较差，要自己处理后才能看清。而且必须在屏幕面积较大的设备上才能看。文本的好处是小巧灵活，字迹清晰，可以在手机等较小的设备上看，缺点是可能存在的文字错误。建议严肃的、学术的、图表较多书籍还有古籍使用扫描格式。流行小说之类的使用文本。

1.2 视频和音频
1.2.1 视频
百家讲堂、世纪大讲堂之类面向公众的，最为通俗，对读者要求也最低，近些年有鱼龙混杂的趋势；
大学校内通识性讲座，尽管面向外专业，但观众的素质高一些且相对比较“内部”，可说的东西多一些；
精品课程的示例，取自真实课堂，保持原汁原味，选的章节也是老师最擅长的地方、讲的最出彩的地方。缺点是一般只有几节课，无法形成系统，当高档一些的讲座也挺好，可以开阔视野
网络学院流出视频，很多大学开了收钱的网络学院，类似以前的电视大学、函授大学。好处是成系统，缺点是并非顶级教授开课，师资力量差一些。当然这也不绝对，而且仔细听一般还是能听出门道的
整套教学视频，规模巨大，一门课要跟一个或多个学期，记录下这门课全部课堂实况，一般放在学校内网上，外面流出来的较少，且经常要收费，能碰上是最好不过的了
网友自制，多是屏录的it教程

1.2.2 音频
实况录音，比如新东方单词之类的，较常见
网友播讲，经常能减到朗读的通俗小说
计算机自动朗读，有软件可以做到，但强烈建议大家不要用，机器朗读和人朗读的感觉完全不同

2、资源的获取
2.1 verycd
经过四五年的积累，vc上已经有大量的资源，建议大家首选这个网站；vc还有个好处是资源介绍详细。如果想快速查找，可以在电驴软件的搜索功能里直接搜索，挑资源数最多的下载。这个办法针对CD、电影尤其管用。只是要根据所在地法律情况使用，以免不必要的麻烦
2.2 Google
把它放第二位主要是搜索到的信息太多，要花很多时间进一步检索。
用“书名+rapidshare”的方式google一下即可得大部分西文书籍的下载链接，下载方式有免费和付费两种，免费够用了
如果找不到，可以用“书名+pdf（或者djvu或者djv）”在Google中搜索。
因为版权问题，Google图书上的电子书常常缺页，其他网站大多没问题
2.3 论坛
很多论坛以附件下载为吸引网友的手段，比如必须上传多少才能下载、必须回复多少才行。有些专业书在这种网站上好找一些。这方面了解的不多，下面几个推荐一下，在专业的只能自己找了
http://www.pkucn.com/index.php
http://www.pinggu.org/
http://www.zwkao.com/
2.4 新浪爱问
个人感觉新浪iask还是挺厚道的，上传下载速度都快，不少冷门的东西也能查到；
2.5 豆丁
这类在线看书的网站还有一些，豆丁算是大的，但限制较多，只能做个替补
2.6 土豆和优酷
把它们放在最后并不是说它们不重要，实际上大多数视频资源都可以在这类网站上找到，包括前面说那些收费的、内部的，总会有好事者上传。这类网站的缺点是分类不专业，经常要搜索半天才能找到需要的东西，所以最好的办法是在其他地方找到想看什么，再到这类网站搜索一下，看看有没有人上传

关于精品课程，这里唠叨几句。不管精品课程的效果如何，至少给象牙塔外面的人一个很好的旁听机会，至于能听到多少只能看自己的造化了
以北大中文系中国古代文学专业的精品课程为例，首页包括：课程简介、教师介绍、教学大纲、授课教案、指定教材、参考文献、考核方法、作业习题、教学录像
所有精品课程网页基本上都是这个框架，其中教学大纲、教案时可以下载的，有些作业习题考试也行。教学录像虽然只有几集，但大都是精华，而且可以下载，值得看看。
网上虽然有精品课程汇总，比如如下网站，但这个汇总比较杂乱。建议大家先看下大学专业排名，再去这个学校的网站--组织机构--对应系所首页，比如北大中文系，一般系所首页就有精品课程的链接。
