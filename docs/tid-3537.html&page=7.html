实际上，我质疑的正是你的论据和结论。我认为人们现有的对原始社会的认识，至少不是全部的真相。

       你不能这么考虑这个问题，以道德考察原始社会时要考虑普遍现象与特例。简单来说就是现在流传下来的关于原始社会美好的说法也许只是“道德闪光点”。这不是道德，也不是原始社会的普遍表现。我们可以与以下的情况做一个类比。我们在网上闲逛的时候经常能看到很多图片是宠物有非常人性化的动作和表现。那么我们能凭此说宠物是具有人性的么？不能吧。我们只能说宠物有一些“人性化的闪光点”。
      就目前对原始社会的认识来说，虽然不是全部的真相，但没有现在意义上的道德是可以确认的。接好上段来说就是。原始社会会有某些行为符合道德，但究其根本是因为行为符合道德而被后人记录下来，但是原始社会做出行为本身不是因为道德。
