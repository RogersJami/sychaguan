之所以转载，是把它作为一个案例，在当下发生的事情，我们应该从各个角度来观察、来思考，把各种不同的见解结合起来，慢慢形成一个尽量全面的认识。因而不会因为盲目，掉入陷阱。

老子说：

圣人欲不欲，而不贵难得之货；
学不学，而复众人之所过。
能辅万物之自然，而弗敢为。

孔子说：

众，恶之，必察焉；
众，好之，必察焉；

苏轼说：横看成岭侧成峰，远近高低各不同，不识庐山真面目，只缘身在此山中。

　新自由主义经济是中国雾霾的始作俑者



柴静女士，你好!


　　我是中石油一名从事环保工作的员工。
　　最近认真的看了一遍“穹顶之下”，虽然我很低调，但是还是想和你谈谈一些石油的事!毕竟我在这个现在似乎已经成了众矢之的的企业从事环保工作，我们应该发出自己的声音。
　　赞扬的话已太多，不必赘述;质疑的声音也不少。应该说不管对这个片子本身的评价如何，你是做了件好事，只要能推动人们环保意识的提高，哪怕能劝阻一些人少扔点儿烟头也是有意义的。对于一件事情本身的评价，就事论事就好，也不必有诛心之论。
　　从环保宣传的角度来说，这部片子是成功的，女性、母亲加文艺青年和知名记者的多重视角，成功引起了很多人的共鸣。


　　但是客观的说，从纪录片本身的水平而言，这部片子并没有给出超出我认识水平的内容。关于雾霾，或者说环境问题的宏观原因也就没有太多可争论的，但是片子的一些刻意的导向我并不认可，先说其中一点。


　　纪录片举伦敦的例子，认为天然气替代煤炭是伦敦雾霾治理成功的关键，这个当然很容易理解，然而话锋一转，以美国有700多家油气开采公司作为对比，将中国天然气缺乏归咎于中石油的垄断。这个结论出现在严肃的纪录片里，未免太过肤浅。美国的数据我一定花时间求证一下，我们一般的了解美国也就是埃克森和雪弗龙、康菲几家大石油公司，不知道其他的700多家产量占比有多大。


　　并不是说我在中石油工作，就一定为中石油说话。纪录片将矛头引向中石油、中石化，固然迎合了一般民众的心理，但就事论事的说，如果真的按照纪录片的潜台词，彻底放开油气生产行业，那绝对是中国环境不可承受之重。证据根本不用再找，纪录片大力批判的煤炭和钢铁行业就是典型的例子。山西的煤炭行业过去全面放开，民资引入可谓彻底，结果呢?矿难几乎成了中国政府的耻辱，在安全掩盖下的环境问题更加触目惊心。最近几年，煤炭行业全面整顿，国企收购小煤矿，才有所好转。有哪个煤炭企业的环保做的比神华这样的央企好?钢铁又是一例，纪录片中的环保官员也承认，河北的钢铁产业已经大到了关闭不了的地步，那些关闭不了又污染严重的钢企都是民营企业，你也可以去做个调查，对比一下宝钢这样的央企做的如何?


　　大气污染只不过因为老百姓感受比较直观，所以共鸣最强，但其实相对来说还是容易治理的，比如apec蓝，关停华北地区的企业，一个月就见成效。而采油采气的环保问题搞不好就是地下水、土壤的全面污染，且容易掩盖，但是治理起来，可就不是关停企业一两个月能解决得了。美国为治理土壤污染，建立1000多亿美元的超级基金，现在也承认失败了。并不是说中石油、中石化的环保就做的有多好，你可以指责国企有腐败、效率低下的问题，然而央企还决不至于为了钱丧心病狂，但是那些私营的油老板、气老板真能，煤老板的示范就在不远，陕北地区就有不少私营老板在开采石油，有心的人可以去调研一下。在中国现行体制下，我也不认为靠加强监管就可以了。


　　总体上我自认为是个自由主义者，但是市场不是万能的，也不能迷信。全世界的石油行业都是垄断的，主要的石油公司不超过10家，这是行业性质决定的。煤炭行业教训不远，如果决策者真的在这种所谓民意的裹挟之下，全面放开油气开采，任谁都能在中国找块地打几个眼，看看有没有石油天然气，那才真的是环境的灾难。

　　河图与洛书，是两个图，这两个图，是一个整体。
　　只是为了把事情说清楚，才只能用分解图的方式表达的。
　　比如说，河图，1与6，放在一起的，2与7放在一起的，3与8,4与9,5与10，是放在一起的。这要表达的就是一个以5为核心的同属性关系。
　　洛书中，1与9，形成了对称，2与8形成了对称，...这要表达的是一个以10为核心的中庸关系。
　　我这里只是简单的说，复杂的东西请去看我之前写的章节。里面还包括了阴阳属性、转化属性等等很多内容。

　　音乐的东西我不太懂，我能够想到的是，
　　你玩吉他，出现的音、声的变化，其实是由琴弦振动的差异而导致的，这就是老子说的“音声之相和也”。没有差异，是体现不了声音的。

　　其次，声音的变化，又由于速度的差异而带来节奏的变化，快慢也是一种差异，再加入一个强弱的差异，由此构成了一个极其丰富的差异化的境地，由于不同的音、不同的节奏、不同的强弱，进行组合，由此构成了一个连续的音乐过程。

　　在这个变化的声音过程中，这个音乐过程之中，为什么有的音乐能够打动人心，有的不能呢？
　　老子讲的很清楚“万物之中都灌注的是道”。就是说都有一个共同的东西在你我之中存在，当你的音乐引起了其它个体的共鸣，或者是内心的感受共鸣的时候，就打动了。

　　那么，音乐所演绎的，其实就是人性本身，甚至就是世界本身，宇宙本身，只不过把它用声音的形式表现出来而已。

　　音乐，一定要有螺旋式的表达方式存在，它可以是一个音乐主题段落在不同音阶、不同档次上面反复的演绎，
　　对于一个作曲者，灵光一闪之间，找到了那个打动自己灵魂的旋律调调，把它作为这个主题，然后开始用螺旋式的表达方式把它跟流畅、更自然、更完美的旋转起来。

　　至于用什么乐器来表达，就要看这些乐器所带来的音调是否与这个旋律的意境符合，由此来做出选择。

　　现在我们再回到河图、回到洛书：
　　当世界越来越高级的时候，它内部的能量结构是越来越复杂的，人类也一定是要向越来越复杂的方向衍化的。人类的精神文明也是与之同步向复杂衍化的。

　　但是，这些复杂的衍化，究其本质，它是由最基本的结构组合而来的。
　　而这个做基本的结构，就是河图洛书结构、就是先天八卦图的结构。
　　宇宙的一切都是这个结构构成的。

　　现代数学里面有一个叫做分形的理论，这个就是讲组合的理论，就是说，最细微的基本结构，组合、组合再组合，构成从微观到宏观的一切。

　　而我要说的，就是，这个最基本的结构，就是河洛图+先天八卦图。

　　行文至此，我建议你去读我写的《道德经》解说，因为，音乐与人类、乃至于宇宙的一切，都是共通的。
　　事物的运动变化的规律，也是音乐的运动变化规律。

啥叫“古之善为道者，微眇玄达，深不可志”？

把那个过程，理得丝丝入扣，巧夺天工，事情的发展情不自禁的按照那个设计的环节，一个一个的发生，每一个节奏的节点、将要出现的触发条件、由此而将要发生的变化轨迹都在预料之中。

而这些一连串的环节，却又那么必然的“合乎自然”。

这种推演，确实不是一般人的脑袋能够承受的，也确实无法用语言来描述的。

作为我们现代最接近这个境界的，推荐大家去读一本书：索罗斯《金融炼金术》。
当然索罗斯还没有达到老子描述的那个水平，不过，他的思维方式，是朝那个方向整的，所以，就有很大的成绩，也因为这个原因，很多人读不懂他的书。

这个世界，总是存在着很多很多的必然联系的链接片段，或者说，存在着因果关系的片段，
这些因果关系，并不是一直连贯的，而是一个一个组件、一个一个片段，
有的片段比较长、有的比较短，

比如：

如果A出现，那么B出现；
这是一个最短的链接片段，

如果A出现，那么B出现，B出现以后有两个可能：C情况或者D情况，如是是C情况，则必然E出现，如果是D情况则必然是F；
这个因果链接的片段就比较长。

世界总是动态链接的，总是由一个个简短的因果链接，组装起来而构成连续不断的运动过程的。

简单的因果链接，叫做“道的基因”，由“道的基因”这种积木，组装起来，就构成为运动的轨迹，这就是道。
所以说：“道，可道也，非恒道也”。

当出现某种多项选择的时候，当出现不足、合适、过度的时候，就至少有三个发展的可能。所以叫做“非恒道也”。

第二步棋轮到英国银行家大展拳脚了。由于1914年第一次世界大战爆发后德国潜艇在大西洋的频繁袭击，英国运送黄金的船只无法出港，导致了英格兰银行不得不宣布暂时停止黄金兑换，英镑的金本位已名存实亡。
    1924年，后来名震英伦的丘吉尔就任英国财政大臣。对金融事务完全没有感觉的丘吉尔在伦敦银行家的鼓噪之下准备恢复金本位，理由是必须捍卫英镑在世界金融领域绝对的权威地位。1925年5月13日，英国通过了《金本位法案》（Gold Standard Act）。当时英国的国力经过战争的剧烈消耗早已严重受损，其经济实力已远逊于新兴的美国，甚至在欧洲也已不是一家独大的局面，强行恢复金本位势必导致英镑坚挺，严重打击本已日益丧失竞争力的英国出口贸易，同时还会造成国内物价下降，工资缩水，失业率大幅上升等经济后果。
    这时，一代宗师凯恩斯横空出世了。凯恩斯在1919年巴黎和会上曾担任英国财政部的代表，他坚决反对对德国的严厉条款，并不惜以辞职来抗议。他力主废除金本位，与伦敦的银行家势力形成水火不容的态势。在英国政府调查金本位的可行性的麦克米兰委员会上，凯恩斯慷慨激昂，痛陈金本位的弊端，在他看来，黄金只是“野蛮的遗迹”，是对经济发展的制约。英格兰银行的诺曼也不示弱，坚称金本位对于诚实的银行家是不可或缺的，无论英国的负担有多重，无论多少行业严重受损，不然何以体现伦敦金融城银行家的超级信誉。英国人民被弄糊涂了。和美国的情况一样，伦敦银行家在人民当中也是名誉不佳，既然是银行家支持的，想必是不好的，而猛烈抨击银行家的观点，应该是向着人民的。
这才是戏的精彩部分。

（一九合十注：“所有敌人支持的，都是我们反对的，所有敌人反对的，都是我们支持的。”这句话也可以作为欺骗的伎俩，而在1924年之前就被国际金融家们大量使用了。）

    来历很不简单的凯恩斯扮演了为民请命的角色，而银行家们则以黄金卫道士的形象出现，这出双簧演得出神入化，舆论和民心就这样被轻松地操纵着。

温和的核聚变             一九合十       http://blog.sina.com.cn/u/1779146080



这是两个半球组合在一起而构成的一个整体：（我不会画图，请会画图的朋友帮我画一个示意图，以便之后贴上了）

在半球的顶部形成旋涡，能量吸入球心内部，再从球心的赤道释放出来。从而形成两个半球式的环流。

之所以是从半球的赤道释放出来，是因为上半球与下半球都是从顶部吸入，两种能量相向而行，在球心对撞，最终从赤道位置释放出来。
在球心区域，相向而来的能量对撞，形成大小不同的能量球，
一些对撞能量球，由于球面旋转同向，彼此压缩，从而构成密度更大的能量球结构，
一些对撞能量球，由于球面旋转反向，彼此排斥，从而密度变大，或者说，变得更为细微。
轻重不同的能量物质，总是朝着总体的能量环流方向运动，速度快的，密度轻的，就形成的是看不见的能量场，再次一些的，形成某种气态、液态、固态物质。

这么看，就发现，地球其实本身就是在不断的进行着化学反应、核反应。本身就是一个工厂。它源源不断的制造着地球上应该出现的东西。

我们的地球是如此，其它星球也是如此。
大至宇宙、银河系、太阳系、地球，小至无穷小，无法感知的能量球，也是如此。

在这种模式之下，事实上，在球心内部是存在着一个“核聚变”的作用的。由于这个核聚变，形成一个动态的能量循环的平衡，重的物质不断的聚集下来，轻的物质不断的翻动释放出去，在向外运动的过程中，发生各种复杂的化学反应，就物质的运动而言，总是存在着不断翻动的效果。

当这个球体越大，球心的能量对撞越强烈，其释放的能量循环越强烈。

所以，地球与太阳所释放出来的能量效果就不同了。而在微观的领域，也是如此类似。

我们现在的误区是，总以为，只有在轻物质上才能找到“核聚变”。
其实，万物都存在着这样的变化方式。只是它们在不同的条件下表现出不同的物质的运动变化而已。

依据这样的原理，能量的转化也是无穷尽的。

然后，我们再把这个模型想复杂一些，想象一下：
有很多不同层次的这样的东西存在着，那么就把能量场，变成了一个螺旋式的运行了。
那么就呈现出，大大小小的螺旋式运动的能量场。

现在再去看太极图，就有感觉了。

因此，我们就不难理解基因为什么是螺旋式的了。
