飞天鸭 发表于 2012-5-28 02:53 static/image/common/back.gif
关于生态，实际上人类除了面临食物的限制外，过分生育还是存在的，自从人类采取了较为静止的生活模式以后 ...

关于生态，实际上人类除了面临食物的限制外，过分生育还是存在的，自从人类采取了较为静止的生活模式以后，大自然的生育控制就对人类不起作用了。

很仔细地阅读了这个贴，你真的让我看到了，数学家的生态学素养和社会问题结合的很良好。或许你是做相关方向的原因？

关于生育成本，一个是生育需要家庭条件，比如祖父母的支持，但也需要社会条件，比如儿童的公共设施等，成本也不仅是经济，还有时间，精力，心力等等。

这个我考虑过，祖父母的介入，这是中国目前社会体制、人的基本素质、社会意识背景下一个相当必然和可行的选择。事实上中国很大部分老年人是隐性就业的，这个隐性就业就是抚幼带来的社会效益。不仅在农村，在城市也是如此。确实抚幼的社会成本构成相当复杂。

其实比较理想的社会模式，应该也不是所有女性都生且只生一个孩子，应该是一部分女性可以多生，另一部分女性不生。所有群体性动物都发展出了这种生育模式。即使对于人类，同时抚养三个孩子的时间精力成本远小于三个家庭抚养一个孩子。所以各生一个模式有很大的效率浪费。

是这样的，因此中国目前的社会强调只生一个是不太有效率和不很符合经济学的措施。不过对于一部分女性多生，另一部分不生，貌似在猴群、象群和狮群里面都不是这样？一些K选择的生物，抚幼时间比较多，幼体成长期比较长，我觉得可以近似地来分析人类的一些收益和选择。

人类的问题是社会进化没有带来相应的生物进化，还停留在部落时代的模式里。相当多的社会都发展出了宗教，很多人选择不生育，这是社会发展的必然，大工业社会应该也不例外。没有生育的可以给别人的孩子当教父教母么，比如大懒虫mm就收了许多孩子么。

应该尊重一部分人的生育权，也相对的应该尊重一部分人的不生育权。就像前面有人认为女性该守贞，而略懂生态学的人都会觉得这个想法或许有点——荒谬。不过现代社会中国的收养关系一般被定义为完全性的抚养，过去干爹干妈那种类似国外教父教母的关系，在现代中国社会中不见得发达。在传统宗法社会里面，亲族是承担了这种义务的，而现在亲族的力量也被弱化了。

传统农业社会，孩子会走路就开始干活，所以生孩子是重压之下父母的迫切需要。工业社会，儿童的成长模式还是在变动之中，问题很多，而且和父母的利益确实并不直接相关。而且父母的社会化问题如果没解决，会影响下一代的社会化问题，类似现在的啃老族和祸害父母问题。

是这样的。不过我还是比较乐观。身边有很多人，成长过程中父母给的扶植教育并不一定是完全正面的，也可能有一些甚至于负面的作用。但是在我看来，一些人他们还是尽力的摆脱了负面作用。也就是说，上一代的问题有影响，但是可能并不完全是决定性的。当然，影响也还是相当大的。

综上，我其实本身是同意你的观点的，不过我还是觉得风险，其实没有那么大，还是值得去冒险下的。生孩子对我们来说可以简化为付出收益问题，但是就类似于感情问题，其实并不是那么简单。当然，无论如何，了解风险是好的。每个父母都应该在了解风险的情况下去生育，我只是不希望看到过于夸大风险导致的后果：大面积丁克，老了想要才发现生不了了。

身边就有这么一对，现在还没有到发现自己生不了的时候。昨天去玩，男的一直抱着别人家孩子不撒手，女的就一直像你说的这样，认为生育风险太大，怕疼，怕麻烦，怕各种因素，她说，可以不要孩子养一群猫。。。我昨天还跟她说，生孩子是跟养猫不同的。我已故去的奶奶曾经感叹养孩子是最有意思的工作，因为养猫养狗养动物获得的收益都是有限的，但是生孩子和抚养孩子获得的收益可能是在感情上和物质上都无法来衡量的。她的话是：过一年，会跑了；再过一年，什么都会说了。多有意思！我大部分同意她的观点。看昨天师兄师姐们抱着孩子的感觉，我很理解当年奶奶的那句话。

　　晨报讯(记者 王歧丰) 近年越来越多的恶性家庭暴力事件表明，仅仅依靠行政处罚和自诉已经无法应对日益蔓延的家暴现象。农工党中央建议，严重家庭暴力事件应该作为公诉案件处理。

　　农工党中央建议，在修改刑事诉讼法时，应加强对妇女权利的保护。比如在处理强奸罪等性侵犯案件时，禁止采纳与被害妇女以往性经历或评价相关的证据。审理相关案件时，要严格保护被害人隐私，一般不公开审理。

　　农工党中央特别指出，要把遭受殴打、虐待等家庭暴力行为的受害妇女纳入法律援助范围。规定使用器械、反复实施等情节严重的家庭暴力现象作为公诉案件办理"
