伊朗遭遇货币战争 里亚尔一周狂贬40%

据海外媒体报道，里亚尔兑美元汇率上周一跌至1美元兑换33500里亚尔的历史低点。不安的伊朗民众纷纷买进硬通货，导致里亚尔持续承压，最新消息显示，伊朗黑市上1美元可兑换3.2万至4万里亚尔。而伊朗央行设定的官方汇率为1美元兑换12260里亚尔，不过多数伊朗人无法按照该汇率兑换到美元。
　　一些政治分析师认为，货币贬值引发的局势动荡可能为西方国家提供更多制约伊朗的筹码。伊朗资深宗教人士艾哈迈德·哈塔米5日呼吁伊朗国内停止争斗，团结一致应对西方国家发动的“经济战”。
　　有媒体报道称，在伊朗肉类已变成奢侈品，伊朗的经济困难已深入到几乎每个伊朗家庭。一位伊朗的肉铺老板说：“我的大多数顾客都只是从窗外看一眼商品，然后就离开。我看见他们去了附近的面包店，全家人都靠吃面包生活。”
　　此外，伊朗首都德黑兰部分商店已关门歇业。一些店主说，汇率不稳让他们难以为商品标价，关店是因为不知道汇率将如何变化。
　　美国约翰斯·霍普金斯大学经济学教授、凯托学会高级研究员史蒂夫·汉克说，根据对伊朗黑市汇率的分析结果，目前伊朗的物价每40天上涨一倍，这不仅耗尽普通伊朗人的存款，也耗尽了他们的购买力，尤其在城市中。
　　伊朗司法部门在当地时间10月4日逮捕了16人，指控他们参与制造了伊朗的货币危机。伊朗司法部门当日发表声明称，伊朗货币危机的制造者已经聚敛了大量非法资金，并在政府的银行系统之外走私外币，从而使外币增值，试图在伊朗制造社会动荡。
　　奥巴马政府承认，美国支持的制裁政策在一定程度上导致了伊朗的经济困境，但最终要归咎于伊朗政府。分析人士认为，伊朗的问题只能部分归因于美国及其盟友为迫使伊朗改变其核政策而施加的严厉制裁，至少具有同等重要性的是总统内贾德倡导的国内政策。这些政策导致一项受欢迎的补贴计划被逐步取消。这项补贴计划人为使许多主食保持低价。
　　去年底至今年初，美国和欧洲宣布对伊朗施加严厉制裁，矛头指向石油部门和银行业，导致里亚尔大幅贬值。今年以来，伊朗货币已累计贬值约60%
