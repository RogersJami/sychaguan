玩钉耙的悟空 发表于 2013-5-10 11:26 static/image/common/back.gif
不能信而泽被天下，就泽被周围人，思炎信而泽被我等茶友。

大道至简，呼吁茶友写帖，一点偶得，一些心想 ...
响应大师兄的号召，写一点感想。

“一切向钱看”， 先要把钱看清楚。 每张钱币上都有发行人名称和币值，还有，美元上有两个签字，欧元上有一个签字，人民币上没有签字，盖了个“行长之章”，章刻得很漂亮，不知出自谁的手。

一张花花绿绿的纸，写上名号和价值，再“签字画押”，就成了一张有价值的字据，和老百姓写的借条差不多，只不过这张借条好使，好多人都抢着要，因为发行人有诚信，有信誉，保证让人拿着借条能换到东西。钱就是信誉。

“一切向钱看”，本质是“一切向信誉看”。“有钱走遍天下”和忙总写的“诚信无敌”，其实是差不多的。

关键是“谁的诚信”，“谁的信誉”。

美元是美国的信誉，人民币是中国的信誉。人民币绑在美元上，美国的信誉也成了中国的信誉。人民币发行量越来越大，中国的信誉水涨船高，中国的信誉有成为美国信誉的苗头，譬如，现在美国市场涨涨跌跌，常常要在中国市场找说法。

钱就是信誉，赚钱就是赚信誉。用信誉去赚信誉，是最妥帖的办法。教孩子讲信誉，就是教孩子怎样赚钱。

信誉是要一点一滴积累起来的。用信誉去赚信誉，只能一点一点来。 现在有人嫌这样太慢，太累，就去动脑筋“使花活儿”，用欺诈去赚信誉。所以有信誉的人（也就是有钱人）要注意保护自己，保护自己的信誉，不要上当受骗，要不有钱人会很快变成没钱人的，自己的信誉被别人骗走了，自已就没钱了。
