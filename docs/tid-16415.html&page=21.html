我说几句语气重点的话。你的回复让我觉得完全没碰到点子上，连个汗毛都没碰到。先说49年建国之后的思想状 ...

      一个理论要想被人们所接受，必须是它对人们有用。春秋战国百家争鸣，最后只有儒家一家独大，因为它对上确立了君臣尊卑的秩序，对下确立了父子孝悌的秩序，这是被农业性质的帝国所需要的，所以能独霸中国人的思想两千年；然而一旦中国开始尝试进入工业社会，它以前的那些优点就全变成了缺点，成了“吃人”的礼教，因为这时候中国已经不再需要它了，它却依然试图阻拦中国的脚步。
      再说马克思主义。在西方，它是工人阶级对抗资产阶级的思想武器，也是人们观察西方社会各种病根的镜子；在东方和世界其他地方，它是被压迫民族反抗殖民压迫的思想武器。所以马克思主义能在19、20世纪在全世界大行其道，不是没有道理的。
      所以，如果你是西方的资本家们，你一定会选择吹捧西方经济学，而打压马恩系列的政治经济学；如果你是高官显贵，你一定会遵奉主张君臣父子的儒家思想，反对、至少是淡化主张人民民主的毛泽东思想。屁股决定脑袋，这是一个很简单的道理。
      所以我说，马克思主义毛泽东思想在今日的中国受到打压，不是人们不欢迎，也不是他们没用，而是政治经济基础发生了变化，屁股坐在那一边的人获得了优势；而近期他们又开始有重新回潮的趋势，原因也很简单——世界性的危机临近了。
