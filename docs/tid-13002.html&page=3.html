18位统一社会信用代码方案揭晓 2017年前覆盖全国

http://www.nbd.com.cn/articles/2015-06-17/923917.html



国家行政学院教授汪玉凯表示，统一代码将会链接各个独立的信用系统，形成一个从政府到个人的完整的社会信用体系，将对未来社会产生巨大的影响。

　每经记者  周程程

统一社会信用代码的总体方案终于揭晓。

6月17日，国务院发布了《关于批转发展改革委等部门法人和其他组织统一社会信用代码制度建设总体方案的通知》（以下简称《通知》），要求建立覆盖全面、稳定且唯一的以组织机构代码为基础的法人和其他组织统一社会信用代码（以下简称统一代码）制度。

《通知》要求，统一代码设计为18位，由登记管理部门代码、机构类别代码、登记管理机关行政区划码、主体标识码（组织机构代码）、校验码五个部分组成。

这相当于让法人和其他组织拥有了一个全国统一的"身份证号"。国家行政学院教授汪玉凯告诉《每日经济新闻》记者，统一代码将会链接各个独立的信用系统，形成一个从政府到个人的完整的社会信用体系，将对未来社会产生巨大的影响。

18位新编码破除"万码奔腾"弊端

《通知》表示，当前我国机构代码不统一，缺乏有效协调管理和信息共享工作机制，大多数代码仅应用于各部门内部管理，一些部门信息数据相互割裂封闭，存在信息孤岛问题。

例如各类机构代码长度、含义、作用不同，有的部门如工商、民政、机构编制部门等，在法人和其他组织成立时赋码；有的部门如人民银行、税务部门等，在行使管理职能过程中再次赋码。

并且，法人和其他组织在设立和办理相关业务时，需到多个部门申请代码，有的还收取费用。多个代码共存现象较为普遍，影响了同一主体信息比对，增加了社会负担，降低了行政效率。

"'万码奔腾'的背后，抬高了企业的各种显性和隐形成本，也增加了公众信息检索的困扰。"宿迁市副市长冯岩曾对《每日经济新闻》记者说，统一规则后实现自动赋码、一次赋码和全程运用，有助于破除权力垄断和部门利益，让利于创业创新者。

《通知》表示，从唯一、统一、共享、便民和低成本转换等角度综合考虑，统一代码设计为18位，由登记管理部门代码、机构类别代码、登记管理机关行政区划码、主体标识码（组织机构代码）、校验码五个部分组成。

其中，第一部分（第1位）为登记管理部门代码，第二部分（第2位）为机构类别代码，第三部分（第3-8位）为登记管理机关行政区划码；第四部分（第9-17位）为主体标识码（组织机构代码）；第五部分（第18位）为校验码。

除第三部分（第3-8位）的登记管理机关行政区划码使用阿拉伯数字表示外，其他部分均按照规定使用阿拉伯数字或英文字母表示。

《通知》认为，为便于行业管理和社会识别，统一代码的第一、二、三部分体现了登记管理部门、机构类别和登记管理机关行政区划，兼容了当前各登记管理部门行之有效的有含义代码功能。为保证唯一性和稳定性，第四部分设计为主体标识码（组织机构代码），充分体现了以组织机构代码为基础建立法人和其他组织统一社会信用代码制度的要求。为防止出现错误，第五部分设计为校验码。

这里恐怕我们的言语背景不同。这是网络交流的特点。

大数据作为一种存在，是有多种属性的。在讲到资本化的时候，我们讨论的是它的社会属性。在你讲到爆炸式成长的时候，我们是在讨论它的自然属性。

那么，什么是它们之间的辩证关系呢？自然属性不等于社会属性。数据爆炸式增长不可能成为一种生产关系。它必须资本化后，才具有社会属性。增长的多少和数据质量成为估价因子，进而作为工具因素影响生产关系。而这个资本化，则就涉及到谁给它资本化了。这是另外一层逻辑关系了。

任何最研究最基本的方法只有两个，历史与哲学。离开了，就容易自说自话。也容易缺乏现实意义。讲历史，都是以史喻今的。类比，也就是在讲边界。活下来的新事物数量远远比出现的新事物少得多。以网络为例，现存一家的背后，有近千计的倒闭和中止运作。所以，历史，可告诉我们如何避险，哲学可以告诉我们如何活下去。

最后一个问题。只能定性讨论。没有人敢把定量数据放到公开场合。而没有定量的讨论，是很难满足你的要求的。比如说，电力部门说有近7500万套房子是用电“死户”。我们只能根据这个数量的信息进行讨论，比如，是现房，代表着凝固资金可能有多少。谁有胆量进一步提供这些房屋的分布？具体的凝固资金的结构？……

所谓空对空的玄谈。是要看对方的知识结构如何来定的。如果没有对电力部门统计的知识储备，对国民经济宏观运行及经济微观运作的知识储备，是很难从7500万套中判断出现房，分析出凝固资金的影响。让这个信息上升为情报。

我最早接触陈浩的理念是一个地方电视台播放他的讲座。他说学习心理学的。这样，我找了他的《筹码分布》一书来看。的确那本书是把人的心理及行为作为分析的起点。有别于当时许多分析证券的著作。

我不知道陈浩的底细。刚才在网上看了陈浩名下的一集天狼教学片，从图像和声音来看，其中的陈浩应该与我看电视讲座中的是同一个人。只是形象上老了许多。我在百度查了一下陈浩的简历，上面说他是清华物理系。与我在电视中得到的信息有一些差别。不过，心理学是属于理工科，学文科的人，还真的不容易学好。所以，他从物理系转为从事心理学研究应该没有什么大困难。我在他的微博上看到他的学术研究方向为“金融行为学”。现在我们所说的心理学是一个通用的说法，一般比较严格的叫法，以大专院校为例，叫“心理与行为科学系”，或者其他的有少许变化的叫法。所以，他研究的方向应该是应用心理学的一个分支：经济行为学，也有叫投资行为学、金融行为学。这样来看，他的学历出身应该没有影响他研究。至于他在电视上的讲他学习心理学的，可能有多种原因吧。

叫“金融行为学”，而不叫金融心理学，主要是看问题的视角，这类主张的人，一般认为人的行为与心理互为表里，但是，人的心理如同黑箱，不容易了解。而把这个黑箱输出输入——行为，作为研究重点，研究效果更好一些。

向葡萄推荐陈浩，主要是葡萄讲到他的操作系统与另外一个参照的操作系统相比，缺少一个关于金融行为心理分析方面的子系统。我记得当时陈浩不但出书，而且还把他的理念溶入了一个叫指南针的证券软件。那么陈浩的理念，应该比较容易被葡萄了解，也容易在软件工程上实现。便向他推荐了陈浩。
