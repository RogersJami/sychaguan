在商场门口等人，旁边有卖烧饼的小推车。过来一男的，牵一十岁左右的男孩，希望我给小男孩买个烧饼。我就让卖烧饼的准备一套（烧饼切开，夹一片卤豆腐），又让加了个鸡蛋。小孩拿着就开吃。我一想，小孩都没吃的，估计大人也还没吃，于是让卖烧饼的照样来一套和一个鸡蛋给大人，总共好像没几块钱。那男的一拿到烧饼和鸡蛋，就和小孩低头走了。卖烧饼的说，父子俩天天来这里要。

排队买火车票。刚在队尾排住，过来一位穿着干净利落的女的，大概二十五、六，对我说：“大哥，我买票差三块钱。”我拿了三块钱给她。她接过去，等在旁边不走，我茫然地看着她，她恍然大悟，说：“是十三块。”我才意识到此非善类。她也不拖泥带水，一看没希望了，一脸胜利的嘲笑，转身飘然而去。
