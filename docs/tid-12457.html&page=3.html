      今天早上还在跟老婆孩子争论对日思想的问题，起因是 ...

澄清几个问题：

一，怎样衡量发动机

发动机有两个特性曲线，以发动机转速横坐标，以功率/扭矩为纵坐标：一条是功率曲线（平常我们说的马力），简单说就是速度高不高，功率越大，最高车速越高，另一条是扭矩曲线，简单说就是提速快不快，也就是爆发力。

最大功率和最大扭矩没有意义，日常使用大多数时间发动机的工作转速对应的平均区间值才有意义：中低转速大扭矩，说明起步性能好，适合城市（日韩车的特性）；中高转速大扭矩，说明高车速超车性能好，适合高速路/飙车（德美系车的特性）。

发动机没有优劣之分，只有特性一说，好比一个人，这个人适合跑100米、200米，那个人适合跑1万米、马拉松，不可能做到全能冠军，就看你需要什么。

二，车买得好不好

总销量没有意义，地区销量才有意义。厂商对不同国家的市场，采用不同的配置（也就是版本），以适应不同的客户需求，地区销售量大，代表这一地区客户对这款车的这个版本认可程度高，比如丰田的卡罗拉，北美版销量大，但中国版表现不好，因为北美版比中国版好太多。

买车，一看怎么用，城市工况多还是高速路快速路工况多，二看同样工况下的市场口碑，三看生产时间，时间长了，橡胶件可能会老化，四看工艺，如果买到问题百出的车，这只能说运气差，个案不可避免，奔驰也有人砸。{:4_268:}

能不能给八一八呢

       首先必须明确的是脚盆目前发展的心神是一架轻型的隐身技术验证机，离开实用化那是相当的远，那么脚盆发展这一款飞机的目的是什么？个人以为必须先从脚盆自主研发的战斗机谈起。到目前为止，脚盆开发过F-1,F-2两款战斗机，F-1脱胎于T-2教练机，而T-2教练机的原型则来源于英法联合研制的美洲虎战斗机（ 一种轻型的以对地攻击为主要任务的战斗机）。而现在装备的F-2简单一句话，F-16的日本版，同样以对地攻击为主，不过可以搭载两发脚盆国产的反舰导弹。在日本空军中，这两款战斗机并不承担多少空战任务，在F-1的时代，空战有F-4“鬼怪”式，而到了F-2，则是F-15“鹰”式。在脚盆装备这两款战斗机的时候，在自主研发战斗机的领域，与我兔相比，他还是有点优势的，但是随着我兔的歼-10大量装备部队，我兔在战斗机技术方面不只是追上了脚盆，甚至于是全面的超越，特别是歼-20的试飞，充分显示我兔在战斗机设计与制造领域的强大实力，同样有着“大国梦”的鬼子怎能不看在眼里，急在心里。所以这款所谓的“四代机”就此粉墨登场。
       仅就外形而言，心神可以说成是F-22的脚盆缩小版，但是是不是等比例缩小就具备了隐身能力了呢？答案显然是否定的，航空理论专家一致认为，空重小于10吨的战斗机几乎不可能实现“四代机”标准的隐身，而我们也看到无论是美帝的F-22，还是我兔的歼-20，还有大毛家的T-50，无一例外都是重型机，虽然F-35披着中型机的皮，却有着重型机的体重，因而有肥电一说（F-35的绰号是“闪电”），心神的空重按照航空专家的推算在8吨左右。接着是心神的发动机，两台，每台的推力是五吨，有航空专家就指出，这种XF5发动机其实还是从美洲虎战斗机上的阿杜尔发动机不断改进而来，可以说已经无潜力可挖了。那么，心神是不是一堆垃圾呢？答案还是否定的，由于鬼子在复合材料以及电子技术方面还是有着相当不错的水平，所以可以比较肯定的是在飞机的制造过程中大量使用了复合材料，这个方面是鬼子领先于世界的地方。另外，由于鬼子的电子技术不错，可能也会在验证机上去试验一下“隐身蒙皮”技术——就是利用电子技术手段来实现隐身，代替目前主要靠外型和涂料来隐身的传统隐身技术，当然，这个技术也是现在军事航空领域最前沿的技术之一，各国都在研究，鬼子肯定也是要试一把的，其它的诸如推力矢量技术神马滴，鬼子木有玩过，他肯定还是想玩一下的。
         除了试验新技术之外，鬼子的目的还是想得到一款比较实用化的作战飞机，不一定用来直接在第一线作战，但是可以做教练机，替换已经老旧的T-2用于培养四代机的飞行员，这个从一般的原型机都是单座，而鬼子的原型机是双座可以初见端倪。当然，如果换上大推力的发动机，比如美帝的F414，无论在飞行性能上，还是武器载荷上都将有很大的提升，搭配适当的导弹也是一种有一定水平的以对地攻击为主的战斗机，这个就回到我们开始讲的脚盆战斗机发展的线索上来了。至于说以后发展成熟了与歼-20来空战，那是想也不要想，这个不是一个量级的对手，鬼子毫无胜算。（文字部分先到这里，以后再上图）
