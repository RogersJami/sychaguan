看了你的点评这里来说一说我对《鹿》的看法，希望你也不吝指教。

首先这是一部完美的小说，没有任何瑕丝。

一：人物塑造饱满。
以往他老人家的作品，不论描写的是什么人物更多的是作为一个故事发展推动的点。如张无忌的优柔寡断就把故事硬生生写满三十回。所以他的人物塑造在《笑傲》和《鹿》之前都显得比较极端。

而《鹿》不同，里面的人物是故事发展中的元素而非重要的推进力。可以这么说《鹿》里面人物虽然众多，但是各有各的特色。而最难得的是不论谁都不是一字形的塑造，而是多面的、复杂的、个性的。

二：历史与现实几乎不分
金庸以前的作品如《射雕》、《天龙》虽然以宏大的历史场景讲述，但是都是以几个点为核心，没有涉及到面。而《鹿》从第一回到最后一回涉及了太多的历史史实，看得出来金庸是花了大力气考证的。

以往的人物如郭靖更多的是历史的陪衬，而《鹿》则把主人公与历史完全融合。打个比方，以前的人物是硬生生的把主人公塞到历史里去，而《鹿》里的韦小宝则是历史的缔造者。

三：故事推进自然流畅
以往金庸的小说，包括《笑傲》都是利用或者离奇的转折，或者完全不沾地气的人物发展下去。而《鹿》，虽然长达五十回，但是却是一气呵成之作。犹如白居易的《长恨歌》一般。

四：对于中国文明与历史的中庸化反思
我前面说了金庸的作品大多是大是大非大爱大恨。而《鹿》却非常平实，平实到韦小宝这个人物一点不像小说人物，仿佛确有其人；平实到小说仿佛如同一本历史教科书。

而对于中国五千年文明里的战争与吞并，他没有像以前一样以一种对立的姿态进行解读，而是以中庸的思想进行诠释。“不偏为中，不易为庸”，他看待满清对于汉人的统治不是以侵略、占有、对立的角度看，而是以兼并、融合的角度来看。的确中国的文明就是黄土文明被不断侵略、不断占领、不断对立转化为中国这个大集体越来越大，民族属性越来越复杂的过程。

所以《鹿》在思想上也有很高的建树。



你说为什么他从《鹿》后封笔。我猜测有两个原因：一，这部小说耗尽了金庸对人生的理解；二，金庸对自己的定位在许多媒体上都能看到，他首先定位自己是一名商人。他或许认为《明报》已经不需要他继续连载小说了吧。

至于电视剧，我怀疑现在引进的印 ...

美国电视剧和我们的文化差距太远，所以我们没办法学习。而日本电视剧最大的特点就是关注“人”这个主题，所以他们的电视剧可以拍的很温暖很隽永。可能是日本国土面积太小而且日本国内民众对于“家国”这样的儒家大命题没兴趣的关系，导致日本国内的电视剧大多显得有点小家子气。

日本的电视剧近些年大多对于现实题材的关注不够。反倒是白日梦温情与鸡血式立志的作品很多（我最近都没看，如果不对希望指出）。日本电视剧有许多精品，如《白色巨塔》就达到了一定高度，但是和国内优秀的现实题材电视剧比还是有明显差距。

其实国内电视剧也不是一无是处。我认为国内电视剧编剧最擅长的就是台词设计，当然了台词写的烂的人也不少。而且国内电视剧的种类更加复杂，其中也不乏精品。如《潜伏》、《雍正王朝》、《三国演义》、《蜗居》就不是日本电视剧可以比拟的。

精华就是由次品支持起来的，只有市场足够大精华才能有生存空间，才能允许创作人花大力气去投入创作。所以兄台莫悲观，国内的电视剧还是能出精品的。
