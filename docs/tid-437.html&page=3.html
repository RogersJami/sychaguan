主要是要个敲门砖，否则，谁给你实战机会，除非家族企业，老爹给机会

   不一定，一个没爹的年轻人，你要实践机会。还有一种办法。
  去认识一些私人企业老板，然后针对某一些人做调查。
  然后调查他们企业的主要问题，跟他现在的苦恼。（别说你不认识，不知道怎么调查。如果这个都不会，你也就不用考虑去做管理了。）
  然后仔细分析后，就定点找其中你认为最跟你脾气相投的，仔细跟他分析他的问题，你的解决办法。让他给你任命，给你机会。可以前期不要钱。（说是这么说，一般老板都不会这么做的，肯定多少都会给你待遇的。）
  最后，在实践中，找到解决的办法，把问题解决了。这经验就来了。
  具体怎么煽动老板，怎么说服，怎么解决。我想这才是难点，而机会却不是，不要抱怨没机会。
