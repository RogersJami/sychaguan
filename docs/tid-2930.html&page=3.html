那一剑的风情 发表于 2012-8-11 14:06:43

看了一下大家的发言, 貌似很多人还理解奥运会是什么东西.

A  奥运会的宗旨是强身健体吗?

不是, 奥运选手都是伤病缠身, 因为奥运是追求 " 更快, 更强, 更 " 没有 " 更健康"

B  奥运会和全面健身有关系吗?

没有任何关系, 因为奥运选手是要夺金牌的, 要负伤训练, 如此跟建身没有任何关系.

C 那全面健身和奥运投资矛盾吗?

没有任何矛盾, 全面健身的搞法跟奥运选拔没有任何交集, 一种是非功利的健身, 一种是极端功力的残酷竞争.

D 奥运的意义?

用来证明国家, 种族, 政治体制的优越性, 如此而已, 绝无例外. 如有异议, 参看纳粹, 美国, 苏联, 中国, 北韩的经验

E 目前中国奥运的意义?

还是为了证明中国, 中国人, 中国政治体制的优越性, 提高民族自豪感, 凝集力.
任何体育运动概莫能外, 如有异议, 可复习新概念英文第四册关于体育的短文.

F 体育能证明人种, 国家体制的优越性吗?

是也不是,

是: 体育搞得好的国家一般来说证明有闲钱投入到运动员的训练中, 比如现在的中国, 韩国; 反观日本, 前苏联, 国力衰退, 体育自然跟着垮台, 体育是一个国家的经济力量的某种指标.

不是: 种族无优劣, 国家兴衰无常, 体育运动无法证明国家体制, 种族的优越性.

G 如何评价中国的奥运经验?

像中国搞得所有的大项目, 搞的是非常惊天地泣鬼神, 短短几十年能有如此进境, 前不见古人, 后不见来着.

H 世界上任何其他国家如果有如此进境肯定非常自豪, 为何中国目前有非常多的质疑的声音?

健身可能投入不足, 另外奥运这个本来就是面子工程的事情没搞清楚, 还像瞎扯很么普世价值, 狗屁不通.

I  中国还有很多穷人, 在奥运投这么多钱值得吗?

是不值的. 不过人类一直如此.

美国都要破产, 10%的失业率, 每年的军费有下降吗?  能不能把钱分给美国的穷人呢? 没门!
