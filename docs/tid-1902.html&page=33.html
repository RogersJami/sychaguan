添个分支：糟辣椒（辣椒酸）+野葱＋蒜茸＋折耳根杆杆+酱油+少许醋+少许白糖+火刨胡辣椒，这做法不放盐的，没野葱就小葱也行。有的喜欢放点水豆豉。 糟辣椒（辣椒酸）为贵州南部独有，就是把红辣椒剁碎+白酒+姜+盐放坛子里封一个星期就有酸辣味了；还有一种做法是剁红辣椒+姜+甜酒（糯米做的，有的地方叫水酒），不放盐，这样做法辣味很少，基本就是酸味，黔南的酸汤鱼就用这样的酸做的{:4_264:}
