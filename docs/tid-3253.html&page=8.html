火控雷达和有没有装弹是有区别的

国防部出面否认了，以兔共的作风，外交上不会睁眼说瞎话让人抽后脚的，而倭寇却数百年来都是毫无公信力的主。

这是从其他军网上看到的分析 可以参考一下︰

航海雷达就是导航雷达，舰艇一动就要开启的（特殊的定位通讯导航雷达除外），而空警雷达是不会开机的，一般情况是在高海况地区开启，平时开启的是海空搜索警戒雷达，而且随着技术的进步，专门的对空或对海雷达都集中在一部雷达上，不再分别使用。
至于普通的机扫或机频扫雷达舰艇上舰舰导弹使用的是两部雷达，一部是海空搜索雷达，一部就是这次照射倭奴的火控雷达，前者搜索大范围目标，用来掌握全海、空域情况，但由于波段和功率的关系，只能获得目标的不精确信息，当目标认定清晰，目标信息确认后，需要对目标采取下一步行动的时候，海空雷达将目标的大致方位高度等信息转入火控雷达跟踪，火控雷达开机，对准目标方位，用窄波段，短脉冲，依照警戒搜索雷达提供的目标信息开始搜索，当目标截获后，进一步加大功率，波束锐化，持续照射，同时将目标精确信息上传至导弹的制导计算机(这才是真正意义上的锁定目标），这个时候，装有雷达接收告警器的对方目标上的告警器就会被激活，发出蜂鸣和红灯警告，而倭奴这次就是享受了这种待遇，当这种情况一出现就意味着只需一个按钮，导弹就将发射开始攻击目标。
  至于相控阵体制的雷达就更简单了，由于相控阵体制的优势，在保持海空搜索的同时，只需将部分发射单元电子相阵对准目标，加大功率，锐化，就可以得到目标的精确信息，直接锁定，从而直接给导弹提供目标的精确信息，继而开展导弹攻击，而且这种对目标强化的跟踪，不改变频段，不再使用单独的特定雷达，故敌方的雷达告警器往往不被激活，被攻击锁定也不知道，这就是MD吹嘘的F22使用的是隐身雷达的原因之一。
  但是，由于保密、雷达寿命、经济性上的考虑，相控阵雷达一般在高海情的时候才开机，平时低海情和巡航的时候都使用普通机扫海空警戒雷达，这也是咱们052C后面还在使用八木天线雷达的原因，当然这个问题可以由双波段相控阵体制雷达来解决。
