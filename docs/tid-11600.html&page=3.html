loy外公很厉害，是救图章的火？还是消防的救火？

欢迎写出来，会是很不错的故事

当然是图章的火啦，我现在都纳闷为什么文革时期会需要那么多的印章，而且不止一次。最可气的是还象征性的发票子，那个年代谁敢揣着，发点吃喝布料多实在。

年代太久远了，我也只知道个大概。据说他的印章技术是用伞骨练出来的，到五十多岁的时候刻章可以不用任何固定工具，左手拿章右手拿刀（不用固定工具容易把手划伤）。我妈算是这个行当里的老师傅了，吃这碗饭吃了三十年，但是据她说还是达不到外公的高度。外公刻的印章家里人都看不懂，不知道好在哪里，但是懂行的人都愿意高价收购（技法方面还是懂的，可能差点心法）。文革期间别家都是没吃没喝的，只有外公家吃肉喝酒，随时都有人来求章。外公不但刻章厉害，书法、绘画、四书五经、木匠、诗词都很精通。在没去世前，还与文友定期作诗写对子，他一个文友送他的两幅画，现在市面价每幅可以卖好几万。不过可惜的是他的艺术才华没有传给子女，包括我。我母亲和几个舅舅虽然都是搞设计、刻章一类工作的，但是他们的作品连我都觉得匠气太重，他们最多也就是一流的匠人罢了。
