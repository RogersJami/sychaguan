哈佛大学的亚沙·蒙克和墨尔本大学的罗伯托·斯蒂芬·福阿最近发表在美国《民主杂志》上的一篇论文说，四分之一的美国“千禧一代”认为民主制度是治理国家“糟糕”或“非常糟糕”的方式。在这些大多出生于20世纪80年代的美国年轻人当中，26%的人说由人民通过自由选举来选择领导人这一点“并不重要”。
与年纪大一些的美国人相比，“千禧一代”也更带有威权主义倾向。他们对于在政府治国不力的情况下由军方接管政权这种情况的接受程度比老年人及婴儿潮一代高出一倍左右（比例分别为43%和19%）。

葡萄 发表于 2016-11-1 16:51:03

福阿和蒙克分析的大部分数据是5年前的，我猜想现在的情况只会更糟。不只是因为像特朗普这种招人讨厌的家伙的出现，也不仅是因为像英国议会爆出的“报销门”丑闻这样的事件，而是因为总体上的失调。比如，民主国家对基础设施的投入不足，可是与此同时，我们却看到像迪拜和新加坡这样的非民主地区却在建设质量远在我们之上的机场和其他设施。
这让我们看到了《民主杂志》上这篇论文中另一个极为惊人的发现。最反对民主的是富人和年轻人。1995年时，在出生于1970年之后的美国富人当中，只有6%的人认为军队在民主制度失败时接管政权是件好事。如今，在同样的人群中，这一比例达到了35%。喜欢强硬领导人胜过选举制度的人（占32%）和认为专家比政府更适合作出重大决策的人（占49%）的比例不断上升都体现了这种反民主趋势。虽然美国的这些数字要比欧洲更引人注目，不过在大西洋对岸的欧洲，反民主的情绪也呈上升势头。
