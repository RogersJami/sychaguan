快乐是什么 发表于 2012-8-15 18:23:03

本帖最后由 快乐是什么 于 2012-8-15 18:25 编辑

1。如果谈历史，多看正史和研究材料，并不难懂的
2。北宋武力并不低，特别是初期的对手缺乏骑兵，北宋几乎战无不胜，燕云十六州也差一点恢复，直到高粱河大败
3。后期北宋失去了马场，但仅靠步兵基本不可能对骑兵进行歼灭战，胜不能歼，败不能逃，对辽、西夏、金都只能防守为主
4。冷兵器时代，步兵守城有优势，关键看将领决心，两宋交接时几乎所有城都守不住是很让人吃惊的，汴梁投降更是难以理解，只能说皇帝和大臣弱智
5。两宋是典型的国穷民富，还花钱买和谐，否则就不会有王安石的改革，也不会有南宋的和议，因为财政养不起兵了。
6。北宋正规军空有规模，没有训练，国家最精锐的禁军充斥着地痞流氓罪犯，成了废物点心，南宋时能打的部队也有限
7。反金义军规模很大，因为民间财富不少，但民兵组织训练水平不足，再加上政府不支持，没成大器
8。宋朝皇帝未必英明，不过对知识分子政策好，所以留下的文字有利
9。杯酒释兵权的目的是为了消灭军阀，也达成了目标，北宋没有军人干政的威胁，但后果是没有出色将领。南宋杀岳飞、废韩世忠都是继续反军阀
10。两宋相对和平时间长，经济繁荣，还是不错的朝代，可惜军事太弱，否则恢复唐朝的辉煌有可能

查了一下南宋对付金军骑兵的办法.
1. 宋朝吴麟的叠阵,就是长矛居前,坐不得起,次强弩,跪以待,次神臂弓.敌人到100步,神臂弓发射,70步强弩并发,最后长矛手长矛前伸组成锋利的矛阵.骑兵从2侧突击敌人侧背.1131年9月此阵在甘肃剡家湾大破金军.
2. 韩世忠,岳飞的背嵬军,从南京,安徽招募敢死士兵,手持长柄斧,麻扎刀.上砍胸,下剁马足,在大仪镇,颖昌,2次大败金军.王德,杨存中在柘皋使万人操长斧,如墙而进,诸军鼓噪奋击,金军大败.而1140年7月岳飞在郾城,对付15000金朝铁甲骑兵,办法是骑兵先冲击,打乱敌人阵势,敢死队步兵手持马扎刀,提刀快速冲入敌阵,目不斜视,只管砍马腿.铁浮图是连环马,3匹1组,用绳索连接,1马倒地,2马无法行走.1140年6月在顺昌宋军用长矛攻击人, 劈山大斧砍马的战法打败3000铁浮图.
3. 宋朝用脚撅弩3人一组,1人发射,1人进弩,1人张弩,轮番射击,保持射速.1130年9月吴阶弩手数千在进入四川的要道尚原的高山设伏,强弓劲弩,分番迭射,连发不绝,箭如雨注.金军却无法反击,结果是重创金军,俘虏千人.
