北京300万的房子，月租金5000左右，租金收益率2%高房价领先于高房租的市场原理

如果是一年以前的我，我会说，这和股票是一个道理。市盈率(价格/每年利润，即房价/每年房租，本例中为50)这么高，说明它马上就要大跌了

经过一年的学习，我仍然会拿股票来打比方，但是我对股票的观念已经改了。我会说，市盈率高，说明市场上对它的未来盈利普遍看涨(未来北京房租要大涨)。当房租大涨之后，北京房价持平，导致市盈率(房价/房租)开始迅速降低，这才是北京房价要开始大跌的时候

具体到北京的例子，现在房价300万，年租金6万，市盈率是300/6=50倍。如果全国房价平均市盈率是20倍，那说明，市场上普遍认为，未来北京房租会上涨，按照普遍的20倍市盈率计算，市场认为北京未来的房租会涨到每年15万。

(崇拜一下英明的葡萄，他半年之前就在茶馆里说了，北京房租将会上涨150%。算算15万比6万，正好上涨了150%)
