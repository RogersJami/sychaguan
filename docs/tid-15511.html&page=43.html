通用电气公司成功测试包含35%增材制造零部件的航空发动机

我是高校的，这几年也接触了不少国内据说增材制造领域顶级专家，对这个比较感兴趣。你这个所谓增材制造部件，我猜测可以分两类，一类就是按照以前传统结构图纸整合以后增材制造出来，这个比较初步也比较简单，但是并没有完全发挥增材制造的潜力。第二类是结构方面完全依据功能和性能要求，根据增材制造的特点设计出来然后制造的。ge如果发动机三分之一属于增材制造，我估计大部分是前者。
目前如果单纯考虑功能和强度问题，能发挥增材制造潜力的优化设计应给没有太大问题。但是如果想涡轮机热端部件如一级叶片等部件，综合强度，气动冷却，振动综合结构优化设计，我估计难度相当大，恐怕还需要更长的时间，包括空间造型，拓扑，等等很多方面的持续工作进步。当然肯定这是未来的方向。
