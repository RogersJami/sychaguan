说一说我的看法，佛、儒、道都说一说吧。

佛教与“普世价值”有些相像，如“众生平等”的口号、《勇敢的心》最后那声“freedom”。可读历史就发现，佛教最后在印度还是莫名其妙的就没了。有的观点认为佛教的消亡是因为佛教不利于统治，还有伊斯兰教的毁灭性打击。可在我看病根还是出在缺乏底层民众支持这点上。相似的，看似“普世价值”最得人心，而实际这种价值观与贿选没有分别。表面上看支持它的人得了现实的好处，问题是长远来看美好“愿景”是否能持续下去。所以“普世价值”几乎就是欺诈。当泡沫无法继续膨胀下去的时候，支持它的人失望过后反而会变得暴戾、残酷。法国大革命在历史教科书中是以“革命”、“进步”、“自由”一类形象示人的，但多看一些历史书就知道那段时期几乎是法国最黑暗的时期。有的观念认为革命后的派系斗争才是问题关键，但没底层人民的支持他们怎么斗？“普世价值”与启蒙时期的进步思想一样，不过是用华丽的外壳包装个人和集体的私欲。有一句话讲得好“历史就在眼前，未来却在身后”迷信“普世价值”的人以我的观察大部分对历史、哲学、政治、经济都没有起码的认知。

儒家在中国的根深蒂固，一是因为它利于统治者的统治；二是它确实符合人类发展规律。相比较马克思提出的“共产主义”设想，实际与乌托邦没什么分别，没有一点操作性。苏联领导人用他们的集体智慧以马克思理论为基础创造了世界上最不人道的国家制度。一部分人确实“共产”了，另外一些人过得连牲口都不如。打着美丽的旗号，既对国内实施残酷的统治，又对东欧小伙伴进行残酷的剥削与压迫。所以主义什么的，都是扯淡，不管它有多华丽。而反观2500年前孔子提出的统治理念，非常地人性化。一方面有意维护统治者阶层；一方面又极大地保障了普通人民的切身利益。更为难得的是它为有能力者提供了一条明确而合理的上升通道（没上升通道肯定要造反）。打个不恰当的比方，与印度的种姓制度有些像。不同的是种姓制度是板上钉钉的，出生的种姓一辈子没法改变（搞不懂为什么底层人民很少有人造反，可能他们的宗教观念太深了吧）；而儒家文化则是变通的，如后来的科举制度，如仁者治天下（也就是为王朝更迭提供理论基础）。儒家文化的核心就是一个“中”字，不管对哪个利益主体都不能太放纵，同时也不能过于打压，如果失去平衡社会就会自然调整。可能是和平的方式；也可能是流血的方式。在我看，现在的“社会主义”道路实际就是改良版的孔子学说。

再来看道家思想。道家思想表面上看是一种消极的避世思想。道家思想的诞生以《道德经》为开端，那是春秋末期，战国初期。那个时期是一个非常混论的时期，有“春秋无义战”之说，战国就更没什么“义战”了。所以与其把道家思想看做是消极避世不如说是一种特殊年代的自我保护。看过一些纪录片和文章，把嵇康那种2货都算作道家学说的一份子，有点荒唐。如果《道德经》《庄子——内篇》都看过就知道道家的核心不是教人做事，不是消极逃避，更不是什么以柔克刚，而是教育人们在乱世或者危局中如何保全自己甚至过得更好。所以一个深刻领会道家学说的人无论是在经济大飞跃的时期还是战乱年代，或者是四十年前的那场灾难中，都应该过得不错。用一句话来概括就是：不极端的生存主义。

佛教不利于统治是一种误读，印度现在这样的身份社会制度如果没有佛教的教义支撑，你都无法想象会是 ...

说得很有道理。我对于佛教的认识来源于佛教经典和一些介绍印度历史的书籍，“佛教不利于统治”的看法就是来源于这些地方。

至于理由嘛，我这里说一下（个人看法，不必较真）。佛教和耆那教的产生都是为了对抗当时的种姓制度，它们的盛行也与刹帝利、吠舍（彼此对抗，而且用于对抗婆罗门）的利益争斗有关。由于当时婆罗门的绝对权威，两个宗教的盛行让第二和第三梯队的人看到了上升的可能。而下层的首陀罗和贱民由于力量微弱就算得到了宗教的理论支持也没有上升通道，所以对于两个宗教不太感冒。也就是因为这个原因，失去了发展的基础。而耆那教之所以能够发展到如今并且在印度很有影响，与部分刹帝利、吠舍阶层的支持密不可分。佛教由于自身的原因，教义不利于“再改造”，最终必定不适合用于统治。如“佛”这个名词的真正含义是“觉醒者”“觉悟者”（而不是“拯救者”），而且佛教的教义就是反对阶层的（而这正是用于统治的必要条件）。更为糟糕的是，佛教其实是一个极度唯心的没有偶像崇拜的宗教。也就是说佛陀其实是一个无神论者。而反观伊斯兰教、基督教都有“神”“偶像”“权威”一类的概念，而佛教没有。如果说佛教利于统治，那也只有一些“空即是色”“十二因缘”“苦集灭道”“六道轮回（对于这个我非常怀疑是不是佛陀首创）”的理论可以用于统治。这些理论确实有教导人们安于现状的意味，但这些理论并不能单独剥离出来，独立地存在（在佛教范畴内）。

佛教在印度的历史长河中存在的时间并不长，但也不算短。只是前期受到刹帝利和吠舍的追捧，而后期自由发展而已。如玄奘就曾经到那烂陀寺求学，并且参加过僧侣们的辩论大赛。佛教创立于2500年前，而玄奘西去的时间则是在公元600年之后，所以并不是一个短命的宗教。而且据记载，玄奘去到那烂陀寺的时候，那里精通三藏的高僧有1000多名（大致印象，我也记忆不清了），所以当时的佛教虽然不如先前辉煌了，但还是有一定底子的。而后来佛教的衰亡除了没有底层民众基础与伊斯兰入侵以外还有一个很重要的原因。那就是婆罗门教的宗教改革。婆罗门教由于受到佛教与耆那教的冲击，于是吸收了佛教中有利于统治的部分并且加以改良形成后来的印度教。由于印度教对于原有婆罗门教教义的完善令得它很快被刹帝利、吠舍接受，而且由于教义的改良令得首陀罗、贱民也更能接受了（没有仔细研究过教义，道听途说而已）。如原有婆罗门教的教义中关于首陀罗的定义是：“于千百世中，你都是首陀罗。”而改良后的教义则是：“今生你修行重下善因，来世就能成为吠舍。”以往的婆罗门教与印度教实际的差别只是字面上而已，可是却对印度社会产生深远影响。而反观佛教，一方面得不到上层的认可；一方面又得不到下层的推崇。所以灭亡也只是迟早的问题了。

对于印度的宗教，尤其是佛教，我只处于纸上谈兵阶段。要想深入理解，必须亲自去一趟印度，与他们各个阶层接触，否则仅仅从不多的经典和考据描述中很难了解清楚，况且我看的书籍也不是很多。这里仅仅只是一些看法，并非真理。先说这么多吧。

       佛教最大的问题是在寻求“不和解”的过程中泯灭了组织意识。佛教是所有宗教中最具备残酷性的，表现为广泛的适应性和真正实现了在“意识”上进行了分层。同一部典籍，看完后，有人伸出了脖子，有人抽出了刀。注定这样的宗教是不能提供给统治阶级以完备的组织意识的。没有组织约束，统治者内部，一个人很可能突然回过头来拔刀相向，而没有人能制止他这么干。各阶层之间失去流动性，在组织形态中是没有给上下层之间流动留有余地。
       佛教从根本上来说是放弃和解的。即便六祖慧能提出人人成佛的构想，但是意识形态框架并不支持这个构想。
