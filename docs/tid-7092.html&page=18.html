茶友在淘宝上有账号吗？如果没有，先去注册一个。然后需要开通网银（保险起见最好去银行办理），这是为了付款方便。网上购物时把购买的物品名称、数量、地址、收件人填写清楚。淘宝付款是这样的，买家付款后并不直接给卖家，而是先进入淘宝特有的支付宝，若买家得到物品验收满意之后，即可点取付款，支付宝会将钱打给卖家。若买家对于在验收时对货物并不满意，可以退货并申请卖家退款，虽然信誉度高的卖家一般都会退款，但最好还是在货物到达时仔细验收。
