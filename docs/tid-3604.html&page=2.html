看来葡萄修改掉了一点吵架的内容，但是文字晦涩依旧。

说到参与这个问题，我觉得你打的赌其实是必输的，组织中的权力是博弈的结果，党同伐异是社会性动物的必然选择，能上位的无论是否公平正当，总是有原因的。这本来就是统治者证明自己合理性的游戏。
对于被排除在政治格局之外的群体来说，在信息和技能都不具备的情况下贸然参与如同让普通人去跟讼棍打复杂的官司，但是真正不合理之处在于被排除在外的过程而不是参与的水平。所以从操作上说，你的指责有合理性，但是你又如何解释开国的那一批人在9成文盲的国家建立起这个共和国而不是像常校长那样的“精英”体制？
从生物学的角度说，人类的个体差异比起在社会中的状况差异要小的多，真正的差别来自于不同的人在组织中所处的不同位置，而这才是平等的根源所在（人对动物生杀予夺时不会给其平等的权力的，而拿人不当人在人类社会中虽然屡见不鲜，但总是越来越没有合法性的）。作为个体的人能力有限、寿命有限，要竞争就要组织起来。一个人一辈子能够在某一领域做出点成绩来已经很不容易，而绝大多数人都是蓦然回首人已老，所以人在构成组织时必然要分工。而分工自然产生协作的要求、管理的问题，权力由此而生。拳头看起来是根本性规则，其实只是组织实力的体现。
真正的问题在于，组织所创造出的成果巨大，而个体和其所在的组织并非绝对利益一致。由此产生两种少付出多获取的方式：上位者可以利用权力占有更多的财富，而决定利益分配的政治权力一直是人类史上最大的垄断行业；而底层的人可以靠混日子实现少干多拿（例如当年干多干少都一样的国企和欧猪福利制度下的无业游民），尽管所得有限，但也算是非暴力不合作的方式。只是天上不会掉馅饼，这两种人一多，组织必然入不敷出，最终走向崩溃解体。一个健康有活力的组织总是要控制这两种类型的食利行为（姑且这么说吧），但是统治阶层的逆向淘汰（其实就是财产、权力的血缘继承和能力不匹配）造成的食利者膨胀和腐化似乎是每个人类组织的最终结局—当然组织完蛋了人还在，就是改朝换代，重组公司。只是这个过程对很多个体来说是灭顶之灾，陈王奋起挥黄钺，天街踏尽公卿骨不是闹着玩的。
从分工开始-有如人从受精卵（全能的干细胞）分裂开始，细胞从胚胎期就开始组织分化，终归会形成不同类型的人体细胞。神经元细胞的日子就是比皮肤中的上皮细胞好得多，而这种结果基本上是不可逆的--阶层固化，现在穿越的大行其道其实就是现实中卢瑟们越来越多的体现—所以越是在困顿的群体中神棍们越是吃得开，当年轮子流行其实就是下岗者没有出路的表现。
当然在特定条件下，细胞可以重新转化(诱导)为干细胞，从而又有一次选择的机会（树挪死人挪活）。低等生物（简单组织）复制再生都很容易，高等生物基本上是只能修复，再生困难的多乃至不可能。
显然以上的类比会有问题，严格来说用生物群落来描述人类社会看起来明显更为合理，因为作为人的个体其实是跟群体并非同生共死的，但是这样一来会有更直白的现实—遇到灾难时总会有部分个体被牺牲掉，对动物来说是老弱病残，弱肉强食的自然规律。对人来说就更复杂，垄断阶级的战争选择一样是solution，但是这取决于计划被牺牲者的反抗行动，死道友不死贫道是个体的理性选择，放在集体博弈中就可能是灾难-这种悲剧在人类历史上太多了。
但我要说的是，越是精密复杂稳固的组织，越接近于第一个高等生物的机体而非后一个种群，变异被抑制的可能就越大，而一旦无法适应环境，就是恐龙的结局。所以哪怕是看起来不够完美，保留进化的能力也至关重要。弱肉强食的丛林法则依然低于适者生存的进化论。

而且吏治的问题到最后落在个体的人品上，其实是对制度建设的最大讽刺，或者说是对群体非理性的的无奈？如前所述，人的个体的力量其实非常有限，所谓能力越大破坏越大其实是权力越大破坏越大，如果参与博弈者很清楚博弈的规则和方式，又怎么会让个体的权力失控—简单的例子是律师事务所，一帮合伙人个个都是人精，谁都没法为所欲为，当然要做重大决定非常困难。
补充一点，Rp这东西很难说不会变，如同“潜伏”里的吴站长，从革命青年到党国蠹虫，简直太有代表性了。利益之下有几个人能坚持原则（何况原则到底是什么）？很多时候利益不仅仅是钱，还有团体、人情，诸般前因后果，哪有那么简单的大公无私？ 何况历史上还有像汪兆铭这样的，要是当初让他“引刀成一快”，恐怕无论历史课本还是人民英雄纪念碑的碑文都要纳入吧。
所以这还是一个Trade-off的问题，没有完美无缺的人和制度，为了某种优势就要付出某些代价-问题是，这个决定谁来做？
