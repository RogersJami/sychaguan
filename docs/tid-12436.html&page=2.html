       楼上老虎兄弟看来还是没有领悟我们这些举措的实质含义，我再解释得详细点。
       我为什么强调人民币要跟着高铁走？我们的高铁不是重载铁路，不能直接大规模运枪运炮的，这也是冷战时期敌对国家间最忌讳的事情。所以你看看苏联、外蒙他用的就是宽轨，搞自己的一套，不跟外界联通，怕的是铁路资源战时为敌方轻易利用。高铁讲究的是人员和商贸的互联和流通。我们中国人别的都不怕，就怕你对我搞封锁，不跟我来往。你跟我通，我就能发挥我的优势。我们的优势是勤劳、肯吃苦（说难听点就是民众对政府的剥削忍耐力比较高）。我们无论是建国后搞工业化，还是改开搞经济建设，原始积累靠的是大规模的民间财富，就是民众创造了财富，政府说你的福利不要过早享受，我拿去投资画更大的蛋糕。。。这是我们的优势，放眼全球，没有哪个国家有中国人勤劳、肯吃苦。
       改革开放30年，我们已经积累起巨大的财富，也有了初步的、配套全面的工业体系。那么目前我们面临的是两个困扰：一是人民对生活品质的要求更高，二是产业的转型升级非常迫切，产业升级是主要矛盾。在国内解决这两个问题存在重大困难。首先是我们的产业升级问题，如果依照市场规律大量淘汰过剩产能，就意味着很大的投资浪费，还意味着巨大的就业压力。主要产业不能完成升级，还意味着巨大的环境和资源压力。同时，我们目前市场上的一些产品质量不能让人放心，发展内需的空间受收入扩张和产品品种、质量等影响而受到抑制。所有的问题都指向产业转型这个根本性问题。解决这个根本性矛盾的出路是腾笼换鸟，不是大海的从粤到黔桂的腾，我们要腾到国外去。我们向外也不全是输出低端的、高污染、高耗能的产业，高铁就是我们的金名片，既是高大上的产业输出，顺带又联通了市场。
       腾出国内市场以后我们才有升级的空间，今后国内主要输出中高端制造业和金融、贸易、科技等中高端服务业，我们的资本会延伸到愿意跟我们交流的每个角落。我上面举的小例子，10年后非洲客户需要的苹果很可能就产自南美或俄罗斯，关键是它用人民币去交易去衡量。
       这次委内瑞拉财政破产，网上一帮小朋友瞎嚷嚷，我们照样贷给他200亿美元，这都是为我们的市场扩张做的必要铺垫，10年后大家就明白了。
       当然，我们的口号是：跟着中国走，大家奔富裕。这是中国人提供给全世界愿意跟中国人做生意的地球梦，也是我们跟美国佬的根本差别所在。
