奇怪的军姿3

英式军姿的缺点，抬臂很高，抬腿挺高，但是跨步不大。步幅频率也比较低。

还有立正，稍息和转身动作，看起来超级费劲，是最土鳖的军姿。

不过，英式军姿是各国军姿的祖宗，是从火枪兵射击队列演变出来的，所以拿着一杆长枪，走英式队列还是很好看的。

然后是法式军姿，这个系列从拿破仑那里开始，是炮兵纵队阵型演变出来的，没那么夸张的立正稍息和转身动作。这个传下去，就是俄罗斯和美国军队继承的风格。

接下去是法西斯式的军姿，就是墨索里尼的黑衫军和纳粹迈的正步，步幅又高又远，步幅还快，这个特点被苏军学去了，所以，实际上最优帝国风貌的军姿是苏军的军姿。

我国的军姿是改开之后又修改的，原来是中指在裤线上，改在食指，军礼扬臂的力度力矩都过度的更加自然。这个成果在建国35周年大阅兵上，产生了巨好的效果。

不过那个时候，我军的三军仪仗队身高在1.78米，排头兵在1.85米，现在平均身高至少高出10公分，所以效果差很多。
