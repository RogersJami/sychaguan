白沙 发表于 2013-10-11 11:23:57

给茶友们提供几点信息供参考：
一、本次台风所带来的雨水是巨量的，覆盖范围是全省的；
二、各主干支流均在大量泄洪，新安江水库等蓄洪设施具体情况不清楚，但拦蓄能力肯定已经达到极限了。钱塘江本来处于低潮位。但是很不幸台风带来的巨大风力驱使大量的海水倒灌进来，造成钱塘江水位处于较高位，给泄洪带来压力；
三、这次台风造成的灾害证明，一个城市的排涝系统再怎么发达，也比不上自然河网系统的排水能力，这从省内各区县市受灾的情况已经得到明证。更加不幸的是，我们的河网系统在近年大量的城市改造过程中受到两大创伤，一是淤塞严重，影响了排水速度；二是破坏严重，很多河道宽度已经被缩窄甚至被填堵，成为了建设用地。
四、我们的应急机动能力是基于主干道路基本畅通，供电通信基干设备基本完好的前提。余姚的特殊情况事实上是超出了预案估计的。在那样特殊的状况下，各类救援力量和救援物资的发散速度都受到极大的限制。

我不做结论。当然群众的呼声是可以理解的，但是面对这样的呼声，给我们提供一个反思，我们需要建设一个什么样规模的城市抗洪排涝体系？如果要做成象法国巴黎那样船可以在地下河穿行的体系当然是一种不错的考虑，但是到时候你是不是又要怀疑那是某个领导的政绩工程呢？确实，我们在很长的时间内或许一次都不需要动用那么庞大的工程。这个反思同样适用于其他跟民生有关的体系建设。

最后，我想说在天灾面前，我们还是要保持些平和的心态。人定胜天只是一种鼓舞人的口号，很多时候我们人类在大自然面前是很脆弱的。

提升应急能力，首先要求城市管理者有以人为本的自觉、未雨绸缪的智慧，将灾害应对 ...

       你的现场报道很生动也很客观，但是有一点或许存在先入为主的判断，就是关于8日的情况。我不太清楚8日余姚的雨况如何，反正我7日晚路过温州附近的高速公路，雨刮器打到最快也只能能够看到前方5--6米远的模糊影像，在那样的状况下，我深深感到人的力量相对于大自然的渺小，唯一希望的是后面不要冒出太多开豪车的鲁莽小子。我的意思是救灾力量的部署也需要一定的外在条件，当这样的条件很糟糕的时候，我们唯一能做的是祈求上帝的关照，我们的民众对政府的期望太高，可政府离上帝的标准还真不是一里两里地的问题，而是差着十万八千里。
       我额外讲点汶川救灾过程中的情况。当时震中纹绣镇是一座水电交通通信全断绝的人间孤岛，而且震后地理条件极端恶劣，很多五万分之一地图上的标识物全部找不到了。如何给纹绣镇人民送去党和政府的关爱是考验着指挥者神经的一个重要决断。关于动用军方力量过程中的一些小波折我就不讲了，最后是挑出了15名小伙子去执行这项艰难的任务。当时他们的遗书全部写好了，因为我军空降部队在这样恶劣的条件下执行任务还从未遇到过，3000米高空跳下去不要说能及时集中起来执行任何一点有意义的抢救活动，这些战士们连生还的机会都是很渺茫的。当然，最后上帝是眷顾这些小伙子和他们所爱着的人民的，他们全部活下来并及时把震中的受灾状况传递给了外界，最最主要的是他们给震中的民众送去了希望，送去了党和政府的关爱。反过来我们想一下，如果这些小伙子牺牲了，他们就会成为一批默默的烈士，而且绝不会有任何让他们去白白送死的疑问。军人的使命是服从党和国家的需要，不允许打任何折扣，但是任何无谓的牺牲事实上也是指挥者的重大渎职，可我们从不去想这些细节的问题。
       最后我提一个问题：作为一名军人，我们可以要求他去执行必死的任务，因为他们不是一个纯粹的人，他们在执行使命的时候只是武装集团的一个螺丝钉。如果在必死的任务面前，要求你去执行同样的任务，你可以做到吗？我想大部分人都只是人生父母养的凡夫俗子而已，我们不能以高得离谱的标准去要求别人做这做那，即使他拿着所谓纳税人的俸禄。

所谓应急框架，能力建设得越强，平时储备越多，所谓的“扰民”也就越多，这是一个硬币的两面。余姚是小政府，大社会的典型例子，而在这个模式下日常管理没有问题但是在突发应急事件前的能力缺失也是不争的事实。救灾动员能力不仅仅是物质准备，更有思想准备。个人、社会组织、政府的能力都有边界的，更何况，如果平时的关注重点在个人与社会组织的能力有多强，那么对其能力范围以外的罕发与突发事件存在盲点也就是大概率事件了。

我想写这篇文章文章的第一个用意。是中国必须建立一个系统，一个在各个部门之间交横连接的矩阵系统。
1，出台一部法律法律，对公共危机管理的部门进行授权，对公共管理部门的职责范围和定义进行明确的法律层面上的界定。给出授权时间期限
2，立法要求各省市县设有相应机构，每个部门或单位都指定具体人员负责，持证上岗，经常配合训练，充分保证下情上达。建立纵向垂直协调管理、横向相互沟通交流、信息资源和社会资源能够充分全面的共享、指挥协调高效、组织机构完备的全国范围的网络系统。
3，关键时能够动员的组织机构，志愿者组织，各级地方居委会，各个私人机构，超市，保险。。。等各级机构，处于一个系统性的横向机构，确保整个地方指挥体系运作。
4，志愿者组织也是分功能性的，如何让各级志愿者组织在国内的系统挂上号，实现救灾时一一功能性的对接是需要认真思考的。

但我们这次在余姚看到的是，地方动员体系基本瘫痪，大家都不知道要干嘛，所谓的动员危机管理体系只不过是一纸空文。有了志愿者送救灾物资反遭哄抢。
而且哪怕救灾不及时，你孬好也要上面知道下面到底出了什么事，这次余姚出事还是在群众的微信和短信中知道的居多，然后才引起上层的重视。上情下达，下情上知。基本是空话。

大家很自然的将日本海啸和中国汶川大地震做一次对比，反证了日本救灾体系不过尔尔。
是的，在两国国家动员层面来说，日本确实很一般，无论是应对局势的指挥，还是呼唤国际救援，乃至运送地方的救灾物资，但就福岛和余姚的地方层面来对比时，我们还认为日本的地方指挥系统比中国差吗。日本的志愿者组织也是能配合地方组织来积极救援。在中国没有地方底层组织配合，志愿者也是无头苍蝇。
