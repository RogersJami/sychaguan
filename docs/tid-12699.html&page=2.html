焊接机器人，我仅仅从机器视觉初步分析，应该可以实现，主要要靠2套视觉系统，一套视觉系统做整体的焊件的 ...

这是可行的，一套视觉系统用于初始焊位的导引和识别，即焊接工件的位置、固定精度等，另一条系统用于焊接动态过程的检测和监控，即焊接过程中熔池里熔化金属的动态行为。

国内的哈工大做过研究和尝试，一个是初始焊位视觉导引技术（激光扫描视觉系统），应用面比较窄，只实现了对一定工件焊缝特征在一定范围内的导引；另一个是视觉伺服和图像识别技术（高性能线阵CCD敏感器），初步探讨了焊缝识别与实时跟踪。

瑞典和美国先进一些，先后研制出基于激光扫描和结构光视觉传感器的焊缝跟踪系统。

但因为增加了辅助设备，很贵，而且系统复杂，应用推广受到一定限制。randy兄在做激光传感器，如果研发团队成员懂焊接工艺，也许可以尝试一下，难点是建模。

焊件位置、固定精度、机器人各构成的容差、焊件变形引起焊路偏差等等因素，造成熔池动态过程非线性、不确定性、时变性和强耦合性，传统的金属热传导数学模型、系统控制数学模型很难精确描述，已经有了神经网络模型和模糊集合模型的研究。

工业4.0工作组，随默克尔访华。

偏重于信息利用与智能控制

这是制造业的趋势，在未来会成为信息产业的一个组成部分，各国都在摸索实现路径，德国人提出工业4.0，美国人提出工业互联网，我国提出互联网+

工业4.0是以物理信息系统建立全球网络，整合用户终端、仓储系统、生产设施，实现人、设备和产品实时联通，最终建立动态生产配置方式，以后发主题细说。

细化一下这两个例子：

1，设备维护

生产车间正在自动运行，探伤检测仪实时扫描出生产设备，就像给人做核磁共振检查，发现设备某个部件有疲劳损失，在1周后失效，这个信息立即显示在生产控制中心的屏幕上，并在线传输给设备维护商，维护商的控制系统自动开出维护方案、材料清单，马上赶过来检修。

2，智能电网

这是美国工业互联网的概念，GE已经在做这样的工作。

美国大部分配电公司是私有的，很难协调统一增加电网基础设施建设，所以电网设备更新换代不如中国，解决办法是优化配电网络：

随用户的分布，附近就地建小型电厂，主要是天然气发电，也包括太阳能、风能等间隙性能源。用户加装智能电表，实行24小时浮动电价，用户通过手机等终端可以看到当前电价，决定是否用电。太阳能、风能不能满足需求时，天然气电厂发电弥补；太阳能、风能满足需求且有剩余时，多余电力用来压缩气体储能或抽水储能，适当时再放送出来。


互联网+应用不太清楚，电商应该属于这一类。
