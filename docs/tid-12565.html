马年回顾

本帖最后由 思炎 于 2015-2-18 11:39 编辑

时光飞逝，混乱的马年已过去。

2014年可以说是人类历史上重要的一个节点，是第一次世界大战爆发一百周年，第二次世界大战爆发75周年，是中日甲午战争120周年。 在这一年里，有国与国的合作，也有战乱与纷争，大国地缘战略争夺加剧、民族宗教问题凸显、恐怖极端势力影响扩大，传统大国与新兴大国之间的博弈也是更趋激烈。

这也是冲突与灾难不断的一年，MH370的失联、MH17被击落、亚航空难、韩国客轮沉没、非洲爆发埃博拉，造成世界的恐慌。美国黑人被枪杀及警察被枪杀、昆明3.01恐怖袭击事件等等。而这些混乱局面似乎延续到了2015年，上海外滩跨年踩踏事件、法国爆发查理事件恐怖袭击案，台湾复兴航空空难。。。。

通过量化宽松，2014年低，奥巴马豪迈的宣布：美国的经济好转，失业率下降。

可是，明显我们也看到美国的全球影响力在走下坡，此消彼长，世界格局也悄然的变化。中国在世界舞台上释放出的正能量与积极进取的风格，自然有人喜有人忧愁，随着美国重返亚太，亚太再平衡战略，亚太地区剑拔弩张，特别是上半年，在南海东海的领土争端不断；下半年整体趋向缓和，但暗涌不断。


由军政府重新掌权的泰国与埃及，尽管受到美国的指责，国家至少目前却相对安稳。

反观在欧洲，乌克兰颜色革命后，烽火连天，从年头打到年尾，领土越打越少，民族撕裂， 进入2015年双方仍未停止交战。。。欧盟北约的东扩，与要想重振声威的俄罗斯，在乌克兰展开角力。不仅撕裂欧洲，美国与俄罗斯的关系，更是在俄罗斯吞并克里米亚后进入冰点。俄罗斯被赶出了G8，被美国与其盟友轮番经济制裁，普京被西方媒体不断攻击，并指其有自闭症。不过，这些都没有改变俄罗斯的行为，普京在国内的民意与声望则达到了历史最高点，并增强了克里姆林宫宫里强硬派的势力力量。

奥巴马领导欧洲盟国反击普京，疲于奔命，还被指责应对软弱，前面几轮对俄罗斯的制裁显得虚张声势。 年尾掀起的“石油大战”，倒给中国印度等国家捡了漏。

除了应对让奥巴马头疼的普京，美国打击恐怖组织IS也显得无力 。在中东的这片古老土地上，依旧充满了杀戮和血腥，美国发动的伊拉克战争打破了中东地元政治的平衡，为推翻叙利亚阿萨德，美国支持培养训练的战士，宗教的对立，滋生了极端恐怖组织IS，反恐却越反越恐。IS残暴行径震惊世界，不仅最忠实的小弟日本的2名人质被杀害，就连来自美国与欧洲盟友的人质也没成功解救。 尼日利亚、索马里、利比亚、马里也成为恐怖主义的天堂。

奥巴马刚向国会申请了战争授权打击IS，拭目以待，今年美国反恐成效。


不管是否有心无力，奥巴马说：美国还要再领导世界1百年。他还说：China wants to write the rules for the world’s fastest-growing region… Why would we let that happen? We should write those rules……

在2014年，中国主要关键词，一个是反腐，一个是外交。欧洲经济陷入低迷，俄美冷战加剧，中国则展示了在经济领域的领导地位， 成为世界舞台中心的亮点。

习近平与李克强走访30多个国家。与俄罗斯签署了西线天然气协议，与韩国完成自贸区协定实质性谈判，与澳大利亚达成自贸区协议，领军成立亚洲基础设施投资银行，金砖银行等。

5月在上海举行的亚信峰会，与11月在北京举办的APEC，展现了大国外交与中国主场外交，积极开拓建构新的国际关系，在国际事务上争取更多主动权，重塑世界政治格局。中国提出亚洲新安全观、核安全观、亚太自贸区、中国海洋观、一带一路战略等。

2014年，中国的外交也从温和转向强势，走和平发展道路，但也绝不放弃正当权益，不能牺牲国家核心利益。在钓鱼岛与南海问题持续博弈，针锋相对，加大巡航力度，加速填海造岛。 2015年，中国自然也继续面临挑战，特别中美的博弈，希望少点灰太狼，多点喜洋洋。

根据预测，到了2030年，中国的经济或将达到美国的两倍以上；现在世界上，越来越多国家与人民反对美国单一霸权，做世界警察耀武扬威， 越欢迎一个崛起的中国在国际事务中发挥政治影响力，并改造由西方主导的现存国际秩序。有西方媒体预计，在过去的一个世纪，世界的权利中心由欧洲转到了美国，而下个世纪，世界的权利中心再次转移到中国。

愿我有生之年，见您君临天下！

在羊年春节来临之际，祝愿世界多一些和平；祖国国泰民安；茶友吉祥如意，收获更多。
