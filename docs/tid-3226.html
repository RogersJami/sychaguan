第一次做的系统
98年前后我们离开国企后，在南汽菲亚特谈了一台专机，回到公司后我找西门子销售要了两件东西：技术支持的电话、802D的简明安装调试手册。当年的调试手册没有汉语的，一个星期看懂并翻成汉语，由于从未干过，心里还是不自信，现在想起与当时与技术支持大腕XX的电话至今还是觉得好笑：
        我：“X工，我是搞机械的，但会用计算机，会用C语言，写过几千行程序，这个软件还得过奖，你看我能自己做这个NC的开发吗？”。其实我想从他那里得到鼓舞。
        他：“那你比我的水平高，但我不能保证你能做出来，我的老师水平也很高，但让他们做一个具体的XX系统也有难度。”
        我：“我想我的表达有错误。你能否告诉我802D这样的系统一个简单实用的开发包括哪些具体的工作？"
        他：“确定组成轴数、电机转矩；做出电气图纸；做盘，编写PLC；联机调试。大概就这些吧，比较重要的是PLC，系统设置机床数据可以让销售的技术支持到现场帮你搞一下，如果调试手册看懂了自己也能做。”
接下来我就把第一个系统做下来了，中间与XX通了几个电话。其中一个比较典型：
        我：“PP板的输出点电压都是直流11.几伏，这是为什么？”
        他：“PP板的接地，特别是公共端的接地没做好。你检查一下。”
        我：“我问过电工，接地了啊。”
        他：“如果能确认接地没问题，那就是PP板坏了，发订货号、服务号给XXX，换一块                        PP板。你在拆PP板时你看一次，是否接地没问题。”
        ....... .........
        我：“X工：你好，昨天的PP板问题已经搞好了，确实是接地有误。另外，我很敬佩你                们的服务，电话就能解决问题，比上门服务好多了，效率高啊。”
        他：“我们公司要求电话解决，你要有问题就打电话。”
后来，他写了一本书，题字后给我寄来一本。当年那些人，优秀。
        还是这套系统，接到用户电话... ... ... ...，我赶紧给销售打过去：“XXX，从用户说的情况看是611U的功率模块坏了，这个客户是TM汽车厂，没法停，今天一定要给我发一套过去。我们已经上车出发了。”西门子销售用武汉到南京的长途车带了一块功率模块到南京。换上，完活。但有一电机声音不好，优化！电缆没带，又是给销售打电话，“销售是你妈呀？”我心里万分歉意，没办法，我打。销售回话很温暖“我们有个工厂就在你们现在的位置的隔壁，我马上托人送来，优化让来人做一下就可以”。
        过了一会电话来了，让我去大门口接人。来人一个年轻女工程师，带到车间十多分钟完活。我对她说，“我们还得跟班看一段时间，傍晚下班才能回去，我先送你出去？”，她说“我也跟着看看吧，我的父母也在这个行业干了一辈子。”
        机床在工作，我们就闲聊。十多年前人们的关注是不一样的，她问，“你们为什么选用西门子的系统，好些公司说西门子价格高不太接受？”我说，“我们觉得西门子系统比较浪漫，就选用它了。”“浪漫？你说的是... ...”，“对，我说的就是浪漫，西门子让人觉得是西门庆的后人，802D系统可能还有潘金莲的血统。”“一点都不好笑。”
        下班了，我请了车间的人，也就给那位西门子的女工程师说“X工，晚上把男朋友约上，我们一起到城里吃顿饭。”她说“吃饭就免了，你还是告诉我为什么用西门子？”我看着她认真的说“六七十年前，就在这个城市发生了可怕的屠杀，有一个叫拉贝的德国人救了很多中国人，你知道吗？拉贝就是你的老上级，西门子中国区的老总。”暮色里，我看见她的眼睛里有东西在闪光，她说“我们公司大厅里就有拉贝的铜像，你明天来我们公司吧，你可以带一束花放到他的铜像前... ...”“今夜我就得往回赶，献花我就不去了。”我没有去献花，但十多年来我们一直用的都是西门子。

后记
        近几天，看到网上一则新闻：北京一男子寥丹，为救老婆刻假印章盖在缴费单据上。看到后想起好多年以前在琉璃厂看过的一幅黄永玉书写的对联，大约是：人到万难须放胆，事逢两可要凭心。
        2007年，在北方一个日企谈了一个项目，11台机床换系统。在使用什么系统上出了问题，对方的设备部门要求使用法那科，主谈的技术部长和我僵持了。我给他讲了拉贝的故事，他看着我不知在想什么，我对他说起了黄永玉的这幅对联，我说“这件事你能让我凭心来做吗？”他怔怔地对我说：“你怎么... ...是这样的人？”
        第二天，技术协议签了，项目除了价格、交货期已经没有其他障碍，那个技术部长递给我一个笔记本对我说“昨天你说的那个对联我记不清了，谢谢你帮我写下来。”我用工整的魏碑写下了“人到万难须放胆，事逢两可要凭心。”

初识NC
        80年代初，晚秋。我随办公室下班的人流往家属区移动，手里还有一个市面上最大规格的铝水壶，下班路上在开水房带一壶开水，是三线厂里的仅有体现效率的一个习惯。
        Y总，66年清华毕业来此，最无领导范儿的总工。“小X ...小X... ...”，长一声短一声的嚷着向我奔来，我提着沉重的水壶看着他心里想“从下乡回来就没有听过这样招呼人的啦，下乡的山区是‘通讯基本靠吼’啊... ...”思绪飞扬之际他已到眼前，“小X，你... ...必须... ...”“Y总，不着急，你慢慢说，我也把水放下。”他塞给我一个文件袋，“赶到青岛去开个会，就是XXX的项目，你参与了，情况也清楚，注意谦虚谨慎，参会的很多部、省领导”我把文件袋里的会议通知放了回去，说“我不去！总工的会，我还差五级啦。”“你们的支书和你谈过话了吧，你是重点培养对象，以后的担子比我还重。这也是个锻炼学习的机会嘛。”
        少不更事啊，走了。第一站必须尽快赶到郑州，在那里才有去青岛的车。到了郑州，拿出中华人民共和国第一机械工业部的会议通知往值班窗口一放，对方说，“票没了，不着急你等会儿”，一会来了个带袖章的女同志，“跟俺走。”一直把我送上车，没坐一直站到兖州，这一路站着没事，我就想：这Y总啊，分明是自己把这开会的事忘了，拿高帽子哄我来吃赶路的苦。下次再敢如此，让他把委任状开好了拿来再说。想到此，心中平衡了些许。

到了青岛就算“找到组织了”，会务组接到XX酒店，办报到手续，重点是要交全国粮票，全国各地来的领导，带省票的一律受到会务组女登记员的训斥。八十年代，部里的司局长到工厂里的职工，生活水平差距并不大，这一点今人是不好想象的，来开一次部属会议也是一次吃大锅饭，部里领导也体谅下面，三线来的同志多吃点海货。老规矩，晚饭每桌两瓶白酒，啤酒不限，但头一晚好多人没到，空桌很多。酒酣饭饱，回到房间同屋是西北XX局的处长，泡个热水澡，一路的委屈辛苦全忘了。
出来时那人已上床了，礼貌的问候“水很好，不洗一下?”，
“喔，一到我就洗了，这地方就是水好。”
“我第一次来青岛，开完会我还想去看看海，你去吗？”
“看哈呢，屋外就是，这楼倒了能把你泡海里。”
“... ...”
我心里想，来时天黑了，也没注意到。再看那人，鼾声响起。
晚上有几次敲门，来人都是来找YXX同学、熟人，分别都说明一番我是来顶替的。

        开会了，我在后排找个位置坐下，打开笔记本（非电脑），在左上角写好时间、地点、会议标题等。酒店漂亮的服务员在分发会议文件、文具、茶水，扬声器放着军港的夜之类的歌曲，我品味不高的欣赏着每次开会都发点小东西不知道悲剧就要来了。

craftsman 发表于 2012-9-20 22:29:46

扬声器响起一个上海口音“XXXXX的Y总请上到主席台就坐”，那是我的单位名称，“怎么会这样，上去坐？还有电视台的摄像机，怎么这么没天理呀？”极不情愿的站起来向台上走去，我想去给主持说一下：“俺是士官代理的正团职，是个代理就列席一下，各位领导就免礼啦。”心里还在贫已经走到了过道上，这时我才注意到穿的是双铸造车间用的翻毛大头皮鞋，抬头再看台上各位“首长”，着装都很严肃，在看那摄像机上还他妈要命的画着“原子在旋转”的台标，我仰天长叹“天绝我也”，这他妈在新闻联播一播，全国人民以为... ... 关键我以后如何在这江湖上行走啊，想掉头回去，不行。用在公交车上放了屁的那一招？向四周看看，谁呀？不行啦就我站着。我想起无数的革命先烈在我们的前面... ... 上台径直大踏步的的走到主持人身边，大头鞋砸的铺了地毯的主席台咚咚作响，把脸贴到主持人的耳边，低声告诉他我是替补，在下面第一排就坐，需要发言就马上上来，稍作停顿，在他反应过来之前大踏步的撤下来了。其实会议涉及我们的内容也很简单所有的电动机要靠上国际标准即现在用的Y系列，主要优势是节能，全国的电机换代这题目就大了，但高效率制造电机的关键设备只有一个国家的两个公司制造，部里规划是“两条腿”进口一部分，自制一部分，等看到样机又发现关键设备之所以关键其实就一个部件可称关键--------福开森分度机构，这个机构的一个三维凸轮无法用传统方法加工，进口国外机床，当年有个巴统（巴黎统筹会）专门限制对共产党国家出口高技术装备。我的工厂任务：做出能加工出核心凸轮的机床！这一串扰口令能理解的就知道工业其实是个体系，缺一个环节就不行。那时你有多少钱人家不卖给你。第一代领袖的贡献非常大，质量有差距，但别人有的咱都有。个别没有也有一两个单位在攻关。
