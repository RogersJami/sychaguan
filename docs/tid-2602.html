2.


软弱的人，往往等着别人来解救。
了不起的人，能够实现自我救赎。
而更了不起的人，不但能解救自己，还能救赎别人。
《肖申克的救赎》讲的就是这样一个关于救赎的故事。

这原本是一部应该获得奥斯卡奖的影片。
但那一年，还有《阿甘正传》。
大片云集的，所以，与奖项擦肩而过。

但反过来想想，好多事情是公道自在人心。
获个大奖，别人说你好，真的有那么重要么？
我相信，即使没获奖，这部电影依然以它的独特魅力征服了观众无数。

紫微星明 发表于 2012-6-22 19:19:37

摩根·弗里曼演的角色瑞德（Red），在影片里谈到过好多次，在监狱里面时间长了，成了监狱这个体制的一部分。

出狱后，在超市打散工，上厕所习惯的举手报告，经理告诉他想去就去不用报告。结果不需要报告无人批准，他一滴尿尿不出来。

这个寓意也好，写实也好，说明天下的人都是一样的。即使监狱这样的体制，也有人最后适应，正是这样的他，这样一个好死不如赖活着的本分思想，反衬出蒂姆罗宾斯在即将刑满出狱的前期，毅然越狱的刺激。

好莱坞影片，一直有个反体制的核，就是鼓励侠盗思想，其实这个思想很迎合小资情调，大家很感动，很过瘾，根本来自于天性的未曾泯灭。
