嗯，按你说的，只要政府不停地盖房子，舆论配合，老百姓就能今天买了小房子，明天听舆论宣传买更大的房子 ...
我可没说“只要，一直，不停”之类的词语。

政府可以想办法找别的甜点让老百姓掏钱，促进消费拉动经济增长啊。
比如说网络游戏，你看那些玩游戏玩的老爹是谁都不知道了的，每个月花几千几万玩游戏的，这就是政府鼓励的产业升级。
还有网络彩票，那些玩彩票玩的倾家荡产，还有贪污公款买彩票的，这也是政府鼓励的产业升级。

房地产只是促进消费拉动经济的一个比较有效的途径而已。新增长点还在培育，没什么不可替代的。巴西人很多对足球的需求高于对住房的需求，足球也一样可以作为拉动经济的增长点。

中国老百姓也可以改变自己的消费习惯，花钱旅游、花钱去酒吧去社交、花钱运动，就是不花钱买房子，对房子失去兴趣。那这个增长点也就不存在了。
