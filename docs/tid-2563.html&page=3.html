《易经》是中国群经之首，展示的古代中国人的朴素辩证法和对基本本质原则，中国人对世界的理解方式：变易（世界在循环往复不断动态变化），不易（世界变化的基本规律天道是不变的：物极必反，否极泰来），简易（事物的本质无非阴阳转化，刚柔相济，同性相斥而已）。
