投机客 发表于 2014-4-11 21:32:19

窃以为认为只要精通溜须就能升职的可能还是经历少一些。

实际上，只要成为对领导有用的人才能升职。注意是对领导有用，不是对单位有用。
有用包括几类：
1、领导需要政绩，能干的人对领导有用。但一般单位能干的人太多，就需要综合素质了。
2、领导有个人需求，能给领导私人办事的人对领导有用。
3、领导有场面需求，能喝又能调气氛的人对领导有用。
.........................

其它还有很多需求，光抱怨没用，还是找自己的闪光点吧。我感觉精通溜须比业务上能干技术含量更高，能一直挠到领导痒处的不是一般人，不要不服。
