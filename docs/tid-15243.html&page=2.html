在明确提出大数据这一概念之前，工业领域一直在应用与此相关的技术。

一，改善物流，降低成本

汽车制造行业，平均90秒产生产一辆轿车，总装厂需要协调上百家供应商，在特定时间把特定的零部件送到特定地点，年产量上百万的总装厂方圆几平方公里内，每小时需要安排上千辆零部件运输车的有序排队。

延展一下，一辆运输车只能运送个位数的发动机，却能运送几千条线束，所以不同零部件供应商有不同的选址经济半径，做产业规划的时候要考虑这个经济半径，有的供应商尽可能靠近总装厂，有的可以远一些，从而降低整车成本，这就是大数据。

二，监控设备，提高效率

发电行业，给蒸汽燃机装上传感器，监控各设备的运行状态，在某个设备故障、失效之前及时主动维护，而不是等到某个设备失效，导致全套装备停车才被动分析根源。纪录片《大国重器》中讲到了陕鼓的这个做法。

三，实验测试，开发新产品

直10设计时实现整机数字化，每个零部件都有计算机三维数据和图纸，零部件制造商拿了数据就可以直接在数控机床上生产。如果再加入制造工艺数字化，在计算机中仿真整个装配过程，就形成平台技术，设计和制造一体化，当特种需要出现时加减某些设备，只要在平台中加减相应的装配过程、夹具、模具、检具，就是产品改型和开发。

四，业务决策

博彩公司根据投注比例，改变输赢球的赔率，保证稳赚不赔。大型企业集团出售、并购业务也属于此类，砍掉投入产出比低的业务。

这是企业层面的大数据，以后还会发展，覆盖产品的设计、测试、制造、销售、维护整个生命周期，联通制造、基建、金融、物流等相关行业，从而提高整个经济体系的效率。

但这只是理想中美好的一面，如果上升到国家和社会层面，必须从工业革命300年以来，市场经济制度对世界的改变说起，大数据可能会呈现可怕的一面。

2.在有关人工智能的进化中，在2008年的时候，我一些专攻人工智能硬件领域硬件的专家朋友，在一次讨论人工智能的时候，他坚决的说冯诺依曼结构在现在的计算机领域不可突破。而我提及参与数据是生产要素讨论的连我在内的三个朋友之中，一个是在三岔口帖子第二部分提及的给出数据主创设的人，另一个是给出相关拓扑结构图让我们得以推导出有关结论的人。他也在算法角度给出，基于冯诺依曼结构有关算法角度的对于人工智能的否定性回答。但是，这一切在前年初，谷歌发布的大型并行计算阵列对于人工自我学习有关突破的时候。之前学硬件的朋友，开始对他当年的论断做出修正，既冯诺依曼结构在单体机依旧突破不了相关人工智能的瓶颈限制，但是通过大量并行机并行处理的方式可能有效突波人工智能自我学习的有关局限性。后来一个新加入讨论的朋友，他是算法领域的博士，他用自己的专业表达为，暴力破解解决问题。既在算法中实现不了的事情，通过海量计算来突破。这种暴力破解，是曾经有先例的。既，图灵在试图破解谜密码机的时候给出了这样的结论，有效破译不可能。但是可以通过算力或者速度覆盖算法的局限性来破解密码。这个是我对暴力破解方面的理解，也是暴力破解对人工智能自我学习部分的相关应用在这次围棋比赛的应用。至少，这是理论上没有超出现在科学的解释。或者说，是可以达到预期的。

3.人工智能的另一个领域，既达到人脑智慧的领域我看到讨论中有这样一个误区。既，多数人在讨论，人工智能能不能达到人的智力水准。我不妨把这个话题表达为，人工智能能达到多少岁的人智力范围的水准。不然，我们无法解释，人工智能在现实中的有关应用问题。好了这事旁话暂且按下。我这里对比谷歌的暴力破解方案，试图表达一些更多的进展。这里触动我的是老槐的关于人脑科学的进展。是的，这里关于人脑科学的进展是很多人期待的真正的工智能的不同路径。既通过模拟人脑的运行机制及其原理，来实现人工智能拟人化的逆向工程。这个路径最大问题是，我们现在对人脑在运作过程中的基本原理还有很多未知领域有待破解，甚至在一些有关人工智能以及人工智能自我学习过程中对人脑运作的拟制，在人脑工作原理方面还没有公认的有效理论。也因此，这个领域的实践与突破各国都保密在保密。我此前，在相关领域贴的有关研究进展，也是因为有朋友在相关美国国家实验室做的工作的突破在公开新闻披露后才能表达一二。这个方面的突破就是，通过技术手段删除人的记忆已经变的可操作，同样技术原理也可以引导植入特定记忆进入人脑。在不久前我看到公开资料的有关进展是，人脑借口，以及通过技术手段复制人脑在梦境中出现特定的记忆已经获得突破。暂时这种突破，在公开资料表达方式是。复原人脑在梦境出现的特定人脸。

有关脑科学和大数据方面，我有一套简单的理解模式。
作个类比的话，人脑是一个即精密又不太精密的多级数据筛除体系，其工作原理，大致就是对数据进行动态逐层过筛，直到筛除出有限可分析数据。如筛除结果无法收敛则转变构型进行换向过筛，直至出现有限数据或者出现多系统的有限数据，我们的理性思维多出现于这一层面，人工智能目前突破的也都集中在这一层面；而如果筛除结果为零形不成结果，此时展现人脑NB的一面就来了，它会自我“补充”数据，并用补充后的数据链重新进行反复过筛，这就是所谓的“创造力”，现实的例子就是，当我们在漫无目的的观看一组画面后，当主持人突然要求我们说出刚才出现多少组人脸，多少个三角时，我们的人脑在度过初期的慌乱后，都会迅速根据破碎的记忆“脑补”出一些实际不曾出现的画面，然后给出各种各样的答案，而如果主持人事先提示了要求，那么往往我们的脑子会非常精确地给出答案。这实质上就是现有人工智能和人脑的风水岭，即会不会自我补充数据，也就是“脑补”。
进入围棋的话题，从棋类的原理来说，电脑打败人类只是时间问题，因为棋谱是有限的，一旦可能的棋谱都被下完，那么电脑将棋谱补齐必然完胜人脑，而之所以电脑至今赢不了人脑，就是因为围棋棋谱太多，把之前所有能追溯的棋谱全部录入，依然有足够的“新手筋”让电脑抓狂。而最近之所以出现突破，只是表明“高手棋谱”出现了大量同质化，下出新手筋的几率变小了，于是电脑的胜率自然就高起来了。
再到大数据，大数据其实探讨的就是人脑的基本工作原理，人脑对数据的过筛有着大量精巧的空间构型和能量利用方面的优势，其空间单位计算能力其实远高于电脑（平面上则远逊），人脑相对于电脑是跨维度方面的优势而非数据累积，换言之电脑的计算力被大量浪费于无效领域，因此即便计算力突跃几个数量级，依然没有超越人脑的希望，这就是冯诺曼构型的死局。而大数据方面恰恰给了我们这方面的一个认识。很多人说大数据无效是因为数据不够大，这么说也对也不对，对是因为，确实只有足够大的数据才会显现出一些特殊的关联性，于是数据才有了意义，但实际上让数据大到出现关联性仅仅是万里长征的第一步，因为对应的这种关联性，人脑可能在几个神经节上就已经实现了，人脑已经在大数据方面领先了电脑不知道多少，它已经把这些需要大量数据得出的关联性直接固化到神经节间的微细过筛上，因此几个构象奇特的神经节就已经能过筛海量的数据，而同等过筛密度下，计算机已经在无效数据的海洋里冒烟了。
说得更白一点，所谓的大数据，只是把人类社会的拟人化推进了一小步，虽然这一步可能是划时代意义，人类社会朝着拥有自己的意识迈出了坚实的一步，而在此之前，整个人类社会所显现的“智商”基本超不过5岁孩童，除非出现伟人级人物的“独裁”，要不怎么说大众是愚蠢的呢？很多人在忧虑那个成为MATRIX的未来，殊不知如果真出现MATRIX，那就是说明生命体进化到下一个数量级（维度）了，嗯，地球只是一个“人”。
