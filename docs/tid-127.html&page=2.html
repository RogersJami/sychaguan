weakleilei 发表于 2012-4-11 18:20:26

说说我最近几个月的港股操作思路：折价空间套利

理论：同股同权，虽然AH上市地点不同，但股票代表的公司权益是相同的，所以内在价值也是相同的。当两个市场出现大幅度价差的时候，就可以做买入低价市场，卖出高价市场的对冲操作。等待差价降低，再反向操作，从而实现在所持股份数量不变的前提下降低成本，套取折价空间带来的收益。

目标对象：民生银行（A股代码600016，港股代码HK01988）

具体操作：2011年9月30日-10月3日期间，H股民生银行价格4.47-4.62元港币区间买入民生银行H股。期间A股民生银行价格5.52元人民币。期间港币汇率81.78.  H股折合人民币3.66-3.78元，A股溢价50%。

2012年4月10日-4月11日，H股7.52港币（折合人民币6.09元）卖出H股，人民币6.21元买入等量A股（A股溢价2%）。

对冲套利完成。
