个人的一点想法：

我觉得忙总前文说过的一些话很有意义，意思就是 TG 很自信，也很有危机感，所以在党校的一些培训课里什么都能讲（甚至在我们看来很反动），所以他们对于一些报纸媒体上的颇有西方自由民主式的言论是放任的态度。

所以，我个人斗胆恶意猜测，08年西藏的事件，09年新疆的事件，其实整个进程都在TG监控和控制下，有故意放任的操作手法，目的就是在一定范围内控制事态，但是也不避免事态的发生。
而在事件发生后通过舆论宣传等手段引导民间的舆论和民意对事件进行强烈批判和强烈反感，这比100次的常规宣传都要强很多（一个灾难就可以改变很多人的思想）。

而我们从事后的实际效果来看，08西藏事件后，国外媒体虚构图片新闻等来抹黑，而事情暴漏后，国内对整个西方媒体的态度大转折啊， CNN等机构在中国臭了好几条大街。
09年新疆事件后，国内民众对疆独份子的恶行都很反感，尤其是在新疆的汉族群众。
最近的黄岩岛事件也给人感觉有类似操作手法，国内的民意汹涌啊，喊打声一片。
至于这几起事件中的受害者，他们确实很无辜，而这也是最邪恶的地方，只能说他们确实做了牺牲品。除了他们的家人得到的经济补偿和其他人的同情外，他们什么也没有得到。他们的最大作用就是用自身的牺牲来成全了大局吧。
古今中外，这种“被牺牲”的小人物数不胜数。就连强大的美国，不也有很多这样的小人物“被牺牲”吗。

我觉得这几年TG对危机事件的处理手法越来越老道，估计也是借鉴了不少美国政府操纵民意和舆论的手法，毕竟要取而代之，美国的一些成功点就要学习，例如如何愚民。
