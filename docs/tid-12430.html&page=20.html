感谢葡萄大哥的坚持！
  别的方面我个人经历浅薄没什么好说的，但借这个地方说说股票我想还是可以的。
我是上次大盘高峰前5800点左右入市的，这之后就是6124，然后就是1664了。到现在也是8年半老股民了，其中的艰辛和困难我想别人很难体会。多年的学习摸索，这几年有幸跟着一位前辈做股票；这位前辈是一位德艺双馨的高人，能遇到他是我做票的转折更是福气。这位前辈的成名战是在川投能源上。当年大盘半年多涨了1500点，川投上下没超过30%，他硬生生的在上面贴死吸了半年血，最后赶走了第3位的股东，后来老大老二也受不了了，川投启动半年涨了10倍。而最开始和他做这个票（当时他是一个股票论坛的版主，已经小有名气）的人已经不到十分之一。而很多之前跟着他在别的票上面大赚的人也到处骂他。这就是人心。
      我是10年开始知道他的，这几年跟着他做了好多牛股，但就我能吃住的就那么几只，收益在众多同学中真是惭愧，我分析没能在启动阶段坚持住、屁股挪来挪去是根本原因。这几年我听到他说的最多的一句话就是：价格最能改变人心。
     先生早就积累够了财富，现在却仍然坚持尽心尽力带我们小散做票（没有任何好处的），他说就是尽一个前辈的经历帮助我们小散能少走弯路。葡萄大哥我想同样也是这样的。我想儒家所讲的 修身齐家治国平天下是不是就是这样的呢？ 感谢这些热心的、耐心的、愿意帮助大家的人。
人生若只如初见，何事秋风悲画扇。

等闲变却故人心，却道故人心易变。

送给所有在资本市场上战斗的人。耐心、坚持、相信，感恩，我想终会有所得。
