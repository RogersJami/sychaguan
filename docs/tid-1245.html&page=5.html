改革是一个重新布局利益分配的过程，责权利都要协调。我们不攻击大架构，从操作层面看，目前某党的弊端在于权利监督不力，收紧就死掉，放松就乱搞。死掉是因为事不关己，死掉就死掉;乱搞是因为行政体制权力集中，监督不利情况下，有利益不吃进，雷锋啊。

因此，改革有两种

一种是容易见效风险低的。就是责权利本来是可以弄得比较顺的。这里改革主要就是理顺关系。

例如学校管理民主化，教育本来就是教师教授个人色彩重，行政化和公司化肯定不行；必须民主管理，公开办学，学术监督，公开监督。这方面，搞得好的国家也不涉及政治架构。中国大学现在反正这么烂，改好一点也容易。

医改也一样。医疗核心是医生，最值钱的是医术医德。现在价格体制，医术的公开价格就是门诊三五块，医生不开大处方吃什么？7年临床大学毕业，难道一个月挣3000？卫生部长是猪脑么？所以要提高诊金，让医生正常工作也能有合理收入。

电灯泡 发表于 2012-3-24 09:48:01

请教忙总：如何看待医改？
按我看到的有些改化比较深入的地方，都是把公立医院定位为基础公益设施，由村-乡（镇）-县-市-省大医院全盘布局。每一级担当的功能都不一样，总体来说都是小病在小医院看，大病大医院看。依靠医保报销比例的不同来分散大医院的就诊压力。
完成医改的医院采用统收统支，药品零差价，所有医护人员原则上工资都跟职称挂钩，不再有绩效工资。而老百姓则参保（按照国家，地方，个人合资的形势），不同地方出资比例不同，报销比例也不同（我想还是跟地方财政有关）。现在乡镇医院都爆满，一个小小的感冒，老百姓要求住院（因为可以报账，更重要的是觉得自己用好药对身体会更好），去年甚至听说有个人因为感冒住院七天还不肯走，最后还要吊几瓶营养品才肯罢休。医生呢，现在没有绩效工资，反正也就是混日子，不出事就行了，也不会对病人说什么。实际上稍微有点常识的人都知道，滥用抗生素对身体副作用极大。从另外一个角度来说，如果这样耗费资源，国家地方财政也是很大的负担。
