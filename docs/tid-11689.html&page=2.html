xulimin84 发表于 2014-9-4 23:38 static/image/common/back.gif
既然这样为什么他们的排名会比内地的高呢

排名的参考因素和权重不一样
比如美国USnews给美国大学做综合排名，像“师生比”这样的数据占的权重就很大，于是私立学校的综合排名里面很占优势。像加州大学伯克利分校，出了几十个诺贝尔，科研能力逆天，但是在USnews的综合排名里面就很吃亏。
再比如，国内武书连做大学综合排名，他设置的参考值里面对学校的规模有很高要求，所以导致很多小而精的学校综合排名很差，比如北邮、上财这样的比较专的学校。
再给你个更直观的例子，西班牙有个机构也做过世界大学排名，其中有一个参考指标权重很大，说出来你可能不信，那就是在google里面该学校被搜索的次数，次数越多，该项得分越高。你自己想想吧，最后得出什么结果。

像英国的泰晤士报、QS做世界大学排名，很喜欢找一帮人给全世界各个大学打分，个人认为这样的参考主观因素太大太大。
而上海交大做的世界大学排名，基本参考的是客观数据，比如该校出多少SCI论文这样的硬指标


总之，排名参考的项目、权重五花八门，排出来的就天壤之别
