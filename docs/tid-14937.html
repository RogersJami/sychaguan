juyern 发表于 2015-11-15 11:40:18

上月初，我从某论坛遇到一个网友，协商之后，接手他的证券账户代为操作。
小弟我虽然2006年就接触股市，颇为痴迷，但是实战经验欠缺，眼高手低，却按捺不住经常在网上发帖夸夸其谈。
接手账户之后，虽然很是用心，但是技术太差，力不从心，在同期指数上涨16.8%的情况下没有实现盈利，当然暂时也没有亏损。
下面我想简要介绍一下投资人（姑且称之为T君）的几点情况：

juyern 发表于 2015-11-15 11:50:30

一 T君此前曾与朋友一共出资70多万，找了据称是在券商上班的操盘手，大概就是今年暴跌这段时间，亏损到只剩30多万。后来我告诉他，找人操作，一般要设置一个平仓线，超过这个限度，操作者要承担责任。T君开始还碍于面子啥的，后来才决定起诉操作者。

二 T君对于定金等全无概念，比较吝啬，要几经说明，据理力争，才好不容易讨得个最低限额。他的逻辑是，你必须先为我赚钱，我才给你钱。这个倒也无可厚非。

juyern 发表于 2015-11-15 12:06:02

三 一开始就要求跟上指数，那时候我还在轻仓买几股观察练手。

四 经常去网上找某大V或者**金融的言论，来质问一番。
五 一有下跌，立刻悲观失望，充满怀疑，貌似要中止合作的样子。
六 时不时要来对具体某笔操作进行评价，每天都盯着账户，还说觉得累。

juyern 发表于 2015-11-15 12:06:18

七 貌似（没有明说，但含义是）要求短线操作，踏准节奏；上涨时要超过指数，下跌是要不亏钱。
八 如果你哪天不操作，就可能会觉得你没有做事。
九 上涨觉得仓位轻，下跌觉得仓位重。
十 觉得资金少，希望满仓快进快出。
