既然都知道这种玩法损人还不利己，那还有什么可担心的呢？

尼赫鲁家族是看到了中美之间的矛盾，在世界经济危机不断加深的过程中无法缓和。印度这样的选择是因为眼下的印度无法工业化。
事情还得从英国殖民时期说起，英国殖民者改造印度的方法是通过改造印度农村基层治理为切入点的。当时有三项措施，第一项是文官下乡，英国把文官科层制度直接引入到了印度乡村，废除在农村广泛实行的“五老会”制度。第二是税制改革，从实物税收改为货币税收。第三是契约农制度，签订契约的农民 其近亲属负有无限连带责任 。第一项政策迫使外来文官不得不依附于城市资本家和高利贷者，后两者在农村大肆兼并土地。第二项政策，在英国货币当局的操纵下，印度农产品市场不堪一击。这两项政策的结果是印度在这一时期出现大规模失地农民。而第三项直接把失地农民变成了农奴。生不如死的农奴压低了大宗农产品的价格，但是这部分利益没有去搞工业化，而是装进了英国殖民者的腰包。换个角度看，印度的土地改革一百多年前就已经完成，只是剪刀差用来供养宗主国了。
印度选择非暴力方式获得独立后，尼赫鲁家族果断选择了手工业工业化，甘地则选择城市为农村服务工业为农业服务的田园牧歌式三农。其实两者是一回事。就是彻底将原先土地所有者的利益固化。而土地所有者们通过压低农产品价格获利的路径是英国殖民者设计好的。于是工业化所需的大量土地在土地畸形价格壁垒面前弱鸡了。
整个二十世纪七十年代印度农村人口大规模失业，面对危机印度启动了建立196万家农村企业的项目，试图在工业化和土地贵族利益之间找到平衡。但是手工业化的农村企业无法进入工业循环。八十年代农村失业人口达到1700万，而此期间年就业人口增长不足190万。到了2000年印度农村失业人口达1.2亿。赤贫的文盲大军终于和新世纪工业信息化的浪潮分道扬镳。
无法工业化的印度和推进一带一路的中国是没有妥协空间的。它和美国之间的利益也并非根深蒂固。但此时印度的精英们选择了站在美国一边。和韩国部署萨德处于同一目的——赌中国在和美国的战争中败下阵来，通过洗劫中国来取得工业化所须资本。
如果我们真的败了，那么“进军拉萨然后进军北京”就不仅仅是印度一厢情愿的豪言壮语。
