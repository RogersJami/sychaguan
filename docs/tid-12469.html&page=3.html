仁兄对天与其时与天气时和无所位而生其本无所本而生其位是怎么理解的？

{:3_41:}你这样问我，就像这个表情，我脸都青了。

缠说过，所有儒家典籍，包括《论语》《易经》本身，都是在论述论语的三个不亦。
论述这三个不亦，有个不变的目的，就是要实现圣人之道，就是共产主义。就是要一以贯之。
天时，是天与其时而与天其时，与天其时而天与其时的结合。
天与其时而与天其时，我理解就是正确的时候做正确的事情。首先，什么事正确的时候，什么是正确的事情，如何做？
套用缠的股票理论，正确的时候做正确的事情，要分以盈利为目的，还是以亏损为目的。以盈利为目的，就是底背驰的时候买，顶背驰的时候卖。以亏为目的反之。这是天与其时而与天其时。
与天其时而天与其时，这个更上一层，继续用缠的股票理论，就是用力量引导出走势，并不是控制，而是引导，四两拨千斤。

圣人之道是金字塔尖的话，三个不亦就是塔尖下一层。无所本而生其位无所位而生其本；不患无位，位所以立，就是再下一层。而客观现实就是最底层，一切从实际出发。

      无所本而生其位，无所谓而生其本。这个我觉得是比较好理解的，因为就是缠解释的那句“为政以德譬，如北辰居其所而众星共之。”就是把北极星确立后，其他的星星就能执持北极星的位置而相应展开。但是北极星不是先天就是北极星，而是刚刚有颗星具有那种性质，所以人为命名为北极星，人为定它做中心。
      那么我们可以看看实际的例子，就是虚数：i，i 的平方是 -1.那么可以说这个 i 就是人创造出来的数。
      看看一下这一篇，感受一下虚数的意义。
      http://www.ruanyifeng.com/blog/2012/09/imaginary_number.html
