与苏联的方式相对的，是美国对南方的支持方式，就是保持南朝鲜政权的基本生存，然后引导其进入美国主导的西方世界市场体系，靠着勤劳革命，逐步发展经济，改善人民生活。这样，在长达30年的时间里，朝鲜北方的人民生活水平是高于南方的。加上美国在南方的驻军，以及南方的军事独裁统治，所以在政治道德制高点的斗争中，北方是占了明显的上风的。现在韩国的偏左翼政党，还有和朝鲜的一些渊源。
但是，到了80年代中期以后，韩国在经济发展和人民生活水平方面，彻底超过了朝鲜。而且，又搞起了民主选举，就此扭转了政治斗争的态势。另一方面，苏联却走上了绝路。苏联解体后，朝鲜一下子断了外援，在经济基本盘上的劣势，一下子显示出来。就人口论，南方近五千万人口，北方两千两百万人口；就自然条件论，北方以山区为主，有少量矿产资源和一定的工业基础，南方以平原为主，又是沿海地区，只要美国接纳支持南方进入自己的市场体系，南方肯定是要比北方发展条件优越的。
当时朝鲜希望在意识形态上和自己一样孤立的中国，能够接过苏联对自己的包山包海的义务，朝鲜就可以就此认中国作老大，也许会从此死心塌地靠着中国混。但中国一方面不具备苏联那样的当冤大头的大心脏——在90年时，按照工农业总产值口径计算，朝鲜的总产值是高于辽宁省的。辽宁省是东北，也是中国重工业最密集的省，也是东北经济的半壁江山，人口是朝鲜的三倍。同时，中国也比较了美苏对朝鲜半岛南北两边的援助模式，决定借鉴美国援助南方的模式。
这样，从朝鲜的立场论，一方面中国不能支持自己迅速扭转对南方的经济竞赛和政治斗争的全面被动局面；另一方面，中国为了自己的利益，不和朝鲜方面协调，直接和韩国建交，属于一个大国对一个小兄弟的“致命的背叛”。在这之后，朝鲜实际上对中国已经不再有基本的信赖。

听风看雨通三统 发表于 2013-2-15 08:43:07

中国在挺过了上世纪末的国企改制的阵痛期之后，胡温组合上台后立即推出振兴东北老工业基地的规划。其中一项重要的内容，就是将朝鲜整合进以东北为核心的东北亚区域经济一体化过程，使朝鲜真正能立住脚。但在朝鲜看来，中国的这个安排，第一是出于在经济上统合朝鲜，最终在政治上彻底控制朝鲜的考虑；第二不可能支持朝鲜重新获得对韩国的经济竞赛和政治斗争的优势，也就不能支持朝鲜主导半岛自主和平统一的过程。所以，中国对这个规划很热心，却不能得到朝鲜方面的积极回应。朝鲜遂决定自己坚持独立自主地搞起核武器，逼迫美国坐到和朝鲜一对一对话的台面上，自己可以左右逢源，两头通吃，在韩国的亲北方政党长期执政的背景下，能够对半岛自主和平统一发挥更大的作用。
中国的应对之道，就是让美国认识到，越过自己和朝鲜直接对话，将遭遇自己不能承受的战略后果，使美国意兴阑珊。然后，就是在保证朝鲜国家安全和经济勉强维持最低水平运转的规模上来援助朝鲜，让它饿得半死不活。最后，就是一次次由总书记和总理出面，耐心说服和等待朝鲜，接受中国的规划。再有，就是支持韩国的大国家党背后的财团在中国大陆市场发展，控制住南方的经济命脉，支持大国家党上台执政且连任，回头追究那些亲北方的领导人的贪腐行为。到了09年，卢武铉被逼自杀，金大中被穷追猛打，朝鲜扶植的左派政党，可以说元气大伤。这才有了今年的朴槿惠顺利当选。
对中国来讲，对西方要讲一个基本逻辑，就是赖斯在联合国讲的“不能让朝鲜因其行为受到奖赏”；在韩国，就是要支持和挟持保守派上台，终结、逆转阳光政策的成果。这样，朝鲜的出路被中国堵死，就只能走上中国规划的道路。
所以说，在枝节问题上，是朝鲜不断为中国制造麻烦。但在战略层面上，中国有能力也有决心，将朝鲜制造的麻烦，反弹到朝鲜自身，堵死朝鲜别的出路，使朝鲜要么半死不活，要么逐步走上中国规划的道路。朝鲜事实上能够选择的空间，其实就是这两样之间选一样。朝鲜很清楚，一旦在中国指导下改革开放，就会被中国彻底控制。所以，只要认为还有中美日之间的矛盾可以被它利用，就不会甘心对中国就范的。所以，大的棋局是中国把控的。而具体的弈棋过程，双方的手段也都亮出来了，后边就是一次次重复而已。直到朝鲜彻底死心，才能真正走上中国规划的道路。

何新对朝鲜核问题的十二点评论：
1、有人认为，朝鲜拥有核武器是为了与美国对抗，下一步问题就是美国何时打朝鲜。这是误判当前新的世界形势了。

2、朝鲜核爆成功，美国对朝其实只是故作姿态虚张声势。短期内美国绝对不会主动攻击朝鲜。美国实际已经默默接受朝鲜拥有核武器的现实。美国深知，朝鲜的原子弹眼前不是美国的麻烦，倒是中国的麻烦。
3、除非朝鲜今后对外公然销售核武器——朝鲜无法以原子弹来讹诈美国。
4、朝鲜今后会对中国进一步予取予求，制造无穷麻烦，美国乐得在旁边看热闹。
5、中国与朝鲜没有共同意识形态也没有共同社会制度。朝鲜不是什么社会主义体制。朝鲜意识形态是金家世袭主体的封建王族意识形态，任何试图把金家意识形态和朝鲜模式引进中国的人，都可以看做要祸害中国人的内奸和汉奸。
［中国曾经为朝鲜付出巨大的民族牺牲，但是在朝鲜的战争纪念馆中基本上没有中国志愿军的位置。在朝鲜的历史表述中，不是中国援救了朝鲜而是朝鲜援救了中国，朝鲜是中国的恩人。］

6、中国应考虑彻底中断对朝经济援助。

如朝鲜今后加大骚扰中国，一定会得到共济会更多资助。
