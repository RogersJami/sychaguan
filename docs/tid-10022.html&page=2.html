操作的级别
经过最近对走势的分析，我终于发现最适合我操作的级别为5f级别，同时利用1f级别的中枢进行对冲。
    为什么不是30f级别甚至日线级别，答：从585上来的走势才刚刚形成一个日线级别的中枢，而很多最近一两个月大涨的股票起上涨的部分就是一个5f级别比如300245,300246,300247,300037，如果按照30f（包括以上）的级别操作，那么运气好的像300037,300245的还能赚一部分，像300246这种的，就光剩下坐电梯了。为什么不是1f级别呢？很简单，因为1f级别你买进去了卖点你不一定出的来，最明显的例子就是从3月18日，前一个线段结束后的走势。
      
    站在马后炮的角度上来说，对于这个中枢从形成到围绕中枢的震荡，我们应该的操作应该是保持仓位不变的情况下在1f级别的买卖点来回对冲。3月18,19日的就不说了，昨天开盘后的急涨我们未必能抢得到，但是大涨到27.98，并且先后2此上冲不能突破27.98，量能萎缩，并且大盘（创业板指）开始跳水时我们完全可以出来，然后在尾盘26.50附近将底仓接回来。同理今天早上24.50的时候没有必要买，但是当走势上破24.45达到24.67时就可以短差一把，然后大涨到27以后的两次反弹上不去后再把底仓出掉，即使是这样滞后的操作，两天时间也可以在保证股数不变的前提下获得超过10%的短差受益。事实上，只要不是3买后背驰或者形成3卖这个过程可以一直继续下去，而不是像我这几天一样明知道上面要形成卖点，下面要形成买点，但是因为T+1的问题不敢进去只能过嘴瘾。

  创业板走势分析（完）
  30f,5f,1f联立起来看，30f上因为24对23的破坏是第一种，而3月18日至3月21日的下跌只构成30f上的一笔且没有破24的低点那么今天10:30开始的这一笔走势很重要，如果先上破1493.39则说明24这个3买后的走势还没结束，如果先下破1378.89则说明3买的线段已经结束，此三买后的走势不但没能形成非背驰类的上涨，甚至连1571都没达到，那么必然要返回前一个30f中枢。当然，考虑到前一个30f中枢已经形成中枢级别的扩张，所以更重要的是这个日线级别的3买卖，以及3买出现的话必然出现的3买后的背驰或者3买后非背驰的上涨。
    而5f级别上今天10:25开始的反弹以第二种方式破坏了前一条线段，所以在形成特征序列的顶分型之前不能确认从1493开始的下跌结束。
    至于1f级别图上今天开始的反弹还不能确认走完一个线段。因此3个级别联立，一旦1f的向上线段结束就要密切关注之后的下跌，只要该走势使得今天5f级别向上走势形成特征序列的顶分型那么意味着一个5f级别的向下线段开始了，而这条线段只要跌破1378.89就意味着30f上3买后的向上段结束，那么其后的走势必然对应着30f级别的中枢3，最少将跌到该中枢的高点1369.12，但是该中枢的上边为1293.94。换句话说，在30f这个级别来看目前处在一个刀口舔血的位置。另一方面因为日线级别中枢的形成，所以除非形成一个日线级别的3卖否则必然会在日线中枢（1194-1219）这个区间上下来回震荡，因此对于中枢震荡的熟练操作将是今后一段时间挣钱的最佳方式而非去年那种单边上涨。

上证指数分析（一）

日线上。因为缠中说禅先生将998点定义为20年大牛市的起点，所以从998开始划线段。

从998到6124以及从6124到1664在日线上只有一条线段。其中6124的破坏时第二种，1664为第一种。其后分别在3478,2319,3186,2661,3067,以及1849形成线段，共8条。8月16日光大事件站在线段级别而言很重要，因为如果将这次乌龙事件当做一次日线笔之间的波动而忽略，那么9月12日2270开始的向下走势只对1849开始的线段构成第二种破坏，那么在走势形成日线级别的特征序列底分型之前并不能说从1849开始的向上线段结束了，但是如果不忽略8月16日的走势那么从1849开始的向上线段就结束于9月12日2270点。

如果站在缠中说禅先生预言的20年大牛市的角度来考虑那么只考虑向下的z字段，那么1234构成一个日线级别中枢，区间为（2319-3478）；5,6,7,8都看做是围绕中枢的震荡，那么8月16日就很重要，因为不忽略8月16日那么2270就构成了中枢（2319-3478）的第三类卖点，那其后最好的情况也是盘整背驰（这个倒是100%成立，毕竟指数除非跌成负的不然不可能比6124下来的线段力度更大了），那么就意味着要形成一个周线级别的中枢，后面的问题就是周线级别的3买卖。当然即使8月16日被忽略了，只要今后的走势在日线上构成特征序列的底分型依然可以确认2270为线段9且为中枢（2319-3478）的三卖。

站在从6124开始的下跌考虑那么只考虑上涨的z字段，则日线中枢区间为（2319-3186），3卖的问题同上，一旦指数跌破1664则意味着盘整背驰，那么除了所谓的奔走型走势指数至少要反弹到该中枢的下沿即2319。

另一方面按照结合律可以把4567看做一个日线级别的中枢，那么比较中枢两端的走势在周线图上可以看出3478到2319的下跌其力度明显大于7-8的下跌，那么就意味着盘整背驰，这样也就意味着8开始的走势理应返回到4567构成的中枢（2661-3067）区间，即使是奔走型的走势也应该触及围绕该日线级别中枢震荡的低点即2319，换句话说8月16日的走势必须忽略，1849开始的上涨不允许结束否者就是理论错了...（还是我错了呢？）
