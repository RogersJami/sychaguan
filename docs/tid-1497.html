人物素描系列之.骗子一二三

电话
A:"请问是XXX吗？"
B:"是的，请问哪位？"
A:"我是国泰君安证券公司的客户经理，想向您推荐一只股票，XXXX....."
B:"打断一下，你应该先问我是不是你们公司的客户，如果不是，向我推荐股票你们也赚不了佣金。"
A:"抱歉，我忘了问，您是我公司的客户吗？"
B:"不是，但如果你给我万一的费率我就是。"
A:"......"

地铁
A:"小姐，需要美容吗？"
B:"谢谢不需要。"
A:"需要减肥吗？"
B:"不需要，谢谢。"
A:"那你需要兼职吗？"
B:"我需要安安静静等车。"
A:"我真的可以帮助你的！"
B:"我真的没有什么需要你帮助的，你下一句是不是要说：你不要把自己包那么紧嘛！"
A:"？？？？"
B:"我天天转车，天天被你们骚扰，你们经典的5句台词我都会背了，有点创新好不好？"
A:"......"

现在骗子太多，竞争激烈，骗局漏洞百出，员工素质下降，一点都不用心表演:(  记忆中有一次被骗得心甘情愿，到现在还回味悠长。

那是刚进大学时，有次到超市买东西，天空下着小雨，有一穿白衬衣的男生在超市门口，低垂着眼静静站着，平时走路从不看人的我，居然留意到这个男生，没办法，超市杂乱的背景，车水马龙的路人，都在突出这位男生的干净与安静，此第一招；我虽然看了他一眼，但也没有继续关注，准备进入超市（神经太粗），那男生在我靠近时，突然抬头对我欲言又止，此第二、三招；他站门口，我总得路过他，不可否认，那男生长得很帅，一位落寞的帅哥近距离对你欲言又止，神经再粗也得有点反应，所以我开口问他有什么事吗，他很不好意思地小声说，自己是北大的学生【名校】，来我们学校找朋友玩没找到，钱包丢了【缘由】，希望我借他点钱坐公交车回宾馆【小钱】，于是我想都没想，很豪爽地给了他－10元钱^_^，帅哥拿着钱连声说谢谢，到此时，我一点都不觉得被骗，感觉是在助人为乐，可是偏偏有下半集 : (

...如果这时我说要去喝水，会不会有人抛砖头？被骗的糗事真不想说诶...

卿本佳人，奈何做贼，我一直认为那位骗子帅哥做电影明星也绰绰有余，可惜没有生在快男超女的时代。发现他是骗子，是在我抱着做了一次好事，高高兴兴准备进入超市时，旁边突然冒出某男挡住我说：“10元钱不够，我们还要吃饭！”边说着边轻巧地抽出我手中的50元，我没带钱包，手里就揣着50元+10元两张钞票，我当时就愣了，傻不溜秋地说，我也要买东西吃啊，某男一边把50元放包里，一边把一旁帅哥手中的10元钱拿给我，说，你拿这个买吃的就好了，我看看他们，再看看手上的钱，脑子还是浆糊，当年的50元是我10天的生活费，帅哥再帅也不至于让我愿意付50元观赏；某男看出我的不乐意，不知从什么地方抽出一个本子和一只铅笔，说，你写下联系方式，我们会还你钱的，帅哥笑眯眯地用手掌抬着本子让我留下大名...

我有气无力地在纸上写下“不用还了，你们好自为之吧”

谁被饿了好几顿都会牢牢记住这个教训的，所以敬业和不敬业的两类骗子我都讨厌。

小飕风风 发表于 2012-4-10 13:58:55

类似得，我有一次差点被骗。

当时刚军训结束，和同学一起在校外买些生活用品。快到校门口时，见到一对衣着很朴素的大爷大娘。他们看到我们，就走上前来，说从外省来到这里看儿子，但是不幸钱被偷了，而且联系不到儿子，饿的很，能否给两块钱买个馒头。
我很同情他们，但是看看口袋里没有1，2块领钱，只有5块钱。5块钱当时够我一天的伙食费，心里还是颇为纠结。不给吧，他们看着很可怜；给了吧，5块钱对我来说有点多了。犹豫了一下，还是同情心占了上风。
就在我掏出钱准备给这两位大爷大娘时，我被同学不由分说的拉走了。为此我心里内疚了好几天。

当然，亮点不在这里。

过了大概一个月，我到另外一个学校去看高中同学，结果在那里又遇到这对大爷大娘。他们大概没有认出我，上来对我又重复了一个月前的说辞。我没好气的说，你们找了一个月儿子，还没有找到么？

类似这样说自己很饿，需要几块钱领钱买吃的的事情，我后来遇到过至少十数次，有在本地遇上的，也有出差到别的城市遇到的。我的对策一律是：饿了是吧，我给你买个面包（或者给你买碗面）。可惜的是，没有一个人愿意和我去吃东西，哪怕饭馆只不过在百米之遥。

然后203 发表于 2012-4-11 16:57:41

碰到过骗子两次，路遇要帮忙啥的不算。

一次是大一暑假回家的车上。在新华路买票先上车后，等客期间，一个小老头上车打量了一下，在我旁边坐下来，拿手机（当时比较稀罕）低低的说：“……灰色的快巴”。我近视很厉害，但听力不错，无意间听到了。

车上路后，先要经过市内，做前排的两个人开始吵架，嫌贫爱富之类互相取笑，我注意到这两个人都是外地话，心生疑窦：这是要下乡的车，江浙外地人到乡下干神马？

第三步，吵架的两个人开始玩牌，出手还很大，互相吹牛比阔。后排的两个人，连最开始那个小老头开始骚动，鼓吹大家参与。这时我明白了，这就是团伙骗子。看看车上的人，可能我是明白得最晚的，因为大家都不做声，司机和售票员绷着脸看着。
最后，我对身边一个学生摸样的同乡说，这就是一伙的，咱们车上的都是老乡，他们几个外地人想干什么？这话没有压低声音，那几个人肯定听到了，车过收费站之前，让司机停车都下去了。
---------------------------------------------------------
这个团伙手法老套，有分工，选择目标比较失败，配合也缺乏默契。
-------------------------------------------------------

另一次是工作之后，某天下班比较晚，路过市政府后面的围墙时，一个高大，漂亮的男子拦住我，说是高速路上出车祸了，要赶紧到客户所在某酒店去，出示工作证、身份证，大名曰吴天祥（武汉市没人不知道这个名字吧）。

先问我怎么走，这个没问题；再说手机欠费了，借我手机用一下，这个也好说；然后说身上的钱物和车辆都给被撞的人扣留了，希望借钱打车到某酒店去。

我对此人第一印象不错，但是心生警惕，而且我身上也没带多少钱，非常坦然的掏出仅有的20元钱，岂料此人十分气愤的拒绝，并认为我严重伤害其自尊……就此告别。
----------------------------------
这家伙外形很好，谈吐也不错，套近乎的手法很熟练。无奈选错了目标，当然也可能是拿我这种无风险的当一回陪练。
-----------------------------------
