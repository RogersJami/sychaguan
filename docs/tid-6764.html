原 创: 四处奔波忙助学 ( 下 )


       离开马石小学时，太阳已经西斜，眼看这时间紧迫，若不抓紧的话，今儿个可得摸着门进屋了。可去的那一条公路，因年久失修，让雨水淘得尽是些坑坑洼洼，不是嘿大的一筒眼，便是些成片的小窟窿，一路上不太好走，速度也不敢加快，尽说路程不远，也花了差不多半小时，一个个颠簸得头昏脑胀，这才赶到凌津滩中学校内。
　　学校领导与有关的老师，早已守候在办公室里，等候我们到来。
　　见面作了个介绍，寒暄几句，话头直奔了主题。不消说，依旧是老一套，直接与救助学生见面聊天扯闲谈，从中了解其家庭的具体情况与事先掌握的情况相核对。
　　左江杏，女，13岁，七年级学生，家住仙人溪村前进组。
　　女孩的父亲因癌症病故，生前为治病全家举债只想治好父亲的病，结果是，鸡飞蛋打一场空！至今欠下债务六万出头，多病的母亲不仅承担着抚养孩子的重担，还得赡养照顾双目失明的奶奶。由此可见，这一家子往后的前景又如何得了呢？！
　　吴高缘，男，14岁，八年级学生，家住岩巴嘴村青龙组。
　　这个伢儿，长得端端正正，相貌堂堂，生成一个机灵相。可问起他的身世，又不禁令人感叹不已。
　　吴高缘自幼丧父，几岁时便失却父爱，年纪轻轻的娘守不住孤苦零丁与那一份无奈，父亲‘走’后，便改嫁出了门，从此，娘儿母子，各是各的一家。
　　娘亲改嫁之后，吴高缘成了个不是孤儿的‘孤儿’，更胜似那无亲无故的‘孤儿’。政策明文规定，失去双亲的孤儿，政府每月有六百块钱的救助。吴高缘失去父亲后，但还有个名义上的母亲，所以，按照政策规定，他算不上是孤儿，可实际境况呢，又要比真正的孤儿还差了些许！如老话所道：他没有爹爱，还有娘痛，可娘却另有个家庭，一点儿也关照不到他身上，实乃有其名，无其实，枉背了“母爱“这一个虚名。有娘的孩子无娘痛呵！小小年纪，走投无路，只好跟随年迈苍苍的爷爷与奶奶。
　　好在穷人的孩子早当家，家境不好，艰难环境能磨炼人的意志。
　　吴高缘自小丧父，懂事得更早更为成熟，深知一饭一粥当来之不易！点点滴滴皆须珍惜！更明白爷爷奶奶含辛茹苦深藏在内心底里对自己的那一番期望，发蒙之后读书学习加倍的勤奋努力，幼小的心灵中时时刻刻怀抱着要用优异成绩回报辛辛苦苦抚养自己长大成人的俩位长辈。
　　我们与吴高缘的班主任刘老师在私下交谈中了解到，吴高缘进入初中以来，学习成绩一直处于班级的前几名，多次联考，在联校始终也是“前十名”之内。这是一颗读书的好苗子！倘若有好心人关怀他，资助他，想必吴高缘会争气赌气，更加发奋地读书，志存高远，一定会成为出类拔萃的学习尖子！成就他的梦想！我询问过他，有志气考取桃源一中啵？他斩钉截铁地回答道：有！请相信我！
　　我深信他！也希望有眼光的爱心人士能关怀他，给予他温暖与资助！帮助一个志存高远的孩子实现自己的崇高理想，这种大义善举，相信必定能盖过建造一座七级浮屠！！！
　　高 娟，女，15岁，九年级学生，家住马石村七组。
　　高 娟的父母已离异，她跟随父亲生活，父亲在家务农为生。家境不太好，很困难。
　　周国庆，男，9 岁，二年级学生，家住凌津滩居委会。2006年，周国庆父母离异，母亲从此改嫁他乡，父亲也外出不归，去向不明，杳无音信。可怜的孩子，一直跟随体弱多病的爷爷，如今爷爷逐年老去，他还很弱小，才九岁，今后的日子蛮长，摆在他面前的路还遥远得很，何去何从，谁又能说得清楚呢！
　　李 攀，男，14岁，七年级学生，家住一甲城村利民组。
　　该生七岁时，母亲不幸亡故，很小便失去母爱。父亲为着生活，常年在外打工，因无人照顾生活起居，一直寄养在姑姑家中。
　　刘伟发，男，13岁，七年级学生，家住岩巴嘴村青龙二组。
　　他的父亲因患肺癌亡故，母亲又是个弱智，如今只剩下母子俩相依为命，其学费与生活费全部依靠亲戚与乡邻们资助，还有位九十高龄的奶奶要赡养照顾，家中事务由村里管理。
　　张佳芳， 16岁，九年级学生，眼看就要进入高中了，家住凌津滩村二组。
　　张佳芳，一个非常好听的名字，也是一个值得关注与同情的女孩。她的名字中有个佳字，佳的字面意思，表示非常之好或顶好，可现实景况对她来说，却是非常的糟糕！还有一个多月，即将升入高中，能继续读书，是她的梦想，也是她改变自己一生命运的重大转机。可她的家境条件却不允许作如此的非分之想！
　　二岁时，她的父亲因病亡故，之后，母亲又远走高飞去，改嫁了他乡，从此没有一点音信。她成为不是孤儿的‘孤儿’，却得不到一点点温暖与关怀。她没有了父母，只能寄人篱下，跟随伯父，伯母一家生活。可伯父，伯母的家境也十分窘迫，生活艰难。在这个一切向‘钱’看的年代，伯父，伯母能收养她，抚育她，显然已超越了一般的亲情，作为旁观者的我们不能苛求于任何人。显然，这小女孩命运不济，乃是天命的使然。在此，我只想说一声，能不能伸出你的援手，拉她一把呢，可怜的小女孩啊！
　　夜色苍茫中，一群人又匆匆登上了车子，奔往回程。
