杂谈印度

本帖最后由 gentlsteve 于 2014-6-4 14:57 编辑

承蒙大家加分，就开个贴谈谈印度。只谈自己经历的，熟悉的，不下笼统的大结论。   


1.      印度司法在我看来纯粹是个笑话。号称“有完整的大英帝国法律体系”，判个案子，没4-5年下不来。法院里积压的案子据说全部判完要10年。我一朋友有天收到一法院判决，他5年前起诉的一个欺诈案终于胜诉了！（执行还早呢）还有个好玩的事：印度国会里面有30%（数字记不太清了）的议员有诉讼在身，其中竟然多数是谋杀之类的刑事罪起诉。

2.      腐败遍地都是。一朋友有急事闯红灯，掏出200卢比，对过来的警察说：我知道一般都是给个100卢比就可以了。今天我有急事，给你200.警察立马放人。听来的故事：一人去政府办事，送上几本自己写的英文书，里面夹上钱。第二天去，以为事肯定能办成了。没想到办事的说：你的书写的很好。我同事们也想要几本看看。几乎没有一个印度政治人物被判贪污受贿的。

3.      俄印关系之密切，可能超过大家想象。好像从尼赫鲁开始，印苏关系就很密切。印度过去也一直是计划经济。新上台的莫迪，亲俄远远超过亲美
      
4.      素质啊，素质。孟买恐怖袭击那会正在孟买。在酒店里看到电视上特种部队的行动（比如：垫起角尖往墙里乱扫），只有一声叹息：别说咱这个鹰那个豹的，就是比咱一般武警也差太多了吧？难怪孟买反恐司令在赶赴现场途中，整车人被2-3个恐怖分子拦下全毙!（几个人从火车站一路杀过来，正好撞上司令的吉普车）
  
5.      印度教，我觉得不能算一个严格意义上的宗教，有点像咱的山海经或神话故事。叫的上名字的神有2千多个，也没有一个完整的理论、哲学体系。但是，印度文化对海外印度人的影响（约束）远比中华文化对海外中国人的影响大。原因有待专家研究
佛教在印度已经式微。其一支奢那教到是还有不少信众，很多是印度教中低种姓的人改宗过来的。印度教低种姓的，改信基督教，伊斯兰的也有不少。
印度国内的穆斯林人口应该是排世界第一（2亿），起码也是第二。这注定它不会和美国走的很近。印度就是个大杂烩，北方人和巴基斯坦人、中亚人一个种，白；南方的就跟非洲黑人差不多了。长得白的，也就在电影电视5星酒店里见见，满大街都是黑和半黑的。
   
有几个印度朋友，不说人家点好话也不好意思：
        
1.      家庭观念重，道德观保守。多数印度家庭是3代同堂，兄弟和未婚姐妹与父母住一起。我一客户，哥俩各开一个公司，不但都和父母一起住，收入都上缴父母。还有一个在孟买工作的MBA，老婆是妈妈从老家找的，小伙结婚前没见过老婆（他说我妈怎么也不会害我啊），而且结婚前是处男。有次宝莱坞一个美女大明星和她的有钱丈夫接受西方一个电视台采访。主持人惊讶地问：你们为什么还和父母住在一起？女明星反应很快，也故做惊讶：啊，你们为什么不和父母住在一起？      
2.      不爱面子，不虚荣。LV之类的奢侈品在印度销量远远比不过中国。人家的妇女都是穿民族服装，有钱也是往身上招呼金镯子，金项链。男人也不大穿名牌，重要场合，穿裁缝做的衣服也很贵。人家不是小气，在海外景点常常碰到印度人，人家都是一大家子7、8口，十几口一起出来玩。去过几个印度工厂，什么大草坪，大理石、马赛克墙面这类装饰很少，工作间也很紧凑。我看见过一个大学，外面水泥墙面甚至于连白漆都没刷。
