一直是穷人 发表于 2013-2-28 14:17 static/image/common/back.gif
说到日本，我认为，日本的出路只有一条：成为中国的一个省。其他都是死路。

我不论述，也不打嘴仗。

东亚各国在历史上已经形成了很好的整合基础，文化同化程度很高。世界大势浩浩汤汤，更高程度的同一化是必然趋势。

说“日本必将成为中国的一个省”比较气人，换种说法就好多了：

实际上，我一贯认为，现在的世界局势，正如历史上兔子的春秋战国之春秋阶段。春秋五霸（既是历史的当时的，又是现在的近代以来的），各领一时风骚；然而在现有条件下，意图建立有效的全球政府（在历史上的春秋时代，就是有效的统一的中央集权的郡县制政府），仍然是不可能完成的任务（Mission Impossible 是我喜欢的著名电影），说直白点就是——赔本买卖，只能落一身骚。因此只能称个霸啊什么的，面子上光鲜一把，顺便搞点抢劫啥的。

因此理想主义者们真正正确的做法，就是推动世界加速进入战国七雄并立的时代（兔子的官话的提法就是推动世界多极化趋势）。这是时代的召唤，只有多极并立，世界才能更加稳固；各地区才能深度的发展。而各地区的深度发展，各大国的良性竞争，才能更大的解放生产力；而更高度的生产力（在内涵和规模上都远超人类现有的）和人类自身的更高度发展（进化），才可能让人类具备在数十亿（以后或许上百亿）人口规模上全球统一的物质基础。

无论在哪个年代，春秋时代都远比战国时代更混乱更野蛮更不人道，发展成果都更多的用于战争、杀戮而不是建设发展和人类幸福。

人权人道也罢，自由民主也罢，主权自主也罢，发展是硬道理也罢，都应当团结到推动多极化趋势这个旗帜下面来，积极推动这个趋势的发展壮大，这应当是各类价值观的共同追求，这也是各类价值观互利共赢的一个利益共同点所在。

按照主要文化体的分布，推动各大地区深度整合，形成新时代的战国七雄局面，显然是最符合社会发展规律的未来发展趋势。在这个大时代背景下，什么“日本成为中国的一个省”就不再仅仅是“中国人的复仇”这样狭隘的东西了。

各国不应当把精力放在什么争霸上，而应当把主要精力放在如何踏踏实实推动落后地区的实实在在的发展上，哪怕只是一点一滴，最终也是人类作为一个整体发展进步的重要组成部分，也是踏踏实实、实实在在的历史性的进步。

推动全人类享有均等的义务教育、基本医疗服务和养老保障等，推动全人类的高度发展，为人类在伟大的大航海时代之后、更伟大的太空时代的出发奠定基础。

人总是要有点精神、有点追求的，否则，人不会幸福。
