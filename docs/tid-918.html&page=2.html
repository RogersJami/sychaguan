fleurrriste 发表于 2012-3-12 21:22:06

很多时候，我在想，那种我们觉得自己长期抑郁但却又不够病理性抑郁症的情况，其实是缺乏解决某些问题的能力，这些问题往往是人际上的处理问题。既然缺乏这些能力，那就会不断在碰到类似的人际交往问题中导向失败，以至于不得不长期抑郁。
在这种情况下，重塑性格与思维方式，釜底抽薪，其实是一个可行的选择。只不过需要凑齐很多条件，以很强的意志力去促成。

当然，更糟糕的情况是，由于这种能力缺乏造成的问题已经形成无法挽回的后果，那这个后果本身就变成了症结的原因，那就等于在抑郁症的海洋里钉了根锚，只能想办法开解自己了。
所以说，最好的办法是趁年轻，历史底子尚薄，改造自己动静比较小的时候赶快把问题解决掉。哪怕可能不成功，也能够更加了解自己的界限在哪里，也就比较好平静地接受而不至于过度抑郁了。

趁年轻努力折腾自己吧

cjf 发表于 2012-3-12 23:19 static/image/common/back.gif
意志力和自控力差不多是一回事，爆发简单，收场难。

我和你说的不是一回事。你依然沉浸在情绪里面。
把它当做一个工作任务来完成，忘掉你自己，一心想着怎么去达成，这样你应该想的就是各种实施步骤方法，根本就不会想什么难不难的问题。你说的是一种情绪的宣泄，前因后果考虑得少才会难以收场。我说的则是策划政变，通过精心设计的小型爆破手段把原来的主导逻辑颠覆掉。

怎样才能颠覆这一种导向失败和抑郁的逻辑？按照我之前说的那种情况，就应该去培养自己所缺乏的那些能力。
如何去实现这个目标？为自己创造一个原本逻辑无法行得通的环境并想办法战胜它，在这个过程中锻炼自己，并培养一些新的思维方式。
如何创造一个原本逻辑无法行得通的环境？有很多种方式，针对自己的问题，尝试一些完全违反自己一贯做法的方式，打破平时与环境互动的逻辑。
缺乏意志力去推动这种自救行为？根据自己想要创造的环境，寻找一个合适的机会，引爆一个推动点，让自己全面陷入那种环境，不再有逃避的权利。
如何收场？根据具体情况仔细衡量和评估每一个角度，对周围的人采取必要的隔离或寻求支持手段，或者换一个新的环境。考虑好可能的风险和撤退方式。

事实上这种方法是可行的，优点在于效果比较彻底，对整个人生都有益，缺点在于难度巨大，要求比较高，风险控制困难。在这里只是提供一种比较反常规的思路，有人愿意不妨试一试，但是不保证后果。
