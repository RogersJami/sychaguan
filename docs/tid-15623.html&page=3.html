gentlsteve 发表于 2016-9-19 11:19 static/image/common/back.gif
诸侯经济只是产能过剩的原因之一。

产能过剩，最根本的原因是同质化竞争。在国内大干快上的局面下，抄袭 ...
为什么会有同质化竞争？别忘了中国改开只有三十多年。英国工业革命原始积累期，哪个有核心竞争力。马克思《资本论》说“无产阶级是劳动力与生产资料的分离”“资本家通过剥削工人剩余价值实现盈利”对应的就是那个时期。

你以为地方政府傻呀？以云南个旧为例，云南锡业集团是云南省国资委控制的股份公司，原来在个旧负担数万（数据不详）退休工人工资、云锡职工医院、两所中学两所小学，其他还有很多不熟悉就不说了，这些包袱个旧市政府统统拿过来，为的是让企业减负好进行产业转型。除了这些，地税负缴纳五六年了，中央补贴个旧的资金很大一部分让云锡吃了。云锡集团倒了个旧也就完蛋了，这就是越大越不能死的典型例子。供给侧难就难在这些地方，地方政府要吃饭，企业也要吃饭，落后产能辐射圈内的相关从业者要吃饭，乱了粮就会出人命。供给侧的思路是淘汰落后，补贴优质产能，在结构上调整产业结构，结构调整完成后用优质产能的盈利通过财政转移的方式补贴落后产能完成“部分”产业升级。落后产能的转型非常难，以我接触到的类似企业看，几十亿丢下去能冒个泡就不错了，不是企业不想转，高利润的行业谁都想进入，但落后产能没技术、没人才、企业结构僵化、老弱病残多的毛病不是一天两天能够扭转的。关键是资金投入进去，还没用在刀刃上就被一张张吃饭的嘴吃干净了。
