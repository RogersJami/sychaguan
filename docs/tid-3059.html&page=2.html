保钓，无论那些香港人是何动机，从政府层面来讲，首先应该掌握主导权，没有主导权，先就输了一招。掌握了主导权，然后在明里或暗里支持、引导。政府有些事可以做，不可以说。这很多人都可以理解，但一味忍让，只能导致周边那些宵小得寸进尺，更加贪得无厌。
日本这个国家，一向是欺软怕硬。它是希望中国和美国干起来，从中得利；美国是希望日本和中国干起来，它可以重振雄风。其实根本的一点是，这两家都不希望自己直接卷入冲突，落个鹬蚌相争渔翁得利的下场。而这两个国家，目前都离不开中国（当然中国也离不开这两个国家）。严重的冲突是不太会发生的。韩国对日本人的心理研究得比较透，所以敢直接了当地去争地盘，日本人反而只敢动嘴不敢动手。中国有这个实力，本来这是一个非常好的机会，韩国带了头，中国如果跟进的话，日本人的第一反应会是指责政府四处出击，做了错误的决策，而不是将指责矛头对准中国。但某些人死抱着“维稳”的心态，不敢去搏一把。这种心态，反而给日本人利用了。其实中国完全可以利用日本的心态，主动做一个局，将日本自卫队的军舰没收，让日本人来要人，逼迫日本人在刺刀下认输。至于美国的反应，战术上重视，做好最坏的各种准备和应对预案，从战略上则可以藐视，从美国不断挑唆中国周边小国的动作来看，美国人根本没有做好亲自出手的打算，再说，即使美国出手，只要中国做好切断补给线的准备，美国的招数有限。
宣传爱国主义的目的，在于凝聚民心，以有利于国家长期稳定的发展。钓鱼岛，日本人敢在中国的领土上执法，中国人就应该用武力保卫国土。可能短时间会有一些经济损失，但是长久来看，打出一拳去，免得百拳来。这也是中国突破美日安保条约的一个很好的机会。现在，政府没有把握好保钓的主导权，那么挨指责的应该是相关部门的失职。这种情况下，如果反过来指责“保钓”的人，老百姓会怎么想，怎么讲？反正我接触到的基层老百姓中，没有听到称赞政府理性的话，都在说政府太软，然后就是对腐败的感慨。凭心而论，我也不赞成现在这种形势下保钓，拖一拖等实力强大了更好办。但事情发生了，就应该随机应变。目前，经济前景不乐观，腐败现象丛生，不少国人的心态很偏激，在这种情况下写这种自相指责的文章，无意中是将政府和民心对立起来。而民心可聚可用不可失。

8月24日，日中新闻社社长韩晓清于曾在《日中新闻》报上发表了一篇标题为“认真反思日中关系，冷静评判香港保钓者行动，香港保钓者登陆钓鱼岛，是爱国，还是害国？”的文章。该文在网民中引起了巨大的反响。一些读者对文中论点提出批评与驳斥。对此，韩晓清特向环球网来信进行回应。她首先向所有保钓人士和团体表示歉意，并承认将保钓行动归为“害国”行动的观点是错误的。同时，她认为一些网友对文章“断章取义”、“无限上纲”，对其进行人身辱骂，这让她感到非常困惑和痛心。她特别强调说，自己是中国籍，是一名“爱国者”。

韩晓清在回应中说，文章为匆忙写就，在重读原文之后，“发现文章中确实有些用词欠妥，有些观点过激，有些论述不够全面。特别是把香港、台湾、大陆保钓者登陆钓鱼岛的行动说成是害国行动的观点显然是错误的，有失公正的。”她在信中对保钓人士表达歉意，写道：“在这里向所有保钓人士和团体表示深深地歉意。同时对这次登陆钓鱼岛的14位勇士的爱国壮举和勇敢的精神表示钦佩和敬意。”

同时，韩晓清对其观点补充说，通过和一些日本大企业家的交谈获悉，一些日本企业对中日关系的未来信心不足，因此没有选择将中国作为其首选海外转移目的地。韩晓清表示，是这样的事例让她得出了保钓行动“不是爱国，而是害国”的偏激观点。

韩晓清进一步解释称，最近日本企业面临着第三次海外大转移，这些企业都是相当高端的高科技产业。她认为“这本来应该是中国吸收日本高端科学技术，引进先进管理理论和方法，实现中国现有技术和产品更新换代的绝佳时机。”如果中国营造出稳定的投资环境和社会环境，使日本高技术含量、高附加值的产业移至中国，这对中国的发展意义非同寻常。韩晓清称，基于这种考虑，她希望中日之间建立稳定、友好的关系。

此外，韩晓清还表示，文章最初是为在日华侨华人读者而写，所以用词是按日本国内的习惯。例如，把中日关系写成“日中关系”等，但这种写法“丝毫没有怠慢中国之意”。她还指出，这篇文章是“用家里人说话的口气，家里人讲道理的方法，说服家里人”。

韩晓清声明称，她的中心观点实为“国家经济真正强大之日就是收回钓鱼岛之时”，“日本在钓鱼岛的一切小动作根本改变不了钓鱼岛属于中国的历史事实”。


对于一些网友就“害国”二字断章取义，称其已“归化”为日本国籍、是“汉奸”、“卖国贼”的攻击言论，韩晓清也在回应中做出郑重声明。她表明自己是中国国籍，并没有加入日本国籍的打算，而且是一位“坚定的、真正的爱国者”。她指出，自己曾多次参加电视辩论，驳斥日本右翼学者和政客的反华谬论，并向全日本及世界观众重申“钓鱼岛是中国的领土”。正因如此，她也受到了一些日本右翼分子的威胁和辱骂，但她“以自己是一个中国人而感到自豪和骄傲”。韩晓清在来信中说，“作为记者，也作为日中新闻的社长，一定会团结日中新闻社的全体员工和全球的华侨华人一起为祖国的经济建设，为早日收回祖国的领土——钓鱼岛，竭尽全力，尽心尽责”。
