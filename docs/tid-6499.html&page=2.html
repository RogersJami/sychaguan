社会发展阶段的问题吧。像日本，只要在日工作的，包括没入籍的，看病都只出三分之一的钱。三分之二可以报销的。
对农民的补贴也很高。由于全国有占六成的农村选区，因此尽管农业产值在国内总产值中所在比例从1960年的9%降到1970年的4.2%，1980年的2.4%，1989年的1.9%,但农业预算在一般会计预算中所占比例，则从1960年的7.9%升至1970年的10.6%，此后虽降低到1980年的7.1%和1989年的4.1%，但份额相对较大。其中农产品生产补贴和价格补贴在农业支出中所占比例，从1960年的23%增长到1970年的55%，直至1985年仍保持30%的高比例。
1961年日本政府颁布了《农业基本法》，开始全国性农业结构改善，农林渔业者对自民党的支持率也从1955年的52%持续上升到1975年的62%。
在政府提供的大米价格补贴支持下，日本米价为美国的8.4倍，泰国的7.5倍。日本政府不惜以引起日美贸易摩擦加剧的代价保护米价的原因，被1978年的农林相渡边美智雄一语道破：“一旦选举，靠得住的还是农村。”

中华文化里的有些东西是永远适应社会的，甚至是可以标榜成普世价值的。 比方说中庸，不极端；比方说天空一样的自强不息和像大地一样的厚德载物；比方说周易和中医里包含的对立统一的矛盾观、辩证观、整体感和平衡观；比方说道家和禅宗里的一些内涵，这些东西，一定是会永远适应各个时代，正因为这样，只要是能窥探到哪怕是这种文化的九牛一毛，都会感觉到自豪。我个人感觉这种文化里有一种莫可名状的平衡感，不太好说清这是种什么东东，但是能感觉得到。
一个文化，如果适应范围非常窄，包容性不大，比较极端和非黑即白的话，这样的文化，本来就没有什么值得产生优越感和自豪感的，一定会是昙花一现的，中华文化不是这样的文化。

另外，中国这个国家，也就是近代两百来年落后于西方国家而已，以前几千年相对西方国家一直要混得更好，一直是老大。 而且现在也正在追赶，重新强大起来只是个时间问题。

当然咯，也不是说中国古代文化全是精华，没有糟粕。但是里面的宝贝确实比较多，充满了智慧，只是我个人慧根浅孽障深，捡不到宝贝，捡到了也消化不了，嘻嘻。
