另外，就这个问题，做一个统一的最后回复：
1，看得出来，大家在分析台湾问题时，带有太多的道德判断和立场选择。我一向的观点是：分析问题时，要有上帝视角，不要带有道德判断和立场选择；解决问题时，必然有立场选择。至于道德判断，本来就是立场选择之后一个产物，公说公有理，成王败寇！

2，大家不是疑惑一个问题吗：大陆的让利不可谓不多，不可谓不好，为什么台湾2300万人的民心却渐行渐远，在台独的道路上狂奔！很显然，大家对台湾的认识误区太多。大陆现在还在讨论阶层固化，讨论既得利益的贪婪，讨论社会不公不义，为何一到台湾就拐了弯？
民族主义与阶级矛盾（或者人权与主权论）不是非此即彼的相互否定，而是一个二者求平衡的跷跷板游戏。马克思的一句话，我到现在才明白：全世界无产者联合起来！——这是一句充满理想主义色彩的世界大同的口号，却也不乏现实性。
阶级矛盾是人类社会最核心、最普遍的矛盾。所以，毛主席说”阶级斗争一抓就灵“。解决西藏问题，就是这种个理论的运用。如果我们还把目前2300万的台独民意简化为”中国人“这种太多的道德判断，最后的结果肯定是悲剧——收拾2300万的民心就是妄想，即使武力统一，只要不是斯大林式肉体消灭的民族政策，你就得面对千万人口级别的民心向背。有一种可能，治理成本的高昂达到了大陆愿意支付的临界点会最终迫使大陆放弃台湾。

我不知道政策制定者们是不是和网友一个想法，但愿不是！

如果要我总结一个对台政策得失，那就是：完全忽略了本省人在推翻外省特权体制奋斗过程中的合理性，没有充分照顾到本省人诉求中的合理部分。
看一看我们涉台部门打交道的台湾政治人物就知道，清一色国民党特权阶级。想通过买办国民党特权阶级收拾2300万人的民意，可能吗？让再多利，也收拾不了的！最后的效果肯定是太阳花学运看到的那样：2300万人与14亿中国人渐行渐远。
但是，要为涉台部门辩护的一句是：他们有一个政治上的困境——2000万本省人中找不到一个在”两岸同属一个中国“原则下打交道的有代表性的政党，民进党是争取2000万本省人民心绕不开的一道坎。——这是民进党与大陆谈判最大的凭借，即使最后武力统一！

不过，按台湾林中斌的观察，大陆现在也在转变策略，加紧对民进党的耕耘。民、共交流非常频繁，层级非常高，只差最高级别的会面。这是可喜的一步，务实的一步。只有接触与交流，买办民进党的权贵，才能影响甚至改变民进党，最终收拾民心。
多说一句，以目前台湾的政治生态和社会环境，大陆不可能绕开国、民两党而直接面对2300万个体，独力完成和平地、有组织性、体系性的政策制定与执行，除非上”井冈山“。
