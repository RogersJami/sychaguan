根据我的经验，读史主要是读正史，以国史为例24史，历经几百年无数读者考验。其他的野史也就是今天甄嬛传 ...

西方的古代历史记录太差了，都是作者听说看到的一些情况写的，没有专业的历史记录机构。
比方说希罗多德的《历史》，充斥着乱离怪神，中国哪怕是最爱记录怪事的晋书都没有到《历史》的那个荒诞程度。

中国古代有专门的人记录皇帝的一言一行，记录政府机构的各种事件，然后形成《起居注》。
中国古代有专门的机构登记保存各种文件。
然后每一任皇帝上任，都要修编前一任皇帝在位期间的详细历史，这就是《实录》。
然后每一个朝代都要根据前朝的《实录》和各种文献编写正史。

所以中国的正史的严谨性是西方历史无法比拟的。
而且中国的儒家学者天生厌恶乱离怪神，传统的儒家精神也要求宁死不能歪曲历史，这也是西方历史学者无法比拟的。
