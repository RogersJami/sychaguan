Y混不好，在我预料之中，混到这步田地，又是我意料之外。因为她的底子太好了。

我在国内的单位，是国内顶尖的勘察设计单位。聚集了国内行业大拿、并承担着国内技术水平最高的项目。在我们单位工作一年学到的东西相当于在其他国内同行几年，与西方同行相比，工作强度大得多、与现场接触紧密得多、交流培训多得多。Y虽然和我同岁，但是作为子弟，本科毕业就得到了到这种顶级公司工作的机会，比我早3年入行，而且专业对口、毕业于工科名校(仅次于清华浙大的下一档次)。正常讲，其技术水平应远高于同龄国外同行。正因为此，刚认识的时候，当Y跟我们吹她在乙方混得如何如何好，老板如何如何器重她时，我一点也不怀疑----因为她应当有这样的技术水平----至少她的技术水平应当比我高。

然而，此后发生的一切出乎我预料。

按理说，单位裁人，有2个因素，工作能力和人脉，简历漂亮的话，公司也倾向于留，再找工作也容易。Y在乙方的时间不算短----至少比我长。然而每逢裁员，毕第一批中枪，形势好，又是最后一波回来的，只能说明她一样都没占上。换句话说，她和10年前刚出国时，没有太多进步，还在原地打转转。

虽然我没和她共事过，但是根据她的为人可以看出，她看不上一点一滴的进步，不愿意一点一点地付出。希望通过忽悠，一步到某个位置；爱耍小聪明，爱占小便宜；别人取得进步，她不是想着如何赶上，而是去去踩、去忽悠别人放弃，问题谁是傻子？

就这样，当别人聚沙成丘的时候，她还要为工作的事情担忧。

美国又何尝不是这样？通过金融手段掠夺、给别国制造混乱、别人取得的进步，它开动宣传机器去妖化、培养公知把别的国家往沟里引......看起来便宜占尽，如果放在几十年里看，其实国力在不断下降，忽悠的是谁？是不是忽悠自己？
