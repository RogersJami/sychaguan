效率的理解一般不会有太大分歧，简单评价就两个:时效，成效。时效上，国企为规避风险，流程繁琐，因此远不如私企，这个我想也基本是公认的。只有在成效上，国企私企各有千秋，很多领域稍有风险，国企就不敢碰，私企却会各种“灵活”应对做成；而也有一些领域，红线很高，私企没有入场资格，或者无利可图，私企不愿进入，则只有国企能做成。综合评价上，国企效率输给私企，这个判断没有大问题。

手机一时发送不上去多点了几次成了反复发送，就顺便多说几个零星的例子

还是电力，以前在西西河写过一个例子，大意是电力基建后面有关国企业多报一个0，然后相关负责人对我提过一般多报一个0能在几年内磨平账目。对了项目本身千万级别，过亿的没经历的不乱说。

还是电力，记得有几年电网大跃进那会搞绩效倒逼。某华中电力国企，在执行过程中也遇到国家大基建的过程中各种奇葩。用当时负责人的原话说就是，基建就是一路打过去的。举3例

1.遇到中石油

因为体制等原因，他们施工中把中石油不能公开项目的管道挖断了，而且出现了爆炸。两边工程都停工。两边都要赶进度，结果谁也不让谁械斗多次无果，上报电力政策优先石油管道铺设，他们赢这一回，然后负责人说总算出了十多年给石油让路的恶气

2.遇到地方腐败

因为体制原因，因为地方政府特别是乡镇级别这种一贯无法无天的政府基层，吃掉了乡民的拆迁补偿款怎么办。一边是理亏，一边是有理说不出。各种汗，只能边打官司边干。

3.遇到讹诈

因为国有工程全面铺开，各种环节寻租不断。说个很小的寻租解决案例。有次他们施工经过某地。当地人突然来了几百人把国企工程工地围了，还冲进工地打砸抢。然后村里长老出面要求说，这里盒饭我们包了，不然明天继续来以后天天来。第二天，他们还是来还是打砸枪。国企的人没抵抗随机撤出，然后包工头带近千把工地围上，长者要耍赖说打死也要利益。包工头当面一句，打死一个包赔五十万，打死多少不论。然后打，打了当然不会死人。但是打服气，对方认赔供应盒饭，但是打砸抢赔偿支出算在盒饭内。为啥包工头那么横，无非利益。因为承包合同说的明白，提前工期一天按照工程总额的比例补偿。电力工程，大家懂的。

用我干电力工程的朋友的话说，大型基建就是一路打出来的。

还是能源部门

1.前面有谁说我举例有硬伤，我不能不承认很多人思维方式有硬伤，很多人被自己认知的行为模式局限了，就和当年我在西西河在军工利润话题和3D  打印话题上的被围攻那会一样，很多人把自己的认知自觉不自觉带入自己不熟悉的领域，我也犯过类似错误。那次是矿难事故，两起矿难都是浸水事故一次井下工人全部罹难。另一次，是因为井下工人还在抢救，但是因为打穿水层附近又近河流所以我基于上一次矿难经验武断判断为工人已经全部罹难，尽管当时有正在参与抢救的有关技术人员给我反复解说我就是不相信矿工能解救然后么工人解救成功无一罹难。事后新闻中的细节给出拯救成功的要素如下，1.同为矿难，前者安全设施老旧且已经承包给私人，安全设施在承包后几乎不存在更新，所以所谓安全防护形同虚设，所以矿难一出随后有关工人全部罹难难以避免。二后者还在国营体制下，对安全设施有所保障，并引入当时还是先进技术的井下安全屋，这在骤然出现浸水事故的事后，有了缓冲区域，救援实际围绕缓冲区展开。2.因为救援有缓冲区，所以救援人员能通过通气管道给井下输送牛奶维持必要生存要素，所以在时间长达近一周的救援过程中，井下人员能维持基本健康状态。并且由于能同时保持一定消息的互通，井下人员在矿难状态下情绪基本保持稳定，这对救援起到了很大帮助。3.虽然矿难的原因是井下浸水，但是浸水的原因不同，事故诱因不同，也因此浸水的反应时间不同，也因此在事故初期反应阶段和事故中期救援阶段与事故最后解决方式方法有很大的不同。这次事故之后，虽然我诚恳的和救援人员有关负责人道歉了。但是在救援中我清晰的记得，他在怎么都和我解释不通的事后，愤然说了一句，一个有井下救援经验20年的热和一个从来对此一无所知还振振有词判定什么结果的人说什么都是多余的。这是，某个政策导向的开始那种倾向于把人和人在不同信息圈闭合区间相互隔离的开始。他们也因此和我打过一个赌，这个赌在西西河的尝试过程中我输了。结果之一，就是群众运动不可能再进入政治博弈核心，主席的路还需要很长时间等待。这是旁话不再多言。

2.说道石油部门，吃肉的事情很多人都知道，那么说几个啃骨头的事。前面提到过，塔里木油田，有部分油区是深达万米的，这里有人理所当然的以为这是打井手段决定的。实际勘测油井的手段局绝对吧只是打井那么单一，不然这一百多年的技术进步就白干了。这里说深度实际在说成本，前面谁提到的油井深度恰恰是地面油井维持在什么深度能产生经济效益的平衡点。这个平衡点，是动态的不仅根据油价本身波动也根据油井开发技术本身而波动。记得，2002年有大庆油田的人和我们这里的圈子推销大庆封存油井开发的事情，事后当时推销部门负责人因为我们这里有石油系统元老子弟，也说了实话这种销售是宰凯子的。因为当时的技术，与当时油价维持一口封存油井的开发成本在120万到200万之间，再开发在当时技术还没突破前提下每年亏损预期是20万左右。实际，当时去尝试买封存油井再开发的，能支撑到盈利阶段都亏损在平均200万/每口井的阶段。这里坑在人员安置，和基础建设重新启动的负担。但是之后的伊拉克战争导致油价激增，并且网络热传的某帅哥院士开发的废弃油井再生技术，在油价顶峰这些油井一度每口日产原有价值过百万。不过，这些封存油井，大多在油价起飞后逐步为国有部门收回。这里，虽然最后还是吃肉，但是也体现了国有部门为维持国有资源部门在困难时期的有限维持都需要拿出坑蒙拐骗的手段。为什么因为他们还承担的一部分战备任务，前面提到的塔里木油田是世界上第一个在沙漠开公路维持开发的油田，这里成本怎么计较，还有说深度，青海西藏都发现油田这里深度不能多说，一个是没有共公开资料作证，一个也会容易被人喷什么毕竟没有对证的。这里能说这部分油田开发的一些心酸细节，记得某院士在讲解中国石油储备的时提过我们储备自给有有保证的就是成本高。当时，在座的学生很质疑啊。于是院士委婉的说了一个他当年经历，就是在西南曾经发现过成十亿规模的油田，但是因为成本问题以及基建问题一直得不到开发利用。直到某一年大地震，恰好改变了油田区域的地下构造，导致相关油储不再具体开采条件。光这个地质勘探，有关人员前后用了几十年时间，到头一场空，这种事不要讨论国营民营谁效率高，换个小点的国家可能干么。

3.上一段说了以前认识一石油部门元老的小孩，与大庆油田当时推销封存油井再开发的事情。当时，他说了他们家家长在石油领域的一些经历。以及相关周边技术应用的开发。这些段子过去都写过，不是啥新故事。大体一件是，当年在东海油气田开发，一次台风事故，当时中国海上油井专家团灭近七成。这类似某龙门吊垮塌事故，中国有关专家几乎团灭。这里开起来说的是专家，但是相关开发部门有关的工人的素养直接决定了很多事情的成败。这里说个凶险的，还是前面小孩提到的专家亲历。因为，海上项目，涉及地质工作。在海上项目休息期间，也作为洋山港开发的一部分，参与周边地质相关工作，其中一个项目就是沿着特定路线造海底隧道，项目在进行一半的事后。有一次小伙子家长陪着大领导视察，发现隧道出现裂缝，当时工程人员直接判断要漏水。而老人，在自己冷静计算后计算结果是，当时及时马上撤离也不可能安全撤出全部人员。当即拍板现场救援，而老人事后赞叹的是现场国企工人的冷静和多年经验积累的准确判断与操作。这才最终让可能的恶性事故，转危为安。这里我多说句，我经历过的国有部门在计划经济时代体会培训阶段，光我为了熟悉精工操作的三个月时间中我熟悉锯子的操作曾经锯掉的废铁就用掉可以以吨计算。同样，那个时代的老工人，以我一个同学母亲为例，她的精准焊接操作让当时电真空电视机显像管一个精准焊接工序直接去掉了当时耗资百万美元的焊接自动化器械，只是打下岗时代，依旧裁撤。这样的大国工匠，直到现在十年才被重新重视，也是时代吧。

这里说说军工吧

1.说道军工，很多人印象应该是高大神秘与投资巨大。这某种角度说都是对的，但是自小我在有印象中，军工子弟对军工印象就是危险，见不到家人和各种奇葩脑洞。这里先说的是涉及有毒有害的危险，前面提过我家长提过他去过某基地的福利。记得还是我初中那会，他有次提及这个基地，是因为电视里发射奥星成功说相关实验基地他去过。我说怎么没听你提及，家长说怎么提。当时出差去正好遇到装填燃料实验，过程中出现燃料泄露（当时我们国家固体燃料还没有完成开发），于是整个基地的人都按计划救援。过了很多年后，才直到因为液体燃料的高腐蚀性并且实验现场条件的复杂性任何一个小实验瑕疵与工程中的疏忽都回导致整个基地的爆炸。这种爆炸一旦发生，方圆多少范围不会有人幸免，当年某苏联元帅在类似事故直接挂在现场。所以按照计划，只能是用人命填进去救援，牺牲一批换一批，直到抢险完成或者基地完蛋，一旦基地本身带有高危性质有关实验残余对外泄露0容忍，那么基本本身就会启动自毁程序，比如从事生化实验的特殊部门有类似程序不是虚化。。实际任何带有军备任务，或者战备任务的国企，多少都会有类似经历。我一个负责电力调度部门的朋友，就遇到过他们部门被在雨季要求给特定项目供电的任务。因为，会有较大危险导致人员伤亡可能，所以调度人员按照条例拒绝，但是军代表一句话不听命令我现在有权枪毙你，以后有事也是我上军事法庭被枪毙。但是现在命令无论如何要执行。如果有人还觉得那是当年特殊年代的特殊案例，那么在在00年后有多少人知道过为保存国家重要资产的事故中依旧以生命不能承受之重去填补与抢险的，时事变迁，承载军工职能的国企依旧。国企承载的社会只能与国家责任，很多人不会理解也多半不想理解与接受，因为里面有时候多半是不可承受之重。

2.如果家里有军人，或则军工人多半会体会家人长期分离的心情。我一个初中同学，家里爷爷父母都是长征火箭开发人员，十岁前记忆就是跟着退休的爷爷奶奶。长大后，后遗症之一就是沉默寡言的。说到军工人，最常见的问题不是长期和家人分离是，健康。因为长期接触有毒有害物质，加上有关认识不足，和医疗手段局限性很多军工人一旦发现身体问题往往是恶性与晚期。我一个小学同学，家长在军工部门做事，主要是生产弹药的部门。家长因为接触有毒有害物质，很早就有肝肾不适，大下岗那会不想家里额外负担，在家里等死而且是活活疼死的去世那会五十才出头。这样的事例在军工部门不适孤立，更是有一部分人群集中被选择性遗忘。还有因为军工研制的危险性，还有长周期性，所以往往一个技术公关完成审核，多少利润军方都接受。部分原因是政策与内外格局需要，部分也是军工研制背后的代价。这里补充一个3D打印的事情，我小时候家里没有人带，因此常常在家长实验室度日，所以对有关金属晶相和有关3D  打印早期积累的技术堆焊并不陌生。哦记忆中，军工研究就是不断是的实验然后存档，极其枯燥，几十年入一日。一开始对这种长期积累不以为然，直到有一天我看到克虏伯宣布根据百年积累的档案资料分析，只要金属承担同一晶像晶想相同成分相同，性能也相同。这让我明白了，3D打印可能成功的原理。这个原理我在西西河有关3D打印的争吵中说明过。尽管有人说，直到原理没什么了不起到重要的是直到如何打印。这种思考方式，不管他出发点如何。我能说的是，因为我的有关讨论有心人看懂了也上心了因此，在多少年后一个相关部门负责人对我说：你对3D打印推进是起到不小作用的，你自己不用谦虚。尽管我知道我实际起到的作用很小很小，但是我能说明白相关原理，是因为我在家长实验室的长期积累，因此对外部信息的理解可以直接获得有效反馈。是的，我实际说军工和国有企业不同其他企业地方就是他们看起来很多人眼里的无用功已积累的庞大数据，是他们厚积薄发形成各种垄断技术的优势所在。还是回到我家长吃肉的技术公关，我家长解释是这样。论水平他不如项目公关时期，几十个本土与海外博士硕士。但是，几十年积累的经验并通过实验数据的量化分析，让他可以直到在什么工艺条件下什么温度能达到特定性能的阶跃，实际差半度都可能天壤之别。这里就是我想说的，积累之余军工，积累之余国企，也因此我在打数据论述中有一个衍生观点，不存在没有用的数据，相关则是后话了。

3.最后这里说的是奇葩的脑洞。说奇葩是因为很多事例脱离普通人认知的常理，这里不是说脱离普通人认知的常识：比如成本说。实际点说，军工为达目的很难兼顾成本。记得以前提过蝉式导弹填充物光可替代的就多达种，还有我家长给某型号设备配套过程中技术要求是上吨的设备在数分钟内能满足正反数千转的要求。还有就是配套政策。既哪怕你技术再先进，其他技术不匹配就不会成套上马。比如，当年某万吨级别的特别管道成型机械完成国有化后，有朋友说我们核电设备全部国产化的最后环节掌握了然后核电能放量。这背后有土共一贯的土鳖作风，这种不惜代价也要国产化的行为方式对很多人就是漫长的等待甚至一生的等待。还有，我们数十年积累，面对大小变局做的安排有时候也许不可能见到任何直接利益，这里说个脑洞大的项目，不在技术多先进在思路。记得十年前有朋友做工程机械给工程配套，他们在新疆开了一条地下隧道，宏大漫长，当时朋友问我这种隧道有啥用我也不明所以。再过几年，英国人公开称发展中国在新疆有几百英里地下隧道，要中国人澄清相关用途，这时候我才想起来好多年前，在一部电视剧中刘烨扮演一个叫钟跃民角色那会正好听人开脑洞做推演课题内容是：如何全歼驻阿联军。那事英国人开盖子后没多少年，联军开始撤离阿富汗或许这就是无巧不成书吧。最后这里想写的脑洞不在于他的天马行空也不在于有多少技术难度，但是这里有军工人的鲜明特点。这事还是能源系统子弟提及的。大意是，如同3d打印有关原理还不能有效覆盖相关现象一般。把煤化油和柴油汽油配比混合也是个难题，我所知道的难点之一就是长期储存的混合油怎么才能不分开。因为没有明确的理论指导，所以只能在不断穷举中试错。其中，就有一个老工人用了20年时间，获得了突破与成功。他是同样在做相关工作中的一个。这也是一种大国工匠，哪怕只有一线曙光，依旧不记成败在坚持中前行。

你说得对，探井和生产井是两码事。

塔里木油田建设成本高，原因是多方面的。首先是修路，大庆油田在冬季冻土期作业，大型设备在冻土上通行无阻，塔里木沙漠则没有这个条件，需要为大型设备修路。其次是高温作业，井架上搭帆布帐篷挡阳光，帆布帐篷下60多度，工人需要不停地喝盐水降温并保持体内盐分，2小时一班，效率较低。再次是没有人烟，大庆最先开发的时候，讲的是先生产、再生活。到了90年代，上层紧吃，下层自然没有那个觉悟，生活、工资成本都很高。另外，地方老百姓觉悟也不像大庆开发的时候那么高，以前是老百姓怕“油鬼子”，到了那个时候，就是老百姓拿油企做肥羊了。塔里木的井深，好像是7000米这个级数吧，当时算是比较深的井了。

石油部88年重组为中石油，中石化成立于84年，中海油更早，成立于82年。98年老朱把中石油、中石化的资产重新划分，胜利油田划给中石化，在此之前，中石化没有石油业务。80年代，石油是按照计划经济价掉拨，中石化作为直接面对客户端，日子比较爽。石油那边呢，主力油田进入回报阶段，加之工资较低，企业办社会基本完成，生产、生活成本都比较低，作为战略行业国家比较重视(286还得出口石油创汇呢)，感觉比较好。90年代上半叶保持了这个惯性。

大约是95年，好像是金陵石化率先用期货的方式购入原油，大赚一笔，并投资几百万建设期货交易中心。这个经验在整个中石化沿海炼化企业推广，然后就悲剧了。所以当时的中石化有个说法：难忘的95、难过的96、难测的97。那个阶段中石油没有受到太大的冲击，下岗也是有些领导为了迎合上意瞎嚷嚷，毕竟没有了海量的前期投入，仅仅维持生产，成本还是比较低的。

中海油当时主要靠花架子，所谓现代企业制度，也就是不像中石油、中石化那样企业办社会。但是中海油技术力量不足。举个例子，渤海油田所有的新发现都是靠合资方(主要是康菲)，渤海自己的勘探部门没有发现任何产油区块。东海后期是靠中石油的技术支援。南海则是中石油留下的底子。中海油直到现在依然非常不专业。比如某个最大的并购案，业内人士的说法是，花白面钱买了棵白菜，这棵白菜还是烂的。
