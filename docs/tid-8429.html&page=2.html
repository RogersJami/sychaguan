2011年10月10日报导：中医表示，咖啡具有蛋白质、脂肪、烟碱酸、咖啡因、生物碱、丹宁酸等成份，有抗自由基的功效；但副作用会导致钙离子的流失，有骨松的问题，也会降低身体的含氧量。建议肝功能不好的人，不要过度去饮用过量的咖啡。中医师表示，咖啡所含的热量较低，具有抗氧化的作用、预防身体的衰老；亦可以减缓肝细胞纤维化的问题。同时，中医观点认为，咖啡具有活血化瘀、除湿、健脾功效；有扩张血管的作用。
所以，喝咖啡对身体有好处也有坏处。好的生理反应在于它具有抗氧化的作用。尤其，它含有咖啡酸、绿原酸等成分，有抗自由基的功效。此外，有提神醒脑的作用，让人精神振奋、愉快、解忧的效果。促进消化，帮助排便及利尿的作用。但是，副作用则是会导致钙离子的流失，有骨松的问题。咖啡因会吸附血液中的铁离子，降低身体的含氧量。肝功能不好的人，不要过度去饮用过量的咖啡。如一天超过2杯以上，造成精神过度地亢奋，便无法适当地让肝脏休息。因此把咖啡当做开水这样子喝，过度饮用，肯定对身体并不好，且会有成瘾性。同时，也提醒怀孕期的妇女，少喝咖啡。或者有胃肠方面的问题、溃疡，及精神方面的疾病。有服用镇定剂的人，也建议少服用咖啡。如果进行保健，医师建议，可准备柴胡1.5钱、黄芩3钱、丹参3钱。用600C.C的水，煮沸后，放温，即可饮用。可疏肝、帮助肝脏血液的循环，顾肝、降肝火的作用。
  咖啡更多的是补充和发散能量。   茶更多是发散之后能宁神，说的玄乎好茶搭配山泉水灵气入体，    哲学层数不是一个境界。

刘枪枪 发表于 2014-2-6 19:32:49

曾经做过两个咖啡店的我来了。几点小心得和大家分享。

这个一啊，咖啡不能多喝，对心脏不好。巴尔扎克当年负债累累，一天喝几十杯咖啡写人间喜剧，最后自己悲剧在咖啡上了。

这个二啊，你喝着开心的喜欢的咖啡就是好咖啡。只要不加毒药和兴奋剂你爱怎么调制怎么调制，爱喝冰的就加冰块，意式浓缩也可以加糖，喜欢就加两份，也很好喝。南美人为什么屁股大呀？吃糖吃的呀。

三啊，什么猫屎咖啡蓝山咖啡都是胡扯。前面让印尼人捉了成千上百只麝猫天天喂咖啡豆逼着它们天天排着队使劲一坨一坨的拉都成了笑话了----拉矢流水线。后者日本政府和牙买加有协议包销他80%的产量。剩下那点都不够牙买加人民塞牙的。我做了这么多年咖啡就03年（？）中联部一姐们陪长春长老去加勒比访问，人牙买加外交部礼宾司送她一小包她不喝转送我了。我遍请老友大家喝完也没觉得多了不起。

四，这个产业链也很可怕，炒作的厉害。我看到很多人就掉坑里了。一例子来说一哥们被忽悠买了一台意大利手工机器，单头单冲，5万块诚意价，不贵不贵。天天要测水质。PH值？大兄弟你弱爆了，还要测水的柔软度啥么的。奶只用德国的一个牌子22一盒我们也就敢用个雀巢9块钱。豆子不仅要讲产地还要看标号，店不大一台烘豆机就占去了三分之一面积。我去看他的时候正例行拆机器做保养，看得我一脑门子的汗啊。我问你这伺候法怎么赚钱？人家看我那眼神，我顿时觉得我在咖啡这行里就不配姓赵。
