从古罗马时代开始，由于战争君主就开始对于个人富商有借贷，人死债消或者驱逐债权权人人出境或者没收。到出现国王对于教会最早的机构借贷，也是不能全部还款。以上的借贷可以理解成特别税或者保护费。法国亨利二世（1519年3月31日—1559年7月10日，1547年登基）为了战争筹款，首先实行了包税制（这个可以看成对于借贷的抵押，以国家税收的形式）。伊丽莎白一世（1533年9月7日--1603年3月24日，1558年11月17日登基，在伦敦建立了证劵市场）。经济中心从热那亚--威尼斯--阿姆斯特丹--伦敦。从借贷双方的历程来看，资本也是由个人行为逐步发展为组织行为，也就是说掌握资本的人为了抗衡王权是逐步建立组织的，不然从力量对比上没法抗衡。还有一个因素，欧洲自查理大帝以后就没有统一过，这个也为资本安全的腾挪创造了空间，不会让人给一勺彻底烩了。从借贷资本以国家税收为抵押开始（包税制），资本控制国家可能性开始逐步显现了。
关于资金的持续来源，如行业垄断--犹太人专营借贷（中世纪犹太人不能拥有土地），英国东印度公司（拥有相当与国家的权力的特权，垄断商业特权）等，不然没法解释资金的持续来源。
中国自秦以后就是中央集权的为标志的统一国家，资本在王权的控制之下。中国的改朝换代，导致了资本难以完整的延续。
