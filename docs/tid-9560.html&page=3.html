嗯，是这样滴，从实际经验来看，初次见面的人之间的群聊，能敞开来谈的，都是相对普遍性的话题。但，在群聊之后，有急于解决问题的群众则要求一对一约谈了，通常是这样的，尤其在江南相对来说较为含蓄的文化中，更是如此。（南北方人之间的差异真的很不小的。）

当然，如果群聊效果很差，那就不会这样。

另外，对于聊儿童教育，这个怕是有误解，教育是跟任何人都有关系的，是任何人都可以关注可以谈论的东西，任何人都可以说得头头是道的。聊天的快感不在比智商和口才，而是在于参与的每个人能就大家谈论的话题自如地发表自己的看法，包括各种跑题，只要能让在场的人都有兴趣继续听、说就是。也就是群众创造了某个时空的氛围。这个基础还是物以类聚，就是心甘情愿地花费自己的宝贵时间和口舌在某些人群中。所以，聊天的快感是来自物以类聚。

诚然，在专业做教育的人们之间的聊，则是不同的。从我个人的风格来说，对泛泛地讨论不感兴趣（那是我大学四年级前就玩儿剩下的）。我是围绕解决一个系统性的问题入手、朝着架构一个解决模型为目标，找到不同专业特长背景的人（必须在至少一方面比我强），从他们各自的视角提供各自的资源，然后每个人都在实施过程中获得成长，并且非常开心（我的人生理想是跟自己喜欢的人们开心地做有意思的事情，目前为止没偏离过这个轨道）。对此比较难得的一点是，能够让强者们互相积极配合，甚至甘于当配角打下手。要做到这个的前提是，与其说是找人，不如说是吸引人，通常是某人很羡慕我们这种做事方式，很喜欢我们这个团队气氛（各种欢乐和进步，互相支持，能及时指出别人的问题并立即改正），然后会主动地要求参加进来。由于我很挑剔，所以门槛不低，没有真才实学的不予考虑，身心不够健康的不予考虑，还有比较变态的是，长相要是对不起的话，也不成。;P 关键还是物以类聚。
