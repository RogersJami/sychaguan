对于日本人我们只能从一些他们的文化作品来看，因为我们无法得到第一手的资料。从他们的文化作品可以说明他们民族的价值趋向。就我所看过的作品来说只有一部《白色巨塔》是具有现实批判主义精神的，其他的要么是做梦式的改革，要么则是蜻蜓点水式的。所以我认为从价值趋向上来看至少是缺乏自我修正的。

而一个缺乏批判精神的民族他以前再怎么牛也只是历史罢了。记得日本有一思想家名字叫新度户稻造写有《修养》一书曾经是日本民族的圣经，今天我想日本的年轻人还有多少知道这本书的，而其中所传递的“苦行僧”式的修行在今天日本的文化产品中已经难得一见了。就算是立志的作品也做的如此的做作与虚假。
