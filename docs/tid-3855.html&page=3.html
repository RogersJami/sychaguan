我老家县城，全县40万人口，城区5万左右（可能不准确）。这两年“城镇化”，相当于建设了3个原城区大小的新城。

当然，不乏深圳致富的同乡大把圈地，好几个号称工业园的项目，地块至今荒凉了几年。

房价去年不到2000，以前我也觉得不可思议，现在算有所了解了：

1，乡镇公职人员，从月薪1500的小学老师、一线政府职员到乡镇领导，从20几岁到50岁出头的，基本全在县城一次或者二次置业了。老妈说逛街时不时偶遇十几年前二十几年前乡镇时的同事、朋友，都准备在县城养老了。

2，家里亲戚，当兵的，大头志愿兵的，几万块钱，退伍回来在县城置业了；表哥表弟堂哥堂弟，有瓜葛的农村亲戚里，外出打工的，30-40年龄段的，有很大部分也在县城买房子了。

3，原有县城居民，基本上90年代建了一栋一栋的私房，这一波，人手一套商品房，都现金置业了；公职人员，普遍三套（一栋2套），开发区建设，部分单位搬迁到较偏远的地方，配套的都建职工宿舍（商品房），500、600的内部价格。

4，补充，国某局局领导加会计，先知先觉，5、6年前于某偏僻区域，人手购买二个门脸的地块，地皮才几千块钱，统一建起了10层还是多少的楼房。年前的消息是，某人年租金收入20万，当地人均工资2000左右。

房子干嘛呢，大都空着，等升值。
