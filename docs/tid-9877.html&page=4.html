“转基因技术对于农业增产增效和提高质量,极为重要!”3月4日下午,全国政协委员、中国农业大学校长柯炳生在参加全国政协十二届二次会议经济、农业界委员联组会议时,当着李克强总理的面发言时说。
柯炳生表示,转基因问题本来是个科技问题,但是现在弄得很复杂。世界上可能没有哪个国家,转基因问题受到如此的误解。他觉得现在最误导人的说法,是说“科学家们对转基因问题有争议”。这个说法,让老百姓忧心忡忡。个别人、个别专家并不代表主流,事实情况是,现在主流科学家、自然科学家,在发展转基因问题上没有争议,都支持。去年,61位中科院、工程院院士联名给总书记、总理写信,呼吁支持转基因产业化。说到这里,柯炳生加重了语气,“我不知道还有什么重大科技项目有过61个院士签名?科学家是负责任的!”
柯炳生说,作为农业大学校长,经常被人问起转基因方面的问题,无法回避。他通常总是给人家说5句话:第一,转基因技术极为重要,潜力巨大。第二,“转基因产品”是否安全,不能一概而论,要看转的是什么基因。这有点像蘑菇,有的能吃,有的不能吃。所以,政府要进行监管审定。第三,“转基因食品”是经过科学检验、政府审定批准生产上市的,相当于超市里的蘑菇,是安全的。第四,美国是转基因食品消费最早、最多的国家,从来没有发生过任何安全问题。其他国家也没有。第五,世界上与食品安全有关的所有权威机构,包括世界卫生组织、联合国粮农组织、美欧日的食品安全监管部门、各国的科学家学会组织等,都对已经上市的转基因食品的安全没有争议。
我们国家每年进口国外大量的转基因大豆和玉米,敞开进口人家的,我们自己不种说不过去。如果我们国家在这个重大问题上继续迟疑不决,将对我国未来粮食安全和农业发展产生极为不利的影响。柯炳生说。

ccaaatt 发表于 2014-3-7 16:44:48

全国政协委员、中央农村工作领导小组副组长、办公室主任陈锡文3月5日表态，转基因育种是全球生命科学的前沿阵地，中国作为一个农业大国在这方面不能落后。

　　陈锡文在接受财新等媒体记者采访时说，国家政策是非常清楚地要支持转基因的发展。转基因是分子级的育种，相较于袁隆平水稻杂交的细胞级育种来说，育种更精准，也更快。转基因农作物的主要优势在于增强抗性，抗病虫害、防除草剂等，从而可以减少农药和除草剂的使用，有利于环境保护，因此要通过不断研究，赶上世界先进水平。

　　对于目前广泛流传的关于转基因食品的传言，陈锡文强调，目前中国经过严格审定，正式批准上市的转基因食品只有番木瓜。并且，尚没有能够举证转基因食品不安全的证据。“转基因是科学问题，如果没有这方面的知识，最好不要去参加讨论这个问题”，因为“一种违反科学常识的讨论如果影响了社会，带来的影响是很不利的”。

　　陈锡文说，生物科学家、业内人士基本上异口同声支持转基因的。“有人倒过来说这是卧底，这对科学太不尊重了”。

　　陈锡文对转基因如此坚决的表态或和目前转基因研究受到的严峻挑战有关。据财新记者了解，全国政协委员、中国农业大学校长柯炳生在4日李克强总理参加的经济界别、农业界别联组讨论上表示，据其调查统计，去年中国转基因生物新品种培育科技重大专项经费从高峰时期的20亿缩水至4亿。多位参与相关研究的科研工作者也告诉财新记者，对于转基因专项经费的大幅削减，他们同样有切身的体会。在反转声势凶猛下，许多研究者担心，这将错失机遇，使中国种业发展更加被动，将市场让于孟山都等跨国公司。

　　陈锡文称，“柯炳生的发言很好。”

　　陈锡文表示，转基因食品对老百姓来讲是食物，要确保它安全。所以政府会经过非常严格的检验确认转基因食品是无害的，最终才能批准它上市。此外，政府有责任要求生产者提供标识，满足百姓的知情权和自主的选择权。陈锡文认为，考虑到这几点，转基因食品是不存在问题的。

国营的老板是“国家”，原则上是“全体人民”，道理也是一样的。
现在的大主管们应该是看清这些道理了：因为他们再不拨乱反正的话。那些正在流失的将会一去不回，那些不仅仅是时间和金钱，也将包括他们手里的“权力”。
