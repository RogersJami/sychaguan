技术线和管理线一般不交叉。技术线的最高级就是技术副总裁（一般也有叫总工程师或者CIO，CTO之类的）。GE的技术线只有10级，比管理线少三级。

一个人一旦决定走管理线，基本就没可能会到技术线了，杰克韦尔奇就是例子。

一般分类是集团总部；战略事业部（例如金融，制造，服务，原材料等等）；专业事业部（例如医疗产品事业部，核电事业部等等）；大区管理战略单位（例如亚太管理总部）；地区管理总部（例如中国事业部）；企业（例如GE汽轮机华电闵行新能源有限公司）。
