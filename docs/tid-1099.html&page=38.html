当时苏军的抵抗是超越了一般对于军人的要求的,确实是殊死搏斗的卫国战争.最后突围,是守军在绝水的情况下的自杀冲锋,在现代战争以重机枪为核心的火力下是没有军事意义的,仅仅是这些军人视死如归,不愿意投降而作出的个人行动的选择.对于这类军人的选择,电影里也拍到被困在城里家居的零散军官,在绝望的处境下使用手枪自尽.
