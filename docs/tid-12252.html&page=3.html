纸张与定价是跟出版社协商后才确定的，本来他们是打算做成精装本，将近200元一套那种，被忙总否定了，忙总说做成最实惠的，价格越便宜越好。

另，这次纸虽然薄了点（换成80克的话，就实在像厚厚的词典了），但这种纸其实质量不错的，保存时间会非常长。比现在市面上大部分书采用的轻型纸的质量要好很多的，那些比较便宜。
