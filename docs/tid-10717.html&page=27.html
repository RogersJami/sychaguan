http://www.sychaguan.com/forum.p ... 7&page=14#pid209997

葡萄在当时做标题这样的判断，确实是带来了热烈的讨论。在过去的日子里，在各种手段使用之后，经济仍在下滑的格局下，我们的政府重新祭起了"房产大法"。如一位茶友说的“中国人买房思维是骨子里的，不需要任何思想动员的”:), 如今，在一线城市楼市的火热带动下，强二线城市也出现了量价齐升的局面。

此时，去回顾一年多前的文字，不胜唏嘘。

1.北京的房产市场价格出现大概20%~50%涨幅不等的行情，结构化现象很明显。在如此高的基础上有这么大的涨幅，超出我的理解。学区次新房涨幅最大，尤其是社区环境好的学区房小区，绝对的一房难求，即使是天价。部分学区差的老破小，涨幅不大，即使在如此火热的市场环境下。

2.现在的市场情况是，卖家在中介那里挂一个比市场成交价高15%~20%左右钓鱼，试探市场行情，也就是买家究竟有多大的购买力。价格接近市场价的成交较快，不过成交量略有萎缩的迹象。

3.看起来有点象2013年的行情，2013年两会前，市场火爆，之后出了调控措施，但市场接下来又冲高了大概半年多。 然后调整了大概1年的样子，到2014年10月左右央行出宽松政策时见底，然后有了这轮上涨。至于今年后面的市场如何，师母已呆
