忙总好文章，补充一下我所了解的几点文章背景，供大家参考：
1、前几年一同学挂职到中部一个省份的一个贫困县当副县长，据他言，由于县里财政入不敷出，有两个衙门县里是不给财政的，一是公安局，一是交通局，其开支全靠创收，这里面有很多小故事，试说一例。话说县里两副县有一回掐了起来，一副县分管招商引资，厂商诉苦说交通局把他们拉货认定为非法营运，要罚8000，这副县便找到分管交通的另一副县，意思是给个面子，不要罚了，结果这副县说，少罚一点，罚5000，都照顾面子，交通局一帮人喝西北风啊。
2、路上设卡的，交警管违章，交通管超载、营运。中国的很多东西离西方差老远，但在要求上规定上基本与老外接轨，所以小辫子是一揪一大把，尤其是货运车。
3、高速公路上一般不允许查车，所以尽管高速收费昂贵，货车们还是愿意走，因为，成本是可预算的，除非不通高速。有些地方坏，喜欢在高速出口地方逮，这个货车们就很无奈了。
4、全国的大部分收费公路，还是国有企业投资，少量的属于民营，对国家而言是一个锅里的，个人认为到路网差不多形成以后，这种模式会渐次消失，说不定在下届政府任期内，就会解决。

1、公路用地是建设用地，这是没错的。
2、按照土地法，建设用地使用有两种方式：一种是有偿使用，一种是划拨使用。
3、军事、公益、能源、交通、水利等，按照土地法，是可以划拨的，但在实际操作中，如果公对公，划拨可以，如果对私，划拨行不了，还是需要补偿一些，但这个补偿标准是很低的，现在一亩地也就是几万块，经济发达省份可能高一些，但也不会高哪儿去。
4、征迁费用是在建设投资内的，不会超过10%。
