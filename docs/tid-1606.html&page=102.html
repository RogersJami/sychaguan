那一剑的风情 发表于 2012-5-3 10:15 static/image/common/back.gif
那倒是可以试一下.

不过还是一件小事, 没有太出格.  人类社会永远不会有解决所有道德伦理问题的那一天 ...
人类都希望能用自己的力量和理性使自己的周围和世界达到一个心中的道德标准，就像武侠小说里的侠客一样

但这是注定不可行的

注意真正的道德一般都是在小团体内达到的，上古的原始部落，几十个人大半辈子同生共死，猜想那时候的人大概才能达到理想道德的水准，没有吃的去和猛兽搏斗，输了就被大自然消灭，赢了就分享。主要精力都在和自然战斗上，弱者和老人就被自然淘汰掉。内部没有害人的能力和必要。

后来人类完全控制了自然，能力增长，人口众多，人类社会就出现了三大问题：垃圾和粪便问题，强奸和斗殴问题，以及老人（病人、残疾人）问题。
这些问题实质是：改变了的生存环境和人类未变的生活状态的冲突，改变了的人际环境和人类适应部落生活的心理行为的冲突，脱离了自然选择环境后面临人工环境无法自然淘汰的伦理问题。

其中后者是最大的体现了人伦道德和人类现状之间的冲突，本来由自然淘汰掉的人现在可以活下来了，养活他们实际上并不会增加生产力，但由周围的亲人亲手杀掉他们或者有意识地主动抛弃他们，又可能会导致更多的健康人精神崩溃，失去创造力和生产力。（饥荒时吃了人的人，还有杀过亲人的人，心理都会扭曲）

孔子，摩西，默罕默德之类就是各民族专门解决这三大问题的理论建立者。

所有这些领袖都发现人类社会必须遵守某种道德，内部才能安定，人心才能有动力，就是说在实用性和精神满足方面要偏向一点后者，给人一个寄托。比如孝顺年迈到一定程度的父母，要求给老人先吃肉，不要让老人负载于路上，就是中华民族的一个传统道德寄托，一个奋斗动力。

当然，内部遵守道德的社会，在面临饥荒时不能随意让一部分人饿死，否则社会秩序就会崩溃，那只能将矛盾外化，而对外人（不同部落）杀戮这一点人类传统上是没有太大心理负担的，所以只要稍微宣传一下，战争就会很容易，”热爱家乡，奋勇杀敌”，以社会内部的认同感为动力， 并妖魔化敌人，这样一般也不会带来道德冲突和心理崩溃，除非是想太多没法洗脑的。灾荒粮食减产，在古代一定会死人，这种压力下社会会分散成更小的团体互相杀戮。事实上起到了灾荒消灭人口的作用，所以战争在古代又叫“天罚”。从这里可以看出人如何在变换的社会环境下保持道德感的。

道德感对古人很重要，因为那是人生价值所系，其实也是一种精神需求，道德感最终产物是宗教信仰。所以判断一个事情道德不道德，就看它是否符合人心的需要，是否能激发人的创造力和进取心，还是让人心迷乱扭曲堕落。

那么现代西方社会的很多构建，比如你说的一面高科技治疗，一面上高科技战场，从这个标准看是符合交战双方内部社会道德的，虽然在整个人类社会来看是有点荒诞的。美国大兵白天打游戏一样远程杀人，晚上回家抱孩子，只要这个大兵心理平静，那么这事对他就符合道德，虽然对被杀的居民来说，是极端不符合道德的。

现在很多人觉得道德没用了，也有很多人更加积极地维护道德，他们都有各自的道理，但工业化社会一个特征是，道德确实越来越没有用。

因为一方面越来越多的道德和前面说的一样，是被建构出来的，轻飘飘的，不值钱的，也不需要代价的，就像电视上和你说挤出你喝得牛奶的牛多么幸福，每买一瓶饮料就给失学儿童捐一分钱一样，让人很容易就自我感觉很好。另一方面侠客精神或者英雄主义在机器文明面前不堪一击，就是唐吉珂德和风车的关系。

但道德虚无也导致了一种普遍堕落的情况，比如吸毒的人越来越多，沉迷于感官刺激的人越来越多。于是又有很多人在这种不好的气氛下为了找到生活目标而追求道德和信仰。

所以总体来看，工业社会人类的道德越来越不值钱；个体层面上，人类越来越倾向享受在道德高地的快感，越来越喜欢谈论和传播道德信仰。

所以我承认现在道德的意义只在个人层面上，就像人一天吃三顿饭，多了少了都不行一样，遵守道德的主要目的是维持个人的心理健康和进取精神。仅此而已。
