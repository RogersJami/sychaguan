davidq 发表于 2013-8-19 17:13 static/image/common/back.gif
这个黄色的封面看起来有灰似的，建议下次封面别用这种颜色。

可能你拿到的这套里这本在装订的时候沾灰了。这个封面纸张原色是接近象牙白的，其他颜色都是染上去的，所以，待干的过程中，会粘灰。颜色不好掌握，不仅是这个黄色，包括红色和蓝色的，都跟我预计想要的效果有差距的。
