终于，买了个烤箱

昨天我用烤箱弄芝士焗番薯，感觉勉强，试多几次，拍几张照片，再来发 ...

关于泡芙的配方。因为没有买秤，所有计量全部靠宜家买的量勺。最大的100ml，第二大的15ml。

配方：
牛奶：160ml（水也可以，味道差不太多），
黄油：50g（包装上有刻度可以切），
糖：2匙：30ml，
盐：如果用的有盐黄油就不加，无盐的就加一小捏
鸡蛋：3只打散（洋鸡蛋2只就差不多了），
面粉：60g（我也是差不多用的100ml，家里只有全麦粉，有什么就用什么吧），

方法：
牛奶＋黄油＋糖＋盐一起煮开
关火筛入面粉。搅拌均匀，至光滑。这时候也冷却了一些，不会把鸡蛋烫成蛋花，倒入鸡蛋液。搅拌均匀。我看的配方说是拿起搅拌勺可以看到有约4cm的尖角。不过我是把材料都倒进去，提起来也都能成型就觉得差不多了。
最好是用挤花袋挤，如果没有，放保鲜袋剪个角来挤也好。后来偷懒，试过用勺子的，形状不好掌握阿。

烤：
首先了解自己烤箱的脾气。重要的是不要烤糊了，然后就是熟，其他的都好说。
我基本上开始是190度左右预热，放入烤10分钟。开始变色，定型。然后转175度再烤15分钟即可。中途不要开烤箱，略冷之后拿出来晾凉。

泡芙考好后里面自然就是空的。侧面切开就可以填充馅料。
如果是以绕圈的方法挤的，开始留个洞，考好后地步也是有个洞的，可以直接挤馅料。

馅料：
这个草莓，加的是卡式达酱
也容易，跟泡芙的配方很接近。
材料：
1.蛋黄：3个
2. 糖粉：2勺
3. 牛奶：250ml（一盒屋顶）
4. 黄油：20g
5. 低粉：25g（我是一勺全麦粉＋半勺淀粉）

做法：
1牛奶＋黄油加热，煮开
2蛋黄＋糖粉 拌匀后，筛入低粉拌匀
1倒入2中搅拌均匀，然后放在炉子上煮一下。一边煮一边搅拌，直至沸腾。

拿到一旁冷却就可以用了。

刚煮好的卡式达酱会有点点的面粉味，放一会之后面分为会降低。

这个东东不用打蛋，很容易的。 试试吧。
