三七是化瘀止血的药物，也就是说在止血和活血化瘀直接有个平衡，止血过了，后果就是血栓，如果化瘀过了，就容易引起出血。
这是从药理活性上分析。
从临床上看，我老师对三七的应用也是比较谨慎的。
活血的确可以祛斑，但是活血的药物有很多。三七只是其中之一。
三七有类似人参的滋补作用，这可能是很多人喜欢三七的原因
