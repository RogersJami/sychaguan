私有化要搞得好，关键就在于“既放得出去又收得回来”，因为现代企业生产的社会属性，是不随所有权的转移而改变的。转移所有权的唯一意义也就仅在于“让有才能又负责任的人来经营它”，从而使各方的合法利益最大化。

现有的私有化模式及其社会制度啊法理啊之类的安排，不能解决企业所有权落到“缺乏才能或者缺乏责任心的人手里”时，及时收回和赋权给其他的适当的人的问题。因此这种私有化模式，必然被能解决这个问题的其他模式，击败和取代。
