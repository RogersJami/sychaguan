用耳朵去读书

这里推荐几个比较有趣有益的有声读物，作品本身很有深度，朗诵者功力也很高，相得益彰。
（都可以在verycd下载）

1. 魏巍  《地球的红飘带》， 曹灿 演播
     非常忠实的反映红军长征的小说，读完之后基本对长征有一个比较全面的了解，故事性也很强。
2. 阿来  《尘埃落定》，        仲维维 演播
    矛盾文学奖获奖作品。描写西藏最后一批土司们的生活，非常有生活的智慧。 阿来原来是写诗的，因此听朗诵版的《尘埃落定》别有一番趣味。
3. 余华  《许三观卖血记》， 仲维维 演播
   许三观是个小人物，每当家庭遇到困难的时候他就去卖一次血，帮助家庭度过难关。这本小说不长，但非常有内涵，可以当作一部中国当代史来读。

4. 金庸 《笑傲江湖》， 周振 演播
  这个不用介绍了。 《笑傲江湖》的有声版本很多，我认为周振是最棒的。
