rafanadal 发表于 2015-3-3 10:25:51

不同的人有不同的立场和关注点，搅在一起就是现在这个样子。

这片子正好应时，资源枯竭环境污染是当年猫论发展生产力的必然代价，就看到什么时候无以为继，十八大报告里的建设美丽中国不是说着玩的。片中采访的对象已经代表了相当一部分体制内的势力，何况片子从发布到炒作的力度足以说明背后有人在造势。

而环保的根本问题，不是道德（柴片依然是站在高地上定向挥棒子的套路），而是成本。环保意味着投入，意味着关停，企业对此比谁都明白，而最终买单的必然还是民众。拿蓝天白云青山绿水忽悠人是容易的，可有多少人真的对这后面的失业、坏账、涨价有清醒的认知？恐怕大多数人高喊支持还是那种轮不到自己头上的心态。看看这些年从PX到垃圾焚烧厂的诸般折腾，是不是叶公好龙真得到时候看。

这两天柴在当年哥本哈根闹剧时对丁院士的访谈又被翻出来了，实际上围绕减排的巨大争议背后是巨大的利益冲突，丁院士非常清楚，柴装圣母只会让人恶心。而我当初恰好是看过电视，当时的感觉不知道柴是真的屁股歪还是一傻托。

从雾霾的成因到雾霾和健康的关系都是复杂的问题，而很多观众自身恐怕连关联和因果都分不清楚-柴自己能不能搞得清楚也存疑，何况专业问题早已是外行无力验证的。就连柴本人是不是老烟枪这种问题都能吵得真假难辨，可见获取信息跟验证信息完全是天壤之别。
不过既然柴都号称胜过央妈了，也别抱怨被人扒光嚼死了，这世界是公平的。

如果柴片只是搜集整理资料，通过现象来提醒关注问题问题，纵有偏颇也不会有太大争议。可要夹带其他私货就没意思了。
相对于风一吹就解决的空气污染，水污染和土壤污染才是真的本乡本土既要命又难以治理的事情。只不过在大城市人群占主导地位的情况下，最惨的更难得到关注。观众无知，柴是不是也无知?
至于能源问题更不是柴这种外行可以胡扯的。中国煤多油少，油气资源的缺乏是困扰已久的战略性问题。真要解决问题就上核电，可铀矿也是要头疼的，到时候一干环保恐怖分子更是麻烦，扯市场化就太歪了，市场化变不出低开采成本的油来。当然都知道这块超大的利益目前正在争夺中，柴片跳进这个大坑就完全丧失了环保的主旨。
真要说排放问题，脱硫除尘小电厂问题大得多，地炼黑炼的油品质量更是比本就不行的两桶油差得多（可要是真的喝上好油现在的N多老发动机又要绵软无力了），能源市场化如果弄成有水快流那就是大跃进的下场。

结论：
秦州牛吃草，益州马腹胀； 天下觅医人，炙猪左臀上。 续曰： 哑人偏会说，聋人偏会听； 何况不聋哑，几时得清净？
