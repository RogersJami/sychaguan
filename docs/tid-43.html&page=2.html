首先是我们都熟悉的，金字塔型的传统货币体系结构





两个货币体系对比，五行货币另一个革命性的改变，是经济体的改变。

传统货币结构，是我们很熟悉的经济、社会、权力的金字塔型组织。底层是大多数人，只能得到很少的利益，越向上人越少，利益越集中。我们被这个体系统治了不知多长时间，几千年，还是几万年？

五行货币体系结构重重叠叠，无穷无尽，无法描述成一个闭合的结构，只能进行区域的描述。在这个体系中，所谓的社会等级不起作用了，重要的是你在这个体系中成长的高度！你成长得越高，从体系中获得的利益越大！

因为在五行货币体系中，最大的特点是大部分的社会价值由人心来衡量，而人心所向的社会价值无法量化。如果你成长的境界很高，被别人发自心底地从内心中承认，那你创造的价值没有边际可衡量，也没有时间来约束，无量无边啊！

比如，人类历史上最伟大的思想、人物、艺术品、建筑等，若全面地以社会价值来衡量，那是无价的啊！

理论上，你可以凭自己的本事，建立一个五行货币的经济体，哪怕是小小的一个，也可影响整个五行货币体系。（这个说法比较简略，但请大家努力用心去理解）

因为其结构的无穷无尽，相对于我们熟悉的传统货币体系，五行货币体系要搞明白的确不大容易。不过一般人只需要搞清楚自己的定位，或者上下各一层的货币体系结构，就可以在体系中应对自如了。现在关键的是要相信，对其有坚定的信心！

各种成长：国家的成长、人群的成长、团队的成长、个人的成长，最后都归到心灵的成长里。在五行货币体系中，将取代身价、汇率、价格等商业概念，成为人们最关心的一件事。

谈到五行货币，一定要理解：货币是表象，分权才是实质。而且，国家将货币五行化了，才是真正的意义上的五行货币。在这之前，只能叫五行分权。

我听得最多的问题，就是利益从何而来。将这类问题很简单地总结了一下，全是市场经济的思维，就那么局限在商业的圈圈内跳不出来，想的全是如何把自己手上的东西卖出去。其实要回答这个问题太简单了，只要将胸怀打开，把提到的五个社会领域都装进去，想想如何去实现自己的价值，别人用何价值来回报你，自然会了然。破除我执，答案是如此地简单。

五行货币，五行分权，若整个体系是一个完整的身体的话，商业只是一条腿。请不要将一条腿看成一个完整的人，谢谢。

如图所示，如果五行货币体系看作一个完整的人的话，五个社会领域分别代表了人的头部和四脚。

商业（木）和文化（火）分别代表着所谓的物质文明与精神文明，是社会向前行走的两条腿。科技（水）和法律（土）是很精细又深邃的，以两只灵巧的双手来代表。行政是统领和指挥社会，进行社会资源调度的，那就是头脑啦。



若要理解五行货币，请抛开现在的市场经济思维。
东方天人合一的思维，着重体现人与人之间的关系，人与天地之间的关系。市场经济的思维，看重的是人与物的关系，人与契约之间的关系。思维一定要转到东方来，眼光要放在人和天地之间的规律上来，否则一头雾水是免不了的。

用来自西方的知识体系去硬套五行，那是用人类拥有的知识的一部分去理解整个宇宙的运行规律，而且这些知识还不一定完全正确，这行得通吗？



我们心里非常明白，搞这个网站的最终目的，是五行货币社会化，去充分体现社会价值。它不是商业项目，也不会为资本所左右，我们也不是为个人图利。请不要完全用商业的口气来提问，最好能全面一点，能涵盖整个社会，能点点滴滴地关照到每一个人。



继续，五行货币/分权体系中，利益从何而来？

说起来也简单，就是将现在依附在商业躯体上的，不属于商业领域的社会价值还原其本来面目。

将文化、法律、行政、科技领域内的价值，脱离商业化的桎梏，归至本位。

让资本回到“人类的工具”的本来原目，不再骑在人们头上张牙舞爪。

很神奇地，我发现大家所熟悉的，马斯洛提出的“人的需要层次”，与五行货币/分权体系中的社会价值不谋而合，而且从高至低是沿着相克的路径发展的。

当然，所谓的从高到底的需求，是对个人而言，对而社会而言，五种需求是同时存在的，不分高低的。

所以啦，五行货币/分权体系中的社会价值并不需要谁去刻意营造的，它是整个社会和所有人都共同需要的。

现在，我们要将这五类社会价值和人们的需要各归其位，各执其权。

这就是五行货币的意义。
