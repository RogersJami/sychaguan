我一开始也懵了。为什么会是昆明遭遇恐怖袭击。按理说西南地区除了昆明外，成都、重庆、南宁（贵阳可能稍差）也是省会或者就是中央的直辖市，这群恐怖分子为何就偏偏选中了——昆明。
幸而，思炎姐提醒看问题要看本质。
而抱朴仙人更是直接指出了云南在国家发展中担负起的重任。在新的时代，已经不能用以前的眼光——蛮夷之地、瘴气之地来看待云南了。现在的云南已经被中央委以：南亚和东南亚突破的重任。
以下是仙人的分析，和茶友们共享一下：

关于昆明火车站301恐怖袭击事件。很多朋友发来消息，表示关心和愤慨。对此，我心怀感激之心。
但是这些信息里有三种看法我不赞成：
1、认为这是某宗教的错，要惩罚之；
2、认为这是某民族的错，要惩罚之；
3、认为云南是是非之地，要回避之。
-----------------------------------------------
我的看法：
1、这不是某种宗教或者民族的集体意志更不是集体委托。认为需要集体惩罚的，就是默认这伙暴徒能代表维吾尔民族或者代表穆斯林主流。而这帮亡命之徒其实谁都代表不了！他们也不配！他们只是一帮在反华势力策划训练鼓动下，妄图搅乱文明社会进步脚步的炮灰而已。随着生存空间压缩，利用价值下降，他们被其主子抛弃只是时间问题，反咬其主人也是意料中事。是历史早就反复证明了的。
2、如果因此事而走上民族对抗，宗教对立的道路，因此而吓得不敢去云南工作，那就遂了这帮暴徒及其主子的心愿，那些鲜血才是白流了。
3、有些势力在努力构建围堵中国的陆上和海上包围圈，包围和反包围是今后一段时间的主要斗争主题。云南和新疆是中国打破围堵的主要突破方向。其中，新疆负责着向中亚的突破，而云南肩负着向南亚东南亚突破。更具体说，云南承担着建设大湄公河次流域合作区和中孟印缅经济走廊的任务。这是从睦邻富邻起步，面向南亚东南亚22亿人口，全面合作的历史使命。这个重大战略根本不可能因为有人捣乱，有人牺牲而放弃。
4、中国将继续走自己的发展道路，将坚持国内民族团结，对外睦邻富邻，和平发展的路线，并为此承受重大牺牲，当然也不会面对别人的挑衅一味忍让无所作为。团结少数民族，打击极端势力，分化瓦解，改善人民生活，消除极端势力的思想影响和生存土壤的基本策略会依旧坚持。
以后估计很多动作会浮上水面。会更多的国际联合，保持对恐怖分子的高压态势，高调施压，公开打击，打击其文化宗教政治保护伞、经济来源和公开代言人，给老百姓一个安心的交代。
5、我虽然不找死，可也不怕死，我倒是害怕无声无臭地烂在家里。要奋斗就会有牺牲，只要死得其所有什么可怕？要是被这帮人渣吓住了，他们不就得逞了？咱还怎么做人？
我是该干啥就干啥，后天就去昆明，继续工作。
为死难者祈福有必要，活着的人继续前行，做好自己的工作更是正道，是对恐怖分子的最有力回击。
