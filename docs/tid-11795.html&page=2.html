玩钉耙的悟空 发表于 2014-9-28 22:45 static/image/common/back.gif
谢谢白兄推荐。

只用来收发短信、拨听电话+路上看新闻，偶尔拍个照，要求待机时间长一点，华为系列中哪 ...
大师兄这需求，xun推荐的3C足以满足。

不过从大师兄的使用习惯上来看（时间久），我觉得大师兄可以考虑直接上荣耀6。

我搞通信的同学告诉我，联通的4G比移动好。因为4G和3G结合起来用最好。
1、联通的3G基站只需要固件升级就可以用4G，而移动得重新搭建4G基站网络。也即联通的4G基站覆盖要比移动好。
2、联通的4G和3G是一个技术体系，技术上来说，4G和3G之间转换流畅。

由上两点，联通的LTE-FDD开通以后，上网使用流畅性上应该要比移动好不少。

如果大师兄希望手机除了接打电话之类的功能，以后上网能更加方便的话，可以考虑直接上荣耀6.
等工信部批准联通的LTE-FDD的牌照以后，大师兄直接可以用联通的4G了。
