jxdz051277 发表于 2012-6-9 19:12:44

被这一帖子炸出来了，口水分泌旺盛，螺蛳粉，既是美食又是乡愁啊！

在柳州吃正点的螺蛳粉，一定是到小吃摊或不知名小巷里面的小店子吃才是最正宗的，一般柳州螺蛳粉的螺蛳味都来自于汤头（猪骨+螺蛳熬出的汤），有店家尝试过用（猪脚+螺蛳)熬汤头的，味道就是不对。然后再配上独门的中药材和香料，那就是一绝了，跟四川火锅似的，吃过出门身上一股味儿！而且柳州螺蛳粉的螺蛳一般用的不是田螺，而是石螺，就是那种手指头大小的螺蛳，螺蛳肉很小，但肉很甜，这样熬出的螺蛳汤才正宗！

柳州螺蛳粉是必须搭上炸过的花生米+炸过的腐竹+酸笋+豆腐果+黄花菜！
在南宁吃的螺蛳粉一般都没有黄花菜了，不知道为什么！
