我在施工企业和监理岗位都干过，房子出大问题（成为危房）主要三大原因：一是地基施工问题导致倾斜；二是设计问题（例如梁柱设计强度不够，或密度或尺寸不够，不过这种情况现在可以忽略不计，因为都是软件计算结构）或结构上偷工减料（主要是降低水泥标号和钢筋标号，或者施工工艺不合格，该焊不焊，该绑扎不绑扎，混凝土有空泡等等几十项影响质量因素），导致结构变形，楼板开裂，墙面开裂；三是使用海沙，导致混凝土因钢筋被腐蚀而膨胀，最后崩裂掉落，结构完全瓦解，导致楼房倒塌。这是目前主要问题，南方沿海海沙房子比比皆是，这种建筑寿命只有25年到35年，以后都得拆除。目前温州，宁波和深圳已经进入高危期，政府焦头烂额，又要压制信息外泄，以免人心恐慌，社会动荡，又要筹钱加固，以免出人命。

所以只要不是上述三种情况的质量问题，一般问题不大，装修注意一点，还是能够住的。上述三种情况是不能住人的，好像温州前一段就是倒塌一幢90年代中期的房子，死人了。
