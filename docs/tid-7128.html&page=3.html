我个人认为，马斯洛的理论说明，尊重和被尊重的需要与是否农耕文明无关，在工业化社会，尊重和被尊重的需要反而更加突出。

农耕文明，一个人一生的生活圈子，也不过是两三个村子。

工业化社会，从小学到中学到大学到工作，从同学到同事，从老师到领导，这个尊重与被尊重的层次更加复杂化更加具体化更涉及到方方面面。

这是人的社会性使然。因此我不同意“中国人爱面子”这一个说法。更不同意把爱面子归结于农耕文明。

有人就会有比较，哪怕是非洲的原始部落，也会比较谁的打猎技术高。
