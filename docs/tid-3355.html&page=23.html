我觉得倒不如加倍竭泽而渔让市场崩溃倒逼改革，要既得利益无故壮士断背是不现实的，自身利益受损才有改革动力，不是每次改革都是被逼的嘛。

现在的地价包含了对未来数十年中国经济高速增长的乐观预期，中国不像香港可以靠金融地产支撑社会，因为香港有大陆输血，大陆本身找谁输血？只要大陆增长暂时停顿，地价就会崩溃。这时乘机推出改革，打击诸侯经济和金融垄断，利用房地产释放出来的资源做全面产业升级重获增长动力。

中央只要保证增长停顿时的政局稳定就可以了，最大的不稳定的因素我觉得是“新城市人”。中央现在应该敦促地方大量建廉租房安置好他们，地方不肯出资建设那就收房产税再剪一次房奴的羊毛，反正他们都提前消费30年了也不差多消费10年，房奴是有恒产有恒心不会出什么乱子的。房产税确实不是什么好东西，将来用促增长的名义再取消就好了。然后只要重获增长动力，什么负资产，坏账之类都是浮云了。

罗杰斯先生对中国崛起非常乐观，但他也认为中国经济要在地产泡沫破裂后才能涅盘重生。崛起过程没发生过大倒退是不可能的，像米国，经历了多少次经济危机。。。
