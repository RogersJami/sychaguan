在河里看的

去西沙有几种方法：

1、通过西沙工委，坐琼沙号去。如果在岛上有认识的人，可以叫对方发一封信，持信就可以购买船票。如果没有关系，最好直接与海南西沙工委联系，参加他们组织的考察活动。这种方法一般就是去永兴岛。

2、通过渔民，坐渔船去。蚂蜂窝这个网站上有一些介绍，也有一些潜水俱乐部或者驴友组织与这些渔民有联系。这种方式的麻烦在于渔船可能比较小，晕船的可能较大，去永兴岛的可能比较小，但可以去甘泉岛这样的小岛。

3、经常有一些活动如联欢、科考、设备维修之类的，需要上西沙，可能有船或飞机前往西沙。

文昌市是个县级市。三沙就算是县级，也是和文昌平级。
海南是全国最先实行省管县垂直管理的省，后来，海口市、三亚市一把手高配，副省级，县级市领导也几乎都高配，副厅级，其余的县，才是正常的县级。
到于今，老百姓发现，这是便宜了一把手，各地方经济起色不大，可官升迁远远比内地容易。所以，在2003年左右的该省两会期间，出现了逆天的言论——
建议海南行政区划调整，成立地级海口、三亚、儋州、琼海、五指山、东方市，市管县。之前，海口、三亚这样的地级市，都是不管县的。还有言论认为，要恢复被撤销的海南黎族苗族自治区，同时成立几个地区……
当然，最厉害的，还是一位该省政协委员的言论：海南的简称“琼”和“穷”同音，不好，应该改。
该委员直接让上海市感到了巨大压力。

谢明中，泄密一事，实在不该。该父母官喜欢把教师和服务员骗上床，数额巨大。

文昌，基础设施的欠账，和贵州大部分县类似。可惜了。

朗朗天 发表于 2012-6-28 23:39:53

去西沙，永兴岛，不是没那么麻烦，而是很不容易。
坐船，渔船另当别论。如果是“琼琼沙2号”，那是不卖票的，三沙工委决定你能不能上船。这船不是客轮，是送给养的，至少两个月才开一次。
一个比较搞笑的办法是，先坐渔船，登岸后，称是被猴子的渔船绑架，自己跳海，偶遇海南的渔船的，本人强烈要求回家乡。鉴于飞机、轮船航班都很少，你就可以在岛上观光一小段时间了。
