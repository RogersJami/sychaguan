还有一个更无语，2003年来深圳的老乡，父母攒了3OW，愣是说等深圳房子跌了买个大的，后果嘛，我就不说了，幸好是个女孩，外貌收入也不错。现在过得也算步入正规了。

我之前就说任某人也好，鼓吹投资房子的也罢。人家在那个时候是没讲错啊。茶馆有人回复我要是当年不放水如何如何，我只想问一句。要是有如果，哪里来的那么多现在扎堆买房的。我搞不懂为什么总喜欢说当年如果，当年都过去了。人家低利率时候已经买了，我同事投资四五套的都有开始套现一半的了。搞不清楚这类投资客会有啥特别坏的下场。
我觉着就是投资就愿赌服输，跌了也别怪人家，当然涨价了也是合法收入，一线城市的房子就现在算到未来十年，我不敢说如何增值，但是是目前能找到的保值最好的工具。
可能从长期看，房子也会回到正常范围，但是这十年混在大城市的没办法，小孩上学，入户等等，用房子来赶人的趋势并没有变化。只能被推着走。活在当下
