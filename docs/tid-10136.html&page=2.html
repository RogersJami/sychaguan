凌晨完整地看了皇马VS马竞，说点观后感。

1. 迪马利亚拯救了皇马，而不是93分钟进球的拉莫斯。

上半场马竞逼抢凶狠，抢下皮球就地进攻，2-3人之间的小配合有声有色，把皇马大牌们玩得团团转。皇马BBC之间的联系被完全隔断，几乎看不到这3个人的身影，整支球队瘫痪，没有战术可言，只在玩抢球游戏，但迪马利亚的2次边路突破威胁足够大，改变了马竞的下半场打法。

为了遏制迪马利亚的威胁，下半场一开场，马竞就龟缩在球门30-35米内防守，任由皇马来攻。这种局面，即使抢下皮球，也不可能迅速进攻，放弃了中前场的逼抢，也就放弃了防守中带进攻的打法。局面被动，西蒙尼换下加西亚，换上索萨，意图加强右路反击，但气势不再，而且思想不统一，防守时队员没有相互之间的呼应，皇马的进攻空间越来越大，几人之间的配合也打了出来。

2.  关于进球，加时之前打成1:1，都是靠角球破门。

rebos和其他茶友早有先见之明，破门靠定位球。

马竞的角球进攻，基本上都能抢到第二落点，特别是在禁区前弧顶区域得球后再往里吊，形成乱战，几次之后，对防守的心理考验非常大。戈丁的头球本来威胁不大，卡西贸然出击有一定责任，但只要是乱战，在所难免。

皇马的角球战术不同，第一层争点队员卡住位置，把防守队员隔绝在角球落点区域之外，第二层进攻队员抢点直接攻门，拉莫斯头球破门之前，屡有其他球员头球攻门。

我不这么认为。
迪马利亚功过五五开吧。造成bbc散步的因素不光是马竞的防守，迪马利亚的过多失误也是一个重要原因。
像皇马马竞这样的比赛，进攻的成功率是非常重要的。
换人前，迪马利亚经常都是在中路靠左侧附近活动，左路金毛不能给迪马利亚很好的支持，好不容易拿到皮球组织进攻，当BBC都百米加速跑到禁区时，迪马利亚已经把球还给马竞了。
这种失误不用多，两次下来再好的球员对跑位的积极主动性都会打折，这直接造成了前场短链。
反观马竞，在很长一段时间能对皇马后防线持续压力，也是没有迪马利亚这种野球选手的结果（对战术的高度执行力）
换人后，迪马利亚更多的在边路活动，伊斯科和马塞洛数次挽救了迪马利亚的丢球失误，这是皇马能够对马竞持续施压的基础。
像迪马利亚这个类型的球员是教练既爱又恨的，尝试过人次数无数，成功率极低，当然造成对手犯规和定位球是可以当开瓶器的，但是一旦丢球，很可能就是对手的开瓶器了。迪马利亚唯一的一次突破成功就是加时那次，也制造了决胜的进球。
不过这场迪马利亚的表现还是回报了安切洛蒂的，安切洛蒂是一个非常迷信感官数据的人，当年不用皮尔洛，也是如此。
真正挽救球队的还是拉莫斯，我们不能说没有迪马利亚的突破传中就没有拉莫斯角球进球的机会。拉莫斯这场攻防两端都是很出色的，在最后阶段一度还加入了组织进攻。
如果没有拉莫斯这粒进球，赛后最受人指责最悲惨的就是卡西和贝尔两人。 卡西的出击失误和贝尔三次不传球浪费良机。
下来就得数迪马利亚了，他浪费的机会和创造的机会一样多甚至更多，这样的球员在现代足球战术体系里都是搅屎棍的角色，有些战术严谨的球队更是没有这样的角色存在。所以当今欧洲足坛著名的野球王已经可以用五根手指数的出了。
虽然西班牙的教练在球员青年时期都会觉得踢野球出来的孩子天赋高，但是在逐渐成年后，这些野球出身的孩子如果不能转正，除非达到了鲁尼迪马利亚的级别，那基本上都是要淘汰的。

虽然我是皇马死忠，可是我不喜欢c罗。
c罗是非常优秀的运动员，勤奋天赋职业而又有野心，欲望。
但是任何一个时期的c罗我都不认为他是世界第一。
我对球员的喜好趋向是受我个人踢球的球风影响的，也可以说是看球影响踢球，然后踢球又影响看球。
在皇马队中，我更喜欢笨马，虽然很慵懒意志力略弱，法国人就这样吧。但是只要他跑回中场接球，发起进攻，那么十有八九都是极具威胁的，笨马为c罗牺牲了很多，笨马更适合影锋而非柱式中锋。也喜欢伊斯科和赫赛。 伊斯科在马拉加时我还专门跑去看过一场主场的比赛，在马拉加他的统治力很强，在皇马的迷失只是暂时。 赫赛去年在卡斯蒂亚时我也过他很多场比赛（因为皇马季票可以免费看卡斯蒂亚青年队的比赛），赫赛身体素质好，技术方面未来能接近小白，就意识方面未可知，不过目前的表现已经很不错了，一度把迪马利亚和贝尔压在替补席上。卡西拉莫斯就不讲了，大爱。另外卡尔哈尔真的是成熟非常快，一个赛季下来，已经有当年拉莫斯的影子了。
下面我讲讲我对c罗贝尔的看法。
我始终认为足球的核心魅力更偏向于集体及个人的技术水平，c罗贝尔这样的田径选手的出现只是当前足球发展的一个极小阶段。 当然足球史上不乏各种快马，但是能为人津津乐道多年的多是一些速度技巧极佳的球员。  就那大罗的钟摆过人来讲， 当今足坛已经可以无法看到看到那样有迷惑性的假动作。 你可以看到大赛上非常有经验的后卫边后卫看到c罗假动作，连动都不会动的，因为几乎没有迷惑性。c罗腿摆的是快，可是身体重心没有变，重心不变根本骗不过高水平后卫。大罗钟摆的每一下重心都在改变，这对膝盖脚踝的伤害非常大，所以我们这些能有幸看大罗那个时代球员踢球的人是非常幸福的，现代球员都在尽量避免这样的损伤。c罗贝尔各有优势，c罗脚下比贝尔灵活些，贝尔的速度更imba点（比巅峰c罗）
c罗大赛迷失的主要原因是个人技术局限，次要原因是皇马的进攻策略问题，可是皇马的进攻策略也是为c罗量身定制的，所以倒推过来原因还是c罗的技术局限。c罗贝尔技术特点重合度太高，有机会皇马会想把c罗卖掉的。虽然遇到强队他俩都能够牵制对方防守，但是本方破防手段也被完全限制了。遇到强队笨马的思路完全就不是一个想进球的前锋的思路，跑位都是在帮助他俩扯空间，笨马这场决赛其实已经很上进了，有三四次回撤拿球组织，平常他都是等着球来，有一半c罗的野心，也不会这么混啊。
再往后讲就有太多细节啦。看世界杯时有没有空摆一摆。
