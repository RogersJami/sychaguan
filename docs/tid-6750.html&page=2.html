我说下一些基层办事的原则吧。这事我就看了微博，隐约记得是好像据陈水总说是基层把他身份证记错了，现在要改身份证来领取低保。
首先，钱从财政划出后，这块属于中央管，中央的想法是，任何政策的财政支出不能恶意的膨胀，为了保障支出总数不会因为突然的剧烈膨胀而导致该项政策的无法再继续，中央会对任何执行的政策门槛实行严厉的不近人情的把关。当任何政策一决定后，基层弄错的，或是群众像跟风的，那是门都没有。毕竟，每个王朝的灭亡首先源于财政的破产，而我国财政的大致政策延续当年山西土老财的做法，异常保守。
接着，我们从民众这方面看，群众看到好处那就是有时那时今天东一榔头，明天西一榔头。因此，有些时候是来不及和社会变化。曾经，大部分共和国一代人改小了年纪，就是为了能在工厂多做几年，多吃几年国家饭。但是，下岗后，发现反而早退休早好，这就是我们的时代太快，谁也不知道下一年是谁得好处。但是，作为一个政策，你今天敢给陈开这条缝，明天就是成百万的人过来要改。
最后，我们从基层看，任何一个基层都不愿做坏人，如果方法可行，谁都愿意做好人，但是这关卡是中央把持的。在生活中，任何几个单位联合办事，不是责任单位，那章改的可是飞快。但是中央要求的是矛盾解决在基层，但是这个问题是基层解决不了的。同时，作为面对当事人的基层，有些话是不能说的，因为这叫把矛盾指向上级。那么一般也只能用一些行动进行暗示，这事实在做不了。最后，碰到想不通的，只能拖了。

derek 发表于 2013-6-20 20:13:46

1.对于一个“按户算拆迁补偿，离婚可以多算补偿面积”的谣言出现，然后办离婚的老百姓就从登记处大厅排队到大街上转几圈的国家（此事前不久曾大热过）。改年龄的口子，是没有那个领导敢开的，也是不应该开的，否则就是彻底的失职渎职。

2.陈水总如此偏激，跟他的个人经历有关。看到报道中讲他们兄弟姐妹8个，4个吸毒，只有大哥有能耐有经济能力逃离那个环境，其他7个都没能力逃离，4个吸毒的就不用讲了，是不可能有钱买房租房的。包括陈水总的其他三个，虽然甚至成立了家庭有了孩子，但是也只能继续和自己吸毒的兄弟姐妹蜗居在一起。

没有人愿意成长在这样一种极端的环境中，更不会愿意长期生存在这种环境中，这种环境中的人的思维的很多方面，跟常人是根本不同的，所以陈水总会干出这种极端的事情来。

也因此，这件事的特殊性太强了，企图从中总结出普遍性的东西来，是很不明智的。而硬拿这件事中总结出来的东西，去做普遍指导的话，难免荒谬。

3.跑22趟，杀47人，其实完全可以安逸的再等一年就把事办了——这是一种典型的病态偏执。

4.按俺的思维，倘若看到文件上有“离婚可以多算补偿面积”的漏洞，本着“有便宜不占王八蛋”的中国人原则，俺也会排队去离婚。倘若看到“哎，我可以开个证明，提前一年领退休金”，我也会跑到有关部门递根烟，拍个马屁，能办了俺多占一年便宜，办不了就算了，无非是“有枣没枣打一竿子”。

实际上，本着这种思维，跑去改年龄的人多得很。有的人改成了，后来被发现了，被纠正了，还被处罚了；有许多没改成，碰一鼻子灰。但是跑出来杀47个人的，实在只有这么一个。因为实在很少人7兄弟姐妹（有的还成家了，例如陈水总本人）挤在28平米的地方，其中还有4个吸毒。

西方倒是有，例如“猜火车”，老兄可以看一看。

这个困难群体是特殊的，正常的困难家庭，都是虽然困难，但是还在努力奋斗，你帮助他们，他们会感恩，会更努力；而且只要有希望，这些家庭就不会垮，甚至苦中作乐，乐在其中，呈现出一种“革命乐观主义精神”。而且在中国整体大发展的背景下，这些家庭你帮着帮着他就出来了，走出困境了。

而这个“特殊的困难群体”是截然相反的，基本上是一句话——“你帮帮帮还帮出罪来了”，很荒唐，但是这个群体就是这样的。
