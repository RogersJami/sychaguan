windows下PDF文件简易制作流程


1.    首先需要安装Adobe Reader。

2.    其次请安装这个软件 pdfFactory Pro（http://ishare.iask.sina.com.cn/f/14625778.html?retcode=0），解压缩后点击setup.exe安装，一路默认安装。

3.    打开要转换为PDF的文件，点击 文件-》打印，在弹出框选（pdfFactory Pro），按确定按钮。（免费版本有水印，输入注册码后就没水印了）。

4.    按弹出窗口左下角的"View PDF" 按钮。

5.    在弹出的Adobe Reader中按 文件-》保存副本， 这就好了。
