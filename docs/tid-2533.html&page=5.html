亦左亦右 发表于 2012-6-30 00:12:24

湖南教育厅为何“躺着”也“中枪”
http://www.changjiangtimes.com/2012/06/395829.html
湖南省教育厅就“崔永元公益基金乡村教师培训”项目回复“不反对、不支持、不参与”，引得崔永元在微博怒斥对方“不努力、不作为、不要脸”，随即成为近日热门的网络话题。

只要否定不了崔永元牵头的那个项目的合法性和公益性，湖南方面除了妥协，其实没有别的路子好走。崔永元的项目要继续实施的话，如果没有官方的某种支持或配合，实际上也办不下去。故而，这件事开骂后的走向，就是沟通对话，达成某种协议。

果然，到6月14日，双方言论均有所缓和。崔永元称“如果湖南教育厅觉得我哪些方面不对或者很过分，我可以飞到长沙去给他们道歉”；湖南省教育厅方面在接受当地媒体采访时回应，当时是出于政府机构不宜代替民间组织直接发文和参与组织所作出的答复，目前已推荐相关机构联系崔永元公益基金会，给予其必要的协助。

双方做出的妥协都很大。崔永元的道歉条件，是对方“觉得不对”就可以，态度极好，似乎好到不讲原则，不过这种好态度似乎“以虚求实”；而湖南方面“给予必要协助”的表示，等于彻底改变了对那个公益项目的观感，由不作为变成有所作为了。

崔永元公益项目在与湖南官方缺乏沟通、得不到起码配合的情况下，实施难度可想而知。一个牵扯到政府内部事务的公益项目，政府要是“不支持、不参与”，实际上就有反对乃至否决之效。受训的人被当地政府管辖或制约，外出参训需要批准，培训效果需要当地政府监测与承认……当地政府在所有事情上不支持、不参与的话，该项目将很难实施。

而今崔永元公益项目有了一个意外收获，即以“网络公开对话”的方式，倒逼当地政府适当作为，做出了一种“示范效应”。由于“名人”的缘故，事先和当地缺乏沟通，成了直接上升到“网络对阵”的必要条件，“透明的对话”取代了餐桌和会议室的不透明协商，使得民间与官方之间的谈判，在众目睽睽之下热热闹闹地进行。这种方式没什么不好，此对话环境可激发各方言无不尽，一切都晒在阳光下，“参与”无数，“评委”众多，对话成本很低。

实话实说，崔永元牵头组织乡村教师培训，在和项目涉及地官方的沟通方面，确实存在不足之处。无论该项目多么名正言顺、资质充分，都需要得到当地政府的认可与支持才行。2007年9月开始实施至今的项目，在当地政府那里却得到“三不”的对待，这就是项目本身的一处败笔，对公益投入和培训效果，确实存在负面影响。

湖南教育厅对崔永元公益项目“不反对、不支持、不参与”，究竟出于何种原因呢？对一个好项目、好策划不该如此，对一个坏策划、坏项目，也不该采取“三不”的态度。也就是说，对有公益活动，无论好事还是坏事，政府都不可以采取“三不主义”。

对湖南教育厅来说，崔永元项目尚未落地之时，就该有所作为。先需要考察组办方人员的资质，做出是否合法、是否合格、是否可行等等基本判断，然后还要分析优点与不足，提醒项目方注意事项，进行效果监测反馈等活动，实施期间若发生严重问题还要采取叫停或指令整改等行政措施。

那是不是个合法、有益的活动？是，不支持就不对，就失职。难得有人主动投身公益，人家出钱出力帮政府的忙、操社会的心，你凭什么不支持？假如不是好事呢？政府“不反对”也不行，不去监管、制止，那也是失职。

当地政府的“三不式”态度，其实是矛盾的，于理不通。还有一种可能，即拿不准、看不透，于是就采取了鸵鸟式动作。但是，这个名为“公益”的项目，葫芦里究竟卖的什么药，正是当地政府必须搞清楚的事情，一时半会拿不准情有可原，人家从事某种活动好几年了你还拿不准，就有点不应该了。

躺着也中枪?不相干、不担责的普通人可以喊冤，但对政府来说，“躺着”正是“中枪”的主要原因之一。和大家都不相干但产生负面后果的事情，很难说与政府不相干，做大家都不做、都不负有责任的事情，本就是政府的分内之事。
