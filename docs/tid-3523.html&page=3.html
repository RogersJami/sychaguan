　　新浪文化读书讯  11月2日，据日本富士电视台报道，在日中国记者萨苏[微博]被日本兵库县警方书类送检，报道称萨苏不仅涉嫌欺诈，骗取日本失业救济金15万日元(按当前汇率，约合11713.8082元人民币)，还协助收集自卫队等情报资讯，为中国媒体撰稿。萨书本人对部分嫌疑予以否认。

　　此事件随后传播开来，甚至被误传成萨苏“骗保”被起诉。但截至今日，萨苏本人并未在公开场合对此事表态。新浪读书今日下午连线到萨苏，萨苏表示，他之所以到现在一直不发声，主要是想保护仍在日本的家人，避免他们成为舆论焦点而受到伤害。此外，国内传闻的所谓日方起诉也并非事实，萨苏说到目前为止日方并未提起诉讼。萨苏称他迟早会公开澄清此事，但要先等家人安全回国再说。(文/邱琪)
