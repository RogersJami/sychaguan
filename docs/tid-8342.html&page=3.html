     世界文明走到今天，对各种古典文明的直接传承都已经很少。都是在旧有基础上修修改改敲敲打打，不断适应新的发展。我们49年以后就已经放弃了对中国古典文明的直接传承，再倒数98年，中国还有过一轮基督教革命。是中国人早就数典忘祖了么？当然不是，是旧有的生产关系不适应新的要求了。自1840年以后，各种思潮和各种组织在中国基本上都试验了一遍。最后中国人选择了马克思主义中国化。
     所以在本帖中我对葡萄的一个回复中说：公元3世纪，基督教伴随着西非奴隶运动崛起不是偶然。

      就目前看来必然不会是以儒家思想为代表的中国传统哲学和建立于其上的生产关系。因为那样的生产关系已经不适应现代社会了。换句话说：想搞也搞不成了，因为没有那个经济基础，上层建筑盖不起来。既然搞不成就不能再搞了。原有体系就必须打破，打成零件，能用的敲敲打打装到现在的体系上，这就是“文化改革”。
      改开35年来我们摸着石头过河，但是这35年也不是铁板一块。有个挺明显的分界线，就是94税改。税改一举打破了改开初期放权给地方政府，最后地方吃肉，中央政府喝汤的局面。开启了中央政府集中力量办大事的新时期。但是也为地方政府卖地财政埋下了伏笔。现在我们都知道坏处在哪，知道了就需要调整一下，既然当初是税改的肇因，就从税改再来。这就是“财税改革”。
