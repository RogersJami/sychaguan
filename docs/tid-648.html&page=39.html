目前我们国家少年体育运动有什么状况呢？

政协委员提案呼吁：“音体美”不能成为“应景课”

2012年03月31日


【全国政协委员徐辉：现在中小学体育课的内容和时间安排不够科学合理，学校往往以平安无事为基本要求，不愿或不敢组织有适当对抗性的体育运动。青少年的健康，是国家的财富，民族的希望。大力推进素质教育和健康教育，积极发展学校体育活动，促进青少年健康成长，是政府、社会和学校义不容辞的共同责任，是贯彻落实国家教育方针的基本要求。要尽快改变学校体育“说起来重要、做起来次要、忙起来不要”的状况，扭转青少年健康滑坡、体质下降的不利局面。】

中小学生体质状况堪忧

【　3月中旬记者做过一次体质健康状况问卷调查，调查数字显示：在被调查的200名中小学生中，67%的学生存在轻、中度近视；57%的学生在体育测试中勉强达标，19%的学生很难达标。在采访过程中，记者了解到，我市的中小学生体质渐呈下降趋势，这其中的原因有很多，“重智轻体”意识、学生运动量不够是主因，而遏制这种体质下降趋势的办法应该是调整课业，加大学生运动量。

　　体质下降的主客观原因

　　去年，新抚区教育局开展了老游戏进校园活动，在千金小学试点时发现多数学生不会跳皮筋、丢手绢等老游戏，也不愿意做室外运动。很多老游戏被淡忘，取而代之的是各类电脑游戏。

　　今年14岁的学生孟淑伟好静，只要一有空闲时间，他便上网聊天、玩游戏、看电视，一坐就是几小时。他不愿意运动的理由简单，一是怕累，二是也没人跟他玩。

　　10岁的邵然喜欢参加各种体育活动，也试着实现自己的想法，却遭到了父母的反对。他只好每天放学后呆在补习班里写作业，等待父母接他回家。每个双休日除了在家学习，就是上补习班、书法课、英语班、二胡班，一坐也是好几个小时，基本没有运动时间。

　　家长学校均需承担相关责任

　　当下各中小学存在一种现象，体育课常常被文化课挤占，跳箱、单杠、鞍马、爬杆等项目因存在安全隐患逐渐淡出课堂。这种现象从个别到普遍的主要原因大概有两点，一是家长担心孩子身体受伤；二是学校不愿意担责任、怕惹麻烦，因此有时会压缩体育课时、停止上述有风险的教学内容。

　　育才小学教师刘晶从事德育工作18年，经常与家长打交道，处理过涉及学生安全的投诉。刘晶说：“在体育教学中，只要存在安全风险的内容，就会遭到部分家长的反对。”基于此，一些学校的体育课开始简化，尽管有些中小学校保留了跳箱等体育器材，但出于安全考虑，这些器材也多处于闲置状态。有的学校在操场翻建时也将单双杆拆除了，这些锻炼学生平衡性、跳跃能力及手足配合能力的体育技能项目，与现今的学生就这样擦肩而过了。】


缺少老师和器材 中小学体育课被严重“边缘化”

【青海新闻网讯 “中小学在体育课时间上语文数学外语，这种做法不但剥夺了学生锻炼身体的机会，也打击了学生学习的积极性。为了孩子的健康，把体育课上起来吧！”一位关心下一代成长的热心人士在给互助县县长的信中这样写道。

　　体育课为何上不起来？笔者带着这个问题走访了互助县几所学校了解到，由于学生的相关考核不包含体育，学校根本不重视体育课，甚至连体育老师都“不受待见”。每周两节的体育课基本上都被语文数学外语“霸占”，课程表上每天一节的活动课则大多是班主任的补习课。

　　一位老师透露，学校为了升学率主要看中的是教学质量，评优评先奖优罚劣都在看教学质量，相比之下，其他的就管不了那么多了，体育课被“边缘化”已经很普遍了。

　　另外，农村中小学缺乏专业体育老师也是体育课上不起来的一大原因。互助县哈拉直沟中学有878名学生，真正具有教学任务的专业体育老师只有4人。“根本忙不过来，一个老师同时带几个班一起上课是常有的事。立正、稍息、解散、自由活动成了体育课的主旋律。”一位姓朱的老师说，“我们缺人更缺专业的体育器材。”】


专家称当前体育教育存两大问题 吁家长转变观念

【对于当下中小学体育教学中存在的问题，沈阳体育学院体育教育学院院长王新坤颇有感触，“一是教学安全问题；二是社会各界，特别是家长要转变观念，要鼓励、支持学生积极参与学校体育课及课外体育活动。 ”

　　体育教学安全问题如何解决？王新坤认为要从根本上解决问题，还需在国家层面制定完善的相关政策，“教育主管部门对目前的情况很了解，正在研究制订相关政策，如意外事件责任如何界定、学生意外伤害保险提升额度等政策，很快就会出台。 ”】

北京重污染天停驶部分公车 中小学停止上体育课

【本报讯(记者白冰)本市今年将完善重污染日应急预案，如果遇到重污染天，中小学生将停体育课，部分重污染企业将停产，在京党政机关及企事业单位停驶部分公车。】

体育老师难就业 体育课改革遇瓶颈

【近日，记者在走访几所中小学时发现了一个普遍现象学校每年在其他学科都会招入新老师，唯独体育老师的人员配置是“十年如一日”。

　　春节过后，记者回到自己毕业的沈阳市沈河区一所小学，发现大多数学科的老师不认识了，唯独来到体育教研组，几乎每一位老师都能叫出名字来。经记者调查，在师资力量的充实中，学校近几年往往是在主科上加强师资，而被边缘化的体育课，只是为了完成教学大纲的学时。】



我的一点小看法，欢迎指正：

第一独子化导致目前家长害怕孩子做大量运动时候受到伤害
第二医疗体系问题不能为孩子运动提供更多保障，学校体育课只能尽量减少难度和运动量
第三追求升学率和就业压力，孩子多数时间是死记硬背没有余暇投入体育锻炼的状态

还有我跟部分教师朋友交流过：

独生子女的教育问题现在更多靠老师哄着，捧着，骂不得打不得害怕被家长投诉，孩子普遍超重营养过剩追求垃圾食品，不爱运动。

农村学校因为资金少器材少，孩子接触东西少，最锻炼的对抗性的球类等等运动接触不多，但是基本运动量比城里要好很多。

就精神素质和运动素质来看，现在年轻人远远达不到毛主席当年的那种气魄还有社会普遍追求自强的那种风气。

农村目前食品安全问题远远比城市要严重很多，新闻报道里屡次出现问题食品专供农村农民工，地沟油等等也是重点向农村倾销（因为城市目前普遍加大监控力度）

就目前而言，中国的崛起还是要靠广大农村青壮年劳动力的，而且就算军事力量来说主体也是农村青年，农村人口的身体素质和健康保障某种程度来说也是和我们国家未来密切相关。
