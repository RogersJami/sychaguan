我不吃青椒 发表于 2013-5-6 20:02:55

说说我的看法。这里的“节制”可以看作是对所占“资源”和“权利”的节制。无论是在哪种组织形态下，你占的多，别人就会占的少。没有节制的占有最终会让组织失去凝聚力，而政治家是靠聚人而彰显才华的。这是节挂的本意。这一挂也不是给人臣的，而是给仁君的。君不见“天地节而四时成，节以制度，不伤财，不害民”？制度不是现在的“制度”的意思。而是以“节”控制“程度”。而能改天换日止离散而聚民心的根本就不是人臣所为。所以拿周总理来举例子并不恰当。要说的话，就说宋文宗，《宋史·仁宗本纪》：“宫中夜饥，思膳烧羊，戒勿宣索，恐膳夫自此戕贼物命，以备不时之须。”膳，进食。宋代叫烧羊，现在叫烤羊。戕贼，摧残、伤害。宋仁宗夜来饥饿，想吃烤羊肉，但他不肯下旨索要，因为怕底下人从此为备不时之需，每日宰杀，天长日久，伤害生命无数。这叫“节”。即使宋文宗很没本事，但凭借着“节”。谥号能得个“仁”字已经不错了。
