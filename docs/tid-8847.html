活动征集原帖：

2013已经进入最后的倒计时了，马上就要引来新的一年 — 2014。 在这辞旧迎新之际，茶馆邀请你参加【茶歌飞扬】卡拉ok活动。
回顾2013年，论坛最重要的事件当属茶馆一大，笑声犹在耳边，歌声未曾走远。 那天，有幸听得了不少茶友的歌声， 在饭店里，葡萄放歌一曲李宗盛的《鬼迷心窍》，还有两位茶友合唱了《人生何处不相逢》，很有意义，恰如其分的歌。  因为我当时正被打岔说下一步去KTV的事，所以只记得了歌声，但不知具体是谁唱的。。。请那两位茶友冒泡认领，也欢迎其他茶友揭发 {:soso_e112:}


一行茶友浩浩荡荡的奔赴KTV，flightp、chalet、Ssun1cn、鹦鹉、emiT、宋兵丙、悟空、laftodeath、dadamm等都一展歌喉，其中不乏令人惊艳的声音；当然那天麦霸奖，当属chalet {:soso_e179:}


可惜，黑暗的骑士刚到KTV就被一通电话叫走，奶大有容原本想好好休闲一下，也因电话急事，意犹未尽的早些离开。所以这里邀请所有茶友，特别是未能唱尽兴的、和遗憾无法参加一大的茶友们，踊跃献声【茶歌飞扬】活动吧。。。。请大胆放歌，想唱就唱,唱得响亮！在你的歌声里，有茶友的微笑和鼓励！

http://soso1.gtimg.cn/sosopic_f/0/15674094230225306312/100


活动要求：歌曲不限风格，不限唱法，民族、美声、通俗均可，也欢迎用以曲艺、戏剧、乐器等表现形式。

*请参与活动的茶友在主题帖前加上前缀【茶歌飞扬】+曲目名，方便颁奖及活动汇总 {:soso_e183:}

活动时间：2013年12月22日 — 2014年1月30日（农历小年夜）

奖励：1.凡参加活动的（主题含原创演绎的音频MP3或视频）将获得100金钱奖励 + 留念勋章一枚（在名下保留6个月）

2. 获得茶友【支持】好评最多的前5名将获得额外奖励：200 金钱
请茶友积极给予献声的“歌手”鼓励，点击主题帖下的【支持】icon （如图）：

http://img15.poco.cn/mypoco/myphoto/20131221/11/64772759201312211158444002391019119_001.jpg




制作录音方式方法提示：

伴奏音乐：1. 伴奏音乐网站：http://ok.wo99.com

2. 酷狗伴奏音乐网http://mp3.sogou.com/music.so?query=%BF%E1%B9%B7%B0%E9%D7%E0%D2%F4%C0%D6%CD%F8&w=02020800&sourcetype=hint_4&p=40130201&dp=1
3. 5Sing伴奏网http://bz.5sing.com/index.shtml

音频视频制作软件： 1.使用免费录音软件Audacity录音，下载地址：http://audacity.sourceforge.net/

详细安装及使用方法可参见本版面置顶的帖（作者：重重无尽） 【旧帖搬家】简易录音程序http://www.sychaguan.com/forum.php?mod=viewthread&tid=3837

使用方法
1.下载Mp3伴奏到本地
2. 打开Audacity，open 文件（伴奏MP3）
3,点击录制，对着话筒唱
4.后期效果制作
5.点击导出，制作成MP3文件
6.将视频转为视频（这一步不会的茶友，可将录制完毕的MP3 email to：admin@sychaguan.com



2. 下载卡拉OK软件，“酷我K歌”，网址：http://k.kuwo.cn/点击网页，可见安装使用方法说明，有电脑版，手机版下载版本。

使用酷我K歌过程中遇到问题，可参见楼中讨论
http://www.sychaguan.com/forum.php?mod=redirect&goto=findpost&ptid=8847&pid=181225


3. 使用Iphone或安卓系统手机唱歌并录音，可用软件“天籁K歌”，软件可以在官网和各大下载网站下载。官方网页：http://www.tlkg.com.cn/





录制完毕后，上传：1. 将音频视频上传到优酷，土豆等分享平台，然后在茶馆贴中的工具键里输入视频地址。

2. 遇到困难的茶友，可将作品制成mp3或wmv格式，再作为附件 email to：admin@sychaguan.com， 在邮件中请注明茶馆ID，简要说明。 然后你将会收到回信，获得可贴到茶馆论坛的视频链接，发主题帖。


虽然看似麻烦，其实只要试过，练练后也是很简单的事。一旦第一次搞定，不仅学会了一样新东西，今后也可以自娱自乐，一个人的卡拉也ok！ 特别对于潜水或懒得码字的茶友来说，这也是桩好消息，吼一把，还得到100金钱呢！

期盼茶友踊跃献声，支持【茶歌飞扬】活动 {:soso_e166:}
