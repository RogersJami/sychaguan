这个就需要定量分析了。
比如说，水泥、钢材产业的工人有多少房奴？农民工有多少房奴？他们中间不是房奴的那一部分，收入增加了就促进其他领域消费了嘛。
至于政府收上来的土地出让金和税收，除了给政府公务员、教师、医生什么的发工资福利促进消费外，还有支付给农民的拆迁补偿款和安置房，道路交通等基础建设，这又是一大笔投入拉动。

其实有一个更简单的算法，把具体花费什么的放在一边，直接算总投入。一个国家是全民高储蓄消费量大，还是一部分人高储蓄（有房无贷款的）、另一部分人无储蓄大规模借贷（房奴）消费大呢？很显然是后者。

把老百姓的储蓄弄出来在国内花掉，不管什么方法，都是促进国内消费的手段。如果花光他们储蓄的同时还能让他们借钱花掉，那毫无疑问就是促进国内消费的非常有效手段了。只要花在国内，肉烂在锅里，怎么花都是次要的。

另外，中国自有住房的比例其实很高，别看网上没房的叫的声音大，其实没有买房的才是一小撮。
