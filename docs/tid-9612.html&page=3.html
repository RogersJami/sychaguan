继续涨知识：鹅毛弄个发动机后置，出于什么考虑呀？
       妹妹好，步战在水上是飘着走的（步战具有浮渡的能力），选择发动机前置，当步战不搭载步兵的时候有时会出现前重后轻的现象，搞不好车头还会扎到水里去。为了解决这一问题，俄毛选择了把发动机后置，由于俄毛的发动机体积小，重量轻，同时为了降低发动机的高度，又把发动机做得很扁。但是发动机后置以后，车体后面步兵上下车的门的高度就比较低，这样又带来了步兵上下车不方便的缺点。
       至于TG，解决问题的思路就很巧妙，本来车首的防浪板就很宽，再把防浪板往前多伸展一些并可以操控，使海浪打上来的时候产生一个抬头的力矩，从而增大了车体前面的浮力，解决了车头扎到水里去的问题。用防浪板提高浮力这项技术就是从二代步战开始的，这样，既保留了发动机前置步兵上下车方便的优点，也解决了发动机前置所带来的车头容易扎到水里去的缺点，让俺们为国防战线上的设计师们的创造精神喝彩。

补充狮子兄一点：
防空弹扔多少个，一起扔还是连续仍是有讲究的。我国防空导弹发射部队当年是在苏联援助下建设的（这个网上有无数资料，去小汤山航博也随便找得到），从建制到装备到操作都师法苏联。接装的导弹根据苏军条令，在某种条件下针对某种目标的单发命中概率是xx（这个显然是机密），简单讲比如面对黑猫的U2，可能单发命中概率是50%，那么从概率论的角度，三发命中概率就提升至快90%了。导弹发射不是随心所欲的，有一个“发射窗口”，时间转瞬即逝，因此到底扔多少个需要指挥员提前制定决心（当然这个需要遵守条令，比如下命令六发齐射的话……他就可以上军事法庭了。而如果下命令12发齐射呢？那显然是伏特加喝多了，一个发射单位就6发导弹），决定是放几发。而我国当年刚开始使用防空导弹的时候自然是一切听老大哥的，标准发射就是三发连续射（可不是齐射）。

为啥不齐射呢？那是因为……技术限制。那导弹需要地面引导，齐射可就是三个引导同时进行，这可是个大工程。当然了，现代的防空导弹受技术发展的恩赐，没这么多的麻烦了（当然也不是，请自行百度“火力通道”，也不是随便可以万蛋齐发的），通常情况下，大家都选择两发连续射。

冒昧一下：防空兵器现在主要是地对空导弹吗？高射炮会不会有点火力不够呀⊙﹏⊙b
自己又看了一遍回复，发现写了一段但是好像还有点木有讲清楚，妹妹莫怪

       “火力够不够”这个要看怎么说，如果说炮弹的威力，现在的战斗机，武装直升机，巡航导弹之类的目标基本上扛不住一发35或57mm炮弹的打击滴，不死也重伤。而轰炸机，运输机之类的大型灰机只要不打中（花动机神马滴要害部位）应该可以顶得住一，两发的。
       如果说命中概率，当然对中，远距离，中，高空的目标，只要木有什么电子干扰，导弹的命中概率是很高的，而这时候，如果选择高炮射击，只能说聊胜于无。但是所有的防空导弹都有一个最小射程（如500米）和最小射高（如30米）的射击范围，出现在这个范围之内的灰行目标，导弹是无能为力的，而在近距离甚至于极近距离，低空，甚至于极低空正是小口径高射炮大显身手的时机，在这个距离（如500米），高度（如30米），由于小口径高炮射速快，精度也行，是具有很高的命中率的，可以填补导弹的射击盲区。所以，在现代防空体系（地对空，舰队空）中小口径高炮是占有一席之地的。

美“再再平衡”战略关注东南亚

2014年02月27日09:14    来源：北京日报 现年59岁的拉塞尔，2009-2011年在白宫国家安全委员会主管日本、韩国和朝鲜事务，后转任总统特别助理和国家安全委员会亚洲事务高级主管，是奥巴马政府“亚太再平衡”战略的主要幕后推手之一。

按照他的说法，美国“亚太再平衡”战略有三大支柱：一是加强和深化美国与地区盟友即日本、韩国、澳大利亚、泰国、菲律宾的同盟关系；二是加强机制建设,深入参与东南亚国家联盟地区安全论坛、东亚峰会等地区机制；三是与地区新兴大国，尤其是中国，构建更好和更牢靠的关系。

对于战略多元化，拉塞尔指出，军力等“硬实力”固然是美国“亚太再平衡”战略的基石，但经济、能源、教育、价值观、民间交流、公共外交等“软实力”才会产生重大和持久的影响，因此，美国应兼顾上述领域，力争实现战略多元化。

对东南亚军援将增50%据美国《星条旗报》网站报道，美国国防部长哈格尔8月25日在吉隆坡说，。“我们的最新预算包括用于东南亚地区的外国军事资助项目以及国际军事教育和训练项目的9000万美元，这比4年前增加了50%。”
     据报道，美国国防威胁降低局将为菲律宾设计并制造海上边界安全系统。该合同是五角大楼与东南亚国家签署的首个边境安全合同。根据合同安排，美雷神公司将为菲律宾设计并建立一个国家海洋监测中心，来自不同渠道的数据将被汇集至该中心；雷神公司还将为菲政府提供自动识别系统的采办、安装以及训练支撑。
