上下左右 发表于 2014-3-5 09:02:48

5个行政垂直等级，仅靠各类规划各项政策肯定是不够的，仅靠每年象征性地搞几次各类型的挂钩推广活动也是不够的。因为每年上级政府的各条线考核依然延续论次排名，实在过不了人情关，就多搞几个并列排名。预算收入这类关键指标，依然是一项领先，综合评比靠先，所以考核的导向决定了同级之间不可能搞一体化合作，更多就是“别苗头”。这几年苏南某市由政府主导的产业转型，企业是搬走了，环境是做优了，但留下的就是大量闲置的商品房和商业。随着制造业的大量搬迁，本市原有资本外流，人口大量外流。现代服务业承接不上，外市资本引不来，人口流入减少，用工成本增加。这一进一出，加剧了现在的阵痛。
群众路线走好后，估计就要好好搭骨架搭班子，建队伍选人才了。现在的行政垂直等级体制肯定是与区域一体化相背。
