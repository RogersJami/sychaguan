纯经济金融背景的，似乎都对各种数字有天然的崇拜或者说畏惧感。

但其实，数字只是一种表象而非本相，在中国尤其如此。数字从来都只是一种工具，而不是反过来让数字去指导你的行为。即便在整天折腾数字的金融行业，数字也只是在统计学意义上作为一种工具让你有更大的概率做对，而不是告诉你到底哪边是对的哪边是错的。

一个有发展前景的国家，从来都不会是结构均衡的，必然是一部分发展过头带动着一部分没发展好的，然后三十年河东三十年河西，换一部分发展过头的再来带动另一部分没发展好的。这样一种非连续循环体系才是发展的常态。美国恰恰是这种体系的最典型例子。而且美国的领先就在于它的循环频率最高，可以主动选择循环中作为引导的那一部分，而且选择最多。

所以不要只看到中国有问题的一个角度，而忽略了整体，因为什么时候你都可以说，发展好的那部分有点过热，发展不好的那部分还差好远。在你没有太多选择主动权的时候，胡乱折腾才是最作死的节奏。
