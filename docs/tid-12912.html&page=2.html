签的协议需要到村委会盖章么？

土地流转，旧社会叫换地契。按目前农村土地承包的法律逻辑来说，是村委承包给农户。农户之间转让承包权，必须到村委一级重新取得承包权，签订新的承包合同，否则，到时必是麻烦。

因为我们的法律明文规定土地不得买卖及变相买卖，土地是不能私相授受的，农户私下的交易可以从法律上认为不合法，交易无效。

在所有权这个问题上，普通老百姓十分无力。不过，我想得很明白，如果中央真打算在这上面作文章，想反悔， ...
封建社会的周期律是大乱---王朝草创---适当均田减赋---王朝中兴---土地兼并---加速兼并---大乱，这么一个周期，土地是伴随封建王朝兴衰的一个主要载体。

历史发展到今天，现代意义上承载着历史兴衰功能的介质不仅有土地，还有资本，资本的作用已经远远超过土地。我一直强调，资本不能失去控制，政权不能控制资本的话，政党和政客迟早会成为资本的代理人。从这个意义上来理解，我赞成适当规模的国企存在。虽然国企有低效的痼疾，有更易腐败的可能，但是社会适度的低效率未必就一定是件坏事。国企是政权控制的资本，这种资本使得执政党可以避免其他社会资本的干扰。关键问题是，这样的资本必须从形式和实质上代表着民众共有的资产，而不能成为某些利益团体的私产。
