      现在全球的主要矛盾其实聚集在美帝自身内部，他有两大主要矛盾迫切需要解决：一是以美元为代表的金融霸权受到全方位的挑战，二是美国自身债务不断累积，如果不能消除债务，他的信用必将破产。
      为什么说美国的金融霸权受到全方位的挑战？首先是各种全球性的货币不断涌现，侵蚀了美元在国际贸易和流通领域的市场空间。早期的日元被打压了，欧元也屡受打压，现在轮到了人民币。但是即使备受打压，欧元和人民币在国际流通领域的空间还是在不断成长，人民币的表现尤其出色。这个游戏已经玩得象个打地鼠的游戏，美国人忙不过来了。其次是在一般贸易领域已经有很多行之有效的规避美元的应对措施，如货币互换，美国的力量无法渗透到世界贸易的每个环节。美国人现在只能死死守住石油贸易的底线，但是现在看来这个事情他也难，俄罗斯伊朗就是个例外。萨达姆被搞死了，不然还得加上伊拉克。最后是金融霸权的实施渠道，就是国际贸易清算体系，现在也开始破冰了。如果金砖银行顺利运作，那么起码在金砖银行用户国层面就多了一个规避美元清算的选择。金融霸权是美国人历经两次世界大战才从英国人手中抢来的最大战利品，现在被一点点蚕食，他岂会甘心，必然要反击、破坏，他反击和破坏的目标就是我上面说的那些环节和参与的国家。
      对于美国自身债务问题来讲，咱们很清楚，美国每年的贸易逆差那么大，政府还要背负巨大赤字养那么庞大一支军队，那么背后支撑这一切的资源实质上来自于全球民众的辛勤劳动。按照等价交换的原则，美国人输出的那些高科技产品和各种服务根本值不了他得到的那些物资和服务的价值。况且真正的高科技他也不输出，高科技产品的定价还存在一个高估的问题。所以这里面的道理就很明白了，美国就是个黑社会，他一边用金融霸权的途径“合法”地剥削全球，定期消除自身的债务（免除对世界各国的支付义务），一边用庞大的美军为这种霸权背书，就是这么个框架。
      现在的状况是大家合力在破美国的金融霸权，而美军是不是强大到可以对抗这种破坏呢？可能大家都在掂量，包括奥黑，包括习大大，当然也包括普大帝。于我们而言，未雨绸缪立足于战争准备才是万全之策。
