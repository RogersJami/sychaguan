小学三年级题目试分解

    前两天见到小学三年级侄子的家长q群，发了一个这样的题目，觉得好玩：
选择题：
    一条毛毛虫，从幼虫长到成虫，每天长大一倍，7天长到20厘米，那么，长到5厘米要（  ）天。
    A，6；B，5；C，4；D，3

第一步：筛选有用信息。（我们的数学提供的每个信息都是有用的，不同于物理）
              一条毛毛虫（无用），从幼虫长到成虫（无用），每天长大一倍（变化），7天长到20厘米（状态）。
              那么，长到5厘米要（  ）天（状态）。
            
第二步：挑选信息。（变化>状态，为何变化大于状态，因为数学的加减乘除本身就意味着一种状态的变化）
              每天长大一倍，这就是个变化信息。

第三步：分析挑选出的变化信息。
              每天长大一倍这个换算成数学语言就是X2每天，在这里我们要引导学生把文字语音换算成相应的数学语言。

第四步：分析数学语言对应的知识点。
              因为我对教材不熟，我想乘法在教材边上对应的是后面的除法，或者是一位数乘法等，多数连续相乘等知识点。
             恰当的分析出所对应的知识要点，我们就能很快的使用相应的数学语音。这步骤是在做熟题目的情况下用来举一
             反三的，在开始做的时候，可以引导，不必强求。

第五步：分析下一个信息，状态信息。如果说分析变化信息是让我们懂得题目所依存的知识点，那么说状态信息就是
              告诉我们题目所提的问题。我们多做题的目的，就是总结这些状态种类，最后把习题写薄，减轻学习压力。
              
              在这题目中两种状态，一是“天”，二是“厘米”。在这里我们就发现对于数学来说，单位的重要性，能很快的
              帮我们把信息分类。
              在以后我们联系中，有一个必要的步骤就是统一单位，即解题的一开始，我们就要把单位直接统一，
              这是解题模块化的要求，当然，现在还没涉及到，我就不多说。

第六步：在分析状态信息时，我们首先分析问句。我们发现 问句是“长到5厘米要（  ）天”，我们要求解（答案）
              的是“天”这个单位，那在下一步分析中，我们首先不看它，因为从问题中我们看不到变化。

              那么这时剩下的信息就是“厘米”这个单位（这步涉及的是信息的筛选）。这里，我们要注意到一个顺序问题，
              一般我们都应该从已知的信息向未知的信息出发。即在这题的语境里，就是从“20厘米”到“5厘米”的变化，
              这是个在数学体系结构中已经能确定的变化。

第七步：比较变化信息与状态信息的变化，看是否相互映衬。如果是，则直接使用此数学语音，在这里就是“X2每天”。
             如果否，就要使得这两种变化想一致，那么必定是变化信息的相应改变。而追查这种改变就是对知识点的追溯。

             在这题里，我们分析到从“20厘米”到“5厘米”这是个缩小的变化，而在三年级孩子涉及的运算会变小的情况，
             用数学语音来描述就是“÷和-” 。这两个数学语音与变化信息 （第三步）所提供的X2有关的就是 “÷”，
             因为除法是乘法的逆运算。在这里面里变化信息就要变成“÷2每天”。
             这步的变化非常重要，数学题目的千变万化很多就是在这一步里面打转。

第八步：变化信息套入状态信息。
              20÷2=10,10÷2=5，即从20厘米经过2次变化才达到5厘米，而每次的变化对应的是1天（每天，这就是
              我在数学语音带入每天这个单位的原因，这个是可以通过多做习题来习惯的）

第九步：得出结果。
              既然用了2天达到这个变化，那么最后的结果就应该是7-2=5（天）

第十步：问题扩展。
              如何通过多种题目变化，比如从16厘米变到1厘米的过程中，解释20÷5=4（倍），4÷2=2（天）这样的方法。

后记：因为本人不从事教育工作，我只是想把数学的结构讲解出来，对小孩是否有用还是个未知之数。如果这个东西，大家觉得没用，就不用理会这些，一笑而过，因为我写得自己都觉得话唠；如果大家觉得有用，欢迎讨论，并点击“支持”和给予“评分”，特别是“威望”这一选项，俺很想要它，当然【反对】大家请看清些，俺是万万不敢要的。

（1）根据已有的信息，这个数只能是从 10 到 31 之间，排除掉相同数字之后只有
13 x 13 = 169
14 x 14 = 196
16 x 16 = 256
17 x 17 = 289
18 x 18 = 324
19 x 19 = 361
23 x 23 = 529
24 x 24 = 576
           25 x 25 = 625
27 x 27 = 729
28 x 28 = 784
          29 x 29 = 841
          31 x 31 = 961
（2）由于甲首先不知道，因此再排除
      25 x 25 = 625 (否则如果甲拿的是数字6就可以立即推断出唯一的 25 x 25 了）
      29 x 29 = 841 (否则如果甲拿的是数字8就可以立即推断出唯一的 29 x 29 了）

1、ABCD×9=DCBA，ABCD=？

三年级加法乘法都学过了吧。这题的关键是对0、1、2、3、4、5、6、7、8、9这些数的尾数规律要清楚——也就是九九乘法口诀表要熟练，其次考查的就是对进位制要熟练。
解题思路：
1、一个4位数乘以9，还是一个4位数。所以A一定是1，大于1的话，乘9必定要成为5位数。
2、由1可以知道，D一定是9。
3、由1、2可以推算B一定是0。因为ABCD×9=DCBA=9CB1，如果B大于等于2，那么2*9会产生向上位进1，而上位由A*9=1*9已经确定是9，如果下位再进1，那么就要变成5位数了。所以B一定要少于2；又A已经是1了，所以B是0；
4、那么现在就变成了10C9*9=9C01了，9*9=81,要向上进8，那么现在就是C*9+8的尾数必须是0，什么数乘9加8尾数是0？只有2+8尾数才能是0，只有8*9=72的尾数是2.所以C是8.
5、那么现在ABCD×9=DCBA已经变成1089*9=9801了，验证一下是否正确即可。

这是给大人看，让小孩慢慢建立数学结构的。

你的思路就错了。应该从小培养各种解决方法，以你所谓的结构只能小孩思维钝化。结构法什么的，等他上了大学数学专业时，或者建模之类，再去考虑。

本题第一感觉就是顺做有点没思路，那么就该逆做看看，逆做还不行，再换思路，比如对比、容斥等等。能解题不算能，要培养孩子能有多种解题思路，然后比较确定最简便最优化的解题思路。练习多了，孩子在碰到题目时，会自然而然的针对某问题直觉的第一思维找到最优解题思路。
