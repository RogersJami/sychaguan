西藏是地理因素，是在是太高了，所以平地上的民族很难待下去，解放后确实靠那一代人的理想主义就像新疆一样也去了很多干部要不然汉族人口会更少，高原不是靠身体或是光精神就能解决的，看到有文章写有些移民过去的汉族生的小孩受同样的教育做数学题也比不过内地的学生。不过也因为西藏环境太恶劣了所以这么古老的藏族人口还不是很多。
其实换一个想法，不用静态的想着移民而就利用西藏特殊的环境开放旅游，当动态的全国各地，全世界的游客达到一定的量就可以当作一种常量使得藏族不会变更封闭而变得更加激进。同于游客必须要和当地发生各种关系，这样这个常量就不用很大就能达到同样去移多的多的人的效果。
再一个可以鼓励一部分藏族融入内地，其实就人的身体来说好的地方待时间长了，也就回不去了。
十世班禅本来被各种运动不能待在西藏然后就在汉地待了几十年没事，然后80年代政策一松，结果一回去就挂了。
现在这个达赖是历任里面活的最长的，而且还很长一点，目前还在活，这算他因祸得福，还能到处跑。说不定是老和尚要是真回到青藏高原了一下子就转世了。
