作为从06、07年开始一直关注房市和股市的工科男讲两句。

房地产调控不是第一次，按照以往经验，由于我党所有调控措施都是focus on杀灭流动性的，因此调控开始后三个月到半年，会把成交量打下去，这期间报价未必会降，但议价空间扩大。然后成交量数据的明显下降引起买方和卖方心理预期降温，这时候报价开始松动。再后面由于销售不畅的传递效应，产业链各环节都开始萧条，媒体上负面报道增加，例如，某家装卖场门可罗雀了，某开发商偿债出现困难了，等等，房价下降的预期形成，报价出现明显下行。再后面地方政府受不了了，开始松绑、刺激...

这一次市场反应比以前似有所加速，北京的二手房日成交量节后迅速腰斩。这可能跟调控政策出台的时点有关，事实上成交量9月中旬之后已经开始回落了。还有一个观点不知有没有道理，移动互联网这一波以后，社交媒体的普及缩短了社会人群的总反射链，房市股市似乎都有这个迹象，这个有点扯远了。

婚房是刚需，其实我是同意年轻人的第一套住房不要太多从投资角度去看待的，因为它属于生活必须基础设施，将来卖掉套现的可能性也不大，顶多是卖小换大。不过机会成本也是能省则省啊，以上分析，希望有用，hand。

就我自己来说，就在这个最高点，利用自己的小房子涨了90%，而目标区位的大房子只涨了50%的有利条件，以小换大了。
小的位置很好（靠近通州新城核心区）就是离上班地方太远，大的在回龙观。
负债变化是房贷比以前翻番，但不用租房了。
支出情况变化不大，原来的房贷+房租 = 目前房贷 ，需要自己填补的就是原来小房子的房租收入没有了，一年多花个两三万吧。
改善需求方面，面积增大了50几平，变三居了，房龄也尚好，几年后再出售也支持高年限贷款。
积蓄方面，当然也基本上都空了，为了少交税费，多给了些首付，剩下的也就够一年房贷的，幸好没有另外和亲戚借钱。
后续大额支出考虑使用信用杠杆，这样虽然有点成本，但总比房子上多交的税费划算。
很多人都跟我说目前机会不好，以后通州还要大涨，但是我从我的支出分析，负债分析，改善需求等方面得出的结论都是现在换最划算，等到热点地区横盘，非热点追涨后恐怕没机会换到大房子啦。

我的想法是不管以后如何，换完后压力没增大多少，居住改善了，资产是否增值的意义不是特别大，毕竟不会老是卖了买买了卖的。
