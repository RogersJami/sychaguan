从装备体系看，中国海军继承了前苏联海军航空兵的对海反舰导弹饱和突击的战术思想。

1962年古巴导弹危机以后，面对美国海军的制海优势、完善的海军打击体系，尤其是强大的航母战斗群，苏联海军感到毫无还击之力。为了对抗美军，苏联海军建立了一整套庞大的饱和攻击体系。1985年底，苏联海军太平洋舰队派出以明斯克号和新罗西斯克号航母为核心的特混编队，穿过日本海，从冲绳岛南部转向东驶入太平洋深处。在此后8天里，新罗西斯克号模拟美国航空母舰，出动多架雅克-38型垂直起降舰载机对苏联发动从海至陆的突击。太平洋舰队则派出一架图-95侦察机，引导20架图-22M逆火超音速轰炸机对新罗西斯克号编队发动模拟反舰导弹饱和攻击。演习结果表明逆火轰炸机使用超音速反舰导弹能够有效打击美国的航母战斗群。由于演习空域位于日本以东，因而对美国和日本造成了极大的震动。

苏联解体以后，这一切都是黄花菜了。目前俄罗斯空军和海军航空兵对海突击力量，仍以上世纪80年代的逆火轰炸机体系为主，尽管还能发射威力巨大的Kh-22N(即AS-4/ AS-6)厨房/王鱼反舰导弹。但整套武器系统早已过时，AS-6导弹使用液体火箭发动机，速度2.5马赫，射程超过400公里，全重达5吨，安装1吨高爆战斗部，采用惯性中制导加主动雷达末制导，常用发射高度为一万米。这导弹是现代舰空导弹的优良靶标，突防能力并不比中国产YJ-83亚音速掠海反舰导弹优异多少。以其60年代水平的电子技术看，估计可能都不需要硬杀伤拦截，用距离欺骗干扰晃一下，这导弹就立马晕头转向，直接落海了。

美国国防部《中国军力报告》指出，鹰击-12是目前世界上最危险的反舰导弹，威力甚至超过了东风-21D反舰弹道导弹。该导弹以每小时4000公里的超高速和超低空掠海飞行能力，可以严重压缩敌方舰队的反应时间。


2016年7月8日，中国海军在南海海域展开大规模军事演习，展示摧毁严密防护下航空母舰编队的能力。首次出现海军轰炸机发射鹰击12反舰导弹画面。鹰击-12导弹采用固冲合一发动机，发射后立刻点燃火箭助推器并以高速飞离，正是演习中发射导弹画面的特点。


海军航空兵的轰6G轰炸机还装备另一种远程反舰导弹——鹰击100反舰导弹，这是长剑20远程巡航导弹的反舰版本。最大有效射程大约为800公里，尽管是亚音速导弹，但可通过复杂航路规划和掠海飞行，提高突防能力。鹰击100的射程超过美军F-18战斗机的紧急截击半径，使用轰6G+鹰击12+鹰击100，可对美军航母战斗群实施双保险的饱和攻击。


此次海军航空兵进入日本海的第二种战机是运8警戒机，日本称之为运8哨戒机。该机是中国海军航空兵对海饱和攻击体系的重要组成部分。一直以来，中国海军就在寻求一种类似美国P3C的优秀海上巡逻侦察机。上世纪七十年代研制水轰五水上飞机时，就为其配备了侧视相控阵雷达，用于海上搜索侦察。到了80年代，中国又从加拿大引进了APS-504V3搜索雷达，安装在运8飞机上，增加国产全向自卫警戒系统，照相系统（高中空光学相机，低空相机，红外相机），红外搜潜仪 ，声纳系统，声纳浮标投放装置，于1986年研制出运8x海上巡逻机。


时间进入90年代，中国海军仍未能拥有一种令人满意的海上预警侦察机，急需能够对大面积海域进行搜索和目标指示的飞机，并且能够协调战区内反舰平台的作战行动。这时候中国对外透露了想要采购新型预警雷达机的愿念，立即引起了西方国家的高度重视，一时间各国纷纷携带自己的机型上门推销，此时中国还处于因为某不可描述事件被西方国家制裁的阶段。但这难不住洋人，甚至美国也拿出E2A改进型，没关系啦，这些都是民用的，以海洋科学考察和探索的名义研发的。

英国人最起劲，他们的最终目的是为了推销迷猎预警机，提出可以提供另外一种先进雷达，安装在运8飞机上，成为高性能雷达警戒机。1996年，中国从英国雷卡尔电子公司购买了6套“空中霸王”机载雷达系统，这雷达又译为搜水2000。于是中国以民用名义进口，理论上用于海洋资源调查和冬季渤海湾的冰情调查。至于当初口头答应的继续引进迷猎预警机的事，没了下文，这令英国人很伤心。我们说过什么吗，您一定听错了@#@%￥


日本航空自卫队拍到的运八海上警戒机

运8警戒机的正式型号为运8J，典型特征是机头有一个异乎寻常的“大鼻子”雷达罩，机身内部也进行了重新布置，能够容纳控制设备和人员的休息设施。中国海军也在该机上装备自行研制的数据链，使之能够畅通地和其他兵种进行信息互享。该机能够在舰载雷达作用距离之外探测到水面目标，可以通过数据链将信息传送到其他空中、水面及水下作战平台。


运8警戒机内部

运-8警戒机续航时间为11小时，可使用空中霸王雷达覆盖350公里远，640公里宽的探测范围。8架运-8警戒机就可以在2个东海区域内保持24小时不间断空中侦察。

在“和平使命”2005中俄联合军演中，运8警戒机首次公开亮相，成功引导我方战机、舰船和潜艇对海上目标实施了精确打击。此外该机也可为水面舰艇提供远程目标指示和战场态势信息，军事新闻多次报道022级导弹艇在运8警戒机的数据链引导下，装定目标数据发射导弹命中目标。


运-8警戒机

配合运8警戒机行动的是运9电子侦察机。该机的具体技术性能从未公布过。只是在军报发布的官方新闻里有过概述：该电子侦察机主要执行战略电子情报侦察，具有一定的雷达、通信侦察能力，同时具有一定的跟踪干扰能力，可担负对空对海电子情报保障，日常战备巡逻任务，以及其他勤务保障任务。据说此次插入日本海，运9电子侦察机模拟了一个轰6大机群的信号，导致日本和美国的误判，以为中国派出了几十架轰炸机。


对外界来说，运9电子侦察机又称高新八号，非常神秘，被戏称为“中国海空军御用摄影师”的日本航空自卫队，在2014年10月5日才首次拍摄到中国运9电子侦察机的照片，这是这种新型电子侦察机首次被拍摄到的高清图片。

运9电子侦察机装有电子情报搜集设备，可通过敌方海上编队的电子通讯和雷达信号搜索敌舰的位置。也装有空中指挥设备，可作为编队的指挥机，分析和处理其他飞机或侦察平台发来的搜索情报和目标指示信息，然后把目标信息分发给舰队或轰炸机，统一指挥调度导弹饱和攻击。在冷战期间，苏联的图-142巡逻机也是用类似的方法在海上展开搜索网，精确定位美国航空母舰编队的。

电子侦察+雷达警戒机+远程轰炸机+超音速反舰导弹，从搜索海上目标开始，到作战决心下达，到导弹攻击航路的规划，再到发射平台的任务分发，直至发起导弹饱和攻击。然后敌方舰队上空将飞过来的是一群鹰击100和鹰击-12重型反舰导弹，以及从大气层外呼啸而入的东风特快。

中国海军航空兵近两年组织的远海演练，说明中国海军的眼光早已超越台湾海峡和第一岛链，要到深海大洋上好好的练一练。美国和日本完全没必要在1月9日表现的那么紧张。相对于苏联当年恐怖的反舰导弹饱和突击，中国还只是刚刚开始呢。

中新网1月18日电 据外媒报道，美国五角大楼的一份报告称，F-35隐形战机至少存在276处缺陷，在2020年之前不具备完全作战能力。战机的新问题层出不穷，修复这些问题可能将花费超过10亿美元。
　　五角大楼作战与评估办公室主任迈克尔·吉尔摩的这份报告，是国防部2016财年大规模年度报告的一部分。他在这份62页的报告中，给F-35的未来描绘了一幅严峻的图景。
　　作为奥巴马政府的官员，吉尔摩在政府换届之时即将离任。尽管负责F-35的官员坚持该战机状况良好，并且在运行测试中发现的问题正在迅速得到修复，但吉尔摩的最终报告明确显示，他不相信这些保证。
　　吉尔摩称，在测试中，F-35战机的Block 3F软件已经出现了276处被认为是“亟须纠正”的缺陷，但正在着手解决的缺陷却只有不到一半。此外，每个月几乎都能发现约20个新的缺陷。

  
　　不仅如此，F-35战机项目还试图跳过诸多早期测试，提前宣布研发阶段结束，并转向定于2017年8月开始的“初步作战测试和评估”。该报告指出，这将会把风险转移到作战测试、后续改进和计划驾驶战机的飞行员身上。在其目前的计划和资源范围内，项目办公室无意用飞行测试来充分修复并验证目前存在的数百个问题。
　　吉尔摩称，多个问题和延迟清楚地表明，即使以最快速度推进，在2018年底或者2019年初之前，该计划也无法展开初步作战测试和评估。吉尔摩还表示，完成系统研发阶段将花费约5亿美元，而成本评估和项目评测办公室估计，花费甚至可能高达11.25亿美元。
　　除了软件问题，飞机还面临结构问题、维护问题等等困扰。
　　这一报告的结论是，美国国防部应该重新考虑是否遵守“大宗购买”(block buy)该战机，因为其当前的状态不符合法律要求和五角大楼“买之前先试飞”的原则。
