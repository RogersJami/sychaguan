雄安新区对北京意味着什么，这里有一个好的比喻（转帖自水木社区）

你，聪明过人，被财上海和任大炮忽悠得认为，京沪永远涨。看到北京房价一涨再涨，觉得新文化街的新华社小区就是硬通货，15万不在乎，瞬间在2月的春天里上了车。

没想到，北京3月的天依旧寒冷。天边雷声袭来，你有点慌张，不过不要紧，你并不在乎，因为你认为雷声过去，夏天就会来的。每次路过北京站和北京西站，看到乌央乌央的年轻面孔，心里盘算着，这座城市，是这个帝国无可取代的明珠，这些面孔会为自己的资产买单的，他们需要你手里的资源。

结果，愚人节的下午，IPHONE上推送了一条新闻。大大犹如一个扳道工，把这些年轻人的火车引到了天安门南150公里（这次不再是天安门南50公里）。

这下，这些年轻人被拯救了，那些北京租牌的人被拯救了，挤在五环外的部委小科员们被拯救了，被房贷折磨得死去活来的码农们被拯救了。

而你的脸却突然肿了，《新闻联播》康主播一句亢奋的“千年大计”告诉你，谁才是庄家，谁才是主人。

今天，春天，北京的夕阳格外刺眼。

明天开始，北京不再是那个北京了。

日子还得过，贷款还得还。

人民共和国会记住你的。

===========================================

个人感想：房价的过度上涨是因为人口的过度集中，而人口的过度集中是因为城市功能和资源的过度集中，认为房价会永远上涨的人其实是不相信中国政府有能力将城市功能和资源主动分散配置——因为发展中国家是没有这个能力和意愿的，大多数发展中国家的政府都是腐败低效短视愚蠢的，并且被利益集团所操控，只有传说中的发达国家才有这个能耐。

但如果中国政府能够主动的有计划地分散配置城市功能和资源，上面所说的前提就不存在了。房价的无限上涨也就没有真正的支撑了。

过去几千年中，中国一直是农耕社会，生产和生活的重点在农村，但随着中国的工业化，生产和生活的重点转化为城市，这意味着必须对城市进行精细安排和设计，不能像农耕时代一样让城市自然生长，而且这也是一个国家的责任感和能动性的体现，更是施政能力的体现。我觉得中国如果能够按照这个思路走下去，把城市的功能和资源科学分散和配置，将繁荣和进步扩散到大城市周围广大的腹地，那么中国的现代化就真的要实现了。那时候即使有人要喊没有地产党就没有新中国，咱们也认了。
或许这才是葡萄所说的解决房地产困局的真正方法。扬汤止沸何如釜底抽薪。
