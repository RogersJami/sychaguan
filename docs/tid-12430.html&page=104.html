股市近期不看好，一大关键因素是制造业整体缺血，临近年关需要结清欠款、发工资奖金过年，所以股市套现动力很强，现在要高利贷都不好借，民间金融出事太多，国家银行基本无指望了，银行都在努力收回社会风险性贷款，只有政策性国家贷款好办。
国家虽然发了N多钞票，实际到不了中小民营企业手上，外部资金解决问题实际还是各式高利贷变相集资的玩法，大量放水一是到金融市场投机（实际深圳沿海相当多的中小老板关了厂子，想在金融投机上搞搞，年后关厂的数量会大增），二是沉淀在地产市场上（如果政府不放宽政策，未来2年是各路房地产公司因为融资成本过大而风险暴露的时间，从12年房子不好卖开始挺，很多就是银行的一个数字）。
所以，外资说不看好，有一定的现实基础。我们相信TG能够有办法，但是目前看来还没有出现好办法。
新中国建立以来的第一次半市场环境下的经济大萧条，可能正在寻找底部之中……对比历史，我们可能还在相当于米国的1930年
