{:soso_e153:} {:soso_e132:}

　　8月3日1时50分，林芝地区行署应急办接波密县报告，川藏公路318国道通麦大桥铆索脱落致部分桥面垮塌，桥上一辆货车坠入河中，车上乘坐两人，均已失踪，交通中断。
　　国际在线消息：据新华网西藏林芝８月３日电，川藏公路３１８国道的咽喉工程－－通麦大桥２日深夜发生垮塌。记者在大桥现场了解到，在确定有两名波密县当地村民失踪后，另有２名游客也在大桥垮塌后失踪，因大桥垮塌失踪人数增至４人。
　　记者在通麦大桥现场经与波密县委、县政府核实了解到，２日２３时２７分，通麦大桥铆索脱落致使桥面垮塌。桥上一辆货车坠入河中，车上乘坐两人，均已失踪，交通中断。经调查，货车上两名司乘人员系波密县易贡乡贡仲村村民扎西群培和其美多吉。
　　另外，大桥垮塌后还有两名游客失踪。两名游客均为徒步进藏，为一男一女：周乐，女，２４岁，江苏省宜兴市高塍镇宋渎村人；苏磊，２５岁，安徽省长丰县三十头乡横店村人。
　　接到大桥垮塌报告后，林芝地区公安、消防、交通等部门迅速启动应急响应机制，消防、交通部门迅速派工作组前往事故现场。林芝地区行署专员旺堆要求公安、武警及消防等力量全力搜救失踪人员。
　　目前，已有当地交通部门在通麦大桥两侧设卡，实行交通管制，劝返、疏导人员和车辆，并做好失踪司乘人员家属的安抚工作。
　　通麦大桥是川藏公路３１８国道的咽喉工程，大桥设计为双塔双跨悬索桥，全长２５８米，主跨为２１０米，高３１．５米。以险著称的川藏公路１４公里便道就位于通麦大桥附近。
　　最新消息：川藏公路通麦大桥垮塌事故失踪人数增至4人
