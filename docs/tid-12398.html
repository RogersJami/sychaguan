聊聊CPA考试


2012年由于工作需要（原单位流程改造和引进管理信息系统）而接触并计划考CPA，两年多来有一些收获，也有不少遗憾。由于中国竞争激烈，人力资本一直保持着上升态势（各种考证热等），今年又取消了一批财税经济类证书考试，可以料想未来中国CPA考试的人群应该更为拥挤。

但应不应该考，为什么考？如何考，考取了又能如何？等等问题是需要在考前弄清楚的。当前社会上存在两种极端思想，一种是CPA无用；另一种则认为考取了就如何如何等等。这其实都不正确。


CPA无用，一是源于会计师门槛本来很低，初级中级会计从业资格即可，这是各个省主办的考试。二来是因为一般的企业用不到CPA的相关知识体系，财务会计最多负责记账、总帐、报税、计算人工成本发工资，如果是工业企业则要涉及到成本会计等等，这都是实用性技能，用不到CPA的财务计划管理、审计和控制以及金融资产报表合并等较复杂的会计处理。第三，如果涉及到转换行业工作，CPA证书的有效性明显不如国外的几门证书，如CFA、AICPA等等（我觉得和CPA协会的学科建设有很大关系）。


至于说CPA考取了就如何如何，那是自欺欺人了。反正我接触的会计师事务所的人没这么认为的。相反很多人认为会计师这碗饭不好吃，吃苦受累不讨喜，压力大薪水低，人员流动率极高，一般3年就跳槽一大批。


总之，CPA考试要和自身的职业规划结合在一起。如果是财会行业中人，本身在做财会工作，有更进一步的想法，不妨考而试之。如果是想跨行业进入会计师行列，则先要踏好门路比较稳妥。因为中国人力资源竞争非常激烈，每年有十多万财会专业大学毕业生，本身是供过于求（这与全世界其他国家都不一样）。
我是第三种情况，因为实际工作，对《经济法》、《公司战略和风险管理》和《财务成本管理》的需要比较迫切，也参考了一些《审计》知识，对《会计》和《税法》仅做大略了解。原准备考完，现在看来没有必要了。


CPA考试说难也难，说易也易。关键是找准适合自己的学习和应考方法。我比较适合看书做笔记然后做习题这个方法，有人适合背书，有人适合背考题等等。由于考试都是在工作之后，对自觉性要求较高。我的经验，每天硬性划拨4个小时学习时间应该就够了，每年报考两科为宜。

先把教材读薄，扎扎实实看两遍并做两遍笔记，应该能达到一周复习完一科，掌握所有考点、脉络、体系，需要专门记诵的做笔记或者卡片。

再把书读厚，做习题，了解出题思路和难点，与之前的笔记互相验证，弄清楚考试的重难点（一般占总分的80%以上），并适应考试。

第三，考前做三五套模拟试题。由于现在是机考，可以使用模拟软件。

总得来说，备考要精细稳妥，切忌侥幸心理。在参考书上，我觉得“轻松过关系列”比中华会计网校的好一些。

我觉得中国CPA考试存在的缺点：
1、教材不权威，不能真正能够代表著名学者和从业人员对行业核心知识的一致意见。横向对比CFA、AICPA教材就能发现：中国CPA教材每年的更新调整并不注重系统性整体性，为改而改的痕迹较重，对合理编排突出重点指导学习考试实践的意义有限。
估算一下，一方面现有教材篇幅可以压缩到300页之内，另一方面需要增添参考资料和试题题库乃至开放式课程等，而不只是一本书。

每年的教材，我都发现不少编排错误或是错别字。

2、考试制度问题，一方面是人力资本加速上升趋势，一方面是影响力还不如会计从业资格考试，另一方面是固步自封。其实每年两次考试是完全可以做到的。

3、就业渠道和信息交流渠道狭窄，这一方面是因为社会现实；另一方面与行业信息垄断不无关系。据说CPA协会20年前就和不少大学进行了合作共建，奈何时至今日其影响力仍然微弱。


最后,祝大家2015年健康快乐!

CPA杂谈
一、我为什么要去考CPA。
我去考CPA的直接原因是：我大学毕业后的第一份工作去了科龙电器。当时进科龙面试我的是原科龙电器CFO姜源（姜宝军），当时他对我说04年科龙白电销售接近100亿，审计师是四大之一的德勤。于是我就是在他的召唤下去了科龙。之后科龙电器的故事大家可以自行BAIDU。我事后反思自己当时第一就不应该轻信相应财务数据，即是审计师是四大。（以后工作中接触了一下四大，个人感觉四大没有那么神奇）至少我应该去看一下科龙公布的财务报表再以自己的所学分析一下。至少能分析出的东西来，如果我没看出点东西，那是我技不如人。基于这种认识，从科龙出来后我开始了CPA考试之路。

CPA杂谈
二、CPA考试的个人感受
我大概花了三年左右的时间考完全部5门科目。除了税法科目一次性通过外，其他科目都是重复考试才通过的。
CPA科目考试我最大的感受是：考试是人为控制通过率而设置的，并没有从实际业务需要出发。
最简单的例子，凡是考过CPA都知道考试时间是基本不够用的。原因很简单，通过率在那里摆着，而控制通过率最好的方法就是加大题量，让你会做但没时间做，考试当然通不过。这一条自然引申出考试的另一个特点：当年考试书上有变化是当年考试的重点。目的就在于使那些往年考试没通过的考生没有科目内容熟悉上的优势。还是乖乖做题吧。
上述考试特点，当然会使只会考试而没有多少专业能力的人通过考试，这也是社会对CPA有所轻视的原因之一。考出来的CPA给人只会考试不会处理实际中复杂经济业务的印象。这种人多了，CPA的名声也就不好听了。

企业需要的是能够解决实际问题的人，而不是一堆证书。证书只是敲门砖，之后在公司的发展和证书就没关系了。我个人的升职就与CPA没多少关系，还是在看你能否处理实际业务。

CPA杂谈
三、CPA审计业务不好做
楼主在主贴中已经说了：会计师事务所这碗饭不好吃。
这也是我没有选择在事务所工作的原因，还是在企业比较好。反正我的理想是从事税务筹划工作，我考CPA完全不是为了去会计师事务所，也没有听信那些考出CPA有多牛的广告宣传。
会计师事务所的审计在一般人看来是很搞笑的一件事情，收了审计客户的审计费，还要对审计客户的财务报表发表审计意见。常言道：吃人嘴软拿人手短。换了你，收了钱会发表什么审计意见！当然是标准无保留意见啦！什么，想发非标准意见，你审计费想不想要了！这是最最一般的场景对话了。
有人说CPA是经济警察，可关键是人家人民警察是政府机关是吃财政饭的，说CPA是经济警察那为什么CPA要低三下四的去求人拉业务？

我所在公司今年审计业务就出现了上述场景：之前与公司合作的会计师事务所，去年在对公司关联公司的鉴证业务中说了点非标的话，结果很简单，今年的财务报表审计换事务所。按财务总监的话：我出钱不是使自己来受气的。
大家想想，这样的会计师事务所有何用处，不就是出个审计报告吗，除了这个还会做什么？这种CPA当然让人看不起了。

CPA杂谈
四、CPA与税
楼上有人说，税务出问题CPA会有麻烦。
其实公司税务有问题，一般是对财务人员有麻烦。一般不会找会计师事务所的会计师的，会计师对审计报告负责。
按一般人的理解，财务有相当一部分是在处理税务的，不就加加减减吗，我也会，我不会的其实就是税务上那些规定，还有增值税专用发票上的那些密码。

CPA不一定知道具体税务规定，而且可能连最基础的税务法律都不清楚。
大家有空可以看看科龙电器2001-2004年的财务报告，请注意报告所列的长期待摊费用科目中的模具项目，按当时的增值税法规，这部分模具的增值税进项税额是不可以抵扣的（实际上科龙是抵扣了这部分不应该抵扣的增值税进项税额），而报告上丝毫没提及这一点。这部分的金额应该足以达到审计重要性标准了。
最搞的是，当地国税局还在税务宣传资料上注明模具不可以抵扣，可科龙这个当地纳税大户就不按这条规定来。会计师也当做没看见。

CPA杂谈
五、CPA应该做什么
CPA考试那么难，CPA那么牛。那CPA应该做什么才能赢得一般人的尊敬。
在我看来，财务工作最基础的核算，也就是算账。没有基础数据支持，之后的进一步工作都是空。
但是核算只是最基础的，这个账房先生是无法赢得人们的尊敬的，因为一般的会计如果有心的话都能做好。
CPA最一般也是考出了两法加财务管理、成本核算以及会计审计。有了这些，应付一般企业的日常财务业务应该足够了。
那CPA工作重点就应该在事前规划，事中监控，事后分析总结上。

以我个人的例子，我以CPA的资格进公司，一开始并没有人拿我的CPA当回事。直到我对年终奖发放事项提出了专业性意见，并针对年终奖个税法规提出有针对性规避方案，领导们才注意到我。

CPA杂谈
二、CPA考试的个人感受
我大概花了三年左右的时间考完全部5门科目。除了税法科目一次性通过外，其 ...
