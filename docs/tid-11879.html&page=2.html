什么叫刚需？是不满足就上瓦岗山的需求吗？

别把老百姓正常的追求幸福美好生活，不断努力改善自己居住条件忽视掉了。

现在社会说的刚需，不是说吃不下饭才叫刚需，而是，不仅要吃得下饭，解决温饱，还得吃上一口好饭，一口有尊严的饭，一口舒心的饭。

什么叫刚需？
那些住筒子楼的人，一生努力奋斗，想买个风景如画的小区2居室，不用三代挤一间小黑屋，这就是刚需；
那些买了个单身小公寓的小年青，努力工作，积攒了一点积蓄，又生了小孩，想买个2居甚至3居的房子，条件好一点，让自己的父母也得住上大城市里的高楼大厦，这就是刚需；
那些住腻了市区的复式楼，大平层豪宅，想买个靠山靠水，有大大的花园草地的庄园，让自己生活的更有品质，这也是刚需。

现代社会，物质生活还算丰富，已经不是过去那种吃饱饭就ok一切的年代，如果现在的生活是美好的，我们还想追求更美好的生活，这就天赋权力。现在，我们就处在这样一种大潮之中。
