比如你可以从文宣中看出，科斯本人不主张市场经济就是私有化。甚至，私有化企业本身也不是西方经济学者的主流。比如，在我们经济学者言必称国际的美国，美国经济主力大多是公众企业。尽管有少数家族利用公共股份的分散实际控制着部分上市企业。但是，这些被某些家族企业控制的上市企业因为本身的公众属性而可以被联邦政府与监管部门以各种法律约束其行为。简单点说，中国要走的市场化，更类似欧美在在罗斯福新政年代之后做的社会企业财团法人的公众化。能不能做到，看下去就是了。现在人事制度改革与财政分配制度改革在三中就要宣布路线如何安排。还有，这样的政治制度改革先行交换的是年初经济改革激进被按下的安排。这并不意味着经济体制改革的激进方案本身会被真正搁置。在习总与李总达成共识的经济增长保7的同时有一个伏笔，外部经济环境不出现剧变的为前提。而美国退出qe，则是可能引起东南亚97问题的一个节点。或者说到三中前谨慎观察。

开车去中亚 发表于 2013-9-11 12:31 static/image/common/back.gif
所谓继续改革引起潜在的阶层固化的担忧，是不是可以换一个角度理解，实质上是进一步工业化和现代化的过程 ...

这里我需要厘清一个中国社会分析的现状。大体上，还是主席在十大关系中阐述的问题与矛盾。这里我不是说，主席英明之类的话题为切入点，而是在说，从主席几十年前的文章说今天的社会现状依旧有着鲜明的社会意义本身是在说中国社会问题的复杂性。而这种复杂性实际就是我这里说的社会固化倾向问题。这种固化，实际因中国社会在工业化进程中还处于一个初步的阶段而变得的更为复杂。我自己在小圈子里讨论，实际表达是，中国的固化问题实际出现封建社会士绅社会中基于血系血缘传承而来的社会固化，和早在马克思韦伯时代就提过的工业社会大分工后，出现在官僚体制内部的僵化与人的社会属性逐步固化的问题，而这种固化到1960年代美国启动福利社会全面改革之后更表现为米尔斯论述的内卷化。而我们之余前者的封建固化，表现是体制内对于子女的各种安排---比如各种年轻的官二代的升官路线图。另一种，就是现在改革针对的体制内部因为既得利益分配，形成自改开以来政治体制权力分配模型下的条条块块对经济与社会发展变革需求的迟滞与基于自身利益团体利益最大化漠视【这个模式发展大极致，就是在冷战时期美苏都不约而同出现的军工产业复合体控制国家机器与社会舆论的进而改变国家发展走向问题】。对于前者的固化，我们可以通过继续经济与政治体制改革来做设计与安排。对于后者，实际即使如主席的远虑，以其余生发动文革来试图做改变共和国体制固化与僵化命运尝试最后在人性明前归于失败。也因此有了汤因比在【历史研究】中说说的，文革是人类最后一次挣扎的感慨。实际点说，只要人类社会，不管中外，只要存在以私有财产为基础的政治经济分配存在，并且社会存续本身依旧基于人与人之间的血系与血缘关系为纽带的组织形式为社会基本单元。以攫取私人或者利益小团体，并通过血系纽带为传承的固化倾向就不可更替。即使，在彻底工业化的后工业化社会分工制度中，不能克服的制度缺陷之一就是继承。依据血缘关系而不是社会分工需要与个人能力竞争性安排的财产继承安排。对于中国的现状而言，基于一代人在社会激烈改革中产生的政治与经济分配问题，以及其社会整体发展需要之间的矛盾。遏制基于两种社会模型自身固化相互结合的倾向，差不多已经是执政党自身首要执政任务。换文宣表达就是，只有保证社会不断的发展，才能给解决内部社会改革与外部生存空间调整争取必须的时间----以社会稳定为前提的发展时间。这就是我说的那件事，那件09年后一个朋友质问一些人，有没有后悔扶持某派系上去的时候，有人断然的回复，只要体制内部走向前苏联晚期那样的彻底僵化与固话，任何尝试都是值得的。换句话说，为打破这种基于建国时期就形成体制自身而来的体制内部彻底僵化与固化的倾向及其趋势，我批评的左派在理论上与今天社会实践中，不仅观念还停留在冷战最激烈的时代。更不要说，在新技术革命已经逐步改变今天的社会生产关系及其社会组织形式过程中，左翼对这一时期的社会发展实践几乎没有多少可行的社会实践创新与理论创新。快要变成纯粹为批判而批判的清流了。我刚才的几句话，不是牢骚，是我看到的现实。社会与时代的前进不会为任何还停留在过去，停留在昨天的人停下变革脚步的，这种进步与发展的需要，也不是任何社会利益团体自身的力量可以无视的。谁能满足并把自己的理念与这种发展的需要结合，谁就能在不远的未来赢得改变现有利益格局平衡的力量，谁不能适应这种发展的需要，其社会影响只能与摩尔速度的模型快速被边缘化。这个就是，我再三提我年初被人告诫，当时代的变化形成潮流与趋势的时候，不跳进去，就会被迅速边缘化的本意。所以，在新中国又一个三十年开局中，在我们面对两种固化模式相互结合，已经严重威胁到社会发展需要拖延中国应对新技术革命带来的国际政治经济利益重新分配变局下的自身调整时候。既然左派不能提出，像前三十年一样，以服务公众利益凝结社会共识缔造公公众社会，结合今时今日中国国家民族与社会需要的理论方针政策以及具体满足社会各行各业发展实际需要的具体工作方式方法。那么，我说的把不同人群加以区别对待，然后利用各种政治与经济改革，诱之以利驱其竞争的右翼主张与方案就是没有选择的选择。或者说，这是人性的选择，或者说是两种固化模型中多数人自己基于帝王将相梦当然的选择吧。

而打赌的事情，追溯上去，就涉及我在西西河提过的，社会资源分配比例的问题。你这里朝上看几个帖子，我这里引进争议的阶级固化回复表述中，实际在当时就是从，政治与经济改革比例中，所谓上层子弟，中层子弟与社会中下层在依托体制内外各种关键职务安排社会各种资源再分配比例来实现。我在西西河有过详细叙述，在06年左右，那时候基本不删帖，有心思细究的自己找，我这里没有义务说多详细。大体比例是这样，我们根据我们自身的能源供应决定的生产力条件，我们能做到的分配是，对社会公众尤其是对社会中下层而言，上层的制度设计方向是，做的比日本分配比例宽松，比美国小一些。而这些年，上面的帖子设计的分配比例实际这部分资源，以至于有学者对此论述是社会资源分配已经超过封建社会最糟糕的时代=安排给普通人的一度已经压缩到百分之7，当然这个是一家之言不多赘述。上面帖子说的社会政治与经济改革目的之一就是，遏制这样两种固化模式相互结合条件出现的社会发展与变革停滞的问题。你尽管可以说这种固化和你关系不大，但是在后面改革过程中，主流与非主流利益分配关系，会有一个三七开的问题。如果占据社会资源百分之七十的人群是我刚才说关键职务非配，是占据社会关键职务已经达到百分之七十比例的中层官僚及其子弟，那么在制度设计里实际在说，社会中绝大多数人实际在百分之30的社会资源中获得再分配的权力。如果这种固化一旦形成僵化，你只要在这个模型中的大多数人的一份子你明白我在说什么后你绝对无法泰然处之的。

实际在05年左右当时的讨论比例，那个比日本宽比美国严的比例，普通人在社会关键职务占据的比例至少应该是百分之四十，而不是现在绝对小于百分之20.这个就是我曾经提过的，普通人改变自己社会阶层的最后机会。差不多这样的模型，才可能保持社会阶层的足够流动性避免僵化与固话的同时兼顾社会稳定。这样的设计恐怕很多人，特别是网络左派要口诛笔伐。但是我和现实的说，我接受这个安排，是因为基于这样几个前提条件决定的；

1.中国社会不出现大乱的基础是维持社会再生产的最起码的物质供应。
2.满足这种最起码物质供应需要的是工业化及适应工业化发展需要的社会组织关系
3.满足工业化发展乃至不断进化的关键之一是符合工业化发展各种需要的能源与资源需要
4.决定当今世界各国工业化发展需要的各种能源与资源需要，不是通过市场竞争而来的，而是通过确立在联合国成立以来的，基于联合国安理会的五常集体安全体制的再分配而来。
5.国内的政治与经济改革水平，只要是基于市场化与全球化而来，就必然受制于这个基于4的能源与资源配额安排。

你可以从这样的前提条件中，清晰的看到，我们现在国内改革在社会共识缺乏下，要推进中，已经不可避免选择了的以利益为诱导的方法论。而我这里喋喋不休的表达的当年的讨论与当年的打赌。也不过是在说，试图通过基于公共利益再分配各种理论到实践的话题中的讨论与沟通，形成与达成至少一部分社会共识，然后和前三十年那样通过执政党组织能力的这个便利，在今天这个世界政治经济利益再分配格局中，通过集体行动攫取更多的市场能源与资源配额是否可能。如果在特定的时间段不可能，或者没有时间等待多数人形成一定共识，那么只能用现在的选择了。新技术革命的时代，必然带来社会组织与之相应的调整，你我或者任何社会利益团体都无法阻挡，只是这一次我们在这200年工业化进化过程中，我们第一次和列强重新站在了同一起跑线。如果我们错过了这次再分配，恐怕这样的再一次重新回到起跑线，要等到星际航海的时代开启了。如果你认为不至于如此，或者很多人以为的，敌人或者对手彻底不行了一天天在烂掉，你不妨看看最近有关实际各大海洋权益国家的圈地图表，在这张图表里，说的很明白，即使中国在所有海洋权益争端中获得胜利，我们在这个世界海洋权益的再分配格局中，位居世界第六，实际权益只有第五名的新西兰的一半左右。这可见一斑，我说的选择无非是，时不我待罢了。
