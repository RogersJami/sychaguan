我们这里不管啥症状，医生开的药方就是打吊针开青霉素。所以要自己学习，少打吊针。
我的天，您可千万要小心了！庸医害人呐！打吊针的风险很大，特别是对于婴幼儿而言。这绝不是开玩笑，您可以在网上找找相关信息。

滥用抗生素，有些情况下的确是迅速见效，大人孩子医生皆大欢喜，而有些情况下（比如仅仅是感冒发烧而没有明显的炎症）则根本没有意义，因为抗生素只对病菌有作用，而对病毒毫无作用。这还是仅从短期来看。而从长期看，会导致孩子体内病毒的抗药性，将来万一真的得了重病，本该发挥作用的强力抗生素药物到时候却失了灵，这岂不是因小失大？

用药的原则，一个是对症下药，一个是不到不得已时不用药。多数感冒就属于后者。目前世界上并没有能治疗感冒的灵丹妙药，那些所谓的感冒药有的其实治疗的是呼吸道感染等感冒的并发症，更多的则干脆就是强行抑制人体正常防御反应（打喷嚏、流鼻涕等）的玩意，跟治病完全无关。其实无论吃不吃药，基本上都是一个星期自愈。只有病情严重——比如高烧不退、有其他严重并发症状——的时候，才应该根据病症有针对性地用药。

至于中医药，别的不谈，“是药三分毒”就是中医自己的观点。有很多人一说中药，就觉得那都是依照多年验方以天然材料制成的，所以可以安心服用。细想想，这岂不很奇怪？

其实对付感冒最有效的，是大量喝水和休息。就这么简单。如果要再加一条，那就是营养要跟上。生病时消化吸收能力可能会下降，所以应该吃些容易消化的食物。

至于您担心孩子感冒是因为抵抗力低，对此我的观念是，周围环境当中的病毒才是引发感冒的罪魁祸首，当然如果人体抵抗力强就不容易被病毒“搞定”。都说国外的孩子不容易感冒云云，其实主要是因为发达国家环境卫生，感冒病毒少而已。把咱们放到南极去，就决不会感冒，因为那里没有感冒病毒。
您家的老大上幼儿园，所以基本可以断定是幼儿园的环境卫生有问题，孩子之间互相传染。当然，不可因噎废食，因为怕被传染感冒就不上幼儿园。在这种情况下，注意个人卫生就格外有意义。勤洗手，勤漱口，在外穿的衣物不在家里穿，这些都很必要。对了，要教会孩子正确的洗手和漱口方法。洗手时要将手指尖、手心、手背、手腕这些容易被忽略的地方统统仔细地洗到，有必要的话可以洗两遍，不能嫌麻烦。漱口时除了口腔外，还要漱嗓子，这时头要尽量后仰，喉咙才漱得干净。另外，据说用淡盐水漱口可以杀菌，这个我信，但不清楚到底有多大的作用，所以仅供参考。

还有一点，其实医院那地方，能不去就不去，原因谁都能想明白：哪里啥样的病菌病毒都有。这既是我自己的分析，也是很多医生当面对我的忠告。
