　　公告说，本月４日至５日，俄情报机关在格鲁吉亚阿布哈兹共和国完成了针对当地匪帮的侦查作战行动。在此期间，他们发现了恐怖分子的１０个军火库，缴获了３套可用于攻击低空飞行目标的“针”和“箭”型防空导弹系统、两枚反坦克导弹，以及迫击炮、火箭筒、地雷、手榴弹、炸药等大量武器。俄联邦安全局查明，恐怖分子计划在２０１２年至２０１４年期间将这些武器运往冬奥会举办城市索契，以便在冬奥会期间实施恐怖活动。

　　索契将于２０１４年２月举办第２２届冬季奥运会。该城市毗邻恐怖活动频发的俄北高加索联邦区，因此有媒体认为，在这里举办奥运会将有遭恐怖袭击风险。近年来，俄罗斯在北高加索联邦区实施了一系列反恐行动。俄联邦安全会议秘书帕特鲁舍夫此前表示，强力部门将在冬奥会筹办和举办期间确保索契的安全。
