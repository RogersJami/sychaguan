其实俺感觉没那么悲观。当然俺仅凭自己的一点见闻和感受，不妥之处请兄台指正。
俺的感觉是，因为主要的移民吊庄区都是在黄河灌溉农业区，俺待过的那个地方就将荒漠变绿洲的灌溉区，建房、修路、自来水、电信、网络等等应有尽有，一个走向现代化的聚集区，当然还是有其民族风格的，零星散落着建筑样式同中有异的清真寺庙（据说他们之间的信仰派系不同）、差不多每周2次的赶集、各种民族化的生活习惯等，但有一个明显的趋势是与现代化接轨。
举几个例子吧：
学校门卫的穆斯林大叔最喜欢看NBA，那时俺常常在他那蹭电视看；很多中学的娃娃都有他喜欢的NBA球星，爱打篮球、踢足球什么的；俺的穆斯林同事就爱拉俺跟他在电脑上踢实况足球；娃娃们对新鲜事物的接纳是很迅速的，唱流行歌，手机，以及后来小镇上开了个网吧之后，上网聊QQ什么的也都流行起来了；俺离开的时候还有娃娃到新开的像馆拍照给俺留恋……反正，很多时候除了一些民族风格的服饰、还有不吃猪肉的禁忌之外，俺感觉不到太大的差别。
在开斋节的时候，跟俺关系很铁的两个穆斯林同事，带俺混入清真寺做礼拜，听了阿訇的布道，发现阿訇也很讲和谐社会，批评小年青抽烟喝酒等不良习惯什么的。
还有一个比较有意思的转变反映在对教育的态度上。据当地的老教师说，俺去的那个回民中学刚成立那会儿，回民普遍不是很信任教育，因为教材是统一的，老师也有不少是汉民，校长就是汉民。他们怕教育会影响孩子们的宗教信仰。学校对他们而言只是个圈养调皮捣蛋的娃娃的地方（这是那个老教师的表述），到农忙了绝对是干活优先。当然，也可能跟他们的子女多，上学负担重，所以普遍不重视有关系。后来，实行了义务教育之后，他们逐渐开始转变对教育的态度，重视子女教育的也多起来，而且不分男女，都希望子女能读好书，考个好高中，然后再上好大学。让俺感动的是，俺家访的几十个家庭里，他们都很尊敬俺，尽管知道俺是汉民，还是很热情地用他们自己的杯子给俺倒水喝，还邀俺吃饭什么的。后来俺听穆斯林同事说，他们普遍不愿意和汉人共用餐具的，但可能是尊敬俺才不介意的。因为当地老师似乎没有家访的先例。
另外，俺的那些穆斯林同事都很好相处，俺和其中的一对夫妻关系特别好，常常蹭他们的饭吃，现在还联系着。虽然，在信仰上有点差异，但生活的其他方面真没有太大的隔阂。现在基本上都是为生计奔忙，房子、孩子、车子、票子都是主要追求的东西。在现代性面前，一切都烟消云散了。
快离开时，俺竟然在那条穿过回民聚集小镇的路边看到了一个养猪场的广告……
只是一些零星的经验感受。总体感觉，生活方式的同化、精神世界的同化过程很明显。
