wxmang 发表于 2012-5-22 11:21 static/image/common/back.gif
这个问题很有意思，明显的突变，一定有原因。
可以提供一个思路。根据我个人的记忆，列一个时间表。另外，城市的和农村的孩子，区别也比较大。上面说的有可能都是城市子女。

一、82年，农村家庭联产责任承包制开始全面推行。

二、82年，回城知青大增，城市就业压力突然加大。

三、82年，shopping正式提出“中国特色社会主义”和“两手抓两手都要硬”。

四、82年，老干部退休制度开始实施。

五、体改委成立。

第一条，农村私有制全面铺开，意味着今后贫富差距将逐渐扩大，有些混不下去的农民，将成为农民工，涌入城市。

第二条，大量早先城里人从农村回来，有气没处撒，天天耍流氓打群架，成为83年严打的导火索。

第三条，中国的集权资本主义诞生。

第四条，功臣群体要隐居幕后了。

第五条，改开主力军登上历史舞台。

综上，可以推导出，82年，面临的环境是突变性的。所以，今后这群人的性格，那也是和之前泾渭分明的。
我自己的观点是：

        由于回城知青和进城第一代农民工的涌入，使得城市人口大量增加，就业问题很严峻，而国家为了解决安置问题，一般都会把大量的城市户口者塞进体制内的企业或者机关，这直接导致这一大群人成为被圈养的动物，丧失了野性，把计划经济里的劣根性，即教条主义和官僚主义，想当然地传授给下一代。而且因为知青们在之前被忽悠下乡，感觉吃过亏，所以在教育下一代子女的问题上，会过多替子女打点一切，避免他们承担风险而重蹈覆辙，倾向于子女还是得“传统点少激动”，结果反而是把子女也给弄得没独立性了。

         同时，那几年是改开后市场经济发展的重头戏，由于社会法制、经济都很不规范，肯定发生了相当多相当“不够社会主义”的做法，这类做法都会被认为是道德堕落或者国家腐败，这无疑加剧了体制内人士的恶心甚至鄙视，他们在教育子女的过程中，会本能地排斥他们做一些不够规矩的事情。

        82年，毛时代彻底过去，建国初的一系列使命结束，加上城市人口剧增，他们有生理需要和身心放松需要，到了年龄就结婚，新一轮的婴儿潮诞生。（比如美帝45年之后，也有一大轮婴儿潮）
        还有，82年以后出生的人，是普遍在一个相对和平，没有政治运动的环境里长大的，再加上是独生子女，要说泡在蜜罐里，也是不过分的。

        最后，中国从82年开始，社会竞争日趋激烈，边际收益率在九十年代末不断降低，父母在外面拼死拼活，普遍也害怕子女今后会受伤害，包办一切，这是可能性大大滴。
