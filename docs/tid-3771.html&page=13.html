说说自己的观察和思考：
1、图书馆确实太少，库存和更新不够。我大学期间翻过武汉几所大学的图书馆，发现库存大同小异，新的藏书更新速度比较慢，量也比较少。从城镇居民占比来看，当前的图书馆分布格局以及藏书品种数量延续着前30年的某些传统，图书馆是为少数人服务的。大多数人到现在仍然不能享受到图书馆的公共服务。改善这个状况不是民间力量做得到的。
2、电子书。超星在这块持续投入了十多年，但是效果并不明显。他们做的也是在打擦边球，没有理顺版权责任（对超星是隐患，也助长盗版）。这一点我觉得和国内文化有关，公共资源不丰富的情况下，人们倾向于选择搭便车。iturns平台那样的可能并不适用。
3、图书和快消品似乎还是不一样的吧。读者绝对数量本身有限（热门畅销书也就几百万），细分一下每个书商都吃不饱。
4、国内这几年图书炒作的手法有明显的进步，销售渠道也拓展了。电子书等新的消费趋势也渐渐出现。未来可能有较大的变化吧。
