一个小疑问，现在的互联网金融，会不会没两年就变成昨日黄花，故事能讲多久呢？

你好，互联网在未来是象水电煤气一样的社会基础物资，股票是炒未来的，炒预期的，如果互联网金融已经成为现实，就是互联网股票的末日，如果大众对互联网股票的预期一致了，说明已经部分照见现实了，就是互联网股票出货的阶段了，上演最后的疯狂。被质疑的股票反而安全，拿不住的股票才有不断上涨的动力。
乐视贾跃亭说过，用未来定义未来，用未来定义现在，股票过去的价格是由现在决定的，现在的价格是由未来决定的！乐视网是创业板我最放心的公司，每一步都领先别人几年布局未来，将来必然会超过5000亿市值。
股票终究只有少数能大赚，只有少数人能站在未来看今天，从而行动改变未来，才会坚定不移，不为短期涨跌所动。
