好在刹车了。

说到这篇文字，想起不少往事了。记得在网络上最早的文字一篇是写岳飞的，一篇是写俾斯麦的。前者本质是民族主义，后者说的是帝国主义。这些年，我的文字实际记录了我的思想的转变。其实我文字记录真正意义的转折点，在2008年奥巴马当选之后。文字发表在西西河，标题是总结，从印度巴西比较等等文字的总结。文字记录的主要内容，是我在奥巴马获得选举胜利当天和朋友的讨论。我在那天接受了朋友的忠告，她对我说如果你想思考中国问题，你就必须从解决世界问题的角度思考，不然没有出路。从那天我开始放弃的是民族沙文主义，汉民族中华民族至上的沙文主义。同样在过了一年多后，在经历很多事情后，我在中帝国一文很明确的表达了这样的观点，帝国是死路。主要原因，前面有人实际已经说到，帝国之路通过对外转嫁危机的方式来缓解内部危机，不仅解决不了内部危机的形成因素，也处理不了造成帝国内部危机最根本的原因—帝国内部的绝对固化与僵化，以及由此而来社会分配内卷化问题。一旦扩张不能继续，必然内部问题和外部问题接踵发作。到那时候再解决内部问题，恐怕早已不可收拾。美国总统前不久有关单边主义与绝对霸权的讲话，是美国人在两伊战争付出以十万亿计的代缴后声明。殷鉴不远，自当引以为戒。


而两篇文章之间发生的一件事对我触动很大。那就是，我曾经提过，当年08危机之后经济救援的当口，我提过通胀数值百分之4很重要。为什么，那数值被突破，就意味着经济调整的手段失效，那么经济结构调整就实际进入行政干预阶段。我这里说起当年的事情，触动在于，围绕当时经济改革与政治改革谁优先的时候，那个在特定时间段中引起博弈的关键考量数值百分之4。尽管当时的国务院，已经竭尽全力试图阻止，那些回头望去的算你狠，逗你玩等等一轮轮的投机与炒作的狂欢。这一轮轮狂欢中自诩代表资本自由的力量，一步步把我们国家的经济体制改革提升到了政治体制改革优先的阶段。而紧接着围绕着这个政治体制改革拉开的左右争斗，最终在去年达到了一个高潮。造成这些事情的根源，今天考一时转嫁内部问题解决不了，什么时候试图通过外部问题掩盖内部问题带来的矛盾与裂缝，都无助于于解决内部问题。甚至，还会带来外部势力干涉的觊觎。最明显就是去年日本政府觊觎，去年初薄熙来案的影响，判断中国将陷入长期内乱进而挑起钓鱼岛国有化事端冲撞中国政府底线赢取国民视线，以此来转移因福岛核事故以及3.11地震之后日本当时的执政党内外交困的潜伏危机。日本政府始料不及的是，正式他们的趁火打劫才成了，去年年初政治冲击在此之后迅速告一段落的契机。让中国人同仇敌忾，一致对外的缓和内部矛盾的台阶。还是殷鉴不远，自当引以为戒。

其实在奥利佛斯通视频回复中，我在写的是说之难。这个说之难，我在这些年网络遭遇还有点体会的。比如写文革了，就有一批人说我是极左。说到，改开三十年后如今社会一些社会层面的现状了，就被说成是为这些现状张目。如此不一而足。前面一段文字，提到百分之4通胀控制目标那时候说房价股价问题，我几乎用尽力气在说服能看到文字的人不要赌。结果这种行为，没少被人说跳大神。我这样说只是在说一个话题，我今天在网络上说这些，实际并没有让我实际得到多少好处。甚至是这样的是是非非，尤其是中帝国中提到的具体事例，不给我带来各种不必要的麻烦已经是万幸了。而且一度因为网络上写了一些我经历过的事情，讨论过的事情，不仅被朋友列为反面教材。而且，在09年一些转折博弈到关键的时候，一度被怀疑过是不是我网络上写东西说多了以至于影响到了什么，说实际点那句是不是你说多了背后，是可以让叫真的人让我今天的所有归0的。【那之后我一度删除过很多文字，这又成了迷途笨狼和海峰这样的人说我故意造假忽悠人的证据。】说实话，我对网络上一些人的行为方式不能理解。我当时写，不过想证明给一些人看，通过坦诚的讨论是可以通过释放一定的提前量信息这种方式提前把涉及社会重大利益的调整消化一部分矛盾与冲突在调整之前的，是可行的与值得考虑的。但是这些年在网络上因为言论与价值观点遇到很多冲突与冲撞。比如像海峰这样的人，一面在讨论中捏造我没有说过的话作为自己攻击我的证据，并且在讨论中被人指出这捏造后还能侃侃而谈说这样做有什么道理，然后没过多久就能在另一些讨论引证中我没有说过的话来证明他的观点。这样的人就是我在西西河看到的很多自称为站在多数人一边的左派。这样的左派，在逐步成为西西河的主流之后，当年一些打赌的事与人看到这里其结论是，多数人引入政治博弈是不可控的。这个结论实际是去年路线斗争的核心，未来十年，群众路线是把群众引入政治博弈，还是引入政治监督。某种角度说，正式海峰这样的网络左派把去年的那条路线逼进绝路。我也是因为这样的事情，成为各种网络媒体的一种潮流。我开始认同，那种把不同人群隔开，然后做什么的模式。我说过，我曾经思考汤因比说的那句话，文革是人类社会最后的挣扎很久。也思考了，主席为什么明知道自己会输给了人性，还是毅然发动了文革。我相信，这里还有很多事情我没有明白。当时我亲眼看到了什么如何最终输给了人性。那么既然什么输给了人性，而我们又必须向前走，向前继续改革。除了选择驱使人性朝着一时之利的引导---这样的右翼方案。而内外矛盾发展到今天，以及新技术革命带来的社会组织重组以及由此开启的国际政治利益再分配，还有就是前面提到的小冰期的预期及其应对，尽管诸公还能侃侃而谈而时不我待如今实际一时已经别无他途。某种角度说，现在的帝国路线实际也是输给了人性。这是讽刺还是幽默，毕竟忽悠人做炮灰实属不易了。就作为此间谈资，聊胜于无吧。
