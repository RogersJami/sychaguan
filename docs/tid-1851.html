饮食疗法----常见食品性味

本帖最后由 newyorker 于 2012-5-7 05:03 编辑

1997年4月17日 发表于新英格兰医学杂志------正式确立西方饮食疗法的疗效。
DASH饮食是由1997年美国的一项大型高血压防治计划（Dietary Approaches to Stop Hypertension；DASH）发展出来的饮食，在这项计划中发现，饮食疗法可以干预高血压。将患者分为（1）典型的美国饮食，（2）低糖且水果蔬菜丰富的食物和（3）联合水果蔬菜丰富且低脂少肉的饮食。后两种饮食方式均可以降低血压，这就是DASH饮食的来源。其核心概念是限制盐的摄入，饮食中如果能摄食足够的蔬菜、水果、低脂(或脱脂)奶，以维持足够的钾、镁、钙等离子的摄取，并尽量减少饮食中油脂(特别是富含饱和脂肪酸的动物性油脂)，以及红肉、甜食和糖的摄入，可以有效地降低血压，如今，DASH饮食和运动一起，已经成为了抵抗高血压的一线治疗方式。《美国新闻与世界报道》对现有的各种饮食进行了评比，DASH 饮食的综合得分最高，地中海饮食名列第二。
DASH 饮食最主要的优点是有利于心脏健康和营养丰富，缺点是不容易坚持和比较贵。具体的评估标准和 DASH 饮食的各项得分如下 (满分为5分)：
•        综合得分：4
•        短期的减肥效果：3.5
•        长期的减肥效果：3
•        方便程度/是否容易坚持：3
•        营养：4.7
•        安全性：5
•        对控制糖尿病的益处：3.5
•        对心脏健康的益处：4.3

但是，饮食疗法对中国人却不陌生。

中国食品中，最符合现今流行、也最被国内外营养学家们推崇的营养标准DASH的食品是饺子。
为防止多种现代病，中外营养学家、医学家们几乎天天都在呼吁人们要平衡饮食，并研究出最为科学的金字塔结构（DASH饮食），进食量从底层到顶层逐渐减少，依次为各种谷物、蔬菜水果、各类动物食品、奶类食品，最后一层是油。但饺子就能让人们轻轻松松地掌握其分寸。饺子最大的优点是总体营养成分配伍合理，人们每天所需要的蛋白质、脂肪、糖类等各种营养成分不仅齐全，而且符合上述金字塔结构给出的比例：面粉做的饺子皮--谷物类食品，青菜--含量一般都比较丰富，优质蛋白--饺子中的猪肉或鸡肉、羊肉等，而且分量适中，因为人们很少吃全肉馅饺子，人们调饺子馅时一般还会放点油特别是植物油--补充了植物类脂肪。人们每吃一个饺子却能将所需要的各种营养素基本摄入。饺子第二大优点是容易定量。饮食定量不仅对某些病人如糖尿病人、立志减肥的人意义重大，它也是现代人保证健康的一大前提，其重要性勿需赘述，饺子论个计量的办法却很少有人不能掌握。

中华食疗文化源远流长，是中国人的传统习惯，通过饮食达到调理身体，强壮体魄的目的。食疗是在古人寻找食物的过程中萌生的。随着食疗经验和知识的积累，食疗理论也产生了。自神农尝百草后人们认识到了食物、药物具有不同的性能功效，商朝出现烹饪鼻祖伊尹创制汤液技术，周朝还专门设立了食医，可见对食疗的重视。《黄帝内经》如此讲食疗：“五谷为养，五果为助，五畜为益，五菜为充，气味合而服之，以补精益气”，奠定了中医食疗学基础。唐朝孙思邈被称为药王，在他的《千金要方》食治篇中说：“为医者当晓病源，如其所犯，以食治之，食疗不愈，然后命药”，这是最早的食疗专篇，他还说：“安身之本，必资于食”，“不知食益者，不足以存生”。李时珍说：“饮食者，人之命脉也”，进一步强调饮食、食疗对生命的重要性。食疗原理也是利用食物的性能功效调整人体的阴阳平衡，协调人体脏腑。

人有病就要用药，但这是不得已而为之的。因为药有阴阳偏性，有偏性就会产生毒副作用。医生高明，设计处方严格按君臣佐使，以降低毒副作用；医生不高明，就不能有效地降低毒副作用，令世上发生药源性和医源性疾病，扩大疾病阵营。食疗学的思想是从医学中提炼出来的，它的中心认识就是“寓医于食”。由于“寓医于食”，食品的力量占据优势，就在很大程度上降低了药品的毒副作用。同时，由于“寓医于食”的食疗诊断的技术要求低于医术诊断，所以也就减少了医源性疾病。

中医食疗学认为，食物与药物同样具备“寒”、“热”、“温”、“凉”四性。只是食物的区分更笼统些，一般就分“寒凉”“温热”以及“平”。但其中食物中相当部分属于“平”，这就是这类食物可长期服用的原因。这一“平”性的特性，既说明了这类食物不具备“寒凉”和“温热”的特质，长期食用不会产生“寒象”和“热象”。中医学就是根据服食后对机体的反映来确定其性质，也说明中医食疗学中的平衡饮食中这类食物应占多大比例?如谷类食物属平性，因此，谷类食物在我国饮食中占重要比例，现代营养学认为，谷类主要含碳水化合物，而碳水化合物占总热能摄入的55%～65%。

所谓「四气」又称「四性」，即寒、凉、温、热。加上不寒不热的平性，又可称为「五性」。对食物作属性鉴别归类，是中国特有的传统。食物的「性」是从食物作用于人体所发生的反应而概括出来的，与食物的食用效果是一致的。这种效果主要反映在功用上，但也可以反映在副作用方面。了解食物的五性，就可以指导人们的饮食宜忌。

【五味」就是食物的辛、甘、酸、苦、咸五种味，实际上还有淡味和涩味，但一般习惯把淡味附于甘味，把涩味附于酸味。中医对五味的认识有两个方面：一是指食物的具体口感味觉，即滋味；二是食物性质的抽象概念，即食物的作用。

newyorker 发表于 2012-5-7 00:30:45

根据中医理论，辛味包括辛辣、芳香味，有宣散风寒，行气活血的作用，例如姜、葱、酒、胡椒、辣椒、玫瑰花等；除了口感觉甜味之外，中医认为甘味还有补益强壮的作用，例如红枣、南瓜、葡萄、饴糖 ，以及多种动物的肉和内脏。甘淡味有利尿除湿作用，例如薏米、冬瓜等；酸味包括酸涩味，有收敛固涩作用，能止汗、止泻，例如柠檬、乌梅（酸梅）、醋等；苦味有清热泻火，燥湿作用，例如苦瓜、豆豉、槐花 等；咸味有软坚散结、润下作用能治痰核、便秘。具有咸味的食物多为海产类及某些肉类，例如盐、海带、紫菜、蟹、海参等。

大　　蒜
性味：生辛热，熟甘温，有小毒。
　　归经：入脾、肾。
　　功效：杀虫除湿，温中消食，化肉消谷，解毒，破恶血，攻冷积，外灸散痈。
　　主治：痈肿疔毒，恶疮发背，水气肿满，泄泻痢疾，腹中冷痛，宿食不消，杀钩虫、蛔虫，解食蟹中毒及蜈蚣、蝎子咬伤。
　　禁忌：阴虚火旺者忌用，肝热目疾者宜戒之。
　　说明：大蒜虽有杀虫解毒祛寒健胃之功，然而性质属于温热之品，同时亦有刺激性，能刺激肝、肺、胃及眼睛，若患有肝热或肝炎患者、脾胃火重者、眼睛痛或有炎症者宜戒之。大蒜多食则伤脾、损肺、坏肝、伤目、生痰、发嗽。

白　菜
性味：甘、平寒、无毒。
　　归经：入肠、胃。
　　功效：清热除烦，解渴利尿，通利肠胃。
　　主治：口干烦渴，大小便不畅，解木薯中毒，外用治漆毒生疮。
　　说明：白菜原产我国，已有数千年历史，栽培广泛，种类也较多，因此一年四季都能吃到鲜嫩可口的白菜，以它的形状来分，大致可分为普通白菜、大白菜和苔用白菜三大类。
　　普通白菜较好的又有十几个品种，如长梗种、短梗种（小白菜），塌地种等；
　　大白菜又称结球白菜、卷心菜或黄芽白，比较好的也有六七个品种；
　　苔用白菜，一般通称为菜苔，红菜苔或白菜苔，它是一种品种多，味道好的蔬菜。
　　白菜性质也较复杂，红菜苔性质偏向平和，而白菜苔和普通白菜则性微寒，大白菜性质寒冷，若煮汤食，性质大寒，虽然清肺热止咳利尿，但又能发诸风冷，凡肠、胃或心、肺虚寒者不宜食或慎食。若白菜经水烫晒干后，寒性则大减。

　　主治：虚渴，病后口干，解酒毒，热性出血（如：吐血、鼻出血），小便不通，血淋尿血。
　　说明：莲的全身都是宝，除莲藕可作治病食用外，其它部分如藕节、荷叶、荷梗、莲子、莲子心、莲须、莲蓬均可入药治病。

南　　瓜
性味：甘、温平、无毒。
　　归经：入胃、大肠。
　　功效：除湿祛虫，退热止痢，止痛，安胎。
　　主治：下肢溃疡，阴囊湿疹，蛲虫，绦虫，蛔虫，骨蒸潮热，痢疾，胎动，胃痛，解鸦片毒。
　　说明：南瓜系日常食物，性质属于平和，外用虽可除湿，然多食则易生湿发黄，令人气壅。

冬　瓜
性味：甘、微寒、无毒。
　　归经：入肺、脾、大小肠。
　　功效：清热毒，利小便，止渴除烦，祛湿解暑，解鱼毒。
　　主治：热毒痈肿，小便不利，心胸烦热，口渴面热，小腹水胀。
　　禁忌：1、久病与阴虚患者忌食。2、服滋补药品期间忌食。
　　说明：冬瓜是一种解热利尿比较理想的日常食物。其性质属于寒凉，而煮汤服食更甚。如要达到解暑清热、止渴利尿作用，就需要煮汤（连皮更好）服食；但体弱肾虚病者食之会引起腰酸痛；体胖之人欲得体瘦轻健者则可常食之。

白萝卜
性味：甘辛、平、无毒。
　　归经：入肺、脾。
　　功效：下气消食，除痰润肺。

葱
性味：辛、温、无毒。
　　归经：入肺、肝、胃。
　　功效：发表解肌，利肺通阳，解毒消肿。
　　主治：外感初起头痛，寒热，无汗，鼻塞，面目浮肿，风痹疼痛，乳痈，乳闭，安胎，大小便不畅，透发麻疹，呕吐腹泻，妊娠下血、小腹痛，杀一切鱼肉毒。
　　说明：本品治病多用葱白，即去掉青色部分葱叶，应用葱头连根须部份。取其辛香气味，用以发表通络，因有表汗作用，故有表虚多汗、阴虚有热患者慎用之。

猪　　肉
性味：甘、咸平、微寒、无毒。
　　归经：入脾、肾。
　　功效：滋养脏腑，滑润肌肤，补中益气。
　　说明：猪肉是日常食品，一般健康人和患有疾病之人均能食之，但多食令人虚肥，大动风痰，多食或冷食易引起胃肠饱胀或腹胀腹泻。对于脂肪肉及猪油，患高血压或偏瘫（中风）病者及肠胃虚寒、宿食不化者应慎食之。一般入药均为猪瘦肉。

牛　　肉
性味：甘、温平、微毒。
　　归经：入胃、脾。
　　功效：安中益气，健脾养胃，强骨壮筋，补虚损，除湿气，消水肿。
　　主治：胃弱脾虚，水肿胀满，腰膝乏力。
　　禁忌：疮毒或皮肤湿疹患者忌食。
　　说明：牛肉是日常食品，有温补脾胃之功，但又是一种发物，对于患有疮毒、湿疹、瘙痒症等皮肤病症者应戒食；而患有肝炎、肾炎者亦应慎食之，以免病情加重或复发。
