如何计算房屋建筑面积
买房的时候，有律师帮忙看合同，没关心技术细节，事后他给了一个计算房屋面积的说明，今天偶然翻到，晒出来给茶友们参考。
建筑面积的计算
房屋建筑面积就是产权证登记的实际建筑面积，根据国标GB/T 50353的规定，由套内面积和公有分摊面积两部分构成。建筑面积S = 套内面积S1 + 公有分摊面积S2
套内面积S1包括使用面积、阳台面积和墙体面积。阳台面积按一半面积计算。墙体面积有两部分组成，自有墙体面积和分摊的公有墙体面积。其中公有墙体是全体业主公有的，包括楼内和楼外分开的外墙（也就是山墙）、户与户之间分开的分户墙、套与公用空间分开的分割墙。套内的自有墙体按全部面积计算，分摊的公有墙体按一半面积计算。S1 = 阳台面积*0.5 + 套内自有墙体面积 + 分摊的公有墙体面积*0.5
公有分摊面积S2计算复杂。公有面积是全部业主共同占有、共同使用的面积，简单说，就是整幢建筑物面积减去各户套内建筑面积之和，由电梯井、楼梯间、走廊、门厅等公有部分和公有墙体的一半组成。各户分摊到的公有面积由各户套内面积乘以“分摊系数”得出。分摊系数是公有面积除以各户套内面积之和得出个一个比例，取到小数点后4位。这个系数往往被保密，是购房者最难得到的房屋信息。S2 = 【（建筑物总面积 – 各户套内面积之和）/ 各户套内面积之和】 * 套内面积
所以，房产证上登记的房屋建筑面积的计算方法是：建筑面积S = 套内面积S1 + 公有分摊面积S2                = （使用面积 + 阳台面积 * 0.5 + 分摊的公有墙体面积 * 0.5 + 套内自有墙体面积） + 【（建筑物总面积 – 各户套内面积之和）/ 各户套内面积之和】 * 套内面积
面积测量计算方法
按国标《房产测量规范》、《建筑工程建筑面积计算规范》进行测量计算。
最简单的测量工具就是卷尺。
测量房屋边长的测点一般取离地面1.2米左右的高度，保持水平；测量墙体的宽度（也就是厚度）的测点一般取与门、窗0.2米的位置，避开包门口等装饰。
按房屋和阳台的形状，比如长方形、正方形、梯形、半圆形等，用对应的数学公式计算，不要忘记加上外墙或分隔墙、分户墙的一半厚度。
购房合同和产权证的阅读
购房合同附件中公有面积构成不填写，可能有问题；
产权证只填一个总面积数，而使用面积、套内面积、分摊的公有面积都是空白，可能有问题；
购房合同约定面积和产权证登记面积是否一致，如果一致，可能有问题，因为施工中很难做到完全精确；
标准层住宅上下楼面积是否一致，不一致，可能有问题；
普通住宅一梯两户分摊的公有面积每户在10-20平米之间，如果达到20-30平米，可能有问题；普通住宅（非豪华、超大户型）公摊面积与套内面积的比例应在20%上下；
阳台面积是否按一半计算；
物业缴费面积、供热费缴费面积和产权证登记的面积是否一致，不一致，可能有问题；
窗台、露台、独立使用的地下室、车库和人防工程不包括在实际的建筑面积里。

很实用的科普，赞一个！
但文中有三处表述还不够严谨：
1.房屋产权面积计算的依据不是GB/T 50353这本规范，而是GB/T17986，两者的用途是有差别的：GB/T 50353应用在工程设计、施工及工程造价计价等领域方面的，而GB/T 17986是应用在房产产权、产籍管理、房地产开发利用、交易、征收税费以及城镇规划建设等领域方面的
2.阳台面积计算分两种情况：a.封闭阳台，此类阳台面积是按全面积计取的，b.非封闭阳台，此类阳台面积才是按一半面积计取
3.外墙并不是山墙，山墙在古代建筑上是特指建筑两个侧面上部成山尖形的横墙。现代演化成指主立面左右侧的外墙。
