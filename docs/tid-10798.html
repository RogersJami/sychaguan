几个图床及使用方法

2015年10月更新：

如果对国内图床poco使用操作有难度，建议使用贴图库图床，tinypic 图床，或以下推荐的其他图床。

贴图库网址：

http://tietuku.com/


——————————————————
——————————————————

poco图床

目前是茶友使用最多的国内图床。贴图方法，可参见

poco图床贴图方法【5月21日更新】
http://www.sychaguan.com/forum.php?mod=viewthread&tid=10617




其实网络上使用图床，贴图的方法大同小异，只要稍微琢磨一下就可以了。希望这些图床中，总有一款适合你。

图床1
http://tinypic.com/

贴图方法简单，可参见羽笛茶友的详细说明帖
http://sychaguan.com/forum.php?mod=redirect&goto=findpost&ptid=10166&pid=198095


以下几个图床也是免费，可选择注册或不注册，使用方法和tinypic 差不多。

图床2：

http://imgur.com/


如图：
步骤1） 点击Computer 选择本地图片，上传。

http://i.imgur.com/xnm8TeM.png

步骤2），这时会看到文件等待上传中。。。点击Start Upload 开始上传。

注意，因为在茶馆，点击图片可见大图。如果茶友发现图片太大，看起来不舒服，可更改图片尺寸（高X宽），一般来说可选择640X480。

对比一下上图片效果。

http://i.imgur.com/hPL0Vqu.png

http://i.imgur.com/hPL0Vqu.png

步骤3）等图片上传完后，贴到论坛：选择 BB Code（ Message Board & Forum），将code黏贴paste 到帖中，即可。
帖子发表后，图片就会显示出来。


http://i.imgur.com/o0VPk8s.png



图床3

http://craftimage.tk/

步骤1）点击 select file

http://craftimage.tk/djP.jpg

2）上传后，点击imagelink， 图片就会打开，得到URL。

3）然后点击茶馆帖子的工具栏【图片】——网络图片——输入URL，即可。

http://craftimage.tk/djQ.jpg



图床4
http://imgly.net/

可以同时上传多图，图片上完后，选择：BB Code Embedd
然后copy & paste 贴到论坛贴中，即可。 帖子发表后，图片变会显示。

http://imgly.net/img/f841d51.jpg




http://imgly.net/img/c85cb3c71.png





图床5
http://www.imagebam.com/

可以直接上传图片。另外支持ZIP格式上传大量图片，单个ZIP文件最大限制为100M，40天无任何使用记录则
会被删除。

如图（这是100X100尺寸的效果），茶友可选择300X300 或根据情况选择尺寸
http://thumbnails112.imagebam.com/32780/a7389c327796716.jpg
