写个能拿奖的小说
====================
750万在京买房的几种方案


　　“我准备在北京买套大房子，后来有人提醒我说也买不了多大的房子，5万多元一平方米，750万元也就是120平方米。”上周，莫言接受媒体采访时如是说。爱操心的人如此之多，大家都想知道，750万元究竟能在北京买到什么样的新房？

　　如果要在北京买和莫先生知名度相配的房子，以目前成交单价已飙升至15万元/平方米的钓鱼台七号院为例，750万元只能买50平方米。退而求其次，我们再考虑正热销的广渠路上的金茂府，三季度的成交均价攀升到54000元/平方米左右，不过这个项目的主力户型至少都在280平方米。

　　如果选择单价在5万元左右、120平方米左右的房子，目前在长安街沿线有两个项目：一个是位于地铁1号线南礼士路周边的金融世家，另一个是万寿路地铁边上的国悦府。在金融世家，据说找套700多万元的房子较难，在售的三居就算小面积的了。相比之下，国悦府的户型选择性多一些，660万元能得手一套60平方米的LOFT，但据说不一定有房了。

　　高端住宅望而不及、城区房子精而稀缺，到北京四、五环外能挑到什么房子呢？上周六刚对外开放样本间的顺义誉皇殿项目，推出380平方米的别墅。按照该项目销售人员的介绍，如果能赶上开盘优惠，800万上下有机会住上套别墅。不过，装修可就没钱了。

　　如果不想让自己有那么大的压力，其实还可以选择大兴、昌平等郊区的大企业开发的项目，如位于大兴旧宫、单价大约2.5万元、155平方米的金第万科朗润园项目；位于大兴线最南头的天宫院站的保利春天里项目。这些项目即使选择大户型总价也在400万元以内，这样莫老还有近乎一半的“银余”自行支配，起码满足了在北京、有大房子两个条件。

　　而另一个不得不提的现实是，有钱不一定能买到中意的房子。无论在金第万科朗润园、还是誉皇殿等项目，销售人员均透露房子与客户的供需比至少都在1:2以上。当然，有业内人士提醒，如果莫言能直接和开发商对话，以莫老的名气，开发商不排除会给开个“后门”，来个“内定房源”；但同时莫老也要有“居住地隐私”被公开的心理准备，保不准在接下来的推销中，就能听到：“连获诺贝尔文学奖的莫言都在我们这里买房了！”

　　还要特别说明的是，总价在500万元以上的房子一般都被中介们定义为高端住宅，除了品质上和物业费要高于普通住宅，还有是买房的契税比普宅要高出一倍，按照非普的3%纳税的话，总价750万元的房子仅契税和公共维修基金两项就要交30万元。
