wxmang 发表于 2012-4-21 12:34 static/image/common/back.gif
理论上所有通胀那都是票子太多的结果。
俺觉得炒作成分，乱涨价，包括农村宗族势力对城市的占领，都是很重要的因素。

现在城市流动人口太多，流动性太大，商家的信誉是不可能保证的，很简单，反正就赚你一次钱，还不如多揩你油。所以价格就乱涨。

还有，中国过去的计划体制，是抑制商业、服务业的，城里人就是工厂大院、部队大院、机关大院、事业单位大院之类的，改开后，相当大程度的第三产业其实是被进城农民所把持的，而农村的宗族势力、村落势力，地域势力（比如西南官话区的小吃）导致他们变相垄断某一行业，和黑社会有一拼，竞争不充分，因此价格粘性太强。
