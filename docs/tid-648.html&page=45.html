【扬州高考报名人数首次下降】
2012-03-18
去年，我市高二学生报考“小高考”人数首次减少，而这一减少的影响很快显露出来。“我市的高考报名人数一直居高不下，在全省考生数减少的基础上不降反升，而2012年高考报名，扬州考生数首次减少。”一位业内人士透露道。“江苏的高考考生、‘小高考’的报考人数近年来一直走低，这种趋势从2008年就已经开始。”

【河南2012年初中生人数下降】
2012-02-09

【北京中小学新生今后三年大幅下降】

2011-2-25

这份研究报告显示，“十一五”期间，北京适龄人口将呈大幅度下降趋势。全市小学入学人数在2008年至2010年预计每年稳定在7.5万人左右，比人口高峰期减少了一半多。与此同时，2005年之后北京初中阶段学龄人口将发生巨大变化，初中毕业生数量大幅减少，其中2005年初三学生总数降到12.6万人，2006年再降到10万人，2007到2009年基本保持在9万人左右，减少一成。
