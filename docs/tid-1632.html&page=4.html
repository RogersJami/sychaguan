penny1003兄，您的问题是相当的精华，您谦虚了！好蜂蜜见过一次后就知道了。呵呵~~
1. 如果您订的是蜂巢蜜，那收到的就是最原始的蜂巢，蜂蜜被蜂蜡封存于巢格里面。若是新巢的话，是可以不用过滤加工直接掰下一块食用的；若是老巢，口感会差一些，您可以将蜂巢用不锈钢水果刀切碎后，经棉纱滤网（用沸水煮消过）过滤即可。卫生问题可以保证，因为原生蜂蜜有一定的灭菌功效。但是野生巢蜜有可能在蜂巢表面会带有一些少许的砂子或小的树枝什么的，这是在采割过程中有的时候难以避免的，并不是故意掺入以增加重量，家蜂巢蜜就没有这种情况了。
2. 蜂蜜的食用方式可以按个人所好。仅需要注意的是，在食用的时候不要让蜂蜜的温度超过40度以上，不然会破坏掉其内最有价值生物酶。
3. 蜂蜜的保存方式很简单：密封好后置于避光处，常温（不宜超过30摄氏度）或低温（10摄氏度左右）均可，重点是密封和避免高温（40摄氏度以上）。大多数蜂蜜在低于20摄氏度以下就会开始缓慢结晶，结晶不会改变蜂蜜的质量，改变的只是它的物理状态。就像水在零度以下会结冰，融化后还是水一样。
3. 单花蜂蜜是蜜蜂主要采集一种花粉而得，蜂蜜成分单一较稳定，功效也比较明确；百花蜂蜜是蜜蜂采集多种花粉而得，一般的野生蜜都是多花蜂蜜，这种蜜成分比较复杂，幼儿和体质较为敏感的同志在食用百花蜜的时候需要注意。百花蜜的药用价值比较高，可作药引，同时保健功能也比较强。在口感上某种单花蜜会好于百花蜜，比如荔枝和龙眼蜜的醇香较为特別，是比百花好，但并不是所有的单花蜜都比百花蜜好。
4. 纯天然蜂蜜的产量受气候影响较大，气候过旱会导致植物开花少甚至不开花；而过涝花蜜又会被冲走；温度不适宜的话有花而又沒有花蜜分泌，所以这个供应是无法估计的。五月份收的是今年的第一次百花蜜，之后是荔枝和龙眼蜜，八月份左右是小叶桉蜜，十月份是大叶桉蜜十二月份至来年一月份是野菊和枇杷蜜。您可根据自己的需要来安排您的计划。

呵呵，市售的未经加工的天然蜂蜜确实是很少，更多是经过加工后的浓缩蜜及假蜜。但不是说浓缩蜜就都是假蜜，其与天然蜂蜜的差别主要是在生物酶的有与无方面。浓缩蜜在加工过程中会因为加热工艺导致生物酶损失殆尽，而生物酶也是蜂蜜重要营养价值的一部分，故而浓缩蜜虽然也是真蜜，但是不是好品质的蜂蜜。但随着网购的普及，越来越多的蜂农会以网店的形式销售其生产的天然蜂蜜，当然，价格自然就不是市售的20-30元一斤所能比拟的了。
