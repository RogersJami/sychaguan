请注意，国宴是分餐制，每人面前都要上这么多食品。


国宴厨师孙应武先生的回忆录中写道：


半个月前接到外交部的命令准备晚宴。一周后，准备工作进入“务实”阶段。孙应武组织厨师们洗菜、切菜、调味和装盘。“即便是冷菜，也得在成熟24小时之内上桌。”
热菜更复杂，以三丝鱼翅来说，先要精选上等的冬笋、鸡胸肉、火腿肉，切成和鱼翅相等的细丝，再把冬笋的涩水处理掉，把鸡胸肉加调料腌制好——“堂菜的鱼翅做法很讲究，基本保持了谭家菜的风格。”孙应武说，要用鸡肉、鸭肉、火腿、蘑菇等配料，小火焖煨约十个小时，这样一来，出锅的鱼翅清香四溢，入口浓香。

孙应武和辽宁省大连市长海县獐子岛当时任潜水队长的王天勇先生说：


外交部打听到美国人喜欢海鲜，决定准备1000公斤新鲜黄海鲍鱼备用。
獐子岛人民公社的潜水队接到了采捕鲍鱼的任务。在零下20多度的天气下，他们冒着鲨鱼袭击的危险进行了数十次采捕，终于在1972年1月底采来了1500公斤鲍鱼，并从中挑选出1000公斤精品，运到北京。
