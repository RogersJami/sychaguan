解决房地产问题与土地财政问题的一个思路



成立“中国国土开发总公司”

把全国国有土地的所有权，划入该公司

由该公司按市场经济的方式，设立在国有土地所有权基础上的使用权，使用权的设立和出售，按现代企业制度，自主经营，自负盈亏
剥离地方政府“城市经营”的商业职能

使地方行政机构，回归到单纯的行政职能，承担商业秩序监管和利益分配工作

国土开发总公司的经营所得，汇总上缴国务院，纳入中央财政收入的池子，由中央财政按各地物价水平，常住人口数量，加权平均分配到各地方，实现全国公共服务水平的均等化

各个地方政府与国土开发总公司下属的地方分公司的关系，界定为地方行政机构与中央企业子公司的关系，按政企分开的原则，互相财务独立

各地地方政府的债务与融资平台的债务，同时划入国土开发公司。

地方政府不再承担债务，也不得再有商业经营活动

地方政府的财政收入与支出，只限于地方行政性事业的支出，功能类似于“全国行政总公司”的子公司

有关居民福利的转移性，分配性支出，由中央财政对居民直接补贴，补贴到个人社保账户


任好大王 发表于 2012-2-22 11:22:02


至于房地产的发展方向

不是限购，或者房产税

而是取消对 投资性或投机性房地产的银行融资

使房地产回归到一般“消费品”的属性

对居民购房的融资服务，只限于第一套房产

类似于“汽车的消费信贷”

贷款风险的担保，以贷款人的偿付能力为限，与汽车一样，房产本身不再视为可抵押品

当然，在实现这一转变之前，要开发出另一种”居民投资载体“，吸引民间剩余资金转入新战场

在新战场当中，银行不再给予个人投资或投机活动融资，不再提供杠杆。

任好大王 发表于 2012-2-22 11:35:04


为居民个人的投机活动，提供金融杠杆，是社会主义经济过程中的一个特例

这个特例的开放，是为了国内经济战场的特需，目的是为了做一个套，绞杀民间剩余资金，吸引中外私有资本与生产资料的脱离，掩护国有资本全面垄断生产力的行动

这一阳谋已经完成任务

那么以后，特例将不再开启

银行将不会再对任何非生产性的经济活动提供杠杆
