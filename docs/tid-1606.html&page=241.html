你在这儿反复的提接管。山某和其它人说的也够多的了。建议你去看一看你说的这几十家的规模和奶源的分布，比如说蒙牛奶源分布，再多少补一些冷链物流的常识。接管有多么不靠谱应该就很明白了。

还是你不断强调的接管能保证消费者的信心。可是事实上，连你自己都多次重复说，你和你周围的人都不买国产奶粉。这可真是一个很奇怪的逻辑。消费者对没有掺二聚的奶业企业有信心，可就是不买这些企业的产品。考虑到你对这些企业还没有接管出现过问题的奶源就已经用行动表示了不信任，很难相信在这些企业接管了一部分出过问题的奶源后，反而会加强你对这些企业的信心。这种一边鼓吹拆散已经形成规模的奶业，一边强调不买国产奶粉的做法很可以解释为在为外国奶业做托。不过你的水平和能力真的达不到为外国这些奶业集团服务的标准，说到底，不过是愤怒的表达者之一。

至于说解决问题，可以明确的只有一点，在这里打打嘴炮解决不了任何问题。山某的原意也不过是说一说在这个事件里所表现出来的危机公关能力的不足和媒体起到的负面作用。尽管你不同意，你争论的帖子恰恰从反面证实了山某的看法，真正的问题其实奶业的从业从员已经解决了大部分。首先是重订了奶业标准，三聚成为检测的目标。二是完善了奶源的管理。这是保证奶质的两条基本措施。三是拿出了二十个亿来赔偿受害者。是不是能够重建消费者对中国奶业的信心还是要看具体措施的实施。这就不是在BBS上写点什么能有用的。

青岛这边，我师兄两口子，刚刚租的房子，60平米两室无厅，在中山路天主教堂附近，装修什么的都不错，一个月才一千二三。如果是普通打工的小两口，租这么个房子，一个月月薪两人加起来五千，我觉得负担率还行。能过得去。

我现在租的地段比较好，是学区房。我主要是因为离单位比较近。套内五十坪多一点，一个月一千七。我师兄他们劝我将来换个地方。

关于打工这种事情，有的人是没得选择。我高中的同学，在河南省内上的，基本大把大把的时间去打工了。无他，穷。没得选择。我一个表哥上学期间学费都是欠着，住五六十人的大寝室，睡通铺，生活费都是自己打工，有时候是真的没有饭吃。所以他尽可能打工，养活自己。而我大学的一个同学，也有没有饭吃的时候。他打工就少一些。不过这位现在也过上了好日子，做上了科研。学校未必都要培养做科研的人才是吧。我表哥现在在一家大型食品企业做销售代理，就是当年打工积攒的工作能力。

其实我对这种问题看的很开。当年我们学校创立校内的打工渠道，大部分其实还是助工助研。像我当年是去实验室做实验刷瓶子啥的。这样其实也是在培养专业能力，因为生物方面对实验操作要求还是有一些的。

我还是坚持认为，专业课的能力不是看考试成绩。这个可能跟你们数学不太一样？我们这个方向的考分最高的几个人后来搞科研的很少，而且跟他们接触的时候我觉得他们的创新性也一般。他们，中间有些人已经对科学心生厌倦，因为他们过去努力学习不过是为了给自己换来待遇，过于强调学习导致对学习产生厌倦。相反，我们这些排名比较靠后的人，反而对科学还充满新奇感。因为我们并不是为了成绩而去学，而是为了自己关注的问题。

目前国内确实没有几个人能沉下心积累的。所以我想试试，看到一位老的准院士，我很感动，也很受启发。因此我会试试看。虽然可能撞墙的。
