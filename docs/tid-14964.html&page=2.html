即便时代变化快，我们不是美国，哪怕有了肌肉，也不会充当世界警察，即便为了保护海外国家利益，我们也不 ...

央视的《长城》纪录片，我的深刻印象是，中国不会当世界警察，更不会追求霸权，只想做成一件事：普天之下，莫非王土，率土之滨，莫非王臣。可怕 的地方还在于，汉人还像蚂蚁一样，前赴后继，子子孙孙无穷尽，缠斗千年。历史上，既有军事征服+政治整合的方式，也有好死赖活地”民间交流“，反正就是打死也不从这些好地方退出去。

所以，谁再说中国人是爱好和平、善良的，我也学着金灿荣问一句：和平？那960万从哪里来的？

所以，农业时代，汉民族为这那点适宜的农地，硬是与游牧民族死扛了2000多年，直到将草原上的游牧民族一个一个地扛死，赶到沙漠的尽头。这种极富侵略性与扩张欲望的民族性，深植于汉人的血液之中，在工业化与信息化时代会加倍发酵。工业化和信息化时代的根本在于对资源的控制和资本的利用。只要有资源和资本的地方就不会缺少中国人的身影。中国人不会缺席全球任何一个地方的资源扩张与争夺。但凡有适宜工业化与信息化时代的资源与资本，都会引发中国人的欲望并尽力扩张。
