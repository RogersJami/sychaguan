    与单县相邻的几个县，先后出现多位开国皇帝，并且全是狠角色。单县东边的丰县，是原沛县的丰邑，出了汉高祖刘邦。东南边的萧县，城东的绥舆里出了南朝宋武皇帝刘裕，篡了司马家的江山，建立了刘宋王朝。南边的砀山更是出了荒淫无道的后梁朱温，废了唐废帝，终结了唐朝200多年的基业。再往南一点的亳州出了一代枭雄魏武帝曹操。再稍微远一点，安徽的凤阳，出了大明太祖朱元璋；河南项城出了83天的洪宪皇帝袁世凯。难怪乾隆皇帝最后一次南巡时悻悻地留下“穷山恶水，泼妇刁民”的朱批，对这一地区的治理也束手无策。

我是农村出来的，十几年了，几乎与农村隔绝了，父母也来上海好多年了，跟家里人联系不多，除了几个实在亲戚。我觉得村里的情况既不那么好，也没那么差，大家宽容些的好。
在一个村子里，资源有限，所以很多时候会为了一些鸡毛蒜皮的小事去争，这也好理解，但如果哪家出了大事，大家还是会出手帮忙的，这也是事实。
对于从农村走出来的，我的做法是，如果老家找上门来需要帮忙，那是尽量要帮的，毕竟你的情况比他的情况好很多，资源也很多。回家的时候，也没必要显摆，什么衣锦还乡之类的，不要对农民要求过高，即使是城里人又怎么样呢？难道不会犯红眼病吗？这是人的天性，不要去要求农民做圣人。
说一个亲身经历的例子，一个村里的亲戚的小孩，前几年读了一个名办的大专，想要在北京找个工作，我工作在上海，在北京只有一个同学，就请那个同学帮忙，推荐到一个他单位的乙方从事技术工作，起薪2500块，这点钱只够这个小孩在北京生活，我跟他说，我也只能帮到这里了，但这个技术工作如果他安心做个三四年，薪水会翻几倍的，这个小孩坚持下来了，今年春节，电话里说他跳槽了，工资涨到了8000。在这个事情里，我觉得最应该感谢的，是我的那个同学，完全是看在跟我同窗（而且是在职）的面子上帮忙。我只是出差的时候，请他吃了顿饭，非常小的礼品几乎算是空手请人办事的。我不在乎个人花个几千块钱，这个小孩如果记住我帮过他，我就不会失望，如果他不提，我也不会记恨，毕竟我做了我该做的，别人怎么做，我没法控制。
