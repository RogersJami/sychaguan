       硬盘还不是最核心的部件，最核心的东西叫保密模块。核心的东西和含重要数据的部件（如硬盘）在飞行的过程中就开始按照紧急处理程序拿榔头乒乒乓乓的砸了。那时候根本不用管热拔插冷拔插的问题，拔下来砸就是了，砸得越碎越好。至于其他的电子设备（如各种板卡）就用斧子劈，劈掉多少算多少。
       国家财产能这么随意破坏吗？话说在平时这么搞是要上军事法庭的，但是在紧急情况下，经过请示（事前或事后）是可以这么做的，那是通过破坏较少的国家财产来保护更大的国家利益的一种非常手段。
