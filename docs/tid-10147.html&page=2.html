该来的不由人。如果不能尽快找到合适的产业升级方向会死的很快。      崩盘会随金融链条不同程度的进入各个领域而不仅仅是房地产，各个行业包括在世界上取得一定技术地位的行业都会因为没有充足的资金而不得不放弃产品研发，并逐步被市场淘汰。这个过程时间越长，死的就越难看。     做为资深屁民+大外行分析一下，崩盘不可能是同一时间发生的。就像是自然灾害，有个爆发扩张蔓延的过程。既然有时间和空间上的差别，就存在分离高风险行业，允许高风险资本逐步崩盘，保护低风险资本不受波及的可能。     首先风险最高的行业是房地产，需要先把房地产行业隔离出来。成立一个只有资金没有地皮的房地产公司（就叫A公司吧），对一线城区人口分布进行统计，以人口数量为排序条件进行排序，然后选出占总人口60%以上的楼盘。收集交易价格对其崩盘风险不断评估。崩盘开始的时候，这些大型社区必然集中了城市中最多的“负翁”。而依照银行纪录对这些负翁的银行账户进行监督。同时对债务做包括社会救济等在内的一揽子还款套餐。目的是迟滞房产崩盘在社区治安和经济方面造成的雪崩效应。      地方政府土地出的问题就用土地还，欠多少钱用土地还多少。国家以土地作价增发货币拖长还款期，并适当通胀。还出的土地成立地级行政区区划税收直接归国家，同时推进城镇化，地皮供给出负面清单，原则上“法不禁止即可为”，目的也是拖时间。      这样银行崩盘的时间可以被拖延出来。在这期间，等于国家以土地为准备金向社会增发了信用。信用不必非要以货币形式发放，能承担一般等价物的东西都可以成为信用，比如土地使用权。它在工业产业链上同样可以起到一般等价物的作用。      由于有土地之锚的存在，对于一直大规模通胀可以起到积极效果。同时地级行政区土地流转增加了资本在社会上流动的速度和规模，有利于抑制通缩。      最后，无论如何新的产业集群必须形成，在国际上新的比较优势必须形成，与之配套的资源利用和消费模式必须形成。否则还是会死的很难看。

理论上去泡沫过程中一国政府需要同时利用至少四种手段：

　　1. 去债务化 包括债务去杠杆（包括债务注销）和债务违约，只有这样才能去除过于沉重的债务，改变旧的经济结构，重新将资本投入真正需要的地方，通过提高劳动生产率，最终渡过经济危机；
　　这一条是以下所在动作的前提，在经济危机中泰国、拉美都拒绝接受这一点，并企图借短期外债维持泡沫，结果他们永远呆在陷阱里。
　　可以说很傻很天真吗？
　　韩国国民即使捐黄金也要还债，最终渡过危机并且重新站起来，虽然过程中卖身给美国，但是要别人买你总要有值得购买之处不是？
　　我们可以看到中国至少央行一直在逼迫银行体系去杠杆，但是尚福林手下的银监会，可以说智小而谋大吗？
　　至于注销负债现在还不到时候，一般用在去泡沫化未期。

　　2. 政府增加投资，缓冲失业
　　方法1）发国债 如果国债发行对象不是央行，相当于回收货币，不会造成无锚印钞。
　　2）社会借贷 这一招已经被瘟神用到极限，成为资产泡沫化的根源。瘟神果然不是盖的。

　　3. 印钞，但理论上仅以抵销因为去泡沫造成的通缩为目标，是补充政府扩大开支不足部分。但前提是要把债务市场的资金赶回储蓄存款。
　　目前超日债、中体厦门信托担保、天威债和中森债，还不够风声鹤唳吗？信托已经发不出来了；但是银行还在疯狂，是时候处理一些银行信托，否则此风常长，倒逼银行发钞（SLO，SLF，逆回购），呵呵他们到是打着主意反正最后还是央行兜底。凭什么要我们这些纳税人给这些疯子买单？

　　4. 其他一些动作，例如压缩行政开支，压缩无法产生经济效益的政府形象工程等，前者已经在做，后者笔者相信很快也会发生。

　　不过作为非货币发行国，还要防止贸易融资拆借大量美元短期借款，拉美、东南亚前车之鉴。

这里我想讨论另外一个问题，在去泡沫化过程中最重要的约束条件是什么？

　　肯定不是资产，而是流动性，没有流动性的支持，资产没有价值。

　　因此去泡沫最重要的工作就是防止泡沫一起暴掉，流动性无法支持。

　　1）在去泡沫的早期主要工具包括债务违约，债务信用延期（我就不给了，你要兑付就清算，延期还能打折支付，以税收为担保的债务可以这么干），资产变现（清算），同时发钞补充流动性。
　　当前处于这一阶段，但以我们的观察，某些吃惯了发钞红利的人显然在把发钞、继续加大杠杆做为主要债务解决方式。而且两种势力的博弈还在继续。悲哀个啊，都这时候这了，这个国家的当权者居然没有形成一致标目。

　　2）去泡沫的中晚期，除了债务信用延期，还有债务主体承接（如美国两房），清理三角债（这个中国在行），债务违约，债务注销（针对个人破产），债务打包收购（如四大资产管理公司）、实务抵债（在这个神奇的国度里这种事情天天发生，作者就收到过一箱抵债花旗参茶作为奖金，说不定某童鞋有一天会收到一套房子作为过去几年欠薪补偿）等等，这些其实操作并不难。

　　最难的是如何约束某些手眼通天的家伙放弃加杠杆。

      假如明天世界就陷入中世纪混战，那么哪种“伟光正”能重新号召起大多数人，那么那种“伟光正”和技术的结合就是现在产业升级的方向。否则我们就还要在漫长的“中世纪”后，在人口减少到可以再次触发资源分配模式后再来一遍。本来还是很看好基督教和罗马那对基友的，比我们的先进很多，但问题是可能要转回去走老路。并且现在的人类也不是当初了，玩不好就成了末世纪。我是希望在有生之年见证一个新的中世纪的开端，或者能见证人类自此走向灭亡也是蛮刺激的。
     但我还是抱着希望，希望人类起码这次抓住一个方向，集中力量拼一下。而我个人比较恶趣味，我觉得人类只保留耗能25瓦的那个大脑就够了。一方面面向未来攫取太空资源血肉之躯不合适。另一方面，摆脱肉体束缚才能摆脱资源人口循环的陷阱，否则人类完蛋那是早100年晚100年的事。
