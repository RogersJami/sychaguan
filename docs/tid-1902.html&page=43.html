morningtiger 发表于 2012-5-15 16:47 static/image/common/back.gif
很多人心里还是隐隐的有民族复兴这个梦想的。
如果没有日本侵略，民族主义精神被激发，毛的分田分地主张会象无数农民起义的口号一样淹没在历史长河里。

毛的伟大之处在于，当民族面临生存危机，蒋政府无计可施时，他给了另外一个现实、可行、必行的选择。

毛是一个民族的解放者。平均地权是民族解放的必要而非充分条件。从太平天国、孙中山到毛泽东，都这样主张的，但只有毛才真正做到了。但平均地权再重要，假如没有民族解放的成分，毛也不过是一个土寇，不会成为蒋介石的分庭抗礼的对手。

中国历史上，农民起义成功建立可持续的中央集权的，总共只有三个：刘邦，朱元璋，毛泽东。刘邦时代，中华民族正在融合为一个大一统民族。朱元璋的口号：“驱逐鞑虏，恢复中华，陈纲立纪，救济斯民”--典型的民族革命。毛泽东也属于民族解放者。

为什么这样成功案例这么少？除非赢得一批知识分子支持，否则不可能在中国这样一个庞大国家建立巩固有效的中央集权。但知识分子多数出身有钱人家，仅凭打土豪分田地的口号，是吸引不了他们的。革命不是请客吃饭，是杀人放火、脑袋挂在裤腰上的勾当。只有让知识分子也感到受压迫受威胁了，他们才会参与革命。而通常只有民族压迫，才能让有钱人家出身的知识分子感到威胁。所以，一般需要民族解放的大背景才能吸引到足够质量、数量的知识分子倾力支持农民起义以建立一个新的中央集权。

太平天国也打土豪、分田地，口头上也有大汉族主义主张，得以蔓延十几省。但莫名其妙地从西方引入了的拜上帝教，妄图以天父天兄之教取代中华民族的父子兄弟孝悌之道，结果不仅得不到知识分子支持，反而几个湖南书生以捍卫孔孟之道的名义彻底击败了它。

抗战是共产党停止土改的时期，但恰恰又是共产党的势力发展最快的时期。如果没有抗战，许多地主家庭出身的知识分子想必不会因为支持土地革命才到延安投奔毛泽东；如果没有报国热情，钱学森、邓稼先之类的书香世家子弟恐怕也不会在新中国草创之时就义无反顾地回来。

到了抗战末期，共产党不仅完全可以做到分庭抗礼，蒋介石即便有美帝支持也必败无疑了。变数仅在于战争打多久而已。

总之，土改是吸引农民支持的一面，但民族解放则是吸引知识分子乃至全民支持的一面。这一面就刻在纪念碑上：“由此上溯至1840年.......”

把这个过程简化为“打土豪、分田地”是对毛泽东思想的一种庸俗化。毛泽东是这个民族的选择，而非仅仅农民的选择。
