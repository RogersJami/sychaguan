因为看到有几个朋友在读缠，觉得有几点要提醒一下：
1. 缠论的内容不仅仅是108课和每日解盘，108课是招式，每日解盘是习题集，其他的系列特别是论语系列，是心法，同样重要，甚至更重要。
2. 单只是108课中就有不少坑，要注意他在谈技术的时候不只在谈技术，讲故事的时候也不只是在讲故事。
3. 没有理论可以让人不劳而获，缠论也不行。不论何种年景在A股市场都能做到每年3-5倍以上收益的钢铁战士绝对有，我知道有人去年的收益远高于这个，但他的勤奋也是很多人比不了的——每天实打实高强度工作8小时以上，把所有的票过一遍，不停总结提高，已经这样坚持了好几年。
4. 缠论跟传统理论并不矛盾。缠论是利刃，传统理论可以看做是刀背；最重要的却是心性，也就是刀把，能把刀舞起来主要靠它。是杀人刀、杀猪刀还是杀鸡刀，甚至是剖腹刀，取决于用刀的人，而非刀的品质。高手用竹刀也可以杀人，废柴拿一把名刀，很可能会死得更快。缠论是唤起了你的贪念、斗志还是佛性，决定了你在市场上的成就。

北京：明确了2013年本市房价控制目标，即全市新建商品住房价格与2012年相比保持稳定；从明日起禁止京籍单身人士购买二套房；严格按个人转让住房所得的20%征收个人所得税，出售五年以上唯一住房免征个税；进一步提高二套房贷首付款比例；新旧政策以买卖合同网签时间为准。

上海：明确2013年度本市新建商品住房价格控制目标；坚决抑制投资投机性购房；增加普通商品住房用地供应；深化完善“四位一体”住房保障体系；加强市场监管，查处违法违规行为；加强房地产市场监测和走势研判。
亮点：无。

重庆：进一步落实对首套房贷的首付比例和贷款利率政策，严格执行第二套住房信贷政策，暂停发放第三套及以上住房贷款。对在主城区个人拥有的独栋商品房、新购的高档住房，以及在重庆无户籍、无企业、无工作个人新购的第二套(含二套)以上普通商品房征房产税；对出售自有住房征收20%个人所得税。
亮点：合理安排土地出让时序和选择出让区域，引导和调节楼面地价不超过当期房价的1/3。
