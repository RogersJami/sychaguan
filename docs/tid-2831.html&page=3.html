实践九号的几种实验：

1. 电推试验。目前DFH系列平台载荷比例低，减重要求迫切，第一个就是要砍掉用于位置保持的燃料，电推尤为重要；还有后续型号要求上电推有些迫不及待
2. 编队飞行试验。类似于德国的TanDEM，两颗卫星可以搞协作模式对地成像，有利于立体观测，军民两用都很重要
3. 星间链路试验。以后北斗二代，天链二号都要上，实践九打头阵理所当然了
4. 关键元器件。现在国外禁运越来约严了，更重要的是把核心元器件寄托在外卖的想法是不对的，搞搞试验是必须的
