【我和它】命题作文--- ---春三月，漫思犬......

1.

三月里的一天，俺正在埋头在杂务堆里的当口，大猪同志电话过来。
俺问，何事？
那边沉吟了下先，才慢吞吞地说，我们单位的老J问，你想不想养小狗？
小狗？俺一时没反应过来。

嗯，他们家的大狗生了一窝小狗，想送你一只。
哦。。。
俺正要搭腔，那边又急忙补充道:
我提醒你，一定要考虑清楚，比如你出差了怎么办?我可替你照顾不了。

哦。。。
俺要说话时，他又说，这样吧，给你十分钟的考虑时间。
说完，没等俺答应，电话就挂了。

2.

于是，俺思前想后地愣了十分钟的神。
他自然晓得，俺是爱狗的。
每每在路上，看到人家牵着大狗神气地走过，俺总会羡慕地追着人家的身影看。

可俺也很明白，这家里除了俺，大猪小猪，爷爷奶奶都是不喜欢狗的。
如果俺真出门了，狗狗咋办？

十分钟很快就过去了。
俺还在浮想联翩的当口，大猪同志的电话又追过来了。
想好没？
没有。

那就算了。
俺忙恳求，再给我半天时间，容俺再考虑考虑。
好吧。听得出电话那头有点勉强。

无奈之下，俺给一个精通周易五行的朋友打电话，问他俺能否养狗？
人家正忙着开会，答曰：等会答复。
俺说好。

3.

午后，朋友发短信，还是别养了，养狗对你不好。
俺谢过朋友，抄起电话给大猪，我想好了，不养了。
对对对，还是算了吧。
听得出那头松了一口气。
俺笑笑，心下暗想，切，既然这么不情愿，干脆别问俺不就没事了？

4.

下午下班时候，走的西门。
远远的就看到在自家楼前，有六七条大小花色不一的狗们在草地上追逐撒欢。
近得前来，赫然发现老J也在，正背对着俺和别的遛狗的聊得正欢。
话题不外是关于爱狗的种种。

我怕他提起白天的事情让俺伤心。
于是，趁他没注意俺，快步走过去。
一边走着，一边拿眼角扫着草地上奔跑的狗狗。
心里想着，到底哪一只是那些小狗的妈妈呢？

5.

于是，一次让俺春心浮动的养狗冲动，就这样磨灭了。
不过还是有点微微的不甘。
俺想，等将来俺退休了，有闲有空的时候，一定要养一条狗。
一定要，一定要。
哼哼，谁也休想拦住俺......
