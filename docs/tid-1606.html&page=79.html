原来还有这么一个渊源，不知汶川大地震时这个系统反应如何？友军您可否做一介绍？
全国调动，行动如军。初期最应急的时候是各地应急队伍赶到后临时分配各处，在确定全国各地对口支援后则按行政区划对应驻扎。各地各级疾控分批次安排梯队轮换。大震之后所有遗体均妥善处理（卫生学意义上），与解放军一起保证了临时供水、饮食、粪便垃圾处理等等的消毒与掩埋、隔离工作。所以汶川没有发生集中的传染病暴发，这在大灾之后是极为罕见的。

现在各类灾害应急都有系统预案，卫生系统也只是其中必备一员，随时应召，相互配合。
