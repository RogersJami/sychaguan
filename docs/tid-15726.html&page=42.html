你怎么知道过度解读了？我还从没有见过这种角度的解读呢。起码这些专业的军事术语知识，除了当兵的和一些 ...

这个还是那句话每个人有自己地世界。曾经有做过舆情分析的人感慨说，每个人对自己不熟悉的东西是本能的抗拒的。后来才慢慢体会，和一些事说他们理解之外的存在是会被反噬的。所以才提过一些不少论坛上的人跟反感的策略。这是历史与人共同选择，而战狼2打开了一种新的选择方式是值得看看有没有希望的。

记得当年在河里提起过几件伊本运作的事，腾讯华为背后有原广州军区的操作。当时还有人很不屑的说他自己特地去差腾讯大股东那是一家在南非的公司。今年公开新闻说，这家在南非的公司有六个主要股东过半是央企比如中信与知名中资银行。圈子里有些话就是点到为止，知道就是知道不知道就是不知道，不知道耶无法求证就搁置。而另一些人，喜欢网络上打听，一定要你说到底，说到底自己不相信就会怀疑你目的。类似的话河里就有不少，最后就是成为公开的集体攻击。这就是我说的，打破被人的认知会被反噬的真实体会。所以那时候就说，不会在说提前量的东西。毕竟不相信的话，网络上动动嘴也就算了。真有人信生活里来找你，还真被坑过的。现在看不只是提前量不能说，就拿正式新闻与公开消息是明牌的事解读下，也是有很大问题的。毕竟破格局的事，带来的震荡不少人的情绪需要宣泄。越往后，时局越超预期，背后涉及的利益就更加激烈与复杂。所以我身边朋友说过你现在和人说这些人家都有选择了，你不是触人家霉头。这几天碰头的朋友，也批评过我和普通人做一些他们生活以外的交流太理想主义了。

不过说到吴京这样的事，不由想起08年前有人对下面的人说，给你个方向，做到行业前三自然有资源配置给你。如今这样的思路与选择逐步变成为以市场为主导的国策。这样的路类似还有一个，李书福。当年吉利蛇吞象吃下沃尔沃，那背后有苗圩赌下所有前途与资源。本周新闻是吉利市值过千亿了。吉利是国内不多专心主业不炒房地产的制造公司之一。这里我想说的是一种类型的资本他们运作的模式与行为逻辑的目的。作为与国同休的资本，起来的是腾讯与华为背后，他们看重的是由此带来的上下游产业链与崛起一个行业方向影响。更重要的是，这样发企业带来的就业率是所有失败的企业加起来的都比不了的。这就业率与社会稳定的权重，多数人看企业几乎不会考虑的。你看看他们考虑的模式，如果有一天你有机会接触到这个层面。一句话，做好自己无改初心就可以了。如果要骗人，就一开始骗骗过所有人包括骗自己能到那个层面也是一种路数，我们喜欢管这个叫气运。我初始说的路子上的对决，实际也就是勿改初衷与一路骗到底两种套路的对决。让市场做选择，本质也就是不管是谁最终都是要争取民心。所以主席最终还是对的，历史是人民创造的。不管什么时代什么人，能走到历史舞台中央的人群他们的最后的选择都是基于人民的选择。顺之者昌，逆之者亡。

还记得我提过的么，当历史形成的潮流成为趋势的时候，毫不犹豫跳进去。这句话结合上面的话，起码后面复杂的格局博弈中你会有所谓的战略定力，这种定力也许对正确的选择没有多大用处，但是做到不是忙中出错还是能有所助益的。
