我的看法是：有什么样的人民，就能产生什么样的政治家。有什么样人民，就决定了这个民族国家的未来。

我评价一个人主要看境界。能力有大小，能力大小是天生的，没办法，所以才会有高低贵贱(这个高低贵贱是我所理解的中国文化理想社会中的高低贵贱，指的是根据能力和境界来安排职位，贵者居上，贱者居下，贵贱两字本无褒贬，纯粹只是社会分工不同，一视同仁，贵者不自以为是，贱者不妄自菲薄)，但是境界可以没有。境界是什么？我认为就是：有自知之明(清楚的知道自己的能力在社会中处于什么水平)，有知人之智(清楚的知道别人的能力在这个社会中处于什么水平)，然后安分守己，人尽其能。

而对于政治家的评价，我个人的观点是：一个国家领袖人物(能力最高者)的境界可以看出这个国家的未来，境界高的政治家是国家的福气，功在当代，利在千秋；境界低的只能叫政客，目光短浅，只能成为历史的垫脚石。

至于是时势造英雄，还是英雄创造历史，我觉得这两个本就是一回事，都是顺应规律的结果，不该强行分开。
