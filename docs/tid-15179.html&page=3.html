人畜无害小白兔 发表于 2016-1-21 14:03 static/image/common/back.gif
我一直都是认为梧桐的可能性更大，因为历来都难有和平回归的先例，港澳是契约，欧洲比较信用，虽然也耍了不 ...

台湾问题的核心不是打不打得下来。而是打下的治理。我就拿三十年前部队的事来讨论。仅供参考。

武装登陆，这不成问题。三十多年前，在师一级的作战室里就有非常详细的台岛军事沙盘。我歼六侦，三头两天在宝岛上走两圈。蒋军一有布置变动，就会这沙盘上标出来。详细到每一个固守碉堡，以及它的战术意义。沙盘演示时，打开电源。每个电珠代表一个火力点，灯光所照射到的地方就是火力覆盖的范围。不同颜色的电珠代表着不同的兵器，等等。那时台军就有一个共识，只要解放军一踏上沙滩，就没有抵抗的必要。台军发展的兵器，都是基于“击敌于彼岸”的（守台要诀：击敌于彼岸，阻敌于中途，摧敌于滩头，歼敌于阵内；点点独立，火力支援，固守待援）。武桐，在那时就不是打不打得下来的问题。

封锁，以台湾的海洋地理特点，极易被封锁。二战时，美军就用潜艇布雷封锁过。当时我军封锁策略是“只出不进”，向岛外转移资本和财产是可以的（美军撤离台军不久后，美资便开始撤离了，后日资填空）。但是，要在物资上向岛内支援是不行的。由于封锁引起内变需要比较长的时间，当时考虑的问题是美军援台样式和动员时间。在现在我国技术兵器突飞猛进（DF26 的战略战役意义非常大），美军要武力援台已经力不从心了。

那么，当时为什么不打？原因有三：一，战争的目的是获得利益和资源，资源包括人力资源；第二，台湾治理，台湾存在四大问题：产业升级换代，毒品及黑社会，青少年问题，省籍矛盾（现在已经发展成台独思潮）；三，同胞血浓于水。其中第二个问题是关键（现在台湾与大陆的矛盾都直接与这四大问题有关）。那时，治理能力是制约我国统一台湾的根本原因。美国非常清楚这一点，所以，美国当时即不主张台湾立马独立，也不许国民党与大陆直接交往。而是把台湾当作与我国交易的筹码。这也是国民党痛恨美国的原因之一。

你的回复中有一个我很赞同的思想亮点，就是以打为后盾，逼台内变。现在的国内国际条件远好于我在役时。国力强盛，装备提升，美国在走下坡路。这些形成促使台湾内变的外部因素。岛内变化已经出现。“时代力量”赢得席位就是一个。时代力量形成于“太阳花运动”，民进党对它也措手不及。想渗透，没有成功。如果我们细究这个运动兴起的原因，就可以发现，它是台湾政治经济矛盾到了局部不可调和后的一个果。学生们既不满意国民党，也不满意民进党。台媒称其为“第三方”。只要这种内变的趋势形成，并发展下去，让台湾政治经济危机全面爆发。这时，大陆不论是武桐还是河桐，都可以在台湾内部找到同盟军，建立统一战线。实现治理好台湾。

“小、确、幸”，主要表现在媒体文化和青少年亚文化之中。就当下台湾这四大矛盾中，主要影响我治理的，倒不是台独了，青少年问题。

目前影响我治理的是前面两个，“产业升级换代”和“毒品及黑社会”。这是目前台湾社会经济矛盾滋生的土壤。前者，台湾如果升级换代不成功，治理的社会经济成本就会非常高。姓蔡的选胜后说软话，是和阿扁当年一样发现接到了台湾经济的“烫山芋”。后者，黑社会正在大规模“白化”，以合法化的形式滋生于整个社会之中，进入议员，政党，公教类，企业，等等。这个问题影响，我们可以看看现在的香港。香港的黑社会“白化”还不是那么严重就已经让港府施政的许多具体措施无法实施（不是指占中这类，占中是政治性的）。

这两个问题相比较，升级换代更重要一些。我国目前一边拉台资入陆，一边在大陆建设晶片，液晶等等台湾半导体支柱产业。釜底抽薪，逼其就范。液晶，已经被南韩打残了，前几年与日本合谋反攻南韩，现在日本却在为美国插刀，已经放慢与台湾的合作步伐，没钱没戏。晶片，台湾不得不进大陆了，不然，等大陆的那几家生产线起来，台湾的投资和预研就打水漂了。我一个位战友退役后就从事这方面的投资管理，他去年说，就是搞14NM量产。而且，我国的异型架构的设计根本不同于它国，解决了“老大难”，性能极好。

一旦台湾产业的升级换代离不开大陆，那么，黑社会问题就相对好解决了。大陆对台湾支柱产业的渗透，或者完全替代时间最长也就十年左右差不多了。我想，台湾回归，祖国统一，我是看得到的。金灿荣说台湾还有十年左右时间可以讨价还价，大概就来源于大陆对台产业升级换代的谋略。
