在我们这里的农活，我几乎想不到我没有做过的。最辛苦的莫过于上山砍柴，挖笋然后挑下山。最不喜欢做的农活就是拔秧·插秧·割稻。凡喜欢的大多是干蛮活比如挑担·翻土。最多一次挑327斤走了有两里路，中间没停歇。

由于自己柔韧性不好，弯腰时间长了膝盖就受不了。以前是种双季，在第一季收割时都在三伏天，在大太阳下，汗流浃背，手背手臂脸上都是被稻叶割的一条一条的红印子，汗水流过既痛又痒，还时不时有小虫子爬到身上。经常割到后半程弯不下腰，单腿跪地割。等割好了就是脱稻谷，在我很小时条件不好，直接拿稻子用力甩在一大木桶脱稻谷（四边形有底，下窄上宽，底部大约1x1 米上边1.2x1.2时间太长连名字也忘了）后来条件后了是脚踩的脱谷机，最后来就是柴油脱谷机。柴油脱谷机省时省事，就是柴油味大，还有就是马力大把稻叶稻芒泥水抛的半天高，脱好了稻谷全身都痒还很脏。现在好了，都是机械效率大增，直接把稻谷运回家晒。

先到这改天有时间在扒。

老弟有点外行了。机械脱粒，打出的麦秸更碎，做烧柴更加容易。麦秸的另一个作用是做草苫子。如果想留麦秸做草苫子，需要用铡刀把麦穗部分割下来，机械脱粒，而麦秸则较完整地留了下来。再后来就是铺在场院上，用小拖拉机拉着铁碾盘(那玩意学名我还真不知道)压。脱粒机太危险了，而且需要人多。

稻草是要留下来的，做草绳、草包。由于稻穗与麦穗不同、脱粒机结构不同，所以稻草可以直接保留下来。

水稻脱粒机的滚筒裸漏在外，加之稻穗不像麦穗那样所有的粒都长在一块，所以人可以站在机器后，手抓稻草捆，一捆一捆地脱，稻草可以完整地留下来。

北方干活，最累的是给玉米锄草。处暑后，玉米比人高，太阳一晒，地里的湿气蒸、玉米叶子拉(比水稻厉害多了)，基本难以忍受。

种水稻最怕蚂蝗。说起来，南方农民比北方农民辛苦。南方人(尤其是江浙)英才也多，这个我是心服口服的。
