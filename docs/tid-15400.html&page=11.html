我觉得游牧民族的文明应该算是远古的原始狩猎文明的一种，都没有进化到奴隶制更别提封建制了。看一个文明 ...

      游牧民族出现比农耕民族要晚不少。类似于地中海文明，游牧是一种商业文明。他们的生存之道是控制亚欧大陆的重要陆上贸易通道。从这个角度上来说，游牧文明也是一种先进文明。但是热兵器的兴起，尤其是步炮协同战术的出现，对任何骑兵主战的作战模式都形成跨代优势。游牧民族式微（其实远程兵器一向能造成骑兵的高战损）。工业革命把热兵器的作战成本降到了可以大规模装备部队，这一时期国家的战争能力就是工业人口，工业人口既战争人口。所谓民主制度这种脱胎于军伍大会的政治制度复兴就很顺理成章了。
      所以从历史角度看一国的政治体制由其军队作战模式决定。
      目前军事发展的趋势是，在信息化支援下地面以合成营为单位，摒弃了大兵团正面对决的作战模式。装甲突击力量得到更丰富的火力支援。空地一体是必然趋势。这就要求自动化工厂和信息化工厂的出现，能够根据战争形势的发展和特点迅速调整生产出各种改型装备武器弹药。满足未来快节奏高消耗战争的需求。
      作战模式改变深刻影响着政治体制。以合成营、空地一体等为主要作战模式的国家会表现出一种殊途同归式的政治运行模式，阶层上两级分化，自动化工厂和军队形成的军产复合体将占据食物链的最上层。中间阶层快速消失，取而代之的是广大的下层为整个体制提供各种社会产品和服务。这些提供产品和服务集中的地区就是城市群。
