浪潮CEO：数据交易要保证“可用不可见”

孙丕恕表示，在信息化平台的基础上，所有的交互、交易和行为，都会产生相应的数据。数据的可传输、可交流、可编辑、可存储和可统一标准的特性，将使其成为打破行业壁垒的融合介质。
