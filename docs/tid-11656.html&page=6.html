曾经想写一点有关南海防空识别区设立的文字，因为最近这个话题比较热门。仔细考虑一下，发现要写这个 ...
对九段线的三种理解：

国界线

即海上疆域线，但从国际惯例来说，国界线只存在于陆地上，而且有具体的标识，有强烈的排他性主权权属，但九段线在实际中却没有起到疆域线的作用，中国从未将断续线内的水域作为国家领土看待。从1947年该断续线公布以来，外国船只都能不断地、自由地航行在线内水域，中国从未对外国船只通过线内水域提出过抗议。因此，不能把九段线作为国界线来看待。

历史性水域

这种看法最初由台湾于1989年提出来，“历史性水域”必须具备：有效管辖、长时间管辖和其他国家的看法这三种因素，但一般被认为是具有“内水”性质的水域，具有排他性，其他国家的船只不能随便在该水域航行，这与南海自由航行的现实不符合，因此也不能成立。而且有一个中国自己的反例，在与越南的海洋边界谈判中，我国曾拒绝承认北部湾是越南的历史性水域。北部湾不可以作为历史性水域，占整个南海面积80%左右的九段线内的水域又怎么可能是历史性水域呢？这种说法也不成立。

岛屿归属线

表示线内的南海诸岛属于中国领土。九段线沿着南海诸岛的外缘岛礁把整个南海诸岛环绕起来，这种方法在19世纪末和20世纪初曾在国际间广泛使用。1867年美国和俄罗斯的阿拉斯加划界、1879年英国准许昆士兰兼并托里斯海峡群岛、1899年英国和德国瓜分所罗门群岛等等。这些划定的仅是界限内的岛屿部分，而没有包括全部水域。
-------------------------------------------------------------------------------------------------------------------

我国长期以来发布的有关声明及各种海洋法规也只针对岛屿的归属权。1958年的领海声明、1992年的领海及毗连区法等，都一再重申中国对东沙群岛、中沙群岛、西沙群岛、南沙群岛及其附近海域拥有无可争辩的主权。这些声明显然指的是九段线内岛屿的归属，而不是线内的全部水域。

我国1996年的《领海基线声明》宣布了大陆领海的部分基线和西沙群岛的领海基线，其中西沙群岛领海基线由28个基点环绕组成。如果主张南海全部海域的主权，就没必要划定这些领海基线。

所以目前的主张并不模糊：

1，九段线之内的岛礁主权在我，并且依据《联合国海洋法公约》享有的各种权利，也就是以岛礁为圆心，划定领海、毗连区和经济专属区，在南海上圈定一个一个飞地。

2，飞地之外的海域是公海。

当然，这片公海是绝不是大公无私，而是TG的势力范围。
