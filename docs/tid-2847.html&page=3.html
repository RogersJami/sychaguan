1、保持了35%以上的投资增长，并且是所有产业中唯一一个远高于全国固定资产投资增速的行业。

2、增加了土地购置的总量，土地的购置面积全年预计约4万公顷左右，约可增长30%以上。

5、虽然全年公开资本市场的融资减少了，但全年开发企业的到位资金首次突破了6万亿元，远远大于已完成的4万多亿元的投资额，保留着充分的发展余地。

稍微多看点历史就会发现，资本是压得住的，而且曾被压得很惨。这个与中国老话所谓“天下大势分久必合合久必分”倒是很相似的运动规律。
欲望是有穷尽的，现代经济学理论中讲述的很多规律正是在说这一点。事实上，欲望既是无穷的，又是有限的。辩证的理解这一点，是理解许多西方经济学理论中的东西的前提，很多人容易只记得前者，这给他们理解相关经济规律带来了很多困难。
