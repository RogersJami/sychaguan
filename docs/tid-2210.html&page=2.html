newyorker 发表于 2012-5-27 21:38:43

      忙总威武，一上来就介绍丰泽园。

     北京市丰泽园饭店以正宗鲁菜享誉中外，是久负盛名的经营山东风味的老字号，1930年代，北京“八大楼”之一新丰楼的名厨、名堂奕学堂、陈焕章等20多位师傅，在同德银号经理姚泽生出资5000元扶植下，开办了丰泽园饭庄。当时，丰泽园在珠市口的店址是四进大院，青堂瓦舍，整齐宽敞。饭庄一开业就以上层人士为服务对象，一度雄居京城各大饭店之首。

      鲁菜是北方菜的代表，分为三大派系，济南，胶东和孔府菜。烟台福山为胶东菜发源地。以烹制各种海鲜而驰名，明末清初被引进入北京。后来发展出以烟台福山为代表的“本帮胶东菜”及以青岛为代表的“改良胶东菜”。 济南历城为济南菜发源地。擅长爆、烧、炸、炒，口味偏重。孔府菜为官府菜，以用料考究、制作精细、自成一格和风味独特而闻名天下。

     您介绍的八大金刚及其他菜，我分了一下类：
济南菜：
清汤燕菜，锅塌鲍鱼，糟熘鱼片，酱爆鸡丁, 油爆双脆
福山菜：
烩乌鱼蛋，葱烧海参、沙锅鱼翅

      广东仁者介绍的山东老家，以济南菜为主，在粤菜家乡，不会用福山菜抢生意。

      哪位能给介绍一下孔府菜？
