你这是个综述，我瞅了下，有个最大的问题你可能没有注意到，他一直是初乳对自己种类的哺乳动物的效果，比如牛对牛猪对猪。但是没有说对其他种类的效果。不是同一物种，很多东西不好说。即使蛋白质是同源的，即使被人体真的吸收进去了，还有个问题是在人的免疫系统里它能不能很好工作。这个很难说的。有时候可能就一个氨基酸的差别都会被免疫系统攻击或者无法被充分活化。蛋白质修饰和显示功能是一个很复杂的系统，并不是说一个序列合适的蛋白质就一定有功能，这个跟它的三维折叠结构有关系。我不是做这方面的说的不是很深，但是基本情况是这样。有的蛋白需要进去身体后受到其他蛋白的修饰作用，比如剪切或者剪接，或者是加一段之类的，才能拥有活性。所以当年中国人合成牛胰岛素，着重点在是有活性的晶体胰岛素。如果你只是合成出来一个肽链，很可能根本就没有活性。

另外还是那个问题，就是牛初乳作为一个复合成分的物质，它被吃下去消化的时候会不会被蛋白酶给剪切碎了。我们熟悉的蛋白酶有忙叔提到的菠萝蛋白酶，你可以试着下次将变性的蛋白质，比如豆腐、猪血、煮熟的鸡蛋、煮熟的豆浆和菠萝放一起看看效果，再将生的牛肉等和菠萝放一起观察下效果。牛的胃里是草，是一个发酵环境，和人的胃里的酸性环境是不同的，我不知道牛初乳的蛋白质活性成分会不会变性；然后再到肠道，受到一系列蛋白酶脂肪酶的影响，具体的后果就更不好说了。当然，也可能有幸存的成分。我看了你提供的论文的摘要，基本是说生长因子和免疫因子的，这些的成分一般都是蛋白质。如果你要是说一般的那些营养成分，可能初乳比其他的营养物质含量多点？这个我就不太清楚了。

我对初乳的效果持保留态度。因为我觉得不同种类的动物的同一个蛋白即使同源，也可能差异比较大。我们做的甲壳动物，细胞色素氧化酶，600bp的一个片段，不同生物差异就能达到很大呢，何况是人和牛这么差距远的生物。同工酶和同源的酶一直是用来做亲缘关系和进化关系的工具，直到现在测序技术较大发展，用pcr加上测序更好做后，才没有什么人去做了。当然，以后蛋白质组学还是发展方向。
