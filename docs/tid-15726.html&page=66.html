资本是过去的劳动时间积累，有折扣
====================
有时间价值，俗称“利息”

这是一个矛盾，
以物质为主体、甚至以信息为主体的资本积累实际都是在“折旧”的，
即使用价值递减，
但是以货币复利系统则是“扩张”的“利息”的，
交换价值递增。

在我看来，
这才是金融资本主义体系的资本主义根本矛盾的表现形式。

马克思的论述即：
使用价值和交换价值不匹配即资本主义基本矛盾。
