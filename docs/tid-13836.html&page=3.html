比雷埃夫斯恐怕会签署扩权协议

      比雷埃夫斯签署驻军扩权协议的可能性是完全存在的，包括欧洲的默许，都有可能。但是，即使这个协议到手，在“中亚--地中海”或“印度洋--红海--地中海”两个线路之一没有可靠打通以前，比港就是刷个存在感，不能期望发挥战略性的作用。相对于海上航线，军事走中亚通道的难度更大，除非以上合模式，否则俄罗斯一关就过不去。所以，如果以后经中亚通道参与中东方向竞合，那么就是与俄罗斯分享成果的模式，在这种模式下，比港对于我自身的战略作用不是特别明显，受俄罗斯因素制约。如果整合埃及，那么可能就是单独作战模式，海路模式下才能发挥比港的战略作用，但是这个模式的风险极大。

      通观目前中美关系整个发展态势，我认为没有那么乐观。短期看大的冲突不会发生了，但是长期竞合的斗争实质没有根本性改变。我并不认为美国默许退出东亚竞争，专注于欧洲利益。那不是美国的个性，于他也不是目前战略态势下的明智选择。所以，我的结论，中美中等强度的竞合还将持续很长一段时间。于我们来讲，化解掀桌子的风险，可能就已经达到短期的战略目的了。今后很长一段时间，大家该干嘛还是干嘛。
