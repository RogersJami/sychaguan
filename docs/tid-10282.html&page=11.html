明道 发表于 2014-6-11 17:09:44

很显然，目前的经济总量远远大于20年前了，因此，大熊体现为温水煮青蛙。下跌更加是有抵抗性的反复地，缓慢地表现。不断挣扎，不断新低。挣扎在于甜水喝了那么久，谁都舍不得放弃。好日子过久了，谁有还记得坏日子呢？
       四五线城市，其实已经基本无药可救了，因为经济总量影响有限，所以我们感觉不大。典型的灾区其实已经有了，不过我们总觉得是个案。没有个案，哪有普遍？鄂尔多斯，温州，营口就是典型中的典型。
       鄂尔多斯在08年的时候就应该死去。08年鄂尔多斯已经是各类的供应量是远远供大于求，销售非常缓慢，甚至停滞。主管金融的一个领导曾经让我上去看看，希望我给他们地产界的朋友说说如何解决困难。我给的意见就是，坚决降价，早走早好。当时的鄂尔多斯的CDB办公楼的供应量，我目测已经超过广州的珠江新城。但是，领导和接触到几乎所有的地产商都不希望依靠降价，在他们心中，就不应该有降价这样的字眼。幸运的是，温的8万亿在半年出台了，理论上来说，他们都得救了。同期，星河湾进去了，简直就是莫名其妙地进去了。据说，星河湾这个项目的亏损高达100亿！星河湾10年来爆赚的利润，一夜之间消失殆尽，元气大伤。其神话人物梁上燕负有不可推卸的责任，只能黯然离去。鄂尔多斯的大起大落，是告诉大家，钱不是万能的，没有需求，一切白搭！
       与鄂尔多斯类似的温州，同样已经目不惨睹了。温州的崩溃在于温商的傲慢。记得我04年去温州考察，那时温州的楼价远高于广州，我很困惑。当地的地产商给我的答案是，我们温州人有钱，有钱就买楼，而且必须在温州买楼，家乡买楼才有面子啊！好吧，温州不过是温州人的温州，顶破天有多少人呢？没有持续的需求，这次的崩盘恐怕再过十年也无法恢复。

明道 发表于 2014-6-11 17:10:21

       营口，几乎所有的地产项目全部停建了。
       本次熊市，在我看来，三四五线城市的崩盘才是真正可怕的。量太大了，而且现在，未来都不可能有什么消化。可以预计的是，在当地政府不断挣扎之后，留下的将是一片狼藉。真有机会见到祖国大地，处处都是烂尾楼。最近的招数无非就是强迫公务员买楼，下一步无非就是强迫农民洗脚上田。他们如果还是不知道节俭过日子的话，可以求变的时间已经不多了。当然，我估计挣扎的时间也就够1年吧。悲哀的是，我判断，不会有多少小城市的政府领导会意识这个问题。

明道 发表于 2014-6-11 17:11:03

       万科，依然是龙头大哥。毕竟是见过世面的，上轮的老熊都过来的且壮大了的，自然知道如何过冬。万科在全国各地的全面大幅度降价的标志，其实已经告诉大家。“老子先撤了，你们谁爱玩谁玩”。完全可以相信，本轮熊市过后，万科更加强大！当然，万科所谓的转型，那就是一个“扯”字。那么大的体量，搞那点经营社区，也就是赚个吆喝，赚个时尚。真正的实力还是在于地产开发，包括商业。待到百花凋谢时，万科将收拾旧河山。

明道 发表于 2014-6-11 17:13:34

     绿城的谢幕其实早就在预料中了。绿城不知道什么叫熊市，当然，也来源于绿城诞生和发展在一个牛市的过程。牛市的成功，自然让宋老板不那么准重市场。绿城的成功在于不断制造出精品楼盘，非常精品。大户型，高端再高端。不理睬市场如何，只认为，只要我绿城在，那就是一网打尽高端客。其实，熊市来得时候，走的最积极是高端客。可惜了，一个出精品的绿城！其实，我本人非常喜欢绿城的房子，做的真好！当然，他真敢砸钱，成本真高。
       和绿城有类似的是广州的星河湾。黄老板诞生在广州地产大熊市的刚刚结束。星河湾的精品横空出世，让过往低端的楼盘一下黯然失色。同样，他的成功也在超级精品，只为富人服务。但熊市的时候，突然发现，富人去哪里了？星河湾项目其实不多，但目前，几乎所有的项目全部严重滞销。动辄500-1000多万一套啊。今年星河湾的新盘在沈阳，最小户型200多，好了，需找沈阳的富人游戏已经开始了。拭目以待吧，我希望他成功，但真不容易！

明道 发表于 2014-6-11 17:14:15

     说大鳄，必须要说说恒大。恒大地产无疑要在中国地产史上留下重重一笔，这一笔更多应该是给许老板。如果在古代，许就是曹操一样的枭雄人物。许是一个典型的草根到中国首富，直至亚冠夺冠时，有人喊出民族英雄（算顶峰吗？）。许的成功应该是无法复制的。每一次，他都是ALL IN。每一次，他总可以度过难关。几次生死，都过来的人，依然在ALL IN。这一次，许还能过关吗？恒大的危机在于其海量库存基本都是3，4,，5线城市。等醒悟过来的时候，回头强夺1,2线的地王不计成本。江湖有个可怕的传言，从现金流的角度看，恒大是传销式的现金流模式。一旦规模无法增加的话，那么崩盘就是一夜之间。地产也好，矿泉水也好，都是许的局，外人无法明白。卖足球，说不缺钱，那是一个传说。
      
       广州的地产人，没有一个看得懂许老板，是真不懂。但我真心佩服他，希望本次他再次过关！

明道 发表于 2014-6-11 17:15:36

      今天微信群很热闹。不就是几个城市限购取消了吗？看那群地产大佬们兴奋啊，喊着，限购取消不行，关键是限贷要取消。我都懒得说话，暗想，早呢？兴奋啥啊。
      
       说说1,2线城市。先说定义。
       1线城市，显然，只有北上广深四个。
       2线城市，仅包括直辖市、省会（青海西藏除外），以及单列市和重要人口经济城市，如苏州、无锡以及特殊地位城市，如三亚。
       2线城市分为强2线和弱2线。合计约40个。
       为什么这样划分，因为2线城市都属于区域中心城市，是有影响力和辐射力的。尤其省会，那是各省的政治中心，相当欧洲一个国家的首都啊。指标意义非常重要。

明道 发表于 2014-6-11 17:16:03

         显然，这一波的重灾区一定是3,4,5线城市，很多城市估计那就是尸骨累累，永世不得翻身了。中国人喜欢一窝蜂，地产兴奋时间太长了，连个小县城都开大型美凯龙！多扯啊。万达也呼呼地进军4,5线城市。3,4,5之所以是永世难以翻身的一个重要原因是：互联网的兴起，让传统的递级式的居住中心发生突变。巨型城市的形成以及区域核心城市的竞争。而普通城市的衰退，甚至消亡。恰恰这些普通城市就是我们的3,4,5线城市。
      
       所以，在一定的悲观中，我们还不是那么悲观。在悲观中机会存在。可惜，不是所有的人都可以理解。尤其是过分依赖买地的政府。
       省会城市明显要远远好于本省其他一般城市（不要把单列市，特殊城市拿来和我争论，我说是普遍现象）。在中国，地域情节是存在的，每个省如果允许独立，我估计都要独立。也就是说，每个省都有足够资源和能力发展。人口也罢，资源也罢。因此，省会的力量是足够消灭一定的泡沫的。甚至，再大的泡沫，只要给时间，那就是可以化解。上一轮大熊典型的例子就是海口和北海。一个是省会，一个是普通城市。明显海口在消灭泡沫的能力远远强于北海。
