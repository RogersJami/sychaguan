    大量经适房稀里糊涂就变成了商品房，最后真正给市民的经适房才占1/3 。

    目前已经证实翟振锋的女儿在兰亭名苑小区拥有11套房产，官方称均为商品房。据记者调查，该小区有多栋楼为商品房。但知情人士向记者提供的当初郑州市发展计划委员会《关于“兰亭公社”(经济适用房)可行性研究报告的批复》中，称该小区为经济适用房，“占地面积110亩，建筑面积10.1万平方米，其中经济适用房建筑面积10万平方米，公建面积0.1万平方米。”

    那么为何到了最后，这个小区的大部分经济适用房变成了商品房？郑州市房管局相关负责人告诉记者，该项目是在05年之前建的，这个小区建的有“补差商品房”，因为有的经适房项目政府配套跟不上，为鼓励开发商建设，当时就允许开发商建25%左右的“补差商品房”，2005年9月郑州市《经适房管理办法》出台后就没有了。这样就出现了新的问题：该小区的商品房比例明显远远不止25%。那么其余的经适房又是根据什么文件精神变成商品房的呢？

    郑州市房管局提供的数字显示，2005年至2009年间，开发商上交至郑州市经适房管理中心供市民公开选购的经适房房源，仅占总房源的1/3，其余的都流入“暗渠”。

    现状：2012年4月，郑州市政府意识到经适房的混乱，宣布停建经适房。当年8月，河南省住建厅负责人称2013年全省将全面取消经适房，原因之一是，经适房资格审查比较难，容易滋生腐败，在操作过程中容易有失误。
报道中称，当初以经适房名义建设的房子，经过重重盘剥，最后只有三分之一作为经适房向社会供应，其余的都变成商品房乃至直接的行贿打点房了（其实变更用途，变更性质，变更容积率，如此等等，在房产界本身就非常普遍，以经适房报批，容易通过，而且还能搞优惠条件，然后扯些理由一变更，所有这些就全变成人家的利润了，爽呆了），不知真假。倘若为真，那么贪官污吏奸商系统真是牛逼死了，三分之二为自己，三分之一为朝廷，朝廷已经必死无疑。
