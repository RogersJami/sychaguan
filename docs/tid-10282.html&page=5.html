最爱庞贝红 发表于 2014-4-18 20:06:20

看到大家讨论的热火朝天，上来冒个泡。。。

1.刚需：
永远是相对的，房价上涨预期在的时候，刚需就都冒出来了，房价下跌的时候，刚需就绝迹了。
如果谈刚需，那就是现在还真正没房子的人。以北京自住房申请的数量来看，优先群体属于真正没房的刚需，也就10万人。
所以，疯狂买房的人，大多数还是本着房子升值来的，如果，预期改变，那么大家就都不会买了。

2. 目前的北京的房市，二手房确实在跌，不仅在跌，也确实没人买。而且很多人去买新房，不买二手房，因为二手房的首付款更高。
新房推盘速度也在加快，以前捂盘惜售，现在都开始卖了。

3. 学区房
炒作的成分更多。我身边有能力，有条件的人，都打算让孩子读个基础教育，然后送出国，没人想去花300万买个20平米的学区房的。另外，小学，初中大家更看重的是，孩子上学离家近，所以，并没有那么在乎小学，中学的教育。当然，在离家比较近的前提下找个相对好学校上，这种现象比较普遍。
也许我说的是一部分人的想法，不代表所有人。但是，在日渐开明的现代人当中，我身边的家长更在乎孩子有个健全的童年，而不是望子成龙，很多反而害怕孩子去那种特好的学校，压力太大，影响了性格的形成。
