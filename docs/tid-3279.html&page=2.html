兄台辛苦,
对于你提到的这几点,第一点,嗯,出来的确实有点晚,我认为这个其实对于微软来说是没办法的事情. ...
微软战略总监Craig Mundie那天还说呢，微软在iPod出来前就有mp3产品，在iPad出来之前就有触摸移动产品，在iPhone出来之前就有在做手机。看看这些年微软的产品非常非常多。但是鲜有成功的，为什么呢？你看这些Mundie说的那些传说中的产品哪一个是他们的CEO在新闻发布会上当着全世界的面发布的呢？没有一个。但是看苹果，每一个产品都是乔布斯专门站在那里展示的。我记得windows发布的时候也是比尔盖茨在那里亲自展示的。这说明什么呢？微软不是没能力，而是微软老总没有战略眼光。没有眼光是什么呢？是在竞争对手出来成功产品之前都想不到成功的产品应该是什么样子的。鲍尔默肯定是发现再过两年如果不搞平板，自己的windows都快没人用了，才不得已拿鞭子抽出一个Surface。不过微软也够慢的。内部研发slate研发了至少两三年了才把Surface出出来。

另外，我说的那些细节的意思是平板的兼容和普通桌面兼容的概念完全不同。平板和手机的应用程序的使用方式也和桌面系统很不一样。你可以大概齐搞出兼容来，那么易用性就不会好。反正你可以问问平板的软件公司现阶段有多少愿意写win8程序，有多少愿意写winRT程序的。他们肯定会说，你先把市场占有率搞上去我肯定写。但是平板上没有应用程序怎么把市场占有率搞上去是个问题。

当然，我也只是猜想。另外，不知道这个论坛里有多少人会买Surface，有多少人会买iPad。顺便，其实我偷换问题了，win8的火爆与否和Surface的火爆与否未必是有很强关联性的。
