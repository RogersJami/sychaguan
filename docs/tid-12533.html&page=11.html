我对网上的视频证据一直态度谨慎，因为不知道前因后果，没有其他与之互相互印证的信息，所以不敢下什么结论。我这里说的视频证据，更多说的是提交给法律机关用。毕竟一个警察、甚至一群警察，不能与整个法律机关划等号。

监控的作用太大了，不过自拍视频也可以起到补充作用。看一个破案的小说，一个发生在高速上的案子，监控视频有死角，最后是通过从案发现场经过的私家车上的行车记录仪录下的视频找到线索。不好意思，用小说当证据，不过听起来是不是还挺合理的，{:4_241:}

补充一条新闻：

巴基斯坦瓜达尔港务局主席贾玛律丁表示，瓜达尔港的基础设施建设目前已到收官阶段，本月内正式启用，年内即可全面投入运营，而中国则具有40年的港口运营权。


瓜达尔港将成为中东至中国石油输送管道的起点，把经由阿拉伯海及马六甲海峡长达12000公里的传统路线缩短为2395公里，终点为新疆。不仅如此，根据“中巴经济走廊”联合委员会达成的共识，该港还将具有集自贸区、机场功能等为一体的多功能码头。

瓜达尔港一旦全面运营后，其将改变整个地区的经济和政治生态，不仅会带动贫困落后的俾路支省乃至整个巴基斯坦的经济发展，还将成为阿富汗、乌兹别克斯坦、塔吉克斯坦等中亚内陆国家最近的出海口，担负起这些国家连接斯里兰卡、孟加拉国、阿曼、阿联酋、伊朗和伊拉克等国甚至与中国新疆等西部省份的海运任务，成为地区转载、仓储、运输的海上中转站。

瓜达尔港全面运营，巴基斯坦与伊朗的石油管道对接后，将降低中国对缅甸的石油通道的战略依赖，也将相应降低马六甲海峡的风险，新加坡再玩“战略平衡”吧（当年拿下该港，拖延了中国战略10年）
