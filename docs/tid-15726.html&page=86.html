skywalker007 发表于 2017-10-21 15:35:29

日本制造现在的困境，根子在于其战后设立的谋求对亚洲，尤其是对中国全面领先的目标，随着中国以及其它亚洲对手的发展，已成为其不能承受之重。

比如一个产品，完美的产品按100分算，75分要用50个工时，85分要100个工时，95分要200个工时，而100分的产品可能要500个工时，这种情况很常见，符合边际成本递增收益递减原则。当日本能生产75分的产品，而中国只能生产50分的产品时，日本人自然会有很强的优越感，当中国能生产60分的产品时，日本会感到危机，当中国也能生产75分的产品时，日本要维持他的领先地位，就必须生产85分的产品，而85分的产品需要100个工时，就意味着日本厂商要付出一倍的时间和精力，而工时翻倍意味着售价也翻倍，但是对顾客来说，如果75分的产品是最经济的，那么85分的产品虽然比75分更好，但是一倍的售价在他们看来很不经济，所以他们仍然会选择75分的产品。站在这个角度上讲，我们认为日本产品失去竞争力，不是因为日本产品不好了，而是因为我们自己的产品上来了，而日本厂商为了自己的所谓的“优势”把自己逼到了吃力不讨好的道路上。

当这种情况开始在更多行业出现时，日本业界面临着边际成本递增边际收益递减，已经力不从心，这个时候，日本业界就面临两个选择，要么乖乖认怂，和中国以及亚洲各方协调，把更多资源集中到少数优势更大的行业，以求可持续的发展；要么就偷工减料弄虚作假。对日本人来讲，日本人选择后一条路，是没有什么悬念的。
