看目前这些城市还没有停止吸纳资源，农民工能到哪去呢？

对工薪阶层来说，在一个城市长期生存，就需要有长期的工作或者不断找到新工作。
在一线城市，从收入来说，多数农民工已经很难坚持了，建筑业/装修业因为特定的市场紧缺，最近应该没问题，等大规模基建和房地产高潮过去（个人认为不会超过十年，应该会短很多），也会是一样的结果。也就是说，随着一线城市产业结构的调整，需要不同工作技能的人，而这些工作技能，都是需要时间培训的，从普遍意义来说，大多数农民工会出局，甚至很多基层白领也一样。
从农民工自身出路看，无法或不想在一线城市立足的，厉害的也许可以利用在外打工积累的眼光、资金、经验，回老家做小型创业（小超市、小工厂等），一般的继续回老家城镇打工（陆续有沿海企业搬迁，中西部基建比沿海慢一拍），年老的估计只好回农村了。
资源从农村向城市、从中西部向沿海、从城镇向一线城市的集聚看起来还将继续，在每个层面，对人力资源的基本要求也是水涨船高的，只能根据自己的能力匹配相应的地方或位置了，大家都是被统治阶级，选择本就不多，从某种角度来说，都是消耗品。
