PX项目不是妖魔鬼怪自2007年厦门、2010年大连“PX事件”之后，PX，这种现代化学工业中极为重要的产品，就成为了公众眼中恐怖与邪恶的代名词。在大连，PX被媒体形容为“令人闻风丧胆的终极绝杀项目”，在厦门，PX据说能抵得上1000枚导弹的威力。

实际上，PX在中国已经被重度误解和妖魔化。PX的学名为“1,4—二甲苯”，又叫“对二甲苯”，是一种无色透明的液体，属芳烃类化合物。作为一种化学界公认的低毒物质，PX的急性毒性低于食盐、阿司匹林、咖啡因和尼古丁。此外，世界卫生组织国际癌症研究机构（IARC）和美国政府工业卫生学家会议（ACGIH），均将PX归为“缺乏对人体致癌性证据的物质”。而国内某些学者宣称的“PX有极高致畸率”的说法，也并未得到国际公认。
相反，PX与人们的生活息息相关。
相反，PX却是与人类生活息息相关的：PX的主要用途就是生产PTA，而绝大多数的PTA，又都用来生产聚酯，聚酯被广泛应用于纺织品、饮料食品包装、光盘等产品中，可以说，聚酯工业是人类正在享受的电子时代、信息时代、自动化时代的基础。一个或许令人尴尬的现实是，不少抗议者手中写满诉求的“旗帜”，就是用PX的下游产品——涤纶制作的。

在美国，一年几乎能生产500万吨PX，建立在美国休斯敦的280万吨/年PX装置，距6万人城市的距离仅为1.2公里；英国南安普敦Fawley埃索炼厂为居民所环绕；设立在日本横滨的NPRC炼厂35万吨/年PX装置，甚至仅与居民区相隔一条高速公路。网络上流传的所谓“PX工厂与居民区的安全距离为100公里”的说法，也并不属实。

任何工业生产都会有污染，PX并不例外，但民众对其污染的恐慌主要来源于泄露和爆炸。对于火灾爆炸事故和化学品泄漏事故，预测结果表明，火灾爆炸事故近距离内对建筑物和人员均会造成严重损害，但影响范围大约为100米。统计显示，石化行业构成对环境重大影响的事故概率在1×10-5次/年，即十万年一遇。

而且一旦PX大量泄漏，也已经有了完善的处理原则：构筑围堤或挖坑收容；用泡沫覆盖，抑制蒸发。用防爆泵转移至槽车或专用收集器内，回收或运至废物处理场所处置。迅速将被二甲苯污染的土壤收集起来，转移到安全地带。对污染地带沿地面加强通风，蒸发残液，排除蒸气。迅速筑坝，切断受污染水体的流动。

walknet 发表于 2012-10-30 10:29:51

目前国内大型化工装置，一般都采用国外的设计包，因此从设计上来说，都是考虑了环保问题的。这些隶属于国有企业的大型装置，在建造过程中都是比较正规，环保设备及设施也都进行了相应的建设。所以整体来说，大型工业装置在环保方面还是比较有保障的。当然，也存在两种情况：
1、当装置在不稳定状态下（比如开停车过程中），或者故障状态下，可能存在一些污染。
2、少数单位从经济效益考虑，不投用环保设备。比如某电厂，原料是煤，装置有除硫、除尘装置，但从经济利益考虑（环保装置投用需要消耗能源），晚上将这些装置关闭，造成空气污染。（该例子仅供大家理解，没有对应上述的化工装置）

而小型装置，特别是私人企业的，从设计建造，到生产投用上，安全管理、应急管理等方面都不规范，污染比大型装置更严重。

看了那些反驳PX污染的文章我笑了。

我自己的出身：化工之家，全家都是搞化工出身的，我母亲恰好在镇海石化工作过几年，然后调到外地后在同行业里车间做了十年，研究所又做了十几年直到被老朱下岗回家，老爸也是做化工的，最后十年做的，是安全技术科，其中的职责之一，就是管化工厂的污水排放的负责人。我老婆做PO生产。

PO生产和PX生产工艺过程类似，都是属于石化行业的一种。生产工艺，是炼油厂那边石油裂解，通过管道提供丙烯，然后我们这边化合聚合生产。工艺生产和PX极类似。最终产品是聚醚和环氧丙烷

问过老爸老妈后，我来说说PX毒在哪吧。

真正毒的，其实根本不在于PX生产过程中释放出的尾气废气啥的，真正的问题，而是从原料石油开始，提炼，裂解石油后产生的一系列废弃物

这些才是污染的大头。比如工艺过程中产生出来的废油，废酸，废水——这些东西有的可以卖钱，但卖不掉的时候就是直接倒掉。
废酸的成份主要是稀盐酸，但要命的是里面含有大量的工艺过程中残留下来的笨化合物和其他重金属
废油，废水里这些成份也都差不多。

废油也可以回收，很多时候是被鞋厂买去充当粘合剂的溶剂。但这种溶剂，老妈说这种东西溶剂会害死鞋厂的工人，因为里面含有大量没有处理干净的笨化合物。

但是这几年国家在抓这一块，废油，废酸的销售渠道很成问题，往往卖不掉，结果怎么办——只能倒掉了。


那些不能回收的废弃物，直接排海里了，这才是真正的污染大户。


网上那些反驳的东西明显都是在避重就轻，唬唬不是化工行业实地工作的网友罢了。我老娘老爸做这一块做了一辈子，怎么回事比谁都清楚。

另一个东西可能非化工厂的人根本不知道的。比如石化行业生产过程中，往往离不开使用烧碱和生石灰（应当说是根本离开不开吧）。以我们厂为例，生产PO（也就是聚醚涂料和环丙，这两种产品其实是绑一起的），这东西和PX生产工艺类似，生产过程中废弃物是含有大量氢氧化钙和碳酸钙，氯化钙，这些东西的重量，是以吨来计算的，一年产生个几万吨的废弃物是很正常的事，这些东西扔哪污染哪里的地下水——事实上，这东西才是污染的大项目之一。

别说什么氢氧化钙氯化钙无毒，几万吨废渣堆哪儿，哪儿的地下水就被盐碱化。这些废渣扔哪儿，才是真正要命的。

另外一点，这是很重要的一点。不是本行业的人根本不会知道，但我把这废渣废弃前的工艺过程和你说明一下，你就知道问题在哪了。
再科普一下更吓人的东西。
PO生产的废渣，也就是上面说的钙化合物在抛弃之前，他是什么情况的。

象是浆糊一样的粘稠状物质，液体成份不是水，恰好是PO的化工产品PO。也就是聚醚。

这最后一道工艺，是用离心机把PO和废渣通过类似甩干机的原理分理开来。

而这些废弃的废渣，再怎么甩干，也是含有大量的PO——这是无法提取出来的成份。拿起一团废渣放手上，依然是象甩干机出来的衣服一样，湿湿的

以上流程不是道听途说，是本人六年前在现场亲眼所见，并和工人，技术人员，以及我老婆交谈时得到资料，那些废渣的物理情况，就象是橡皮泥，粘土一样的物质

PX的情况估计也差不多。

然后这些废渣就这么扔了，其成份，大概是氯化钙，氢氧化钙，碳酸钙之类的各种钙化合物以及大量无法回收的PO产品

现在你回答我，这些废渣会不会污染地下水


PX虽然工艺不同，但情况也一样的。 石油化工和烧碱，生石灰绑在一起，PX也跑不了。

这些废渣，废水，废油，才是真正的污染大户，至炼油厂的尾气，烟冲烧废气传过来的臭味啥的，真的算不了什么。
你们问问那些反驳的人，PX生产搞出来的废渣，废水，废油是怎么处理，看他们怎么答，他们敢答吗？

要证据的话：问问福建湄洲湾地区的当地人，他们敢吃当地生产的海砺吗？这里恰好就有一座大型炼油厂和一家PO化工厂。也是半一体化合作的，也在搞一体化，可以视为缩小山寨版的镇海
化工生产，学校的理论研究和工厂的具体生产，中间的距离是巨大的。至于污染起源和产物，想要从几个分子方程式上看出问题，这根本是笑话。只有真正地进基层，进入生产环节的人，才能真正明白污染的位置所在。那些反驳的文章不过是唬唬没在具体化工行业干过的人玩文字游戏罢了。
