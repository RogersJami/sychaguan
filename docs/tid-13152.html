大跌以后怎么办

好久没发帖了，市场上周大跌，大家有所疑虑，出来说说话。
首先，这里可以确定的就是这是一次中期调整。后面应该还有新高。为什么？因为支撑牛市的主要因素都还在，没听说要改变。
既然还是牛市里面，那么就看什么时候见底。下面的支撑有大致三条线，分别对应前面三个低点。特别是第一条线（4432附近），同时也是上证指数的60天线附近。加上短线波动下周重新向上，那么大概率就在这里反弹！如果空方力量太强，继续下跌，那么就在下一条线等着。如果三条线皆破，那么牛市格局打破，政府要想再拉起来那是千难万难的。
不过即使后市大盘还有新高，很多个股可能没法新高了，所以大家要仔细分析，判断自己的票是否泡沫太大，如果是的，逢高出货。
反弹后有几种走法，需要注意的是其中一种双底的走法，如果出这个走法，杀伤力极强！

rafanadal 发表于 2015-6-27 12:29:28

哈，果然对于股市不能幻想。

暴跌先打爆高杠杆的场外配资，然后跌破重要点位再诱发自动交易以及恐惧性杀跌抛盘。这时候就没道理可讲。

不管牛熊分界是4100还是3800, 接下来的平仓盘和杀跌盘会是什么状态难猜，不过证监会说了是市场自身调整--意思是别指望政策？反正“继续抑制涉嫌违反账户实名制规定的场外配资活动；将查处大股东及高管违规减持”倒是正常职能。

几个想法：
1. 经过史无前例的7年长熊之后，20000亿成交额堆出的牛市5000点就是顶说不通。当然有些股票就此走上屠杀一切抄底盘的征程并不奇怪。
2. 06-07的牛市是世纪初以来工业化爆发的迟到反映，或者说此前股市是被非正常的压低，而08金融危机标志着一个全球化模式的结束。
3. 本次牛市面临经济结构巨大调整，在N多不确定性的影响下，就算有人拿07和15年的K线做对比，我也不相信历史会简单重复，这次的牛市或许比习惯了牛短熊长的老股民们想象的更长，但是在各种杠杆、对冲工具的助力下恐怕折腾得也更狠。
4. 就算跌到3500点会如何? 大盘指标股不暴跌股指的下探空间就有限，难道把央企蓝筹再次打折给外资捡地板价？
5. 控制仓位永远是生存法则，一天跌8%算什么？查查90年代股市有过多少暴涨暴跌。中国股市在T+1涨跌幅限制之下已经很仁慈了，港股美股都上演过一个交易日股指下跌1/4甚至1/3，想像一下一个没有涨跌幅限制，可以T+0, 裸卖空的股市吧，天台哪里够用。

我在股市里明白了一件事，人的欲望的增速永远超过股票的涨跌，无论贪婪还是恐惧。
对于信息弱势的散户来说，准确估值做不到，政策变化跟不上，市场的非理性几乎是唯一可靠的获利途经了。
