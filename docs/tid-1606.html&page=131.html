我觉得在核大战条件下，第一第二产业的从业人员和政府科教文化等领域的人员要尽量保护，而平日那些以小资自诩的市井子弟被消灭是一个不算太坏的选择。
所以在和平时期，一个国家要尽量少创造小资等脱离劳动人民阶层的生长环境。既然在和平时期无法快速消灭他们，那我们就尽量不制造他们。
