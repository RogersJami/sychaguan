不是跟您抬杠哈，实在是职业因素，跟甲醛打交道打的太多了，所以只能说几句，不说不舒服。

俺是做浮游动物鉴定的，就是把小虫子放福尔马林溶液里面（3-5%福尔马林，福尔马林是30%多的甲醛），然后放鼻子底下天天对着显微镜闻。俺这行，得白血病的不是没有，但是真不算特别多。大家互相开玩笑说是，有福尔马林天天给我们这些人保鲜，所以得病率反而比较低。俺们室内的甲醛蒸汽浓度多大呢？基本上，不熟悉我们这种工作环境的人是不愿意进屋的，熏得慌。而俺们就基本上要在这样的环境里面工作，而且鼻子正好对着那滩甲醛溶液。。。

事实上甲醛普遍用于水产品饲养消毒环节，在食品工业中用途颇广。就我所知，养殖场常常用甲醛来泼洒养殖池进行消毒，看中的就是它易于挥发的特质。给您个链接吧：

http://wenku.baidu.com/view/29078df9700abb68a982fb0e.html

这个是比较新的，比较老的俺就不贴了，避免被说是材料太老了。

甲醛导致白血病是有的，但是实际上，家具装饰里面最应该强调的并不止是甲醛，更严重的是苯，那个也相当的有毒。事实上苯在家具装饰室内污染方面很可能要超过甲醛的危害，不过现在炒作的不严重而已。俺腹黑一下，很可能炒作甲醛在室内装修中的污染的目的是托市，让那些搞甲醛清理的企业赚钱。而苯很可能相对来说难去除一些，或者检测起来麻烦些所以并没有被炒作。

为何甲醛挥发的时间很长，甚至几年都挥发不完？其实要知道，室内装修的甲醛挥发是从使用的那些复合材料和胶里面出来的，而且很多家具都有封边，因此那些反应有可能进行的很慢，会逐渐挥发，而不是一下子挥发出来。

事实上，如向大白菜泼洒甲醛这种情况，甲醛挥发的是相当的快的。保守的说，在冬季敞开环境下，如果见风的话一天用不完就挥发掉了。在堆成堆储存的过程中可能甲醛还挥发的慢一些，正是在这个过程中可能需要用甲醛来防腐。

俺们做实验用的标本，如果敞开了吹风，大概几天就一点味道都没有了（根据室温不同情况不同）。

baidu百科关于甲醛的介绍：http://baike.baidu.com/view/4913.htm
