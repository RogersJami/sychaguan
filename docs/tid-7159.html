房价的上涨与否

本帖最后由 fhqiplj 于 2013-8-15 13:25 编辑

1>房价上涨的好处，获得好处的人群，获得坏处的人群
   1. 社会总资产价格上涨 已经购房的所有人群 需要购房的人群
   2. 房价上涨对地方政府有极大的好处，因为地方政府可以通过售卖的手段获得现金，然后通过这部分现金进行市政建设来进行下一步的土地出手。这样直接好处就是城市越摊越大，越來越好看；市政设施逐渐完善，方便了市民的生活。
   3. 房价上涨的收益最大者应该是地方官员，通过建设城市的过程，在政治上和经济上都能够获得较多的收益，而且这种收益是0成本的。
   4. 开发商在上涨过程中也可以获得部分收益，但是个人认为这种收益大部分会转让出去，也就是说开发商很多程度上应该是白手套的角色。
   5. 承建商和原材料供应商在06年前基本上都是普通收益，甚至有些前期垫资都拿不回来(当时有很多的民工讨薪案)，在06年后赶上了飞速发展的快车道，目前看收益都是比较高的。
2>房价平涨的好处，获得好处的人群，获得坏处的人群
   1.社会总资产不变， 无                        无
3>房价下跌的好处，获得好处的人群，获得坏处的人群
   1.社会总资产下跌 需要购房的人群    需要售房的人群
