“10·26”茶馆第一次聚会记录

今天参加了茶馆在上海的第一次大型聚会，不知道k歌的现在散了没有？我18点多先走的，不知后事如何？

第一次参加聚会么，虽然已经看过思炎姐姐和Dada的网络头像，但是到场一看，好多人不认识啊（第一次见真人，当然不认识了。。）

还好思炎想得周到，事先准备好了给每个人分一个标有 ID 的扑克牌，我拿到的是红桃Q。根据这个牌，可以把真人和ID对起来了。吃饭的时候，每个人都把这张牌竖在桌面，好象领导们开会，面前放一个一样（可以参考新闻联播的政府会议新闻画面。。）

一共有四个桌，我在的一桌是(应该没漏人吧)：

玩钉耙的悟空
刹那芳华
shen0807
黑暗的骑士
萨满
芒果
爽秋虫
宋兵丙
奶大有容
emiT
sammoy
和我

首先，几位外地赶来的茶友痛陈赶路的艰难，尤以奶大有容、宋兵丙为典型。以地理距离计算，sammoy从帝都远道而来。外地来的还有emiT。住在上海的茶友纷纷表示感叹，有道是：茶馆聚会魅力多，火（车）汽（车）的士全来过。

然后就是海阔天空地聊天，我印象深刻的是宋兵丙茶友，不光带来了由他亲自采摘、他母亲亲自炒的绿茶，而且自暴了一些少年时的密闻，一桌子听了哈哈大笑。（不知道他愿不愿意在网络上公开？）

黑暗的骑士是我们这桌的外交大使，拿着啤酒杯，轮番请思炎、葡萄和西瓜子来我们这桌发表讲话。。玩钉耙的悟空 和思炎姐姐干了一大杯，估计有1/3瓶啤酒。

思炎姐姐真的很用心，还安排了抽奖活动，奖品是两瓶干白和一瓶香水。我们这桌独中香水（爽秋虫）和一瓶白干（玩钉耙的悟空），运气真好啊！！可惜不是我。。。

吃得差不多了，思炎请葡萄高歌了一曲。不能尽兴，茶友们继续奔赴去唱歌（男女比例太悬殊，因为你可以听到两位男茶友在那里对唱我爱你、你爱我，顿时激情四射！）

然后我提前走了，请其他茶友补充后续故事。
