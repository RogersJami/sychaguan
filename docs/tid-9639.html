葱姜龙虾


龙虾在国内算是一种高帅富食材，卖得很贵，一些高档的海鲜酒楼里龙虾赤身售价一千多元一份。而龙虾在澳洲，美国，加拿大则便宜多了，澳洲大龙虾赫赫有名，但据说美国波士顿和加拿大龙虾，最为价廉味美。

在澳洲悉尼时，曾与一帮华人去唐人街吃大龙虾，端上来时，看到那分量十足满满一大盘龙虾，不禁惊艳了一下，盘子大小是西式主菜餐盘尺寸的1.5倍还多。

同桌吃饭的几乎都是初次相识，吃饭是一种社交活动，礼仪姿态就必须端着，美食在前无法大快朵颐，也更别提“食客的意境”。 撤盘时，居然留了一半以上的龙虾，真暴殄天物 。

其实我本身对大龙虾并不感冒，虽然在欧洲不比北美澳洲的便宜，但比国内还是好很多，偶尔也会买来做了吃。但之前只做过西式的芝士蒜蓉烤龙虾，或清蒸蘸煮融熟牛油，或者直接用牛油煎， 味道也可以的，却总无法满足我的中国胃。

某天晚饭时间窗外细雨萌萌，与某人通话，正值家中有龙虾食材，便问及龙虾的中式做法。。。。。我也学会做简单又美味的葱姜龙虾。


葱姜龙虾做法步骤

1.由于我买来的龙虾是已经处理好的，就省去了放尿步骤。 欢迎茶友补充如何放尿，处理龙虾方法。

2. 将料酒、鸡汤（味精）、砂糖、盐、生抽和少许芡粉调成一小碗料汁。

3. 油锅热后，将龙虾放入油锅中炸，虾壳变红后即可捞起。

4. 锅中放少许油，将葱段和姜片放入锅中煸炒出香味 ，再将3的龙虾放入同炒 (几分钟即可)。

5. 倒入步骤2 料汁。焖锅收水，大约5分钟左右，搞定出锅装盘。

wisele 发表于 2014-2-12 19:49:37

还有物尽其用，吃完龙虾留下的壳，可以留起，有空时把龙虾壳用焗炉焗香后，再加上 「大地鱼干」一起熬出一大碗香味浓烈的汤，这就是一碗极品的广东式云吞面汤底，据说有些餐厅用龙虾壳熬龙虾汤。也据说蟹壳一样有这种效果。

大地鱼干的处理是先把鱼干用锅干煎 （不加油），煎成焦黄后把肉拆下来，跟龙虾壳一起熬汤，汤熬好后把虾壳和鱼干隔滤不要。
