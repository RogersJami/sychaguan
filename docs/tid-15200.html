读韩非子

读韩非子 引子，李世民和魏征的关系
魏征犯颜直谏，李世民从善如流，被传颂为主明臣直的典范。这是典型的冠冕堂皇，如果信了，就被骗了。
李世民和魏征关系好，时不时因为进谏，口头表扬外，嘉奖几匹绢，但并非真的那么重视魏征的意见。
魏征的意见分虚实，虚的是普通原则，诸如要节俭，不要穷奢极欲大兴土木折腾老百姓；要尊重法律，不要仅凭个人喜好裁判案件之类，李世民当然肯定，不肯定摆明了就是昏君。实的军政大事，李世民采纳的并不多，仅举2例。
灭了东西突厥之后，面临怎样安置10万西突厥投降人口的问题，百官各抒己见。魏征主张遣返回原地，不可留在唐朝的国境内，防止几年以后人口增长而扩散到内地，成为心腹大患。李世民弃之不用，采纳温彦博的意见，异地安置，迁徙到原先的东突厥和唐朝之间的边境地带，互为仇敌的东西突厥杂处，并分而治之，重新划分东突厥的地域区划，设置4个都督府看管起来。
另一例是分封皇室和勋贵。李渊集团的根基在关中，崛起过程中收编了多地的一时割据势力，大多给个官衔就地当官，鱼龙混杂，州县非常多，一统天下后合并同类项，全国分为10个道。李世民还不放心，往地方掺沙子，提出来封建诸侯，魏征明确反对，养的人太多，只会增加财政负担，根本起不到军事拱卫中央的作用。李世民没有采纳，5年内先后封了21个宗室皇亲和14个功臣出任刺史，功臣们很识相，一个也没有去上任。
除此之外，有一次朝会结束之后，李世民回到后宫大发雷霆，“会须杀此田舍翁”，“魏征每廷辱我”，翻译过来就是，魏征这个乡巴佬，经常在朝廷上当着百官的面提意见，让我难堪，总有一天砍了他的吃饭家伙。
但这些并不妨碍两人关系要好，因为魏征有一个独特的重要价值，他是唐王朝和山东豪杰之间的联系纽带，有利于维持唐王朝的政治稳定，这是李世民的核心价值，也是李世民看重魏征的真正原因。
灭了王世充和窦建德之后，窦建德部下刘黑闼等人在山东叛乱，虽然被剿灭，但唐王朝要巩固，一定要摆平山东豪杰。而魏征出身山东集团，早在李渊势力仅限于关中之时，主动请缨去搞统一战线，出使山东招抚老朋友，把一代名将李世勣拉入李渊的队伍，扩大了李渊集团的地盘。
魏征原先为李建成效力，经常劝李建成早点干掉李世民。玄武门之变血雨腥风，唐王朝中央权力大重组，李世民大变戏法，事变一个月内即派魏征安抚山东集团，消除权力重构对地方势力的震动，同时表明不念旧恶不赶尽杀绝，连魏征这样的李建成死党都能受重用，其他人当然会有个好的归宿。
李世民是杰出的政治家，魏征绝对是政治人精，两人一唱一和，共同营造了主明臣直的佳话，政治利益使然。
历史其实是一部利益分赃史，就看怎么分，手段巧妙+份额恰当=名利双收，手段粗暴+份额不当=国破家亡。
我为什么读韩非子
韩非子是本政治学著作，主张君主独揽利益分配特权，法、术、势兼用，用法选拔人，用术驾驭人，用势统治人，达到稳坐江山、富国强兵、称王称霸的目的。
这是统治者的斗争哲学，也就是老子口中“不可以示人”的“国之利器”。我读韩非子，当然不是为了当统治者搞政治斗争，而是为了知人论世行走江湖——老于世故+返璞归真。
所谓“人为财死，鸟为食亡”，利益当前，绝少六根清净的高洁之士，却不乏利欲熏心之徒。只要人和人之间还存在利益关系，就必然计较利害得失，计较利害得失就必然你争我夺地算计，所以现实世界并非那么纯粹，有美好也有邪恶，有真诚也有诡诈，有热忱也有阴险，有仁爱也有刻薄，有慈善也有暴虐，有给予也有掠夺，有风和日丽也有波诡云谲，有廉直忠正也有贪枉奸邪。
老于世故就是少一点无知的天真和愚蠢的幼稚，分得清真话和假话，识破卑鄙不被骗，不被无偿利用；听得懂言外之意和弦外之音，辨别肮脏不被伤害，不被人卖了还替人数钱。返璞归真就是多一点切实的世道人情，正视和尊重各自的利益，在分享利益的基础上精诚合作，你拿你的蛋糕，我拿我的大饼，独占利益的事绝对不干。
人心从来不是简单的道德问题，更是利益问题。透视人性中的自私自利，打上这支免疫针，金钟罩和降龙掌左右开弓，防守严密，不被卑鄙肮脏吞噬；进攻犀利，平衡各方利益，必要时以卑鄙肮脏对卑鄙肮脏。

《兵以诈立》以及《唯一的规则-孙子的斗争哲学》

有一个感觉，老祖宗敬鬼神，但是从来都是立足自身探寻 ...

老祖宗的知识系统分两大类，一类是研究人的人文科学，一类是研究大自然的自然科学，迷信是古人在研究自然。

人文科学出了儒墨道法等百家，汇集成《老子》《论语》《孙子兵法》等经典论著，自然科学出了巫祝、中医、方技。横跨两大学科，用自然现象解释人文规律出了《周易》。

要解古人的改造和控制事、势，不作专业研究，我建议读历史和部分典籍。先读历史，后读典籍，用历史感受典籍，因为从感性到理性，中间不能少了感受这个阶段。之后回过头再来读历史，有了初步理解之后再读典籍，领会体悟，读出一些个人心得，不至于背死书死背书。

我曾经不分先后次序乱读，回过头来想，读历史应该按照这个次序：

史记 岳麓书社
培养读史兴趣，文字朗朗上口，人物鲜活灵动。

三国志 岳麓书社
摸门槛，读后学会梳理人物关系，参考两本书：《三国前传》 于涛 中华书局；《论曹操》 朱永嘉 上海社会科学出版社

魏晋南北朝史论丛 唐长孺
隋唐史 岑仲勉
20世纪中国史学名著系列，河北教育出版社。
入门槛，历史片段的面，了解朝代更替时官制、军事、民族、文化等各方面的演变路径。

秦始皇传 张分田 人民出版社
苏东坡传 林语堂 百花文艺出版社
张居正大传 朱东润 百花文艺出版社
朱元璋传 吴晗 百花文艺出版社
李鸿章传 梁启超 百花文艺出版社
曾国藩传 萧一山 海南出版社
看门道，面中的点，以点见面，以小见大，设身处地，理解并同情。

资治通鉴 岳麓书社
找门道，特别关注两个历史时间段，大乱到大治和大治到大乱，找出核心因素，参考《韩非子校注》 周勋初修订 凤凰出版社

典籍太多，建议两本集大成者，先《孙子兵法》，参考李零的《兵以诈立》；后《周易》，参考《周易译注》 黄寿祺 上海古籍出版社

最近想到一个问题，老子思想的来源是什么？人生百年，唯有经历并细心体察过生住坏灭的人才可能超越自己、 ...

道家思想来源于兵家，继承和发展了兵家的辩证主义思想雏形，把军事上的辩证主义发展为政治上的辩证主义，提出统治、管理国家的根本原则和方略，并提升到形而上的哲学高度。下一贴细说。

这与老子的背景有关，首先，他是国家图书馆院长和档案馆馆长，而且身处政治变革期，西周的礼仪宗法体系正向春秋的实力体系演变。这两个背景使他有机会全面观察和思考氏族邦国灭亡的历史，《老子》就是这一思考的高度概括。
