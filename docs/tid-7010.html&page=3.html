葡萄 发表于 2013-7-23 06:08 static/image/common/back.gif
关注下票据贴现市场化吧，变数在这里。题眼也在这里。基辛格来不是白来的。现在他们不来以后就没多少蛋糕可 ...

意思是，以前的银行贴现率是跟着再贴现率走（跟着货币政策走），现在跟着市场行情走。也就是说如果再贴现率高，而市场贴现率低的话，其他银行就不会给不到期的票据贴现。那么部分出票银行信用扩张也就走到头了。这时候定向放水就是一轮收割。
