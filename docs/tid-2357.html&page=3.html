俺比较困惑的是那个小女生知错不改。。。她一方面看来对科学事业心向往之，另一方面却没想有认真看到自己和客观上要求的差距。这样的一个心态，其实是非常有害的。我觉得，上硕士就要看清目标，要么就是坚定的上完了就业，要么就是埋头搞科研别管其他的。我身边见到的这种摸不清情况，昏头转向的上下来，适应社会的能力丢了，科研能力也没有培养出来的人不在少数。这种最后的结果就是杯具。

这几年来做实验的小女生见过一大把，其实也算见怪不怪了。我对她们基本不怎么管的，反正不是自己的学生，也不是自己所在单位的学生。去年一个师弟做实验，比较困惑，我背地里跟他聊，提出我的想法。我们这边的学生有什么问题，只要我知道的，知无不言言无不尽。我的资料都随便给他们复印的。但是说真的这种外单位的小孩我是不乐意管的，管的好不好，自己没有什么好处，她或者他有时候还觉得你是给他们穿小鞋，何必。

俺不是名校，甚至算是国产博士里面面上成绩比较差的那群。俺本科毕业的时候之所以对一些问题有所注意，一方面是因为大学的时候曾经有所接触，当年我们某些课程，老师偷懒，因此让自己写小论文来着。其实就是比着书去抄，但是要写成那样的格式，相对也是一种磨练。另一方面是因为俺觉得一些东西，真没有什么好讲的，从cnki里面下一篇硕士论文，照着格式来设置，看别人每一部分写的是什么，自己往里面填就行了。再不懂，google或者baidu一下就出来了。俺第一次写paper的时候也是baidu了一个论文的格式，然后比着套自己的。既然网上能下到类似的，先看看，套路都是一样的。如果连这个模仿的欲望也没有，或者根本就不知道该去哪里找东西来模仿，那我只能说能力有问题，或者个人想法有问题。

我本科毕业的时候就被一个硕士指使给她配试剂，算是正儿八经的实验一点没有做。因为她的实验是免疫蛋白的分离纯化，我也插不上手。最后结束的时候，给我一张ps出来的图（她那个时候那个蛋白还没有分离出来），让我自己写写就行了。我就自己去查教科书、参考书、参考文献，自己忽悠出来一篇。当然是够烂，但是起码格式和结构是正确的。现在想想，虽然正儿八经的实验没有做，还是学到一些东西，比如免疫活性的测定方法，跟着看了下。还有蛋白的纯化方法，等等。现在做东西也是，有时候不出结果，但是还是能总结出来一些收获，俺觉得这就够了。既然是做科学，不可能每次尝试都有结果。那个小女生的问题，俺就觉得，我一训她就不理我了。。。17号交论文到现在也没有给我稿子再让我修改。。。这个抗压能力，也太差了也。

美国的那种社区大学是什么鸟大学？是不是类似技校或者XX职业技术学院

不是，社区大学是两年制，给个associate degree。但是它可以申请转进牛校。举个例子，假如你很穷，付不起斯坦福的学费，可以考虑去斯坦福附近的社区大学念两年书，然后申请转学斯坦福。一旦成功，你在社区大学年念的课程可以抵斯坦福要求修的课程，于是你就省了上万美元的学费。

所以说，社区大学更像一个预备大学，是好大学的有益补充。
