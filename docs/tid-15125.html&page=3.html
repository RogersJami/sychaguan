给政策、给荣誉、给职位 绍兴打造乡贤参事会推进乡村治理
2015年12月30日 18:40:57 来源： 浙江在线 作者： 浙江在线 首席记者/施宇翔
【摘要】 近年来，乡贤成为推动美丽乡村建设不可忽视的力量。昨天下午，绍兴市委政法委副书记马永定、绍兴市越城区委政法委副书记梁成浩，绍兴袍江新区马山镇尚巷村的党支部书记骆国庆走进浙江在线演播室，共话“乡贤”。

　　浙江在线杭州12月29日讯(浙江在线 首席记者/施宇翔 编辑/沈正玺) 乡愁牵动乡情，乡情凝聚乡贤。近年来，乡贤成为推动美丽乡村建设不可忽视的力量。乡贤及其组织在乡村治理中究竟发挥着什么样的作用？也成为当下大家讨论的话题之一。

　　昨天下午，绍兴市委政法委副书记马永定、绍兴市越城区委政法委副书记梁成浩，绍兴袍江新区马山镇尚巷村的党支部书记骆国庆走进浙江在线演播室，共话“乡贤”。

　　乡贤的“前世今生”

　　始于东汉如今是时代的精英

　　绍兴市越城区委政法委副书记梁成浩说，“乡贤”一词最早见于东汉，是国家对有作为的官员，或有崇高威望、为社会做出重大贡献的贤达予以表彰的荣誉称号，是对享有这一称号的人的肯定。

　　古往今来，乡贤的作用悄然发生了变化。

　　绍兴市委政法委副书记马永定认为，如今的“新乡贤”，已经不同于传统时代的以功名身份为核心的乡绅阶层，他们是在各行各业取得成功的人才和时代精英，如农村优秀基层干部、道德模范、身边好人等先进典型，在乡民邻里间威望高、口碑好。

　　近几年来，我省大力推进美丽乡村建设，而乡贤也成为了美丽乡村建设不可忽视的一支力量。

　　2014年8月，绍兴在全市部署开展“坚持发展枫桥经验推进乡村治理现代化”工作，明确提出以“乡贤参事会”为突破口和切入点，来推进乡村治理现代化工作。

　　如何保证乡贤参事会的积极性、长效性？

　　绍兴：给政策、给荣誉、给职位

　　马永定说，乡贤参事会是乡贤的一个自治性的组织，是以参与农村公益事业、化解邻里纠纷、引领先进文化、促进村民自治为主要职责的，具有公益性、服务性、互助性、地域性、非营利性的基层民主协商和村民自治组织。

　　在推进乡贤发展工作过程中，乡贤参事会的作用功不可没，它的设立使乡贤有了自己的一个“家”。

　　乡贤参事会听起来作用很大，可如何保持乡贤参事会参与乡村治理的积极性、长效性呢？这几年，绍兴在这方面也花费了大量心思。

　　“基层有很多社会组织，刚开始的时候轰轰烈烈，但时间一长，就偃旗息鼓了。因此，我们也想了很多办法，主要是希望用制度化来确保长效化。”马永定说，绍兴主要从给政策、给荣誉、给职位三方面下手，作了很多尝试。

　　比如说，在市、县两级层面，制订出台《关于培育和发展乡贤参事会的意见》及《实施细则》，对乡贤及其组织参与农村治理予以明确。比如说，以村歌、村史、公德榜、乡贤祠等形式，把乡贤公德载入史册；再比如说，把乡贤提拔起来，成为村里的干部，来提高他们的积极性。

　　马永定说了一组数据，截至今年12月中旬，绍兴全市共有1209名乡贤担任村干部，其中有25人担任村党组织书记或村委会主任。

　　绍兴6成行政村建立村级乡贤参事会

　　化解矛盾3000余起

　　近年来，绍兴在培育新乡贤及其组织以推动乡村治理的做法，被中央电视台、《光明日报》等媒体宣传报道，也得到了上级领导的肯定。

　　截至2015年12中旬，绍兴全市已建立村级乡贤参事会1381个，占全市行政村总数的60.6%，会员16113名，募集各类资金11631.25万元，发放困难户慰问金754.26万元，提供决策咨询2311条，收集村情民意5895条，化解矛盾纠纷3000余起。

　　马永定在访谈时表示，广大乡贤及其组织积极参与农村公益事业建设、矛盾纠纷化解、社会治理以及“美丽乡村”、“平安绍兴”建设和“五水共治”、“三改一拆”等中心工作，并发挥了重要作用。

　　梁成浩说，乡贤对乡村社会矛盾纠纷的化解起到调和作用，使乡村治理更人性、更低成本。

　　骆国庆则以绍兴袍江新区马山镇尚巷村前后8年的变化作了类比。8年前，村里垃圾填满河道，路上都是烂泥；而现在则是粉墙黛瓦、古色古香的民房，河里的水清澈见底……乡贤及其组织在其中发挥了重要作用。

老兄之所以这么想，其实是还没看到TG今后将会面临的惊涛骇浪。
很多人还是以为TG是会“和平崛起”的，包括我自己也经常会有这个情绪，觉得只要稳扎稳打，中国就能稳步跃升第一，所以犯不着用过于激进的政策。可是如果仔细分析进去，很多事真的就和愿望相违背了。事实上TG这几年的一系列政策都已经透露出在国运方面的巨赌，已经不可能停下来了，现在的局势或许已经是最好的状态。
归结起来，就是三个问题：美国今天和我们比，究竟有多少比较优势？这些优势需要多少时间可以抹平甚至反超？美国还会给我们多少时间？
今天中国的成功，来源于三架马车，其根源在于美国对我们开放的全球市场，正是有了这个全球市场，所以哪怕我们被WTO绑定了各种各样的不平等条款，我们依然在喂饱欧美乃至全球的同时，也让自己变得异常强壮。但是不要忘了，所有的这一切，均来源于美国对我们的开放甚至用他们擅长的海空霸主为我们的商场和商船护航，虽然这和他们推广美元帝国的目标是一致的。我们今天的世界地位，靠的就是10几年对世界的无私奉献和对世界的“无害”换来的，我们的武装落后，关键物资缺乏（铁、石油、天然气、橡胶、粮食等等）。可是自从反卫星和种种针对美国的先进武器曝光后，中国将要逐步开始自己承担那些美国帮我们承担掉的费用。
其一，军事层面，这包括一整套海外威慑力量和一支经得起实战考验的军队，同时要能够和美军保持军事威慑。前者需要我们迅速改变建军思维，并恶补近30年的“和平债”，后者则需要在高精尖领域持续砸钱，请注意在战略领域我们将要开始完全未知的试错，这个投入可不是几倍这么简单。无论如何，军事领域都将是一笔非常巨大的开销，并在今后的岁月越来越大。
其二，科技层面，美欧科技的山寨红利即将枯竭，因为师傅已经快被我们学死了，今后在试错方面的成本大家都会差不多，而TG更麻烦的地方在于，你要转成创新模式，需要彻底“洗心革面”重设系统了。这又是一项巨大的开销，同时又需要颠覆性的重组，光靠清华这种“哈佛预科”，你觉得有多少成功可能？
其三，文化层面，TG在这些年近乎“全盘美化”，自己的文化战线几乎全面放弃，好的一面是对接国际社会异常成功，坏的一面是如今想要搞出自己的一套东西来宣传也变得难上加难。中宣部这些年没少挨骂，但他成功地将美式价值观融进了本土，可算是完成了任务。而随着《寻龙诀》、《琅琊版》这类的影视作品出现，国家在文化领域终于开始转型，可以预见，这一领域针对美式价值观的砸钱互掐会愈演愈烈。
其四，经济层面，以上三个层面，可以预见都是赔本花钱的，甚至是巨额花销，那么作为唯一产出的层面，经济上我们目前是什么局面呢？首先，传统三架马车都问题挺大，投资，自从温4万亿造了大量多余产能，不仅提早预支了投资预备队，还随着碳排放规则的通过，成了负资产，TG白白自己空折腾了好几年，而随着村村通公路，封闭经济圈基本全部打通，经济低洼地这些国内的经济预备队已经全部启动，现有形态下未来国内投资产生的生钱速度基本不太可能出现飞跃，换句话说投资的产出正效应不用多久就会趋于饱和；出口，TG已经选择了自立RMB门户，那么以往基于美元体系下的便利都会逐步丧失，根据美媒报导，中国已经成为排在美国之后对各国反政府组织资助第二大的金主，上贡转为政治献金，至少短期看，TG的RMB战略不会有太多的盈余可用，甚至不排除因为与美元的图穷匕见而迅速转为赤字；消费，消费的根本目的，在于提高工作积极性来获取更多生产，因此美国的消费多少有些误入歧途，而中国的消费除了房地产，目前看来都相当健康，这是TG产出最为可观的一项，但光靠这项来支撑起前面几个层面的巨额开销，恐怕不容乐观。
对于TG建设的一面，我完全不担心，并相信能够做得好，但最大的变数来源于与美国的关系，新旧霸主的对抗是迟早的问题，而一旦中国彻底走上前台，中美很多方面的合作也将戛然而止。很现实的问题，比如未来非洲某国挑起事端，中国必须在美国的隐性干预下，一战立威干掉对手，这需要多少时间、多少投入中国才能做到？根据葡萄的预测，美国给予中国的时间窗口不会有多少时间，如果到时中国处理不好，不仅海外利益大幅受损，国内物资进口、民心、治安等等都会出现严重问题，弄不好全面崩盘都未必危言耸听，因为我们的经济正循环不仅短板较多，也远不如欧美那么成熟自信。TG现在的国策既然已经选择彻底赌出，那么为了未来可能的挫折缓冲，必须准备好在军事、科技、文化、经济上全面超越美国，没有这个觉悟，那就必输无疑。那回过头看，老兄认为在现有体系下修修补补，可能完成这个任务吗？
