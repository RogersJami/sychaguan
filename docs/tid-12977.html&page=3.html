对当前的战略思维作一些补充性的说明

1、美国的快速败落并不是我们现阶段甚至更长时间内的追求。我这么说可能很多人会失望，但是从现实的角度说，美国的快速败落对我们并不是最好的结果。在全球政治力量的发展进程中，霸权的陨落必然带来世界的动荡，比如上世纪90年代以来十几年间世界政治格局的快速变迁。中国或其他政治力量在没有做好充分准备以前，这样一种局面的出现对任何人都不是最好的结局。相信美国决策高层也充分认识到这一点，认定中国在相当长的时间内并没有对美国根本性的恶意。所以美国才会在一些具体争端上产生反复、犹疑，他目前的这种心态是对自身状况的忧虑和对中国快速发展的忧虑结合在一起的双重忧虑心态。

2、如果认定第1点成立，那么中国目前决策的基本思维是什么？我认为，核心的内容是做好两个融合，一个是东南亚的融合，一个是东北亚的融合。做好这两个家门口的融合，是我们谋划长远发展的根本立足点。这种融合是基于互利共赢、共同发展的，是保持地区政治实体内部高度内政独立的。要消除地区国家对中国采取传统的势力拓展方式的担忧。在这两个战略发展方向上，东南亚是重点，是目前工作的重心。东北亚方向上，韩国目前小有进展，有待更大发展。朝鲜是刺头，估计是不撞南墙不回头了，先让他闹腾去吧，有他哭的时候。日本很微妙，可争取。我这么说可能又引起一堆人的不满。从现实角度说，你不争取，那他就是竞争对手阵营的势力，从博弈的角度来讲是不智的选择。当然，韩国，尤其是日本，他的政治和文化基因必须要改造，这种改造既要靠他自身的努力，也要持续施加外部的压力，要双管齐下。东北亚的难度比东南亚要大很多，但是我们的努力不能放弃。

正是因为美国看清我们的策略，所以他阻扰和破坏的资源也主要投在这块。当前及今后很长一段时间，这两个方向上，尤其是东南亚不会太平，时不时出点幺蛾子是大概率事件。

3、基于第2点的融合思维，我们有一路一带的全球拓展战略，但是这个战略的优先度不能超越第2点，投入的资源也不能超过第2点两个方向上。一路一带的拓展，从国际政治上，我们是广交朋友，把朋友搞得多多的，把敌人搞得尽量少。在经济和人文交流上，我们跟世界上很多国家实现各种互补，拓展我们的市场和原材料补充空间。同时，我们也给全球很多国家提供一个不一样的选择，有机会参与中国的发展进程，共享中国的发展成果。而不再是以前那种辛辛苦苦几十年一夜回到解放前的状态，相信世界上很多国家乐意多了与中国做朋友、做生意这样一种选择。

4、如何应对美国的“重返东亚”的战略？正如前面有朋友指出来的那样，这个战略会是美国很长一段时期内的根本性战略，不会产生动摇，所以对美国会在很长时间内遏制中国发展这一点不要犯迷糊，要有清醒的认识。对于他这个战略，我们要采取切香肠的战术，在战略投送能力可以达到，并与美国形成基本的均势的位置上要积极争取。要稳步发展，不要急功近利，不可能一口吃成个胖子。对于领导阶层来讲，要有功成不必在我的心态。对于民间大众来讲，要全力支持中央的决策，形成上下一心，则我们的事业必成。那么，在这么个我进敌退的态势下，有没有发展的边界？我认为还是要有的，按目前态势来看，20年内发展到第二岛链的话，就是很乐观的结果。即使发展到那一步，我们也要让美国以及区域内国家相信，我们的主要目的是拓展自己的可靠生存空间，我们没有做排他性的选择。到时候你美国的军事力量也可以参与进来啊，但是你必须尊重我的重大利益，不能以目前这种霸道的方式存在。今后，这个地区的和平安全与发展议题主要还是以区域内国家为主来决定，美国可以参与，但只能是其中一员，不能以美国的意志为主。

这点对美国来说可能有点难以接受，那么在一些具体争端上就可能以比较激烈的方式来改变他的态度，比如说目前的南海态势。真理永远在大炮的射程内，对于习惯了霸主地位的美国来讲，没有硬的手段想改变他是不现实的，所以该出手时还是要出手。

5、上面有朋友提到俄罗斯，俄罗斯的战略重心始终在欧洲，以前如此，今后还是如此，对这点不要有怀疑。对俄罗斯和欧洲要以统战的思维去争取，以世界政治多极化的愿景去感召。我们不称霸，欢迎欧洲和俄罗斯正常发展，成为世界政治力量的重要角色。我想，对于有机会当上武林大会的真正一员，没有人是愿意稀里糊涂地与美国合流，成为美国马仔的角色。
