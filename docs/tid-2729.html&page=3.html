raodingchen 发表于 2012-7-13 15:31:08

转发一些越南的历史
公元939年，交趾地方豪族吴权自中国五代南汉政权独立称王，建立政权（吴朝），越北交趾地区脱离中国一千多年的统治，事实独立，但并未建立国号与使用年号。968年，丁部领（丁先皇）以武力征服境内各割据势力，建国号大瞿越（丁朝），两年后（970年）又自称皇帝，年号太平，定都华闾（今越南宁平市），这被认为是越南正式脱离中国而自主之始。之后，丁部领接受宋太祖册封为交趾郡王，这一政治行为，标准着中国皇帝正式承认越南是自治的藩属国，而不再是直接管理的中国本土。



980年，权臣黎桓篡丁，建立前黎朝。北宋曾派军欲助丁氏母子，被黎桓打败，越史称之为“第一次抗宋战争”。1010年，前黎朝大将李公蕴建立李朝，并迁都升龙（今越南河内）。1075年（宋神宗熙宁七年），李朝与北宋爆发宋越熙宁战争（越南称“第二次抗宋战争”），李朝北侵宋土，占领邕州（今南宁）大肆屠城，军民罹难达5万以上。其后宋军展开反击，进至李朝境内的富良江（今红河），两国讲和撤军。是役，宋国军民死亡达25万之众，李朝军队死亡2万人以上，掳获大量北宋工匠以及农人归国，促进了国内经济的发展。



1225年，陈煚（李朝女皇李昭皇之夫，为其叔权臣陈守度拥立为帝。陈氏中国闽人，或说桂林人）建立陈朝。蒙古（元）政权对陈朝发动三次征越战争（李朝圣宗时，越南改国号大瞿越为大越），分别在1257年至1258年，1284年至1285年，1287年至1288年。



第一次侵越战争，蒙古云南驻军3万人，由西征名将兀良合台（二次西征时，曾随拜答尔深入孛烈儿（即波兰)﹑捏迷思(即德意志)等地，攻陷古波兰首都克拉科夫。后又在莱格尼察战役中大败波德联军）率领，攻陷陈朝首都升龙（今越南河内），陈煚出逃，君臣意志低落甚至想举国投宋，但由于蒙古军的作战意图是从南面威胁南宋，并未有长期占领陈朝的打算，故不久就行撤退，此战蒙古军军纪严明，不掠民物，被当地人称为“佛贼”。



第二次侵越战争，元世祖命第九子镇南王脱欢，率兵10余万，号称50万征越。元军虽顺利占领升龙，收降了一些陈朝贵族和亡宋遗军，但在越南民族英雄兴道王陈国峻等杰出将领以及陈朝军民的抵抗下，此后作战连番失利，尤其是南宋降军新附军战意消极，成为陈朝军队的首选攻击对象，而陈军中的亡宋遗军，也参加了对元军的进攻，进一步影响并扰乱了元军军心。战至炎夏，元军中爆发瘟疫，士气低落，撤军北回，沿途被陈朝军队袭扰打击，灭宋名将唆都、李恒相继战死，总管张显降越。



第三次侵越战争，元世祖立越南宗室陈益稷（陈煚之子，封昭国王）为安南国王，命镇南王脱欢为统帅，发江淮、江西、湖广三行省的蒙古、汉军、新附军，以及云南兵、海外四州黎兵近10万，号称30万，分道入侵陈朝。再占升龙后，元军疲于应战，同时粮道不畅，脱欢等人决意北返，在途中为陈朝军队阻击于白藤江，元军大败逃回，数名将领被俘杀。



元军两次南侵，20万大军损失太半，将领死伤颇众，世祖大怒之余，敕令镇南王永镇扬州，终生不得入觐。又复调兵五万，谋划再攻陈朝，仓促未能成旅，陈朝又贡金人求和，世祖迫于事实暂时应允。数年后，又暗中设官集兵准备侵越，但因其不久逝世而终止。元廷扶立的傀儡国王陈益稷，也最终客死鄂州（元末陈友谅曾派使节联络陈朝协力攻元，越史《大越史记全书》称，陈友谅为陈益稷子）。



1400年，陈朝权臣黎季犛改姓“胡”，废陈少帝，篡夺皇位，建胡朝，并以虞舜的后代自居，改国号大越为大虞。1406年，明朝借口为陈氏复国，出兵灭胡朝，占领安南国。之后的1407年至1428年，明朝以陈朝宗室被杀尽为名，在升龙设立交趾布政司（行省），进行直接统治。1418年，豪族黎利发动蓝山起义，抗击明朝。在安南民众的支持下，黎利屡次击败明朝讨伐军，明军师老兵疲，不堪再战。最终迫使明宣宗于1427年下令从交趾撤军，并册封黎利为安南国王，明朝结束对当地统治，黎利自立为帝，建后黎朝。安南属明期间，明军先后死伤数十万人，安远侯柳升等多名将领战死。



1471年，后黎朝攻灭南方占婆国，国土进一步向南扩充。1527年，后黎朝南北分裂，北部由莫登庸建立的莫朝控制。1540年，出身藩王一生讲究正统的明世宗，派兵南下准备征讨莫登庸篡弑之罪。是年11月，莫登庸亲至广西镇南关向明朝纳降求封，明朝下诏降安南国为安南都统使司，封莫登庸为安南都统使，轶从二品，世袭，银印。明朝在赚得足够的面子后，两国罢兵。1592年，后黎朝复辟，北部由郑氏家族挟持后黎朝皇帝为傀儡控制，南部则由阮氏家族控制。1698年，阮氏家族出兵吞并下高棉（今湄公河三角洲），越南今日领土疆界基本形成。



1771年，安南爆发西山三兄弟起义（阮文岳、阮文侣、阮文惠），义军先后灭阮主、郑主，统一全国，奉后黎朝为正朔，拥立后黎愍宗为帝。之后的1788年底至1789年初，清朝与越南西山军之间发生战争，史称清越战争。



1787年，义军扶立的愍宗被废，携族逃至中越边界，皇太后阮氏则逃入中国，遣使向清朝求救。两广总督孙士毅向乾隆帝上表奏称此事，建议发兵讨伐安南，扶愍宗重登皇位。次年10月，孙士毅奉旨，调两广、云贵之兵2万余人出征安南。战争初期，清军势如劈竹，占领升龙；但后来阮文惠自富春（今越南顺化）率军反击，战局迅速逆转，西山军趁清军准备欢度春节之机，在除夕夜大败清军，清军死伤万余人，退回国内。最终阮文惠主动向清朝请和，乾隆帝同意讲和并承认其为安南的新君主。而后黎朝的君主黎愍宗，乾隆帝称他先后两次失去了皇位，说明后黎朝已为上天唾弃，因此拒绝再支持其复位，并将愍宗及其宗室、遗臣迁到京师，编入汉军八旗中。
