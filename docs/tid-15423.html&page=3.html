给个背景分析

前天去滴水湖散心，湖畔大家讨论一个核心是，围绕特定时间为节点做安排。有朋友愿意参与，也有朋友选择顺势或听天由命。这个时间节点不乱猜测，仅以公开资料做如下推论。


1，08危机后世界范围的利益调整再博弈中，中国受益之全球化首当其冲。
2，四万亿后，受益于全球化的改开路线瓶颈明显后继乏力。从四万亿刺激，到互联网金融。现在国内格局下，任何局部调整都因为外部环境挤压归于无效。
3，中国基于自身的崛起，面对明显不利于中国利益扩大再生产需求的全球现有分配制度。在破局中，我们在危机中将不可避免经受考验。
4，这样的考验，在威斯特伐利亚体系构建以来，利益再分配不仅仅是基于大国之间的正面碰撞来解决的。而且基于该体系的等级利益分配制度，一个国家要打破自身身处的利益等级，要么正面击败霸权国家重新获得全面利益再分配。要么击败和自己身处同一等级强国获得进一步生存空间。四百年来暂时没有例外。
5，基于如上叙述，既我们面临内外时间节点。对于我们于时间最有利节点在我们积聚民心与实力之后五到八年。而基于我们对手而言，时间则越近越好。而今年恰逢美国等对手大选，且大选后调整尚需时间。所以对于我们对手而言，他们能选择的最近时间点在一年半到两年之间。

这就是我这几天说的大考
