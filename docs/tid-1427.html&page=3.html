再说点关于对乌有的看法。

支持发展毛泽东思想，是没错的，但是他们本身的地位角度，使他们都无力对共和国各个发展阶段，做出准确全面的评价。尤其对改开三十年，彻底否认，是相当错误的。这种错误更体现了他们本身的局限性。

说点一些新闻界的老故事。

改开初，某报老记者得知同学任烟酒专卖局高层，欣然拜访，同学间畅谈未来，顺便做了篇采访，完成的采访稿当然相当八股，某单位事业发展蒸蒸日上，某同志意气风发斗志昂扬不拉不拉……

文稿间加了一些数据，大概老同学当跟领导汇报，成绩吗……老记者也没想就写上了。

报纸已经付印了，老记者又看了遍稿子，发现有个数据，有点晕，赶紧给同学打电话，问这个数字能公开发表吗？同学一听就急了：你想保住我的脑袋，就把这个数字赶紧撤下来……

后来听人描述的时候，我听到这个数字我也晕，原来改开起始（30多年前），我国经济居然这个样子，当年我国烟草收入居然占国民生产总值的XX%，是全国烟民在养活共和国。

所以，对乌有的看法，俺的结论，是右派的教授有屁股，左派的教授有很强的自我保护意识，
但是归根结底，都是教授，有动脑能力，没有操作能力，尤其是政治操作，这种机会，不在体制里，根本就不知全貌。
所以在评判政治问题的时候，一定要在教授和记者之间选项，
还是选记者，别看记者说了些什么，还要看他们没说什么。
