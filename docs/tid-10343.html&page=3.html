五古·感念才女陈郢客之二

给西河才女陈郢客献诗，吾早有此意。一一年，便给河内女杰南方有嘉木、飞天鸭和玉垒关献过打油诗。越明年，给洗心大姐献诗完毕，才凑齐东南西北四方，欲合五行之数，非陈郢客君不足以当之。奈何诗思不畅，后闻陈君染疾，莫能助之，更觉意兴阑珊。时至今日，方草就两首，略表推重之意与思慕之情。唯愿天怜才女，善人多助，使陈君之著述传诸四海，以警世人，以励来兹。能如是，陈君可无憾矣。

不见郢客久，吾侪思慕深。

爱其高格调，文章善铺陈。

讵知弱柳质，亦怀志士心。

纶巾愧未及，施施望绝伦。

仙才鬼魅嫉，玉叶凋泥尘。

哀哀泉将涸，寂寂居无辚。

入山寻赤李，渡海候潮音。

何日降甘露？嫏嬛满目春。
