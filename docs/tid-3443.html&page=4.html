是这样的。
但这不是我的论点。

曾经和同学聊明史，感慨中国老百姓之坚韧无比，在明末那样的烘炉里面也能忍耐100多年，还得遇到连续天灾、外患实在是没办法活下去了才会揭竿而起。

在自然经济体系下面，小民真的如同草芥一样。但是工业社会里面显然不能再套用明清时代的经验。社会的自组织、资源的自组织，知识技术的传播以及人才的聚集都不一样。能够借鉴的只能是欧美社会变革的经验。

但中国和欧美的自由化、寡头化政治模式社会又不一样。社会结构不一样，对冲击的承受能力、重建能力不一样。
当然最关键的还是资源占有量不一样。

很难想像工业化社会里面失去能源、粮食、以及现代人习以为常的生活必须的各种服务体系——医疗、教育、交通、通讯，还有很多不起眼但是不可或缺的服务。

当这些服务企业不能获得发展的资源和空间，那么只能停留在初级的单一的小微型的发展阶段，还必须持续不断的满足体制相关的贪婪，那么结果显然会很悲惨。

如果有持续天灾、周边不间断的冲突、社会群体骚动不已，粮食产量大幅度减少，等等这些都会大幅减少政策施行的条件。

现在还有条件实行一定的改变，再过20年，没有战争记忆，没有基层联系的人掌权，那么更不会有人关心社会中微型主体的死活。

项羽自刎前说，天亡我。到那一步田地，真的是天亡我，因为已经没有政策施行的条件
