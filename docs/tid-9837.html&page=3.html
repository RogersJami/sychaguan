欧盟(European Union)给俄罗斯向欧洲输送更多天然气的两条大型管道项目踩下刹车，针对俄罗斯入侵克里米亚地区进一步施压。

此举将对俄罗斯增加天然气出口并绕开乌克兰进行输送造成打击。欧盟正在权衡其他选择方案，以降低对俄罗斯天然气的依赖。欧盟的天然气供应有三分之一来自俄罗斯。包括保加利亚和立陶宛在内的6个欧盟成员国的天然气完全依赖俄罗斯。

欧盟委员会(European Commission)本周表示，将冻结South Stream项目的高层谈判，South Stream是一条输送管道，计划经由黑海向欧洲输送占其年需求量至多15%的天然气。该项目定于2018年完工。

本周，欧盟委员会还阻止了俄罗斯天然气工业股份公司(Gazprom)通过OPAL管道输送更多天然气的计划。OPAL是一条长290英里（合470公里）的天然气输送管道，将俄罗斯天然气经由德国输送至捷克边境。

欧盟规定禁止俄罗斯天然气工业股份公司获得OPAL的全部使用权，称这条管道利用率较低。俄罗斯天然气工业股份公司曾要求豁免。欧盟委员会原定于周一批准其要求，但现在称将暂不作出决定。

一个问题：新修的通过黑海的天然气管道由谁投资？欧盟是不是认可这个供气渠道？
管道是俄罗斯投资，自己运营。这个当然欧盟不认可，这是地缘上欧盟和俄罗斯不可能解开的结，就拿里海的石油、天然气外送举例子
对于欧盟来说，石油是小事，天然气是大事。
法新社有个图，目前管道情况http://zjdaily.zjol.com.cn/images/2006-01/05/zjrb2006010500004v01b005.jpg

现在这样天然气大多控制在俄罗斯手里，欧盟真是寝食难安。尤其是目前俄罗斯北线已经竣工，即上图正在建设中那条，可以从维堡直接送气到西欧。至于文中说的南线，最早是和意大利合作，从黑海由海底线路进保加利亚，然后通过塞尔维亚和黑山（这俩都是俄罗斯的好弟兄，完全是送一笔过境费），进意大利、匈牙利、奥地利，然后分流到德国、法国等地。至于文中所说保加利亚，拥有境内管道一半的股份，所以很难想象股东反对自家企业。这条管线要是建成的话，直接可以绕开乌克兰、波兰、捷克等等这几个东欧所谓新欧洲国家，对俄罗斯大有好处。因此对欧盟自然是大不利。
这条南线，就是要过克里米亚的那条线。
（话说，乌克兰这国家实在是奇葩，想当年俄罗斯停了乌克兰的供应，乌克兰就从管道里偷俄罗斯输送给西欧的天然气，搞得西欧国家没得用怨声载道，俄罗斯焦头烂额。后来又赖账不还，俄罗斯真是深恶痛绝）

至于北非阿尔及利亚出口欧洲的天然气，出口量与俄罗斯对欧盟的出口量相比，大约比例在1:2的关系。

欧盟是要搞一条从里海到土耳其，完全绕开俄罗斯的管道，nabucco，俄罗斯在全力阻碍这条管道的实施。目前还没看到有什么大的进展。

至于美国说的加大天然气的出口，基本可以看做是一种威胁，也仅存在口头上。毕竟建立天然气出口站需要很长的时间，而且会影响美国国内的能源价格。

再说一点LNG海运和管道的比较。从经济性上来考量的话，一定是管道的经济型极大的超过LNG船海运。
比如说，用项目来举例子，澳大利亚的一个项目和中俄天然气西线管道对比。

中俄管道方面，根据2009年中俄两国签署的有关协议，“十二五”期间中俄两国建设中俄天然气管道项目，其西线方案起点为俄罗斯的克拉斯诺雅尔斯克，经戈尔诺—阿尔泰斯克至我国的新疆地区。设计年输气能力为300亿立方米，里程约4500公里，计划投资140亿美元。

澳大利亚的LNG项目，整个上下游，包括采气-净化-管道输送（550公里）-液化加工和相关设施，两条生产线大约900万吨/年。项目总投资120亿美元左右。

通常情况下一般认为10万吨就是1.4亿方气。900万吨大约是126亿立方米的天然气。虽然例子不是很恰当对比性不强，考虑到LNG在产地和销售地都需要单独建立加工厂，固定资产投入极大，因此天然气贸易领域除了日本那样的国家没办法只能进口LNG之外，还是管道贸易占主导地位。
