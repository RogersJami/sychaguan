个人看法，西进是虚招，南下是实招。
前进策略应立足于工业化立足于产业布局，产业布局就要解决资金、技术、通路、通水、通电、场地平整这六个环节。西进的立足点是喀什，定位类似于新加坡这种物流中转基地。吃的是附近中亚五斯坦+阿富汗+小巴等人口近两三亿规模的市场。辐射方式就是铁路公路。喀什的短板在水，出路在于阿里地区的狮泉河、象泉河、兵营河等国际河流（狮泉河象泉河兵营河这名字让我想起了西游记的狮驼国，唐僧法师不会真的经过阿里地区吧），修建水利工程南水北调一千公里以上至喀什，这三条河流年流量毛估估有10亿立方，供应喀什一两千万规模工业人口有压力，吃水没问题，用水估计得抽调地下水补充，或者控制工业人口规模。可惜西进的南水北调计划看起来很美，做起来问题多多：1、狮泉河象泉河兵营河是印度河支流，虽说流量小只有两百分之一，会不会引起印巴反弹？2、南水北调必经缺水的南疆，你即使用管道化的下埋方式通过，这也对兔子南疆的实控能力提出挑战。
南下立足点在西双版纳，定位应该类似于东京都市圈，吃的是中南半岛近三亿规模的市场。辐射方式是湄公河内河运输+铁路公路。西双版纳的短板在于平地少，得移山。这对当下移山填海技能点满的兔子来说根本不是个事儿。
综上，所谓的西进就是喀什二道贩子，并无造血能力，南下的西双版纳才是又一个人造真正的大工厂，是真正的工业化产业化群。
