9月70城房价无一上涨 最近1年买房大部分亏了



[摘要]房价在9月份依然是惯性下调，11月有望跌幅放缓，成交量可能在4季度出现触底迹象，但从价格来看，明年一季度才会出现明显的企稳。

9月70大中城市房价环比无一上涨

新建商品住宅价格下降城市69个，分析称房地产市场黄金十年结束

新京报讯

(记者李蕾)9月份70大中城市房价环比无一上涨。国家统计局10月24日发布数据显示，9月70个大中城市中，除厦门新建商品住宅价格环比持平外，其余城市新建商品住宅和二手住宅价格环比均下降。

最近一年买房大部分亏了

国家统计局数据显示，从环比来看，9月70个大中城市中，新建商品住宅价格下降的城市有69个，持平的城市仅有厦门1个。环比价格变动中，最小降幅为0.4%，最大降幅为1.9%。70个大中城市二手住宅价格环比全部下降。

从同比来看，9月70个大中城市中，新建商品住宅价格下降的城市有58个，持平的城市有2个，上涨的城市有10个。同比价格变动中，最高涨幅为4.9%，最低为下降7.9%。二手住宅价格下降的城市有52个，持平的城市有1个，上涨的城市有17个。

中原地产市场研究部的数据显示，9月70城市新房房价数据同比下跌，跌幅达到1.2%，二手房同比跌幅达到1.28%。中原地产首席分析师张大伟表示，数据说明最近一年来买房的人大部分都亏了。而9月房价环比只有一个城市持平，其余全面下跌也创了历史纪录。

张大伟表示，2014年楼市变化的速度远远超过了市场之前的预期，房地产市场以本轮深度调整基本宣告黄金十年的结束。房价在9月份依然是惯性下调，11月有望跌幅放缓，成交量可能在4季度出现触底迹象，但从价格来看，明年一季度才会出现明显的企稳。

房价环比降幅有所收窄

从国家统计局数据来看，虽然房价环比继续下降，但降幅有所收窄，70个大中城市新建商品住宅价格环比综合平均降幅比8月份收窄0.2个百分点。

光大证券称，新房方面，9月一二三线城市房价环比继续下跌，不过跌幅较前月有所收窄。具体到北上广深，虽然这四大城市房价也继续走低，但北京、上海和深圳的跌幅均已收窄至1%以内，仅广州跌幅由前月的1.3%微升至1.4%。二手房方面，一二线城市下跌势头明显，在北上广深四座城市中，北京二手房房价表现最差，较前月环比下跌了1.4%。

国家统计局城市司高级统计师刘建伟表示，值得注意的是，9月份70个大中城市新建住宅和新建商品住宅销售套数比8月份均增长8%左右，销售回升对房价的影响还需进一步观察。

分析

“救市”政策或助房价企稳

分析称，在房价全面下调的情况下，政府救市政策愈加频繁，这可能带来房价企稳复苏。

张大伟表示，房价下调影响整体经济，包括银行限贷松绑，都是为了房地产市场企稳，土地财政的存在使得各地在楼市降温的情况下，救市的政策出台愈加频繁，力度越来越大，救市政策出台的底线越来越低。各地将在银行释放救市政策后，通过财政、税费补贴等形式刺激楼市，尺度越来越大。

信贷松绑政策对市场影响逐渐体现。国庆以来各地成交量明显上涨。中原地产数据显示，超过40个城市公司销售都有明显上涨。张大伟称，各地的救市热情将持续上涨，年内市场存在出现反弹的可能性。

光大证券也认为，考虑到9月新房房价跌幅略有收窄，同时政府近期已经陆续出台了一些地产放松政策，地产行业很难再加速恶化下去，10月地产销售延续环比改善势头将是大概率事件。

（新京报 李蕾）

农民工仅仅是进城打工而已，要买房落户，还有很长的路要走。远郊土地变现不容易。

所以生活需求一直旺盛，只是没钱。
