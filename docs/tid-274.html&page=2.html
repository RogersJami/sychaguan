成奎花 发表于 2012-4-5 21:35 static/image/common/back.gif
这个拉闸限电据我所知是为了要完成当年年度碳排放的减排指标。
拉闸限电很简单粗暴，但从某种意义上说是不得已而为之。

中国的资源限制，不是书本上的，到下面走走，就知道是很现实、很紧迫的。别的不说，
每年夏天电网供应不足的时候，农村拉闸以保证城市供电在不少地方是家常便饭，而且没有通知，说停就停。2010年夏天我在河北一农民家里调研，
停了一上午电，午饭吃到一半电来了，夫妻两个立刻放下筷子， "趁着有电，赶紧去地里浇水"， 我跟着他们两个急急跑到地里，
浇了不到十分钟，电又停了。 农村抽水抗旱和城里人用空调，哪一个是更基本更急迫的需要？ 可是，人之道损不足补有余。
在一个关于可持续发展的讨论会上，我听到发改委城市和小城镇改革发展中心主任李铁沉痛又无奈的提到，每年夏天用电高峰，周围四省轮流拉闸保北京。十一五末期为了完成节能减排一些城市拉闸限电意见很大，殊不知这在不少农村其实根本不算什么，早就习惯了。

然而，阶梯电价《征求意见稿》甫出， 有相当多的意见是第一档电量标准不应该一刀切，
而要考虑到地区差别，比如北京、上海的第一档电量，不应该是全国户均用电的87度，而是北京户均用电的100多度。原来说北京、上海等发达地区的居民，就应该比农村的居民享有更多的廉价能源？

让城市人口也尝尝拉闸限电的滋味，才知道资源不是天上掉下来的，可以永远那么廉价。多拉几次，大概才会知道节约的重要，和涨电价之类的经济调控手段少一些抵触。
