我没有遇见这样的情况，没法帮你支招。

但是我可以给你举个我同学的例子帮助权衡一下利弊。他在海外上研究生时要的老二，是个女孩。老大是男孩，当时刚两岁。同学学习非常辛苦，没有时间照顾家庭。老婆一个人要照顾两个孩子，还要给一家人做饭也相当地累人。中间之有老婆做月子的时候婆婆匆匆来两个月帮忙。同学两年后毕业，听他讲，男孩的性格变得怪，脾气大，分不清家里和外面的区别，甚至到老师要求做社区心理辅导的地步。现在男孩已经上高中了，有时还是控制不好情绪，孩子人非常善良。我同学很后悔在老大小的时候光想着照顾小的，忽略了大的。

我感觉在两个孩子年龄相差非常近的情况下，关注度如果不能兼顾，宁可暂时更倾斜一个，我趋向于倾斜向大的，因为从我的经历来讲，小孩子是白眼狼，谁对他好，他就对谁有反应，所以在1岁前是分不出事母亲还是其他家庭成员给的关爱，只要生理达到满足就行。稍大点再弥补也是可以的。而大一些的孩子已经能区分家庭成员的不同关系，已经能够意识到有另外一个在和他争抢爱，所以心灵受到的影响更大一些，从母亲那里没有得到足够的爱，安全感降低，会伴随各种大人不能理解的现象，其实那是在和小的争夺关注感。大人对孩子的这种心理没有足够关注的话，我认为会影响到社交心理和技能的发展。

我的建议是在你不能兼顾而影响到自己的健康的情况下给出的，如果有更好的办法兼顾，当然用更好的办法。老槐哥有专业背景可以给你更专业的辅导，我讲的是从生活中来的，仅供参考。

西溪老槐 发表于 2014-12-16 19:31:29

风云际会：你好！

上次上来已经看到你的帖子。大陆当时是一胎政策，多子女教育我和朋友们多没有实践经验。与发展心理学的朋友讨论后，有这样两点是关键，加一个建议。

首先，重点是老大。一，老大对人际的感知已经到了比较关键的时候，重建的亲子关系容易反复。二，老大与老二的亲子感知是不同的。老大原来是千宠万爱在一身，现在有人分享你对他的爱，这种失落是老二没有的。老大容易产生人际挫折感。

其次，成人之间不能当着两个宝宝的面谈论他们的关系，谈论他们的对错。不要拿一个做榜样去批评另一个。而是要用鼓励的言语向另一个学习。这是许多家庭多子女教育中常犯的错误。

建议，与单个宝宝，或者两个宝宝同时交往时，试着低下身来，与他们眼睛平视的高度来交往。这个方法简单，但是，心理效应很多。不展开了。

其他的内容，在私信里聊。跟帖中许多茶友的思路和方法都是可以借鉴的。

祝你顺利。
