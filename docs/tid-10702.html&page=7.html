最近由于中越南海争端等一系列原因导致越南总理阮晋勇和国家主席张晋创的矛盾持续升级。据一位不愿透露姓名的越南高层说，阮晋勇在最近的一次饭局上表示：越共只有走普世价值，才能继续为民执政，否则永远也改变不了变相独裁的本相，若无法改变，当立即解散！
　　此前《国际金融报》6月30日曾报道，越南内斗的导火索，是越南党内就如何定性一系列反华暴力事件，产生了矛盾。6月1日，越共政治局发布了一个告知全党书，公开指出最近发生在越南境内的一系列反华暴力事件，“是有关反共反社会主义的，以民族主义为借口发生的暴力行为”。不料，此公告一出，也就是6月3日，总理阮晋勇就公开表态说，“我对这个有保留。”这一公开言论，立即撕裂了本就脆弱的越共共识。6月5日，反击就开始了。越共中央监察委员会提出，将对越南总理阮晋勇进行专项廉政检查。
　　一方面，阮晋勇当众表示解散越共，“说明越共内部斗争已经白热化了”。此前多维新闻曾经指出，无论是越南的“反华”暴行，还是中越西沙海上对峙，越南混乱的内政矛盾是一个重要的推手。

2013年媒体曾报道，越南有意将国名从现在的“越南社会主义共和国”改为“越南民主共和国”。在胡志明市，美元始终是畅通无阻的硬通货，甚至比通行东南亚的人民币都要显得坚挺。
