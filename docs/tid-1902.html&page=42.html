对组织而言，有理想没钱，理想撑不下去；有钱没理想，钱撑不下去。
其实说理想太局限了，理想到底还是世界观价值观的一部分。
而世界观就影响对利益的认识。
当远景超出大部分人视野的时候，不宣传理想不宣传世界观，长远的利益就没法和眼前结合，组织就难以长远规划。然后算的长的打败算的短的。
大多企业根本没本钱承诺员工一辈子养老，不讲理想怎么让人十年十年的砸下去攻关。
也没谁能永远有钱，顶不了逆风，一败就乱的，会被历史淘汰。
