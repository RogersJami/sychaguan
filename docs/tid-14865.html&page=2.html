指读（手指头指着读）的方法很简单，介绍如下：
把孩子抱到怀里，拿幼儿图文画册（画面鲜艳，文字要大），指着画册的文字给孩子讲故事，当讲到孩子兴趣上来后，家长就有意识的降低指读速度，这时孩子会着急，她就自己往前看了。如此反复，孩子就逐步认识汉字了。

孩子的幼儿图书早都送人了，实在是想不起来多少了，我记得当时是迪士尼的图书居多，原因是色彩鲜艳，故事编排的可读性比较强，容易引起孩子的兴趣；我记得有中英文混排的，也有只有中文的。

我觉得中国的图书符合：色彩鲜艳，故事编排的可读性比较强这两点的，也是非常好的。
