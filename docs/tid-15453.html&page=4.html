河中兽 发表于 2016-6-11 11:45:35

脱离经济谈教育公平，在我国社会环境下是无解的，最终一定是摆立场、讲蛮理。

多少人觉得，如果各地都搞素质教育，京沪也就没有优势了。这个说法反正是个假设，对错也没法校验，但是问题是，怎么做得到各地都搞所谓素质教育呢？ 师资呢？ 资源呢？ 京沪招个中学老师，现在开口起步价就往硕士上面喊；偏远地区中专直接做教师。非要说这两类人教出来学生没差别，可能吗？ 真当人家那么多年书白读了？ 反过来，京沪等资源优势地区的教师，当年也是玩过题海的，带队伍玩题海，怎么就一定输了？ 这里面要弄明白一件事，不是资源优势地区搞不出题海战，而是人家根本没那个必要。

说一千道一万，脱离经济基本面谈教育公平，要么教师队伍做雷锋，要么国家政策出面做雷锋，单纯以现在市场环境，其实拼进了北上广的家长不也就是要个教育优势么？ 有几个愿意把孩子塞回老家去刷题考试？ 在这个问题上，家长们都是拿脚投票。但凡上了车，一定先冲着司机大喊一声“师傅关门”。
    无关高尚与否，就是地区经济和其他资源分布不平衡决定了教育本身也没法达到期望的“公平”。指望教育部门来解决这件事，本来也没戏，所以教育部出手搞这种那种改革，最后基本都是在一片口水中变成这个时代最常见的骂战。
