“其实，这要从正反两方面来看。此次国家队奋尽全力，吸引民间资本进入股市，主要作用前面已经说过，还有一点就是要保持这种高位的吸引性，这就需要更多的资金进入。而更多的资金进入又会继续拉抬股市吸引资金，这是一种活循环。只是，国内的资金是有限的，也是很难锁住的，所以，不但要拉高股市，更要让实体经济有所表现，一带一路配合亚投行就是保证活循环成功的基础。只是，这不但需要外部资金进入，更需要外部基础建设的需求。所以，大掌柜此次在亚非会议上要做的事情有两个，一个是让优质经济体投钱，比如日本；一个是要让基础建设需求国提出规划蓝图，保证这些资金的有效使用性。这样的话，一潭活水才能流动起来。”

“哦，原来如此，看来大掌柜此次任务很重啊。”

“是的，一个不小心，后继无力或基础建设谈不拢的话，对于国内的经济伤害都是无与伦比的，这就是兵行险招。所以我们看到，在亚非会议上，缅甸还可以接着傲娇，暗背还可以接着拖延，因为他们深深地知道掌柜们的命门所在。不捞够筹码是不会松口的。但同时他们也知道，只要中国挺过这一关，他们就会全盘皆输，这不但是意志的较量，也是智慧的博弈。成败，就看我们能够拿到多少基建项目来支撑先期的亚投行运转，而这些项目是否是优质资源也是加入亚投行或准备加入亚投行的各国紧盯的焦点。假如我们拿到的都是一些还款能力弱，国家安全差的建设项目，恐怕后期的运转就难以为继了，相反，要是能拿到那些投资回报大，预期运行时间长的项目，到时候，只怕亚投行就会变成世投行。到了那个时候，你说说，股市的这点涨幅算得了什么？而精明的日本人正在观察到这个节点，他们才不愿错过这个时机，但如果是个泥潭，他们也不愿踏进。是时机还是泥潭，尽在掌柜们的决心和智慧了。”

“嗯，不错，你说的很有道理哈。这么说，我还有一点私房钱，是不是也要拿出来支持一下掌柜们呢？”

“哼。。哼。。你这个出家人尚且如此择时，还要怪其他人或国家吗？所谓天之道---自然，人之道—难缠。掌柜们，任重道远啊．．．．．．”
