宁波巧用大数据突破网络订餐监管瓶颈

打开智能监控系统，输入查询关键词，不过一会儿就从数万家店铺中发现5家店铺存在违规行为，了解线索后，监管人员第一时间赶到现场，当场查处一家黄焖鸡店暗藏“黑作坊”。
线上检查与线下监管密切联动，巧用大数据突破网络订餐监管瓶颈，这是记者近日在食品安全宣传周上了解到的监管部门的外卖监管新办法。

　食品安全关系国计民生，是全面建成小康社会的重要标志，也是党中央关心的重大民生工程。在老百姓普遍关心的网络餐饮监管方面，宁波智能化监控系统树立智能监管新坐标，引起与会人员关注。

　据陈元刚介绍，智能化监控系统上线前，宁波第三方平台商铺证照不合规率达30％，经过今年上半年运行，现在证照不合规率已大幅下降到2％左右。

以网治网，网络订餐问题才能得到有效解决。据李由宙介绍，全国主要第三方平台中1200万家入驻商铺的数据，这一系统只用一天时间就可以采集到商铺公开的所有菜品、证照等信息。
