港灿闹事只会便宜挑拨离间的台巴子

港灿闹事只会便宜挑拨离间的台巴子作者博客笨狼出山 http://mememevvv.blog.tianya.cn 作者西西河文集 http://www.cchere.com/user/迷途笨狼

港岛一小撮反华走狗闹到今天这地步，如果说没有新军1统特务暗中煽风点火，我是不信的：这边示1威如火如荼的时候，那边代表团就迫不及待的拜见习近平了，虽说国军传统就是“友军有难，不动如山”，不过抢主顾的吃相忒难看，一把年纪的马娘娘明目张胆的送秋波，连欲遮还羞都做不到。

也难怪马娘娘着急，选战不是民进党对手，内部也勾心斗角指挥不灵，岛内经济又不景气，只好求大陆增加施舍了。全球大环境，世界范围内金融危机尚无消除迹象，中国大陆国力却日渐昌盛，工业用电量差不多是排名第二的霉菌3倍，抢地盘放狠话的时候，一向嚣张的美帝也不敢接茬。随着解放军新装备形成战斗力，干爹山姆已经护不住炮党在卧榻一角（还不是卧榻之畔）偏安了，更别说对大陆在钓鱼岛巡航近乎视而不见的倭寇。再顽冥不化，一旦动武鸡飞蛋打是肯定的，还不如趁着港灿犯傻卖个好价钱。有奶就是娘是炮党尤其高层元老的悠久传统，某种意义上炮党诞生、发展、壮大、战败偏安史就是倭寇、毛子、霉菌政治宠物豢养日记一部分的集合，早年不断换恩主现在或稍晚从良都是很理性的选择，真心选他们当盟友不扒层皮就怪了（遥想tg当年，6万多党员几个月后只剩下1万多）。炮党跑到台湾后内斗行径了解有限，在大陆混时哪怕中央军内部自己人也没少在背后下绊子甚至下刀子，现在不顾反共统一战线大局、先煽动港灿闹事再跟大陆眉来眼去不过是本性使然。

虽说炮党很多事上象开血汗工厂的台1巴1子那样唯利是图，不过百足之虫，死而不僵，“三民主义反攻大陆”的准备还是处心积虑的，“西沙战事要紧”和抗日名将张灵甫等不过是新军统杰作一小部分。以西沙为例，历史真相是即使南越把南沙（包括太平岛）、西沙划入版图，跟tg鏖战之后，国军驻西贡代表还帮着南越猴子抗击北方共军。当年炮党《中央日报》，国防部发言人的说法是：“太平岛我军严阵以待,如有越南军人上岛, 将好言劝告；如有共军上岸, 将痛击。”因为转这个笨狼居然被tianya封掉微记等很多权限，转大陆贪官污吏新闻没事，想转新的都不行，辗转到指定版面投诉也不搭理，看来40年过去了，阶级仇仍然大于民族恨，炮党对大陆媒体渗透也可见一斑！港灿行吗？这么强大的暗黑力量，只管挑拨离间大陆和香港，把港人放火上烤，不劝导马娘娘坚持反共统一战线，把冒险上街的港灿当作跟大陆讨价还价的筹码，是可忍孰不可忍！
如果没有炮党的背叛，港灿可以一部分唱红脸一部分唱白脸，继续玩“会哭的孩子有奶吃”的把戏坑大陆银子，被出卖后只怕例行压岁钱都可能成问题了。要知道香港经济最重要支柱就是金融、旅游（地产就算了，娱乐业还在衰弱），想好过都离不开大陆扶持，最近起劲闹腾等于为卖友求荣的炮党做嫁衣，十一黄金周香港旅游业损失还不知道有多少！如果示威头目大多不是新军统或苏格兰场、中情局等机构特务，看迹象现在还没回过味来，来大陆的富豪已经明白，有什么办法或采取什么措施挽回吗？

笨狼建议以彼之道还彼之身，按照常凯申“攘外必先安内”信条，先把吃里爬外的新老军统特务揪出来示众再说其它。想出气想保住人民币离岸结算中心位置之一也简单：把台1巴1子开办的血汗工厂残酷剥削真相（别忘了主要清单）大白于世、让尽可能多的大陆同胞了解，还有炮党当年干的龌龊事，诸如向倭寇出卖东北，向毛子出卖蒙古，纵容越南猴子侵占南沙等……大陆校园里新老右派太多，官方不提，很多事不关心政治的知识分子都不知道，所以“西沙战事要紧”等谣言满天飞。还有打铁还需自身硬，最好主动揭批金庸等作者明显违背史实的小说暗示，突出代表就是《碧血剑》里对猿崇焕正面抗战，不——抗金或满清的吹捧，事实真相说白了就是靠毛文龙敌后游击拖满清后腿仗着坚城利炮硬挺虚报战功罢了，还向满蒙走私军资，害死毛文龙后重金打造的关宁防线就不灵了。还有《鹿鼎记》开头讴歌的查伊璜，就是金庸或查良镛先辈，历史上就是著名文1字1狱《明史》案首告，吴之荣不过是奉命行事或职责所在罢了，祖上脏水泼到人家身上不合适。因为逃到香港的地富子弟太多，违背史实或明或暗抹黑中华民族的东西太多了，还历史真相，榨出某些家伙“皮袍下的小”来，可以作为一个很好的经济增长点吃很多年（欢迎出版笨狼作品），也算是向中央输诚，早点搞，相关文化产业和金融中心还有希望。好奇最近闹事，股市暴跌究竟坑谁呢？想要民主应该多搞几次，让更多港人认清反华势力丑恶嘴脸。

笨狼判断不错的话，马娘娘快回大陆了，希望还犯迷糊的港灿们早日回头是岸。

袁崇焕到底该不该杀?

讲史得讲“人”，也要讲“制”，要“以人为本，以制为纲”，不然要就犯“马克思大跳街舞”的蹩足错误，并得出蹩足结论。

客观地说，袁崇焕杀毛文龙就是犯了死罪。

实际上，老袁也曾和自己的顶头上司王在晋、高第有过意见分歧，甚至是冲撞，如果都凭着“官大一级压死人”的观念，想杀就杀，那岂不乱套了？

所谓“尚方宝剑”，毛文龙手里也有，你这把“尚方宝剑”凭什么能斩他那把“尚方宝剑”？

一个大军区司令可以公然处决一个省军级干部，而不用事先向中央请示汇报吗？

而让崇祯最无法接受的还有六点：

第一，毛文龙死后，崇祯曾自己下谕，把袁崇焕杀毛文龙的负责全部解释为“圣意”（即皇上的意思），但不久之后（史载只有1个月时间），皇太极便大摇大摆地绕道漠北蒙古地区，自长城入侵京畿。作为辽东经略，袁崇焕确有失察之罪。
当时，有朝臣立即弹骇袁崇焕“擅杀边帅，纵敌长驱”。或许，我们不能过分地责怪朝臣的弹骇。
要说到毛文龙，还先得说皮岛。皮岛是位于朝鲜湾中的一个小海岛。史载，毛文龙多次指使部下自岛而上，采用“游击战术”攻击后金，随后又退入海岛之上，由于后金八旗不善长于水战，因此，毛文龙把努尔哈赤和皇太极的头都搞大了。虽然《清史稿》多使用带贬义的句子来形容毛文龙的所作所为，什么“逃入海”、“败走”之类的，但实际上，这也从一个侧面反映出了毛文龙以及以他为核心的“皮岛军事力量”确实对后金形成了制肘的效果。
从史料上看，毛文龙原是李成梁的部下，皮岛则是毛文龙一手经营和创建起来的一块“海上根据地”。的确，毛文龙在皮岛上开设了商品交易市场，把从后金地盘偷采到的山珍、明朝的贵重货物、朝鲜的土特产拿来交易，同时还设立了所谓的“关税”（主要针对朝鲜至中国的商船），但他这样做，也实在不至于罪当杀头。由于这位毛大帅“赚财有方”，史载，当时皮岛一度由一个荒凉小岛变成了商品集散地。我不否认毛文龙有“军阀”的某些特征，但当时辽东已乱，群臣皆不敢守之，如果皇帝授一个什么辽东的官职，朝中那帮大臣就像被杀了头一样难过。而在这种情况下，毛文龙能“独驻敌后”，确实不能说他无胆无识。而事实上，根据当时的史料记载，除了一些捕风捉影的传闻外，还拿不出什么像样的证据来证明毛文龙要反明投金了。至于什么“毛文龙通敌书信”之类的东西，一是考据不详，二是无法佐证。照此说来，那袁崇焕还和皇太极互致书信，甚至大谈“议和”呐，难道我们就能以此断定：袁崇焕必然是在通敌卖国吗？
[ 转自铁血社区 http://bbs.tiexue.net/ ]
其实，无论从“资历”、“经营”和“战功”上讲，毛文龙在袁崇焕面前有时显得比较“大牌”，也是可以理解的。但这下好了，袁崇焕居然以“十二条罪”把毛文龙杀了，而且杀前不奏报。
所以，当时的朝臣弹骇袁崇焕并非完全没有道理，也并不是说就等于“陷害”。

第二，在战略上，袁崇焕不是立即联络朝鲜，夹击沈阳，迫使皇太极回师，而是率着关宁铁骑几乎放弃了宁远，“千里勤王”，这意味着：本来敌人的腿很弱，可以给他一个扫堂腿，杀杀他锐气的，但现在你却偏偏不给他扫堂腿，而是用你的手去和敌人很强的拳头比力气。
到了京畿地区之后，袁军又不死狙八旗，而是把大部队开到河西务去开了一个“军事会议”，讨论是不是需要“入京勤王”——放谁当皇帝也想不通这个。
其实在此之前，袁军的一位部将就提出“必御敌于北京之外”的建议，但不知道袁崇焕是怎么想的，似乎没这意思。
以关宁铁骑（即根据“辽人守辽土”的军事原则组织起来的明军野战部队）的速度和野战能力，如果死扑八旗，我不敢说有“百分之百的胜算”，但至少可以使敌受制。所谓“效命”，正是在这时体现啊。诚然，我们可以用“粮草不足，军需不济”来解释这时的关宁铁骑，但这又何尝不是袁崇焕“失察之罪”的一种延伸？
[ 转自铁血社区 http://bbs.tiexue.net/ ]

第三，八旗在北京周边大肆抢掠、破坏，毁了不少王公大臣在郊区的产业，他们几乎是天天上折子弹骇袁崇焕。如果你处在崇祯的位置，将作何感想？
打个比方，如果有人天天在领导面前“弹骇”你，恐怕你的日子也不会比袁崇焕好过吧？如果有人天天在你面前“弹骇”某个人，你对这个人的印象会好得起来吗？更何况是在战乱时期。

第四，袁崇焕在第一次平台召对时，明明说的是“五年复辽”，这其实相当于在崇祯面前立下了“军令状”。而崇祯走后，袁崇焕又对担心自己的人说：“或许可以吧。我这样说只不过是为了安慰安慰陛下。”这不是成了“军中有戏言”了吗？
崇祯好歹也算是个“军委主席”，你一个大军区司令居然敢说：“我这样说，是为了安慰安慰咱‘军委主席’。”这不是洗刷“军委主席”的权威吗？
如果不说“五年复辽”，而说“八年复辽”，甚至“十年复辽”，又有何妨？当皇帝被骗时，他或许比听到“八年复辽”、“十年复辽”更觉得难以接受——后者顶多也就是个能力问题，而前者则是“欺君妄上”，是道德问题、政治问题。
事实上，正因为有袁崇焕“五年复辽”的承诺，所以，崇祯对他全力支持，在这几年里，崇祯对弹骇袁崇焕的折子几乎是“视而不见”，对辽饷更是毫不吝啬，有资料证明：在这几年里，国库银两大多充了辽饷。
[ 转自铁血社区 http://bbs.tiexue.net/ ]
崇祯的心情其实不难理解，如果你是崇祯，正在北京皇宫里想着“五年复辽”的大计时，忽然某一天，在北京城外传来密密麻麻的八旗兵马声，你说你会作何感想？

第五，袁崇焕私下和皇太极“议和”（曰“款”），当然，我们姑且把这个理解为袁崇焕的一种战略考虑，因为在某种特定的情况下，用用“议和”这样的计策也未尝不可，不能一概而论，全面否定。但是，正是在“议和”期间，皇太极兵进朝鲜，使得朝鲜王国这个明王朝世代相依的盟友和臣藩一夜之间兵溃君逃，最终不得不与后金签立城下之盟，结为所谓的“兄弟之邦”，几乎有掉过头来对付明王朝的意思了。而袁崇焕在此期间却只是在宁远城“修防设池”，强调“城防工势”的重要性，几乎没对后金采取任何军事行动。
聪明的“议和”，换来的应该是自己在战略上的有利地位，这位袁督师倒好，他和后金的“议和”竟换来了皇太极在战略上的有利地位。显然，在这场“议和”中，袁督师处在了一个相对“笨拙”的位置，可以说，毫无军事大家和卓越指挥员的风采，既无制敌之奇策，也无攻伐之良谋。
我们知道，历史上杰出的军事指挥家和军事将领，如白起，如田单，如司马懿，如高仙芝，如史万岁，往往能够“游刃于战场之上，制权于方寸之间”，而在“后金平定朝鲜”的战争中，我们似乎没有看到袁崇焕杰出的军事指挥才干，不仅是我们，或许也包括了当年那位崇祯皇帝。
我们现在的明清史家通常只能用“修筑城防，加强工势，修整士卒，准备粮草，争取时间”来解释袁崇焕当年在那场战争中所做的这一切。我不禁想大胆的问一句：难道打仗如同修房子，要先修好了房子，再搞装修？
换个位置想，如果你是崇祯皇帝，忽然听到“袁军未动一卒，后金平定朝鲜”的消息，你又会作何感想？
[ 转自铁血社区 http://bbs.tiexue.net/ ]

第六，抗旨开市卖粮给“蒙古”人。有学者根据《国榷》等史籍认为，当时袁崇焕卖粮给所谓的“蒙古”人，其实就包括了后金人。这些后金人是混在蒙古人中来买粮的。而事实上，当时，蒙古早就和后金达成了“战略同盟”，并结下了“儿女亲家”，孝庄太后就是不折不扣的蒙古人；而自皇太极征林丹汗之后，除林丹汗走死青海大草滩之外，他的儿子额哲也归顺了后金，甚至连林丹汗的老婆都嫁给了皇太极。
最关键的是崇祯根本没同意过袁崇焕“开市卖粮”，对于蒙古的所谓“缺粮”，当时崇祯还是比较宽仁的，他给袁崇焕的圣旨明确写道：“按人数救济”，并要求袁崇焕严格盘查，“务不使粮食流入东夷（后金）”，否则“以通夷论处”。但袁崇焕似乎对明令没有太在意，仍然是“我开我卖”。那些粮食究竟流入当时严重缺粮的后金国没有？谁敢拍着胸口保证：绝对没有？
事实上，当时的后金国由于采取了血腥的种族政策，致使“壮丁锐减，人口外逃，田地荒芜，居无完室”（这就是后来“逃人法”出台的背景）——严重缺粮的正是后金。
试想，辽东士卒的大部分粮食都要从内地运去，你袁崇焕竟然还有这么多的“余粮”拿出来卖。而且，三翻五次地公然抗旨不遵，如果换成你是崇祯，你又作何感想？

不错，袁崇焕在辽东战场上确实有他的贡献和过人之处，例如主张“凭坚城，用大炮”、提倡“辽人守辽土”，并发现了祖大寿一批忠勇的辽将，同时，在孙承宗的基础之上，袁崇焕加固了宁远防线，但我们不能把一个人描述成高大全的“完人”。作为当时辽东战场最高军事指挥官的袁崇焕，在晚期擅杀边帅、抗旨开市、不察敌情、坐失战机，均是致命的问题，甚至够称死罪。

作为崇祯来讲，他可以一次原谅袁崇焕（杀毛文龙未追究）、二次体谅袁崇焕（抗旨开市未深究），但还可能有三次、四次、五次吗？“五年复辽”未成，而虏兵已至北京城下，而且喊杀声震天动地！如果是你，放在当时那种环境下，你还会相信有三次、四次、五次……直至最终“成功复辽”吗？
