乡镇那点事之三：状纸

本帖最后由 玩钉耙的悟空 于 2013-1-24 17:21 编辑

这些天事情赶人，没时间填坑，先晒一下八股状纸。下周出差结束，及时填坑。
某次回乡下，正赶上乡亲“串联”，商量着去上访，被要求代写状纸，稀里糊涂掺合了一把。
玩着钉耙给茶友们作揖，如果曾看到过这张状纸，可别把我晒出来，实在不想被参观，而且事情还没有完全落停。



致XX镇政府申请书主题词新农村建设筑路建桥 工程信息不清 传言部分住户拆迁 标准不明程序不清本组预留宅基地被他组占建 建桥占用本组河道 补偿情况从未说明申请以合情合理合法的组织程序公开公示上述信息邻村整体拆迁办法和事实 公路通车危害住房安全 河道防护栏缺失和消防安全隐患长期以来环境污染 癌症多发 近5年来已有15人患癌症去世请愿整体拆迁

XX镇政府及各套班子领导：
您们好！

我们是XX村XX组居民，一贯拥护并配合党和政府的城镇建设，欣闻新农村建设更是举双手赞成。

最近，政府在本组居民住宅区附近铺设公路修建桥梁，围绕该工程的修建发生一系列的误解、误会和纠纷，并引起居民对长期以来居住环境污染的不安、不满和恐慌，并多次到村委会、镇信访办和规划办等处反映情况，申请政府部门公开相关信息并有所作为，但至今没有下文。所以流言漫天：纷纷传言政府处事不公不作为，干部枉法，征地补偿隐匿不实，倒买倒卖宅基地；干部家庭及官二代动迁时多占多建宽敞逍遥、官三代襁褓中已批得宅基地，而村民无立锥之地，后代只能雨天打伞暴露风雨中，甚至出现“是共产党执政还是国民党当政？”、“二十年后农民再次起义”的言论，人心不稳，民情不定，民怨沸腾，出现“以死相拼，以命抗争”的冲动情绪。

我们愿意并一直坚信，坚持三个代表思想、落实科学发展观的人民政府一定“情为民所系，权为民所用，利为民所谋”，只是本组居民历次以来可能没有清晰完整地表达自己的关切和合理请求，也因此导致政府各套班子没有获悉相关的真实情况，造成信息不畅通而下情不能上达。

现将基本事实陈述如下，并提出以下申请：

一、        信息公开公布申请根据中华人民共和国的《行政许可法》、《信息公开条例》、《村民委员会组织法》等法律法规及相关规章制度，特别是胡锦涛总书记多次强调人民群众享有知情权、表达权、参与权和监督权，本组居民申请镇政府责成有关部门和村委会公布、澄清下列信息：

1.        公路桥梁项目规划详情居民普遍担忧，该项目施工过程中和建成通车后因震动危害住房安全。提请以工程告示板的形式立即公布该工程规划图，以打消居民的担忧。多少住房在规划红线内，公路桥梁的规格、宽度是多少，计划中开工和完工是什么时间，工程承建方是哪个单位，具体负责人、指定协调人是谁以及其他必要的相关信息。

2.        拆迁标准和拆迁户拟定详情目前公路规划并没有公布，但传言部分住户拆迁，已根据不同的住房面积评估且准备补偿相应的拆迁费用，有33万元、47万元、30万元等不同数目。其他住户质疑拆迁户拟定过程中存在的人为随意性——拆迁的标准是什么：为什么有的住户拆迁，有的住户不拆迁？拆迁户拟定的程序是什么：是“由下而上”经村委指定后上报规划动迁部门核准，还是“自上而下”经规划部门论证后通知村委？但邻组XX村XX组因少部分居民住房在规划红线内而整体拆迁，为什么缘起同一工程组与组之间有此差异？

3.        预留宅基地被占建详情2008年征收土地时村委在情况说明会上保证，为本组预留宅基地，并指定具体方位，共有前任村书记徐XXX、现任村书记XXX、村干部XXX等在场。在未征求本组居民同意的情况下，现该地块已被邻组（XX村XX组）占建，用作该组拆迁的宅基地。本组居民感到被欺骗和愚弄，是什么人在什么情况下在什么时候做出此决定？

4.        占用河道补偿信息据路桥施工方陈述和实际已施工进程显示，该工程肯定占用本组XX浜河道，但至今没有相关的职能部门说明占用面积和补偿问题。占用多少面积？是否补偿？补偿标准是什么？居民多次分别向村委会和镇规划办询问，村委答复是应该补偿，镇规划办的说法是不补偿。居民云里雾里，不知哪种说法可信可靠。

以上信息，请有关部门、组织和公职人员依照中华人民共和国相关的法律、法规和条例，以公正公平便民的原则、程序和形式，予以公开和澄清。


二、        本组住户整体拆迁请愿本组居民请愿整体拆迁，理由如下：

1.        邻组整体拆迁邻组XX村XX组少部分居民住房在该公路规划走向红线之内，该组已整体拆迁。按此事实，本组居民申请照此办法整体拆迁。

2.        公路通车危害住房安全因为居民住房修建于1980年代和1990年代，由于当时经济条件普遍困难，房屋质量本身并不牢固，经过二三十年的住用已是脆弱不堪，有的即将成为危房。居民深恐公路通车后因震动对住房构成不同程度的破坏，引起墙体开裂、房基沉降甚至房屋倒塌，危及居民的生命财产安全。

3.        消防隐患严重本组居民住房之间的间隔狭小，有的住房更是两家共用一面墙体而联排，根本没有消防车进出通道空间。一旦失火，消防车不能出入而扑救不及时，极易导致整片过火，存在严重的消防隐患。

4.        河道防护栏缺失的安全隐患本组住宅区内的村道紧临河道，岸堤于2008年加固，但没有加装防护栏，极易发生落水事故。青壮年外出上班，日常家中只有老人和儿童，一旦发生落水意外，后果不堪设想。

5.        水体污染的恶劣影响和后果——先后有15名居民患癌症不幸去世流经本组的河道XX浜污染严重，常年水体暗浊漂浮各种垃圾，有时呈乳白色，有时呈酱红色，有时呈墨绿色。一到夏天更是臭气熏天，根本不能开门开窗，特别是XX二区建设以来，生活污水和粪便长期直接排入河道，污染一天比一天严重。居民苦不堪言，多次向村委会、水务和环境部门反映，后有专人清理漂浮垃圾，但水体污染并没有得到有效治理而改善。

近四五年来，本组居民患癌症人数明显增长，因病不幸先后去世者共有15名，无一例外全部为癌症患者——呼吸系统肺癌6人，消化系统食道癌胃癌5人，肝癌2人，骨癌脑癌各1人，——且受害者明显有年轻化趋势，年龄最小的不幸者只有13岁。

公路建成通车后，必然带来大量的扬尘、噪声和汽车尾气排放污染，加上已存在多年的水体污染，居民即将生活在水和空气的全面立体污染中，除了拆迁他处，实在没有更好的办法。

这些由环境污染导致的恶性疾病，不但给患者带来巨大的病痛，还给家属亲人带来沉重的经济负担和无法弥补的情感伤害。居民们时时刻刻为自己的健康担忧，情绪低落，人心慌慌，不知哪一天疾病不幸又降临到哪个人头上，降临到哪个家庭中。纸面冰冷的数字后面是一个个逝去的鲜活生命，是一个个饱受折磨和不幸的家庭，失去的是民心，失去的是党和政府的威信。

基于上述实际情况，为居民生命健康大声呼吁，为子孙后代生存发展泣血请命，所以特此致政府请愿书，恳请政府考虑给予本组居民整体拆迁。


我们相信，有良知的政府、有党性的共产党人，不会置群众的切身利益、生存不闻不顾，漠视人民的人身健康和生命安全，一定会倾听群众的合理呼声，顺应人民的切实恳求，造福一方造福子孙后代。

此致
敬礼

XX村XX组全体居民（签字）
