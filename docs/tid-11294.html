一，引子
房价会怎么走？
看涨派认为，中国城镇化还没有完成，潜在的庞大进城人口所带来的需求足以支撑房价长期上涨；看跌派认为，房奴不堪重负，房价与收入比例严重不协调，已经到了拐点。
看涨派的说法看上去很美，问题是人口进城的钱谁来掏。如果政府掏，政府的收入来源于税收；如果是民众掏，有工作才有收入，这都依赖经济保持长期稳定增长。
看跌派的说法切中时弊，商品房已明显具有投资属性，而当年海南调控房地产（挤压房地产泡沫）的事实说明，房价走势与资本是否放水息息相关，只要把银行注入房地产的资本水龙头一关，房价立即应声下跌。
看涨看跌，其实是经济增长机制这枚硬币的两面。短期的一面是，根据过去的经济增长经验，要实现目标GDP，资本放水必须高于目标GDP5-6个百分点，这种GDP高速增长模式是否继续有效？长期的另一面是，“经济运行在合理区间” 这种犹抱琵琶半遮面的用词下，未来的GDP会以多快速度增长？
  
二，主要内容
通常说，投资、出口和消费是拉动我国经济增长的三驾马车。这种说法通俗易懂，但不免误导，很容易把投资仅仅理解成资本放水，过去多年的资本放水强化了这种认识，甚把中国经济简单地看成投资驱动性增长，这是只看表象的一派胡言。
投入能够持续，是因为有回报，不然就是坏账。一遇到经济不景气，就鼓吹资本放水，仿佛只要铁公基一上马，就能立竿见影拉动经济，甚至极端地说，一批人挖坑，一批人填坑，付给这两批人工资就创造GDP。这样的资本放水是会拉高GDP，但这种GDP根本就是GP，因为不能促进生产力发展，从而无法增加社会财富，只会有两种结果，要么是金圆券的崩溃，要么是日本失去的20年。
从经济意义的供求来看，投资是供给的一部分内容，不是全部；消费是内部需求，而出口外部是需求，所以三驾马车的说法不如改成这样：供给拉动和需求推动是促进我国经济增长的两个车轮。
先说供给。
供给促进GDP分两块，一块是要素投入增长，包括固定资本（通常说的资本放水）、人力资本和劳力，类似物理堆积的量变增长；另一块是上述要素（即全要素）的单位生产率增长，投入1块钱的资本要素产生大于1块钱的综合收益，类似化学反应的质变增长。
为了行文简单，用一个核算方程说明过去这两块各自对GDP的贡献，看看哪些改革促进了经济增长，未来什么样的重要经济制度、政策和社会问题会影响经济增长，增长路径是怎么样？
生产核算过程太复杂，不做详细介绍，这里只交代核算方程涉及到的英文代号：
C =Capital = 固定资本；
L =Labor = 简单劳动力，以前帖子里说的“人手”，比如农民进城成了农民工；
H =Human Resource = 人力资本，即受过良好教育的劳动力，以前帖子里说的“人脑”，比如科研和管理人员；
TFP = TotalFactor Productivity = 全要素生产率，即C、L、H等各要素生产率。
最后说需求。
继续扩大出口是否可能，能持续多久？在外部需求疲软促使出口停滞的情况下（这不可避免），国内消费是否有足够的动力推动经济增长？
虽然预测经济增长是件不讨好的事，被打脸的概率很高，但预测可以加深对我国经济增长机制的理解，所以这个帖子的重点不在于预测结果是否正确，而放在讨论影响经济发展的主要因素上，并从这些主要因素出发，试着理解三中公报这个纲领性文件为什么会是我们见到论述，哪些领域会在未来10年发生转折型变化。
至于房价走势，如果帖子中的逻辑成立，结论显而易见。

关于GDP的说明：GDP数据来自国家统计局，经过统计局事后修正，即使这样，也并不精确，而只是一种合理的估算。国外经常强调我国夸大GDP，但同样有理由怀疑GDP被严重低估，原因有2个，一是大量的现金交易经济活动产值被排除在外，以服务业为甚，比如菜市场，可以称为统计外的菜市场经济；另一个是统计数据不反映质量的提高，比如家电、汽车、电子消费品等制造业部门产品质量不断提高的同时，产品价格明显下降，统计数据低估了当期价格指数。

建国后的1952-1957年，我国迅速稳定了货币，结束了数十年的恶性通货膨胀，农业集体化和工商业国有化国营化之后，恢复电力、刺激棉花这类商业农产品种植和贸易等简单措施就使产出大规模增长，这是因为战争导致的闲置资本和劳动力恢复了活力。从战争破坏转向生产建设促进了生产力发展，全要素生产率TFP实现了令人印象深刻的增长，这一阶段GDP增长的3/4归功于全要素生产率TFP增，反映在居民生活上就是终于吃上饭了。
接下来的1957-1978年间，以WG为标志分为两个阶段。1957-1965年间经历了大跃进、三年自然灾害、苏联取消技术援助，同时又因为禁止外商投资，而美国又对我国实行贸易禁运，我国只能获得最有限的技术，不得不自力更生，这都使政府投资成为最主要的经济发展形式。通过剪刀差提取的利润成为新的投入资本，投资在这一阶段占GDP的93%。随后的1966-1978年间，文革使经济决策处于瘫痪状态，投资占GDP的65%。
与建国初对比，高速且不断加快的资本投入代替了全要素生产率TFP增长，成为GDP增长的主要贡献力量。全要素生产率TFP的负增长导致还停留在吃饭阶段，人均消费每年只略微上升2%。大部分的消费发生在城市，或者说被政府主导的分配制耗费尽了，而大部分农村居民的人均消费在改开前20年里几乎没有任何改善。
从1978年至今，改革经历了4个主要时期：1979-1984年的第一阶段，开放对外贸易，实施家庭联产承包，放开乡、村市场和很多消费品市场；1985-1988年的第二阶段，放开工业品销售市场，乡镇企业的繁荣支持了经济高增长；92年286讲话后的第三阶段，建立社会主义市场经济，更多市场化改革；1990年代中后期至今的第四阶段，外商直接投资FDI快速增长，加入WTO后取消了大多数国内和国际贸易限制，政府在工业部门的所有权和控制权大幅度减少。
在整个改革阶段，国有和民营部门将更大份额的资源从消费转向投资，高额储蓄率正是这种转变的注解。储蓄和投资带来了更高的GDP增长，但资本贡献的上升并不独立于生产率的上升而发生的，全要素生产率TFP以每年3.8%的速度增长，从1978年以后的30年里占据着全部增长的40%。全要素生产率TFP增长的结果是消费品不断丰富，从吃饭进入吃肉阶段，越来越多的人口加入到致富行列中，最典型的注解就是1亿农民工进城打工和大学以上学历在劳动者中的比例不断提高。高速增长的全要素生产率TFP反过来又要求更高增长的投资，所以，资本增长本身就是全要素生产率TFP增长加速的结果之一。我国经济发展首先是由于要素投入的增长，特别是资本投入的增长，进而推动全要素生产率增长的一个过程。从这点来看，未来我国经济持续高速增长的关键取决于全要素持续增长的能力。
先前加速的全要素生产率TFP增长是一系列改革的结果，这些改革包括实施家庭联产承包责任制、从计划转向市场、取消大多数国内和国际贸易限制、以及伴随着从中央计划想市场经济转变一起发生的许多根本性变革。这种转变促使我国大步向市场经济迈进，但未来与过去的那种放开工业品在市场上出售，或者促进FDI增长的影响不同，特别是农业部门不可能对GDP增长造成很大影响，因为农业占GDP的比重已经不足10%，农业人口向工业、服务业转移依旧很重要，前提是第二、第三产业能够创造出新的工作机会。因此，在接下来的10年里，未来的生产率和GDP增长主要取决于工业、服务业的产出和生产率的发展变化，这些部门会带来新的资本和全要素生产率TFP增长模式吗？

四，未来的增长动力
1，储蓄和投资率依然会很高吗？
虽然我国深受外资青睐，但固定资本年增量的大部分还是来自国内投资，而大多数国内投资又由国内储蓄融资。因此，即使假设有稳固的投资机会，未来10年的国内储蓄水平将在很大程度上决定投资步伐和资本存量增长率。
国内储蓄有3个来源，占总量1/2多的居民个人储蓄、约1/3企业储蓄以及约15%的政府储蓄。居民个人储蓄在整个改开时期都保持在高位，1/4的可支配收入成为储蓄，约为GDP的1/6。但近年来已经有了很大的变化，储蓄越来越多被用于消费，房子和车子是目前的大头，而随着二次分配向保障民生倾斜，为养老、医疗、教育而储蓄的意愿更弱，所以居民个人储蓄会逐年降低。企业储蓄来自利润，规模以上工业企业利润集中度高，有利于储蓄的提高，占GDP的比重不会下降很多。我国政府是全球最富的政府，大规模的财政盈余保证了政府储蓄水平，但加大力度改善收入分配的补贴和福利政策会减少储蓄。
所以，储蓄率会在未来10年逐年、缓步降低，投资也会因此减少，但2013年各类国内储蓄约占GDP的50%，经济发展不可能因为储蓄不足而受到太大的限制。正是因为这种经济结构的变化趋势，“用好增量，盘活存量”的提法在三中之后应时出笼。
2，劳动力和人力资本的增长
人口政策帖已经做过相关内容的详细说明。
我国的劳动力人口将在2015年达到顶峰，并会伴随着人口老龄化逐年下降，但没必要担忧这种趋势对经济增长的拖累，原因有2个。
首先，农村人口向城市转移会促进劳动生产率的提高，这种转移占改开以来产出增长的绝大部分。近年来的调查数据显示，城市劳动力产出是相同教育水平农村劳动力的2倍多，而且因为城市规模经济的存在，劳动力从农村转移到城市，不仅能增加移民自身的生产率，同时也能够增加现有城市劳动力的生产率。人口再分布能抵消劳动力缓慢增长对经济扩张带来的负面影响，提高未来的增长，所以三中公报提出一系列改善人口转移政策，包括新型城镇化下有限制放开农村人口落户城市、社保账户随人口流动在省际间转移等。但农村人口在城市安家落户有一个前提，城市必须提供足够多的工作岗位吸纳农村劳动力，促进就业增长的政策和制度安排理所当然成为下一步改革的重点，包括清理和下放行政审批权、开放对民营企业行业的限制、鼓励发展第三产业和创办小微企业等。
另外，改善人力资本将提高劳动生产率。据人口普查数据预测，初中毕业生将继续成为我国工人的最大组成部分，稳定在总量1/3上下，而高中以上毕业生的比重在逐年上升，到2025年占35%以上，小学以下人口降到只有1/4。扩招政策一直为人诟病，但并非我国独创，日本在1955-1975年之间实行类似的招生模式，这种模式有一个好处，短期之内迅速拉升人口素质，而且将对下一代带来积极影响。
所以教育领域改革会很有看头，越来越注重实效性。名不副实的研究型大学转为切合实际的应用型大学已经提上日程，近10年升格的大学可能会“倒退”为高等技校；在职业教育受到前所未有重视的情况下，中等职校会走两个方向，一是中等职校联合办学，走一专多通的路子，另一条是和同等专业方向的高等职校合并。英语退出高考也是出于实用性目的，在向农村生源倾斜提高录取率的同时，却变相分流学生，英语基础好的学生流向重点名牌大学，基础不好的学生进入技校通道。
3，两种GDP增长率的假设
假设未来10年GDP每年增长6%和9%，使用生产核算方法进行回归分析，得到2个供给拉动结果，即新的资本和全要素生产率TFP增长模式。
模式一，假设GDP年增长率为6%

模式二，假设GDP年增长率为9%

http://img16.poco.cn/mypoco/myphoto/20140706/22/1737968742014070622202802.png?806x393_130
这2张图不是预测，而是分析的起点。
根据对两种GDP和投资增长的假设，固定资本在未来10年对GDP增长的贡献在2.4%-3.9%之间变化，也就是说，如果增长完全依赖劳动（包括教育）和固定投资，那么GDP的增长率不会超过4.7%，这仅仅是过去10年的一半左右。
因此，全要素生产率TFP显得极为重要。为了使9%的增长率成为可能，每年的全要素生产率TFP必须保持在4%以上，这一增长率高于过去10年的3.8%，也高于2005年之前10年的3.1%。如此高的全要素生产率TFP增长率不大可能实现，仅仅保持近年来的全要素生产率TFP已经是一大挑战，在储蓄率下降的情况下，也不可能用资本投资来弥补。所以，GDP年增长率在未来10年不可能达到9%。
过去5年GDP的平均增长率已经徘徊在8%，近3年在7.7%左右，在经济运行在合理区间的措辞下，有理由相信，GDP还会下行。此外，所有的假设都建立在内外部和平环境的基础上，在局势日益严峻的条件下，必定有一部分资源会投入到安全保障中沉淀下来，所以，连续保持6%的GDP增长率会更加困难，未来10年的增长率最有可能在5%-7%之间震荡。
其他国家的增长过程提供了一个参照，日本以平均8%的速度增长了20年，在人均购买力水平达到13800美元之后的1971年开始减速；韩国以平均8%的速度增长了30年，在人均购买力水平达到13370美元之后的1992年开始减速，这两个国家减速后再也没有恢复到过去的水平，增长率略微强于5%。但我国人口基数大，城镇化率还有潜力可挖，有庞大的国内市场作为后盾，增长表现会比日本、韩国更好。
4，需求推动
我国出口依存度一度（出口占GDP的比重）达到35%，给人过度依赖国外需求的印象，但扣除加工贸易中的进口额，出口净附加值约占GDP的17%-20%，这一比率与大部分经济体一致。但我国是个制造业大国，进口国家不可能长期忍受为适应大规模进口增加而快速调整本国经济结构，所以不仅发达国家针对我国的反倾销调查逐年增加，发展中国家也越来越多加入制裁行列。
如果继续把出口作为支撑总需求的一个主要因素，要使未来10年GDP以每年9%的速度增长，出口增长率必须每年保持在12%左右，这与近些年7%左右的出口有相当差距；而年出口增长率连续10年保持7%，约占9%GDP增长的10%，这显然与出口净附加值约占GDP的17%-20%不符，所以GDP9%的增长率不可持续，出口率年增长7%也不可持续。
当出口在总需求增量比例中下降时，不得不寻找办法刺激国内需求。国内需求分两类，一类是家庭消费，常年不到GDP的40%，另一个类是政府消费（即政府支出），以铁公基为主。
三中公报指出了刺激的方向，针对家庭消费，财政补贴社保、医保和教育，让民众大胆花钱；针对政府消费，提出加大开放边贸力度，铁公基配套势在必行。
