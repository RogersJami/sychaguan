西藏这块地方如何？

西藏这地方， 海拔太高。 热量条件不足，降水也不稳定，形成稳定的生态系统不容易。 又是三条母亲河源头。 虽然地方也大，还是弄成保护区好点。 那地方太脆弱。

新疆不一样， 新疆热量光照极好。 就是缺水。再恶化也恶化不到哪去，因为已经干到极点了。 如果能把区域外的水弄到新疆发展农业， 那真是前途无量。

加州就是个成功的例子。 加州的热量条件也非常好，但缺水。 美帝把落基山的雪水弄来加州灌溉。 加州成了非常非常牛X的州。 加州种的辣椒都比其它地方大一号。
