每周一卦是否速度太快？

从一个读者角度说，我觉得每周一卦的速度太快了，易经中每一卦信息量都很大，且晦涩难懂，要理解领悟很费时间。以前，忙总的一卦还没真正理解，新的一卦就出来了，为了不错​过新卦的讨论，只能连爬带踹的跟上去，总有力不能及的感觉。从个人角度，两周一卦的速度会比较好，这样可以把一卦的内容多看几遍，能多学点东西。

另一方面，解析易经绝非易事，要做到自圆其说恐怕要查阅不少资料，忙总对卦形义理解释都很清楚，想必花了不少功夫。所以个人建议忙总适当放慢速度，保重身体为要。如果空闲​时间很多，可以考虑写一些休闲类的文章，如菜谱、个人经历等。

最后，很高兴能再次看到忙总的文章，祝身体健康！
