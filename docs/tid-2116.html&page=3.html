蚁族
    一个朋友的兄弟，车祸死了。
    在广东打着工，一个小小的运货司机。那天，货车在高速出了问题，被迫停在路边，正检查着，一辆货车据说是想超车，从临时车道就串了出来，直愣愣的撞在人上，再把停着的货车一把撞出了护栏。

    这消息青天霹雳般传回，从上午9点传到了下午4点。朋友听到了，匆匆的上了往广东的车。这事儿，可不小，朋友家也浩浩荡荡的来了二三十人。到了局子，负责的交警批头就说，你兄弟尾灯没开，车轴脱出，停靠偏外，得负主要责任。还拿出了现场的照片，不但这些，对方的车歪着横躺在中间的路上。朋友这些直接就闷了，既没拍照，也没录音，愣愣的看着，不相信这个事实。

     晚上，躺着的时候，终于想到不对了。这出事的兄弟，平时最老实，搭他的便车时对交通规则可是记得清清楚楚，对安全问题可是从来不大意啊。感谢现在无所不在的手机摄像头，在当地网络论坛翻啊翻，总算找到了事故后他人拍摄的视频。看了一眼，眼泪就不能控制的流了下来。尾灯在那平静的一闪接着一闪，车轴好好的在原处呆着，对方的车横躺着，但是都有半个车头在临时车道了。而那个给他们出示证据的交警，就在一旁一笔一划的记录着。
    第二次再去，为了不拖时间，外地的人也拖不起，直接出示了证据，那个交警也只好开始重新整理了措辞。赔偿问题总算能开始说些了，一次次看交通事故的折磨总算没白费。

    逝者灯灰，生者如蚁。收拾房间的时候，3平方米的小屋，放了衣服后就是张床；不小心登陆进他的qq空间，发现他在絮叨着生活的劳累；出发的时候，他的母亲还担心着孩子会不会判刑，回来的时候，只能轻轻的告诉她可能无期，永远没法再见了···
     卑微者如尘土
