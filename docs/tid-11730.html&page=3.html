囧，发重了
找了一圈删除不了，那就借这层楼说点我知道的。
我曾经搞过文艺研究，因为天赋的原因，后来逐步淡出了，但还是保留了一圈搞文艺的朋友。其中一些大牛，我接触过后的感觉，就是相比之下现在市面上所谓的大师，都不过是一些横行的二逼。这个不多谈，也许我是站在朋友的立场失去了客观也不一定。
最近这圈人突然开始一个话题，这个话题是如此突兀，我反正是插不上话。这个话题的中心，是，如何将儒家和马列主义毛泽东思想统一。
目前讨论的方向，是将二者在更高层次上进行统一。目前主要说的是儒家。君臣纲常伦理视为农业时代的经典生产关系，以古代中国生产力作为当时已知世界的顶点，儒家纲常理论恰好就是最适应生产力的生产关系、社会结构的理论体系，所以可以维持系统运作两千年。与之应征在于黑格尔批中国两千年没有变化，这是显而易见，系统达到顶点之后没有外力介入，显然不再具备再进化的内在动力。
这个话题很牛逼，在于其讨论做成理论系统之后，就是前后统一两千多年的思想体系不中断，连清末鼎革废科举、行西学以及本朝那些有洋味又多少有点走样的舶来品，都一并包含其中了。是真正意义上的继往开来，继往圣绝学。
比较有意思的是，他们开始话题的时间，这个刚好是董事长提到的前后三十年统一的问题，刚好是在同一时间点。我不太相信这帮人会接受官方指导，我看中宣部，未必有那个眼光来搞统一两千年思想，就算有这个心，也肯定没有这个学术水平，就算有这个水平，也没那个能力指挥得动这些懒散文人。

大数据对生产领域的影响远比消费领域要大，可以这么理解吗？

“中石化成品油零售板块将来不是一个卖油的概念，而是一个综合服务商的概念。我们有3万座加油站、2.4万座便利店和8000万客户，将来要把服务8000万客户作为重点，要负责他们吃穿用住行的消费。任何一个厂商、消费品制造商都想拉这8000万的客户，为此会上我们平台销售他们的产品。”傅成玉说。
　　在他看来，未来中石化的8000万客户甚至可以不花钱加油，因为所有消费都可以折成油或现金。广义来看，8000万客户所要求服务的任何一个领域都可以通过这个平台去做到，这就是平台的意义。
　　“将来成品油销售可能成为副业，利润、销售额的成长空间都很大。”傅成玉表示，过去，非油品板块的盈利、销售额占比都非常低，未来都会有20多倍的空间。如果把电商、移动互联网、大数据联到一起，将产生一种新的业态和新的商业模式。

http://finance.sina.com.cn/360desktop/chanjing/gsnews/20140825/201720113849.shtml

我三月知道中石化核心业务以后不是卖油了，这个话是整合中石化大数据的干部说的。傅成玉说的话还离我们对中石化的衍生业务的理解很远。我不是说我们比谁懂，因为我们都和在蒸汽机发明时代的多数人一样，只能从我们的生活实践去推论，大数据能改变我们什么。实际，仅仅从生产力整合而言，大数据已经超过我对现有生产力进化模型的任何理解。所以我说，我实际操作不考虑半年后的事情。其他变观察边在小圈子讨论。

还有你关心生产，我实际更关心这一轮生产力升级下的新技术革命对人类生命科学的影响。这里一些话题，是圈子里禁止我公开讨论的。原因就是，多数人对未知本能的恐惧与拒绝。
