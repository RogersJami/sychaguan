这次景嘉微没操作，是严格操作纪律：既回避有政策类风险的股票池。这里林左鸣的中航集团，就是其中之一。但是，这不是说他没有实质利好，而是利好太明确。基于国内利益再分配原则，这块肥肉肯定有个再分配过程。从公开的新闻披露中运20需求国内近千架。而对应的战斗机爆量，只多不少。具体数字只多不能说。战斗机爆量，到这个级别就是我说的，如果为真那么对应南海，台海乃至亚太格局会出现性质变化。记得去年中美战略对话，美国要求我们明确承诺诺干年内不对越南印度动手。而，我们回应是我们要的特定年限得不到，我们自己拿。美帝舞剑背后我们自有对应。这里我对茶馆网友说过，这里政策风向变化之快，普通人难以揣测。今天的朋友，明天的对手，反之亦然，普通人在这里赌政治风向，短期十赌九输，中长期每赌必输。安心自己小钱钱既可，和股市等标志性阳线那样等那个明确的结果出来锦上添花也不迟。

回到景嘉微，他的明确在于他是中航GPU的绝对垄断供应商，具体份额目前百分之七十以上只多不少。而且基于中航的增量需求，还有潜在的外销需求。（比如我提到得沙特20采购谈判）他们的增长前景，确定且可观。而，回到上一年，景嘉微销售2.6亿中毛利2.2亿。如果撇开特定的开支，基于有关产品一代前期成本基本摊入前一阶段的空军增量。实际这部分产品成本，折旧后几近于0。然后随着内外销需求，以及应对新一代战机的研发推进。一代产品销售增量，与二代产品升级，以及随后带来的衍生服务，对于公司未来发展前景想象空间很大。这里更不要提，对应通用航空开放领域，这一块的中航公开的信息提示的增量是前面提到的20及其他军用飞机两者总和的增量还将倍之。而，中航系目前的困难是暂时的。当改再分配的归于过去，增量需求有其特定时间表，产销自以及利润跟上来，到时候无论持有什么核心产品的中航系股票都会有一个重新焕发生命力的时代。然后就是，耐心信心与细心。大时代下，无非如此。
