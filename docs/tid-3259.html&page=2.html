震撼揭秘：


基辅市舍夫琴科国立大学国际关系学院国际关系和对外政策教研室教授佩列佩利查认为，中国在“瓦良格”方面对乌克兰没有任何义务，哪怕最微不足道的义务。他解释称，航母当时是通过离岸贸易的方式出售的，因此中方不对乌方承担任何义务，它实质上不是国家间协议。况且，当时乌方出售的不是成型航母，而只是船体。中国将其建造完工，改装为教学航母，对乌克兰来说不会产生任何国际后果。不过可以确定的是，上世纪90年代乌克兰领导人如此不负责任地出售航母显然缺乏远见。但是也有理由认为，乌克兰当年实际上遇到了极其难得的机遇，完全能够从已经绝望的航母项目中(通过与中国的合作)获利。相比之下，同样未完工的“乌克兰”号导弹巡洋舰至今还停泊在尼古拉耶夫市黑海造船厂内无人问津，只有俄罗斯表示准备无偿接收。当年美国充分利用中国拖运“瓦良格”号航母的问题，当作索回悍然侵犯中国领空的EP-3侦察机和机组人员的筹码，和中国讨价还价。问题解决之后，土耳其在美国的授意下立即允许“瓦良格”号过境进入地中海。这就是最为鲜明的例证，表明美国当时对中国建造航母舰队的计划实际上并不是特别担心，他们根本不相信中国能够胜任如此重大的任务，因为中国当时既没有经验，也没有相关技术，最重要的是，还没有舰载歼击机，而没有飞机，航母就失去了任何意义。

乌专家指出，后来还是乌克兰帮助了中国，2005年向中国出售了苏联解体后遗留下来的苏-33舰载机的验证机T-10K，中国正是在此飞机基础上成功研制出了首型舰载歼击机歼-15。中国军方还参观乌克兰克里米亚半岛上的尼特卡训练设施，研究舰载航空兵飞行员地面模拟训练方法。有消息称，中国在建造完工“瓦良格”航母时聘请了乌克兰黑海造船厂和乌克兰特种出口公司的专家作为顾问。尼古拉耶夫市“朝霞-机械设计”燃气涡轮机制造科研生产综合体为中国航母提供了动力装置。也就是说，乌克兰不仅了解“瓦良格”号航母在卖给中国后所发生的一切，还切实参与了航母建造完工和改装全过程。所有这些都有理由认为，中国在完全自主建造国产航母时，同样离不开乌克兰的参与。尽管乌中军事技术合作有自己的特点，即很少采用购买某种武器和军事装备样品的形式，更多的则是建议转让产品生产技术。乌克兰应当重视中国这个非常有前景的伙伴，发展在航母建造领域的商业联系，互利互惠的合作方案总是能够找到，毕竟这种合作的基础已经奠定。或许，根据中方的倡议，乌当局在中国航母方面的合作只是一些互不相关的合同，没有深思熟虑的多阶段深度合作计划。
