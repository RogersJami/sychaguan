国民政府和人民政府的推进，不过都是继承了这一倡导，当然选字都是根据一定规则和沿革的。

3D打印是怎么回事？据说很神奇的样子。

根据工件三维模型，将粉末和结合剂逐层打印，升温后固定，得到工件。
你可以想象成一本书，然后每页形状不一样，叠到一起后就可以做出任意的三维形状。

好处是制造快，不需要任何其它模具和加工机床就可以制造出任意形状的工件。
缺点是只能单件或小批量生产，目前用途很有限。
