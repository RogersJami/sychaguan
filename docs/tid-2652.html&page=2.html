西伊 发表于 2012-6-29 17:03:50

两千多年前的经典，其具体内容不见得每一条都适用于当代。甚至有不少东西，恐怕从来就只是想像。
不过，如果仔细捉摸文字背后的价值观，常常会发现，有好多东西似乎一直没有变。
说得美妙、或浪漫一点，那叫得以传承的文化。说得现实一点，那正是些适合于这片土地上的人们的生活方式、生存方式的东西，有正面的，也有……不那么正面的。
反过来，也能看出，两千多年了，这片土地上的人们的生活方式、生存方式当中有着那么多变化不大的东西。研读论语，其真正的价值并不在于理解古人的思想或理想，而在于理解我们自己。
如果能更进一步，以经典中传递的价值观为正面或反面的参考，结合现实重新反思自身的价值观和为人处事的方式，也许会更有收获。不过，一定要谨防走火入魔{:soso_e104:}
