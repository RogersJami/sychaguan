中国有没有经历过奴隶社会——封建社会？

我同意前面的看法，西方意义上的奴隶社会和封建社会从来没有成为中国社会的普遍现实.

希腊罗马的奴隶社会是这样一种复杂精密的社会制度，可能不亚于今天的工业社会下的资本主义。捕获其他部落的成员，使其为自己部落服务，这是史前普遍发生过的事情，但这种简单的行为实际上不能保证建立奴隶社会。试想新成员开始地位低下，慢慢结婚生育，下一代从婴儿时期开始，被周围接受是人性的趋势，这样就基本融入了，没有制度或信仰来强化这种界限的话，简单的抓奴隶是无法持续让奴隶做奴隶的。

而为了确保这种界限，大部分的罗马奴隶是被禁止结婚生育的，这是保证真正的奴隶社会能够持续存在的前提，这样的制度安排能够让所有的希腊罗马公民都无需在日常生活上花心思，可以把全部精力用在公共辩论，政治活动，以及思考抽象问题上，才缔造了辉煌的希腊罗马文明。即使说西方文明那帮人在人种上甚至历史上都和古希腊罗马文明无关，也无法否认希腊罗马文明在各种抽象问题和发明创造上进行思考的深度和广度，是后世所难以企及的。欧洲人不是这个文明的直接继承者，但他们知道这个文明而成为其间接继承者，就和鲜卑部落学习并发扬了华夏文明一样。

奴隶社会的持续需要有以下几个条件

稳定的奴隶来源
稳定的奴隶买卖渠道
严密的奴隶培训、管理制度，以及严格的社会界限
这三条都可以在历史资料里找到明确的记载，这里就不多说了

最后，奴隶制度的破坏有以下几个基本原因
一是战争技术的发展，导致精英打仗的作法无以为继，必须依靠雇佣军或者异族，参加战争的人肯定会有政治上的要求，所以奴隶社会是没法让纯奴隶作战的。但依靠异族作战同样是难以稳定的，原因同上，罗马最终就像唐朝一样亡于雇佣打仗的异族。

下次有空再谈谈奴隶社会到封建社会的有趣转折。
