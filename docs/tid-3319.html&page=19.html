衡量泡沫一般用的比较多的指标是M2/GDP，以此为标准中国比美国泡沫大很多。

未必。中美情况不同，中国太依靠间接融资（银行），而美国依赖直接融资（主要是债券）。现在中国直接融资比例增大后，央行都说不能光看M2增长了。更相关的指标是社会总负债率/GDP。从那个指标可以看出中国整体杠杆率低于美国，但这四年增长太快，非常非常的危险。
