保障性住房都取得这么大成就了

看完忙总的损卦文章，确实觉得和年初感觉不太一样。那时候忙总对保障房资金募集还比较乐观。特别是对北京的保障房建设比较认可。今天读到这篇文章觉得真是发凉。这个描述的是帝都吗？

据北京市住建委的统计显示，2007—2011年，全市累计开工建设、收购各类保障性住房70余万套，保障房新开工面积从“十一五”初期的115万平方米，增加到2011年的1660万平方米，新开工套数占住房新开工套数比重由5.8%增至60%以上，初步构建成“多层次、多渠道、多方式”的基本住房保障制度，并通过各类保障性安居工程解决了近50万户中低收入家庭住房困难问题。

二毛 发表于 2012-9-26 18:55:42

根据财政部二○一二年一月十八日的财综5号文件《关于切实做好2012年保障性安居工程财政资金安排等相关工作的通知》，一般情况下，地方城市保障住房资金组成基本是地方卖地钱占30％，地方财政性投入10％，中央转移支付和财政投入10％，剩下50％银行抵押贷款。卖地好的的地方，保障性住房资金筹集容易一点，当然成就就大一点。

建议把所有省会城市的关于保障性住房资金问题的内容搜索一下，资金到位率惨不忍睹，钱不到位，吹什么都是假的。

二毛 发表于 2012-9-26 19:15:26

根据住建部数据，完成今年计划的1000万套保障房任务对资金的总需求约为14920亿元，包括中央和地方政府需要支付的资金6520亿元，以及需要通过贴息等方式吸引社会资金参与8400亿元。目前来看，包括中央财政补助、地方土地出让金和公积金增值收益提取部分、中央代地方发债、国家开发银行保障性住房贷款在内，政府方面较为明确的资金来源有3730亿元，各级地方政府还需要通过其他方式，如地方财政预算资金、融资平台贷款和发行平台企业债等筹集大约2790亿元资金，但是目前没有完全落实，同时地方政府土地出让金收益锐减，资金筹集更加捉襟见肘。而8400亿元社会资金更是没有把握。

       此外土地问题也是大头，2011年开工建设的1000万套保障性住房中，除400万套棚户区改造房不用另行征地外，剩余600万套保障房都需要地方政府无偿划拨土地。这对地方财政收入将造成双重影响，一方面土地收入锐减，另一方面无偿划拨的土地还需要支付大量征地和土地平整费用。按平均每套60平方米、3倍容积率、每平方米土地费用1600元估算，政府为提供这些土地需支付征地、拆迁和平整费用1920亿元。　　

       目前情况看，地方政府可能的筹资来源包括5个方面，中央代地方发行的地方债（2010和2011年代发债规模均为2000亿元）、地方土地出让金净收益（不低于10％）、专为支持保障房建设而保留的保障性安居工程建设融资平台、地方住房公积金贷款收益、地方政府财政预算资金。社会和市场融资渠道包括，国家开发银行中长期贷款、商业银行贷款、保险公司贷款、社保基金贷款和社会机构直接投资等。

       但是目前事实证明上述融资渠道并不能够完全支撑起现有保障性住房的资金缺口。

       所以住建部建议大力发展公租房的房地产投资信托基金（REITs）来解无米之炊。

我的一点题外话：
说到医院的人满为患，我觉得国家应该考虑大规模培训和推广中医，从我个人的生活经验看中医的用药价格比西医低太多了，只不过好中医培养困难，估计比好西医培养还困难得多。

从患者角度讲，我觉得很多人看病都有点过了，比如普通的感冒，其实自己吃点药，休息一周基本差不多，或者去正规小门诊或社区门诊就行，可很多人还是愿意去大医院看，觉得保险安全，所以大医院永远人满为患。
还有一些特殊疾病，比如癌症，说的不好听，这种基本病基本就是死缓，大医院都肯定要你开刀啥的，我觉得如果得了这种病，如果个人心理素质还可以的话，就干脆愉快的享受剩余的人生，不要把剩余的时光都花在医院的手术台上，那样做的结果就是个人身体和心理受伤极大，家属心理和身体也很疲劳痛苦。

我们家现在对于一般感冒之类的病就是自己吃中成药，或者中药偏方，每次花费绝对不超100元，依靠认识的中医进行平常的保健和预防，同时和家里老人达成基本共识：如果得了如癌症类的绝症，不去医院开刀受罪，安安心心过完余生就可。

所以我的医疗观念总结起来就是：平时多保健预防，用中医的理论调养，多养心养身，要有苦中作乐的精神。
