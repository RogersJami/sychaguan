所有事情，历史上都能找到他的表兄弟或堂兄弟。

帮忙总找个表兄弟：
中国古代正税也就是地税一般归中央政府，而杂税（如人头税、各种摊牌）则供地方政府支出，当然经手人的各种陋规也从附着在这些税的征收过程中。有很多人认为，类似摊丁入亩（包括一条鞭法和唐朝的两税法）可以减轻农民的负担，可以减少各种陋规。其实这是一种天真的想法，因为将各种杂税合并入征税本质上是中央政府钱不够用后，侵夺地方政府的财权，虽然杂税往往伴随着大量陋规，但这不能否认杂税本身的合理性，毕竟地方政府也要运转，也有很多公共服务要提供，所以税外加费不可避免，正税虽然增加，杂税又会以其它的名目很快恢复。唐朝的两税法，明朝的一条鞭法，清朝的摊丁入亩都逃不过这个规律。虽然这种改革能够暂时缓解中央财政的压力，但是本质上这是一种加税行为，只不过中央自己立牌坊、让地方政府做婊子而已。百姓的负担反而是越来越重、不调整的话，整个统治的崩溃只是时间问题而已。

一孔之见，纯粹纸上谈兵，有用的大家可以讨论，没用的多骂骂，让我也学点东西：

要想避免这种责权不一的情况继续扩大，应该系统得调整税制。现在中央和地方的利益交换，基本上都是临时起意：某些政策实施不下去了，就搞个交换；放权放太多了，就给地方加责任。最后责权越来越不统一，地方上辛苦，中央累。要搞利益交换，就要中央先把计划放长远点，对每个权利都自己估价沽好，多点安排，想好可能的对等交换，在中国发展这么快点情况下，这么搞是有点困难，可总比总是临时起意的好，既然要搞利益交换，索性就搞个大范围的，别总是走一步看一步，最后整得面目全非，都不知道该怎么入手了；

中国国家那么大，情况那么复杂，在巩固中央领导地位的情况下，应该扩大点地方的独立性，把什么都集中到中央，把地方卡得死死的，让省委天天想着怎么跑部钱进，也不是长久之计
