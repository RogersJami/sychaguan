简单的幸福 - 简易酸辣汤

本帖最后由 思炎 于 2012-2-22 09:03 编辑

思炎很喜欢喝酸辣汤，在英国这么一个小碗（如图）的酸辣汤竟然要卖3个多英镑，甚至更多 {:soso_e103:}

http://atopi-59386768.atopi-wear.com/tbimg.php?mid=/bao/uploaded/i4/T12e5jXelBXXbxK1w9_102911.jpg_160x160.jpg所以，我就自己动手丰衣足食，虽然正宗的做不了，时常做做简易酸辣汤也能解燃眉之渴 {:soso__7848749051764303562_1:}


材料：

豆腐，冬笋（我这次用的是袋装笋尖），香菇，火腿（我这里用的是梅林牌的火腿午餐肉，也可用熟鸡肉丝、肉丝），尖辣椒、葱、鸡蛋。

*也可以加入其他材料，比方鱿鱼，虾仁之类的，做成海鲜酸辣汤。


调料：

高汤（可用太太乐高汤精，或者清水），醋，酱油，料酒，胡椒粉，盐，水淀粉，香油。

http://img2081.poco.cn/mypoco/myphoto_anonymous/20120222/08/0201202220056561925501675299_000.jpg   




做法：

1、将适量胡椒粉倒入小碗内，加适量醋将胡椒粉冲开备用。


2、将豆腐切成丁状，最好过水焯一下。 笋尖、火腿午餐肉切成丝状；香菇泡发洗净，切成丝；尖辣椒、葱花切碎，鸡蛋打散备用。

3. 将锅置大火上，倒入高汤（或清水），烧开后先放入豆腐略煮一下，然后放入尖辣椒碎、笋尖丝，香菇丝，火腿午餐肉丝，待煮开后加入酱油，料酒，盐调色调味，

4. 用水淀粉勾芡。

5、缓缓均匀得将蛋液撒入锅中，然后加入（1）冲开的胡椒粉和醋，淋入香油再撒上一些葱花即可 http://www.fengyanforum.org/forum/images/smilies/chanzui.gif






因为个人不喜欢放太多水淀粉，因此汤看起来不太稠；做时没有刻意追求刀工，所以，还是比较粗糙的http://www.fengyanforum.org/forum/images/smilies/shy2.gif

但味道不错，欢迎大家伙指正，切磋、分享酸辣汤的做法与各种心得 http://www.fengyanforum.org/forum/images/smilies/Erainbow.gif
