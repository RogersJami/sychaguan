油茶

寒冷的早晨，回想起每年冬季的晚上，自己躺在家里烧得暖暖的炕上，两手支撑着下巴，看妈妈在火边一 ...

羊油炒油茶会不会膻？

我家住在回族聚集的镇上，回族几乎垄断了县城的饮食，回族以羊油包包子，因此我妹妹坚决不吃回民的包子，因为怕膻。

我对膻味一般敏感，还是有点敏感的。不过我和你不同，我吃羊的心、肝、肺、少量羊肚，但是不吃羊肠和羊脑羊头等。

我bf就只吃羊肉了。

这里面的美食，我吃过羊肉胡萝卜馅，味道不错。通常里面流出来的油都是红红的，被胡萝卜上色了。

过油肉我一直不知道是不是小酥肉。看着跟河南的小酥肉是一个东西，炒法也差不多，也是加蒜薹、木耳等来炒制。加蘑菇的我还真很少吃。不过说真的，我家乡那边的传说也说很多人是从山西洪洞迁来的，所以饮食习惯有些相似是可以理解的。

莜面栲栳栳没吃过。。。。

我家那边的最出名的传统美食，是清真的五香牛肉，现在做成真空包装都打入世博会了。

在山东这么长时间，吃过的最好吃的面食是锅贴和炉包。最喜欢这边的锅贴，其次是炉包。所谓的山东泰山烧饼，吃起来也就那回事。但是锅贴还是相当的绝的。不带馅的面食基本没有遇上多少好吃的，包括山东煎饼，都不太喜欢。
