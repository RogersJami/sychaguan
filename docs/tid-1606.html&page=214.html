幸运的活力东东 发表于 2012-5-19 15:57 static/image/common/back.gif
轮子是挺好玩的，记得美国关于某个小行星有过科学论文，然后老李同志出的书里面居然就有关于小行星的发功 ...
我当时看的是老李的第一本，印刷质量给我感觉也不好。要是我工作（做出版的）后看这种印刷的，估计连翻都给省略了。

我没有当作幽默来看，而是很严肃地来研究的。

对于高校教师、教授们痴迷那个的确从常识不容易解释，我估计是被其中某些他们自己熟悉的东西给迷惑了，比如搞物理的它就弄些现代物理的一些东西进去。再者可能跟身体素质比较差、身体不好有关系，身体不好的人相对更容易不由自主，不自信，身体素质比较有问题的，还有到处找“灵丹妙药”的心态，你找我推的这就碰上了。还有一些应该是有些其他想法和目的的，人数多到那个程度，随便忽悠忽悠就能达到名利了。

我多多少少接触过一些传统功法。仅仅从功法角度来看，当时就能看出轮子功它是善于乐于也主要是拼凑的，抄的。传下来的三教九流五花八门的很多功法，选一个坚持去锻炼一下的话，多多少少都会起些效果，这个功法对这些有效果，那个对那些有效果……它都给抄过来算自己的轮子功。结果就”效果很好“了。那时候忽悠我的就是跟我说练了后会有什么什么效果之类的，而且还说得很玄乎，说不同的人出不同的效果（其实因为是根本说不清）。

我当时看书就能看出极度不舒服，是因为拼凑的东西气场太杂乱。按照那个去练的话，气机会紊乱，如果加上态度专注地刻苦修炼，搞不好很快就能出神经病的，所以他们练出疯子的应该是体质比较敏感的。其实老李自己也是知道的。所以我在看书前被特地告知说这种神书需要一点点地看，不能一下子看完。所以当我说我一个小时全部看完并且很不舒服时候，我室友的姐姐就说“早就警告过了不能一口气看完。但从中看出你悟性的确很好，坚持学下去……”——仅仅凭这点，我就想不通那些大学教授们是怎么想的，读书都读到傻掉了：听说过什么书不能一口气看完么？
