兄台能不能多说点，IBM是什么路子？是怎样的商业模式？
我先说我的。我的行业是城市通卡，某省会城市，统 ...

大数据之所以被认为是革命性进步，以我个人肤浅的来看，主要是大数据不同于过去的海量数据，是因为大数据的数据具有完备性。也就是说我们已经搜集了关于用户的所有信息。现在的智能手机已经实现了不同用户网页的定点推送，每个人的看到的新闻网页都有所不同，基本是定制的。
大数据的魅力之处就在于，甚至连用户家里的亲人受伤了，孩子在学校被老师批评了都被完整收集。那么这些数据被分析之后，当你出门的时候就会在你经过的橱窗提供你所需要的任何服务。其革命性的进步体现在变现有的局部最优为全局最优。
以你所说的公交卡为例，这些数据如果被充分分析的话，我们就能够统计出顾客的出行流，替用户找出最佳的出行方案，恰当的在用户需要的时候，提供合适的公交线路，并且线路是动态可调的，可大幅提高公交运行的效率。如果与其他行业结合分析用户的行为的话，将产生更加广阔的应用前景。
但大数据的发展带来的问题是个人再也没有隐私可言，你的所有行为甚至包括心理活动都会被记录，那么基于这些数据预测个人的行为将成为可能。

引文中的一段话

统计学家们正争先恐后地为大数据开发新的工具。这些新的工具当然很重要，但它们只有在吸取而不是遗忘过去统计学精髓的基础上才能成功。
最后，我们再回头来看看大数据的四个基础信条。其一，如果简单地忽略掉那些反面的数据，比如Target的怀孕预测算法，那么我们很容易就会过高地估计算法的精确度。其二，如果我们在一个固定不变的环境里做预测，你可以认为因果关系不再重要。而当我们处在一个变化的世界中（例如流感趋势预测所遇到的那样），或者是我们自己就想要改变这个环境，这种想法就很危险了。其三，“N=所有”，以及采样偏差无关紧要，这些前提在绝大多数的实际情况下都是不成立的。最后，当数据里的假象远远超过真相的时候，还持有“数据足够大的时候，就可以自己说出结论了”这种观点，就显得过于天真了。
大数据已经到来，但它并没有带来新的真理。现在的挑战是要吸取统计学中老的教训，在比以前大得多的数据规模下去解决新的问题、获取新的答案。
