只做仰卧起坐不行。
首先得确定自己的意图，是健身还是减肥。
在卧室（没有专门健身房间）应该是选择3、4个动作，练30分钟，将全身肌肉都练到。
比如，仰卧起坐要躺在床边地面、将腿搭到床上，使腰和大腿成90度，然后做。如果没劲，可以先将手放在胸前，随着腹肌增强，将手放在脑前、耳旁、颅骨后。

chalet 发表于 2013-4-19 11:51:53

回jolery的  “跑步百利惟伤膝”，对膝盖的保护很重要。有种说法不要每天都跑，每周间隔跑个三次，不知道是否合理？

跑多久、怎么跑，主要看你的目的是什么。如果是简单的保持身体状况，我觉得每周跑上3次，不用太快，每次在30分钟以上，就可以达到最基本的运动量。这个指标只是一个基本参照：对长久不动的人来说，也需要不短的时间才能达到；而对经常户外运动的人来说，还达不到体力储备的目的。

其实伤膝这个问题，如果能注意保护，问题应该不大，主要是对那些长久没运动的人需要注意点，一开始多做腿部肌肉的锻炼，少跑一点，不做跳跃，穿好点的跑鞋，有个1-2个月的过渡期，应该就没什么大问题了。

我自己开始锻炼的时候，在2个月后发现左膝有点酸，究其原因，可能是之前2年踢球和人碰过，有点旧伤，加之穿了回力板鞋，还做不少的立定跳之类的动作。休息了一段时间之后，再跑就没有酸痛了。除了上述提到的预防措施之外，我还在吃硫酸葡糖胺。另外，有人说膝盖不好要多跑上坡路，我的感觉也是上坡跑起来挺舒服。仅供参考吧。
