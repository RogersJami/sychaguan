中国官方主导着手开发自有处理器架构

2012-04-25 22:09:56 5516 人阅读 编辑：雪花 [复制链接] [我要爆料]
中国官方在上个月启动了一个计划，目标是定义出国家级的处理器架构；如果计划成功，采用该标准架构的处理器将会成为所有寻求政府资金之专案必须具备的条件，例如政府相关机构的电脑与智能手机采购案。

据了解，至少有5种现存处理器架构正被考虑作为中国国家处理器的基础；该计划也将定义出中国自有指令级架构（ISA），或是将某个现有指令级架构进行扩充。中国工信部官员是在3月份召开此“中国国家指令级架构计划（China National Instruction Set Architecture initiative）”的首次会议，有来自约20个中国本地相关厂商团体的代表出席，包括通讯设备大厂华为（Huawei）与中兴（ZTE），以及数个学术单位。

上述计划是中国官方致力于建立自有标准的数个行动之一，目标是藉此取得自有智慧财产权（IP）、不用再支付权利金给外国业者；目前中国已经建立各种电子产品标准，从CD/DVD播放器到保全视讯系统。而中国主导的3G蜂巢式行动通讯标准 TD-SCDMA ，可说是到目前为止最引人瞩目的一个，虽然其结果有好有坏，但已让全球最大电信业者中国移动（China Mobile）取得足够市场动力；该公司已经开始进行中国4G行动通讯标准TD-LTE的实地测试。

中国开发自有处理器标准的想法已经酝酿多年，其龙芯（Longsoon/Godson）处理器的诞生是最明显的宣示；该款处理器大致上是采用MIPS架构。产业观察家对于中国最新自有处理器计划是否会成功，则是意见不一。

MIPS业务发展副总裁Robert Bismuth表示，中国自有处理器标准建立工作是至少需要花费数月时间才会有成果的：“但我确实认为会有成果。”他指出，龙芯处理器就真的已经运用在中国政府机关采购的系统中。但这也有可能是MIPS一厢情愿的看法──该公司因营收下滑，正传出打算出售；有一位要求匿名的中国厂商高层甚至指出：“中国虽有数个处理器核心是以MIPS 为基础，但MIPS最后还是会倒闭并出售其IP与专利。”

至于ARM核心则对部分想找较低价替代方案的中国电子业者来说太过昂贵，上述匿名中国厂商高层的公司曾取得ARM核心与PowerPC核心的授权，他表示：“我们不能只有ARM这个选项，所以PowerPC也有了个机会。”要取得ARM高阶Cortex A9核心授权，至少要500万美元；该位厂商高层表示，如果不选择ARM处理器与Android 平台的搭配，这个金额足够一个采用PowerPC处理器与Linux平台的平板装置开发案所需。

一位ARM高层则对于中国的处理器计划抱持怀疑态度：“我们当然会注意这个计划，这不是新的题目，已经被讨论了很多很多个月；”ARM总裁Tudor Brown在接受电子邮件采访时回覆：“我们了解中国的初衷是开发自有ISA，我们也将持续与关键人士进行会谈与合作，希望能有一个最佳解决方案。”

Brown补充指出：“关键议题并非ISA本身，而是围绕着任何一种ISA的生态系统。”他特别强调了ARM所具备的产业生态系统规模：“定义一个ISA相对所花的时间是比较短的，而建立并佈署一个生气勃勃的生态系统，会需要更长的时间。”

确实，ARM不只是在主流的行动装置市场地位稳固，也拥有庞大的中国晶片业者客户群；根据一份EETimes美国版在2011年底所做的中国无晶圆厂设计业报告，目前ARM在中国有34家授权客户，MIPS则是超过20家；而在电脑领域，英特尔（Intel）与x86处理器架构几乎称霸所有种类的系统应用，包括在中国市场。要与这些檯面上的大厂竞争，为一款新的PC或行动装置处理器架构创造生态系统，会是个困难的任务。

但MIPS的Bismuth表示，中国政府的目标是：“让中国能与西方世界平起平坐；他们想要一套通用的软体生态系统，而要达成此目标的唯一途径就是通用ISA。”他补充指出：“他们愿意取得一种现有处理器架构的授权，并从中开出一条新路──他们并非不愿意付钱。”(文/光电新闻网)

近日，华为科技高层放言，手机芯片业务未来将成为重要业务，华为将会向其他手机厂商提供芯片方案。华为执行副总裁徐直军当天在年度分析师会议上表示：“未来，不论是移动宽带设备、平板电脑，还是智能手机，华为将可以提供自己的核心芯片解决方案。”

当被问及华为未来手机业务盈利前景时，这位高层表示，如果华为不能从智能手机上赚钱，仍然可以通过芯片组赚钱，“如果我们能够从每一个智能手机芯片中赚钱，这将很可观。”

目前，华为大部分智能手机使用的是其他厂商的芯片。不过华为旗下有一个子公司海思，该公司除了无线基带处理器之外，还研发了手机芯片K3V2，这种四核应用处理器基于ARM架构，主频从1.2GHz到1.5GHz。今年初，华为在展示Ascend Quad系列智能手机时，也对外推介了自己的芯片产品。

媒体评论称，进入移动终端芯片市场，意味着华为将成为高通、NVIDIA、德州仪器等公司的对手，这些公司的芯片，已经被用于热门手机和平板产品。

华为相关人士也表示，K3V2芯片目前只在华为内部使用，但是华为希望向其他手机厂商供应。徐直军表示，从华为角度看，在未来两到三年内，华为在智能手机硬件上的竞争力将没有问题。(文/搜狐IT)
