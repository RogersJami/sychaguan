        对大一的新生，尤其是迷茫，学习动力不足的时候，我忒别爱用下面这个故事来开导他们：
     
        我是80后，你们是90后；坦诚的说，我特别羡慕你们；羡慕你什么什么呢？不是羡慕你们手拿Iphone耍酷，不是羡慕你们开车来读书，不是羡慕你们比我们当年更优越的生活条件；而是羡慕你们年轻、朝气、更羡慕你们有充足时间和精力用来学习求知；
      
       我把你们90后大学生的时间和精力比喻成手中的一副牌，或者是桌子上的一副牌；尽管它有些凌乱，但只要稍加整理，就可以打出一副好牌来；而我这位80后也有这样一副牌，可是这副牌已经从教学楼的八楼抛洒下去了，散落在了校园的每一个角落；这个时候要把散落的牌搜集起来打一幅好牌，何其难也？ 为什么艰难，因为我们的时间精力和你们一样，也是有限的；不仅仅用来上课，还要用来陪伴妻儿老小，还要用来工作应酬，还要用来操心房子 车子 妻子 孩子等等；在这种情况下，你还想静下心来读书学习，很难做到，真的很难做到；

       坦诚的说，大学四年，是宝贵的四年，如果这四年用来 玩耍，用来游戏，用来恋爱，我并不反对，但是我提醒一下各位，错过这四年的学习时间，以后都不会有这样的机会来让你们学习了；想玩、想恋爱、想打游戏，你们就是到了81岁仍然可以去做，但不可能有现在这样的心态、时间、精力、效率来供你们静下心来充实自己；

       以上是我当班主任的时候对大一新生关于学习话题的谈话，仅供参考一二；
