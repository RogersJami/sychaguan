个人体会是案例的主人公的情况是：口是心非

说超然了，却没有真正的超然

楼上快乐是什么已经说得很好了，嵇康自己在野，但一直干预朝政，写文章煽动士子，跟司马家敌对，是自取灭亡，钟会只是煽风点火。下面加一点我对钟会的理解

钟会出身的钟家是名门望族，父亲位列三公，兄长也是重臣。他为人机敏、长于计谋，但同时虚荣、小心眼，司马昭夫人说他“见利忘义，好为事端”。

他揣测上意非常在行，很受司马师和司马昭器重，被司马昭称为张良。但司马昭并不完全信任他，在他当主帅伐蜀的时候也在他身边安了几颗钉子，最终就是这几颗钉子镇压了钟会的叛乱。

钟会为人非常小心眼又爱面子。他成名之前想结交嵇康，怕当面受嵇康奚落(竹林七贤之间互相奚落是常事)，就远远的把自己的作品扔进嵇康家院子里，掉头就跑。他成名之后，带着大群宾客来拜访嵇康，嵇康不予理睬，他记恨下来。之后寻衅参了嵇康一本，嵇康被处死。

伐蜀的时候，邓艾孤军深入，在成都门口决战胜利，接受了刘禅投降。钟会一是功劳不如邓艾大，二是看不惯邓艾受嘉奖之后有超越自己的势头，三是一直以来自己代表的名门望族跟邓艾代表的草根就不合，四是有点动了取代司马昭的念头。于是他招纳了姜维，对司马昭陷害邓艾。杀了邓艾之后，准备驱动军队加上蜀国降兵造反，但是被司马昭在身边安排的钉子发动兵变杀死了。

钟会这一生极为精明，但也树敌众多，最终掌握的权势超出了他心胸能容纳的范围，招致横死。

金源 发表于 2012-6-21 06:25:03

昨晚临睡前匆匆看了忙总大作，心有所感，说了几句，一梦醒来，感觉意犹未尽，多说几句，求教诸贤。
颐卦，震下艮上，卦面上看，是外阳内阴，两阳中夹4阴，外刚内柔，外方内圆，简单点说，就是内积累功德，外求自保，呵呵，这种情况，一般是老年之人，或者历经风雨，或者饱经沧桑，做过大事的人，从幼时一腔热血，经历世事的折磨锤炼，猛然醒悟，原来阳为阴之用，于是沉寂心灵，以中和心态处事，去灾免祸，颐养天年。忙总现在在网上呈现的状态，与此卦很相似。
从五行看，震为胆木，象征年轻人充满豪气，立志做大事成大业，雷在水中，不鸣则已，一鸣震乾坤。艮为脾土，也是山，象征身体肌肉饱满，做事稳重且有耐心毅力。
颐卦两阳夹四阴，两阳中间有条通道，意味着人明白了自己从那里来的，还是回那里去，人生不过是一游戏。但人活着，精神存在的基础是有个身体，所以，用自己经历获得的知识和自己的行动及创造能力，都转移到自己的身体上，也就是开始注意自己的身体了，这才是颐卦的含义所在，我印象中，颐字也有快乐的含义，但是百度没有查到，只有明白生命的意义，人才会真正的快乐，也就是知天命而乐。
颐卦也揭示了修身的关键，是可以返老还童的，就是一阳幼儿之体，历经养阴，可以回到一阳之体，修炼途径忙总文中说过，但是，没有阐述具体的实施办法，这就好比说航母是用钢铁造的，但是，关键的航母甲板钢板制造方法，是一个秘密。
感谢忙总，我才知道嵇康写有《养生论》，粗粗看了看，写的不错，也是一些大道理，知易行难啊，即使嵇康自己，也没有按照自然规律来做，嵇康出名了，出名一般是巽风，无影而有声，阳气勃发待用，不用不行，不用要出麻烦的，出名后必须做一些事，以成就名声，慢慢的，成事多了，阳气逐渐用尽，自然会出错，出错之后，引咎归隐，这些名声带来的麻烦自然就消失了，这时候他的养生论才会派上用场。即使是我自己，看到嵇康的问题，对于自己的问题，能真的看清楚吗？知易行难，知易行难啊！
感谢忙总，以前我一般在五行和八卦中寻找事物的落脚点和规律，看来易经64卦更有一番深刻的道理，值得好好学习。
