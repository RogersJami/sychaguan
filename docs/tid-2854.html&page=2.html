以2007年为例，庄家至少从散户那里掏走一万亿人民币。

没有发达的二级市场，没有天文数字的二级市场交易，就没有庄家对散户的剥夺。庄家坑散户，无非是“高位接盘，低位割肉”，反复来搞。

事实上，经济史上利用房地产市场割羊毛是很多的，其原理无非也是反复“高位接盘，低位割肉”，但中国的房地产市场并非如此，中国的房地产市场是每年实实在在的生产一千多万套住宅出来。与割羊毛时利用少数中心城市稀缺地皮一会拉高一会打压（与股市类似）不同。但正因为如此，楼市的调控不力会对中国以及世界的经济增长产生巨大影响。

江城如画里 发表于 2012-8-4 22:11:49

这里一个关键就是住房是不是商品的问题。如果按商品处理，那么目前就是个垄断和稀缺的昂贵商品，自然是不可能低价下来。以北京为例，现在观望的人是多，但价格稍有松动，就有有能力接盘的人接上来了，总的来说确实还是由于供应量不足。
但是，这个问题得从另一方面看。目前住房问题已经成了介于经济与政治之间的问题了，或者说直白点，成了影响TG统治长治久安的问题，成了TG长期统治合法化的问题了。这个问题对于第五代来说更是现实。
那么这个时候TG会怎么选择？显然是经济问题让位于政治问题。
我对中国房价的看法是，五代的策略应该是大建公租廉租和共有产权房，简而言之，就是选择一条长期的路线，使住房的商品属性降低，公共产品属性提高。这样房价问题就不能简单的从经济角度出发分析了。
如果五代的策略成功，未来的十年应该是商品房价格缓慢下降的十年，因为这类公租廉租和共有产权房出量是有一个过程的。解决地方资金来源问题是最重要的一环，这个环节解决了，就会有潮水般的新产权形式房出现，从而使得老产权形式的房子逐渐贬值，形成一条缓慢下降的曲线。我猜五代心中的理想情况就是这样。
如果这个策略不成功，就会比较惨烈了。我和一些地方官员以及中央党校的人有过比较坦诚和开放的交流，给我的一种感觉就是，对于TG的这些精英层来说，维持统治比什么都重要。感觉住房的去商品化可能成为未来十年一个激烈交锋的战场。如果平和的去商品化过程完成不了，就可能有激烈的去商品化，比如取消商品房预售制度，物业税等等。
基本上，在这个过程中，个人只能跟随大潮，所以我也基本上同意忙总说的。你要等得起，十年后的收入价格比会比现在令人满意，但是真正的刚需，咬咬牙买了吧。因为房价会在十年的周期内缓慢下降，并不会出现崩盘。
