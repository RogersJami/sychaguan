derek 发表于 2012-7-4 14:15:19

日本人干的这种事情在企业经营中是一种很常见的现象。

企业从创业开始，从小到大的过程中，往往主动寻求面对高风险机会放手拼搏，快速成长起来的企业无不是这种大胆策略的产物，除此之外，往往还在必要（不必要）的时候敢于不仁不义，只求抓住机会获取利润和成长机会。

呵呵，大家联想一下我国这些年那些神一般速度发展起来的大企业，是不是有这样的特征呢。这种事情我们已经司空见惯了。

然而，当企业壮大到进入“可持续发展”为重的阶段，就应当及时以持重守成策略替换大胆狂突猛进的策略，否则“玩火自焚”就将成为必然。然而，似乎鄙人目力所及，这个转型阶段之下，能有意识地自觉转型的是少数，而转型成功的则更是少之又少。

譬若华为的几次命运大赌博，譬若长虹当年的彩管大战，譬如UT斯达康，譬如盛大，譬如。。。。。。

所以我的结论是：日本是一家缺乏转型意识的以创业思维运营需要稳健发展策略的大型企业。
