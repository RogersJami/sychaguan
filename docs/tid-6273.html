跑步的肥羊

发这个帖子之前想了很久，因为我不是专家、达人，这个贴子不能带给大家知识和经验，我的文笔也不好，不能让大家赏心悦目，这只是个单纯的记录，记录我自己跑步的点滴，也希望能记录我的恒心和毅力，这是我所缺乏的。

跑步之前是走路，一开始每天3km，后来加到5km，总觉得不过瘾，所以懂了跑步的念头。
走路之前是骑车，每天来回24km，后来看网上说骑车时间长了对男的不好，加之昆明现在大兴土木，春城变灰城，遂放弃。

跑步是从2012年的11月份开始的，1km、2km、3km，慢慢来，时间上也是断断续续的。我是晨跑，坐公交到半路，走1km开始跑。这里藏了个小心眼，坐到半路下车，如果反悔想接着坐车是不可能的，公交车已经爆满了。

上周跑了两天，路程都是3km，时间都是29分，后面几天陪着去检查、去医院等等没跑。

本周，周一外出培训，搭同事的车，没跑。

周二（4月16日），跑了3.7km加上过路口的300m，用时24分53秒，自己觉得速度不快，能够接受，结果比之前少用时4分钟。跑完了大腿有些酸涨。在距离上比原来有突破。

chalet 发表于 2013-4-17 10:52:40

一般人的跑步主要还是锻炼心肺功能的收效比较大，对减肥来说，相对作用比较小，除非饮食控制得好，原因是一般人跑步的量离直接调动脂肪供能还远着呢。

刚开始跑步有几个要注意的：
注意保护膝盖，如果体重比较大更要小心。因为长期不运动后，刚开始运动时腿部肌肉还不够强劲，对韧带还不能很好保护，所以韧带会容易受伤。相对而言，跑上坡路对膝盖的负担小，而对心肺功能锻炼更强，有条件的话建议多跑跑上坡路。此外，注意锻炼腿部肌肉的力量，这个网上应该有专门的帖子可以参考。也可以先戴个护膝，有保护的那种。万一有不适，一定要注意，该休息就休息，不要硬挺。

不要做洗尘器。现在城市空气太脏，跑步时肺部的通气量要增加好几倍，也就是说跑二十分钟可能就吸了平时一两个小时的空气。所以有条件的话，尽量到空气好的地方去跑，不行的话，去好点的健身房吧，至少新风系统能阻挡一点空气中的颗粒。

zju 发表于 2013-4-18 09:10:25

跟楼主一样,我也是从1公里开始跑起,坚持下来有三年多时间.
现在每天跑7/8公里,每周4/5次的样子. (跑更长也可以,可是没有那么多时间).
体会是,每次最好坚持30-40分钟,这样才有效果. 对每天的精神状态,工作效率改善很多.
至于减重,最初开始跑的那段时间效果很明显,后来基本上稳定.如果真的想减,还是要配合饮食习惯的调整.
充分的准备活动和舒适的鞋子很重要,可以避免肌肉和关节受伤.
我一般备4双鞋子,轮流用,基本保证一双鞋子每周只穿一次,要知道,鞋底保持足够的弹性对保护膝盖是很重要的.
