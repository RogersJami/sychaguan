      我认为强干弱枝的政策还是会推行的，伴随着技术的进步，中央的权力一直在扩张中。而相对于刚解放的时候，即使是相对于刚改革的时候，我们的技术已经进步太多，这是一种本能。
       像这些集中财富、权势于自身的本能。我们可以清楚的看到，法国在大革命前的集权导致了巴黎的异常繁荣，别人已经评价巴黎郊外都是农村，都有着一样的习惯，都无从的看着巴黎。法国的君主集权趋势深刻的、隐约的从巴黎的吸血中折射了出来。因此，在中央加强集权的情况下，有一个最明显的特征就是城市化，特别是超级城市的出现。而在我国改革开放以来，城市化是一个强烈的趋势，而这个趋势如今是更在加速和放大中。
       同时，伴随着集权的进行，还有一个是社会扁平化，例如，社会的阶层减少，上升途径的困难。这个扁平化发挥着重要作用的就是银行，现在很多财政支付都已经直接补助到人，绕过了很多部门，现在基层越来越像个信息反馈者了，如今，政府里面的信息职能越来越重。扁平化的另一个表现就是教育，这个就看教育改革如何突破城乡分野了。另一个考察社会扁平化的重要指标就是上访，可以从上访的群体的变化了解到社会底层内部是否有着异动，而乡镇的功能以前很多的是把上访的冲动镇压在苗头中，当上访需求变弱时，乡镇的最大职责社会稳定的减弱，肯定导致乡镇权力的变小。而且，当网络化发生时，中央的信息源扩大，他更能把自己放在仲裁者的角度上看双方博弈，同时以前为了相互监督的内部博弈而设立的机构也能进一步的削弱。
