明德 发表于 2012-4-19 14:35:07

说下浪费的问题
饭菜浪费分几种情况，饭店多点菜大多出现在礼仪宴席和公款吃喝中，后者要依靠三公经费、政务公开来管理，前者数量较少，也就是婚礼
日常吃饭浪费总体数量最多，其原因是饭菜提供方无法进行精确分类。例如公司吃盒饭，每人饭量不一样，一人一份必然浪费，饭店吃盖饭、面条等等同样；大学食堂属于饭菜质量无法精确分类，很难吃的菜一份就那么多，想少要卖饭的不干，高质量的又被限制涨价；家里吃饭有浪费的，我没见过，比例问题；
减少日常吃饭浪费，需要在饭菜质量、数量分类方面进行精确划分，这需要投入人力分装，对饭店来说有人事成本；对顾客来说需要等待分装，有时间成本；因此这么多饭店没有做，这可不是因为他们不想少送点饭菜降低采购成本
以涨价来遏制粮食浪费，完全是不食人间烟火的神仙做法，与胡不食肉糜有异曲同工之妙
