原 创: 草 药 郎 中 ( 十 五 )



      俺住在张家隔壁，隔三差五，常看到有乡下人提了鸡蛋，茶油，鸡呀，鸭呀什么的，寻上门来，求老太婆医治这病。病人一来，她家的大门便关得梆紧，过一会儿，便听得里面像杀猪一般的嚎叫起来，痛得病人啾爷喊娘。每每听到屋里边在啾爷喊娘，便晓得，治那种病的人又来了。　　常有人痛得受不了，少不得要呜呜咽咽，哭不断声。这个时候，便把老太婆的牛脾气给撩发。
    老太婆的牛脾气一撩发，断乎少不得一顿子地臭骂：“哭！哭！哭个卵不哭呀！你快活的时候怎么不哭呢！给你烧哈灯火，你就忍受不了！真是丢脸呢！还哭我就懒得帮你治了！……” 　　骂过一阵，老太婆又是一顿好话安慰起来：“好了！好了！莫要哭了！这刚才就没得事哒的……”。　　老太婆治风湿病也有手绝招。　　有一回，俺正在家里做午饭，炒菜时发现灶头上没了盐，去买吧，一口气来不及。没法子，只好去隔壁张家讨一勺子盐回来救哈急吧。　　去的时候，正好看到老太婆儿子手里头拿着块瓷针（即用破瓷碗片砸成锋利的针状，俗称瓷针），俯下身子，正全神贯注为病人治风湿呢。桂林他妈，也就是张老太婆，坐在一旁当指导。　　桂林手里头拿着块破瓷片，正急速地，不停顿地在病人的腿上剌来剌去，随着瓷片连连剌下，病人腿上不断有红得发乌的血点子涌出来。那位病人呢，痛得像是吃了一包辣椒末儿的，嘴巴里头发出啧啧不已的嗖嗖声，疼得直呲牙裂嘴。　　候到一条腿剌完毕，便开始拔火罐。　　老太婆从抽屉内掏出一大堆竹筒来。　　竹筒长短不一，大小不等。桂林拿起个竹筒，先把筒口朝下，一只手抓了张卷成筒状的黄纸，搁一旁正燃着的煤油灯盏上面就火点着，等到黄纸燃烧起来，便把竹筒在火苗上晃上几晃，趁竹筒晃得热呼呼的，迅即把竹筒口往病人腿上一按，稍倾，那竹筒便牢牢地粘稳在腿上。等到竹筒内的热空气一冷，火罐便自动掉下来。　　一忽儿功夫，病人的腿上巴满了大大小小的竹筒，有十来个。　　火罐脱落，扎过瓷针的针眼内，冒出来的全是红中夹杂着发乌的血液。我在一旁看得是胆战心惊。　　等会儿再过去瞧看时，火罐已拔清白，正在敷药膏。敷上药膏之后，照例是用绷带缠起来。
