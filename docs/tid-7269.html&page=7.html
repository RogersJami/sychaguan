你说的由校长担当，我想这校长一定不简单。因为督学和督导是2个概念。
学校的管理并非是课堂管理。条条块块很多。

管理学校比管理企业难度更高。企业有硬标准，产品决定一切。而学校的标准是活的，在一定的区域内，多所学校比对，然后决定优劣，你可以想象。
督学一般是政府编制，权力体现的协调单位的督导检查工作上。学校巴结没用，因为结果不是他说了算，他只是协助指导你工作，但结果是上级督导评比得来的。而评比的标准是一定区域多所学校的高低来决定的。
