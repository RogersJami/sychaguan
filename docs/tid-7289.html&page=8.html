瑞典房价15年涨两倍 或引发新一轮房贷高峰

2013年10月28日 01:14  每日经济新闻  我有话说(74人参与)
　　
每经实习记者 覃钰景

　　中国房地产市场正面临着日益增长的空房率和居高不下的房价的困扰，人们担心，房地产泡沫破裂之时，会给中国经济带来巨大的影响。

　　无独有偶，远在欧洲北部的瑞典也正饱受着来自房地产市场的压力。由于房屋建造数量远远低于人口增长速度，房价同样居高不下，已经严重影响到了瑞典的经济。

　　住房供应落后于人口增长

　　据路透社消息，近日公布的第三季度斯德哥尔摩及其周边的普通家庭住房售价已达到62.3万美元一套，几乎是15年前的3倍。据英国经济和商业研究中心数据显示，这一价格与大伦敦地区62万美元几乎相同。

　　据了解，在过去的几十年中，瑞典新房屋的建造速度远远落后于人口增长速度，并带来了劳动力流动性锐减、住房价格攀升、按揭贷款利率居高不下等令政府头疼的问题。

　　和中国滚滚的周末看房大军一样，瑞典的购房者热情也并未被高首付、高利率和严格的抵押还款细则阻碍。“购房者们从来不会提及这些问题。”一位房产经纪人在介绍一间斯梅兹兰登火车站附近的139平方米的公寓时如是说道。大概有40位顾客看了这一处93万美元的房产——尽管需要彻底翻新——在分配的45分钟看房时间的头15分钟里，阻隔看房者鞋底泥污的蓝色塑料(11265, 50.00, 0.45%)鞋套就被迅速用完了。

　　“过去的20年中，我们建造的房屋只有其他国家的一半，不得不说我们现在面对的是一个长期的问题。”瑞典公共行政和住房部长斯特凡-艾特弗尔表示，“另外在招聘劳动力上，由于国家的限制性规定，大大地降低了建筑公司的长期竞争力，所以很多劳动者都不愿意应聘这类公司。”

　　住房短缺系房价上涨主因

　　另外，分析人士指出，复杂的建筑法规也是导致住房短缺的原因之一。在瑞典，开发商获得规划许可可能就需要10年时间。瑞典需要解除对零售市场的限制并简化获取建筑许可的程序，并将规划权交给区域性团体，以避免“不在我的后院”的心态。

　　财政部长安德斯博格说过，政府不会容忍房价进一步上涨并已采取措施为贷款需求降温。但分析人士认为，无论是提高信贷门槛、为按揭违约设置缓冲区，还是提高初次购房者限制都不能解决根本性的问题。“更高的银行信贷门槛并不能提高房屋供应量。”瑞典银行首席执行官迈克尔沃尔夫说道。

　　据预测，到2030年为止，瑞典首都斯德哥尔摩的人口会增长60万，而目前该区域就已经短缺了约11万套住房。根据瑞典房产协会的研究显示，到2030年，瑞典一半的GDP都会由斯德哥尔摩地区贡献。如果住房建造速度跟不上发展的节奏，瑞典将每年损失19.4亿美元的潜在增长。
　　新一轮房贷高峰的担忧

　　国际货币基金组织[微博]、欧盟和瑞典政府正在对欧洲新一轮房贷高峰表示担忧。1990年，瑞典曾有过相似经历，居高不下的房价和信贷增长最终引发了地产崩盘，导致经济进入长期衰退。

　　由于住房紧缺、劳动力流动性锐减，导致政府需要更多的人去工作，才能维持瑞典的高福利运作，然而失业率却依然高居不下，大约有三分之一的人失业超过6个月。青年失业率为12.4%，远高于欧元区的平均水平9.6%。

　　国际货币基金组织认为，当前挪威的房产市场被高估了40%。德国中央银行也对其房产市场发出警告，欧洲中央银行韩伟森亦对挪威房产泡沫发出警报。在瑞典，对信贷增长的担忧让央行采取了除降息外的所有手段。

　　瑞典AAA的顶级投资评级使其成为了过去五年全球经济重压下投资者们的安全港，但房产价格却酝酿着一场动荡。“这是瑞典经济最大的结构性问题，”瑞典银行首席经济学家安娜费兰德说。
