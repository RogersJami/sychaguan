首先，08年后，李显龙政府已经过度使用了财政政策，大规模的基础投资是在让我这个中国人都觉得太多了。而且这些项目现在基本上都进入了稳定发展阶段，实际上无法提供更多的外来资金了。新加坡已经达到了增长的极限，没有进一步扩展的能力了。至于PX之类的，我的认为是，如果欧美经济能够非常快速的恢复，那么，这种出口型项目可以获得回报，但是很明显这种情况不会发生了，那么他们就只能依靠对华出口了。。。

实际上，我们能感觉到的是，新加坡拼命在犹豫，他们知道美国人靠不住了，但是也不知道中国人能不能靠得住。
