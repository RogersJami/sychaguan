而茶馆其实是忙总倡导的新的一种模式。在茶馆里，大家都是平等的茶客，大牛相对来说可以很放松，有好的文章了可以开个主题贴讨论；没有好的创意和文章了，大牛也可以在歪楼回帖中吹吹牛侃侃大山，或者表达一下自己不熟悉的和不成熟的观点，即使被拍砖也不用背上原创的负担，不过是喝茶吹牛而已嘛，大家都退一步又何尝不可了？这样的模式原创的精华贴虽然少了点，但话里话外的字字珠玑却多了起来，对大牛也没有那么大的压力了，我看忙总在茶馆的心情要比在河里快乐多了，也不用总想着写好文章给大家献礼了。
另外，茶馆里平头百姓也可出来喝口茶随心所欲的扯两句了，就交流的广度而言也大多了。当然，茶馆的交流深度还需要各位潜在大牛的支持，多上好茶来供大家吹牛兑水，互相娱乐！

土匪王 发表于 2012-4-12 00:31:38

西西河的问题，还是在于老铁的政治立场的问题。西西河的每一次风波，基本上都可以说是因为政治立场引起来的。很少是因为学术争论引起来的。

西西河是一个很左的网站。这要从它的历史来源说起。我们都晓得西西河是从文学城的一个版块分出来的。文学城当时是海外第一大网站。现在也还是有声有色，经常会有好帖子，很多人在西西河发帖子后，会在文学城也发一份。但是，文学城好像真正的自由主义者比较多。老实说，我觉得文学城更对我的胃口，但是文学城的版面有点糟糕。正是因为文学城的自由主义者多，而且好像当年的老板是台湾人，所以上面的台湾人id也比较多。而当时海外华人几乎都上这个网站，自然，有军二代背景的人就聚集到一起了(这一点whatever和laozhong都说过)。因为当年能够出国的，都是有背景的居多，尤其是有大院背景的居多。这些人在文学城上跟台湾的id们辩论，跟真正自由主义的id们辩论，最后成了一小撮了。就破出文学城，搞出了西西河。

西西河早期的大牛里面有自由主义倾向的有很多。然而架不住有军二代，大院背景的id更多。最后，自然早期的有自由主义倾向的大牛和id们逐步被排挤，到退出。温相是明显有自由主义倾向的，老冰是明显有自由主义倾向的，秋原是明显有自由主义倾向的，忘情是明显有自由主义倾向的。而一些把持管理权限的id们，明显是军二代。

军二代比较矛盾。首先要跟蓝军斗，在文学城没有斗赢，就破门而出搞了个西西河。到西西河之后又分成毛左派和邓左派。温相是个邓左派，军二代大多是毛左派。所以，温相不见容于军二代，温相被驱逐。老冰是完完全全的右派，既被毛左批，又被邓左批。架着文章写得好，虽有一批人支持，然而终究不敌毛左和邓左，被驱逐。秋原也是自由主义者，被毛左们排挤，淡出了。忘情因为是实事求是的写，加上是体制内的，从维护铁路的观点来揭露铁路，所以，毛左邓左对他没有痛下杀手，他还在西西河。萨苏开始表现是个左派，讨了日本老婆的爱国主义者，民族主义者，写的又多是给体制唱赞歌的文章，所以，一时锋头无两。加上搞什么采访抗日老兵等活动，得到毛左和邓左的喜爱，结果，萨苏为了表现高尚的民族主义情怀，只要是中华民族的，就都要歌颂，把雄奇兄歌颂了一番，惹翻了毛左和邓左们，从此在管理层里有些孤立了。后来的白衣入蜀事件，又把广大的自由主义者得罪了。从此走下神坛。老铁为了维护萨苏，也是在广大毛左和邓左的怂恿下，对广大自由主义者们大开杀戒，杀了非常多的自由主义者id，一时血流成河。油人好像就是这次被杀了。这次，西西河元气大伤。成了左派的一桶浆糊了。而稍微有点自由主义倾向的id，都被排挤打压。老忙在西西河的3次争吵，除了最后的媚如春事件之外，前两次都是左右之争。老忙之所以，开始在西西河声望很高，有经验，有见识，文字有干货之外，主要还是因为，是体制派，为我党唱赞歌的多。所以能够得到毛左和邓左的支持。然而，老忙还是个花衣派，立马就被毛左littles骂了。自由主义者们支持老忙，跟毛左和邓左又干了起来。最后老忙出走。回来之后，碰到鸡胸脯事件。鸡胸脯事件中的陈郢客是个毛左，因此，一直以来，在西西河的声望很高。毛左和邓左把持西西河嘛。但是，这次，碰到了一个在欧洲做IT的自由主义者？邓左？，看不惯陈郢客的毛左，请她吃鸡胸脯，让她补补。这次事件，牵涉到老铁的政治立场了。老铁首先是个左派，其次是个邓左，但是他是平民出身，还有一些自由主义倾向。所以，这次没有支持毛左陈郢客，而是支持了邓左或者自由主义？的蜀生。他的支持，也不是明确的判陈郢客负，而是调查这个鸡胸脯的来龙去脉，致使鸡胸脯在陈郢客面前放了好几天之久。这件事情，把毛左得罪了，把一些认为有道德感的自由主义者得罪了。而邓左们在后面推波助澜。最后导致了西西河分裂出阿玛尼和风眼。然而西西河还是毛左和邓左的天下。毛左和认为有道德感的自由主义者分出去一些之后，现在自由主义者们慢慢有一些抬头，因为以前一手遮天的毛左，所以，现在，有些人又感觉西西河有点开始好转的样子。这好转，主要在于曾经把持管理的毛左分裂出去了一些的原因。而且，现在，西西河的管理员是自荐的，有时还凑不齐管理员数。所以，自由主义的风气，又有点盛行了。所以，西西河又有一点变清的样子。然而，如果分裂出去的毛左回来之后，西西河，又会变成火药桶的样子。所以，这要看铁手的政治立场。而不是看铁手的管理方法。很多人认为是铁手的管理方法不对头，我认为不对，关键是看铁手的政治立场。铁手是偏左的，邓左，等到铁手变成完全的自由主义者后，西西河才能成为真正的清澈的西西河。
