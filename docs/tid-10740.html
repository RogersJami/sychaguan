原创：行善终有善报，作恶必有恶还！ （下）



      再说这孟鼎鉴，虽说是当地的首富，不过是家里的金子，票子和田土多一点罢了，凭什么警察们对他要如此的恭敬客气呢，且又还礼让他三分？难不成他是个什么人物头儿呗？
      其实呢，真要盘根究底起来，追寻孟鼎鉴此人的身世，这就恰如本地一句俚语所道的那样子：那是锈剪刀剪铁丝一——简（剪）不单（断）啦。
      孟鼎鉴早年一直追随国民党元老宋教仁先生，又是同盟会的骨干成员，兼有土豪与绅士的双重身份。其哥哥也跟随孙中山多年，兄弟俩同为同盟会的老资格，人称‘孟氏双雄’。孟家又是桃源有名的大户人家，在民国政要们眼中，孟家那是腿杆子上面绑着一面大锣——走到哪，响到哪。休要说区区一县警察局局长，就是桃源县的七品县令，在他面前也得唯唯诺诺，小心翼翼，作揖打躬，陪着小心回话。小小的一警察局长，岂敢在他面前拿大装蒜滴！？所以说，尽管警察局的同志们包围了孟家大院，连只鸟儿都插翅难飞过，但却不敢迈腿跨进孟家大门一步。在局长大人还没与孟老爷通气商量弄妥贴稳当之前，谁又敢吃了豹子胆主动去惹事生非滴？倘若不听招呼，贸然行事，不由分说，大摇大摆地冲进屋内抓人，除非是他那局长不想当了呗！
      孟鼎鉴能搭救这张唯一，凭的是自身身份与名气的使然，这种事在他来说，那是提起千斤重，放下四两轻，况且做得也是极为隐密，自然是小菜一碟，小事一桩了，况且也不破费他分文的钱财，自当是举手之劳。事过境迁，也就忘脱了霞影！
      被搭救的张唯一呢，倒是一位受人点水之恩，必当涌泉相报的正人君子，虽说逃过了这场劫难，却始终没忘记孟鼎鉴于危难中救助他的这一份深情厚意，时刻想到要报答他的救助之恩。
      解放前两年，张唯一特地给孟天章写了好几封信，一再地告诫他：一石田（十亩）一斤的肉，你要放肆地吃，一块光洋一斗（一亩）的土，千万莫要买了！说来说去，信中意思，无非是提醒孟鼎鉴，赶快将土地卖得光光的为妙！省得将来被打成个地主，拖上台挨斗背家伙！
       孟鼎鉴乃是读书人出身，可谓见多识广之人，在当年的那个时代，家中就安得有一部专门的手摇电话，与南京，北平……联络广泛，又订得有各式的报纸及各种书刋杂志，虽是闲坐家中，倒是个秀才不出门，能知天下事的。对于外边翻天覆地的时局变化和即将改朝换代的大趋势，清楚得瞭如指掌。
      收到张唯一的来信后，孟家立马行动起来，四处寻找买主，于一夜之间趁热打铁，卖掉了几百亩良田好土。有传说说的是，孟家一夜间卖出了三百多个土地(土地菩萨)（意即土地很多）！
      及至大军南下，桃源刚刚解放的那刻，张唯一还特地安排了一个营的兵力，全副武装，开进到仙人溪这大山的深处，将孟鼎鉴一家，专车接至省城长沙定居下来，湖南全省解放时，孟鼎鉴一家的土地已然卖完，土改运动中，孟家成为破落地主，孟本人因为同盟会的老资格，再加上危难时刻救助过共产党，成为民主人士，被安排为湖南省的首届政协委员，享受政府的保护。
      可张唯一的一位亲老表呢，却没享受到孟鼎鉴这般的待遇与呵护，据乡邻们说，张唯一的老表听说张是共产党的一位‘大官’儿，曾当面向张求过情，要求他出面帮个忙，在新政府中寻一个事儿做哈。可张唯一却说：没什么好事可做，但求平安无事则可。这位亲老表土改时被打成恶霸地主，一枪给嘣了。
