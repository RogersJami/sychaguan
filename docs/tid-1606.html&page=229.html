关于供应链的问题已经在前面说过了。没有你想的这么容易。

至于问题。山某不认为是关一家两家企业就能解决。出问题的三聚首先是不在国家规定的检测标准上。添加三聚的目的是钻了法规的空子。在这种情况下，解决问题的方式应该从法规开始，然后是完善检测手段，最后是建立检测体系。这些都不是某一家企业就能完成的。注意看了一下，三聚风波以后，国家做了几件事。首先是重订了国家标准。把原来的对于国内奶业太高的蛋白质含量降了下来，增加三聚作为检测的一项。然后设立了一个二十个亿的赔偿基金，对受三聚影响的用户进行补偿。最后，应该是各个乳业的企业对自己收奶做了改进，对三聚的检测进行优化控制。最后一点没有见到报道，只是山某从后续所道中，三聚可以说基本上控制在偶发事件的范围内这一事实推论出来的。到此为止，受害者得到了补偿，有害因素得到了控制，三聚问题基本上可以说已经解决。所以你说的没有动力解决只是没有事实支持的想法。

利用舆论造势，明打一种产品暗推另一种产品是国外利益集团经常用的手法。丰田刹车事件是一个很典型的案例。因为美国三大持续被日系车压制，在经济不好的情况下，把从上个世纪90年代就散见于新闻的问题拖出来大加报道。形成的舆论压力使丰田的出售下滑了几个百分点。这还是在双方都是个中高手过招的情况下，一边是舆论，一边是危机公关。都知道游戏的玩法，可动用的资源也差不多。所以后果也在双方可以接受的范围之内。在中国乳业的这次风波里，双方基本上不是一个量级。一边是资源丰富的国际奶业，另一边是没有准备的中国奶业。丰富的经验对上笨重的官僚系统加上不负责任的媒体，绑架一下感性多于理性的中国舆论并不是一件很困难的工作。毕竟合适的口号要比正确的理论更容易羸得舆论上的多数是不单单适用于外国的。
