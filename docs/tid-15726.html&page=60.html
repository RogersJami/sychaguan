北京时间9月8日早晨8点，飓风艾玛的中心已位于美国佛罗里达州第二大城市迈阿密东南约1000公里。超级计算机已达成高度一致：艾玛将绕开海地岛、古巴岛，于9月10日以近巅峰强度在美国本土登陆，最大可能是由南到北穿透整个佛罗里达，随后北上佐治亚、南卡罗来纳、田纳西等州，再而转向从华府或纽约附近出海，将几乎所有能量释放在美国境内。


欧洲数值对“艾玛”的路径集成分析，可见数值成员对艾玛的路径的判断已高度一致

目前，艾玛虽然稍有减弱，但仍为最高等级的五级飓风。事实上它已经是历史性的大飓风：它已连续“五级飓风”60小时，远远打破之前纪录，成为大西洋历史上连续维持五级飓风强度最长的飓风；它的巅峰强度高达160kt(相当于我国标准70米/秒，17级以上)，是大西洋外洋(不包括加勒比海、墨西哥湾)历史最强飓风。如果对比西北太平洋，它堪称大西洋莫兰蒂，虽然巅峰强度还略逊莫兰蒂一筹，但巅峰时间已经远远超过。


北京时间9月8日凌晨的飓风艾玛云图，来自NOAA卫星

艾玛飓风已经是“毁灭者”的代名词，它刚刚给大西洋岛屿带去前所未有的、广泛惨烈的破坏。它眼墙扫过的地方，安提瓜和巴布达的房屋100%损坏；位于荷属圣马丁的茱莉安娜公主国际机场被飓风摧毁；圣马丁岛几乎被夷为平地，惨状如同被战斗机轰炸；荷属圣马丁至少10人死亡，法属圣马丁至少4人死亡，法国总统马克龙紧急开会部署救灾；波多黎各已有至少10人死亡；而风眼直接经过的维尔京群岛交通通讯中断，至今失联。


圣马丁岛损毁情况

艾玛的极端强度在数据上清清楚楚：位于巴布达岛上的美国NOAA水文站测到69.3米/秒的阵风、位于圣马丁岛上的法国气象站测到67米/秒的阵风后，均告损坏；美国NOAA和空军飞机以没3小时一次的频率密集穿越飓风眼，风速数据惊人地稳定在150kt左右。


位于巴布达岛的NOAA水文站实测数据图


美国空军探测飞机在“艾玛”中心拍摄到经典的“体育场”型风眼


艾玛飓风底层微波扫描，核心对流极其雄厚。来自NRL

但艾玛觉得还不够，它还要创造新的灾难历史。在完美避开海地岛高山直接削弱后，艾玛还将避开古巴岛，进入全大西洋最暖的区域：巴哈马暖池。在这里，墨西哥湾暖流和北赤道暖流相遇，海温高达30度以上，且暖水深厚。
一句话，艾玛很可能在登陆美国之前再度加强，以创纪录的强度登陆佛罗里达。
艾玛已经获得大西洋跨赤道气流的支援，在进入巴哈马暖池后还会获得加勒比海西南暖湿气流的支持，高空西风急流也将为艾玛打开。因此在登陆前，艾玛的暴风圈和飓风圈还将迅速增大3倍左右。
因此对美国来说，这可能是前所未有的一次飓风巨灾，无前例可循。目前来说，地势低平的佛罗里达，全州都可能被12级以上大风横扫，包括迈阿密、奥兰多、杰克逊维尔在内，所有大城市一个也跑不掉；全佛州沿海、全佐治亚州、南卡沿海都会迎来飓风海啸，目前没人知道到底有多大。中国驻休斯敦总领事馆已于日前向佛罗里达州、美属波多黎各地区的华人华侨、中资企业人员及中国留学生发出预警提示，提醒大家采取防范、避险措施。
现在距离飓风登陆还有2天时间，我们只能说，在佛州，以及佐治亚州和南卡沿海的华人朋友，立即撤离，去亚特兰大乃至更西更北的地方，现在还来得及。几天之后，佛州将沦为灾区，很多美景和繁华将毁于一旦。
