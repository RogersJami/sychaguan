请教一个问题，海参化皮是怎么回事？

个人认为应主要是自溶酶在起作用。

很多海洋生物体内的自溶酶都很发达，不及时的采取某种措施保存的话就自溶掉了。

所以海参这种东西上来就要尽快保存，煮熟或者盐或者糖之类的都可以破坏自溶酶。

我们曾经拖网抓到一只小海参，回来放冰箱，结果还是自溶掉了。

我还曾经弄过一点样品，不是海参是其他的，一个小时没保存，后来提取dna根本就什么都弄不出来了，全成碎片了。

中新网海口4月3日电(王子谦)3日，海南首次引进的集捕捞、冷冻、生产加工等为一体的“巨无霸”渔船——3.2万吨级“海南宝沙001号”综合鱼品加工船驶达海口港。据船主方海南宝沙渔业有限公司介绍，此生产规模渔船全球仅有4艘。

　　该船是海南三家渔业企业与希腊RESTIS集团共同投资15亿元组建的现代化海洋渔业生产船编队中的一艘，船上拥有4间工厂，14条生产线，可容纳600余名工人生产生活，可连续在外海工作9个月左右。船上拥有不同鱼种生产加工作业线，配备不同标准质量检测仪器，每天可加工处理2100吨渔货物，生产35万听罐头，660吨低温速冻鱼，70吨鱼粉等，还可与小渔船互补，收购鲜鱼进行加工，未来还将运用水上飞机保障联络运输。

　　据了解，海南现有海洋渔船2.65万多艘，但不足30吨位的小渔船占85%，续航能力差、生产设备简陋、保鲜加工设备缺失，导致海洋渔业生产堆挤在近海海域，加速近海渔业资源衰退枯竭和环境污染；与此同时，西、南、中沙海域209至370万吨的渔业资源实际年开采量仅为8.3万吨，增强捕捞能力迫在眉睫。
