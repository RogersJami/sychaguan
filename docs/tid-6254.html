原 创： 炒制红烧肉 ( 上 )

本帖最后由 老光 于 2013-4-14 07:25 编辑

        前几日， 有朋友喊应我，说是在这《吃吃喝喝》栏目中，扯哈“红烧肉”吧。默神一想，现今的人，多讲究养生之道，于这肥腻的“红烧肉”，唯恐避之还来不及。故此，一直不敢在这儿献丑。没想到，有人跟着呵起懒形，说是平生最爱的就是这玩意儿。话说到这份上，再不扯这“红烧肉”恐怕有点“拿大”之嫌！没得法，丑媳妇终究是要见公婆的。只好将去年在本地网上所发的文章转载于此，以飨列位。
        
       前几天，网友‘土狼’发了个回复给俺：光大厨，何不来钵‘红烧肉’噻？这东西而今的人只怕一般不会吃哒，话不收起港（说），完（俺）也是做伢儿的时候吃啊哒滴，记得那肉是肥而不腻，细火慢炖出，越嚼越有味道. 可惜那时完不会喝得酒。…… 我想80前多多少少都试过此‘美味’吧？
    收到回复，过细一想，也是滴！既然扯美味，这"红烧肉"那是万万不能少却滴！
    “红烧肉”也是俺的最爱。
    先前下放农村时候，但凡乡下办红白喜事，八个人，排排坐，围绕一张八仙桌儿，桌面上，无论四钵，六盘，还是大小拢共十二碟，这当家的一盘‘红烧肉’，那是不可缺少的一门主菜！
    那会儿，家家户户，人穷底子也薄，一年到头，辛辛苦苦，累得黑汗水直流，也捞不着点滴儿的荤腥。想吃肉的话，就盼着谁谁家办红白喜事。喜事一来噻，三亲六眷，隔壁二往，都得拢场凑份子打热闹屁股哄一餐‘场伙’打哈‘牙祭’。
    等到主菜上齐毕，客亲们你揖我让，纷纷就桌。筷儿最先落的地点，便是这一盘“红烧肉”。主家做场大事，不容也不易，首先想到的便是：一回的酒水，一世的下巴，这一句古训， 为的是不让客人背后嚼牙巴骨(讲闲话)。请得的掌勺师傅，多是些会弄也会吃的角儿。酱红的斑汁，将那砣砣肉块浸染得红抛抛，亮晃晃颜色，令人眼睛一瞄到，嘴巴内的哈拉子忍不住就要往下直滴。那家伙！就像‘土狼’港（说）的：肉是肥而不腻，细火慢炖，越嚼越有味道！真正的好味道！味道好！吸引得众人的筷子似雨点，你伸我夹不歇炉火。不消片刻儿功夫，一盘‘红烧肉”便抢得精哒哈噜光！
    眼看这中秋，国庆连袂而至，算哈起来，有个七八天的假期，闲坐在家中，合家大团圆，何不弄它一钵“红烧肉”试哈看呢！据说这“红烧肉”的作法，有十数种之多，俺这里且先讲它两种，没得事的，趁闲暇有空试它一试。
    第一种：炒制红烧肉：
    去市场上买一砣五花肉回来。这五花肉得像下面图片中的模样儿，连精带肥都得有哇。
    先切成2——3厘米见方的块块或是砣砣，得记着：块块可不能太小，太小的肉块经过烧制炖煮，做出来象似一盘花生米的粒粒，不光是模样儿不好看，也没个红烧肉丰富多汁那一种感觉。
    得切成这个样儿为好（如图）。
    接下来，把肉块搁进锅里头一顿翻炒。要注意的是：这锅里无须放油。咋不放油呢？莫要急嘛，你先看着，看到后面就会明白滴。
    肉块放入锅中，烧起中火翻炒。刚开始，这肉会炒出水来，你且把这水倒掉即可，再继续又翻炒。
    翻炒到这个样子，肉块的表面会带上点焦色，空气中也有股子肉被烧焦的味道（要晓得，这可不是真的烧焦了，只是有点儿肉的香味而已）。
    肉块抄到这个样儿便可以收手。
    接着，将锅里头的肉倒哈出来，再用清水淘洗它一遍，洗净之后，用个盆子盛起，放一边备用。
    炒的时候，肉块会呕出好多滴油来。你若嫌它肥腻，可把这油倒哈出来用作炒菜，这样的话，烧制出的肉便显得不腻也不肥。另外，炒出油的肉块，受热后肉皮会绷得紧紧，呈现出一定的弧度，外型那是非常滴漂亮。再者，如今人讲究养生，多不喜欢油腻，可又想尝一尝这红烧肉味道。如此样的弄法，兼顾到全面。这就是不放油抄肉的目的嘛！明白了？？？
    跟着，把锅儿洗刷干净，重新烧起小火，再往锅内放上点油。
    乘锅未烧滚时放入几勺白糖或是冰糖，用小火慢慢地炒制糖色（俗称斑汁，代替酱油上色所用），所谓小火就是你能控制到的最小的火候。
    耐心地炒成这个样子，要耐心，顺其自然，不要着急。
    等到糖色炒制成这样子，跟着倒入炒好的肉块，用锅铲翻炒个1至2分钟，让糖色充分包裹住肉块，就是下面这模样。
    然后加入开水，水刚好淹平肉块即可。
    再放入点老抽，就是深颜色的酱油。目的是除了帮助上色之外，再给增添点酱香的味儿。也可用普通酱油代替。
    接下来，把砂锅洗净，将桂皮，八角，花椒，生姜，草果，香叶，盐，还有少许的糖，搁入锅底内，再把炒制好的肉块倒在上面。桂皮和八角少加一点，尤其是八角，两个即可，香料酌多了味道反而怪怪。然后，盖上盖用文火熬炖个1小时。用文火熬炖出的红烧肉，又酥又软又烂又香。
    肉块炖好后，倒入锅中烧起大火翻炒一顿，翻炒为的是收汁，即收干肉中的水分。
    翻炒至水分收干，出锅装盘！
    来个近景，很好看吧？很肉欲吧？请尝它一块！味道如何？？？

temptemple 发表于 2013-4-23 17:59:46

这个周末也做了红烧肉，就是创新加实验。
步骤相同的就不说了，不过我是飞水后，用豆瓣酱、姜葱蒜放入水中慢煮30分，起锅沥水。
因为冰糖没有买，用冬蜜的糖精弄些下去，效果一样，蜜糖香味出来，加香料、小火熬制下，等酱色上后，
在电饭锅垫上切好的白菜，放入可乐和盐、一点白醋（估计作用不大），胡椒，再放上肉
因为怕水不够（失败的地方），又加了水
但是煮的时候，发现出了太多的水，敞开盖煮都没有能顺利收汁。
但是肉味鲜美，纯的肉味已经出来，肉质弹软，味道偏甜，可乐估计还是应该放置在油锅里面收汁。
