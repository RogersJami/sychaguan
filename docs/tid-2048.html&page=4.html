onlyforus 发表于 2012-5-15 21:29 static/image/common/back.gif
去一线也一样

资本家把大学生当民工使唤非常普遍，累死累活给的钱还少

在我看来，大学生和民工没有区别，如果他们的贡献是一样的，同工同酬，不能就凭一个学历要求不一样的待遇。我在一个企业做，里面的技术老大30岁，就是民工出身，手下干活的倒都是大学生。不能有学历歧视，毕竟现在不是20世纪了，本科和高中的差距，和以前高中毕业与初中毕业的差距差不多。

什么叫“血汗员工”？富士康是轻工业，没有什么重体力活，而且是空调的环境，估计比船厂、钢铁厂之类的环境好多了。跳楼的人其实在10多万人的工厂里所占比率不过是千分之一左右，不排除有的是心理疾病的，当然，工作枯燥，环境无聊，混乱的男女关系也是诱因。但工作本身没有什么压力，而且上班也可以有怠工的机会。工资给的不错，生产线加上加班可以3000元。

另外，现场不代表就是生产线，而是作业发生地，如果销售的，就是销售场所，比如：大卖场，客户地点，所销售设备的使用地点等等。
