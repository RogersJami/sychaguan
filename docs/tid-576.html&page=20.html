你以为科学院很好混啊，现在竞争那么激烈。

当然比大学还是爽一点点，起码拿的钱多，生活有保证。

现在北大清华场子大了也是什么鸟都有。我有个同学就是北大金融学的，硕士毕业基本什么所谓的研究都没有做过。中科院很多所不好混日子的，比如北京那一批。生存环境超级恶劣，你不做出来点什么东西就很容易被赶走，因为周围都是牛人虎视眈眈你的岗位。博士毕业留工作单位留不下，留京是多难的事情呀，没有岗位没有编制没有户籍，一个月三四千块钱发给你拉倒。

地方上的科学院单位还行，基本上硕士混毕业还是比较容易的，但是博士的话也还是有点难度的。虽然难度不高。现在都有硬指标了，一个人要发多少篇论文，档次在多少多少，才能毕业。我们单位现在连副研究员都有论文的总数指标了，sci要至少3篇以上才能获得升副研究员的资格（不是一定能升上，此外是除了毕业申请用的论文之外三篇）。这样其实对一些边缘学科和工科是很不利的，其实我不认为这个很正确，有些大学的教授也没有发过几篇sci呢。但是既然定个标准也不能说是坏事情，起码比没有标准好。

地方上科学院单位压力小一些，环境松一些。

现在我身边的人想着考寄托出去的很少，打够级的人也不多。大部分就是做好本职工作，有口饭吃，就不错了。我想的也比较简单，踏踏实实的把自己该打的基础打下来，发展好自己的那块领域，按照生态学上的话说就叫做认清自己的生态位，占领自己的生态位。
