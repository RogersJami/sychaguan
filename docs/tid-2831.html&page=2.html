新华网西昌７月２６日电（记者李清华、江宏景）２５日２３时４３分，我国在西昌卫星发射中心用长征三号丙运载火箭成功发射“天链一号０３星”，卫星顺利进入太空预定轨道。这次发射成功后，“天链一号”卫星将实现全球组网运行，标志着我国第一代中继卫星系统正式建成。

报道说，“天链一号０３星”是我国发射的第三颗地球同步轨道数据中继卫星，由中国航天科技集团公司所属中国空间技术研究院为主研制。经一段时间在轨验证和系统联调后，“天链一号０３星”将与２００８年发射的０１星、２０１１年发射的０２星实现全球组网运行，建成比较完备的中继卫星系统。这一系统将进一步提高我国载人航天飞行任务的测控覆盖率，为我国神舟飞船以及未来空间实验室、空间站建设提供数据中继和测控服务。同时，还将为我国中、低轨道资源卫星提供数据中继服务，为航天器发射提供测控支持。

看到这则消息，青衫老祖不禁再次拍手称快。因为，天链一号完成组网，其意义不亚于中国组建“三沙市”；从国家进军太空、开疆拓土的长期战略来看，其意义甚至高于组建“三沙市”。组建三沙市，已经令美帝恼羞成怒，菲越心惊胆战；“天链一号”组网成功，不知美帝尚能安睡否？

据介绍，中继卫星号称“卫星的卫星”。是半个多世纪以来人类在探索太空的进程中创造的新型空间信息传输系统，相当于把地面测控站搬到了距地面 36000公里的地球同步轨道上。那么，它的意义何在呢？根据青衫老祖的观察，可以定义为“三员”。

一是太空侦察员。作为中国第一代中继卫星，天链一号01、02、03共三颗卫星平衡部署在赤道上空36000公里的地球同步轨道上，能够对200公里以上、2000公里以下的空间全轨道覆盖，覆盖率近100%。也就是说，它们就像三个24小时不间断值守的优秀警卫员，眼睛像鹰一样紧盯着太空各个角落，守护着中国的全部太空资产，一旦发现敌情，立即报告。

二是通信接线员。中继，其实就是接线员的意思。中国目前在轨的各类航天器应该不少于100件，承担着资源勘测、环境监控、敌情侦测、信息传输、科学试验等各类民用国防任务，中级卫星的任务就是把它们所搜集的各类情报、产生的各类数据集中起来，形成一个有机的数据传输整体，第一提高传输效率，第二实现信息资源共享，第三实现应急的协调联动。

三是太空导航员。中继卫星高高在上，下边的情况一清二楚，就可以适时为在太空航行的航天器、空间站提供准确预警预报，导航定位，让中国的太空航行更安全、更准。据报道，“天链一号”０１星、０２星两颗中继卫星组网运行以来，先后参加了天宫一号与神舟八号、神舟九号交会对接任务并发挥了重要作用。任务过程中，中继卫星发挥高码速率、高动态、高轨道覆盖率的优势，提升了飞船和天宫一号的测控覆盖率和数据传输能力，增强了空间交会对接任务实施的安全性和可靠性，为实施手控交会对接、开展空间科学实验等提供了稳妥高效的天基测控通信保障。随着03号卫星成功入轨并完成天链一号组网，天链一号对我太空航行的服务支持将实现全方位、无盲区。

综上所述，有“天链一号”在，中国的太空资产将获得更安全的生存保障。任何来自敌对国家对我太空资产的攻击，都难逃天链一号的神眼；任何被天链一号发现的敌对势力，都将遭到中国地空导弹、太空武器的反击。

或问，如果敌对势力直接攻击“天链一号”咋办？青衫老祖只能说，除非从36000公里以上的外太空发起攻击，否则都不可能突破中国的太空防御网。但是，截至目前，还没有任何国家具备这样的能力！唯一可能的就是激光致盲或摧毁。这一点，青衫老祖认为也可以放心。因为，中国的激光科技在全球是领先的、一流的。中国知道怎么造激光武器，就知道怎么防激光攻击。相信“天链一号”具备防激光打击、防电磁干扰功能。所以，在可预见的时间内，“天链一号”是安全的，这可不是随便YY。
