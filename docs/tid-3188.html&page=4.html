lewisl 发表于 2012-10-14 11:23:34

从公开资料上看，辽宁号实际上主要肩负的是实验、训练任务，兼具作战任务。

我们的航母起点很高，直接从前苏联的第三代起飞（第一代是“莫斯科”级直升机航母，第二代是“基辅”级航母，第三代是“库兹涅佐夫海军元帅”级航母），前苏联从大型巡洋舰一路走来的艰辛我们都跳过了。
但这种超大型军舰的建造、使用、保养，对 PLA来说，都是崭新的课题，我们无法一步跨越；
同时，我们舰载机的设计定型，在航母上的使用、保养，加上配套舰艇的建设、航母舰队的编组训练等等，我们还有很多功课要做。
所以，作为过来人的霉菌，他们评估我们的航母要形成战斗力的时间表，还是有一定参考价值的，因为所有这些功课必须我们自己走一道，甚至来回走N 道......

所有这些基础性功课的完成，都离不开科研、训练、制造基地的强力支持。

因此，我们的航母在成熟编队、形成作战能力之前，不宜离基地太远。首舰辽宁号，应花更多时间在大连等北方基地。
后续航母变动大道，可以考虑南海基地。
