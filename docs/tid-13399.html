三明治

Earl of Sandwich是一个英格兰的贵族爵位，由查理二世在1660年设立。

爵位是世袭的，到了18世纪，约翰·孟塔古 John Montagu成为了第四代三明治伯爵，他也是英国的政治家、军人，曾三任第一海军大臣。 据说孟塔古嗜赌如命，赌起来，都没时间离开赌桌吃饭，于是就用面包夹着熟肉，边赌边吃。 这种吃法逐渐的流行起来，起初是用大写的S来命名这种吃法，后来逐渐演变成使用三明治伯爵的封号。

有人对此质疑，认为面包夹肉的吃法，是古罗马征服英国时带入的。

无论如何，今天，在两片面包中夹肉，奶酪或蔬菜，再加上调味料或酱料的东西，通常惯例被称为“三明治”。


由于制作方便，经济实惠、便于携带、吃起来节省时间，三明治成为欧美人的饮食、文化的一部分。 三明治的做法不拘形式，比方在法国长棍面包里夹上各种馅料，就成为法式三明治：french baguette sandwich。

那么，在面包中加点肉松，咸菜或老干妈辣酱等，也就成了中式三明治了。

但如果是将白面馒头对半掰开，中间加上肉松咸菜等，哪怕是夹了蔬菜火腿，估计也还得称为馒头。
由此可见，核心区别是：面包还是馒头  {:soso_e112:}


英国人对三明治情有独钟，除了作为茶点外也是上班族最抢手的午餐选择。 在各大小超市、饮食店、便利店等都可以见到三明治的身影，它们通常被摆放在入口或最显眼地方。 如果去的晚了，很可能货架上只剩下几只孤零零的三明治。 很多地方专门卖超值午餐套餐（Meal Deal），价格2—3英镑左右。 套餐其实是一个三明治+一瓶软饮料（或一只水果）+一包薯片 （口味均可自选）。

诺丁汉郡的一家三明治加工厂雇用2000员工手工制作三明治（员工大多来自波兰），每星期供应给超市300万个三明治（这只是一家厂的数字）。 英国的饮食连锁店，茶餐厅咖啡厅等地方，三明治大多是由店里的员工现做现卖的，再加上各家自制，英国人每年消耗三明治的数量可是相当可观的。

自己很少吃面包，所以通常在买来新鲜面包后，立即将之放入冰柜冷藏。这样一举两得，即可保鲜面包，也填充冰柜。据说，冷藏室太空的话，费电且增加机器工作负荷。 反正，在国外一袋面包的价格很便宜，即便保存时间过长，扔掉也不心疼。

在没有充裕的时间做饭时，或想换换口味时，随时可以取出面包充饥或制作三明治等食物。


做三明治通常使用的涂酱：

mayonnaise（蛋黄酱）
butter （黄油）
margarine （植物黄油）
在面包上抹上油脂，不仅增加美味，也起到隔离蔬菜的水分的作用，避免三明治变得湿湿烂烂的。


通常使用的蔬菜：
生菜（在家制作，不用切成丝状）
番茄片
黄瓜片


个人经验分享：

用海鲜类（虾仁、三文鱼，金枪鱼吞拿鱼等）、烟熏肉（bacon），蛋类等馅料，适合与mayonnaise（蛋黄酱）搭配。

而各种火腿（ham），香肠，牛肉，火鸡等则更适合在面包上涂抹植物黄油或黄油。



下面几款是近期制作的，是属于做法相对简单，口味classic的三明治。

1. Club sandwich 总汇三明治

正宗的总汇三明治做法，面包要先经过烘烤。 最好先煎bacon烟熏肉，同时煎蛋，快完成时，立即烤面包。
然后在烤好的面包一面均匀涂抹mayonnaise（蛋黄酱），铺上生菜与番茄片，再加上煎蛋与烟熏肉。这道三明治最好是现做现吃，冷热搭配，口感脆脆咸咸，清爽，咸淡适中。

未经烘烤的面包与冷掉的烟熏肉，将会使这款三明治大打折扣。


如何确定面包需要烤或不烤呢？其实很简单，准则是：
*如果是现做现吃的，面包烤一下，三明治的口感更好。
* 如果不立即吃，比方用来做便当，等亲朋好友一起吃，或者是工作Buffet点心，店里出售等情况下，面包则不要烤。烤了也会软掉，多此一举还影响色面。

2. Ham sandwich 火腿三明治

面包上涂植物黄油；如果想涂抹黄油，最好先将面包烤一下。
再配上生菜，番茄片或黄瓜片与火腿。

3.smoked salmon sandwich 烟熏三文鱼三明治

面包上均匀的抹上蛋黄酱，再依次铺上生菜（番茄片）与烟熏三文鱼。

http://img17.poco.cn/mypoco/myphoto/20150727/10/64772759201507271012331296270491161_005.jpg?1568x1252_120



4. tuna sandwich 吞拿鱼三明治

在国外，吞拿鱼罐头价格相当便宜，营养丰富又便于储藏，所以我家里会常年备上几罐，以防万一时应急。

做法：
1.用开罐器将罐头盖割开，不要完全割到底，留一点，然后将盖子压下去，将罐里的油挤出去（丢掉），留下干的吞拿鱼肉。 取出吞拿鱼肉放入一碗中，待用。
2.将半颗洋葱，切碎末后，也放入在碗中，待用。
3.放入2-3汤匙蛋黄酱， 适量盐，少量黑胡椒粒，将吞拿鱼与洋葱末搅拌均，馅料完成了。
4. 面包上涂抹少量的蛋黄酱，放入馅料，一份简单的吞拿鱼三明治就做好了。可配上生菜与番茄片。


吞拿鱼的鱼味比较重，这里由于添加了蛋黄酱与蔬菜，使得鱼味不会太过浓。 对留学生们来说，这是一种经济实惠的补充Omega3 的方法。

OK，回到土豆

这种是烤土豆 roasted potatoes

做法，

方法一（考究些做法，特别是烤小土豆或用于招待客人的配菜）

1.土豆洗净，切块。如果是小土豆的话，对半切。

2. 锅中放水，加点盐，将土豆放入。煮沸后，转小火煮2分钟左右，淋干水分。

3. 然后在土豆中加入橄榄油、蒜粉，rosemary干香叶碎，黑胡椒粒，拌匀后，平铺入烤盘。


*烤箱预热200 C / 400F / Gas 6， 放入烤箱烤30分钟（至金黄色）。


方法二（更接近图片中的效果）

1. 土豆洗净，切块。锅中放水，加点盐，将土豆放入。 煮沸后，小火煮2分钟左右，淋干水分。

2. 在土豆上均匀的撒点面粉（很薄很薄的一层）。

3. 烤箱预热200 C / 400F / Gas 6，在烤盘（或烤盘中的锡纸上）洒橄榄油，预热后将土豆放入烤15分钟左右，翻个继续烤15分钟左右。 根据效果，放回再烤10-20分钟， 直到外焦内软。

*或烤箱预热250C/500F/Gas10烤35分钟


注意烤箱很热，拿出拿进要小心，避免被烫到。
