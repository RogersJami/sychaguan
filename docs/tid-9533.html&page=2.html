丈夫在中美恢复关系之后回了泰国。

3年之后的1977年，臧健和带着两个女儿辞掉单位去泰国寻亲。

大丈夫三妻四妾，几千年一直如此。
按照现在电影、小说里烂熟故事，这将是一个宫斗的故事，一个腹黑女的故事，一个侧室干掉正室咸鱼大翻身的故事，一个生出儿子来的故事。

结果这个人选择了这条道路：

1977年她带着两个年幼的女儿，在去泰国寻亲的途中，因为意外被迫停留在香港，开始以打短工、摆小摊谋生。
1978年臧健和在香港湾仔码头开始卖手工包制的水饺 　　
1985年在香港设立第一家湾仔码头水饺工厂 　　
1997年美国Pillsbury食品公司(现已与美国通用磨坊公司合并)入资“湾仔码头”，并在上海、广州建生产基地 　　
1998年3月在上海建立冷冻食品合资企业 　　
1998年7月在上海推出手工包制的“湾仔码头”水饺和馄饨 　　
1998年9月在北京、广州推出手工包制的“湾仔码头”水饺和馄饨 　　
1998年12月推出手工包制的“湾仔码头”汤圆 　　
2001年9月重新推出改良后的“湾仔码头”馄饨

她是这样说的：“我从小受的教育和自己的尊严让我无法接受这种一夫多妻的现实，为了我自己为了孩子的前途，我只能离开。”

再说日本的：

她生于日本东京附近的静冈县热海市，幼年入读东京御茶水女子大学附属小学校，并从6岁起学习弹钢琴。

12岁时因父亲任日本驻奥地利大使而举家移居维也纳，1961年考入维也纳音乐学院，先后师从理查·豪瑟（Richard Hauser）、威廉·肯普夫及斯特凡·阿斯肯纳瑟等人，并在14岁时首次于维也纳金色大厅举行公开演奏。

在1969年，维也纳的贝多芬钢琴比赛获得冠军；
其后复于1970年的萧邦国际钢琴比赛获得亚军；
1973年在琉森音乐节钢琴比赛获亚军；
后在1975年英国的列斯国际钢琴比赛中再度夺得亚军。
在2002年至2007年期间，她曾任美国克利夫兰管弦乐团的驻院艺术家，期间主持为莫札特的钢琴协奏曲独奏作品进行演奏。

她尤精于演奏莫札特、贝多芬、舒伯特、萧邦、德布西及荀白克的作品。
她曾经灌录莫札特所有的钢琴奏鸣曲作品，另外，她还曾与英国的英格兰室内管弦乐团及其指挥杰弗里·泰特（Jeffrey Tate）合作，灌录莫札特所有的钢琴协奏曲作品。
除了莫札特以外，她亦曾经和指挥家库特·桑德林（Kurt Sanderling）合作灌录贝多芬所有钢琴协奏曲作品，此外亦有灌录贝多芬的钢琴奏鸣曲作品，以及舒伯特的一份联篇钢琴曲。

在1986年及1996年，分别获日本的三得利音乐赏（サントリー音楽赏）及朝日赏，在2008年4月，《BBC音乐杂志》评定她为年度器乐家，并向她授予年度唱片奖。

她对音乐的贡献屡获日本及英国两地政府肯定，日本政府曾于2005年向她授予“文化功劳者”称号；
英国政府亦曾于2001年新年荣誉名单中授予CBE勋衔，但由于当时她仍非英国国民，故有关勋衔仅属名誉性质。
而在2009年的英女皇寿辰荣誉名单中再获勋DBE勋衔，这时她已取得英籍，故可使用“女爵士”头衔。

她现居于英国伦敦，未婚，没有任何子女，她尝言自己热爱音乐，所以其音乐事业亦不容许她有组织家庭的空间。
