做it工资水平常见的是 ：工作年限X2+1，好一点的是：工作年限X3；比如哦工作3年的常见是7K，好的是9K，这还是4年前广州的水平；至于公司对工资外的缴费，见过很多外包公司都是按最低水平交的。外包要招到好点的人，一般要比市场价高20%~30%才能招到人。
感觉外包不是长久之项目，中国随着通货膨胀之类的，再加上产业升级成功的话，在现有工资水平基础上涨个2、3倍个人觉的不是什么难事。到时候国内外基本一样的工资水平，外包还可能回流。
