被处理

茶馆开张两月又两天， 在茶友们和管理团队的支持和共同努力下，取得了一定的发展，正在逐步稳定。

目前，来到茶馆的大部分网友是理解认同茶馆理念的，赞同为大家提供一个开放、宽容（畅所欲言、求同存异），心平气和交流平台的目标，大都自律、自觉的维护茶馆的和谐氛围和秩序，这是茶馆能够继续办好的主要力量。

茶馆规矩是约法三章，也在注册页面明确提出且表示认同的得以加入，请新老茶友不忘初心，温故知新。。。

【茶馆规矩】

1. 茶馆定位：虚拟空间的闲谈聊天交流的地方。

2. 茶馆内容：风花雪月，生活琐事，政治经济，历史学术，读书心得，艺术娱乐等等。

3. 茶馆目标：海纳百川，和而不同，互助互学，碰撞激荡，开阔视野，了解现实，展望未来。

4. 茶馆特点：不设门槛，不限原创*，来去自由，平等民主。

5. 茶馆禁止：故意侮辱，挑逗矛盾，煽动造反，传播谣言，变相传教，其他恶意行为。

6. 茶馆禁止：推销营销等行为，或将论坛作为个人商业谋利平台。


7.茶馆建议：己所不欲，勿施于人； 三人行必有我师，虚心一点，宽容一点，平和一点。

既然是喝茶，就不必搞阶级斗争、左右站队，求同存异。

*不限原创：但禁止全文转载。转来的文字内容主要是用来提供您的主题帖中观点或资料的支撑，或组织整理成新文。或对引用的文章加以自己的简评、意见或提出问题，以便他人进行讨论

以上规矩，对违规及相应处理办法最终解释权属于茶馆论坛 。

---------------------

当然，随着新会员逐步加入，茶馆越来越热闹，也不可避免出现一些争吵和意气用事。

为了更好的维护环境，保持聊天秩序的稳定，对违反茶馆规矩的言行及现象，茶馆会视情节轻重做出相应处理，处置。


茶馆禁止的恶意行为：

1. 发广告/Spam、黄色信息，性骚扰（淫秽图片及言语）。

2. 禁止推销营销等行为，或将论坛作为个人商业谋利平台。

3. 危及论坛安全的碰高压线帖子，例如故意挑逗民族矛盾，煽动造反，变相传播邪教，造谣污蔑政府TG及其领导人等等

4. 恶意灌水，捣乱，对网友骚扰，人身攻击，侮辱辩论对手人格，恶意死缠烂打，无差别投反对评分，四处追杀、恶意举报，及其他恶意行为。

*对违规判定，最终解释权属于茶馆论坛



网友举报：

采取民不报官不究原则。
论坛人和帖子多了，版主和管理员未必每次都看到那些违规的帖子，所以论坛的管理也依靠网友的举报。
当然，多次恶意举报的ID也将受到处理。



相应措施

对于在交流中出现一般的情绪激烈，或言辞过分，不理性行为者，版主和管理员会给予适当的劝说提醒，酌情屏蔽删帖，或给予警告等处理方式。

对于屡犯者或行为严重的，则将酌情采取短封、永封的处理方式，绝不迁就姑息。

对于故意来捣乱，踢馆的，则一律会被请出茶馆，永久封禁，甚至吊销账号。

*被永久封禁的账号将会在被处理版面通报； 但具体情况具体处理，比方故意捣乱，发广告、淫秽内容，故意煽动造反的，一经发现立即永久封禁，吊销账号，不会浪费时间精力另做通报。
