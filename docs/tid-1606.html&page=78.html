"生物药剂学"里不会有关于透皮吸收的内容吧，要有也应该是"现代药剂学"之类的书。

药物的透皮吸收是个很大的难点，人类皮肤的角质层是个极大的障碍。皮肤的亲脂或亲水组织是一层一层分布的，选择有合适亲水亲油系数的药物很难。总之，能有效穿透皮肤并上市的药物制剂很少。

激素之类药物分子一般是脂溶性较强，是否容易穿透皮肤还要看看文献的报道。
