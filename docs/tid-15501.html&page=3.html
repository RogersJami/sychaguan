房价应该要降，但是不敢降。 为什么？ 因为一旦降价就是雪崩。 炒作的成分在房价里头太大， 大家都是期望房价上涨来获得收益。 没有这个预期，就没人买房。 自助的成分所占比例已经很小了。

所以必须用 地王涌现这个现象来加强房价上涨的预期。 所以地王现象只是个托市的工具罢了。  至于是不是真的地王， 都值得怀疑。 地方政府和地产商合起伙来报个价， 不是很容易么？

等着吧。 就要泰山崩于前了， 看多少人能做到面不改色。

现在的问题是，除了经济增长速度、投资、消费受到房地产市场影响，更大的问题是，
经济稳定、社会稳定也受房地产影响，因为房地产问题后面是债务风险和金融风险。由
于购买住房需要大量货币，房地产的典型趋势是买涨不买跌。从经济学基础原理来说，
供求决定价格，按道理是相互作用的，但是房地产市场有特殊属性。现在房地产市场要
去库存，中央希望开发商降价促销，但是一旦价格一跌下来，市场就不行了。也就是说
，降价促销跟去库存模式相矛盾。
