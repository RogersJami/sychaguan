我觉得对人的成长来说，体验体悟，自动自发向上走是最重要的，学校教育只是外在因素之一，现在的人们把学校教育拔高太多了。
条件合适的时候，我能向一切人学习，也能学一切东西。

个人成长很大程度上受先天性格的影响，年纪越大越觉得智商不是最重要的。这几天看了一个日本华人拍摄的纪录片《我们留学的生活——在日本的日子》有几个人物特别打动我。一个是26岁的官宦子弟韩松，一个是35岁到日本求学的李仲生。前一个我压根不看好他，可他居然2年时间零基础过了日本语水平考试，还考上了明治大学（私立），这期间经济独立。后一个因为政治成分问题两次报考国内的大学被挡在门外，由于日本的大学没有年龄限制他居然跑到日本求学，一口气读了13年，从未就职。两个人的选择我看都是错误的，但那种拼搏精神震撼到了我。还有一个叫陈晨的北京小哥，流落街头睡24h便利店依然谈笑风生。

这里讲教育，推荐家里有孩子的朋友带着孩子看一看这个纪录片。你看这片子我相信多少会有些共鸣的。

我自己总结，能够挣扎着 ...

初高中老师的作用是训练紧迫感，通过考试训练，也就是成绩排名。想要拿高分，直觉好+速度快，所以必须刷够一定量的题，有排名压力，才会有刷题的紧迫感。我认为成绩排名是必须的。

大学老师的作用是搭建学科知识框架、展示逻辑思维和提示学科前沿。学科解决什么问题，解决这个问题需要哪些知识，这些知识怎样相互组合，老师讲解学科框架的过程也就是展示逻辑思维的过程，再配合一点学科发展史、学科前沿。考个好大学的好专业，不然遇不到这样的好老师。

最重要的是三观，父母塑造，个人体悟，其他人只起辅助作用，在即将突破的时候提点一下，不在突破的临界点，都是苍白的说教。

猪吃了睡、睡了吃，舒服到死，但没有一点意义，人可以认知、领悟和改变环境，过程中困难免不了，有困难就要战斗，要战斗就有痛苦（最大的痛苦是对自己狠），因为痛苦生命才有质量。

我也觉得报班，学各种才艺，不如帮孩子养成阅读的好习惯，对孩子的一生都有益。十年前，我在帝都带孩子，孩子当时上小学四年级，以前一直是爷爷奶奶带的，没有上过课外班，到了我这里也延续了这个传统，不过每个周末，都会到孩子去帝都名胜古迹玩，也曾去过王府井的儿童艺术剧院看演出。累了，就到王府井书店看看，孩子一到那里就走不动了。上各种班，不如带孩子去图书馆或者博物馆。我记得以前读过一个家长写得育子心得，这位家长平时工作非常忙，但再忙，每周都带孩子去图书馆，家长自己看书学习，孩子安安静静地读书，不限制孩子读什么书，孩子什么书都读，养成阅读习惯，这个孩子长大后很出色，这位家长觉得书籍给孩子潜移默化的影响。我的一些朋友，小时候大人没时间陪伴，但因公或因私有丰富的藏书，因而小时候读了很多书，理解力和习作能力都很高，对他们后来得成长非常有帮助。

我家孩子在小学阶段主要关注了阅读和逻辑思维能力。逻辑思维能力在孩子三四年级的时候主要是用了加德纳趣味数学系列，孩子是自己看着玩，不会做就看答案，答案能给出些思路。六七八年级在爸爸的辅导下刷奥数题，过程很痛苦，因为题目做不出哭了不少鼻子，但也打下了数学基础，现在大学里就是选的数学专业。

也非常赞同然后兄对个人成长的总结，确实内在的驱动是孩子成长的最大动力，尤其是孩子挫败感之下产生的想改变自己的意愿，要比家长的唠叨有效的多。家长越是给孩子创造优越的条件，越是怕孩子跌跟头，事事都给孩子想好了，反而剥夺了孩子在挫折中成长的机会。当然老槐兄以前也说过这种挫折感不能太大，要孩子能承受得起。

学习是跑马拉松，前面跑得快没有用的，能否夺冠考的是耐力而不是速度。小学刷奥数得奖，大学学个线性代数都头疼；中学学六年英语年年第一却不会基本的对话技能，甚至听都听不懂，在外留学半年内必须掌握基本的语法词汇，否则生存都困难。这些例子多如牛毛。

孩子的好奇心是有保质期的，不精心呵护就会腐烂变质。没有好奇心进入大学后某些难点课程几乎没有学会的可能。如数论在离散数学中应用范围很广，网络的信息传输安全就是靠这门学科保证的。没有兴趣学三年也是读书皮。

没有好奇心至少保证好胜心，关键时候对自己能狠。这个我有一个现成的训练方法，拿周围的小孩试验过，效果很好。引导孩子玩电子竞技游戏，最好是SC2、LOL、DotA2一类耐玩的游戏。观察孩子输赢后的情绪变化。如果输赢都很平静很不在乎，这个麻烦就大了。
