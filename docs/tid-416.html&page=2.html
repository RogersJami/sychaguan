mopfish 发表于 2012-4-22 12:57:23

中国滞涨的根源在于随着人口红利的结束，经济的潜在增长率在下降，但主观上不承认这点，依然相信通过投资可以保持一个高增长，必然导致货币增发，引起通胀。其实，通胀就是一个经济体勉强保持高增长的一个约束。

在低增长和高增长不可得兼的情况下，人的本能会反复试探。所以看到了从08年开始，一会紧缩，一会放水的反反复复。但回旋余地越来越小。在约束的条件下，政策犹如水流，一定会向压力最小的地方流。

在一个官僚体系下，保增长就是保政权，高增长就是政权合法性的来源，而民生不是，相对来说是可以牺牲的。所以，增长和通胀冲突的情况下，增长一定优先于通胀。

在民意和民生对政权的压力和约束最弱的情况下，最后一招可以修改统计数据。事实上，普遍的看法是中国的CPI完全无法反应经济实际的通胀情况。人民可以不满，但无法最终决定经济政策，但需要承担经济政策的后果。
