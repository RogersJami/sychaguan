您说大城市的土地成本高，拼起来的产业生存不下去。
但是我认为土地的价值本身就决定于产业，现在的大城市的土地成本之所以高，是因为有发达的高端产业，支撑了这样的高地价。
拼积木起来的大城市缺乏高端产业，地价本身就做不上去（地价高了，弱企业生存不了，那土地就卖不掉。土地卖不掉自然就会跌价，跌倒弱企业能生存为止）。
