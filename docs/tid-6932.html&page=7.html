  从理论到实验室成功，从实验室到量产都是不一样的；很多理论在纸面上，可能要等几十或百年才能进入实验室，而进入实验室即使实验成功的东西可能永远无法量产或出于成本效率的原因要等到某个适当时候才能投入应用。
更有很多理论在当时发现的年代是根本无用的，但要等很长时间才能在实际中发现价值。
比如现在热火的2样技术：
3D打印，理论在19世纪末就成形，但第一台样机是80年代中研制成功，一直到这几年才变得火热。所以3D打印号称是：19世纪的思想，20世纪的技术，21世纪的市场。
而页岩油，在70年代从技术到实践都已成功，但迫于成本因素，一直被束之高阁，一直到07,08年，油价上80美元后才有实际经济价值。
