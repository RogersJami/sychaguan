现在考研的二本的，有的混日子都懒得在这行里面混，甚至连考研不过是随大流，根本不具备做这方面的素质和心态。考上了又能如何，当然从个人角度来说，素质能提高点，但是对增进整体的科研界的水平来说可能很小。

我有个师兄在二本做系主任，每年送来若干本科生让我们带做毕业论文。

直接就无法交流，你都不能想象他们的底子。最要命的是连几个数据库都玩不转，不知道该怎么把自己想要的论文找出来，更不知道怎么组织材料。

就比如今年带的一个，这个女生刚刚被中科院贵阳地化所录取了。然后我问她，你现在的导师是做什么的，答曰不知道，是所里给她换的导师。

然后我问，你有搜过他的论文吗，搜搜就知道了啊，答曰没有。我说那你去数据库搜他中文论文近几年做的。我心想现在让你短期内弄英文的你也搞不懂，还是先搜中文的有个印象。我说你搜完了发给我看看，并且告诉她可以搜论文找她导师的联系方式云云，然后小女生就自己去搜了。

然后搜了一堆反馈给我，还算比较仔细，分每年的搞的很详细。不过我打开看看就不对了，有个同名的，明显做的东西根本不是一个学科的。。。她那个导师是做地球化学的，里面杂了篇学医的是咋回事。。。

然后我跟她说，你去搜下我要让你写的东西，先搞个背景来，搜关于长江口浮游动物的，中文的，五十篇左右，先大体看看。

然后她的回馈是：关于长江口浮游动物的cnki里面只有二十多篇。。。我立即猜到她是怎么回事，说，你是不是直接搜“长江口浮游动物”？答曰是的。我说，那你先搜长江口或者先搜浮游动物，然后在结果中查找。。。

让总结别人做过什么的时候，估计会让人更不能忍受。

说实话在国内有时候确实憋屈。想学到的没有人教，不想学的被逼着做，好容易有个想法会被认为是异想天开，没有想法的时候会被认为是你能力不行，你有了想法数据不够会被认为是虚头花脑跟主流学术界挑事，数据够了会被认为是方法有问题，或者还不够说服力，不够老不够底子厚实，然后还会有一堆的问题。。。。

只有发了英文的if很高的paper，才是唯一的证明自己的办法，但是上面这些内容都存在的情况下，怎么让人发很好的paper？

这次，如果搞不定工作的事情，没有我自己想发挥想法的空间了，要么跟bf直接转行，要么就也申请出国做博后算了。实在是快没有希望了。
