http://finance.sina.com.cn/stock ... /083211734010.shtml

根据中国社科院世界社保研究中心的计算，由于目前个人账户中“坐实”的账户记账额仅为2000亿元，因此全国养老金的“空账”缺口相当于1.7万亿元。而由于养老金至今未实现全国统筹，在统筹层次低下的省区，养老金“缺口”尤其严重。

这种压力，还可能随着老龄化社会的加速而加剧。根据2011年12月中国社科院发布的《2011中国养老金发展报告》，2010年中国基本养老保险基金总支出增速，已高于同期基金总收入近2%。

一个典型的例子，便是养老金“缺口”高达近200亿元的辽宁省。该省在进行了几年做实个人账户的试点之后，最终由于离开了中央财政补贴便陷入“死账”，而导致改革陷入停滞。
