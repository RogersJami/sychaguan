给大家献个快速自家版的做法：
1、绿豆加入小苏打泡水30分钟-1小时；
2、按照1:8比例加水，大火沸腾5-10分钟，小火熬制1小时；
3、关火，加入冰糖皆可

糖的味道有什么分别呢，我看红烧肉很多也是要冰糖、红糖上色的。

冰糖的口感会比较清甜，这是在广东学到的个词。
放冰糖的跟放砂糖的对比，味道就区别开了。

我家绿豆汤就是，少水，高压锅煮上汽煮5分钟。然后自然放气之后，加到适量的水,放冰糖，压10分钟。出品也很糯，省煤气。
