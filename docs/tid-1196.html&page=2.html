方解石 发表于 2012-3-21 17:27:36

发个我的懒人面条，虽然好像在哪里发过了。

面条做法如下：

鸡蛋一个，打碎，加入少量的盐和十三香；

西红柿一个或两个，切块（不用太碎，因为炒的时候可以拿锅铲压碎）。

姜切片或者丝，随便。

葱加不加均可，要加的话稍微来一点，不用切太小。

干虾仁或者虾皮少许。

菠菜一把，约五六棵，或者更多，随便。我一般直接拿剪子将根部剪下来，这样叶片也都分离开了。然后拿水淘两下。

步骤：

下油，炒鸡蛋。鸡蛋刚熟即盛出，放刚才打鸡蛋的碗里面。

略加油，如果第一步剩油多也可以不加。下虾皮和姜，或者有葱稍加点葱。

到虾皮呈现黄色，即可。下西红柿，炒出红色酱汁。

将刚才的鸡蛋倒入，以刚才那个碗舀水，适量，添入国内，此为面条的汤汁（顺便刷碗）。

加少许盐和十三香。烧开。加面条。面条要转着圈放入，以免摞在一起粘结。

将火适当调小。盖锅盖，焖煮约几分钟，待面条八九成熟，下菠菜（或者小青菜或者白菜或者小白菜）。

少煮一下，关火出锅。

这个面条，bf百吃不厌。连续一个月，几乎天天拿这个做早餐。

最重要的是，只需要洗一个锅一个碗就行了。因为用的油也不太多，所以也很好刷。从开始做到吃完，我可以半个小时搞定。我觉得如果赶时间可以更快。
