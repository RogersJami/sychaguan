斗胆解读一下：杀人要杀死，救人要救活。
既然决定了欲“杀”（广义的：不仅仅是字面上的含义）的对象，那必是已犯下了该“杀”之罪，就不可以念情手软，半途而废。
既然决定了欲“救”（也是广义上的）的对象，那也必是该救该助之人，就必须好事做到底，送佛上西天。

hans 发表于 2013-6-26 07:46 static/image/common/back.gif
恩所加，则思无因喜以谬赏；罚所及，则思无以怒而滥刑.

翻译为杀人要杀死，救人要救活。

忙总那是个麻辣翻译，类似当年明月说的：要么不做，要么做绝。

我觉得，人家原文的意思是要皇帝老儿做事谨慎，不要自由心证。

这个《十思疏》对普通人更有价值的是文章开头“求木之长者，必固其根本；欲流之远者，必浚其泉源；思国之安者，必积其德义。源不深而望流之远，根不固而求木之长，德不厚而思国之安，臣虽下愚，知其不可，而况于明哲乎？”
现在，很多人都无法静心固本浚源，一心只想挣快钱，做食利族。
