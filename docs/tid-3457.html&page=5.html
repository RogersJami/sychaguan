毛志飞 发表于 2013-8-7 21:31:27

看来还要普法一下。增值税的管理，实际操作上分增值税一般纳税人和小规模纳税人，其中一般纳税人的标准是工业年销售额50万以上，商业年销售额80万以上，"营改增“企业年销售500万以上，对于达不到销售额标准的，可以选择按小规模纳税人征税，征收率是3%（注意这里是征收率，不是税率）
-------------------------------------------------------------------------------------
实际生活中，税务机关日常管理中是以发票为凭据核定销售额的。打个比方，你开个便利店，每月销售为5万元，但是你只开1万的发票，那你可以按1万申报销售。这在税务机关有个术语叫以票定税，以票管税。在我们国家的生活中，个人消费很少有开发票的，个人消费者通常只有在报销需要的时候，或者看某个商家不顺眼，想保留证据的时候会索要发票。顺便说一句，只有开了发票的销售，才有可能会被统计，所以在个人消费环节，中国的统计数据是失真的，消费占GDP的比重绝对比统计数据里的比重高。
--------------------------------------------------------------------
我做过浙江沿海某地的调查，该地国税2013年1到6月增值税税收入库3亿4千多万，个体户入库税收只有300多万，不到1%的比例，交过增值税的个体工商户，数量只有400多户，而该地国税登记的个体工商户户数为9000多户。同时，在该地国税登记为公司性质的小规模纳税人为2000多户，如果按2万的线划，只有大约10%户数需要纳税。该地纳税排名前100的企业，缴纳了超过90%的增值税。再顺便说句，当地2家银行缴纳了超过80%比例的所得税。只2家银行的利息收入，就超过了当地所有非金融企业的账面利润。
2万的免税线，其实从户数上让一个沿海城市90%的工商业户不要缴税，这条线并不是随便划的。
--------------------------------------------------------------------------------------------------------------------------
虽然实际管理中是以票定税，但是税法上的概念里，业户是要据实申报的，所以如果你的店子月销售5万，开票1万，你申报一万，OK没问题，但是你永远在税法上是有原罪的，税务部门可以查你帐，这种权利税务部门很少使用，但不是不能使用。我说过例子，去年浙江温州有个商人，吹牛自己在美国买了个叫大西洋银行的银行，当时的背景下，温州的民营经济和金融改革都需要找到新的出路，这个商人的吹牛让当地政府信以为真，树为典型，最后商人被省长接见，表扬，没几天，有媒体爆料美国根本不存在大西洋银行。几个月后，该商人因为虚开增值税专用发票坐牢。当然这个商人确实是虚开了专用发票逃税。
我说这些，就是想说在中国讨论政府经济政策，法律，一定不能按书本，按条文来讨论。必须结合现实生活来讨论。

具体到月定额2万免税这个事情，在浙江沿海，可以让90%企业免税，在西北某地，也许就只能让50%业主免税，在某个财政极具恶化的城市，甚至会出现税务机关强令某个原来开票只1万，月纳税300的纳税人按每个月2万申报，纳税600的可能。这大概就是网上讨论问题经常有鸡同鸭讲的感觉的原因，我在浙江沿海，我调查过，我就觉得这个政策很好，这条线划的很巧妙，某个生活在债城的办小企业的网友可能就觉得共产党真他妈操蛋，减税怎么变成增税。
