秦波仁者 发表于 2012-10-6 15:35:16

案例分析：
                                   ‘重庆’牌60度老窖
                                         行销误区
                                 
                                  一、和谐酒厂通吃市场
和谐酒厂位于北京，秉承第一任毛周老总、第二任邓总之营销理念，一贯奉行‘通吃市场’战略---对外宣称‘统一战线’。
近三十年，和谐酒厂在通吃市场主体思想指导下，又引进了‘宝洁’公司多品牌战略【注：海飞丝、潘婷、飘柔---都属于母公司‘宝洁’】。
现在，和谐酒厂旗下有主打北上广一线城市‘资本’系列、主攻二三四五线城市‘中产’系列、主守农村小城镇‘小康’系列。
                                     二、重庆老窖分蛋糕
2007年，小康系列白酒中又横空出世‘重庆’牌60度老窖，该产品以占领低端市场为重心，力求拓展小康系列比较薄弱之空白区域。因为策划思路超前、切入点准、价位通路对口，该品牌白酒风靡全国终端，线上线下一片飘红。
2011年中，重庆牌欲抢占中高端市场，并命名此次战役为‘分蛋糕’。同时，该战役与北非、中东若干白酒品牌‘抢滩登陆’行动有形成共振之可能，这不免引起和谐总厂警觉，于是重庆牌被紧急叫停。
                               三、让喝低档酒的人群能喝得起中档
和谐总厂依然秉承通吃战略，但总部认为：过分张扬‘低端’品项无疑会损害整体品牌形象，降低档次并失去中高端金牛消费群，迫使能贡献很大利润的受众选择美英等西方‘普萄’酒。况且，重庆牌60度老窖所拥有消费群都是一些：就着二两花生米能喝一斤的酒仙，酒一上头又喜欢上街的主儿。所以，重庆分公司经理与打假部门负责人被调离就在所难免。
另外，重庆分公司与总厂营销思路也存在差异，总厂认为：应该等待国民收入倍增计划逐步实现，引导低端消费群饮用‘中产’系列---毕竟这是和谐酒厂利润最大一块。而重庆分公司刻意迎合低端做法是短视，而且无益于长远布局。
                                     四、白酒界要转型
鉴于重庆公司在市场开拓中成功做法，总厂决定继续保留副总老黄职位并嘱其改弦更张与总厂保持战略一致；
至于分公司经理老薄的问题可能没有以前国焘、高岗两位分厂负责人严重，所以暂时没被冠以‘分裂总厂、反总厂’罪名。
2012年中，因为‘钓鱼岛’品牌市场归属之争，部分‘粉丝’上街晃悠，所幸重庆牌60度老窖‘下架’，喜欢这一口的老兄们没有‘上头’，事态才不至于白热化。
如此看来，总厂对重庆分公司的处理还算及时？
