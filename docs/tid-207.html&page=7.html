12自然法

一、“礼即自然法”有着其正确和不足。

二、在中国传统中，当礼与法相抵触，那么法一定是虚假的。

三、礼的基本概念构想着一个以义务为中心且依靠义务来调整的社会。自然法的本质是种正义论，礼的本质则是家族的伦常。礼法是一个封闭的、非革命性的体系，罗马的自然法对风俗习惯进行有见识的批判，而礼并非与风俗、人情相悖，想象中的和谐状态恰好就是自然。

四、中国礼法没有超验的世界，反映就是“天人合一”。

五、宇宙本根，乃人伦道德之根本，人伦道德，乃宇宙本根之流行发现，这是超验主义缺乏的根本。

六、中国历代的思维未曾改变。

七、我们本身是完善的，无西方的原罪说。

八、礼的本质是自然的，“失礼则入刑”。

九、一切，源于天道和谐的观点。天道，就是自然的自然法。

读后感：“礼即自然法”，这种说法，是他们的应用范畴等同起来，从本质上来说，礼是一种秩序，礼外还有世界，中国的华夷之辨就是这种潜在观点的现实反映，真正的触及到了我们这个民族的文化核心；而自然法，则是万法之法，凌驾于一切之上，控制世间万物。这两种说法，在其思想内核的本质上是截然不同，并有着相互对立的方面。中国传统中，礼和法是两个层次，正因为我们之外还有世界，还有着各式各样的法则，这种法则对更高层次礼的侵犯，都是不可原谅的，在我们眼中，维护社会的法和万物自由竞争的自然法则，是平等的，以及低层次的。礼的本质既然是种义务，那么就意味着我们在这个集体中，本身就得到了天大的恩惠，我们与其说是性善论，不如说是组织或是国家对于个人的汲取方面，天生就是无罪的。承认这个礼，就等于承认我们都是社会人，我们的天性有部分肯定是社会性的，个人主义不可能孕育。超验的本身即是一种建立在精神世界上相互关联的秩序，我们即是秩序的本身，所以超验是不存在的。“天人合一”的天不是万物，而是道，是秩序。天道，并不是自然的自然法，而是建立在自然万物基础上的组织体系的自然法。

13转捩点：过去与未来（本书完）

一、中国法律的历史。

二、清末的法律体系，外国列强伴随着侵略深深地影响着法律，文化冲突第一次影响着中国人。

三、中国所面临世界的挑战，当文明机体有益的东西也显示出弊害时，是一个文明濒临死地的征兆。

四、现代法学所诞生的需求，中国并没有，价值体系的接受必然变为挑战。

读后感：这篇文章写于文明危难之际，当时作者的判断是中国传统文明正处于濒死的征兆，家已经分解，国企因为承担太多的责任，也濒临死亡。一个文明，当出现不了自身价值的载体，注定要灭亡，换一种说法就是，文明的第一要务就是要有自身的价值观，并在社会中存在其载体。中国人本身就是秩序的反映，这个实现，决定了我们每个人都是中华文明，因此我们的文明表现了强大的韧性，这是被低估的部分。当秩序的本身反映在资本上时，国家资本在我们的社会具有天然正义；当秩序反映在法律上面，我们发现每个人身上的法律各不相同，也许，职业法（行规）更符合中国对法的需求，是法律中国化的一个突破口。目前，法律作为规则而成为不了秩序，这是中国现代法的先天缺陷；而在市井阶层中，又受到财富的强大冲击，处于上下不讨好的境界，中国法律，将何去何从。
