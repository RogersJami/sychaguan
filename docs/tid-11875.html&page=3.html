大国小白菜 发表于 2014-10-19 11:48 static/image/common/back.gif
不是三胖将军无能，实在是旁边土共蜀黍太狡猾，土共表示这么多年的包袱积压到这一代，得丢掉一些了，不然 ...

      你这个理解似是而非，基本上是不正确的。
      我们不讲小胖子能力大小，也不讲蜀黍是不是狡猾的问题，单独来研究一下包袱的问题。
      什么叫包袱呢？包袱是指为你带来负荷却不能产生效益的人和事。近些年来关于包袱的奇谈怪论很多，远的讲到当年援助阿尔巴尼亚的事情，近的讲到嫦娥划不划算，金家的小朝廷自然也是历来的热门话题。我们不要去跟那些有小国寡民思维的家伙计较，那些人的思想就是国家把钱发到他的口袋里才是物尽其用，要不然全部是浪费。也不要去研究半岛的缓冲能力问题，半岛的战术效能在下降，但是战略效能一点都没有下降，这都已经是有定论的事情，不必再费口水去评说。
      那么朝鲜对于我们来说是不是纯粹的包袱呢？当然不是！蜀黍很狡猾，自然不会干那种很2的事情。根本原因在于蜀黍的指导思想发生了变化，而且是根本性的变化。我们之前谋划国际局势的时候，侧重于谋势，对短期的实利不太重视，或者说有时候是故意模糊。比较典型的就是对阿尔巴尼亚的援助，那个势谋得过于感情化了，长远利益打了水漂。现在我们更加侧重于维护自身的核心利益，这个核心利益既有眼前的利益，也有长远的利益。这种变化就好比下围棋的棋手转变了下棋的风格，之前光布局，很少涉及局部争空，现在既要布局，也要争空。
      所以，朝鲜不是包袱，我们不会丢的。但是我们会以更加务实的态度去处理对朝关系，一切决策的评判标准是是否符合我们的核心利益，在符合核心利益的前提下，才可以继续畅谈各种友谊，否则，该干嘛干嘛。
