建议你看看《五百年来谁著史》一书，诸多促进近现代国家发展兴盛的因素中，国家能力、科学技术、地理大发现等因素都扮演着重要的角色，而资本的作用则远没有我们原来想像的那么大。对比一下英、法、荷、德四国的发展历程就可以看出来，最纯粹的资本主义国家荷兰，因为资本的投机性和软弱性最先出局；西欧的原生大国法国，在大革命后有大批的政治精英被铲除，国家能力大幅度削弱，导致其在后来的欧洲和世界争霸中屡战屡败；英国的大资本家大部分是从贵族转化而来，能文能商，国家能力和经济科技同时大幅度发展，再加上孤悬海外的岛国优势，使英国终成日不落帝国；德国则是在资本发展的先机已经不在的情况下，大力发展国家资本主义，使德国后来居上。
