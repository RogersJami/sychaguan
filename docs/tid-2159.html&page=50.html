解决这个问题的办法有很多,比如:
1)家长多存些积蓄,为孩子多负担一些学杂费(为啥亚裔家庭可以做到,而其他族裔做不到?);
2)政府保证对州立院校的资金支持,抑制学费增长的幅度;
3)鼓励企业招收新毕业生,提供财政补贴;

作为老百姓,我们只能做到(1).
