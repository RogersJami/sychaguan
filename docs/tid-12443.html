极品女的故事

本帖最后由 qli872 于 2015-1-10 09:34 编辑

是不是写这个故事，我纠结了好久。因为中国人讲究骂人不揭短，把这个故事写出来，不够厚道。但是这个故事又比较典型、有教育意义，不写出来实在可惜。考虑到茶馆比较小众，写出来也不至于流传出去，就把它写出来吧。

1

我出国是比较早的，当时还是2字头的年龄。周围的同胞大都在35岁、甚至40岁以上。远在异国他乡，自然是抱团取暖。当时结交的一帮朋友，一直交往至今。我们有空就往一块聚，有事就一起商量，相互帮衬。谁家孩子上了大学，我们一起庆祝，谁家孩子结婚，我们是亲友团。十几年下来，在这远离祖国和亲人的地方，我们真的像骨肉相连的亲人一样。

这些比我年长的大哥、大姐们一直对我照顾有加。尤其是生活上，他们比我会做饭，经常给我开小灶改善生活。我也把他们当自己的亲哥、亲姐对待。我不喝酒，所以聚会的时候我就是司机、跑堂；一起出去玩也都是我来张罗。有一次出去玩船，我就给大家做船长。

10几年下来，各人的际遇不同，生活情况各异，但是相处起来和刚认识的时候没有太多差别。

这其中，有一个不和谐音符，大家都很烦她，但是又不好意思把她听出去。她呢，也就厚着脸皮跟我们混。每次聚会、活动，只要有她在，气氛就会差很多。暂且称她为 X 吧。

X 把我惹翻，是在我们这伙人认识10周年的时候。

有一次我们一起聚会，突然发现我们认识快十年了。就商量庆祝一下的事情。我说，我有阵子没张罗了，到我家去吧。W说，“你儿子喜欢到我家玩，趁着天好，让他多来玩玩，还是到我家吧。”W家有个小农场，还有个大房车，我儿子特别喜欢。我想想也是，就说，“干脆，你也别准备了。我到唐人街买几个菜，直接带着酒菜过来。用你家的场地，我来张罗。”事情就这么定了。

到了聚会那天，我去唐人街的xxxxx中餐厅买了些凉菜、热菜，包好，又沽了些酒，开车直奔W家的农场。W又烤了些肉，10周年聚会的准备工作就差不多了。

等大家都到了，X才姗姗来迟。X到了以后，第一个问题是，“你这菜都是从哪儿买的呀？” 我顺口回答，“xxxxx中餐厅。” X 接下来的话差点把我的鼻子给气歪了:

X说，“xxxxx的菜码挺大，就是味不咋好，我们去吃过几次，就不再去了。”

我之所以生气，有几个原因：

1、由于X一直租单元房住，她又不舍得租个大点的，没有场地。因此，过去多少年，她没组织过聚会，一直是跟着大家白吃。她离婚嫁了个老外，又生了个孩子，连满月酒都是T老姐在家给她操办的。

2、以我对她的了解，她那段话无非是想表达两个意思：1、她下过的馆子比我们多，比我们有档次；2、我们常吃的xxxxx中餐厅是她看不上的餐馆。

妈的，白吃白喝，还要挑；一家n口人挤一套一居室，还要显得比我们住独栋的有档次。

什么人那。

极品女为什么一直租公寓呢？不是因为没钱，也不是因为工作不好，原因非常奇葩。我称之为极品，这算是主要原因。

我们这波人初识的时候，条件最好的是W家。他们俩不仅出国前底子厚，而且到这边收入也不错，一直引导潮流：第一个买房、有房车、现在住在城市边的农场里。我们这拨人的很多事都是他挑头张罗。W比我大10岁，对我极好，我也很尊重他。他张罗事的时候，我一直是他的助手。

除W家外，论当时的综合条件，我家和X家各有千秋。X的前夫在国内是个干部子弟、做生意，因此他们的底子厚，但是她的前夫出国后没有一技之长，因此于我家比显得后劲不足。还有一条，就是我们俩比X两口子年轻。

出国后必然要走的两步是找工作和买房。

前文说过：W家不仅出国前底子厚，而且到这边很快都有了不错的工作，因此，第一个买房。W家的第一套房，价格N万。

W家看房的时候，X家也在看房。每次聚会都会跟我们说，她上个星期刚刚看了什么样的房子(这个时候，她的前夫一般保持沉默)。

W家买房后，X还在看房。不过，她看房的价位升到了(N+2)万以上。每次聚会，她还是会跟我们说，她上个星期刚刚看了什么样的房子(都是(N+2)万以上的)。

不久后，我家买了房。由于房价一直稳步增长，加之我家从W家买房取得了些经验教训，我家买房的价格是(N+2.5)万。

X接着看房，不过价位升到了(N+5)万以上。每次聚会，她还是会跟我们说，她上个星期刚刚看了什么样的房子，都是(N+5)万以上的，听她那口气，这个价位的房子她手到擒来，只是没找到满意的。

我们这拨人中，条件最差的是K。K他们两口子都是50后。英语不好，专业也比较偏，只能打工；来自国内小城市，家底也不厚。我们这帮兄弟，先是帮K找了个还算凑合的专业工作，又担保K买了个小房。

就这样，连K都买房了，X还在看房。

然后，由于利率长期走低，房价开始飙升。每次聚会，X还会跟我们说，她上个星期刚刚看了什么样的房子。看房的价格，也随着房价的增长而增长。

然后，房价在几年内翻番。

再然后，X把前夫踹了，嫁了个有个小房的当地人。

(这只是开始)

4.

X 一直想一把买个大房把大家都压下去。奈何她省吃俭用、抠抠唆嗦省下的几个钱赶不上房价上涨的速度，离自己的目标越来越远。最后仅仅因为面子没买房子，狠狠地亏了一笔。可是其另辟蹊径，换了个洋老公，算是扳回了劣势。洋老公有个小房，在房价飙升的背景下，有个小房和没房差得可多了去了。加上离婚分得的财产，其帐面看起来一点不差。

可是.........

X 和洋老公结婚后突然不想工作了，辞职做全职太太。按她的话说，她的洋婆婆一辈子没工作过，那才是她的目标。

这边的全职太太并不少，相夫教子，全力培养下一代也是一个十分有意义的任务。不过，对生活的要求就不能像两口子工作那么高。

问题是，X 还惦记着买个大房子呢，一个人的收入无论怎么节俭都是不够的。于是，婚后不久，洋老公就去了现场(现场环境恶劣，补助很高，一个人可以挣两个人的钱)。洋老公的新工作是基于现场的，因此公司只付搬家费、不付房租。以X的抠门，这边房子自己住，那边房子要付房租，如何了得。于是，洋老公拿到新工作后不久，X 就卖掉了小房，带着和前夫的孩子，搬到了现场附近那个镇子。

这是我称其为极品的另外一个原因。但凡有学龄孩子的中国父母，在选择住处、工作、乃至所生活的城市时，考虑的主要因素之一就是孩子上学。她直接带着孩子去这么小的一个居民以红脖为主的镇子去上学，我都不知道说什么好。我认识几个在此工作的技术工人，都是把老婆孩子留在本市，自己出钱在某镇租一个简单的地方，然后来回跑。把孩子搬到某镇上学的，就我知道的，仅此一例。

X 在某镇生活期间，做了两件非常奇葩的事情。

一是，X在此期间做家庭主妇，然而接送孩子上学的却是她来探亲的父亲。其父有一次送孩子上学，被狗咬伤；
二是，某镇生活实在无聊，X忍无可忍，把前夫的孩子留给洋老公，自己回国旅游数月。
