道可道 发表于 2012-4-15 13:58:22

首先呢，既然老美是上司，要先想想值不值得和他理论，会不会被穿小鞋。如果还是想理论的话，必须要承认，近现代确实是西风压倒东风，科学技术上是西方引领潮流；文化艺术上，东西方不存在谁领先谁的问题，本来就是不同流派。

所以再看创新能力这个问题，老外的论点是偏科学技术，还是文化艺术：如果是文化艺术，没什么辨的，文无第一；如果是科学技术，确实西方创新的多一些，但是说中国人没有创新能力，是绝对不可能的。

问题是，你要是举一两个例子来证明中国人有创新能力，老外可能还是偏执的认为太少啊，个例啊什么的，那就没什么辩论的了，他就是有偏见，浆糊脑袋不值得一争。要是你老婆觉得一两个例子能理论的话，可以再去找，我相信肯定能找到。

至于盖茨和乔布斯，他们成功的原因是多方面的，并不仅仅是所谓的创新能力，而是美国社会的一个系统土壤的条件。不过，从小处来说，他们都是copy了施乐公司创造的图形用户界面(Graphic User Interface), 乔布斯先copy, 盖茨再跟进，可以参考此链接：GUI。
当然，这界面也是西方人发明的，这一论据仅仅是针对老美说的盖茨乔布斯而已，要是老美继续死皮赖脸说施乐也是西方的，还是和他的观点一致，那就没啥理论的了。

其实呢，只要有资源，实力，人才会有的，创造也会有的，现在的中国，在某些领域应该已经在培养自己的创新人才了。
