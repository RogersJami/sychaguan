按照你提供的数据，中国发展确实是遇到了缺水的障碍。但是作为一个行业来说或者单纯就一个自来水厂来说，在现在或者说今后5年时间会获得很大收益，这个恐怕未必啊。

楼主所提出的讨论题目是【10年后什么行业职业最火？】
参照当下情况，所谓的‘火’应该定义为：从业人员收益远远超出该国其他行业。
那么现在在中国哪些行业很火呢？
个人认为是国家垄断的——电信行业（移动、联通、中国电信）、电力行业（南方电网、国家电网）、3桶油（中石油、中石化、中海油），在一些省份比如说 云南，还可以添加上烟草行业。

这些可算作是没有争议的‘火’的行业了吧。至于说待遇好到什么地步？其中、高层收入，龙战无法给出量化的数据。其基层员工的收入，我倒是了解一些。比如我老家的电力公司基层工作人员，只要是从业3年以上的，其每月收入都在3500左右。当地平均工资也就2500~~2600的样子。换言之，在当地他们的收入属于中高档的，同时也比其他行业的人达到这个工资水平要少奋斗一些时间。

为什么这些行业能拿到超出平均工资40%的收入？我认为其一是国家垄断（能从市场上获得巨额利润，利用自身优势有的时候甚至能少缴税，比如前些年三桶油中就有人到中央诉苦说亏钱了，需要政府提供补贴等等）；其二是需求旺盛（这个多亏了中国经济发展给力啊）。

水处理行业，能具备第二个条件，但是不具备第一个条件。水厂属于市政公用设施，在定价的时候要在发改委组织下进行价格听证会。只要头上还有这个箍（政府指导价），它就不会成为那种‘火’的行业。

关于“水”的问题，有几点要说。
按照你提供的数据，中国发展确实是遇到了缺水的障碍。但是作为一个行业 ...

水资源，这个概念比公用事业用水、水处理的范围更大。比如节水技术、设备、瓶装水、气候变化导致的水资源分布（冰川融化、持续干旱）变化、海水倒灌、海水入侵地下水等等都在其中。

目前水处理行业是不太景气，但并不表示未来依然不景气。当前在某些情况下（工矿业污染、废水废气毒气泄漏），人们愿意付出十几倍的价钱来得到清洁的饮用水，这已经说明行业潜力了。
