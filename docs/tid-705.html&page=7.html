为啥说物流专业比较有发展前途?

atene 发表于 2012-4-9 03:18:36

本帖最后由 atene 于 2012-4-9 03:34 编辑

从大面讲是因为，国际贸易和网络信息技术的发展使得物流管理在经济和贸易中扮演越来越重要的角色；就微观层面而言，过去企业对前（中）端（销售、客户服务、制造流程等）一般比较上心，但是对后端的供应商管理、物流管理的重视不够，这一大块其实对企业提高效率、保证客户满意度、优化程序、降低费用等能做出很大的贡献，近些年越来越多的企业开始挖掘这方面的潜能，所以个人觉得这方面应该比较有发展前途。

另，个人觉得工业设计也是很有前途的职业，从那末多人排队购买Apple家的产品就可一窥，中国制造要升级的一个重要环节就是产品的设计，当然还有品牌的管理。可惜，相对于物流管理，工业设计的门槛比较高，一般人不太可能转行过去。
