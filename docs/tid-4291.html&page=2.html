在大别山区看过一个生态沼气项目，各农户家的猪舍集中，自己负责喂养，猪粪统一流入发酵池，沼气 ...

相比之下，也许发展养牛业，利用牛粪生产沼气更为有用。一般的沼气池需要大量的水，同时产生的废液难以利用。利用牛粪生产沼气不需要水，只要采用堆肥方式即可。因为牛粪便中含有大量甲烷细菌，发酵更快。同时牛粪发酵产甲烷后可以减少甲烷的排放，发酵后牛粪作为熟有机肥也可以方便用于农田。多养牛，多发酵，多产粮，粮食产量让三哥国再也不能吹牛，利多而弊少。
