如何毁掉一个一线大牌

本帖最后由 qli872 于 2015-3-28 02:37 编辑

假定：

1、您成功地忽悠了某一线大牌(如夏奈尔，LV，etc.)的老板，获得了独家推广该品牌本季新产品的机会。你想借此机会彻底毁掉这个品牌。
2、您一旦开始干坏事，必须尽快把该品牌摧毁。否则该品牌就有机会把你炒掉、然后慢慢挽回。

请做一个推广计划。

  1. 先全媒体造势，让所有人都知道知道要出新品了，而且时间精确到小时，比如3/20 12：00，准时开售。
  
  2. 在所有渠道销售，特别是电商一个都不能漏下。

  3. 大量备货。

  4. 第一天正常开卖。

  5. 第二天全线打1折，利用网络/微信/短信等散布消息，可以参考《穹顶》手法。价格便宜量又足，让组装厂小弟小妹都人手一个。

  6. 第三天，直接闪人。

一线大牌，比如LV就是奢侈品或者说时尚品了。

那么它的销售人群主要是社会上层等高收入阶层了，当然不排除某些花几个月工资买包的小资了，而且应该有很强的时间性，一个季节的卖不好，就很产生很大的影响。

而这些人所注重的是某种奢侈品效应，或者我用这些东西，说明我比较时尚。

所以我觉得最好花钱找一些一线的时尚评论杂志或者网络在一个时间点爆出大量差评（什么黑工童工了，血钻战争犯了等等，设计不够时尚），然后自己开新闻发布会承认这些差评的合理性和承认失败。

找一家乡村地下加工厂，最大程度的节省成本，偷工减料，粗制滥造，最后贴上大牌logo。

发起一轮又一轮的宣传攻势，介绍产品是如何的设计精美，用料讲究，是精英与上流人士的不二之选。

宣布：x月x日正式销售。 一拖再拖，正式销售日期更改N次。

终于正式开卖，先在专卖店推出，对VIP与忠实顾客不减价，且服务态度恶劣。对顾客质询与投诉回应： 你不懂，不配用。

半个月后，在淘宝上买一送一促销，甩卖。
