怡沫清泉 发表于 2017-12-28 19:09 static/image/common/back.gif
月中到月底，天然气价格国内最高到1.2万，现在已经跌到7000不到了。

这个可以到上海石油天然气交易中心的官网看看，有各省每天的LNG出厂批发价公布。

其实此轮包括“煤改气”在内的环保大突击的肇因在于2016年12月25日通过，将在2018年1月1日实施的《中华人民共和国环境保护税法》。不过由于此法早在去年就通过了，留出了比较充裕的计划和准备时间。特别是在近年全球天然气供大于求的的情况之下，原本只要做好规划完全可以泰然处之。

但前后两任环保部长均是“新官上任三把火”，在8月份联合6部委加码下发的《京津冀及周边地区2017-2018年秋冬季大气污染综合治理攻坚行动方案》及帝都率先垂范的效应下，打破了各地和三桶油在年初的统筹安排。而在下半年一切为“十九大”让路的气氛下，多一事不如少一事也让各级官员直到大会尘埃落定之后才开始紧急执行。由于大幅扩展“煤改气”适用范围导致国内LNG供需严重失衡，出现多地面临缺气以及需要大范围调气的局面，一时之间闹得鸡飞狗跳，与对岸由于民粹裹挟的“废核”行动倒颇有几分相似之处。

不过经此一役，由于矫枉过正而出现的政策反复也及时为过于激进的环保政策敲响了警钟，为今后如何实践“实事求是”思想路线，并与经济发展相适应提出了新要求。

而由于近三十年“摸着石头过河”的常态，“先污染后治理”仍旧具有一定的体制惯性。所以最近一系列强力环保新规则的出台也为即将到来的“撸起袖子加油干”设立了一个底线思维。

或许在发展与环保之间取得制衡，这就是所谓开创新时代的“新常态”吧。

希望经过此番折腾之后，全国百姓都能过个安心年。

也祝茶友们元旦快乐！
