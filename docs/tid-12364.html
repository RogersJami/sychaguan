政经杂谈二：米国在第一岛链的工业资产 与 中国的国防安全

众所周知，中国的经济中心，主要工商业，分布在东不沿海省份！
而中国近代面临的外来军事入侵，主要来自东部海上！
因此，将国防安全的前沿阵地，向东面的外海推进，从军事上，控制第一岛链，至观重要！

目前的第一岛链，处于米国的军事控制之下，
韩国、日本、台湾、新加坡，有米国的大量高端工业资产！
在谈判桌上，米国不太可能放弃这些工业资产，将相关地盘，移交给中国的！

随着中美军事力量的消涨，第一岛链，迟早是中国的盘中餐！
那么，这个问题，怎么解决呢？

一、米国将这些资产转移走。

这个问题，首先面临的是，往哪里转移？
日本有1.26亿人口，韩国4900万，台湾2300万，新加坡540万，合计约2亿人口！
这还不仅仅是人口数量的问题，还涉及到人口素质、教育水平、社会组织性、政治经济制度、基础设施等等因素，
去哪里找一个适合这些高端工业转移的地区呢？

1.1 拉美？
拉美的人口数量、资源储量，确实不错！
但是，拉美地区早就实现政治上的多党民主化了，早就从社会心理文化上去工业化了，只适合充当资源供应地和产品倾销市场，并不适合搞高端工业化！
再说了，米国只想让拉美做政治经济附庸，没有在哪里搞高端工业化的政治意愿！

1.2 非洲
非洲的人口素质、社会心理文化、政治经济制度、基础设施等方面，并不适合搞工业化，况且，列强也没有在非洲搞工业化的政治意愿，无须考虑。

1.3 孟加拉、印度、巴基斯坦等南亚国家
既然第一岛链将不可避免地落入中国手里，那么，南亚，自然也是难以避免的。
转移到这里，岂不是从中国的左边口袋拿出来，又送进中国的右边口袋吗？

1.4 中东欧国家
这个地区的人口规模、人口素质、社会心理文化、政治经济制度、基础设施等情况，看起来确实不错！
再者，西边有欧盟这个发达经济体，东边有毛子这个能源资源供应地，各方面都很理想！
可是，欠缺的，还是政治意愿、地缘政治因素！
当今世界的经济格局，中美欧三强并立！
如果欧盟和毛子媾和，就能在未来跟中美争雄；如果欧盟和毛子彻底翻脸，老死不相往来，那么丫的就出局了！
这个，也正是米国主导北约不断东扩，中美挑起乌克兰危机的根本原因，撕裂欧毛！
米国要是把第一岛链的工业资产，转移到中东欧国家去，如何保证它对局势的控制？落入欧盟或者毛子之手，怎么办？
如果某一天，欧盟、中东欧国家和毛子媾和了，将形成一个经济巨无霸，那岂不是为渊驱鱼、为丛驱雀？搬起石头砸自己的脚？

二、将这些工业资产，卖给中国。

对于华尔街肥猫们，兄弟连，来说，无非是股权换债权！
