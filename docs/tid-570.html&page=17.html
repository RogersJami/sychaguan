嘿嘿，我觉得人脑只不过是一台人肉电脑，人脑能算电脑也就能算，只不过需要建立一套复杂的神经网络模型罢了。日本现在正研究这东西，拿一个超级电脑模拟人脑已知的各种神经网络模型，包括那种自动激发的意识网络模型，想把那电脑做成个人。。。我看过相关的几篇文章，小日本在这方面搞的还是挺细的。
人脑有其局限性，由于是生物结构的，所以神经元在做运算时的背景噪声相当大，大多数的神经链接的作用不是思考而是为了平均信号以降低噪声，所以真正起作用的神经元要少的多。这就是为啥现在的傻瓜相机用一个小芯片就能完成人脸识别。随着芯片结构进一步的进步我觉得人类用电脑进行科学研究完全是可能的。
