suqier 发表于 2012-5-6 12:52 static/image/common/back.gif
缺氧环境应该怎么做？封闭容器吹氮气？
点根蜡烛在里面是不是会改变水的ph值所以不能用？



你点蜡烛在里面，只能造成空气环境缺氧，短时间内水体里面还是缺不了氧。

其实就是气体在水中的溶解度，氧分压，让水表面的氧分压减少，另外往里面充氮气，增大氮气的氧分压。但是一定要密闭容器，也就是说水面上的氧气分压也是减少到近似没有的。

大量含水的生物，就是测出来体积后，用元素分析仪求氮和氧的含量来估算它身体能量的含量。体积用图像扫描的办法再根据公式近似求出来。用排水量法太麻烦了，而且很难求出来体积很小的物体的体积。
