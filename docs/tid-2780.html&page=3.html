stevezhouxj 发表于 2013-10-9 10:14:02

个人的一点看法：

北京、上海、广州这类资本和人口都处于虹吸状态下的一线大城市，房价要跌太难了，难到几乎不会发生，除非是经济危机、自然灾害危机或战争危机发生。

一些二线的地方，比如浙江的温州等二线城市，如果处于本地资本富裕时期，就会发生房价爆炒暴涨，可一旦发生资金流出或短缺危机，就会暴跌。这些地方的人口流入不如北上广，刚需们撑不起太高的房价来。
温州最近的暴跌就是资本短缺导致。

至于如辽宁营口，山东海阳这类三、四线小城市，房子很难卖高价，除非是本地被某个开放商彻底垄断搞饥饿营销。
比如我知道的情况是海阳的房子3000多一平都卖不动，原来以为建核电站后会好卖，结果房子太多，进来的人口发现可选择的房源如此多，租房最划算，结果都是租房不买房（此信息来自某远亲，他是去海阳建核电站的，本打算在当地买房，后发现租房更划算，他的同事们都是如此）。
