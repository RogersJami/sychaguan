redapp 发表于 2014-4-2 00:38 static/image/common/back.gif
今天（应是昨天），听了一个消息来源比较靠谱的人说，中央对大多数城市的楼市限令，马上都要下放给地方政 ...
取消限令死得更快，新土豆在菜贩手中，人家负债高，得算时间成本的，现金流出问题的关键时候敢于出血大降价。二手土豆买家一般不计算土豆时间成本，只看账面，负债比例低，现金流断的可能性也小，被套死扛的多，割肉止损的少，轻易不会降价。

如果二手土豆限令取消，很多新土豆买主分流到二手土豆，新土豆的回款更少，还不了债，逼菜贩降价回款。

所以场内这点资金，用于新土豆则维持价格时间能长点，用于二手土豆，新土豆会降菜贩也敢降，二手土豆自然也得降，恶性循环。
