葡萄 发表于 2013-3-13 04:19:04

给一个约束范围，以前和朋友讨论经济政策的有效循环的时候范围约束在这里--既经济政策有效部分是只投入能产生利润并支持扩大再生产的人群部分。这部分人必须有可支配收入，购买产业导向过程中需要通过不断升级所带来的不断推出的心产品。只有这样产业升级才有利润不断继续投入。这部分人按照中国的贫困标准，中国有三千万不在经济虚幻之中的。如果按照联合国的绝对标准，每天收入在2美元以下的这部分人群实际将扩大到以亿来计算。如果是人均收入每天五美元的话。恐怕这部分人口要占到中国人口总量的四分之一。而你前面列举的通胀是把这部分人计算全国平均水平的。这样的数据会吧区域性通胀。尤其是在调控环节中涉及债务链条及其传导的债务转移，最终都只会集中在我说的那2.3亿人之中产生较大的再分配。也就是说，产生通胀实际影响社会阶层再分配，我们不能从全国十三亿人角度来看待。而是从通胀实际负担中，把已经退出经济增长再循环的相对贫困人口从模型里剔除。在这个基础上，我说的其实已经在那个时代发生过。而如何解决因结构调整带来的各种问题尤其是我们这里提到的恶性通胀，我们看新领导班子会如何应对。

王石说房地产泡沫破灭会引起革命，算是对习总关于汲取苏联亡国教训的一个来自民营企业家的一个解读。李稻葵的观点也代表了社会部分阶层的利益倾向。而不管领导层最后选的什么，就在眼前了。看年中变数就是了。具体啥实际不重要，理解什么是高度的不确定就可以了。
