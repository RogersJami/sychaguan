舒亦逡 发表于 2012-4-8 08:43 static/image/common/back.gif
信息时代怎么了，在所谓互联网上存在的信息不及实际中的十分之一。当然，我拿不出证据。

任何信息在一定范围内都是“公开的秘密”。信息时代的区别不过在与“广而告之”的成本极低、速度极快、规模极大。会不会“人所皆知”只在于：受体是否感兴趣接收，“一定范围内”是否存在想传播的人。像万向这种事情，目前这种状况下，“一定范围内”缺乏有意愿对外传播的人，此外也缺乏乐意接受的人。见惯不惊，绝大多数人在自己的日常生活中看到类似的事情已经很多，对“万向”不感兴趣是自然的。
问题在于，当某些条件发生变化时，“万向”这样的事情就会大大刺激全国民众。一个是“一定范围内”希望传播这些消息的人大增到必然此类事情会被传播出去；另一个是民众躁动不安，愿意以此类消息为导火索骚动起来。信息化和工业化后，社会管理与传统社会管理的不同主要在于此，而不是其他。为了应对这种不同，很多东西不得不做相应调整，否则管理失效的后果是非常不愉快的。
