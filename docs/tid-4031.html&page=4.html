潜水良久，在茶馆受益颇多，是个万年伸手党。看到大家谈股票，也厚着脸皮来交流一下过去几年做票的感想。

同主力相比，小散的信息来源、专业分析以及资金号召力都无法同主力相提并论，唯一优势就在于资金小，可以快进快出。再加上A股是绝对的投机市，普通中小投资者，按巴菲特那一套来搞，很难成功。依托技术，跟随趋势，跑得快，更适合小散。

在入市之前除了一些基本知识，还要研究一下相关的技术。稳妥一点的，先通过虚拟交易建立并完善自己交易系统，有点心得了，再用小资金交易一段时间，能稳定盈利了，可以考虑慢慢增加投入。市场永远不缺少机会，不要因为一时的贪念去做市场的炮灰。

这个交易系统最好能满足以下两点：
1. 要有完备的买卖点，也就是说，你在买之前，要明白为什么买，同时也要清楚该何时卖（买入逻辑被破坏）
2. 要有资金管理的意识

单就技术而言，主要流派有道氏、江恩、波浪、混沌、缠论等等。
所有在市场上存活的理论都有其高明之处，用好了都能赚钱。最好的技术理论，是在级别的基础上对市场进行解构，并结合动力学，寻找大概率买点。
结构分解做得最好的，我认为是缠论。但越灵活越根本性的理论，越难把握，而且因为缠师的文字比较有性格，容易把人带沟里去。据说死在缠论上的人特别多，我自己也一直在努力学习中，目前还活着{:soso_e113:}

无论用哪种技术，对散户而言要想赚钱，要留意以下几点：
1. 关注板块的轮动，分析题材热点，哪些热点可以中长期关注，哪些是搞一把就死，心理大概有个数。这算是散户的基本面分析。
2. 确立自己操作的级别，只做该级别上升通道中的股票，所有均线向下发散的那种不要碰，因为你根本不知道他什么时候能活过来，活过来后也大多先有个恢复期再拉升。一般来讲，上涨的第二波是速度最快，涨幅也最大的，波浪理论管这个叫三浪，优先做这个。
3. 要专注，不要拿太多票，关照不过来。与其广撒网，不如努力提高技术，增加准确率，再加上合适的资金管理，会有更好的收益。
4. 不管学哪种理论，先读读livermore和江恩，里面有很多原则性的东西，是两位大牛用绳命写成的，非常宝贵。
5. 不废一法，不立一法，能赚钱的理论就是好理论。
6. 炒股没有捷径，不是一朝一夕就可得道升天。勤于思考，谨慎操作，坚持复盘，不断总结反思，希望能有一天终成大道{:soso_e113:}
