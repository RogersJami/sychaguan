看到一个关于油价的信息（来自［占豪评论］）：
    俄罗斯对华供气暂时的规模是380亿立方米每年，有报道称是4000亿美元。按这个价格是每千立方米350美元。这个价格比俄罗斯输往欧洲的价格低30美元，比中国从土库曼斯坦进口的气价高30到50美元估计，中国从土库曼斯坦进口的是290美元到420美元，最低价格是中石油在那开采所以价格低。但这个价格只是媒体猜测，并非公布的价格，俄罗斯方面称价格是商业机密，不能透露。
    根据过去双方谈判的价格，中国希望价格和土库曼斯坦进口的低价相当，即290美元左右；俄罗斯则希望和欧洲看齐，即380美元左右（俄罗斯向一些亲欧小国供气价格比对欧供气要高100美元，譬如向立陶宛供气价480美元）。
    中国一直不同意，结果一直拖着。从谈判进展上看，在普京访华前这一刻，俄罗斯也没有答应中国的要求，所以中国这边的消息是可能签不了。但是，这个协议突然签了，说明俄罗斯在价格上作出了让步，而中国也应该作出了一些让步。那么，这个价格可能到不了350美元，但也不会是290美元，最可能的是在330美元。当然，这个价格都是暂时的，不可能用这个价格来衡量30年的供气价格，价格是根据国际油气价格、开采成本等因素的变化而变化的。譬如，2008年俄罗斯对欧供气价格一度超过500美元每千立方米，如今价格也降到了380美元。所以，价格肯定不是绝对固定的，但中国的价格显然比欧洲的价格至少要低30—50美元每千立方米。
