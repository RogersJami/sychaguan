减肥的食物---详见常见食品性味（本栏中）

      中国素来有药食同源的说法。就是有一些中药既可以当食品来吃，同时还有一定的调理功效。一般情况下，这些药食同源的中药是没副作用的。当然也需要用的恰当，同时所用的食物或者说是中药要和自身的体质相适合。否则用的不当，或是不适合自己的体质也会有一定的不良反应。体力活动少和食物摄入热量高是引起肥胖的两大重要因素。常吃有利减肥的食物，是减肥的重要方式，记住这些减肥食物。

newyorker 发表于 2012-5-11 09:56:14

1。紫菜 紫菜除了含有丰富的维生素A、B1及B2，还可以帮助排走身体内废物及积聚的水分。

2。芝麻 芝麻中的亚麻仁油酸可以去除附在血管内的胆固醇。

newyorker 发表于 2012-5-11 09:56:41

3。香蕉 香蕉虽然卡路里很高，但脂肪却很低，而且含有丰富的钾，又饱肚又低脂，可减少脂肪在下身积聚，是减肥时候的理想食品。

newyorker 发表于 2012-5-11 09:56:53

4。苹果 苹果含独有的苹果酸，可以加速代谢，减少下身的脂肪。

newyorker 发表于 2012-5-11 09:57:03

5。红豆 红豆所含的石碱酸可以增加大肠蠕动，促进排尿及减少便秘，清除下身脂肪。

newyorker 发表于 2012-5-11 09:57:13

6。鸡蛋 蛋内的维生素B2有助去除脂肪，它蕴含的菸碱酸及维生素B1可以去除下半身的肥肉。

7。西柚 卡路里极低，多吃也不会肥，亦含丰富钾质，有助减少下半身的脂肪和水分积聚。

newyorker 发表于 2012-5-11 09:57:35

8。魔芋 所含成分以葡萄甘露聚糖为主，含有大量的食物纤维，呈碱性，吸水后体积可膨胀100倍，粘度大。可促进肠内酵素分泌，加强酶的活性，消除肠壁上的分泌物。

newyorker 发表于 2012-5-11 09:57:45

9。洋葱 有明显的降血脂和增强纤维蛋白溶解酶活性的作用。此外还有前列腺素A1，有降血压的功能，可增强体力，帮助分解体内毒素，促进血液循环等。

newyorker 发表于 2012-5-11 09:57:59

10。白萝卜 萝卜中含有大量的维生素、精氨酸、胆碱及酶类，对健康非常有益。可用于食积胀满，利尿止渴等。

11。海带 海带中含有丰富的碘，以及其他微量元素和无机盐，具有抗癌作用，还可以促进新陈代谢。海带可降脂祛肥，为美容食品之一。

newyorker 发表于 2012-5-11 09:58:24

12。芹菜 含有丰富的无机盐和微量元素，大量的粗纤维及维生素等。

13。绿豆芽  水分较多，被身体吸收产生热量较少，且不容易使脂肪堆积在皮下。减肥收腹健康绿豆芽

14。黄瓜  含有丙醇醋酸，能够抑制食物中的碳水化合物在体内转化成脂肪。

newyorker 发表于 2012-5-11 09:59:03

15。燕麦片 燕麦所具有的医疗价值和保健作用已经被古今中外医学界所公认。用燕麦面取代常规面粉，家中自制饼干、煎饼或面包等。此法有助摄入双倍膳食纤维，而且热量更低，所以更容易产生饱感，有助成功减肥

16。胡萝卜含有胡萝卜素、叶酸、钙质、纤维以及各种维生素，使人体吸收营养更全面。含有的膳食纤维能提高人体新陈代谢，达到自然减重的目的。不仅能抑制人体进食甜食和油腻食物的欲望。

17.酸白菜　　酸白菜的益生菌绝对不会比酸奶少哦，增强消化功能，加快减肥，提供免疫力好处多多。经研究表明，酸白菜中还含有丰富的维生素和钾，而且每份只有27卡的热量，放在三明治里作为配菜或是夹在烤土豆里都是很不错的。

19. 鱿鱼　　鱿鱼富含钙、磷、铁元素，利于骨骼发育和造血。很多人认为鱿鱼的胆固醇含量很高，其实它的可食部分胆固醇含量并不高。鱿鱼除富含蛋白质和人体所需的氨基酸外，鱿鱼还含有大量的牛黄酸，可抑制血液中的胆固醇含量，缓解疲劳，恢复视力，改善肝脏功能。

20. 辣椒：辣椒中含有丰富辣椒素。能促进脂肪代谢，并可溶解脂肪，抑制脂肪在休内蓄积。

newyorker 发表于 2012-5-12 03:51:26

21.番茄酱
　　与其选择甜辣酱还不如番茄酱，甜辣酱中的糖足以让你的胃涨大，但番茄酱算算甜甜，不含添加剂。最适合做意大利面的拌酱，也可以勾芡鱼、玉米饼等食物。没有脂肪又能加快热量燃烧，真的很赞。

newyorker 发表于 2012-5-12 03:51:50

22.亚麻　　把亚麻籽磨成细碎，做为食物的佐料。虽然亚麻含油量也不低，但同样纤维含量也很可观。每天咀嚼上两汤匙亚麻籽，维持健康苗条一点也不难。或者可以在下午茶的点心上撒上一些亚麻籽。

newyorker 发表于 2012-5-12 03:52:07

23.开口杏仁
　　多吃些坚果类食品做为点心，你的主食分量就会越来越小。一个杏仁+一个水果，高纤维、高维生素，对于经常吃快餐和泡面的上班族非常适合。

newyorker 发表于 2012-5-12 03:52:30

24.朝鲜蓟
　　粗纤维食物绝对是减肥者的最爱，除了竹笋以外，朝鲜蓟也是个好选择。一个水煮的朝鲜蓟只有150卡的热量，超过10克得蛋白质和大量的自然钙和叶酸；也可以配到一些家常菜里，对于祛脂消油很有帮助。

newyorker 发表于 2012-5-12 03:53:05

25． 精益黑豆
　　豆是世界上最健康的食品，它们的超能抗氧化剂、调节血糖能力，有助于避免暴饮暴食；完美蛋白质，有助于调节饮食平衡；黑豆浆、黑豆素饼、黑豆汤，都是很好的减肥食品。
