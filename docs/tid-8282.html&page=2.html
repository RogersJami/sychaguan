众所周知，广东工潮最近闹得很大面很广，如广州医学院有十二个维权工人一次性被逮捕，这是共和国历史上从未有过的事。
一切说明广东经济已到了很困窘的地步，广东经济，除产能过剩，人民币升值，物价上涨，人口红利吃光，环保要求严格，各地之间竞争，劳动力涨价等全国普遍因素外，广东还有，
一，              到了设备更换期，也被称作产业升级，广东由于改革开放早，是私企外资最早投资发展之地，自80年代起已有二三十年了，机器设备陈旧该换了，其实产业升级只是用于对银行进行贷款的漂亮词，一般的低端产业没多大升级头，如缝纫机就是转动次数略高点而已。因不明前景老板不愿或没钱投巨资更换设备，企业生存发展遇到瓶颈，故而企业艰难，就业困难。
二，              由于地皮升值巨大，有地皮的想卖地产房产走人，没地皮的付不起涨价太多的厂房租金，企业难以维持。
三，              企业利润极薄，一般小老板的收入不如体制内教授高，特别是房产涨价过高，老板们宁愿搞房地产去了，而不愿搞实业。老板也难拿出钱来给农民工加薪，广东的工人一般就就两三千一月，国企合同工，如长城钢厂月薪只有1200元，根本没法维持生活。
四，              广东由于企业大的较多，一般农民工只能接触得到工头一级，而工头除了维护老板利益不可能为工人改善什么，他们替老板分忧对工人更直接更狠，往往激化矛盾，使得工潮增多。
五，              农民工因太累，且工作多年，有点积蓄找工作就不再饥不择食了，也想休息调整一下，有适合的工作才干。同时90后由于独生子女多，宁肯坐办公室拿低薪也不愿干重活，也造成了一方面招工难，另一方面找不到工作，企业因此也影响稳定。
六，              特别是广东农民工干了近三十年，农民工最多，而且不少已举家还带亲戚老表迁来，他们要扎根广东，因此要求生活稳定，旱涝有生活能过，市民待遇如各种保险医疗，子女上学等，故要各方面维权，所以广东显得比其他省问题严重。
提了南方，顺带提提北方，北方有几千人代表近70万的银行买断职工在维权，他们包括七千万下岗职工都面临的问题，即当初他们的买断费还算钱，但由于国家持续不断的印钞，而印钞只有体制内或准体制的才能分享，广大体制外，也包括广大私企，并不能享受印钞红利，而且是他们手中的钱被摊薄，如三万元在十八年前还值钱相当于副教授一年的工资，而现在只是副教授年薪的五分之一不到了。说严重直接点，就是印钞是体制内剥削体制外的手段。因此，国家必须停止印钞刺激经济的手段，让体制外和体制内公平竞争。

国家体制内费用极高，国家所有费用都摊在了企业头上，而企业是社会最底层，必须承受体制的各种盘剥，而且准体制的改革，教育医疗产业化加剧了农民工的负担，水电气等价也一涨再涨，农民工增加点收入完全被涨价因素吞噬，特别是房价。一句话，所有的改革红利都被体制或准体制垄断拿去。因此必须大力削减体制内支出，消除教育，医疗，房产等政府垄断操盘参与控制行业的涨价因素。


现最大的困境是：国家已使用光了所有刺激政策，如温家宝的四万亿已不能再用，国家只有在萧条厉害时印点钞票，稍有复苏又停止印钞，不知这种办法会使经济缓过气来，调整向好，还是加剧困难，最后总爆发。总之阵痛期不可避免，其表现就有工潮剧增。

三中全会的改革措施对钢材市场的影响
       三中全会引起了市场的广泛关注，11月15日，三中全会公布了“中共中央关于全面深化改革若干重大问题的决定”，这份纲领性文件将引导中国社会走向巨大的变革，对钢材期货而言，我们主要关注未来的改革措施将对钢材市场产生那些影响。        公报指出，经济体制改革是全面深化改革的重点，核心问题是处理好政府和市场的关系，使市场在资源配臵中起决定性作用。
       这一表述需要引起投资者高度关注。我们认为，长期以来，由政府主导的固定资产投资高速增长，推动了中国钢铁需求的高速增长，尤其是2009年以后，城镇化快速推进，房地产和政府主导的大型基础实施投资高速增长，更使全球钢铁的生产和消费，持续向中国集中，中国粗钢产量在全球粗钢中的占比达到了50% 。
      但政府主导投资所产生的负面效应正在制约中国经济的健康发展，主要表现为：投资在国民经济中所占的比重过高，我国经济增长主要靠投资拉动。而投资的高增长是建立在货币扩张和债务扩张基础之上，2008年以后，地方政府和企业经历了新一轮债务扩张，目前很多企业资产负债超过了70%，债务扩张已难以为继，对很多地方政府来讲，如何还债，成了大问题，这必然影响投资增长，进而影响经济增长。
         未来的改革措施可能会导致以下结果：
       一是行政体制改革全面推进，政府主导投资的局面或将发生改变。政府对资源配臵的权力过大，对微观经济活动介入过多，形成经济结构的严重扭曲，压抑了市场活力。改革后，地方政府的投资冲动或将有所收敛，固定资产投资增长放缓，钢材需求增长放缓。
       二是利率市场化加速推进将，央行可能放开对银行存款利率的限制，这样市场利率将进一步走高，利率上升对高负债企业是个致命打击，一批企业必将走向破产。而根据三中全会的改革精神，政府将不再补贴和救助那些经营不善的企业，允许国有企业、甚至是大型国有企业破产，未来一批钢厂、甚至是大型钢厂走向破产重组是预料之中的事情。
       三是企业破产将导致银行坏账增大，商业银行将自动收紧信贷。在确立的金融改革措施中，一方面允许具备条件的民间资本依法发起设立中小型银行等金融机构，一方面建立存款保险制度，完善金融机构市场化退出机制，这意味着后期，商业银行也是允许破产的，这将逼迫商业银行增强信贷的风险意识。如果未来地方政府或国有企业还不起债务，亏损后果只能由商业银行自己承担，因此一旦债务违约大量产生，商业银行将自动收紧信贷，而收紧信贷将进一步抑制投资增长。
        习近平强调，改革是由问题倒逼而产生，中国经济发展中累积的种种问题，倒逼三中全会确立了全面深化改革的决定。但改革首先需要付出成本，改革的收益需要经过较长的时间才能体现。对产能过剩、污染严重、债台高筑的钢铁行业来说，三中全会的改革措施带来的首先不会是利好，改革中整个中国钢铁行业要面临巨大的阵痛，对此投资者应该做好足够的心理准备。

农民持有的土地原始股可以由后代继承，并缴纳遗产税；

这样农民就有动力去追求土地效益的保值增值，免得因为抽遗产税后留给子孙的股份太少；

这样农民和他的子孙们的权益可以得到永久的保障；

同时也彻底解决了大量农业人口涌入城市的问题

当所有土地都IPO之后，一个统一的农村土地市场就形成了，谁用土地，谁去购买该土地的股票，并且每年按照评定的土地价值支付给农民一笔使用费，这样可以防止有人低价囤积土地，放着不用。

这样也可以解决农村超生的问题，生的越多，每人继承的股份就越少
