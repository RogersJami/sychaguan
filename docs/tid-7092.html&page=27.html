并且，现如今有多少人能静下心来读这类书呢？你个人的看法不能代表其他人的。粉丝群里愿意仔细看《易经》原文而非“牵强附会”部分的，我看撑死也不会超过20个的。不看的话，虽然出于某些原因买了，在我看来，也是浪费。
