前几年去扬州老家的时候专门问过，那时候80年代初老家那边的公社基本上都用拖拉机耕地了。土地也是几十亩（ ...

拖拉机的问题其实很简单，在80年代初，农民不用拖拉机是很正确的选择。
为什么？
原因就是当时劳动力的价格极低，用人力比用拖拉机替代便宜，农民自己用牛拉犁、甚至自己拉，收入比用拖拉机好。在6、70年代，政府强推农业机械化，其中政府出一部分钱，而且还有廉价柴油等优惠，但农民还是要自己掏钱的。一般，在农机推广区，公社的公积金少一半都投到农机上了，但实际上，我国除黑吉新等少数地区以外，人均耕作面积极小，客观条件就对农机化不太好，更主要的是，由于农业劳动力实际严重过剩，所以农业的劳动力价格极低，所以，在不能吸纳转移过剩农业劳动力的情况下，任何试图替代人力的行为都会失败，因为这会把隐性失业变成显性的。同时由于农业机械化本身需要投入，在无法增加产出的情况下，人均所得会更少。这就是农民当时放弃拖拉机的原因。举个相反的例子，现在我国小麦的机收率90％多，田都是农民的小块承包田，收割机基本全是私人所有的，反而实现机械化，为何？因为机收比人工收麦便宜，更不用说不耽误打工了。

CHENGTING 发表于 2013-1-20 19:23:01

这两天茶馆登陆很慢啊，今天终于有空，先来上几个片段吧。我的家乡与一般内陆农村还是有一定区别的，也许不具有代表性，请大家一定要注意！

我生于70年代初，15岁就离家上学，很久远的事回忆起来不那么清晰，请勿过于苛求数据和根据。
步入正题：我的家乡位于苏北黄海之滨，历史非常短暂，是解放后军队新开垦的一个军垦农场，所以行政编制沿用部队编制多年。后来军队逐步撤出后，从周边农村迁来了第一批居民---我的父辈们。

拓荒者的艰辛一言难述，仅从生活和劳作两方面来说几个片段吧：
衣：几乎都是手工纺织的绵粗布，颜色洗不了几次就褪了，知青下乡前从没见过衣着光鲜的人。一年四季没有帽子、不穿袜子，鞋子全是手工做的布鞋。夏天还好，冬天就难熬了，根本就没有棉衣棉裤。

食：肯定是吃不饱的，主粮有大米、小麦、大麦、玉米、高粱、红薯等，蔬菜极少，水果没见过（第一次见知青吃苹果都不知道那是什么，偷偷捡了苹果皮尝了才知道水果好吃）。食用油、食糖供应极少，肉类要到逢年过节才能吃到。

住：就地取材，当地有小山包，但是没有火药、铁器等工具无法采石，惟一能做的就是泥土了。泥巴加水、碎草搅拌后倒入磨具，晒制成土坯就可以垒墙了，房顶是木头粱、芦苇扎的把子，最上面就是茅草了。这就是正宗的茅草屋，倒也冬暖夏凉，只是不通风、采光差。

行：没有汽车、没有拖拉机、没有自行车，惟一能搭的便车是老牛车。我小学毕业前都没有离家超过十公里。
文化：没有电视机、收音机、报刊、杂志，惟一的娱乐和消息来源就是村头的大喇叭，农闲时两三个月会有一场露天电影。
教育：没有幼儿园（知情来了后才有幼儿园，也就是仅仅看管一下孩子），每个营有一、二所小学（师资力量薄弱、校舍简陋，课桌是木板搭的，板凳自带），每个营部有一所初中学校，团部才有高中。

医疗：每个连有一到两名赤脚医生，营部有卫生室，团部有医院。医疗器械匮乏，药品稀少，大都是些中草药。

劳作：所有的生产资料都是集体所有，集体土地、集体农具、集体牛群、集体饲养场、集体菜园、集体打谷场。。。。。。个人家中几乎没有多余钱粮。生产形式也是大集体的，全团生产安排几乎一样，只是种植农作物稍有不同。营级以上干部几乎不干农活，连级以下干部也是要干农活的，尤其是排长（相当于村小组长）都是挑选的侍候庄稼的行家里手，也有地方是轮换当得排长，连长均是由营部任命的。那时候大人们都是干劲十足，没人愿意落后。有些人力气活干不动或手脚笨拙干活不快只要不是故意的，一般也没人过于计较。

农忙时，天天都是天没亮就听到出工号了，晚上要到太阳落山才收工，秋收季节有时候要忙通宵甚至连续几个通宵，经常听大人说谁谁上工时睡着了。
农闲时，男劳力要上河工，女劳力要沤农肥，政治学习是不论什么时候都少不了的，开会时女人们会利用起来赶做针线活。小孩是没人带的，好在那时每家都有5、6个小孩，都是大的带小的过来的。
小孩子上学都是自己起来做早饭，自己去上学，上学回来还要喂猪、喂鸡，做一家人的饭菜。饭菜很简单，通常是些咸菜、萝卜干之类的，能烧上个把蔬菜就很好了。曾经有人家买了榨菜、豆腐乳就被认为很奢侈了。

那时候大人的唯一希望就是能养家糊口，不要让家人饿肚子，能过上一个好年。这一切一直到70年代末才有了明显变化。

为什么70年代后期才有大的增长？

化肥产业的原料基本上是天然气和石油，我国们从60年代开始开发大庆油田，发现油田、开采、炼化，一整套产业链的建立差不多花了10年时间，所以国内的化肥形成大规模生产能力也是70年代开始。而到了70年代引入了“四三方案”，80年代初投产，又使化肥产量上了一个台阶。
