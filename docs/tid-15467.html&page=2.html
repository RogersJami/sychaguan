wild007 发表于 2016-6-3 16:43:54

如果说工业社会最重大的区别是使得物资产出速率超过了人口增长速率，那么现在对于教育来说也到了一个关键的拐点，那就是人才的产出速率超过了岗位增长速率。在这样的背景条件下，一味追求体系化的高强度高竞争性教育，已经很难持续。实际上现在传统的经院教育体制已经无法提供给社会足够的助力，因为大量毕业即失业和各种岗位关系盛行，以及企业对有经验者的偏好，都说明社会大学的成效显著超过了高校。最近盛行的高考地域不公平讨论只是传统经院教育失效的一个侧面表现。
好比工业社会对农业社会的摧枯拉朽式地摧毁，当人才体制的拐点到来时，传统教育的未来也将面临全面的颠覆，即便你现在依然能举出传统教育体制这种或那种的优点，但供需的整体大环境已经逆转。因此所谓的九品中正制的回归，本质上是人才快速贬值后的一种反转效应，真正来临的是社会对于人才价值体系的重新定位和重启。
而这其中最可能快速到来的，莫过于全社会的人才自由竞争态，即经院教育直接放开，人才认证全部网络化，实用化（类似于点赞或好评系统），即数据化认证取代证书认证。

你这个帖子回复很难，难在时代变迁中大事小事的不确定。尤其是科技革命背后的科学革命的再三十年。记得有人很认真的和我说过，这轮信息化也许是蒸汽机改变人类社会以来最深刻的一次。这也意味着，往往过去几代人遇到的经济与社会变迁，在我们余生自己我们后辈的余生将经历至少两次。如果我自己做的模型没有错，未来二十年中仅电子信息领域，以摩尔速度更新产品降遍及全产业链。而，围绕其衍生的社会经济组织及其管理方式将面临的不是变革是革命。我以自己亲身经历的一件事来做案例你来参考。2012年讨论大数据时代组织变迁中，一个朋友这样提醒过比特币是垄断资本作为新技术革命一个突破的尝试。当时有关体会和经验的历程斗写在茶馆了，现在回头看相当感慨。12年过后是13年，当时一个北京的朋友在一个中量资本集团炒比特币，那个中量资本规模五百亿，当时他对我说中国市场最顶尖的玩家都在琢磨比特币。随后，我在讨论大数据过程中结识的新朋友在2015年初提醒我大数据纳入固定资产已经放在他桌面等签字了。而到2015年底，不仅以美国九大投行为首组建区块链同盟，随后央行主导区块链开发正式宣布。我因此到北京方面的咨询是，区块链一定成事，到今天中国区块链同盟成行在即。而我自己对这里影响未来的冲击理解，说几个新技术改变未来例子

1.过去在家工作都能拿五六十万底薪的高级职业人，他们职业技术都存在被新技术替代的可能。比如，区块链之余审计师，比如人工智能之余报社编辑，比如阿发狗在金融分析领域的扩张？

2.2013年一篇来自牛津大学的报告说，围绕人工智能研究社会各种工作。544项高中低职业范围内，三分之二可以被当下人工智能在自我学习中替代。被替代职务从总经理，到餐厅与超市服务员，售票员涵盖范围几乎对社会全覆盖。

3未来基于2的变化能在将来就业保持持久优势的反而是掌握符合职业技能的岗位，比如保安岗位。

最后响应大师兄的帖子，我说一点我的理解。

1，确立终身学习的理念以应对终身变化的时代。

2，基于1，确立不断自我更新体系的方法与方法论，不能在获得成就后自满或固步自封，因为一旦对世界的认知发生固化，因时代很难避免循环下行。

3，基于1和2，确立对自己与他人诚实的学习态度。这样才能做到，区分不可以改变的事实与可以改变的事物之间的差异。这样才能做到，在时代改变来临的时候，拥有勇于行动的智慧。

摘抄一点，
例子是韩都衣舍，他现在的组织管理跟传统工业企业完全不同，它下面有286个小组，并且数量不断在增长中。它打造了以小组制为核心的单品生产体系，每一个小组3个人，有一个设计师，一个表现师，就是负责通过网络去表现产品，最后一个是负责供应量，就是通过供应链实现产品。生产计划是动态的，企业不给每个小组下达生产计划，做什么完全依靠各小组自己。这些小组自己根据市场去设计、运作、而企业提供的都是相应的操作资源，也就是赋能。
现在海尔也是这样进行经营的。
目前正在出现这么一个大型组织裂变、小型组织巨变的过程，这与整个工业时代恰恰是相反的。
