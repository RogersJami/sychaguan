       看着过瘾，我也瞎嚷嚷几句。
        顺风顺水的时候什么道理都对。逆境的才是炼真金的时候。这恐怕是党自改开30年来最大的一次考验——高考来了。过了这个坎，对经济和社会发展有了客观认识，对市场经济的活跃性和不可持续性有了深刻认识，在能力上就能望美国兄弟连的项背。过不了这个坎，苏共就是前车之鉴。当然我们走不了俄罗斯那条路。一方面我们国家人口太多人均资源相对很少，没有俄罗斯卖资源求生存的基础（卖了也不能让人都有饭吃，没吃的还得造反）。另一方面，中国人没有俄罗斯人的那种末世情节，俄罗斯人没饭吃敢把地球毁了大家一起删号重练，中国人不敢。所以西方也不会给我们留一丝生存空间。
       现在华山一条路了。看看国内，精英们还指望着中国毁了干爹能把他们接走呢。太幼稚了。有中国这把大刀，干爹才把精英当回事，没中国，美国就缺精英了？笑话。
       打个比方，现在人信什么，信钱。为什么信钱？钱能买粮食。钱能买粮食是政府们最后的信。到任何时候这一条都是底线。现在是全世界超发出来的货币没有对价的实物，美国就有那么多实物给他的货币做准备金么？肯定没有。有的话就不用担心经济危机了。世界不是没资源，而是资源不能立刻转变成供人消费的商品。或者是转变成了供人消费的商品而没有人去买（因为不缺乏）。这部分没有使用价值的商品是不能作为货币的对价的。对于四万亿，我的看法是有利有弊，在当时，经济危机刚刚抬头。人还都没有反应过来，四万亿砸下去就是在抢资源。因为我们一直都在捆绑美元。所以当初是能占多少是多少。当然这四万亿早晚有一天要坏事。这就是此政策弊端。天底下没有什么绝对好和绝对坏的事，都是因应当时情况。
       危机来了，两种方法，一种是开发新的产业链，让人都觉得缺乏这玩意，去消费，把资源变的有使用价值，等于给货币背书。另一种方法，就是打仗。消灭敌人的货币，把自己的产能强加于对方身上。对于新兴产业，无论是通讯、电子还是制造，都需要有换代的东西出现。也就是相较与以往产品有质的飞跃。那就看看新一代技术里有什么可以做到这一点，能够快速推广。因为留给我们的时间不多。
       提个东西，现在3D打印的精度虽然还不太好，但是打印食物尤其是容易和分子料理结合，混合口味的水平肯定是没问题，食材质量容易把关，容易制定标准，对原材料的要求不高（口感不依赖食材本身）。能不能作为一个项目发展一下。各位有能力的茶友不妨琢磨一下。个人觉得应该不错。

redapp 发表于 2013-7-1 00:38:23

我查了一下今年中药材的价格走势，各品种走势各异，说明中药材还是比较受实际供求关系影响的。主要是供给端，中药材产地五花八门，各地难免受灾什么的，另外之前一季的价格也会严重影响本季的播种面积，这就造成价格波动剧烈。需求端，应该比较稳定。所以可以把中药材视为类似蔬菜等的小宗农产品。
单独炒中药材价格，属于赌性比较强的选择，而且跟所有农业种植类公司一样，很难搞清其是否造假，例子可看紫鑫药业。这个是难点。
但中药的龙头公司，往往有比较强的成本转移能力（就是能涨价），所以原料价格波动对毛利影响小，比如白药。它们的问题，一是中药的毒副性，有可能造成黑天鹅，但以我观之，我朝人民目前还是对此不太敏感，有强大的文化惯性，这与食品、西药都不同，所以我以为起码在十年内，都不要太担心中药的毒副性（尤其是a股上有名的中药的龙头公司），会造成奶粉事件那样股价黑天鹅。担心中药的毒副性，自己少吃就得了。
问题二，是成长性，真倒是个真问题，尤其是龙头公司。这一是会受制于国家的医保政策（太贵国家出不起钱），二是中药公司普遍在推中药的衍生品，比如白药牙膏和广药的王老几，搞好了，当然有成长性，但跨界到快销品市场，是有市场风险的，所以还得看其市场做的如何。
总结一下，中药的龙头公司，还是有托底（传统业务）的，所以不用担心股价跌没了，二，传统业务虽然成长有限，但因为能提价，起码不会输给社会中值。拓展业务，是有一定风险，但依仗强大的文化心理，总比什么都没有好。所以中药是属于未来十几年，风险小，收益虽然不是最高，但会在中位之上的一个长期看好的板块。做个对照，现在的银行股，身上不仅背着4万亿的炸弹，而且由于利率市场化改革，整体利润率一定会下降（不利率市场化改革，实体经济也受不了），既没钱途又风险巨大，不能玩这种股。
