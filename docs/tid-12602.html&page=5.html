大师兄一针见血。
国企的技术储备人才储备是十多年前敢朝装备制造业攀登的底气所在。

现在的问题一是低端产品市场饱和而且受各种因素影响起伏波动很大，比如原油价格下降趋势一确定，所有的能源相关机械产品市场迅速萎缩。
中国战备制造业现在爬到半山腰了，要争抢中高端市场。中高端受危机影响小得多，有的这几年甚至迅速增长。但中高端的成龙配套不是前些年只要投入技改资金就能解决的，不仅仅是研发和产线改造的问题，系统化集成化也不是简单的产品升级（有些单位以为加上产品认证就行了）。
这就是第二个问题，经营管理体系的优化，如何主动调整自身适应市场变化的同时，取得资源配置优化的成果。而不是简单的问题导向产品导向头疼医头脚疼医脚。
做到这个，既要有一定的经营管理基础（人员素质、机构设置、基础管理水准、资源配置方法……）又要有相当的前瞻、洞察力和经营手腕。

虽然国企的技术力量看着让人口水三丈，但我隐约感觉这并非未来的决战决胜因素，假如市场主导权在别人手里，国内装备制造业能做的，和以前配套产品也没什么区别，啃一堆硬骨头，拿一些边边角角的订单。
掌握主导权的一是欧美大公司，或掌握核心技术主导，或掌握市场话语权，如几大油服巨头。再就是国内的巨型国企，大家伙们把持大多数市场，而且有自身的产业链条企业，吸取整个行业大多数资源，也有能力榨取其他厂商的剩余价值。这种现象如果长此以往，我觉得不是国家之福。

国企改革如果能解决这个问题，形成良性循环，才称得上中国创造，制造业强国。

见笑。以我所在层级，算是杞人忧天吧。

自主创新还需要一些制度配套，企业的科研资金作为成本在税前抵扣，打通产学研人才流动，把SCI文章等机理研究集成到项目中
