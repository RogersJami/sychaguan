奶农破产也没有法子么，破产了再给补贴呗

补贴很容易。有钱就可以了。可奶牛的产能的发展是要时间的，一个三聚把中国奶业至少送回去十年。同时，也把中国奶业彻底抺黑了。以中国奶业当时的情况来看，三聚多数发生在奶牛外包的产品中。在国内的奶业里应该还是很有一些奶品集团没有或是说涉及很少的品牌。可是，在媒体上，至少是在山某看到的媒体上，没有人出来说为这些品牌说话。就说在这里，差不多每个人都知道什么三鹿，蒙牛的。可有没有人知道那个品牌是没有三聚添加的？相比之下，美国前两年一个肉类加工的巨头被FDA查出有肉类污染问题。前前后后见报的至少有三所学校的几百名学生受了这个污染了的肉的影响。最后这个肉类加工集团召回了将近四十万磅产品。这个事件无论是影响范围还是受害人数都和这个三聚在同一个规模上。可是美国的媒体报道非常认真的写出是那一批，而且写明主要是肉馅，并且指明主要是销往那几个州的那几个超市集团。同样是报道，同样是爆料，两个事件的两种处理方式起到了完全不同的社会效果。结果当然也是完全不同的。没有美国人会因为这个事件就不再吃美国的牛肉，而且美国的牛肉甚至连出口都没受影响。同时，这种事件也决不是一个偶然事件。每个月在美国因为操作不当引起肉类污染从而造成急性食物中毒的大约都在两位数。出于对公共安全的考虑，官方从来不把这类的食物中毒单列。说穿了，就是国家不能允许自己的公民对自己国产的一种主要食品原料失去信任。再进一步，以山某来看，这些企业还真的算不上是黑心。如果换做在美国这种上下线几万人的利益捆在一起的情况下，至少会在北京，天津，石家庄这几个城市看到几千农民赶着上万的奶牛在媒体前面要说法。试试看，是那几个孩子和几个媒体有本事，还是这些生计受到危害的农民有本事。没走到这一步，中国农民失去了组织，TG领导人求稳的作风还是起了相当重要的作用的。

三聚氰胺事件的影响只能靠时间来消除，有些人确实从此对国产牛奶敬而远之，当然大多数人别无选择。后来的人会慢慢替代现在的心理敏感群体。
进口奶粉受宠，结果是价格虚高，有朋友出差去德国，奶粉价格便宜多了。美国人鲜奶多，奶粉主要婴儿的配方奶粉。
抱怨也没用，国内奶业混到这个程度完全是咎由自取。
最初大家都比较淳朴，农民也就是兑点水--后来上含氮量测定-然后用尿素增加氮含量-容易识破-终于有“天才”发明了加三聚氰胺的方法-丑闻-天量广告费（最大受益者就是广告和传媒业）-再出事-再做广告
如果是某个地下加工厂干的，那还好，可是出事的企业是花了无数广告费的大品牌；
如果是个别人的短期行为，也可以控制，问题是报道出来众多的品牌都中标--有一点一直是报道中含糊不清的，加三聚氰胺到底是在企业生产的环节还是在收奶的供应商环节，还是奶农自己--公众的印象是洪洞县里无好人；
如果加三聚氰胺的奶粉是成人奶粉而非婴幼儿奶粉，那么危害不会那么大，而对公众的心理冲击也不会那么巨大，这个跟当年的“大头娃娃”奶粉很像，属于心足够黑而脑子也足够笨的行为；
而三聚氰胺的奶粉到底销毁了多少也是个谜。

当时的报道方式可以商榷，但真正让大家彻底绝望的是后来层出不穷的食品卫生问题，反正是无路可逃。最近很火的食品卫生网站创办人说大家是“易屎而食”，食品生产商都知道自己的东西有问题，自己不吃。当年我从食品供应商那里听到过同样的话-做香肠的不吃香肠，做方便面的不吃方便面，做奶茶的不喝奶茶，做蜜饯的不吃蜜饯......实际上是大家互相投毒，很有同归于尽的感觉。
从博弈的角度说，这类似于囚徒困境，大家都盼着别人不乱来，但是因为别人违法获利，自己也参与到投毒的队伍当中--
一句话，就是没有底线。
这点在婴儿奶粉屡出问题上体现得尤为明显--害人幼子这种事都肆无忌惮。
从根本上讲，法律是道德的底线，如果道德没有了，法律也就成了摆设。不改变这种大家争创道德新低（如时果还有的话）的局面，法律其实没用，第一法律是由人执行的，执法者也可以没有底线；二是法不责众。
宣传部门的可叹之处就是天天拿道德模范忽悠人，却不能教育一下大家遵守起码的道德的好处。
如果每个人都希望靠不择手段在竞争中获胜，大家就可以体验丛林社会了。那么就变本加厉的为恶，恶到所有人都受不了要求遵守起码的规则为止(有这个勇气吗？)--唯一的挑战是活到大家不得不接受规则之时，但这对绝大多数人来说意味着毁灭。
