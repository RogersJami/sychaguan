网海讲讲阿，清水白菜有什么特别的阿

清水白菜：

    1.说是白菜，其实只取大白菜中间的那点发黄的嫩心，将熟未透时的白菜心是最好的。

　　2、所谓“开水”，是将老母鸡、老母鸭、云南宣威火腿上的蹄子、排骨、干贝等鲜货分别入沸水锅中，除清血水和杂质捞出再洗净，一起放入汤锅内，加入足量清水、姜、葱，烧开后打去浮沫，加料酒，改用小火保持微开不沸，慢慢地熬至汤出鲜味(至少4个小时)。

　　3、将净鸡脯肉打成茸，用凉的鲜汤把肉茸搅成豆浆状，倒入烧开的鲜汤中，这时鲜汤会出现奇妙的景观：汤中的杂质争先恐后地吸附在肉茸上，慢慢地形成一个球状物，10分钟左右将球状物捞起，弃而不用。如此反复2-3次，直到把汤“清”得如开水般透澈。

　　4、微黄的“开水”做成，把白菜心放入沸水中断生，再用清水漂冷，去尽菜腥后放入漏勺中，用“开水”自上淋下，直至白菜烫熟，烫过白菜的“开水”则另作他用，不能用回开水白菜之中。将烫好菜心垫在汤盆底，轻轻倒入“开水”，即成。

闲云野鹤 发表于 2012-5-28 08:49 static/image/common/back.gif
我是按装修价格推测的，如何能看出这花瓶是真品赝品呢？黄花梨的家具和上漆的仿制品有什么区别？真是外行 ...

装修价格有待考证，花瓶中清花瓷是上品，越素雅简单的越高贵，大黄大红的属下品，艳俗，看不到底色就无法鉴别矿址窑址。图中之瓶到处都能看到也就显不出其高贵之处（这种瓶江西福建广东都出的）。

黄花梨主要分海黄和越黄（前者名贵后者一般），由于料少（相对于大红酸枝），所以同样一套座椅，尺寸较老红木的小一些，一般做成明式的，用料少，图中座椅是清式的，无论尺寸颜色都不是黄花梨，黄花梨虽然也俗称红木，但不是红色的，色泽偏黄木纹含金丝。

真正的老红木家具只上蜡不上漆的，而且还要防水防阳光，否则色泽会变淡。好这口的人连蜡都不上的，因为红木是会转化的，里面的油会滋出来，越用色泽越深手感越光滑，时间久了竟不似木而是像玉般的润了，润玉润玉，你会越看越舒服越摸越光滑，暖暖的但不会有光亮的感觉。上了漆，虽然看起来很光亮，但转化就没有了，失去了灵性，就称不上是高挡货了，就如一块金砖放在你面前，它就是块金属，没有任何艺术上的感受心灵上的慰籍，而红木家具上的雕刻（我只喜欢手工苏作的），无论是蝙蝠祥云百子图龙凤呈祥松鹤图梅兰竹菊小桥流水，，，，都是那么的令人舒服爱不释手，强烈建议先去无锡苏州南通的红木家具厂参观学习一番，跟老板交上朋友后择机预订，一般一年后就能取货（其间每星期抽几天去抽查），放在家里既享受又保值，还能传给子孙，忙总提醒我们货币贬值是常态，购买老红木家具不失为对抗通涨的手段之一。
