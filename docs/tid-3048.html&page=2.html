《货币战争和人民币战略》续一

缠中说禅 2006年底


其实历史总是在重复的

就在缠中说禅2003年预测、美国2007年走出多头陷阱走势之后，2009-2012年全球资本市场就重复了一遍2007-2012年的多头陷阱走势。

2008-2009年，美国资本市场同样是从高位迅速下跌至低位，按照缠中说禅的理论，为防止2009之后，资本在市场的回升之中逃出美国，美联储再次实行了复制2003-2007走势的美元贬值政策。无独有偶，中国再次实行了人民币跟随美元贬值的吸血政策。同样重复出现的是，2009-2012年，大量热钱涌入中国。

按照缠中说禅在<货币战争和人民币战略(上)>中的理论，2009-2012年，美国如同2003-2007之间再次丧失了大量的资本，被中国吸取。因此2012-2013年，美国资本市场如同2008-2009年一般，失去了大量赖以维持的资本，再次面临着一次断崖式下跌。

而中国的资本市场状况就有些令人费解。如果按照2003-2007年走势去理解的话，中国的资本市场早就应该雄起了。当然，现在的存款准备金率比较高，周小川的“池子”里积蓄了不少2009-2012年涌入的“水”，是原因之一。;P
