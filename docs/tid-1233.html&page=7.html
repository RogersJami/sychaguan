LCC 发表于 2012-4-20 16:05:47

请芒总点评下《随园食单》里蹄髈四种做法如何：
1. 蹄膀一只，不用爪，白水煮烂，去汤，好酒一斤，清酱油杯半，陈皮一钱，红枣四五个，煨烂。起锅时，用葱、椒、酒泼入，去陈皮、红枣，此一法也
2. 先用虾米煎汤代水，加酒、秋油煨之。
3. 用蹄膀一只，先煮熟，用素油灼皱其皮，再加作料红煨。有土人好先掇食其皮，号称“揭单被”。
4. 蹄膀一个，两钵合之，加酒，加秋油，隔水蒸之，以二枝香为度，号“神仙肉”。
尤其最后一个隔水蒸，您觉得靠谱吗？

我没吃过鲍鱼，你觉得那东西是不是特别好吃？我总想不通为啥有的东西就能特别好吃

我觉得鲜鲍鱼煮着吃味道一般，我们单位原来举办过学生的烹饪大赛，有个女生弄过。煮熟了直接切片的，我觉得味道一般。不过那天查了下，看到忙叔说的那种极品鲍鱼是晒干了，经过特别炮制的。我想起来原来在厦门鼓浪屿看到当地晒很多的鲍鱼。这种原来好像在婚宴上也吃过，味道也就那回事，当然，忙叔他们吃的等级肯定是很高的等级，跟我们这类吃的不一样。

特别好吃的东西，不怕你笑话，早上自己下了个捞面条，浇头是香椿鸡蛋，我感觉味道不错。人和人的口味是不同的，我觉得像我这种偶尔吃个海鲜，吃个香椿鸡蛋啥的已经是齿颊留香了。

有的东西确实好吃，比如螃蟹。那是真鲜。据说大闸蟹是最鲜美的，我吃的少，大部分都是吃海蟹。有一次拿煮海鲜的汤，上清倒出来下面条，稍微拌点豆瓣酱，那个鲜美确实是无法形容。

鲜美的味觉一部分是由氨基酸决定的，可能还有一些芳香化合物在里面。此外一些酯类也能让人感觉到鲜美。口感这个问题就比较复杂了，应该是由食物的物理性质决定的。像海鲜类，就是氨基酸含量高，加上可能还有一些其他的成分。像香椿应该就是里面的一些芳香化合物。

对于更高档的宴席来说，有时候可能就是吃个价格和面子吧。当然有的确实能做的很好吃，但是其实仔细探讨起来，有时候用更便宜的价格得到的心理享受更大。
