蓝精灵怨怨 ( Grouchy )

本帖最后由 履虎 于 2014-6-4 13:45 编辑

女儿一个人在房间里有一会儿了，我伸头进去看看她在干什么。

她在玩游戏，iPad上面的，Smurf 蓝精灵游戏，她很喜欢，玩起来无师自通，以前就看过蓝精灵的动画片，最近又看了蓝精灵的3D电影。

女儿的外婆也走了近来，开始跟女儿聊天：

外婆：Rachel，你看完蓝精灵电影以后，哪个蓝精灵小朋友最不讨人喜欢啊。。。

女儿：笨笨？惰惰？哦，是怨怨，怨怨我不喜欢。

外婆：为什么啊？

女儿：怨怨老是埋怨个不停，自己又不做事，烦死个人。

外婆：是啊，你也不能像怨怨那样，没人喜欢的，不开心了抱怨几句可以，天天那样就没人喜欢了，对吧。

女儿哦了一声，又低头玩游戏去了。
