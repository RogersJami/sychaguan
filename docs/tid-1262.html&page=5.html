来自百度百科


众所周知，普通氢弹是在原子弹的基础上，外面包一层热核材料（氘、氚），由裂变反应放出热量导致聚变反应，进而释放出更多的能量。因为它是通过重核裂变触发轻核聚变，所以，在物理学上又将氢弹称为双相弹。三相弹是在普通氢弹外再包一层贫铀(铀238)材料。铀238这种用于坦克装甲和穿甲弹的廉价材料虽然平时很安分，但当氢弹发生核聚变时会产生大量高能中子，铀238的铀核会引起裂变，产生出能量和裂变中子，前者增强了杀伤威力，而后者反过来冲击氢弹中的锂-6材料，制造出新的氚，加剧新一轮的热核聚变，接下来的良性循环不用再多说了。可见其原理是核裂变-核聚变-核裂变三个过程，所以叫三相弹。它使普通二相弹的威力得到了成倍提高。  

三相弹效果
　　三相弹也称氢铀弹，爆炸时先由中心的铀235或钚239裂变产生超高温，在这条件下氘(dao)和氚(chuan)【均为氢的同位素】进行热核反应，如同氢弹一样释放出巨大能量，产生大量快速冲击中子，其速度超出每秒五万千米，能量很大，在这样高速的中子流的持续轰击下，外层的铀-238这种平时不易裂变的原子也发生裂变，释放巨大的能量，从而获得氢弹和原子弹的双重爆炸威力。同时，这种爆炸产生的铀-238碎片很多，于地面形成的放射性污染也很严重，从杀伤力的角度来说更加威慑。所以，目前氢铀弹的破坏力为核弹之首。 　


　打个通俗的比方，有个叫原子弹的家伙穿上件热核材料做的马甲就成了氢弹，后来又在外面套上件贫铀材料（铀235或钚239）做的防弹马甲就改叫三相弹了。 　　


前面说了，三相弹就是一种氢弹。印象中，世界上大多数氢弹，特别是用于战略核武器的大当量氢弹，都应该是三相弹。虽然没看到中国官方宣称进行过三相弹试爆，但我相信，我们现在拥有的多数氢弹应该是属于三相弹。 　　

三相弹的制作不难，中国没有它反而是不可思议的。三相弹是为增大威力而产生的，现在不大流行过大威力核弹了。而且因为要经历两次裂变，三相弹威力中差不多一半是来自裂变，所以造成的放射性沾染严重，是不环保的典型的脏弹。

很多军迷只是关注这东西数量和公开资料上有几个运载工具。其实更重要的是妥善率。
验证核武器可靠性的最好工具就是核试验。特别是进行核武库的抽样试验检查扳机的可靠性，不然即使核武器能够发射出去那也只是个脏弹而已。美国核武器的优势就在这里，不在于数量，而在于可靠性。利用之前数百次的核试验数据建立计算机模拟系统，而且这玩意本身就是娇气东西，要定期送回制造它的实验室去定检。即使这样。公开资料显示，美国也不能保证其核武库的妥善率超过八成。更何况财力物力和技术储备不如他的中俄。我觉着周边国家仅仅是过了核门槛，或者打个比方说，刚刚学会射击的新兵，和随时能够掏枪并且击中目标的老兵还差得远。我觉着我国还是古代的拉规律，来自北方的核武威胁最大。   
     以我国经济高速发展的实力，应该采用核潜艇为主的方法，这也是大家公认的。因为即使是陆基机动发射，其生存力也不如海基的十分之一。以美国为例，其大型路基洲际导弹 诸如MX这种，性能指标超过SS-18（除了威力，那是导弹大小决定的）但是综合考量生存力和成本，人家只保留路基的民兵，至于当年和SS-25对抗的侏儒导弹，也被评估后认为时陆基机动生存力仍然在第一次打击中几乎不能存活，因此仍然重点发展海基。我觉着我国可以借鉴。因为即使花费远超路基，基于生存力带来的威慑力的成倍收益，还是划算的。核武器乃国之大器，应当数量适中，质量精湛，一剑封喉，又不能劳民伤财。一点拙见，大家拍砖
