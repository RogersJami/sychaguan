白沙 发表于 2013-6-28 10:25:57

      最近大家讨论较多，迷茫也很多，产生迷茫的原因是信息不够，无法彻底领会高层的真正意思以及及时掌握高层的一些具体变化。
      下面我对对下半年的运作思路提几点看法，对或不对可以留待时间检验。
      一、确保总体增长率在7---8个点之间；
      首先看这个数据的必要性，既有就业的压力，也有政绩的需求。如果达不到这个增长，失业人数增加，社会会动乱，总经理会有压力。现在可能会有对达到这个增长数字的一些担忧或疑问，我认为完全没必要。首先智囊团肯定做过测算，上层对这个数字应该是有把握的；其次按照惯性规律来看，这个数字的实现也是完全可能的；再次是手中的筹码还是有一些的，实在不行的话，该投一点项目还是要投的。看二季度的形势，可能已经在着手做一些这方面的工作了，具体可参照近期发改委批复的一些重大投资项目的进度。
      二、不希望突破8
      高层希望经济降温，为改革留出空间，如果突破8，则回旋的空间较小。619金融动荡，现在看来更象是一次战役预演。此次预演的目的可能有二：一是测试社会的反应，是探空气球；二是实际测试抑制对象的抗击能力，为下一步采取更有针对性的措施做测试。经济降温，社会总体的日子会不好过一些，但是某些局部可能会更不好过，比如一些大规模举债的地方政府，再比如玩钱生钱游戏的一些资金。通过抓住资金源头这个牛鼻子，形成倒逼，促进经济总体形势按预期发展。
      三、最佳走势
      如果把现在的经济总盘子比喻做一个雨季的池塘的话，现在池塘里水将溢出，捕鱼的最佳手段是在池塘地势稍低的位置布好网，并掘开口子，在水自然流出的过程中，收获的各类鱼苗可投放到更大的鱼塘中去。至于不小心潜进来的大鱼，自然是要美餐一顿的。当然了，上面只是我的美好愿望。比较糟糕的场面是小鱼惊恐自杀性冲击，大鱼破网而去，那是最最糟糕的结局。
