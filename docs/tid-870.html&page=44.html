二牛 发表于 2012-3-12 22:36 static/image/common/back.gif
如果方便，请详细解释一下特供，谢谢。

有茶友有兴趣聊物流这个话题，就再涂几句：

我不是干这行的，但作为客户等与他们打过些交道，见识过一些，就自己的感受谈点：
1．        那个茶友的回帖里提到的，在中国也开展了业务的这两家，好像属于散货（中小件）类的物流公司。实际上，物流业要广大的多了。这俩是还轮不到巨无霸的位置的。
2．         我回帖里提到的修车业是“Smash Repairs” ( 残车修复)，不是那些作汽车维护保养，机械修理的Garages 。后者比较规范，前者则较混乱，容易钻孔子。一般来说，保险公司需要你提供三份以上的报价单，随后并有专业人士检核审批。但是，警方为了清理路况，需尽快地将残车拖离现场。那个第一个拿到车的人，就有机会在客户的允同下，利用自己的专业经验和获利目标，对车来“改头换面”，将车体在碰撞方向的破坏肆意地扩展。我早期就亲历过这样的，他们用大锤把客户想换“新”的地方都砸烂后，本来千余元的生意，瞬息之间会上升为近万元。。。。。。 这就是为什么警商勾结，内鬼（保险公司）外鬼结合等，屡见不鲜，屡禁不止的道理。
3．         专业物流有以下几个特点：
a.        经营方式特殊：“收储发”等都有自己的特有的方式，和特别的渠道等。
b.        人员技能和培训特殊：很多需要按照有关规范处理。（上岗证等）
c.        装备特殊：大多需要对运输车辆进行改装。
。。。。。。
你留心一下路上跑的，就会发现一些了。

不想写那些需高科技高投入的例子了，写个国外屋主们常常会打交道的：
重建，或者翻修自己的房子，是大家都会经历的。这样，建筑垃圾的处理就变成了某些人的生财之道。这样的公司会提供特制的各种容量的大存器租给你，按天付费。也会代你付一部分钱给那些拆旧房的工人，好把那些还能回用的旧式门窗，木料，砖瓦分类妥置，让他们回收。因为这里的不少人们就是偏爱这些东西，肯出大价钱到他们那里去挑货（自己心爱的旧房上的一块块瓦碎了，是不肯将就一下，用不同年代的新瓦来替换的。那个法国式的花窗，是多少人的梦求啊！）。老建筑的有些纤维板，现在属有毒物质，需专拆专收专丢，这些都是他们的额外利润。。。。有些更灵活和利害的，把下家的垃圾分类，处理和再生等都囊括了，肥水不流外人田。
乱涂一点点，供扩展思路，物流是有大文章可做的。
