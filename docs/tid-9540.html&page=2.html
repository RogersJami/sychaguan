2月7日：
1、        中俄将共同举办抗日战争胜利70周年活动——习近平抵达俄罗斯索契，应邀出席冬奥会开幕式。中俄元首索契会晤，对能源、经贸、军事、安全等领域的合作进行规划部署：普皇表示，愿共同推动油气开发、核能、航空航天、通信、人文等领域合作，愿将俄方跨欧亚铁路与“丝绸之路经济带”以及“海上丝绸之路”对接；还表态道，欧洲纳粹势力与日本军国主义的二战罪行不能被遗忘，愿同中方共同努力，搞好世界反法西斯战争暨中国人民抗日战争胜利70周年庆祝活动。——安倍桑也将出席开幕式，并与普京会谈。所以，共同举办个纪念活动，也是象征意义大于实质内涵。话说，“置于死地而后生”，万一美日俄联手反华，我们也不能任人宰割不是。做最坏的打算，谈判桌上才有底气。
2、        中俄元首共同与正在参加叙利亚化武联合护航的中俄军舰舰长视频通话——中方舰长李鹏程，百度了一下，原海军作战部长，现北海舰队副参谋长，大校，看照片比较年青。2013年底，海军第一次同步组织三大舰队赴西太平洋进行远海训练，李鹏程为蓝军指挥，红方指挥则为南海舰队司令。这次又派出去护航，看来以后要肩负重任的。
3、        习近平会见联合国秘书长潘基文，表示希望联合国办好反法西斯战争纪念活动——秘书长大人木有直接回应，不过肯定也不敢当耳旁风。另外，据潘基文所讲，中国居然是五常里面派出维和部队最多的国家，惊讶
4、        习近平会见捷克总统，表示双方要推进制造、基础设施建设、新能源、农业、医疗、投资等领域合作
5、        习近平会见希腊总统，强调要共同搞好中方参与经营希腊比雷埃夫斯港口项目——比雷埃夫斯港，地中海东部地区最大的集装箱港口之一，并与欧洲铁路系统相连接。那啥，习总跟捷克、希腊元首会晤时都笑眯眯的，对潘基文则没什么好脸色。
6、        我国将建立全国统一的城乡居民基本养老保险制度：李克强主持国务院常务会议，强调将两会建议和提案办理纳入国务院及各部门年度督查工作计划；决定建立全国统一的城乡居民基本养老保险制度，推行全国统一的社会保障卡；表示基本筹资方式为个人缴、集体助、政府补，中央财政按基础养老金标准，对中西部地区给予全额补助，对东部地区给予50%的补助。——这应该是很大一笔开支吧？不了解社保政策，希望可以落到实处。
7、        【新春走基层】 让民族信仰在家风中传承——露珠属于放养长大的，爹妈貌似也没对俺报太大希望
8、        关注乌克兰局势：美助理国务卿爆粗口侮辱欧盟——美欧在乌克兰问题上的立场有很大不同？？
9、        鲍卡斯正式出任美国新驻华大使——美国参议院财经委员会主席，搞经济的出身，应该会更务实吧？
