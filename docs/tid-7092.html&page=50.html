嗯，不如你说说牛牛同学怎么不好管了吧～我看看能否出点主意先。

我自己书稿的事，还是要解释一下的。一个是实在没足够的时间来完成修改工作，因为最终我想做出这个效果：读者看了后，不会觉得“说得有道理有水平，但结合我的实际问题的话，却不知道怎么做”，而是看了以后，会觉得“这些道理我也懂，而且我现在知道怎么去做了。”

修改工作之所以慢，是因为要达到以上的目的，需要：一方面是要全部去掉理论学术的味道，尽量通俗，每段论述都针对问题解决的思路展开，但又不是经验之谈，涵盖了群众关心的基本所有的问题，但内在逻辑又很强，还能让我的老师们这个级别的专家读出我的教育思想来（我母校的教育理论这块是国内最强的）。另一方面是要具备可操作性，提炼规律和共性的东西，给出方法论，避免给出具体教条化的方法，又能让人觉得不难操作。再有，我想这个作品在好几年内不会过时，所以，要体现长效性。另外，文字的粗糙性也要去掉。所有这些都不是一蹴而就的（我写作速度是很快的，初稿15万字我用了十天多点就完成了，但是修改工作已经超过2年半了，现在再看初稿，面目全非，修改是很值得的）。

当前国内没有人这么写这种教育类的书，写作难度是很大的。需要作者有很强的理论修养及独到的视角，足够丰富全面的涵盖教育所有领域的实践经验，以及足够好的文字表达能力，几方面缺一不可。所以，如果我能完成这个作品，那成就感是杠杠的。

一些出版社的编辑朋友一直盯着我这本书，因为跟他们稍微说了里面的一些内容及我的表达方式，他们对号入座自己的教育孩子的问题，就判断这本书会卖得很好，都翘首以盼了很久了，这几天又有人建议我先出第一版，以后不断出修订版的方式来表达我的修改意图，因为按我的想法再改下去，根本没有底。此外，我也有自知之明的，需要一个很合适我的身份来说出这书里的话，所以也一直迟迟不完稿（因为我的定位在很长时间内不是也不会是专家，并且我成功的做到了这一点，但若是出了书说了这种专家说的话，就容易让别人搞混对我的角色认识，喧宾夺主，不利于我开展工作），但这个问题在近期居然以一种意想不到的方式被解决了，我现在有了一个合适的身份角色来表达了。所以，说不定今年内就能出了这第一版呢。

但我今年还是会先出忙总的图书的。所以，请耐心等待啦。
