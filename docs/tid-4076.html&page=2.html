猛一看，还有些不信，以为忙总解读完易经就没下篇了。

打开一看，还真是忙总大作。看来2013年，我们一样会有期待的幸福。

看完忙总读《资治通鉴》第一篇，有一点特别强烈的感觉，想当年忙总刚下河时，对政治还有些生疏，虽然忙总一再谦虚对政治并不懂，但看其文字，却也是实情。
看忙总刚开始的文字，不像现在这样有思想高度和系统性，理论性。但下河三年，忙总的进步却让我感觉吃惊。刚下河时，忙总还真是一个纯粹的技术工头，埋头干实事，有些小小地愤青。现在忙总抬头望天的水平也是令人景仰了。

回想自己三年，却没一丁点长进。忙总真是树立了一个标杆，正印证了忙总一句话：我要是还只三十岁，睡觉都要笑醒了，太年轻，什么事都可以从头干起。
