我那可怜的拖延症发作得真不轻。一直在关注《忙总管理笔记》的出版进程，也很有决心的对自己说，等大懒虫妹妹把书做出来，一定第一时间购买。结果拖啊拖啊，一直拖到了今天，大懒虫都写完8月卖书小结了，我才开始行动。为了避免忙总一再批评提醒过的“君子动口不动手；或者想了等于说了，说了等于做了，做了等于做成功；或者是光说不练假把式。”，以下列出俺的行动计划。按大懒虫的计算，500铁杆忙粉，每人推荐成功购买10本，就把5000本一销而空了，所以我的目标就是完成10本有效购买。
1.        公司培训购书，为公司的管理干部提供该书做自我学习提高之用。先预计8本。这部分需要发票，收据也行。还未下单，下周一和老大谈，获得批准后，9/18下周二之前落实付款报销流程，下单落定。
2.        作为礼物送给一些我热爱尊敬喜欢的人，8本先吧，已下单。
3.        一些我生命中的有缘人和朋友，我愿意直接推荐和分享这套书给他们，我会提供这套书的信息给他们，向他们推荐这套书，把五味书社淘宝地址 告诉他们，由他们自行购买。 15人。
4.        最近同学聚会很流行，和很多同学又通过同学会建立了松散的联系，有QQ群微信群什么的，我会在这些同学会中推荐。
5.        在我的微信朋友圈中推荐
6.        在一个管理微信圈中推荐。
7.        我自己业余在做义务组织者，组织孩子们参加童子军活动。忙总的书也可以做人生观教育书，我们这个家长群里也有管理者，我会在家长群里推荐。其实父母也是一个管理岗位，每天也要做很多家庭决策，比如是去上各种补习班，还是参加和学科考试没有直接关系的童子军活动。
