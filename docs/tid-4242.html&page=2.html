还有男性线粒体的研究如何了 ...

不是那个意思。我对非洲夏娃的理解，指我们都是12（？）个非洲女性的后裔。也就是非洲夏娃不是一个，而是多个。因为线粒体DNA只能由母亲提供，父亲没有影响，所以是分析世代交替的方便工具。通过计算人的DNA自然变异速率，可以归为12组，所以我们都是12个女性的某一个生的。

男性线粒体也是母亲给的，所以无法看父亲的世系。要看那个，只能去看Y染色体，唯一可以确保是父亲给的东东。但是Y染色体理论上来讲是缺失的X染色体，突变率高，而且带动整个男性DNA的突变率都比女性高三倍（这就是为什么女性长寿，而大部分癌症都是男性得的），所以很不好研究。
