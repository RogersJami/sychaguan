至于中美合作的领域，远比新闻 ...
目前大多数的定义下，国家是最大也最具有独立利益主体的组织了（当然一些跨国公司的势力不比一些中等国家的实力弱--公司帝国。）。个组织之间既有竞争又有合作的关系，烈度看情况看需要。独立性就是，你不是他，他也不能是你，好的穿一条裤子还是两个人！斯大林和德国在二战还签了友好条约呢，都准备一起收拾英国了。苏联都准备运兵过去了，德国要求人员与装备分离（我看到的是这个解释），以德国偷袭开始，苏德战争刚开始苏联那么惨的原因之一！
多极吗，多交朋友少数敌人。家大了要照顾的多，资源也多，多元化也是必然。
其实，根本分歧到是没有，都希望祖国强大，个人及未来的子孙后代在强大的祖国庇护下好好生活。个人掌握和理解的信息不大一样，描述的侧重点不同。
我的信息就是公开的新闻报道，而且就是一个方向的信息都不能保证全面和准确。所以精细化不了，只能是一个趋势。
