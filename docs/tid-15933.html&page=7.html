中日关系在未来很难处。

中美在东亚博弈的背景下，从难往易推。


五百里甸服：百里赋纳緫，二百里纳铚，三百里纳秸服，四百里粟，五百里米。  
五百里侯服：百里采，二百里男邦，三百里诸侯。  
五百里绥服：三百里揆文教，二百里奋武卫。  
五百里要服：三百里夷，二百里蔡。  
五百里荒服：三百里蛮，二百里流。  
《禹贡》

这个讲地理远近的不同制度安排。
大同是理想没实现前，远近亲疏才是现实的选择。

攻、守是两种不同的状态，其实讲的是一回事--战略态势。 就一国来说，你能攻多远，是战力的极限问题，背后是国力的支持问题。可国力不是一层不变的。就西域（新疆）来说就是几得几失，汉唐得，到元再续，中间相隔千年。守就是被进攻时还能稳住的地盘。

以上是军力，还有认同的问题。这个是核心问题，不仅仅是地缘问题。攻下来的地盘要能形成认同，融合以后才能长期稳定的管理。中国的文化就像摊大饼这个比较形象，有动荡的时候不会轻易分裂出去。

全球博弈是多方参与的过程，不仅仅是一家在想。技术的进步会改很多东西--管理的工具和方法、深度和广度，但是文化是个慢变量，而且也是意识构成的核心。这个在很大程度上影响人的行为。看看欧盟现在的情况就知道了。

根本看，中国需要什么，不同的影响因素有不同的半径，几项叠加就会形成几个同心圆，最小的那个就是我们一定要守住的，开拓出来能守住的还是需要文化的认同和融合--可拓展的（包含地缘、资源考量等）。再向外，其实就是利益圈了，轻重也不同，各种方法能稳定的拿回利益就行，其实不用背上国民管理的社会成本。

好处着眼，坏处着手。
