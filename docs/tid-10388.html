【整理】青岛多国海上联合演习之舰艇

每天忙于俗事，今天经过馆主提醒才发现在青岛开始了多国的海上联合演习。
这次演习是人民海军成立65周年纪念活动的一个重要部分，旨在进一步加强各国海军之间的交流和务实合作，提高各国海军遂行海上联合搜救任务的能力，及时应对、处置海上突发问题。这个主题在今年各种坠机、沉船的情况下比较应景。
参加的国家除了中国，还包括文莱、印尼、马来西亚、新加坡、印度、孟加拉和巴基斯坦，基本上是东南亚或者南亚国家。
下面一点一点介绍下参加演习的舰艇。

先来介绍巴基斯坦参加演习的舰艇。
巴基斯坦参加演习的是F-22P  Zulfiquar-Class 佐勒菲卡尔级护卫舰“沙姆谢尔”号护卫舰。
沙姆谢尔号于2007年7月在上海沪东造船厂铺设龙骨，2008年10月下水，2009年12月正式服役巴基斯坦海军。
F-22P是我国为巴基斯坦建造的多用途护卫舰，总共出售了4艘护卫舰和配套的Z-9EC反潜直升机，合同金额大约7.5亿美元。
F-22P来源自我国海军自用的江卫级Ⅱ型（053H3）护卫舰，但比053H3型排水量更大，舰艇设计更加适合远洋航行。

介绍完了巴基斯坦的舰船，然后再介绍下巴基斯坦的近邻印度这次前来参加演习的舰船。
这次印度派出的是Shivalik-class  什瓦里克级护卫舰的首舰 什瓦里克号。
什瓦里克号于2001年7月在印度的Mazagon船厂铺设龙骨，2003年4月下水，2010年4月正式服役。
同F-22P参考了053H3类似，什瓦里克级是基于俄罗斯塔尔瓦级护卫舰进行了改良和设计。
F-22P排水量大约3000吨，而什瓦里克的满水排水量达到6200吨。因此，虽然一样称为护卫舰，但是什瓦里克的作战定位与
F-22P完全不一样。

下面这张什瓦里克级的图附带了武器系统和电子系统的介绍。
从这张图来看什瓦里克的外形进行了隐身设计。
http://photos.tuchong.com/413064/l/6892864.jpg

下面这张图是俯视图。从图中可以看到，什瓦里克直升机的机库相对较大，它可以搭载2架卡-31或者2架MK-42B海王直升机。
这会增强什瓦里克的反潜能力。
http://photos.tuchong.com/413064/l/6892870.jpg
在烟囱后面装有4组8单元的以色列制的Barak 1 近防导弹垂直发射系统，射程12公里。
没有找到什瓦里克的Barak 1的图，下面这张是印度另外一款舰艇装备Barak1 的图。
现在基本上Barak1 已经成为了印度海军的标准近防系统，同时它还结合了AK630防空机炮，成为所谓弹炮合一的系统。
http://photos.tuchong.com/413064/l/6894227.jpg
下面这张图什瓦里克的主要的中程区域防空武器，一座3S90单臂防空导弹发射器，发射俄制SA-N-7/12防空导弹，射程32公里，射高15000米。但可惜的是印度的这款导弹并没有采用垂直发射，而是仍然采用了80年代普遍采用的单臂发射器。
虽然据称发射间隔已经缩小到10秒内，同时重装时间也很短，但和垂发相比还是有很大的距离。
值得注意的是我国的红旗16导弹也是从俄制SA-N-7防空导弹仿制发展而来，同时我国已经实现红旗16的垂直发射系统。
http://photos.tuchong.com/413064/l/6894319.jpg

下图舰炮后面的垂发系统就是我国的红旗16的垂直发射系统。
http://photos.tuchong.com/413064/l/6894400.jpg

下图是什瓦里克的主要反舰武器，俄罗斯和印度联合研制的布拉莫斯（Brahmos）超音速反舰导弹，射程300~500公里。
比较有意思的事，什瓦里克采用垂直发射系统发射此类导弹。一座8联装垂直发射器安装在单臂防空导弹发射器后面。
http://photos.tuchong.com/413064/l/6894435.jpg

同时什瓦里克还装有意大利奥托梅莱拉76mm舰炮的超级快速型（Super Rapid Gun Mount，SRGM），射速高达120发/分，是此炮标准型的1.5倍，并使用隐身炮塔壳。同时舰桥前方装有一具12联装RBU-6000反潜火箭发射器 ，此外还有两组隐藏于舰体内的双联装533mm DTA-53-956固定式鱼雷发射器（也可能改用印度自制的产品）。

什瓦里克可以说拥有相当强大的火力，也是印度自制军舰上的一个标志性舰艇。

下面来介绍下孟加拉国参加演习的舰艇。
孟加拉国参加演习的舰艇是 abu bakar 阿布·巴卡尔号护卫舰。阿布·巴卡尔号护卫舰原为我国053H2型导弹护卫舰黄石号。
黄石号2013年4月在北海舰队退役，2014年1月被转让给了孟加拉国海军，同时被转让给孟加拉海军的还有053H2型导弹护卫舰芜湖号和053H1型导弹护卫舰湘潭号。这次参加演习算是回娘家了。
黄石号，江湖III型(053H2)导弹护卫舰首舰，1984年10月在上海沪东造船厂开工建造，1985年12月下水，1986年12月正式服役东海舰队，后于2000年调整去到北海舰队。

阿布·巴卡尔号护卫舰满载排水量1960吨，是目前介绍的护卫舰例吨位数最小的。

下面介绍下文莱海军的参演舰艇。
文莱海军派出了“达鲁伊山”号巡逻舰参加此次演习。

下面介绍新加坡的参演舰艇，可畏号护卫舰。
可畏号护卫舰是新加坡可畏级护卫舰的首舰。它于2004年1月下水，2007年5月服役。
可畏级护卫舰是法国基于拉斐特级护卫舰为基础进行设计，由法国和新加坡共同生产的新一代隐形护卫舰。
总共6艘舰艇，除了第一艘可畏号是在法国生产的，其他都是在新加坡生产的。
有意思的是，法国除了向新加坡出售了这型护卫舰以外，还曾经向台湾和沙特阿拉伯出售了此型护卫舰。
台湾在1990年初向法国购买了6艘拉斐特级护卫舰，命名为康定级。
这几型拉斐特级护卫舰满载排水量大约在3600吨左右。
为了给大家一点直观印象，我把新加坡、台湾和法国的拉斐特级都贴了出来。

下面来介绍印度尼西亚的参演舰艇，班加马辛号船坞登陆舰。
班加马辛号是印尼Makassar class 船坞登陆舰的第三艘。Makassar class 是韩国大宇公司为印尼设计建造的登陆舰。
Makassar class 前两艘在韩国生产，后面两艘在印尼生产。目前秘鲁和菲律宾海军也分别订购了2艘该型登陆舰。
班加马辛号满载排水量11000吨，最高航速16节，可搭载40辆步兵战车、218名海军陆战队员。
班加马辛号可起降5架直升机，机库足够容纳2架欧洲直升机公司的超级美洲狮中型直升机（这个类型的直升机有很多型号，在世界各地都有服役，包括大陆、香港和台湾~~）。
