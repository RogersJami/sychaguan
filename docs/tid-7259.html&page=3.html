浮云 发表于 2013-9-2 22:49 static/image/common/back.gif
那些令人不舒服的信息谁遇上都不会开心，不过懒虫诸如便宜东西、奢侈品云云情绪，没必要。买奢侈品的人内 ...
谢谢理解。其实我自己的情绪还好的，可能我的文字中给群众的感觉很郁闷，其实没有的，我平静得很。说歧视什么的，的确带着些情绪的，说话也不严整了。其实我想表达的是内在贫穷的人，能把好东西都拉低档次成垃圾。（其实我在内心是把所做的图书，看成是奢侈品的。）内心富裕的人，穿个假的阿玛尼，别人都以为是真的，因为人有那个自信，内心贫穷的人，戴个真的劳力士，别人都不相信，因为没那个气场。

对各种不友善，经过这段时间的锻炼，我已经非常适应了。之前之所以不舒服，是因为自以为是地把喜欢忙总的人群的总体修养人为地拔高了。其实，人群中修养好的人，总是极少数比例的，不管是什么人群。现在，当看到有人很“老道”地评论我做的这套书“应该不是正版”时，我很平静地回复他：“请找出正版来，然后欢迎您到国家新闻出版总署去举报我们。”（这个总署现在改名了，但对外还是习惯这么说）。我才不会跟人说“如假包换，假一罚百，假一罚万”神马的话，是真的，它跟假的就不必要相提并论。
