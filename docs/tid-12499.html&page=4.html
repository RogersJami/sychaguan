我一个同事这个车，一年不到，换了四个轮胎！呵呵，也不知道怎么回事情！昨天刚好座她车，一个轮胎又报警 ...

哇，这么费轮胎，该修了，可能的原因：

1，左右两侧胎压不一致，自己检查一下即可；

2，自己测试一下是否跑偏：找条没人的直路面，起步后双手松开方向盘，低速直行100米，起点与终点最大偏差超过5米就是跑偏了，需要在4S店重新做定位。

3，做了定位后重新按照2的方法测试一下，还有跑偏，可能减震器或者弹簧受损，或者是悬挂有位移、变形，需要送4S店彻底检修。

30万的3个选择：奥迪A4，低配；帕萨特，高配；奥迪Q3，中配。推荐奥迪Q3。

奥迪A4和帕萨特的比较：

1，动力总成（发动机+变速箱）完全相同
2，车身加工技术相同，只是外观不一样，A4中性，家用商用两宜，帕萨特保守，商用。
3，A4低配和帕萨特高配的差别在底盘，帕萨特高配的技术含量高，更舒适

Q3和A4、帕萨特的比较：工艺水平一致，但Q3进口件多，可靠性更好；外观活泼，家用最宜。

买奥迪要考虑保养成本，我这次换车时候看过奥迪Q3，,车不算贵，但是保养太贵。反正我还是工薪，养不起。
具体对比数据选车我已经没有这个积极性了。第二次买车，首先挑自己喜欢的，再结合需求选车型。第三，我在意的是保养贵不贵，可靠性高不高。比如说20多万的车，性能足够我用了。
我选的是歌诗图，首先就是喜欢这个外形，喜欢就是感性了，没什么好说的。其次就是日本车在可靠性不错和用车成本上是不高，这个不用网上对比，问问同事就知道了，日本车在整车质量这一块口碑确实做得好。第二次买车，已经没有了大的激情，无非就是把车当成一个工具，这一次把工具改善一下，提高一下品质。我会把可靠性和养车成本放在很重要的位置。其实小毛病多的车原因有两个，一个是技术不成熟，一个是装配质量，后者涉及到品质管理问题，有人可以接受这些，但我可能从事机械维修的原因，特别讨厌可靠性不高的东西
不要以为保养是小钱，以奥迪为例，小保养一次1300看似不贵，但其实到一定程度上随着你车毛病的增加，实际每次不止这个钱。有人说可以不去4S，我所在的深圳我没发现施工质量和SOP这一块能超过4S的路边店。
至于安全性我是这么看的，不伤筋动骨的擦碰，日本车吃亏，这跟设计理念有关，即便是凌志，油漆也不厚，但是涉及到人身安全，也就是拼车架的事故，没有明显数据说有人身伤亡的交通事故里，日本车死人多。用我一个干交警的同学话说，过了120KM，什么车都是馅饼。
总之大家家用车的话，喜欢和需求之外，少折腾是不得不考虑的因素，我第一辆车就是福特，大毛病没有，小毛病很烦。这次就是想买个不折腾的。所以选日本牌子，个人认为德国汽车要体验的精髓，至少要过40W。
