发现看不懂的地方真多。勉强理解一下：

1. 土地流转改革，改变土地供给的基本情况，使得房地产市场不再因物以稀为贵，以市场资源调配的方式而不是行政限制的手段来调整房地产市场。地方政府靠卖地收钱的财政模式应该也会失去基础，那么房地产税是否也该同时推出，至少要给地方政府续上口活气？

2. 收拾谣言传播，说明利用舆论起到监督、制衡、清理官员体系的作用已经开始起效，现在开始控制舆论开放的负面，逐步减少因网络和移动平台假消息泛滥造成的社会负面影响。结合最近央视的正面宣传行动，可以想象对社会公义的宣传和倡导将大大加强。如果要类比，也许和80年代初的大杀一批稳定社会再跟上五讲四美三热爱、四有新人宣传是一个路子。其实我很怀念那时候的风气。民心可用，首先要建立在还有民心的基础上。反贪和反谣，这是一件事物的两个方面。

3. 对内，光靠反贪反谣是不够的，搬掉新的三座大山是重建社会良好经济秩序不可缺少的基础。房地产前面说了，我很感兴趣的是本次针对跨国药企的动作，是否是医疗体系改革的一步？习总说再难也要改，是否是真的要动刀的时候？虽然药企一被查，我们这些服务商不管有罪还是无辜，基本都是灭顶之灾，但是如果这个能带来医疗体系的重建，我个人认为还是值得的。就怕虎头蛇尾，最后我们只是神仙打架殃及的无辜，死得毫无价值。

4. 对外，感觉中美关系和第二次国共合作真有类似之处，互相依存，斗而不破，不过心眼深处都希望一招制敌。在这个过程中，如何实事求是、随机应变，利用一切对TG有利的机会壮大自己，是非常有挑战的。也许在这场战争中的每个人都应该看看《毛泽东传》，主席在实事求是、具体问题具体分析、时刻分清主次矛盾上真称得上是英明神武。这回的对手不再是混乱低能的GMD，不过，精英社会决定了对手中聪明的人不会太多，农村包围城市、从下到上的战略很可能再次奏效。但是，我们也需要一支健康、没有太多短板的队伍。

总之，既然帝国之路不可避免，我们只能争取尽量战斗得漂亮、灵巧，尽量小的损失，而不要打成消耗战。在这之前，强健自身，活血逐瘀，打通内部的任督二脉，获得强有力的内力源泉，而不是仅仅靠一身野蛮生长的横练肌肉去打天下，这点至关重要。这一点，应该是这届班子最重要的任务。
