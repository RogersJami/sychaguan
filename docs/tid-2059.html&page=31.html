忙总，能讲下你对李约瑟难题的看法吗？我最近对这个感兴趣，但感觉很难找到有效的资料。目前粗浅的认识是 ...

资本主义本质上说就是释放私欲，长期来看是一条自取灭亡之路。中国已经被用这种模式思考问题的秦朝统治者折腾过一次了，吸取了教训，所以汉代到清代都在抑制私欲的无限膨胀。但由于土地被以商鞅为始的那一拨人私有化了，所以中国帝制时期一直缺乏按照毛时代路数工业化的物质基础。所以帝制时代的中国是经济上的资本主义，文化上的社会主义，处于一种不上不下非常难受的阶段。
我觉得咱们要问自己的不应该是中国为什么没产生资本主义工业化之类的问题，而应该是中国总体上应采用什么样的模式生存发展延续，细节上采用什么方法解决问题。跟着李约瑟的问题走，潜意识就在跟着西方的模式走了，吃错了药，就会像国民党一样，进退失据，消化不良。
