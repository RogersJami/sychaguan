葡萄 发表于 2017-3-18 15:14 static/image/common/back.gif
日耳曼中心史观与罗马中心史观是欧洲两大中心史观，我对条顿堡森林战役及其衍生叙述偏日耳曼中心史观。
对于罗马的崩溃，你偏重技术原因，但还有政治制度和经济原因。

罗马是军事帝国制度，马上得天下也马上治天下，征服一地，派军团驻守，军团必须倚仗当地利益集团，久而久之，相互融合，成为藩镇，当藩镇强大到一定程度，回师挑战中央，屡次出现军人干政。

罗马是地区性的经济体系，经济基础是城市，所谓条条大路通罗马，交通干道上的大城市、港口是经济枢纽，一个大城市背后就有一张地区性的经济网络，地区与地区之间可以往来，也可以不往来，一旦破裂，重建的意愿并不迫切。

所以罗马崩溃之后，欧洲进入1000年混战，而中国不同。

从政治制度上说，中国的主流是马上得天下，郡县治天下，地区的实力有限，不足以挑战中央。

从经济上说，中国是网络经济，基础在农村，交通干道上是城市，支路上是县镇，末梢是农村，物资在整个网络中聚、散，即使破裂，重建需求强烈。这种重建，就是一次经济大动员，恢复得很快，而且经济上的整合往往先于政治上的统一，汉匈对立还有马市。

历史上中国的强大，一是政治稳定，二是经济动员，短短30多年能走到今天，就是在政治稳定下的经济动员。

我们的路径是清晰的，一带一路和不称霸。不称霸就是边界。

有必要补充一下。

在当下，政治稳定就是收买利益集团，从15年以来的金融版图变化可以窥见一斑，特别是魔都的变化，非常剧烈。

收买不好听，可不贬义，民众也在收买之列，除了民生措施，最大的收买就是经济动员，一带一路总动员。
