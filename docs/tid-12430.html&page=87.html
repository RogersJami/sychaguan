如果说周四之前还有很多人众志成城，周五这个大跌真的是让所有人都明白这不是调整而是股灾了，包括我，跌穿了我的心理底线3800点。我的股票满仓，就是中航飞机和招商轮船，各一半。之前这2个股都比较抗跌，我都不在意，但周四大跌，周五直接跌停价开盘，我就知道形势不对了。一般来说如果是调整行情的话，最抗跌的股票在调整完毕之后会最强势。但如果是股灾，那后面这2支前期抗跌的股票就要补跌。现在飞机离高点才跌了四分之一，按照再跌四分之一计算，就要跌到24块附近。当然这要看各大基金的清盘情况。之前这2个股票抗跌是因为是大盘股，主要由大基金持仓。在前3周的大跌中，起因是清理配资盘，而配资盘炒的一般是小盘股和创业板，现在配资盘清理的差不多了，所以其实创业板已经跌的差不多到位了。但从周四开始这一波大跌开始渐渐到达一些1：1的融资盘的平仓线，而融资盘很多买的是大盘股。所以融资客开始抛大盘股。而大盘股下跌就影响到大基金。周四开始大基金为了自保开始争先恐后的清仓，抛售大盘股。基金开始互相逃命互相踩踏。我猜想周四晚上一定是有不少基金做出了逃命的决策，所以周五我的飞机轮船都是跌停价开盘，很明显大基金在抛货逃生了。周五消息一个国内前三的私募宣布清盘，应该是周五这一天跑出去了。如果周末这2天政府没有明确的强有力的政策，周一周二2天会有基金大规模踩踏状况发生，飞机连续2个跌停的概率很大。极端状况些做好3个跌停的准备，也就是回到24块附近，刚好是高点的腰斩价。
       从周五的盘面看，已经有抄底资金开始进场，我知道有很多是5100附近逃顶的，他们开始回来了。但还不够，下周应该会有很多，资金总量很大不用愁，只要跌势企稳，抄底资金立即大规模出现。大盘这次的底在哪，也许3500，也许还要低些，但我估计下周出现未来3个月的低点应该是大概率的。我手上的股票都不打算动了，周末准备再筹资一些，下周进场继续买飞机和轮船。
