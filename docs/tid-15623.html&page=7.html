买方

契税：买方非唯一住房全额3%；唯一全额1%。

卖方

土地增值税：增值部分累进税率计算或者全额1%。
个人所得税：利润部分20%或者全额1%。
营业税：未满两年全额5.5%，满两年免征。营改增后换名增值税，税率总额5%。
城建税、教育附加税和地方附加税占比过小，略。

以上为普通住房，上海的征收标准。

投资买房税费成本=契税（买入）+（土地增值税+个人所得税+营业税）（卖出），

分两种情况讨论：1）假设有增值，并且税费没有减免，并且短期内再次出售，则

契税=X*0.03，

土地增值税=（X+t）*0.01或者a，

个人所得税=（X+t）*0.01或者b，

营业税=（X+t）*0.05，

由上，投资买房税费成本=X*0.03+（X+t）*0.01或者a+（X+t）*0.01或者b+（X+t）*0.05，
表达式1=X*0.03+（X+t）*0.07=X*0.1+0.07*t，短期内X远大于t，t可以忽略不计，设X为1，

表达式1=10%，
表达式2=0.08+a+b。

2）假设有增值，并且税费可以减免（如持有两年以上），并且两年内不做出售处理，则

契税=0.03，

土地增值税=a，

个人所得税=b，

营业税=0，

投资买房税费成本=0.03+a+b。

结论：1）的10%低于你说的12%，必要税费是8%，a、b只对增值部分收税，不增值不收税；2）的3%是必要税费，a、b同1。不考虑通胀，短期投机涨幅超过8%即账面盈利，两年后超过3%亦同。
