一个可能是 对付流亡的反抗的江南楚人。结果是中原王朝获得广州的小小殖民地。

    汉初，南越国其实是汉人上层，老百姓是百越就壮族先民，然后汉武帝灭南越，对于南越加强管理，随后几百年广州站稳，以珠江三角洲为基地，不断挤压百越（壮族）生活空间，壮族逐渐跑到广西云南贵州大山里面。

    对于这些深山里面，在古代冷兵器时代，征服是痴心妄想，政府军过来压服，政府军走就没有管理，有也就是宋朝之后北方汉人大规模南逃，江南那片好地方在东晋后面南北朝已经被人占了，只能继续向南向南，这样的人口压力才决定性的把百越的后人（壮族）不断向穷山恶水这些汉人呢，来自北方的就是客家人，南方的珠江三角洲基地的就是广府人。



    在这些大山之中，当然是维持土司制度，所以只要政府要加强管理，自然就是反。广西土司在明代还是很听中央政府话的。很多土司追随中央打仗。



      这种壮族的状态，和苗族、傣族等等西南少数民族是一个经历，都是生存斗争中被赶进山里的。



     所以是有造反传统的民族。不过现在60年已经好多了，基本上国境线内少数民族已经没有区别了，不存在造反可能

frog198 发表于 2012-6-6 00:20:06

而历史上，南诏国其实是东南亚各民族的母体。当南诏国被元灭亡了以后，人口往南移，形成了泰民族。往东和北移，形成了现在的布依族和壮族。所以你看泰国的历史，就发现这个民族是12世纪突然从泰国北部冒出来的。
泰民族与我国的布依族和壮族，渊源体现在语言上，都属于属汉藏语系壮侗语族壮傣语支。以前说布衣语于泰语很相似，我还不信。后来到了泰国旅游，才发现很多单词都是一样的。
比如管水和汤叫“囊”，把小孩叫“侬”，狗叫“马”，鸡叫“给”，猪叫“母”等等等等。
有意思的是，布衣语和壮语的有些词，在现在的泰语中不说，即使是学泰语的也不知道。但泰国人还是知道得，但不怎么说，或者是不正式的场合说。
如布衣语把你叫“蒙”，我叫“顾”。而现代泰语没有这个说法，我和当地留学生交流，他们都不知道。但一位布依族的留学生说，她和当地人交流，发现他们其实也有这个称呼。但这个称呼是古泰语，现在不怎么说了，只是用于非常亲密的朋友中。我在看泰国电影的时候，发现好像还是有把我叫“顾”的说法。
从语言上，就可以看出泰民族于中国的布依族以及壮族的渊源了
