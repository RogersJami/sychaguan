小黑贝在4个月大的时候最可爱，一个耳朵站立，一个耳朵耷拉。
忙总，这是特有的么？

以前家里的狗狗，刚买回来时耳朵是站立的，长大一点两个都是半耷拉——人家都怀疑，这怎么是警犬后代啊，还要花200块卖（95年），耳朵居然耷拉？就在大家狐疑的氛围中，人家的一个耳朵就慢慢站起来了；再过一阵，两个都站立起来。

爱家模范的表现

有时候不小心挣脱出去，基本上口不走空——总要叼个鸡鸭什么的回来。尤其农村的小鸡仔们，散养惯了，基本都会普通几下翅膀做些飞行动作，很难就擒。不得不佩服。当然，也有反面教材：自己家养的鸡鸭也容易惨遭毒口。但理论上，没教过她什么抓捕技术的。


胆子大的表现多了

那年月家里还偶尔杀猪。结果，每次往猪圈里赶猪都很头痛，不管是一头还是一群，更无论大小。通常是怒吠着冲上去就死死咬住——猪脸、猪耳朵、猪鼻子，逮哪咬哪。而且，屡教不改。后来，家里养的马，也被欺负；养的母牛，更是悲催，尤其是牛怀孕的时候搞得大家都紧张。把牛马签回圈里，必然得两个人分工——一个负责牵牲口，一个负责把狗狗拽到小黑屋。

记忆力好就没的说

后来去上大学，经常一两年假期都出去疯而不回家。回到家，自己都觉得很陌生的时候，狗狗从来都表示热情欢迎。摇尾晃脑，搞得其他人都很嫉妒。


不过，说起来最佩服的还是她的食性：杂，不挑食

这条狗狗，基本历经家里的几次变故，伙食不固定，质量也不一定保证。但，她从来没什么怨言，佩服啊。要是小猫，因为伙食问题，恐怕多数会叛逃而离家出走——有一两只，就是如此，于是受伤而极其不喜欢猫。

狗狗小时候，经常直接喂生肉。尤其赶集卖剩下的猪肉、骨头，在储存前总要捡些扔给她——也算蜜罐里边长大吧。后来，基本上人吃什么，她吃什么——剩下的饭菜泡馒头，也算可以。碰上过年了，所有的猪血基本都是她的——新鲜的，加工熟的——来年，加工熟的猪血化冻，要多难看有多难看，她照吃不误。家里的骨头，也不需要特别处理——多了的时候，她自己会藏起来。

后来，家里境况有段时间不太好，狗狗也能同甘共苦。那段时间，她的口粮就是牛口粮——不同的是，煮好的玉米糊糊、麸子之类的，给她吃之前特意放一勺子猪油。煮红薯，可就没有油水了。但她都吃得津津有味，总会莫名感动。这段时间大概两三年吧。再后来，她的伙食水平基本恢复了。


杂，除了吃东西多外，还表现出吃的技巧。嗑花生还不牛的话，人家还会磕瓜子。啃西瓜，能啃到只剩下最硬的那层外皮。也喜欢大白菜，偶尔还对牛马的草感兴趣。啃玉米棒，只能算及格。抓鱼吃鱼，尤其是买回来的海鱼处理剩下的杂碎——头尾、鱼骨等等，也因此还会被卡到过，也因此见识了其呕吐功底。


狗狗还有项特别的能力，呕吐

不知道其他狗狗会不会，但我家这条狗狗吃到自觉不好的东西，就会经常呕吐。观察多了，我也摸索到了规律，所以现在在外边应酬喝酒就算多了个随身本领——真要豁出去了，咱还可以不借助任何其他道具就可以去吐酒，回来再对付那些不顺眼的。



另外，看到忙总贴的第二张图片，貌似还真有点像我家那条狗狗。
