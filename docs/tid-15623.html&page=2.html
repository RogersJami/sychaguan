诸侯经济只是产能过剩的原因之一。

产能过剩，最根本的原因是同质化竞争。在国内大干快上的局面下，抄袭别人赚钱的项目，是最简单最快的。同质化竞争，怎么办？降价。为了应付降价，怎么降成本最快？ 扩产。你降我也降，你扩我也扩，死循环。
（不少行业都是中国前三名产能加起来，基本到全世界需求的80%-90%了）

解决的办法：
1.限制同业跳槽。对劳动合同里有规定若干年内不得在同业执业的，要严格处罚。
2.严格对技术抄袭的处罚。
3. 地方政府不得对本地区亏损的过剩产能与以补贴。
4. 亏损企业让它去亏损。 没有企业破产，产能怎么减的下来？
