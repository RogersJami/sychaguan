tiantian 发表于 2012-3-13 15:40 static/image/common/back.gif
忙总，你对转基因如何看？



试着答一下，

首先，现在成熟的转基因作物可以减轻田间管理的劳动力支出。比如抗虫Bt蛋白，抗草甘膦的转基因大豆，这样可以减轻很多田间劳务。省去反复除草和多次除虫打药。中国当前劳动力人口逐渐枯竭，这些成熟的转基因技术是可以缓解农业对劳动力人口的需求。

其次，现在农业土壤退化严重。比如，木瓜，甘蔗，蔬菜等种植地长期种植单一物种，土壤里面专性寄生的土壤线虫很多，导致蔬菜烂根，减产。转基因种是可以快速提高植物抗性，可以应对地力退化这一问题。

第三，转基因暂时做不到增产的可能，但是也是有可能实现增产的。比如袁隆平设想的把碳4基因转入C3的水稻中，这一可以提高水稻的光合效率。这也是很好的方向。

第四，中国育种行业问题很大，转基因是解决育种业的一个技术，但并不能全面解决中国育种业的问题。这涉及到管理，市场，机构改革等等。

第五，有机农业和生态农业，比如用有机肥，不打农药。是可行的，但是生态农业对劳力和土地需求都大，而且没有数据表明有机农业在面上推开，还能保持高产。比如，如何在大区域上供用有机肥？

在里面的几个人中，李家洋人品没的说，中国生物学在90年代极度落后于世界领先水平，现在植物学已经追平美国。这里面有李家洋的功劳。李家洋即是一个优秀的科学家，又是一个优秀的领导者和管理者。
