       包括本楼楼主在内，一直以来大家有一个错误的观点，以为中美之间存在第二种选项，我的观点可能令很多人失望甚至愤慨，不要产生这种奢望，一旦有这种错误的指导思想，迟早要栽大跟头的。我的观点，今后的世界局势不会有G2，只能是多极平衡。
       为什么不能有G2？要回答这个问题首先要回答美国的霸权是从哪里来的。美国霸权的支撑体系是美元和美军，但是归根到底是美元，因为美军的强大是建立在美元强大的基础之上的，没有强势美元支撑的美军实力会快速消退。美元的强大是因为欧洲一众二战伤残国的抬举，抬举美元说彻底一点就是集中西方政经资源对抗华约集团的冷战需要。现在美元走弱了，难道仅仅是因为人民币的冲击？显然不是。仅仅凭人民币是冲击不了美元的，冷战后共同敌人的消失，欧洲伙伴不愿再给美国佬抬轿子才是美元走弱的根本性原因。欧洲人日本人都不肯抬轿子了，美元怎么维持强势？G2体系能否取代以往的美欧日体系？中国在G2体系内处于一个什么样的地位？搞清楚美元走弱的根本原因，相信也就不难回答上面几个问题了。所以不要迷信中美合作统治世界之类的鬼话，谁信谁就准备交学费。
       既然美元无法维持强势地位，那么美国是否就一发不可收拾？会不会产生一种取代美元的强势货币？这两个问题属于回答了G2假设是否成立问题的后续问题。对于美国到底会衰弱到何种程度，这个谁都无法预测，中间会发生很多波折，美国未来的对手不完全是中国，还有欧洲、俄罗斯等主要政治集团，中间会有激烈的博弈，但是我相信这个进程会在20--30年以后达到一个相对稳定的平衡状态。新的强势货币不会再产生，取而代之的是几种主要货币，相信人民币会是其中的一种。
       总之，我相信20--30年以后的世界会是一个多极化平衡的世界。中美之间的博弈既非简单的零和游戏，也非双赢统治未来世界的美梦，中美的博弈必须要放到全球背景下去进行。

       对于日本社会近期的一些列举止行为我简单做一些论断性的判断：
       1、目前整个日本社会主体已经右倾化了，闹哄哄的场面只不过在互相配合着演一出双簧，就如我很早就说的那样，国家正常化是挡不住的步伐，只要有点脑子的日本人都会有这个基本认识；
       2、日本走上正常化道路不一定就于我有害，这里牵涉到中美日之间复杂的博弈关系，都在互相利用。日本走上正常化道路是他在激烈变化的世界局势下的一个明智选择，正常国家才有选边站的权利，被绑在别人战车上的非正常国家是没有选边站的权利；

       4、日本在台湾、南海、中南半岛甚至非洲等区域上更多地是对我形成掣肘，延缓我的进步，削弱我的影响。但是在这些区域的发展他没有根本性的战略规划，属于见招拆招的被动应对，长远来看他不是我们的对手；
       5、日本的战略关注点在东北亚，在朝鲜半岛、我东北、外蒙、俄罗斯西伯利亚一带；
       6、我们要提防日本的不是既有战争模式，而是下一代战争模式，比如生物战。
