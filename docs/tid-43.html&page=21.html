我最喜欢的藏语


不是藏族，但是知道一些简单的藏语，比如“尼玛”是“太阳”的意思，“达娃”是“月亮”的意思，“卓玛”是“仙女”的意思。当然，全国人民都知道“扎西德勒”是“吉祥如意”的意思。

其实我最喜欢的是“金珠玛米”，小时候曾经问过老妈，这是啥意思，她说是“解放军”的意思。我一直以为这是藏族同胞造的一个专有名词，刚才查了一下才发现，这个名词和藏语的其他词是一样的，两个字一个意思，“金珠”=“救苦救难”，“玛米”=“兵”，也就是说解放军是“救苦救难的兵”，在西藏农奴解放的时候，他们也把解放军叫做“砸碎锁链的兵”，一个意思。

在还不知道金珠玛米的意思的时候我就喜欢上了它的读法，不知道怎么了，觉得很好听。

后来知道了意思后还很奇怪，看来西藏的人是真的很喜欢解放军啊，起这么好听的名字。现在才知道，完全是巧合，这两个个词是原本就有的，只是做了一个组合而已，却产生了神奇的效果。

西藏去年经历了“3.14”打砸抢，不管是汉族还是藏族，都遭到了暴乱分子的袭击。后来藏族地区又有地方地震了，“金珠玛米”们又去救灾。我想但凡是有点良知的人心里就应该明白，到底是谁对他们好。

“金珠玛米”衍生出来的就是“金珠玛米呀咕嘟”，就是“解放军好”的意思。可是刚才在网上居然看到有人说听到“金珠玛米亚克西”的说法，呵呵，其实懂一点维语和藏语的人就知道，这种说法是不可能的。

“金珠玛米”是藏语，“亚克西”是维语，一个人怎么可能把两种语言混在一起呢。

《金珠玛米呀咕嘟》还是一首很老的歌，也许很多人都没有听过。藏族人民用朴实的语言赞扬着解放军，给了他最好听的名字。所以不要随便使用这个名字，免得玷污了他在藏族人心中的神圣地位。
