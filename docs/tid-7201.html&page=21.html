但是你不能只看结论，数据来源和分析方法更重要。

在全国2585个县/区中随机抽取80个县/区，每个县/区抽取4个社区，每个社区平均抽取25户共80*4*25=8000户做为样本
1.我认为随机性太大，80个县/区只占全国县/区总数的3%，加上地区发展不平衡
2.8000户家庭样本数量太少
3.按照统计的"家庭"的定义，成年子女与父母同住也算拥有住房
城市中的三口之家，即使子女已经成年并有自己的工作，只要子女与父母同吃同住，家庭的消费支出也是依照其收入共同规划的，子女与父母就同属一个“家庭”


有一套住房的城市家庭占69.05%，拥有两套住房的城市家庭占15.44%，拥有三套及以上住房的城市家庭为3.63%
真正能从房价上涨中得益的是有多套住房的家庭占30%，算不上绝大多数
