在企业中什么是是数据资产？

对于企业，数据资产就是由生产、仓储物流、销售、客户管理、财务等软件产生的数据组成，但这些数据还不能叫为数据资产。

现在的企业无论大小都内部拥有了：货品进销存、供应商收付、客户资料管理、客户收付的软件。传统销售企业以直接实体店面销售为主，少部分企业使用第三方的，如京东、陶宝、或微信公众号等网络销 售平台。

实体店面，客户软件由销售人员记录客户的特征及消费记录，但过程中客户的行为数据，是采集不到的或不全的，如：客户在A产品和B产品之间选择了B产品，可以记录结果消费了B产品，但不会记录A与B的选择原因及过程。

使用第三方平台，有可能从客户各个产品浏览页面的次数和停留时间来分析出客户的选择原因。但这个数据企业是不容易得到的，并且这个数据没有能和原来的软件平台形成一个系统，企业得到的数据是不完整的。

因此我的理解，只有在一个应用平台中：生产单元，能直接根据客户的行为、特征生产出产品。客户能快利用平台快速消费企业提供的产品，形成了一个消费循环的环路后，这时产生的数据才能叫一个完整的数据资产。
  
现在能看到的可以形成数据资产有阿里、京东和腾讯。
银行、气象局、地理信息系公司、天眼查企业等这些专业的数据可以直接进行交易，但能不能把这些数据也叫数据资产呢？
或许数据资产，不是我想像的这样。

古代瘟疫的大规模流行一般都是发生在气候急剧变迁（主要是变冷）的大背景之下。无论是汉末的“伤寒”还是罗马的“瘟疫”，抑或中世纪欧洲的“黑死病”还是明末的“鼠疫”大流行，都是在经过了一段时期的极端气候之后才大面积爆发，这既有由于长期气候干冷导致粮食歉收，从而时人营养不良导致身体素质较差抵抗力低下（在“三年困难时期”前后出生的国人仍有类似情况），也有同期欧亚大草原游牧民族为求生存大规模南迁四面出击导致东西方文明交流及病毒加速传播有关。

而且，气候的急剧变化也是加速政治崩坏的一个重要原因，而延续近400年直到隋唐初年的“汉末小冰期”正是有文字记录以来人类经历的最寒冷一段时期。

这个在之前《勒石燕然居然找到了》就有过论述。即对于中原农耕文明，因为小冰期导致的干冷气候，会使北方可耕地大幅缩减，汉末小冰期时牧场更是延伸了到中原地区的黄河以南。而且农田单位粮食也会大幅减产，这对于平均亩产只有1~200斤时代的的农业社会，往往会带来严重冲击，特别是在和平时期繁衍了大量人口之后。这也是16世纪小冰期导致明中后期出现流民痼疾的一个重要原因。

同时，在工业社会之前，古今中外任何农耕文明承平日久，必然会发生土地兼并，豪强并起，危及皇权统一。这也是为何历代王朝大多采取措施抑制土地兼并的原因所在，但大多归于失败。

在气候变冷，土地贫瘠之后，地主阶级为维护自身利益，势必加重盘剥和加强兼并。这会加深底层困苦，破产农民增多，求生而为兵源或流寇，治安环境恶化，社会不稳定因素飙升，进而导致维护政权稳定的皇权与看重自身眼前利益的地主官僚间矛盾冲突急剧上升。这就是各类政治文化等表面现象背后的深层次经济原因。如遇“天时地利人和”兴利除弊进行改良或可中兴，但一般到了朝代末年都已沉疴日久，无药可救，虽遇“明主”或有心杀贼，却已无力回天。

这也是为何在三国归一统之后西晋没有像其他久经战乱后的统一王朝一样呈现一派平和气象，而是在短短10年之后就爆发了断送社稷的“八王之乱”的重要背景。其后异族大举入关，相继作乱，终致五胡乱华，生灵涂炭，中原糜烂，天下板荡400年，汉族曾到过历史上最接近亡族灭种的地步。

而农业文明主要看天吃饭，游牧民族则更甚。同一时期，从蒙古高原到东欧平原的亚欧大陆广阔草原地带，到处都是一波波蛮族在向南迁徙。导致的后果在东亚是“五胡乱华”，西亚是嚈哒入侵萨珊波斯，欧洲则是“上帝之鞭”摧毁了罗马帝国。
