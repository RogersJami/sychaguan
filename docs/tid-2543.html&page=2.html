卦意中，不贪不急显然是养生重点，能做到不贪不急，自然也不容易忿怒。忙总再三教导的这三点不仅是事业人生的关键，也是养生的原则。

对稽康不太了解，但感觉是个虚伪的人，过于在乎名誉，类似现在小姑娘们的要风度不要温度。不要温度可能感冒，为了名誉，稽康是把命都送掉了，尽管在刑场演奏广陵散这一场景很飘逸。看他的一生所为，养生实践可不怎么样。他号称不在乎权贵，但身边全是，连老婆都是；他公开与山涛决裂，但却把儿子托付给山涛；说是要隐居，偏偏喜欢评论世人政治。拿对待钟会来说，钟会是当时公认的贵族才俊和政治接班人，他父亲是曹操最信任的手下之一，书法至今有名，钟会自己也很有才，喜欢结交高人，来拜会稽康其实很给他面子。如果思想交流，双方都可能受益，即使不想聊，也不必那样狂妄。公开地诚心恶心钟会，除了虚名，我不知道稽康还能得到什么。最后那句问话，其实反而暴露了稽康内心欲望，倒是钟会的回答不卑不亢，绵里藏针。稽康这样的人，心口不一而且狂谬，既贪且忿，又身处乱世，最后的下场实在不冤枉。
