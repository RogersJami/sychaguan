原 创: 草 药 郎 中 ( 七 )

　  

    疮治好后，简师傅自然是隆重地备下酒菜要感谢何师傅的仗义出手，倘若不是何师傅，背脊上的这个‘天疮’，又何时是好哦？　　可何师傅呢，拉都拉不拢去，甚至连药钱也分文不取。何师傅说：咱们都是早不相见，晚上见，低头不见抬头见的几个老熟人，朋友有难，出手相助，这是本份，你又何必如此啊？　　简师傅说，这药钱总得收一点吧！何师傅却说：自己采摘的，自己配制的，都是些根头草脑，也就几味药，不值钱，不必弄得这么生分！　　给别人施药诊病，即不喝人家的酒，也不收人家的钱，什么意思？多年后又一场经历揭开了这个谜底，此乃后话，慢慢地往下说吧。　　常言道：好事不出门，恶事传千里。　　人见这简师傅不再赤膊上阵，背上的那个‘天疮’也给治好了，自然，一传十，十传百，人人知道本厂内有位不显山露水的何师傅，有绝门手艺在身。自此以后，上门问候的人络绎不绝。可这何师傅脾气古怪，依旧拒绝任何上门求药者，用当地俚语讲：是人不是人他懒得呲你。　　老话也说：人吃下的是五谷杂粮，又哪有不得百病的人？　　七一年，砖瓦厂从乡下招来一批青壮，年龄都在二十上下，一个个精力充足，体力强健，全是跌一跟头卵浆直刹的后生子们。内中有位青年，姓张，大热天里，腰中间，也就是齐肚子围腰的那一圈，长了一圈看似痱子，又不似痱子的东西，这东西长在腰部，奇痒无比，渐渐地从小红点点又长大成一个个的大水泡，又慢慢地由中间向尾部发展开去，如同一条蛇，头部不停地向前蠕动，一点一点快接近到尾部即脊骨的正中了。　　小张先是到厂医务室就诊，之后，医院也去过，大包，小包的药也吃下不少，各种膏药也涂过，消炎的各种针剂也打遍了，可总不见有好转，你说它是丹毒吧？炙火也烧过，总不见这‘痱子’回头，并且越发越很！越发越多。　　可怜这小张，先前活蹦乱跳的一青年，眼下一日黑只能巴巴的呆在宿舍里，衣服也穿不得，身上穿件汗衫，撩发得整个腰部火烧火辣般的痛疼，无奈之下，这小张整日整夜只好穿一件三角裤躲藏在室内，不敢迈出房门一步，一日三餐的吃食全靠同室好友帮忙递送。　　小张的大人闻讯后也从乡下赶哈来，帮着求医问药，寻找了不少的民间郎中总不见有点好转。为此，也曾上门找过这何师傅，可何师傅却一毛不拔，拒人于千里之外，回说是不懂也不会，你还是另请高明去吧！　　眼看病情越来越很，还伴发起高烧来，去了无数趟医院却一直没有好转的迹象，万般无奈之下，父母双亲只得求救于厂领导了。　　厂领导专程上了何师傅家登门拜访，又给他作了大量的工作，说了许多的好话，并一再承诺，如果出了问题由厂里承担。　　如此这般，何师傅这才应承下，答应给看一看，只见小张腰际的红肿不消，白色的水泡中间有那么一点点的黑，如同眼睛眸子一样，由红色渐渐变成白色，且不断的向中间发展开去，形状如一条腰带，布满在整个腰间。　　有人一旁问何师傅，小张这腰间长的是个么得东西？何师傅仔细看毕，一口气肯定的说：这是‘蛇丹’，又叫‘斩腰丹’，是沾染上了蛇毒所引起。并指着腰部的那一点黑泡泡说：这就是蛇的眼晴，它在慢慢地向前爬。如果整个腰都发满了的话，毒气便会攻心，毒气攻心，这人必死无疑。并对小张的父母说：眼下就诊还来得急，丹毒还没长拢来，我先去配点药，然后我再告诉你怎么使用。　　依然是白色的药沫子用酒调成糊膏状，再用鹅毛蘸上糊膏涂在腰际，一天数次。又根据丹毒变化再调整用药及药量。     就这样涂抹了几天，只见红肿渐渐消退，白泡慢慢枯萎，又见向前爬伸的丹毒开始回头，个把礼拜，腰间的丹毒变成黑色并结了痂，丹毒枯萎，衣服也穿得了。半个月之后，黑痂完全脱落且长出了新肉，又如先前一样。     自然，小张一家子感谢不尽，要掏钱付医药费，何师傅依旧不收半分文，只是盛情难却，秋凉时节，被小张一家子拉去家中，喝了餐惬意的酒。     何师傅从不张扬自己的医术与药，乃是怕自己没正式拜师学艺，倘若轻易出手，接治病人，莫万一弄出个什么问题，怕吃不了蔸着走，到那时，悔之晚矣！因为敬畏这一点，故而一直不敢张扬。
