凯恩斯主义学派的核心观点

1936年2月，英国经济学家凯恩斯John Maynard Keynes发表其代表作《就业、利息和货币通论》，一举奠定了现代宏观经济学以财政政策和货币政策为核心的理论体系，同时也开创了凯恩斯主义学派。


凯恩斯主义的思想孕育于1930年代的世界经济大萧条，其核心观点是：市场会失灵，灵活地执行税收和支出调节能实现不衰退增长，即主张国家采用扩张性的经济政策，扩大政府开支，实行财政赤字，通过增加需求达到刺激经济增长的目的，同时将失业率保持在一个最低的水平上。这一学派凭借二战期间让政府干预经济而殊荣备至，代表人物凯恩斯出任战时英国的财政部顾问，哈里·怀特Harry Dexter White时任美国财政部部长助理。

凯恩斯主义的思想孕育于1930年代的世界经济大萧条，其核心观点是：市场会失灵，灵活地执行税收和支出调节能实现不衰退增长，即主张国家采用扩张性的经济政策，扩大政府开支，实行财政赤字，通过增加需求达到刺激经济增长的目的，同时将失业率保持在一个最低的水平上。这一学派凭借二战期间让政府干预经济而殊荣备至，代表人物凯恩斯出任战时英国的财政部顾问，哈里·怀特Harry Dexter White时任美国财政部部长助理。
