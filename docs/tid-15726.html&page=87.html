技术人才断层的原因是培养中断。
培养中断的原因：
第一，行业不景气，不招收新人。老人的经验无从传授，新人无处可学。
第二，拒付学费，经验实际上从试错中得来的。现在管理过于讲究SOP，试图让白痴完美地复制成有经验的工程师。降成本逼迫过甚。
第三，信息技术普及，使得年轻人以为知识易得，看轻前辈的经验，不愿再学。
第四，宽泛内容的大学培养脱离实际需要，实习时间太少。
第五，低难度，高回报的创新行业吸引了过多的优秀人才。
