没办法，举个例子，一个单位300号人，顶在一线干活的6、70，剩下的负责喝茶、看报、找干活人的碴，比如办公 ...

       你举的这个例子我认为要从多方面去考量。
       首先我完全了解并认可你描述的状态，现在机关内部确实存在比较严重的人浮于事的现象，而且往往是干活的流汗还流泪，耍嘴皮子的轻松且快活，有一股歪风。虽然我知道一些部门存在的必要性，但还是从心底里瞧不起那些玩花架子的部门。当然，这主要是党口的部门，不是他的活不重要，而是干这些活的人太次。一天到晚的考察评估，考来考去都是玩些部门间的游戏，跟民众的生活完全不相干。当党政部门的活动变成一种自娱自乐的时候，这些部门的存在必要性是要受到怀疑的。除了玩虚的，再就是推脱责任。做事的总免不了错误，总免不了得罪人，只有不做事的人他才不会犯错。其实对这个事情，早就有好的制度设计，就是首问负责制，不过基层没有好好去落实。再一个就是部门内部的轮岗机制很不健全，家长制还很明显，有些人干活没水平，跟领导处的时间和精力大大地，他就能谋个好差事，这是做人的差距。
       其次呢我还是要劝你要放正心态。机关中规划、组织等务虚的部门还是有生产力的，别光以为就自己窗口或一线执行的部门在干活。统率机关的方向指引非常重要，他的方向指错了，你们流再多的汗都是白流。另外象办公室等部门，其实也是很锻炼人的，里面大有学问。
       总的说来，一个有活力的机关，应该是自身服务人员占少数、内部协调精力占少数的机关，可惜现在这样的机关很少。
