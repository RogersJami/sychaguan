虽然这是政治版块，但是我不是政治参与者，我只是被政治影响者。

我不在意谁执政，不在意是否民选，我在意的是我的财产、生命是否安全，我的未来是否存在希望。

至于前苏联、今俄罗斯的政治家是否符合历史评判，跟我鸟相干？


我最近在东北工作，经常走长深高速。今春以来，经常看到路边的农田里，几个老人家弯腰耕种，所以神十上天又如何？我父母辈的农民的耕作方式与我小时候没区别；在我老家，几亩农田、一户宅基地，因为南水北调要征用，政府发出拆迁通知，每户补助几千块，其他地方还几亩薄田来（好一点的田早就被村民分走了，临时调剂的只能是低产田）。

土改的时候口号是耕者有其田，请问“有”是不是意味着所有权？但是如今呢？为什么说剥夺就可以剥夺？

滚他妈的什么历史罪人，谁让我的父母、朋友、孩子，付出了相应的努力，却得不到相应的安全保障和上升通道，才是历史罪人。
