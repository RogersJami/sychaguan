愚者乐言 发表于 2013-4-8 01:06:14

事后盘点一下，关于对李总理“城镇化战略”的解读，我始终认为这个判断不错
关键在于城市化不意味着每个进城居住的都得拥有房子的产权，全世界都没这道理，有房子住就算OK了，你能租到房政府的城镇化战略就算完成任务了
很多人的逻辑出现误判的心理根源就是不肯面对城市化会带来的贫富分化，买不起房的人，按照经济学道理来讲要么选择贫民窟，要么选择政府出钱补贴建廉租屋
前者由于政府要面子和意识形态不肯承认反而打击（群租房市场和城中村市场可以视为变相的贫民窟 ）
后者十年来政府一直毫不作为
这样一来，房价除了蹭蹭蹭往上走没有第二种选择了，这样的单边行情不出现才是没道理的
所以说，中央政府高层的学识和能力还是不容怀疑的，政府一直的调控口吻都是“防止房价过快上涨”，请各位注意红字的两个字
那么十年来的房产调控失败在什么地方呢？就失败在领导层不能像90年代的朱镕基那一届政府，有所历史担当，不怕背负政治的骂名
所以政策就成了开发商做婊子，某人立牌坊。而且是一个大大的牌坊
不管他从本心而讲愿不愿立，或者扯什么其他人为什么不立牌坊也不挨骂，他有今天的一切非议都是他自找的
