说起农村土地流转，对于中国大多数的农民而言，人均的耕地有多少？一亩地流转能够农民多少收入？要知道，对于大多数耕地而言，流转以后的用途还是耕地，就目前的粮价而言，你觉得地方政府会给农民多少钱？

举个老家的例子，老家在江淮流域某地，这个地方的田已经算得上中国有限的良田了。如果遇上风调雨顺的年份，一亩田按两季算，麦季大概是八百斤左右，稻季大概在1200斤。小麦今年农民拿到手的收购价是1.14每斤（目前为止的最高收购价，大多数农户的卖价在1.1以下），水稻的收购价，今年的还没有出来，往年也就在1块出头的样子，这样，一亩地一年的毛收入也就2500不到，除去种子、农药、化肥、人工、灌溉、收割的费用，一亩地的收益不到2000。注意，这还是风调雨顺的时候，正常年景是没有这个产量了。小麦在四五月份的时候如果遇上高温高湿，极易发生病变，这个会严重地影响产量，前两年的亩均产不到500斤。就这样的田，人均也就一亩左右。如果流转到种粮大户手里的话，一年至多也就给农民800元-1000元。也就是说，土地流转之后，一个四口之家，一年从土地上得到的流转收益不会超过4000元。

再说宅基地。集中给农民盖回迁房，然后让农民免费居住？这样的做法在少数地方做可以，在大多数地方做不到，因为地方政府没有那个财力。现在的政策就是，宅基地的房子补偿，然后再去买回迁房，回迁房与老房的差价每平方至少在800元。不过，现在就是这样的措施也实施不下去了，第一个，农民不干；第二个，土地现在已经没有前两年那么好卖了，地方政府拆迁的利益急剧减少，地方政府也懒得干了。老家，年前的说法是，全面拆；四五月份时候的说法是沿着省道两侧拆；现在的说法是，不拆了。就算是前几年拆的，比如镇江某地，说好的免费回迁房已经推迟了一年多了，还没有建好。

我倒是想问问，土地流转了，农民怎么就成了既得利益者了？
