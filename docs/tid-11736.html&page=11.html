看电子书觉得静不下心来，还是想读纸质书，不过没看过第一套，可以直接看第二套的吧
其实忙总的主要思想与特点，用心看几篇网上的帖子就能领略大概。读不读纸质书，另当别论。

关于第一套书的说明：忙总认为他在网上与大家交流的各种闲聊中，最有价值的部分大概是谈他对管理的认识。我从一个忙总学生的角度，一个教育工作者以及一个管理者的角度，觉得忙总谈的管理，如果整理一下，或许会对很多想上进的年轻人的成长（尤其是职业成长）有所启迪，所以经忙总许可，从忙总3年内先后在4个论坛里发布过的数百万文字里面，选取了其中一部分，编辑成第一套书。第一套书的主要出发点是：希望这些年跟随忙总学习请教交流闲谈的群众，能把它推荐给身边的年轻人。第一套书推出去之后，我调查了大量第一次因这书而读忙总文字的人，群众反馈是很有启发，即使他们中很多并不在企业工作，也不从事于管理工作。因为忙总管理文字中始终透着：做事＝做人，做人＝做事，踏实做事，诚实做人。（本来我是想先出忙总读《易经》心得那数十万文字的，因为我自己读打印版的觉得不方便。但是考虑到这部分的读者数量会很少，所以就暂时不出了。）

之所以有第二套的《忙总管理笔记》，是托第一套的福，这个从第二套书忙总写的“前言”中就能看出，忙总看了第一套之后，觉得有必要补充一些内容。但是忙总毕竟只能掏出来高级干货，所以这第二套书，管理升级了，对从事于总经理或有志于从事总经理的人群来说，这第二套书是相当实用的。市面上的管理类图书，是不会这么写的，因为没有人像忙总那样乐意且能够掏干货、指导年轻人的成长。

如果说第一套书是老少咸宜的普及本，那第一套书的“企业管理”的味道就会浓厚许多，适合给在做事的而不是搞研究的人看的。
