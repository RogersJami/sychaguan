低附加值产业中国这匹角马没机会成为狮子吗？比如运动鞋这个领域中国做的还不错啊，慢慢地把美欧的一些挣 ...
很多产品已经成为狮子了，就如你所举的运动鞋这一块，基本如下发展：

8X年是来料加工国际品牌，间接帮我们培养了一大批人才。
随后人才分流出企业，成立了许多小作坊小工厂开始仿冒国际品牌，前期那个质量惨不忍睹，当时一朋友买的假冒阿迪达斯，踢球一个月后就裂口。
9X年经过强劲竞争，血战脱颖而出的一些公司开始尝试OEM+ODM混合模式。
至于现在不用说了，国内大部分市场份额基本上都被国内公司垄断。

以上流程可以在大部分行业中看到，基本类似。
