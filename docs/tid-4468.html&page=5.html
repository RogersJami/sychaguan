做擅长的事，资深的技术专家值得尊重，也是很好的活法，并不是每个人都适合领导岗位
我觉得，作为领导，除了能力、经验以外，有一些最基本的东西更重要：一是主人翁精神，一是良好的工作态度(哪怕是装的)。

先说主人翁精神，就是做事要以公司利益为先，比如说，裁员的时候，马屁精要照顾，老黄牛的也要适当照顾；分蛋糕的时候，哥们弟兄要想到，业务骨干也要想到。

乙方公司在上次提了一批经理后不久碰上萧条裁了不少人，不少老老实实能干活的被裁掉了，而马屁精、味精一个都没动。我带的一个黑人小伙，当地本科、研究生毕业，特能干，被裁了(这事涉及两个混蛋经理，一个我写过了，另外一个还没写)。接他的是一个亚裔姑娘，过去4年生了俩孩子，我查了一下她过去52周的工时，才800来个小时，当时尚不能全职上班(裁员后她改全职了)，她告诉我，自己都没想到自己能留下(没有后台，但是跟领导跟得比较紧，长得不错，有点嗲)。另一个专业，剩了6个人，俩老头(一个是组长)+某总监的儿子及其女友+俩身材火爆的金发美女(我称她们为味精)。

再一个就是工作态度，领导吊儿郎当很影响士气的 ---- 吊儿郎当的人，能力多强都不能提拔(个人观点)。

资深的技术专家往往是需要领导提携的。想在公司内被认可，需要领导帮着宣传；想在业界有名气，需要参加一些会议什么的，更需要领导支持。其实行业协会也喜欢吸收领导 ---- 因为领导能量大，掌握资源多。
