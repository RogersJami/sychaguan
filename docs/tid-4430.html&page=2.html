   四十五 完美进阶练气三层
  　日出日落，日子悠然过。
　　灵田的事情告一段落，这之后，依灵的生活作息又规范了起来，一天三次的修习玄气诀，心态也放开了，契机来了，就进阶，不强求，不怕来，就这么自然而然的。
　　剑法是每天的清晨和黄昏各练一遍，也自是日益熟练，使出法来虽说和先生那样的飘然若仙还差的远，但是也可以说是穿花绕树自在行云了。看样子，这剑法，依灵是越练越觉得顺手，越练越感觉到此剑法的博大精深。
　　书法是每天用一个时辰的时间来用功修习的，现在的她，字体还没到先生的火候，但是骨架也渐渐的明朗起来，有属于她自己的独特风格，清丽而隽永。
　　其他的时间，藏书室倒成了依灵长期的去处，趁着现在年纪小，修为不高，还没有到修习术法的时候，多看点基础类的书籍，把地基打牢固，万丈高楼平地起，其中地基都是重中之重，无论是修为还是境界，玄法还是剑法，基础类的通透了，往后才能更好的触类旁通。
　　每天晚上入睡前都会进去须弥戒陪小九儿说说话，游玩嬉闹一阵，或是跟着小九一起去一些依灵现在的修为可以到达的地方。寻找土系或是木系的晶体，当然，这样找到的多数是依灵自己可以用到的灵晶，小九倒是玩得不亦乐乎，开开心心的数着灵晶，数好后就直接给依灵装到储物手镯里，或者是笑闹着要依灵直接服下。
　　又一次的服下灵晶之后，依灵感觉体内灵气的运转一下子猛烈了很多，分出一点的精力打开石屋山谷的禁制，一边按着每天修习的运行路线调息。
　　半年的时间，她不仅将玄气诀修炼到将要突破练气第二层的地步，九天玄元剑法的初级也是修炼到了小成的地步，并且，因为灵晶的存在，依灵体内的灵气修为增进的速度也快了许多。并且因为灵晶是天养地生的灵物，不含其他杂质，也就没有所谓的副作用，再加上依灵也不是个太过贪吃的人，每次都是等到体内将原来的灵晶消化吸收完才会用其他的。
　　其实土灵晶和木系灵晶都有比较温和的特性，对人体的更是没有什么副作用，依灵体内的灵气目前已经有十根头发丝粗细的程度，并且灵气如玉般雪白，依灵也没有见过其他人的什么样，以为同阶的别人都是一个样子的呢，也就没有太过在意。
　　进入练气第三层以后，就可以去传道院学习，学习简单的术法，阵法，炼器，或是符箓，甚至是进修剑法，都可以的，这里也代表着正式踏入了修仙者的大门，也为练气五层以后的学习法术和技能打下坚实的基础，在这一个阶段往往它的象征意义就是一个选择，选择你以后修仙的路上要走的是那条道。
　　目前，泰一峰他们这一批杂役中的徐阿九和李玲珑已经在传道院学习了，并且听说这几天有可能还会转到法术院去，王采幽也在一个月前转去了传道院，只是王采幽的进阶家人的帮助和她自身的努力都是相辅相成的，依灵可是听说，她的一位远祖因为她的用功赏她一枚进阶丹，再加上她本人将近一个月的闭关，才有了那样的成就，这份执着，这份毅力，对于他们这些还都不是太大的孩子来说，也很是的难能可贵的，依灵还是很佩服她的。
　　听到这些的时候，说依灵一点点都不羡慕是假的，但是，各人有各人的造化，别人有的，你不见得会有，同理，你有的，别人也不见得也有，想要什么样，就要自己去努力，这样才有意义。
　　依灵深深的明白一个道理，那就是，天下没有免费的午餐，付出和收获多数时候都是呈正比的，你有多大的付出，就有多少的回报。
　　如果说，王采幽的进阶有家族的帮助身影的话，那么徐阿九和李玲珑的进阶都是靠自身的意志和毅力硬抗过去的，闭关两个月，两个人相差三天，忍受无边的心魔考验，无穷的幻境实战，还有无边的寂寥，最终过了那一关，进阶练气期五层。
　　依灵为有这样的朋友而自豪，现在，她自己也要尝试着冲击第二层障碍的的关卡了。雄赳赳，气昂昂，向着第三层的关卡，进发！
　　其实，在依灵第一次尝这灵晶的时候，体内的灵气就累积的差不多了，只是依灵想要自己的境界更稳，灵气更充足，灵魂也越发圆润自然的时候，就这样，她一直的压着，一直的等待着，而现在，这个契机到了，依灵知道，现在这个时机成熟了，冲关进阶的时候到了。
　　须弥戒内的灵气比着外面还是要浓郁的多，先生这里的灵气比着泰一峰其他地方还是要浓郁的，只是和须弥戒里面灵气浓重的程度还是略有不如的，只是现在的依灵可不敢现在在须弥戒里面进阶，那不是借力，那是找虐，里面的灵气，仙气，混沌气息，什么都有，甚至还有着那么一些魔气，天地灵气那里齐了，谁知道会是怎样的一锅粥呀，至少，依灵现在还没有那个胆子在那里试。
　　灵气不断的向着依灵的方向聚集，依灵不敢大意，一手握着一颗先生留下来的极品灵石，是先生专门为她进阶的时候准备的，一手握着一颗灵晶，当到时候万一灵气不济，也可以备不时之需。

　　但是对于风灵子来说，极品灵石还是有的，并且不少，也不知道是出于什么心理，临走之前，给了依灵一百颗左右的极品灵石，也幸亏依灵对于灵石的物件不太在意，储物戒里的多是那位先生放的，储物手镯里面是外祖父给的，也有不少，因为这些，在依灵的潜意识里，这些东西都是别人暂时放在她这里的，说不定什么时候就会收回去，所以，他也就从来没有惦念过这些东西。
　　并且一直认为自己很穷，除了那一千亩地就是穷的叮当响的那种，
　　现在是进阶的重要阶段，先生的嘱咐很清楚，进阶的时候，开启阵法，用极品灵石，其实，在修仙界，有这样的一个传说，一个修真者，从引气成功，以后这中间的每一次进阶，一直都以自己本身的灵气来进阶，可以辅助灵石，但是不能服用药物，（因为副作用和丹毒的原因）一直到他筑基成功，这样的进阶筑基就称之为完美进阶筑基，能够获得成功的修士，将来的成就将无可限量。

　　依灵的体内的三个月前已经到了可以进阶的地步，只是因为心境的原因，她压制着，现在体内的灵气更精，更纯，心境也是圆融无间的。再放了一颗葡萄大点的灵晶到口中，入口又是唇齿留香，灵气四溢。
