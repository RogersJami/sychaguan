10. 面试

我一直觉得，我家LD无论是做mentor还是做concept definition lead，都挺亏的。

1. 工资没有动：职位动一般工资会动，如果不动，也会有个说法。乙方的人都是甲方出钱养着的，涨工资需要甲方批准。由于LD是甲方任命的，稍涨点工资甲方肯定会批，对于乙方领导来说这是个顺水人情。问题是R经理装作这事没发生。

2. 简历上不能写：甲方任命LD的这个组长，比B组长的那个含金量高的多，但是简历上不能写。因为LD是乙方员工，乙方的人事档案上没有反应。大的业主公司，雇正式员工要进行独立的背景调查。这种调查一般是直接联系人事，写在简历上容易惹麻烦。

不觉得甲方老大的安排对LD找工作能起什么作用。

面试的是一个重点项目项目经理职位，项目的规模要大一些。业主公司之所以雇合同工，估计有以下几个原因：1)试用，不喜欢不续合同就行了；2)重点项目不像生产项目，不是老有，一旦断流，公司安置起来余地比较多；3)因为税务原因，有些老法师(老游击队员)更愿意做合同工。

LD的项目管理业绩，以及多年在业内老大系统里的历练，使她对项目管理有本质的理解，有自己的、成熟的项目管理思想和理念，对面试经理的问题，不仅能回答，而且能延伸，不仅能举出实例，而且能上升到理论高度(这与我们经常一起探讨有关系)。她的背景十分难得。一般业主背景的项目经理/项目工程师缺乏项目执行经历，而咨询公司背景的则缺乏和业主相关部门直接对接协调的经验。她两者兼有，面试经理是一个非常有经验的老法师，不可能不注意到这点。

面试中，LD提到自己是甲方任命的Concept Definition Lead，并且拿出了甲方老大发布的组织结构表。组织结构表上甲方员工和乙方员工用不同颜色表示，很显然LD是以乙方员工的身份统领甲方的项目定义团队。甲方公司是业内老大，面试经理很清楚这个任命背后的意味。

面试的另一个人问了个问题，“既然甲方那么喜欢你，为什么不直接雇你呢？”，LD说，“他们正在做这件事情，但是不能确保能成。”----甲方公司直接从乙方雇人，涉及到赔偿问题，即便没有O老板，也需要花时间走很多程序，这个面试经理是了解的。

由于是合同工，本来不需要再一轮面试，但是没过多久，LD又被他们叫过去谈。这次见了不少人，这时LD清楚，已经拿下了。

拿下了，又有些纠结。因为毕竟是个合同工职位。因为放弃的不仅是多年奋斗打开的局面，还有甲方老大给她画的饼。
