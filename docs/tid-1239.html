秋玲的故事

（一）我是在一个朋友家的聚会上认识秋玲的， 她和我一见如故。 秋玲来自江苏常熟， 她身高大概1.58 左右，皮肤很白，眼睛也大，可惜有一点水泡眼。 但她还是个美女， 高高的鼻梁，喜欢“咯咯”的笑，并且笑起来时嘴角自然的往上翘,很迷人并给她增添了很多魅力。


她当时拿的是三个月的学生签证（读英语），每三个月续签（如果幸运没被拒签的话），当时在新加坡拿学生签证是不允许打工的。

从那次聚会相识后，秋玲就会经常给我打电话聊天，也会到我住的地方玩。当我觉得自己讲的话并不是很幽默好笑， 她听了也会笑时我就会开她玩笑，“骂”她“花痴”，她听了还是笑， 并不介意。 总之，她是个性格开朗的年轻女子。

有一次，她来找我，她看来不像往常那样开心，好像有什么烦心事的样子。

到我的房间坐下后，问我：能不能在我这住两天。 我便问她发生了什么事。 她犹豫了一下，便告诉我。。。





（二）
当时还在中国时，秋玲的一个亲戚在生意上和一个新加坡人 Vincent有来往，在一次饭局上秋玲和Vincent见过面， Vincent答应帮秋玲办签证并说到时可以住在他家，秋玲和她家人自然相当感激他的帮助 。

我见过Vincent，整个人的样貌简直令人无法恭维。他比秋玲大10岁左右，个子比秋玲还矮，又瘦又黑，还戴着副眼镜。。。这些不提也罢了，最主要他还长着张歪嘴。 但尽管如此，他有一份很不错的职业，在一家大公司做高层主管经理，属于事业有成之人。

其实，自从Vincent那次在中国见到秋玲就对她一见钟情，但由于他心里知道自己的样子配不上秋玲，便不敢表白，就想等秋玲过来新加坡后再慢慢培养感情。Vincent对秋玲真的很好，不仅让秋玲免费吃住，还经常买小礼物给秋玲，（学费等其他费用是秋玲自己付的）。Vincent有空就会带秋玲四处走走，并请她吃饭，秋玲心里很感激并把他当作年长的朋友看待。


虽然Vincent没有当面求爱表白，时间长了， 秋玲还是感觉到他的意思，可是就是无法爱上他， 这样一来便觉得再住在那里每天面对着他非常难受，天天还要故意回避装傻相当辛苦。



（三）
秋玲在我这住了三天后，便问我能不能正式搬来与我同住，并表示愿意分担房租。 我这个人很需要个人空间的，除了亲人我是很不习惯与他人同住一个房间的，而且我那是个单人房间 。

我便对秋玲说：“求求你了。。。我真的不喜欢与别人住在一起的。我这里又小， 我看你还是另找一处，我也可以帮你找找看”。


在她回去前，我还跟她谈心，说：其实Vincent这个人虽然长得不讨好，但他对你倒是很真心的。。。

秋玲叹气，道：“唉，这我知道。。。他平时在出门上班前，总是对我百般关照的，我都不对他好声好气，还说他罗嗦，烦死了。。。他都一点也不生气，但不知为何，他越是对我好，我就越讨厌他。。。”。

秋玲回去以后， 有一段时间没和我联系。 直到有一天傍晚，在路上碰到她 。。。

（四）

秋玲看上去憔悴很多，原先白净的皮肤变得苍白，眼睛里带着血丝，眼神也相当疲惫的样子。 但见到我，她非常惊喜立刻上来挽着我的胳膊，笑着：“哎呀，太巧了，碰到你。我还一直想给你打电话，但就是没空。。。”。

我便问她在忙什么。她带点神秘的样子告诉我她在上班赚钱，等有了钱她就可以搬出去自己住了。

我一听有些诧异，便问：上什么班？在哪里？。。。。做家教吗？（当时在新加坡中国学生唯一被许可“做家教”来赚钱补贴自己的）。

秋玲说不是家教，并邀请我一起吃晚饭，然后带我去她上班的地方看看。

一路上，我带着好奇心和秋玲闲聊着。我们在裕郎的一个巴士站下了车，秋玲指着前方的一个小型夜总会对我说：就是这里。

一看，我的脸就变色了，对着秋玲压低声音，怒责道：“你疯了，你怎么竟然到这种鬼地方上班！这里是些什么样的人来的地方你难道不知道？ 再说了，这种地方警察时常突击检查的，万一被他们抓住是要坐牢的！”。

秋玲苍白的脸一下子涨红了，有些紧张又有些羞愧， 急辩道：“你别急嘛，其实这里的客人还好了，就是一般的陪他们喝喝酒，唱唱歌的。小姐不答应“出街”，他们也不会强来的。 。。 还有，这里很安全的，老板说他搞得定的，警察来检查前有内线会给他们通风报信的。。。”并拉着我的胳膊“我其实来这上班还不到一星期，既然已经到了，进去看看吧，现在还早没客人还没来呢。。。”。

我看着她这样真不知要说什么才好，心里一转念：也好，我先看看这地方到底是什么样的，这样，我才能更好的劝她“悬崖勒马”。




（五）

一进夜总会， 就见一个风韵犹存的中年女子迎了上来：“秋玲，今天来得很早啊。。”她上下打量了我一下，对秋玲说“这个是你介绍来的小姐？嗯。。。很好很好。。。”。 我连忙摆手否认。

秋玲：“妈咪，这是我的朋友，正好路过，我就拖她来这里看看，见一下世面而已。”妈咪没再说什么，见我们在一边坐下，还叫人送来两杯啤酒。

这时，夜总会里的确还没有客人，有三四个小姐聚在一起聊天。 在昏暗的灯光下，我环顾一下四周，稍稍定下心，开始劝秋玲不要在这种地方工作，还不如找份家教做做。

秋玲说：“咳。。。我试过，但最近家教难找，现在正是学生假期，很多家长都带孩子去国外旅行了”。

我接着问：“那你到这来，Vincent 知道吗？”。

一听我提Vincent秋玲沉默下来，低着头用手指顺着杯口划圈， 十几秒钟后，她又抬头声音有点懊恼，道：“都是因为他我才来这上班，我晚上上班，回到家时他已经睡着了，我就不用整晚的面对他。。。 有一次他没睡等我，我一进门他便问我去那了，怎么这么晚回来。。。我当场就冲他：这是我的私事，你管得太多了。。。他就不敢再问了。。。反正我要想办法搬出去”。

我听了轻轻叹了一口气，也不知该说什么好，自己又帮不上什么忙，这毕竟是他俩之间的事，秋玲实在不爱他也无法强求。

这时，进来两个40多岁的男子，妈咪笑咪咪的迎上前一阵寒暄，其中一男人还用他那猥琐的眼神把我和秋玲从头到脚的细细打量起来。。。并对妈咪说：就那两个吧。。。

见妈咪向我们走过来， 我赶紧对秋玲说，我要回去了，你也和我一起走吧，不要再呆在这样的地方了。 但秋玲不愿跟我走，离开前我便叮嘱她，千万要小心点。





（六）

两天后，我突然接到Vincent的电话，他的声音听起来有些着急，问：“秋玲，这两天是不是在你这里？”。我回答：“没有啊，怎么了？她没跟你说去哪里吗”。我突然有一种不祥的感觉，很有可能是秋玲出事了。

Vincent一听，更焦急了：“她没有跟我讲过， 她在这里没有很多朋友，所以我就想到可能在你这里”。。。

我一边听着一边纠结着是否应该告诉他，秋玲在夜总会上班的事和我的担心。

Vincent在电话的那一头叹了口气, 说:“我真的担心死了，不知道她跑到那里去了，又不敢胡乱报警怕对她不利。。。咳。。。既然不在你这那就打扰了，不好意思 。。。谢谢你了”。


就在他快要挂掉电话的当口，我突然叫他不要挂，我有事要说。。。就这样，我把事情的来龙去脉和我的猜测担心跟Vincent讲了

听完我一习话，Vincent在电话的那头沉默了许久， 然后连连叹气，声音有些颤抖，道：其实，我是真心喜欢秋玲的。。。但她若无法接受我，不也会勉强她的，她完全可以同我讲明，我会帮她另租房子的，真的不用这样。。。。

挂了电话， 我发现自己早已一身冷汗，在新加坡如此炎热的天气下，我竟然浑身打着冷颤，心想自己还算幸运：那天我如果没有早些离开那个夜总会的话， 很有可能我也被抓了，那就是“跳进黄河也洗不清"了。

我心里也为秋玲的命运而焦急担心， 她如今到底在哪里呢？

就这样，我每分钟都盼望着得到秋玲的消息和她的下落。

（七）

又过了两天，我在房间里听到外面一阵“敲门声“，打开房门一看：是秋玲！！！

看到她，我相当惊喜：“是你，你这些天到哪里去了，可把我急死了。。。”。

一把将她拉到房间里坐下后，我仔细的打量秋玲，她看上去还是有些疲惫，但总的来说别来无恙。

我便想可能是我之前的猜测错了，一下子感到非常不好意思，急着表明到：对不起，Vincent前两天打电话到我这找你，我还当你出了事。。。就把你的事情告诉他了。。。我真的不是故意“泄露秘密”的，你知道我不是那样的人啦，请你不要怪我啊。。。

秋玲抿了抿嘴，面上带着羞愧，过来拉着我说：“我怎么会怪你呢，我是来感谢你的。。。。”见我一头雾水的便跟我把事情说明。。。。

原来，秋玲的确是在那晚出事了，警察到那个夜总会突击检查，把所有没执照的小姐包括秋玲抓走了。

Vincent 和我通完电话后，立即根据我告诉他的夜总会的名字和地点， 通过他的一个做警察的朋友查到专门负责这次行动的警署，并到了那边为秋玲辩解，同警官说：秋玲是他的未婚妻，因为他的错两人之间发生一些误会和争吵 ，她不开心才跑到夜总会散心的，等等。

虽然在新加坡办事很严，但对于本地人特别是有些地位、身份的人，他们还是比较相信和稍微宽松的，不然秋玲这样一定会判起码三个月的狱刑的。

秋玲被放了出来，但被勒令在一个星期内离境， 并且没收了担保人（Vincent）新币5000元的保证金。

我听了不禁感叹，道：这真是不幸中的万幸啊， Vincent真是个好人。。。那。。。你就要回国了？真舍不得啊。。。

秋玲微微一笑，脸上泛起了红晕，说：不要紧，我很快就会回来的。。。我答应Vincent的求婚了。

见我张大嘴，不可置信望着她，秋玲声音相当平静，悠悠的说：本来，我一直想我还年轻呢，对婚姻和未来伴侣有太多憧憬。。。也梦想自己有一个长相英俊的男朋友和老公。 我以前总是嫌Vincent长得丑又老，看不上他。但通过这次我才真正的知道Vincent这个人的心肠很好，是真正对我好的人。。。。

我一边听一边望着秋玲，心想：虽然她比我大很多，但因为单纯开朗反而一直觉得她比我小似的，如今她这样静下来，却另有一番样貌，我向她眨眨眼，说： 你倒是真的一夜之间成熟了。

她脸红，道：是啊，我现在对Vincent印象完全改观了， 当他来接我时，我见到他一下子有了亲人的感觉。。。以前他把我当他的女神一样对待， 所以这次我便觉得没脸再面对他了，我回到家，便躲在屋里不出来。。。直到晚饭时间，Vincent来敲门并对我说他不介意所发生的一切， 并说其实应该怪他自己让我感到压力。。。并向我求婚，说他一定会去中国把我娶回来的。

我伸出手一把抱住了秋玲： 太好了， 我听了真为你高兴啊， 没想到这一桩坏事到有了个好收场。。。Vincent也真是应了那句“塞翁失马，焉知非福”，他终于得到你了。

（八）

当秋玲离开新加坡后，Vincent真的在三个月后，把她又带回来，并邀请我参加他们的婚礼。

婚礼上客人很多， Vincent公司里的人几乎全参加了，他们都不停地赞叹新娘子漂亮。 秋玲那天容光焕发，身穿象白色锻子婚礼服，发髻上别着胡姬花，印得她像一朵娇艳的花一般美丽。当他俩来到我这里敬酒时，我由衷的祝福他们白头偕老。新郎乐呵呵的搂着他的新娘接受祝福，有点醉意。

那以后，我又见过他们一次。 在他们的女儿满月后，我去她家看望，女儿很幸运长得像秋玲，没有像爸爸。 秋玲对Vincent很好，很温柔，特别有了那段“不好“的经历后，心里也有一些愧疚，也不再对Vincent“高高在上”了。

趁着小宝宝睡觉，我和秋玲在屋里聊天。我悄悄的问秋玲，Vincent是否在婚后俩人出现争执时提到她那段经历。秋玲告诉我：他从来都没提过，就象这事不曾发生过一样。

这时，小宝宝哭了起来，Vincent赶过来帮忙，并充满着爱意和柔情的看着忙着给baby换尿布的妻子。。。

我望着这一幅乐意融融，温馨的家庭画，心想： 是啊，每个年轻姑娘都梦想自己能找到既英俊潇洒又爱自己的另一半， 有这想法也是很正常的， 当然两全其美是最好的了。 如不然的话，找个心地善良和真心待你的人也一样会得到真正的幸福。



（完）
