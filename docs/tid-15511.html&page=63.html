环球时报：应激活《反分裂国家法》，缺席审判，全球通缉甚至实际抓捕“台独头号人物”

2016-12-25 23:17:00环球时报

美国总统奥巴马23日签署“2017财年国防授权法案”，该法案首度允许未来美国国防部助理部长以上级别文官及现役军官赴台交流，这被认为是继1979年美台“断交”以来，双方提升军事交流级别的一次突破。
　　
奥巴马抢在卸任之前签署了国会参众两院本月初通过的法案，把特朗普可能会做的事提前做了。不仅中美关系出现新的难题，台海局势也将增加新的不确定性。
　　
台湾方面千万别以为他们将因此得到华盛顿的额外保护，台湾从此“更加安全”。台湾已经被大陆的力量环抱，美军失去了主导台海局势的能力。台美加强军事交流更多具有的恐怕是政治意义，如果大陆下决心武力收复台湾，美军持什么态度已不是决定性的。

而且很重要的一点是，台美升级军事交流，肯定会削弱大陆方面和平解决台湾问题的信心，推动大陆认真考虑军事解决台湾问题的选择。自特朗普与蔡英文通电话以来，大陆社会关于武力收复台湾的探讨大量增多。这决非说气话，或者故意这样说，吓唬台湾，而是很多人确实认为加强军事斗争是台海的必然趋势。
　　
自1996年围绕台湾岛进行导弹试射以来，大陆这边再没有搞警告“台独”的军事行动。自民进党再次上台后，“台独”重新活跃，华盛顿方面也出现利用“台独”作为对华博弈筹码的新动向，如何采取实际行动惩戒“台独”成为大陆不容回避的现实课题。
　　
圣多美和普林西比与台湾断交是“台独”遭到的第一个惩戒，但是这还远远不够。除了从外交上打击“台独”，大陆方面恐怕需要采取更能震慑“台独”的军事动作，打击民进党当局的嚣张气焰。
　　
我们在之前的社评中提出大陆军舰和军机越过台海中线，军机飞越台湾岛，这些建议迅速得到大陆舆论和学术界的响应、支持。我们相信，大陆的军事牌还有很多，针对台湾靠近大陆的离岛采取一些行动尤其容易。
　　
现在的问题是，台湾方面不太相信大陆真的会有军事动作，“台独”仍然认为主动权在他们一边。大陆有必要打破“台独”优哉游哉的感觉，采取一个切实震动全岛的行动，打乱蔡英文当局的施政计划，让民进党付出代价。
　　
我们还应激活《反分裂国家法》，由大陆司法机关收集“台独”头面人物及极端组织分裂国家的罪行，大陆法院对他们进行缺席审判，对他们实施全球通缉，并且实际抓捕他们当中的罪大恶极者，对“台独”施加精神压力。
　　
总之大陆有多得多的两岸对抗资本，既然“台独”不想好好过，就不妨和他们好好斗一把，搞得他们鸡犬不宁，彻底给他们一个教训。
　　

大陆不需担心美国方面会插手。蔡英文不承认“九二共识”，“川蔡通话”、台美强化军事交流，这些举动给两岸关系的稳定带来了严重伤害。特朗普政权如果要加强与中国地缘政治摩擦，那么把这些摩擦放到台海，对我们最有利。因为在台海，我们有更牢固的军事控制力和更强大的决心，中美在此一搏，我们获胜的把握最大。
　　
大陆方面只需祭出几招，蔡英文的支持率就将大跌，台湾经济也将蒙受损失。这不意味着战争将在台海爆发，但台海冲突的压力将是台湾不可承受之重。台海战与和的最后决定权只有大陆才有，无论台湾还是美国，都拿不走。
