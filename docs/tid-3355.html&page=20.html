想给自己刚出生的儿女买房投资的，就不需要了，你的儿女到2040年才30左右，现在城市年轻居民普遍结婚晚，到2040年房市都开始没落了，有钱还不如投资点别的（比如买成金条，名人字画等留给后代）。

其实我认为穷人投资任何市场或东西，都不如投资自己的身体，没钱了，就别瞎想，穷开心，把自己身体搞好，争取少吃药不打针，多活几年，多从国家社会身上割点肉，那就赚了，比如退休了，你才开2000/月，他是公务员开5000/月，但你比他多活20年，你不就赚了。

混沌，滞胀，死结
==========
国家统计局公布2012年9月份，全国居民消费价格总水平同比上涨1.9%，全国工业生产者出厂价格同比下降3.6%，降幅创2009年10月以来新低。就此，北京领先国际金融资讯公司首席经济师董先安对和讯网表示，2%左右的低通胀水平难以重现，未来5-6个季度内，只能怀念2%时代。今年4季度以及明年中国地产市场高概率出现预期继续紊乱，价格以及交易量呈现大波动，政策讨论仍缓慢回归财政改革与土地确权。

　　通胀风险并未降低

　　通过一个较长视角来考察通胀和增长前景，把短期宏观周期因素置于发展和转型框架之中，董先安判断，很可能在未来5-6个季度内，只能怀念刚刚过去的2012年3季度，因为7-9月2%左右的低通胀水平难以重现。

　　9月CPI环比比8月放缓，尽管PPI环比延续负增长，但水泥等上游价格反弹。与7月下旬、8月上旬不同，蔬菜等分项价格探底带动9月食品价格涨幅放缓，而蔬菜价格向下余地很小。并且，猪肉、水产品价格相继展开反弹，从流动性、猪粮比等因素综合判断，猪肉价格反弹或延续15个月以上。粗略估计2012年4季度-2013年全年CPI同比均值分别处于2-3%与3-4%区间。

　　从总需求和货币角度来考虑，9月贸易数据、官方以及HSBC/Markit PMI已显露内外需筑底迹象，2季度政府基础设施投资已显著回升，货币扩张也超出预期，3季度流行的通缩担心被证伪，结合地产市场动态，4季度通胀风险上升，宽松政策告一段落，避免滞后超调。

　　总之，中长期来看中国经济潜在增速处于下行通道中，这是由资源、海外市场、人力资本和亟待改革的社会经济制度共同内生决定，一个较低增长和较高通胀的新趋势正水落石出。这一时期，通胀与房价“一点就着”，政策偏重“防过热”，甚至不得不采用歧视性的房、车“限购”行政措施，以争取时间来改革根除旧模式痼疾。整体来看，未来3-5年，中国经济与政策反应大概率延续2010-2012年间已露冰山一角的新趋势。

　　地产市场短期呈现“量跌价涨”的不利情形

　　国庆假期购房人观望情绪较重，而地产商前3季销售目标完成较好，上调价格预期增强，地产市场短期呈现“量跌价涨”的不利情形。在土地、财政、人口、转型与发展框架下梳理真实利率、库存调整、政策预期等诸多因素，把握住政策之内生性，地产走势不难推理。类似地，我们很可能将怀念2012年4月份之前，因为未来3-5年一线城市地产市场难以再现低至八折的优惠力度。而房价每上涨一成，潜在风险就增加一成，且拖累人民币资本项目可兑换的真正实现。

　　延续一贯的逻辑框架，今年4季度以及明年中国地产市场高概率出现预期继续紊乱，价格停停涨涨，交易量大波动，政策讨论仍缓慢回归财政改革与土地确权。

　　积极信号是：朝野接近形成危机共识与改革必要性共识，但具体改革路径离达成共识距离较远。
