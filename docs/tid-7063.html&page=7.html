       其次，国家民族的前途不是小朋友过家家。都说战争是政治矛盾最后的解决手段，这个道理谁都懂，可我在这个帖子里真没看到几个人懂。快意恩仇是武林高手的境界，习李不是那样的境界，我相信安倍也不是那样的境界。
       我们不当东郭先生，但政治上要以我为主，这也是毛爷爷传下来的精神财富之一。建议你们仔细琢磨一下187楼传达的所谓阴谋，不过他的阴谋从我以前表述的双簧变成了三簧，看来世界变化真快。
       打落牙和血吞，这也是一种境界。在这个地球上，你比别人更好地活着才是最大的道理。如果人都毁灭了，理想自然也就毁灭了。今天我写了个日志，发表了一点对拉登的看法。他跟毛爷爷相比，就差很大一个境界。毛爷爷同样是个伟大的革命理想主义者，但是在实现他的革命理想的过程中，一样体现着他的超现实主义精神。没有这样的超现实主义精神，我们就无法理解72年的中美接触，就无法理解72年的中日建交。以我们目前的眼光来看，我们在中日建交的过程中诚然是有一些瑕疵的，以至于成为今天某些人攻击的借口。对于这样的攻击，我轻蔑地回应一句：他们那些人是站着说话不腰疼！
