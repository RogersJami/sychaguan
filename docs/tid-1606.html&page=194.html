“主张让消费者继续安心吃问题奶粉（反正渔樵山人自己不吃，别人的孩子死不完），直到建好产业链为止。” ...

食品安全问题的发生，大概有这么几个原因：

1、盛世下的浮华心态，是其土壤。
典型案例是“楚王好细腰，宫中多饿死”。程度差点的，“以肥为美”。
在物资匮乏或社会不稳或其他情况下，理性消费可以用脚投票，不行了还能用拳头、枪炮等暴力来解决，这个问题不会太大。毕竟，生死问题大过健康与否。极端了，还有引鸩止渴呢。（插科打诨下，曹操打寿春时候导演双簧杀粮官，放大了也可以当作食品安全问题。）
浮华时代，觉得自己腰包鼓的人并不习惯也不乐意理性消费，所以也最适合概念传销。例子很多，比如万能绿豆之类的。需要牛奶，还是不需要牛奶，这是个问题？！总体上，概念传销又赢了——其实，我们老祖宗并不太常饮食奶及奶制品。（更别提什么牛奶与癌症的零散报道了 http://food.9939.com/sjys/djys/2010/0114/859947.shtml              http://old.jfdaily.com/gb/node2/node17/node38/node32705/node32716/userobject1ai486241.html）

2、商业化是食品安全问题的根。
本来商业极大促进了人类社会的进步，可惜当“以满足人类需求的基本商业”进化到“以物化的人作为对象来创造需求的高级商业社会”，所谓的“无奸不商，无商不奸”（说好听点就是一个信息不对称）与商人逐利就开始残酷的展示商业作为魔鬼的一面。这是辩证法，商业于人类社会也不例外。
“以物化的人作为对象来创造需求的高级商业社会”，这是生造的说法（如有雷同，纯属巧合）。首先是把人物化，即将人视为改造的物体对象，从而大肆开发所谓的商业价值，最典型的两个行业就是减肥与整容。其次还是概念营销，创造非必需的需求，典型如比如用房产权混淆居住权而制造买房需求，以及奢侈品消费。
这个估计得靠共产主义来消灭吧，不然食品安全问题没有根绝的一天，只能是限制在越来越小的危害级别内。

3、资本主义与工业化的兴起为食品安全问题的突兀提供了工具，其催化与放大作用“功”不可没。
对资本主义的控诉，马克思的那段描述现在还是触目惊心。在一切向钱看的号召下，有钱能使磨推鬼啊。而按照辩证法，工业化带来的危害也会被催化乃至放大。大量的标准化产品，直接将食品安全上升到公共安全的层面，让每一个个体都感受到实实在在威胁。所以三鹿、蒙牛等是以毒害一代人的架势展开的，而疯牛病也成了流行的传染病。

有几个大的约束，目前处理
喝中国牛奶的人相对少了。牧民又不是一月不卖牛奶就饿死，即便重新签合同。只 ...

不要说几个月，养牛的连一个月都熬不起，这个在美国早期是有人做过测算的。没有控制，奶价的波动可以在3 －5倍之间波动，而且因为产能和需求之间的变化受养牛成长期的限制，所有的市场反馈都会以产能的崩溃反映出来。所以，在有反垄断法的美国才会有一个有权掌握养殖到奶价这样一个产业链的奶业协会。而且这个协会是公然的垄断。没有奶业协会的认可，没有一家奶制品企业可以得到奶。奶业协会认定谁有资格养牛，养多少，谁有资格送奶，奶卖多少钱，买奶的至少要买多少，买谁的。在这个体系下，养奶牛的利益得到充分的保证。所以，基本上没有奶牛场主会笨到做什么事情破坏奶牛协会的规定。当然，在所有其它的保证以外，奶的质量也得到了保障。

看看中国养牛户的现状：养牛的成本
在一头牛一年只能带来千把元收入的情况下，这些人拿什么来熬着。在走投无路的情况下被迫签合同这些人的利益更是处于无人保护的状态。这种在最低收入线上的生产是无法保证质量的。反正也是在低水平上挣扎，受苦受累的养牛又没给这些人带来高于其它同样受苦受累的利益，这些人只有钻空子的动机，没有守规矩必要。

除了养牛以外，依附在奶业企业上的另一重要环节是收购系统。以一头牛年产出产4000公斤来算，最小的正式罐车大约是二吨半的，差不多是二百头左右的产奶牛才能够一车。一天挤两次奶，就是四百头奶牛的产能才能够一车。鲜货冷链，对于没有冷藏车的散户来说，挤出的奶送到收购站最多不能超过一个小时。以蒙牛现有3700多个奶站，9000多吨的日收购量来说，牵涉的物流问题，特别是冷链物流，一旦受损，重建是一个非常困难，并且要花大量时间来完善的工作。不要说破坏后重建，只是接收这样一个系统并投入运作没有几个月都是做不成的。

还有就是这样一个系统的检测问题。三聚氰胺检测方法
说到检测就不能不说浓度。以咱们愤怒斗士的资料，蒙牛的奶粉是70毫克每公斤。奶粉比奶是八比一，所以奶中的含量是差不多10毫克每公斤。也就是10ppm.。类比一下，当年著名的致癌物苯在水中的允许含量是5ppm。当然现代检测技术发展了二十年以后，现存标准是5ppb。学过一些化学分析的人都知道，当浓度到了ppm的级别上，检测是一件技术含量要求非常高的工作。通常也是一件很耗时的事。对于控制现有这种收奶方式的质量来说，在收奶站配备这种技术能力是不现实的。不仅仅是钱的问题，鲜奶也不允许这种时间上的消耗。所以一定是一种粗检加精检的方式才是可行的。以当时的情况来说，在奶中添加三聚本身就是一种钻空子的行为，检测方法有漏洞是必要条件。所以大家纷给中招也没有什么奇怪的。这种事情，跺脚，骂娘，杀几个，关几个都没用。空子之所以是空子就是因为人家想到了你没想到。接受教训的方法一方面是严格已有的检测制度，堵上这个空子。另一方面是未雨绸缪，自己养一批专家自己找自己的空子，争取先找到下一个可能的三聚，事先做出预案。

最后是那个著名的日本羽。第一，这个企业没死。第二，这个企业是乳类加工业。在日本，乳业和养牛是没关系的。企业关门是不会影响到牛奶的产能的。关几个厂子不过是另外一些厂子多开几个小时就是了。反古加工能力有很大的潜力。中国之所以在这次三聚风波中这么狼狈，就是因为养牛业依附在乳类加工业上。
