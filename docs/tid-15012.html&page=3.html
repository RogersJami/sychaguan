1958年中国撤军之前，中苏想联合把金日成搞下去，金日成祭出了“主体思想”，从批判“事大主义”入手，彻底争取了朝鲜本土派和民族主义派，使得延安派和亲苏派被彻底清肃。

在这之前，延安派的势力最大（把持党政军关键位置），其次亲苏派，金日成的影响力很小。

这段历史，是迷雾中的迷雾。中国最终没能像袁世凯控制闵妃一样控制住金氏政权，相反，不得不妥协，驻朝的庞大军队撤回国内，延安派一生无法返回朝鲜，老死中国。虽然之后几次谈判，中方曾主张让延安派部分人士回到朝鲜，也劝说金日成向TG学习，不要搞肉体消灭，不要搞清一色，应该容许不同意见的人物存在，给延安派余脉留条生路，要给出路（这理由，好高大上，其实满满的私货;P），但最后都因各种原因需要金日成合作，只好作罢。

这一役之后，中国领导层应该非常强烈地感受到了朝鲜民族自立自主自强的不屈之心，朝鲜再也不可能回到过去甘当“大清属国”的附庸地位了。

今天朝鲜的种种表现，应该不会太出乎中国领导层的预期。

所以不要拿普通的内部斗争的常识去看朝鲜党内斗争。人家那真正是，胜者生，败者亡。所以不是说他宁愿和苏联好，而是第一苏派的人不影响他地位，第二，人家大腿粗。但也不能说朝鲜一直抱苏联大腿，中苏论战以及之前的会议上，朝鲜都是支持中国的。当然，你要说白头山问题，我觉得，如果属实，那么是他们在支持我们之后，要得一个回报，而不是因为那个才支持我们。要知道，对于国家统治者来说，让国家顺利快速实现工业化比领土更加重要。

2。 摆脱中华文化，中国影响力，基本上每一个从中华文化圈独立出去的都这么干。哪个不这么干才奇怪。

3。对朝鲜来说，现在面临最大问题是生存，不是发展。而核武器基本是他能生存的最廉价的手段。 这里的人应该还记得葡萄提过，当年搞核武器，只有主席一个人赞成，其他人都反对，最后主席胜利的原因就是大家计算发现，核武器是能对抗美国的最廉价手段。所以朝鲜发展核武器是他自身发展的必然需要，不可能放弃，除非他准备投降。

4。对中国来说，朝鲜就和巴基斯坦一样，其实都不是绝对受我们控制的国家，他们对我们的意义只有一个，缓冲区，不让MD大部队安安稳稳的在中国边境呆着。这就是我们必然要支持朝鲜的原因。内部有一些问题，那是肯定的，一致对外就可以了。所谓巴铁一样是和MD勾搭

当然，朝鲜脱离中国投入MD怀抱不是不可能，真要那样了，形势会比现在更加严峻，所以说，朝鲜现在越跳得高，对中国其实也就越好。
