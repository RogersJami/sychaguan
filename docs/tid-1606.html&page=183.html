我的智力发育最快的时期是在3岁左右，在2岁的时候父母读书，开始将文字和语言的意义联系在一起，汉字大部分是自己学的，父母读书就将方块字自动转化为汉字。我之后的十几年完全就是自己把自己的智商水准往下拉。这个往下拉的过程到了高考复习那年达到了高峰。在狂学猛用功一段时间之后身体出现故障，发烧，低烧不退，神经衰弱。后来一位农村老中医给我调整了一下，从天天发烧的状态到神经衰弱用功不能超过半个小时的状态，当时的感觉是几乎自己要废掉了。但是在这种状态下参加高考，又超常发挥了一把，然后进了一个垫底的985的一个不算好的专业。但是这个专业其实是很对我胃口的，mm你该理解。

大学头两年，学校考试很少不提前交卷的，无他，神经衰弱未完全痊愈。一旦用功时间长了就会觉得胃里有呕吐感，同时大脑有被毛巾拧着的感觉。现在偶尔也会有，这个强烈打击我的信心和信念。
