葡萄的务虚3一直不写

1.关于永生

永生是指思维意识的永续存在。
物质是第一性的，意识是第二性的，人的思维意识不能脱离大脑而存在。一切灵魂、转生、天堂都是迷信。
意识是一个脑细胞网络中的电信号震荡，是一种量子化的动态，因此无法转移到电脑中，量子化的存在不可复制。
如果用人脑细胞培养出一个人工大脑，这是一个黑箱，无法判定这个AI是否有自我意识，还是只是一个超大的应答题库。
人脑脱离人体后的长久持续存在，或者人体寿命的极大延长，涉及到的医学知识太多，不是百年内能解决的。
人体内的各种精妙平衡太多，在百万年进化中，这个身体只是用来活30-50年的，现在要用来活几百年，太过艰难了。

假设永生/超长寿命可行，社会会怎样演化？
要看永生是基于基因改造，还是基于某些资源的占有（如医疗资源）。
若是基于基因改造，在计划生育体制下，经过三四代人，就全社会普及了。这和建国以来把人均寿命从30多岁提高到70多岁，没有本质区别。
只有基于资源占有，使得少数人的寿命极长而大部份人寿命正常，才可能会有社会阶级问题。

一种是如彭祖、张三丰这样隔绝人世，不让世人知道，从而不会被当权者抓去切片研究。
一种是如某个活了80多岁的古埃及法老，以全社会资源供养一人。不过现在民智已开，独夫如穆巴拉克/金三胖也需要一群支持者。
沙俄依赖20万贵族阶级去统治一亿多民众，技术进步使得被统治阶级对统治阶级的人数比值可以进一步加大。总资源的多少，和被统治阶级的反抗能力，决定了统治阶级会释放多少资源出来让中层和底层分润。
另一个问题是统治阶级内部的矛盾，如奥林匹克诸神，一代代地子弑父，是永生族群的必然结局。

2. 大数据下的社会资源配置

社会资源配置，一直有计划经济和市场经济两种机制。相互间的边界，就是科斯所说的，由交易成本/信息传递失真决定的，企业的边界。
IT几十年的大发展，使得企业内的管理变得扁平，计划体制的边界扩张。
大数据会进一步地扩张计划体制的边界。之前中苏计划经济体制的失败，有信息收集的困难，有信息传递的失真，于是价格信息被持续地扭曲，直到整个体系崩溃。但是今天我们看到市场体制下有着另一种的价格持续扭曲，另一种体系崩溃。
大数据是加强了计划体制的能力。虽然依然没有回归到计划经济体制下的线性，但是对于社会和市场需求的信息把握，要确实得多。
Big Brother式的社会管理，也是要依赖人来做的。即使监控能象NSA那样自动化，执行的暴力机构也是要依赖人来做的。不管统治阶级在科技进步地帮助下能缩到多小的人数，一小撮人是不可能实现对帝国的统治的，特别是大国家主义下对社会的全面管理。统治阶级的人数越少，对于暴力机构的失控的机率也就越大，最后就不免象日本的武士阶级取代天皇与公卿一样。

3. 反腐

这次反腐，虽然真正倒台的只有康师傅的派系和一群辽宁将军，但是江曾李贺等一只接一只大老虎的影子露出来，是出乎我的意料的。
以前有人说过，只有在经济形势很糟糕时，TG才会认真反腐，把free rider清理掉减轻负担。这是不是说明经济形势其实很糟？
蔡小心多次点名批评386和影帝。放在“开国功臣子弟 vs. 走资派/改开既得利益派”的格局框架下，说明双方的对立相当尖锐。
尤其此次海军阅舰式被取消，蔡说之前邓公出访时，华主席宣布阅舰式，邓回国后以总参谋长身份取消了。先例既在，说明维尼熊遇到的阻力极大。

4. 乌克兰

本来俄国有一个乌克兰做盟友，现在只有半个了，即使对克里米亚的控制由台下变成台上，也是亏损。
从乌新政府的表现看，和叙利亚反对派的表现一样烂，完全没有执政能力。区别大概是乌新政府有西半乌克兰的民意支持，叙反对派只能指望西方援军。
