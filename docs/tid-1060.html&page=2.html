报告，我07年2月去印度旅游过，来回共9天。印象至今还是比较好的，尤其是饮食这块，实在是太好吃了。我们一行8个亲友团成员，平均体重增加6斤是起码的，我自己就增加8斤。去之前我们有人带了方便面的，但结果一点也没吃。可能跟我们跟的是豪华团有关系（平均每人给旅游社就近2万），一路包车，吃住主要在四五星级酒店，坐过一次火车，是卧铺包厢，在境内坐过一次私人飞机。我们这批人，都是吃货，但一致对那次印度行的食物赞不绝口。至今怀念。

恒河去过两次，早晚各一。比传说中的干净多了，恒河日出是很美的。恒河边焚烧死人的那段也去走过，夜里和白天各一，弥漫着蛋白质燃烧后的臭气。

但印度真的是两极分化很严重的。上流社会的生活的确很不错。

不过首都新德里机场出入，真的速度很慢。返回的时候我们排队4个小时，还差点误机。
路上牛也在走，各种交通工具都有，速度极快。车没有后视镜。但一次没遇到堵车或车祸。真奇怪。车技都很好。曾亲眼看到群众从没有门且到站不停的公交车上跳下来的壮观景象……
