产业升级对技工的要求非常高，低端加工人才顶不上去，会造成摩擦性失业。已经重视对技工的培养，一个技校生每个学期补贴600元学费，尽管少，聊胜于无吧。

问题出在技工和大学生培养上不接地气，理论和应用不承接，应付实习。

绝大部分技校实习就是生产流水线上的操作工，和普通工人干一样的活，工厂怕实习生“捣乱”，影响生产，不敢让实习生有机会参与技术方面的活。而大学生不愿意干“工人的活”，“万般皆下品，唯有读书高”的官本位思想作怪。我大学工程方面的老师，大多在学生期间参与从苏联引进的156项工程的技改。

现在是把以前的中专转成职业学校，以后可能会把一些普通大学转成工程技术方面的“技校”，在学制中强制性拉长实习期，比如一年，国家按人头给工厂一部分补贴，工厂每录用一个技校生，返还几倍的补贴。技工是一个工厂最宝贵的财富，工作中谁都不买账，就买高级技工的帐，走一个少一个。
