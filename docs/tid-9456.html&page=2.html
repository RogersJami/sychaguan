玩钉耙的悟空 发表于 2014-2-6 20:39:08


宋兄不易，收款最难最痛苦。

分级现存的账款和账期，对不同的人、不同的账期、不同的金额，采用不同的方式回款，做一个计划，有理有据，该软则软，该硬则硬。

出货前，摸一下对方的商业信誉和为人，分级，按不同的等级要求对应的付款方式，比如有些人，出货之时除了部分现款，要求写下收据和具体的还款时间；有些情况，斩件供货，基本收到上一批货款之后再发新货。

讲人情，但生意是生意，储备粮草过冬，已经成为一种共识。

宋兵丙 发表于 2014-2-7 21:24:15

三，高中关系最好的同学！


        这位同学我认为很能干，虽然现在有困难以后会好起来的。

        还是从前年说起，大概在5月份给我电话，说厂里资金比较紧，想借点资金周转下，待货款收来还我，估计最多一个月。我说和老婆商量下，明天给答复。结果老婆不给借，说上次说好一个月还结果拖了这么长时间，信誉不好。那我就只能另想办法了，就瞒着老婆给在广东的前同事打电话，同事也很爽快，当天就把这笔资金汇过来（这位同事偶尔还在这里潜水，看到了先谢下！）

        时间很快过了5个月，到国庆时，我看同学怎么一点音讯都没有，就给他打电话。他说现在在宁波这几天回来就还我。我等了一个星期，没有音讯，又给电话，他说下午就转过来。等到晚上查了账户没有转，又再给电话，他说是叫朋友转的，看来没转，他明天自己转。


      这样又过了一天，晚上再查，又没有。再给电话，不接。过一个月后电话，不接。我知道他肯定出了啥事了。而同事那里我先还了一部分，剩余部分年前货款收好再给。


      其实这笔款不多，但这件事我很心痛，心痛的不是钱，而是同学之间纯真的友谊。真要有难处就直接告诉我，我即使帮不上大忙，至少让我知道最好同学的近况也好啊！人与人之间交往诚字最可贵。

      还好，就在去年的大年三十晚上，他给我发了个信息，说是今年资金好转就还我。看到这个信息很高兴，至少心里他还惦记着我这个同学。

宋兵丙 发表于 2014-2-7 21:43:41

四，初中关系做好的同学！


     这位同学是我初中的同桌，有20年的交情了。

     事情是这样去年4月左右，给我电话说急需资金，明天就要，不用一个月就还。我问了多少，就让他发个卡号过来。第二天一大早就转了过去。

      过一个月，没音讯。

      过两个月，没音讯。

      过三个月，没音讯。

      四个月后我实在忍不住了，就给个电话。我忍不住的是担心对方出了啥大事，按我对对方的了解，没啥大事应该会实时还我，要不然不会拖这么长时间。还有一个担心就是，没啥大事，就这么拖着不还也不给个答复，同学的友谊没了。


      电话通后我就问，你是不是出了啥事啊？他回答说是把钱压货里了，这段时间一直没给你电话真不好意思！我说：“你这小子，没事你也给个电话啊，我为你担心好几个月了，钱先不用还，你急就先用着，等我要的时候再给你电话。”

    一个是高中最好的同学，一个是初中最好的同学，几乎都一样，我都怀疑是不是我人品的问题了。现实生活中我朋友不多，一个不剩，就真成孤家寡人了:'(。

     年前给他电话说年底要还债，他说现在都压在货上，年底出不了货，只能过正月十五再给我。好吧看下十五后他是否会给我。
