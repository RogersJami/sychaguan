从最近络绎不断的迹象看，这轮“天然气”大棋的轮廓已经越来越清晰了。

http://i.guancha.cn/news/2017/12/14/20171214135630754.jpeg

中俄东线天然气管道建设加速将通往根本不缺气的魔都，这是要为成立刚一周年的上海石油天然气交易中心破解天然气东亚溢价和争取人民币定价权而拼命扩大内需啊！

因为2016年上海全市天然气总供应量73.8亿立方米，其中“西气东输”供气37.7亿立方米。而根据西气东输工程开工前与中国石油签订20年“照付不议”合同，“西气”到上海白鹤站的价格是1.32元/立方米。

其它缺额部分气源由“川气东送”（每年12亿立方米，部分补贴页岩气故价格较贵）、东海气田（供应有限）及LNG（液化天然气）等补充，其中最主要的是LNG。目前上海洋山港LNG码头已建成的一期项目年输转量约40亿立方米，可满足上海一半的天然气供给需求，正在建设二期工程在目前基础上再增加一倍，基本可以满足上海全市的用气需求。在2013年国际油气市场崩盘导致严重供大于求后，这几年的LNG到岸价格基本都维持在1元/立方米左右。

但由于近期国内大规模“煤改气”导致东亚LNG现货价格短期内接近翻番，目前LNG到岸价已近10美元/mmBtu，约合天然气价格为2.2元/立方米。如果加上LNG再气化成本的话，大概在2.4元/立方米左右。在之前川皇访华期间，中美签署的每年采购300~700万吨LNG（中石油项目未公布具体数字，根据签约金额大致推算）、约合45~100亿立方米天然气的长约，按照近一年美国LNG到岸价约为1.85元/立方米。

而如果使用俄气，根据之前中俄签署的购气大单估计（虽然一直没有透露具体价格，但大致可参照俄罗斯供应德国的天然气价格），在中国入境点的成本就要0.35美元/立方米，约合2.3元/立方米。这还没有包括天然气管道的“运费”等其它费用。

根据8月30日国家发改委最新核准的西气东输各段管线运费分别为0.14元、0.12元、0.24元/千立方米·公里。由于中俄天然气工程为新建管道，成本肯定比多年前建成的西气东输要高出不少。这里仅取西气东输最高运费0.24元/千立方米·公里计算，从黑河到上海3371公里管道运费就需0.8元/立方米左右，即俄气到上海接收站的净成本就要超过3.1元/立方米，是“西气东输”价格的2倍以上。接近三桶油在2013年LNG泡沫最高峰20美元/mmBtu时签的长约价格。

而目前含税、“最后一公里”建设管理等费用的上海居民天然气第一档价格从2014年9月1日起由2.50元调整为3.00元/立方米（非居民用户2014年后已多次调降），可以想见如果引入俄气后上海的天然气价格必将大幅上涨。

当然，以上都是建立在中俄天然气管道一定会修到上海的基础之上，但也不排除有可能是争取国际天然气以人民币计价故布的疑阵。

补个遗，如果要整合工人阶级，需要从社会认同或者说阶级需求着手。工人阶级的特点或者方法论是脚踏实地靠技能吃饭，然而后三十年的大开放，太多一夜暴富一飞冲天的奇迹让这样的方法论成为笑话，工人阶级这种顺民是不可能出头的已经成为一代共识，至少是改开这一代的共识。然而大环境变了：信息技术与人工智能会导致数据垄断，意味着增量套利机会越来越集中在少数人周围；前三十年的红利在后三十年已然耗尽，意味着能够一夜暴富的存量套利宝藏越来越少；轮回压力带来的不安定大环境，意味着积累无法达到质变（参考三个加强师的段子）的套利者迟早会被收割（或收编）以缓解矛盾。也就是说，后三十年最好走的路再走是要碰壁的，大多数人碰壁多了，社会认同迟早会反转，或者说绝望。然而事情向来两面，绝望之后总有希望，当大多数人发现自身离无用阶级不远而且反抗无用之时，就是求存寻路之时，而这条就是大多数人都有资格进去的工人阶级之路，走的人多了，也便成了路，或者说新时代的社会认同。接下来再说工人阶级的需求，技能依赖技术，而技术总在进步，这意味着工人阶级所走的路是一条永无止境的学习进化之路，如果停止进化，就会被淘汰，大多数程序猿的中年危机便是如此，既没有后三十年必须的关系网，又没有继续跟进新技术潮流的动力，到点儿了只能无路可走。危是机，忧虑也是需求，进化需求与港湾需求就是工人阶级最大的需求。进化需求对应不断学习进化，换句话说就是希望有一个能够随时接受教育的平台，程序猿好歹还有各种开源平台，而大多数技能岗位并没这个待遇，因为技术工人早就被现有的教育体制拒绝乃至鄙视。港湾需求对应能被治愈，换句话说，绝望来临时可以被鼓励被安慰被引导然后调整方向重新开始，跨国企业与军队中好歹还有心理医生乃至牧师，然而社会主义市场经济中的工会不过就是个花架子。想动员一个阶级，就得从这个阶级的需求出发，并且用社会认同度量，不然只能是口号，没有教育体制与工会组织等基础改革，人数再多也不会有凝聚力。

挺普大帝是个很小很短期的目标，只是出于维持一定时期内中俄战略伙伴的稳定角度考虑。

而且目前俄气只占中国进口天然气总量的不到1%（中国天然气进口依存度在1/3左右），即使到2030年前后俄罗斯输华天然气管道全部达到最高设计目标，估计也不会超过25%，比目前德国对俄气的依赖程度还低。而作为俄气的最大客户，俄罗斯给德国天然气价格在欧洲市场是属于中间偏低的。除了德俄之间的战略利益关系外，德国的一个制衡手段就是来自中东（主要是卡塔尔）的LNG气源。

但目前包括中日韩在内的东亚国家虽然消费了全球LNG出口量的近70%，却由于日韩天然气需求基本完全依靠进口LNG，因此在国际天然气市场长期存在着不合理的“东亚溢价”，同样来源的LNG价格几乎比欧洲贵50~100%（根据季节不同）。

而随着中国天然气市场需求的快速提升（从目前天然气在一次能源中占比6%，到2030年提升4倍到世界平均水平的24%），中国有望成为全世界最重要的天然气市场，来自中东、美国、澳洲、非洲的LNG与来自缅甸、中亚、俄罗斯的管道天然气之间都存在着竞争关系（目前LNG与管道天然气进口量约各占一半）。并且作为全球页岩气最大储藏国，维持一定的天然气价格有利于国内页岩气稳定开发与相关技术发展，同时也将限制进口天然气价格的上线。这都将加强中国在全球天然气市场的定价权，有望打破LNG市场长期的“东亚溢价”问题。

而随着中国国力与话语权的不断提升，最终必定会指向天然气的人民币定价权。而已经一周岁的上海石油天然气交易中心就是为了这个战略目标而设立。不过上涨的油气价格可能推升全球通胀进而迫使央行加速加息，面在美国进入加息缩表周期吸引美元回流，谁家的泡沫会先被戳破呢？
