1、航空发动机代差是一定的。我们以常规循环的WS10已经成熟的时候，对方已经从变循环开始冲击自适应循环发动机了。航空发动机我不想说任何专业术语细节，发动机看循环参数和循环模型就知道是不是代差。忙总说的50年我没看到，但是20年是乐观很多了。Y20我要特别说一句，这是个跟得上世界先进水平的运输机。设计之初是考虑了使用发动机引气的吹气襟翼的，可以大幅提高野战起落性能，最后因为发动机不行还是使用了常规襟翼方案。别说什么发动机重大突破，只要一看这些，看一下涡轮前总温一目了然。
中国目前是解决了第三代涡轮风扇引擎的问题，正在冲击总增压比20-25，燃烧室出口温度在1800K左右的第四代常规循环发动机，对标的就是F119。五常里我国发动机最快超过的国家是俄罗斯或者法国。制造抗衡RR或者CFMI商用高涵道比发动机，20年内没戏。随便举个例子，宽弦长编制叶片技术都不会，没有这个技术，你拿到FAA或者EASA第四阶段噪音标准都不行。此外我们的FADEC技术刚刚成熟，你再去看看湍达700的FADEC的调节精度，是不是代差
2、J20的气动是我国的突破，但不是世界第一，是我国从望尘莫及到望其项背的产物，很大的进步。这里面突破了几个研制四代机必须的技术，隐形技术，高压液压系统（没这个就别谈弹舱内发射空空导弹）。有这两个就足够了。可是还有很多是代差军事网站怎么不说，必须跨战区航程和光洁构型超音速巡航因发动机没达到，引擎的外场维护时间只需要20个最常用工具就可以完成（这些标准我国研制时候 基本就是对标美国标准）也不行。军迷网站不说吧 要我看这就是代差。不说了 再说惹祸
