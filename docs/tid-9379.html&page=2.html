楚国一直出不了好的董事长，搞不定国内，一代不如一代

楚国一直到国灭，国君都深受国内大贵族的影响，例如屈原的区姓也是当时四大贵族之一。
而从春申君的权力暴涨和旋灭，感觉是国君与大贵族的制衡的一个表现，春申君的权力无疑来自君王，也是很大的；但是覆灭的时候，却对贵族们没什么冲击，这么看来，贵族和国君的权力划分的边界是很明确的。
看楚国后面的国君，很像是生于深宫，长于妇人之手这些，因为庄王父亲弑父，熊掌难熟，之后的太子估计都会像唐朝一样，终唐一世，对太子提防的厉害，加上贵族政治的结果，弱的君王才符合他们的利益，所以太子既不能打造自己班底，对贵族有依重，所以楚之后的董事长越来越差。
