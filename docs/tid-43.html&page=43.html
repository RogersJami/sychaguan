华族时代1

在昆仑山滴最高峰，K2（乔戈里峰）峰上，在18000年前，就已经是一个很老的名胜古迹。在5万年前，那里还是地球的最高峰。那时候，就有人类在那里建设了最初的——娱乐设施。

到了华族帝国的1400年的时候，一种叫“乱流”的游戏，出现在这座高耸的山峰的封顶。这是最早的利用高度不同的重力差，引导穿越者进入华族帝国其他时间空间的区域。

这种游戏往往是一家人一同参与，所谓乱流，是指每个人感觉面前空气开始加速，但是每个人都不知道自己会被穿越到何时何地何处……结局是大家在经历共同的时间长度之后，再次回到当下的时空，不过地点不是这个起点，而是帝国的任何区域。

这种穿越的奇妙之处，是所有人的参与感，体验分离的紧张感，体验重聚的幸福感。还有因为这种经历得到的对幸福珍惜等等。

我国人体再生复原科学取得巨大进展

光明网9月16日讯（记者 李然）我国著名生命科学家徐荣祥在北京钓鱼台国宾馆举办人体再生复原科学成果展示会，展示了包括胃肠在内的各类组织器的原位再生复原成果。徐荣祥表示，这是再生复原科学在实际应用方面取得的又一巨大进展，今后将为该成果早日实现产业化而努力，为延续人类生命，造福人类健康服务。

美国著名抗衰老专家，南加州大学著名教授Dr.Shannon、中华全国新闻工作者协会党组书记翟惠生先生、国务院监事会主席寻寰中先生、著名心血管科学家，泰达国际心血管病医学院长刘晓程先生、中国中西医结合学会秘书长穆大伟先生等政商学界知名人士、外国驻华使节、各大跨国企业驻华代表、世界各大通讯社及各大媒体200余人应邀出席了发布会。

早在07年，徐荣祥及其团队就成功复制了人体206个组织器官，并取得包括中国在内的多国专利，在全世界引起巨大震动。徐荣祥希望此次通过公布胃肠原位再生成果展示，早日将科研成果转变成产业化，延缓人类的衰老问题，造福全人类。


在徐荣祥进行有图有真相的胃肠再生复原效果展示时，全场皆惊，热议纷纷。现场展示大量的胃肠再生实验效果对比图显示，经再生复原后的胃肠表面绒毛较再生复原前更显浓密、粗壮。徐荣祥说：“胃肠是人体的重要器官，只有胃肠功能正常，才能有效消化吸收各类营养物质，保障身体健康，延缓机体衰老速度，提高生命质量。”医学研究表明，30岁后胃黏膜开始出现萎缩，40岁后胃肠蠕动变慢，小肠黏膜开始老化。胃肠再生复原技术或将改变这一状况，为延缓人体机体衰老赢得时间。
