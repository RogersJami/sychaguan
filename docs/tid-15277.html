目前的电商模式能够持续吗？

电商，优势在送货上门，其实加上处理退货投诉之类的成本很高的。目前电商高歌猛进是在特定条件下的产物，未来能否持续，存疑。

电商生存的基础：
1.      资本市场长期超低利息以及对网络经济的狂热在长期低息的环境下，社会预期资本收益率降低，预期投资回收期变长。充裕的资本无处可去，大量投资涌入电商。而电商无一例外都在持续烧钱吸引客户。最早开始做电商的亚玛逊还在一直亏损，赢利遥遥无期。京东就更不用提了。淘宝是靠大量的假货得以生存。

电商亏损，意味着是投资人在出钱补贴消费者。实体店和电商根本不在一个公平的起跑线上。实体店亏一二年早就关门了。而亏了很多年的亚玛逊还在经营。

亏本销售，就是倾销，是不折不扣的“不正当竞争”行为。理应在“反不正当竞争法”下被处罚。
资本家不是做慈善，投资早晚是要赚钱的。早晚电商要把它的成本和投资人的利润加倍地加到商品的价格上。到那时电商对实体店还会有这么大的优势吗？

有人说到时消费者行成习惯了。放心，消费者改个习惯很容易。滴滴一取消补贴，马上用支付宝付车费的人的数量就下来了。等大潮退去，大量投资会撤出，很多电商关门是必然的。

2.      低油价低人工费低油价和中国特有的低人工成本，使得送货的成本变的很低廉。在发达国家电商为什么没兴起？人工成本高是一个重要因素。油价低是美帝打击俄毛的暂时需要而已。长期低油价对美帝会有很大的害处，特别是对金融市场，石油美元而言。所以未来几年，油价和人工成本肯定要上升。电商的送货成本会越来越贵。
核电，风电，太阳能，确实都有不少进展，但是革命性的突破还没有。化石能源，用一吨就少一吨。在沿海地区建大量的核电站，风险充分评估过吗？战争情况下呢？

3.      很高的居住密集程度中国东部沿海的居住密集程度，全世界少有地高。正是在这种居住密度下，才使得电商的壮大成为可能。电商不流行的欧美是没有这种居住密度的，而淘宝一再提的所谓开发农村市场也一直没有实现。

上层据说还在策划上亿人口的城市圈。真心希望这不是在电商网络公司的忽悠下的举动。上亿人口城市圈带来的污染问题，出行问题等等，基本无解。和太祖的大三线战略相比，战时危险性大大增加了，而且培养出无数的沉溺虚拟空间的宅男宅女。
请仔细客观地想一想，这么高的人口居住密度，真的是一件好事吗？请多为中华民族的子孙后代想想吧。

4.      电商免税的政策
电商一直免税。而实体店各种税费的负担很重。对电商最起码征一个5%-6%的营业税就够了。因为大量电商的毛利率非常低。
5. 现有法规的滞后发达国家“产品责任法”，“消费者权益法”非常健全。对于商家，任何一个小的错误都处罚很严的。国内的法规缺失，执行难客观上使广大销费者处于非常弱势的地位。这才使得电商得以野蛮生长。

当这些基础不再的时候，电商的优势会有所降低，未来必然是实体店和电商并存的格局。
