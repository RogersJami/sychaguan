会晒大桥建成通车 世界最大自由贸易区将实现无缝连接
      12月11日，老挝连接泰国的会晒大桥建成通车，意味着5年前就已通车的昆曼公路将实现全面贯通。今后，从昆明自驾到曼谷将只需要20个小时的时间。老挝会晒跨湄公河大桥项目是昆曼公路的重要组成部分，链接老挝波乔省会会晒与泰国清莱省清孔区，项目全长11.6公里，其中主体大桥长630米，宽14.7米，由中泰两国政府各出资50％分别建设老泰两侧工程。其中，老挝侧工程由中国政府无偿援建建设，整个大桥项目资金总额达4800多万美元，于2010年6月开工建设，现在已全部建成。

　　昆曼公路是中国西南地区经云南、老挝、到泰国的一条重要的国际陆路通道。是大湄公河区域经济发展的基础。昆曼公路的建成将有利于中国与东南亚国家的国际关系的进一步加强与改善，在国际和平与稳定方面，具有重要的意义和促进作用。同时对我国西南地区及东南半岛国家的经济贸易、旅游业的发展具有重大的促进作用。

　　昆曼公路起于昆明市，途经玉溪、思茅、西双版纳，从磨憨口岸进入老挝境内的磨丁口岸，经老挝的会晒口岸进入泰国清孔口岸，经清莱、清迈最后抵达首都曼谷，于2008年通车。公路全长1807公里，其中，中国境内688公里，老挝境内229公里，泰国境内约890公里。昆曼公路经过泰国与马来西亚的国际高速公路相接，最终抵达中南半岛最南端的新加坡，是中国进入中南半岛最便捷的交通走廊。
