白沙 发表于 2015-8-23 11:23:39

对于一些朋友的不断质疑，说点出格的话。

首先我并没有否认世家或山头的存在，也不否认极个别所谓世家有意识地搞一些人身依附的实际举措，比如文先生描述的那种情况。
但是，我们看问题要看全面，看发展趋势。

其次，到底哪些可以算世家？楼主的表述里面，实际上是把改开以来党内统治阶层分成了两大流派：传统的红色世家和改开中的精英阶层。按照某种推理逻辑，依附于传统世家的精英阶层，或许可以保留既得利益，在当前的变革中生存下来。而与传统世家发生重大利益冲突的精英阶层，则可能成为这波变革的牺牲品。目前的局势发展似乎从侧面印证了这个逻辑。

但是，这个逻辑存在逻辑自洽的问题。我们不去考察那些贫民子弟奋斗出来最后垮台的那些案例，比如何家成，比如郭老虎徐老虎等。我们考察一些更敏感的案例，比如处于风口浪尖的庆亲王，比如目前非常尴尬的岭南王家族（具体参考广东省公安厅那个拿下的武警少将，看看他什么身份），比如已经倒台的薄家。那么这些考察对象算不算世家？

我们假定另外一种逻辑，通过一些家族打击另一些家族，进而实现利益的交换。但是，这同样存在致命的问题，就是今天的胜利者不能保证明天还是胜利者。这样演变的后果是，几十年后可能就没有世家了，如果按照老百姓通常表述的500家的说法。斗争的结果只能是世家越来越少，而世家的生成是需要时间积累的。有人可能说，倒下的世家可能重新崛起，就象文革期间那样。如果是这样，我只能说局面会更好看。

出格的话就说这么多，其实我想上面也是很困惑的，他们对前景的展望并不见得比老百姓更清晰明确，要不然就不会有摸着石头过河的说法。

我们继续看看，不要急着下结论。
