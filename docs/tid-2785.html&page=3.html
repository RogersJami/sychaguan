烂文一篇，实在没法看下去。捡看到的随便说三点：

1、前边有朋友已经提到了，一吨800美元，远高于WTI价格。其实WTI是benchmark price，多数原油卖不到这个价格，这个差价叫differential，最高的时候，这个differential超过30美元。至于石油部长报出到岸价，连保险都包，更是笑话。西方人报出的价格一般是bare bone，比如说，你买辆车，他给你的价连空调、收音机都不含。更不要说运费、保险这些不在他们控制范围的东西。再说到岸是到那个岸？是茂名还是秦皇岛？

2、私企搞炼油那是笑话。举个例子，90年代北京市花100多亿搞了个小乙烯，建成后开车一直开不起来，最后在97年罐区爆炸，最后还是燕山石化收拾烂摊子。爆炸原因是因为一个法兰的材料问题，该用不锈钢的用了碳钢(设计问题)导致腐蚀泄露；施工、操作也有问题，就不细说了。北京市都玩不转的东西，民企玩得转？重化工这种玩意，装置越大，效率越高，污染越小；现在上马的项目，都是1000万吨炼油+80万吨乙烯，这么大规模、复杂的装置，民企玩得转？

3、按文中所述，吉林油田一度有41家民企，产量100万吨。大家想想是什么概念。区区100万吨的年产量，围着41家小企业，后果是什么？我都不用想：破坏性开发、掠夺性开采、高能耗、低效率；还有一个更要命的：为了节省投资，把环保、安全设备能省的全省了。国外的小公司很多也是这样的---不过国外小公司形成和背景比较复杂，不细说了。
