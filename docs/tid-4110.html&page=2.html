DF31 发表于 2013-1-31 11:54:01

　　人民网1月31日电 据中评社报道，全球华人保钓联盟会长黄定为(原名黄锡麟)30日控告日本海上保安厅指挥官后，接受中评社访问时透露，1月24日保钓行动，日舰以水柱直接攻击保钓船，“海巡署”派小型船过来帮忙挡，反而双双受损，他深信，日舰是因大陆海监船到场后，才停止攻击。

　　资深保钓人士黄定为原名黄锡麟，为了宣示保钓一定有作为才改名。他说，台湾“海巡署”立场是“不挑衅、不回避、不冲突”，日舰则是存心挑衅，也不回避，硬拗，而且态度愈来愈强硬，因为像去年7月他领军的那次保钓行动，是“偷偷摸摸”去钓鱼岛，日本猝不及防，派了3艘船，从头到尾都没有喷水，“因为他们不晓得我们去！”但是这次行动，日方是直接喷渔船，不一样。

　　黄定为表示，以前保钓船一出海到钓鱼岛海域，“海巡署”都会派船在左、右两翼，开一条水道给他们走，这次不一样，是保钓船单刀直入在前面，海巡船舰在后方0.3海浬尾随，反而是日本海上保安厅的船舰在左右两翼，不断冲撞他们，直到要攻击了，“海巡署”才过来广播说：“他们要喷水了！”

　　黄定为回忆，结果声音一停“啪！”一下子，直径像路灯杆一般粗的水柱就过来了，如果知道他们要喷水，“海巡署”应该派大一点的船过来挡才对，因为和星舰船比较高，结果过来的是跟“全家福号”差不多100吨大的PP10018船，结果两艘船都被打中。“海巡署”应该派2000吨的“和星舰”在前面，他们就不敢打了，结果派小型船舰来，而且是在保钓船被打得体无完肤的时候才过来。

　　黄定为认为，当时大陆海监船具有安定的力量，日舰连续攻击完之后，他爬到船板最高的地方看损失情形，瞥见左前方，大陆海监船来了，然后大陆海监船到右翼回转过来，停在那个地方。黄定为强调，是大陆海监船有过来，日本巡防舰才停止攻击“全家福号”。
