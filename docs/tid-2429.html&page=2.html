报道1:
日本大米在中国市场一向被视为高级农产品的代表，而日本民间也一向对自己的国产大米自豪，在食材上十分“排外”。而现在，中国大米在日本市场销售火暴，并且价格已逼近以高价闻名的日本国产大米。

据日本新闻网17日报道，在日本福井县一些大型超市中，中国大米已经成为货架上的重要品种，受到部分顾客追捧，价格也水涨船高。10公斤装的中国大米已卖到2580日元（约合215元人民币），价格仅比日本国产米便宜10%左右。该超市负责人在接受采访时说：“日本消费者对于中国大米没有什么抵触，销售情况非常好。”而日本最大连锁超市之一的西友超市也从今年3月开始在旗下主要超市出售中国大米，销量也十分不错。

事实上，中国大米在日本热销的主要原因是去年日本大地震和核泄漏导致国产米供应紧张，价格不断上升，加上消费者对来自日本大米主产区东北地区的大米心存疑虑，担心核辐射会影响大米的食用安全，因此中国大米作为替代品而走俏。受到核辐射影响严重的日本东北角，以福岛为首的地区本就是日本产米区，对于辐射的忧虑极大打击了日本大米的市场。同时，供货商和超市认为，由于历史原因，中国大米特别是中国东北地区的大米品种与日本大米类似，都属于短粒品种，味道也无太大差别，因此很容易被日本消费者接受。


日本目前国内的大米消费量每年为800万吨，为了保护本国农民，日本政府目前规定的外国大米的进口额度为77万吨，约为总消费量的10%。考虑到品质和成本因素，日本超市和快餐店希望扩大中国大米的进口量，但日本农林水产省方面表示，短期内扩大中国大米进口“比较为难”。

而日本民众不敢吃的远不止大米。

事实上，自从核辐射事件发生以来，日本东北部农产品的销售出现严重问题。东京本身属于关东地区，其蔬菜、水果等农产品本多来自于关东、东北地区，但核辐射事件后越来越多东京人选用南日本甚或国外的农产品。日本的超市和蔬菜店则开始更加注重标明原产地，因为这会很大程度上影响消费者的选购。

日本是水产品消费大国。但今年4月，东京电力公司向海中排放了1万多吨低浓度放射性污水，污水中放射性铯的含量达840亿贝克勒尔。目前，日本福岛、宫城附近海域几乎每天都发现水产品放射性物质超标事例。很多日本人开始减少水产品消费，或者只购买南部海域的水产品。

除了水产品，日本人对自来水也充满忧虑。东京等13个都县自来水中检测出放射性物质，很多家庭已经大量囤积矿泉水、纯净水，更不会像以前那样直接将自来水当做饮用水。

一度以安全可靠著称的日本奶粉也出现了问题。去年12月6日，日本乳业巨头明治奶粉公司宣布，该公司制造的罐装“明治STEP”奶粉中被检出含有活度达到每公斤30.8贝克勒尔的放射性铯。这是福岛核事故以来，日本奶粉中首次检出放射性物质。此次检出放射性物质的奶粉是3月14日至20日期间在崎玉县春日部市的工厂生产的，明治公司决定对40万罐奶粉无偿换货。

东日本大地震和福岛核辐射带来的问题正在慢慢发酵，诸如电力、食品安全这样的问题短期内不会得到轻松解决，日本的灾后重建任重道远，而其中最为困难的是如何重塑日本民众的信任。
