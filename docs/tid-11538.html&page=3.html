叻沙laksa 是娘惹菜。 当年，随郑和下西洋的众多随从，在南洋一带生活下来，和当地人通婚，生下的小孩，女孩叫“娘惹”，男孩叫“峇峇（Baba）”，

以前女孩从小到嫁人都要会烧饭的，所以娘惹菜是中国菜中加入和马来特色，逐渐形成了一种特殊菜肴——娘惹菜，其中最著名的是叻沙（Laksa）和娘惹糕点（我在【房东记】小文中有提到）。

我更喜欢新加坡加东风味的叻沙，汤头是以咖哩汤汁混合椰浆和香料熬制而成，口感浓郁，色香味俱全，配料可以自由选择，但基本的要有米粉，油豆腐，鱼蛋（或鱼饼）或海鲜（虾等）
