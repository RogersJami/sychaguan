Judychan 发表于 2014-5-4 13:42:35

一季度总结（1）：

　　1）央行、保监会、银监会在采取措施去杠杆，但没能阻止社会整体债务继续扩大 ；
　　2）银行针对私营部门在收缩信贷，但对公部门贷款还在增加；风险加速向表外转移，各大银行中间业务收益猛增70%以上，
　　3）金融系统信用债增1820亿，利率债增加率债2600亿；以上没有考虑影子银行的其他形式负债加；


一季度总结（2）利率：
　　总体特征：短期货币市场宽松，长期货币市场利率吃紧；
　　原因分析：
　　1）银行出于风险考虑，改变原先长短错配情况，因此短期货币市场宽松；
　　2）市场出于风险考虑，不愿持有长期债，因端长期端货币吃紧；
　　3）大量债务无法缩减，只能延期，因此长期端货币需求大；

　　结论：中国处于衰退早期，迟一点长期端债务无以为继，长期利率也开始下降，并且下降幅度远远快于短期利率，表示衰退第二阶段，中国正加速进入探底；

　　如果不注水，两阶段的转换时间估计在9月之后；届时一级市场房产会下跌；
　　如果注水，两阶段的转的时间会拖到明年美国加息后。


毛国庆的讲话，分析的人很多，我注意到三点：

　　1. 很多大型房地产公司欠了很多外债--如果操作不当，房地产可能是做空中国的一个支点；
　　2. 北京3.7万的房子，卖2.5万很好卖--北京五环外的房子，如果降价30%就很好卖了，北京房地产市场支撑位一目了然；
　　3. 北京房子的订价权掌握在政府手里，如果价高会被禁止入市；同理如果价低，会被罚款。--北京的房地产绝对是一个庄家市场，北京市政府就是庄家。


一些推论：
　　1. 按照市中心与效区降价比经验公式，假设北京五环外未来降价空间约为30%，那么三环内降价空间约为10%。整个北京房市平均降价空间约为15%。
　　2. 从（1）中进一步推论，四个一线城市平均降价空间不会超过20%；
　　3.按照比价关系，三，四线城市降价空间在50%左右；
　　4. 按照比价关系，二线城市降价空间将在30%左右。

关于汇率与通涨
       5. 按照以上降价幅度，肯定不足以完成社会总负债去债务化，所以未来通过发行货币注销债务，形成通涨不可避免。
　　6. 同理，汇率将同步下跌
