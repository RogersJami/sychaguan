        时间有限，先给大家上一个范例看大牛们是如何分析丝带试飞的。理性的分析很精彩。真是不枉卧草党们拍摄PP和视频的辛苦啊！{:soso_e105:}
        今年2月初，我在CD上发现一段丝带试飞的视频（链接：http://lt.cjdby.net/thread-1335501-1-1.html），拍摄的非常给力，于是立即推荐给了晨枫。
         晨大很专业，很快就做出了分析。一篇专业的帖子出现在了西西河上：（链接：http://www.cchere.com/article/3676224）
目测歼-20
1、歼20的操纵面动作非常灵敏，动作幅度很大，说明控制功率够大，控制律的权限也很大，否则这样的控制动作非常容易over control，进入振荡。F-22的控制权限也有那么大，但进入了PIO。成飞不可能不知道这里面的厉害，敢用这么大的控制权限，一是对控制律有信心，而是对硬件、软件有信心。这个相当牛。
2、歼20的起飞滑跑距离比歼10双座至少短了1/3，这说明升力大，推力大。这不是表演起飞，也不是量产型飞机，还是试飞，估计没有发挥全部的潜力。目测感觉，起飞距离至少和F-22相当。
3、稳盘很厉害，看他压的那个坡度，至少有7g。再次提请注意，这还是试飞，而且不一定是最大性能试飞。如果没有猜错，这都可能还是试验机阶段，而没有到生产定型阶段，或者说相当于YF-22，能拉这样的稳盘很厉害了。
4、没有看到垂直爬升，可能那天没有这个科目，也可能还在等真正的大推。现在在飞的歼20传说是在用WS10或者AL31，推力比F119差一大截。如果WS15（？）能有传说中的推力，这歼20的性能真能赶上甚至超过F-22。
不知道美国人看了这一段有什么感想，我想印度人看了肯定有感想的，尤其是HAL的人：LCA飞了那么多年，可能还没有飞到歼20这个进度呢。
真想哪天能把歼20、T-50、F-22拉到一起，同台表演，肯定精彩！

上图。丝带和J10并列在跑道的尽头，其大小可自行分析。        河友： 莫愁湖水怪立即根据照片按比例推算出丝带的长度（链接：http://www.cchere.com/article/3676718）
关于20的大小，我根据这个视频的截图计算了一下 [ 莫愁湖水怪 ]
我不知道图怎么发的，因为拍摄地点没动，根据截图，我用尺子量出歼10和背景建筑物的两个门之间的比例为：1.16，计算出歼20和两个门之间的比例为1.459016，然后用1.459016除以1.16得出歼20和歼10A型的长度比例为：1.257773（以上比例均不包括空速管），在百度上查到歼10A的长度为16.43（不包括空速管），进而计算出歼20长度为20.665.因为我测量的误差，我认为歼20的长度应该超过20米，但应该不超过21米。这样就可以得到三个四代重歼的长度分别为：
F-22:18.92米
T-50：20.00米
歼20:20.66米。
我看了一下百度上的数据，歼20的长度为21.26米，我测量的为发动机喷口到机头的长度，如果计算从腹翼到机头的距离，估计差不多20.66米（腹翼好像超过了发动机喷口）。看来百度的数据还准确的。

关于J20稳盘和翻滚的数据计算 [ diamond ] 于:2012-03-31 03:25:19 复:3676224

在视频中大约6分钟前后的地方，有一段非常完整的J20稳盘视频，盘旋角度超过360。利用这段视频计算，向心加速度a=3.2g，盘旋角速度w=13/sec，盘旋半径r=619m，空速v=504km/hr，过程和结果见下图。图中选取了6个特征时间点，从1到6依次是：1－开始盘旋；2－尾部正对拍摄者；3,4－距离拍摄者最远点；5－头部正对拍摄者；6－刚好完成整圈盘旋，时间标记统一以1为零点。
http://picture.cchere.com/0,1203/56118_31031013.jpg
基本物理知识告诉我们，匀速圆周运动物体的向心加速a、半径r、线速度v和角速度w有如下关系：a=v×w，r=v/w。所以四个未知数中知道任意两个，就可以得到另外两个。由于视频是从地面拍摄空中，没有任何背景参照物，所以无法计算半径r。利用飞机和地面的倾角可以直接计算出a，但同样因为缺少地面参照物，我们无法确认镜头和地面的相对角度，摄像机可能是倾斜的，所以无法简单从视频中测量飞机的倾角，只能用做参考数据，后面会给出计算结果。
可见，准确的测量只能依靠v和w。
先说角速度w，这个相对容易，只需要计算完成一周所用的时间即可（姑且假设是匀速的）：
盘旋角速度：w = 360 / 27.78sec = 0.226rad/sec = 13/sec
困难之处在于v，天空上没有任何背景可供计算位置。万幸的是，当J20飞到最远处的时候，画面中出现了一根树枝，这真是雪中送炭啊。利用飞机和树枝交错而过的两幅画面，因为时间间隔很短，我们可以假设树枝是固定的，然后在利用飞机自身的长度作尺子，以树枝作参考点，得到这段时间内飞过的距离，于是速度就有了：
盘旋线速度：v = (26.0+30.0)m/(13.06-12.66)sec = 140m/s = 504km/hr
其中飞机的长度是关键数据，这里采用njyd（20.68m，链接出处）和莫愁湖水怪（20.66m，链接出处）测算的平均值：20.67m。后面的事情就简单了：
向心加速度：a = v×w = 140 m/s * 0.226 rad/sec / 9.8m/s^2 = 3.2 g
盘旋半径：r = v/w = 140 m/s * 0.226 rad/sec = 619m
最后，利用重力、升力和向心力的几何关系，我们还可以用飞机的倾角来估计向心加速度：a(图2) = tan71×g = 2.9g，a(图5) = tan73×g = 3.3g，和上面的计算基本吻合，同时说明摄影机的角度摆得相当正。这和代码ABC的估计也比较接近链接出处。
仔细观察这几张图，我们还能得到以下信息：
一，图6中的飞机大于图1的，这表明飞机不是严格地飞了一个标准圆圈，后来更靠近拍摄者了。通过对比图1、图3和图6中飞机的长度，可以知道拍摄者位置距离圆心约1.3km，还可以知道图1和图6中飞机的位置前后变化大约100m，这会带来一些计算误差。
二，图3和图4中飞机的轴线地面大约有+3的夹角。前面说过，摄影机的角度基本是正的，所以这个夹角说明飞机不是严格在水平面内盘旋的，而是稍稍有所爬升。估算一圈下来大约爬升了200m。
三，在图2和图5的时间点上，飞机的轴线恰好指向拍摄者，假定飞机是大致匀速的，通过这两个角度可以给草丛党定位。不过缺乏地面参照物，只能得到和飞机的相对位置，意义不大，就放过了吧。
另外，简单测算了一下视频里J20的翻滚速率，大约是120/sec，不算很高，看来这还不是一次极限试飞。
主帖里晨大估测了视频中J20稳盘的加速度是7g，看来是太乐观了，本帖的数据可能有点泼冷水。不过毕竟是刚刚试飞一年而已，何况3g的加速度也不算低了，来日方长，相信成飞不会让大伙儿失望的。
