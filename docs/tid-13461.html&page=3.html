现任的反腐被指过度消耗社会资源、理想化，总经理的经济政策也有待商榷，但有一点需要明确：这个时期政治改良的重要性大于经济发展。因为现任卸任后以后就没有人能够做这件事情了，以苏共的历史看从赫鲁晓夫、勃列日涅夫到戈尔巴乔夫，即无动力也无能力改良政治体系。宋一朝的政治制度决定了宋只能在积贫积弱中逐渐走向灭亡（重文轻武、文人兵役花销巨大），所以庆历新政与熙宁变法实际上只是用新问题解决旧问题，现如今一味强调经济忽视政治建设就与宋一朝类似。

现如今的难题是新时期社会主义制度该怎么走，具体来说就是缺乏相应的理论。这种理论学术研究是没有用的，只能由红二代根据中国国情具体设计。

技术进步带来的反馈，整合以及结构调整的合理，正在把系统里的人齿轮化，螺丝钉化，系统本身不用担心某人出局，就会造成整个系统崩溃的风险。反而是系统的缺陷会把整个系统带上和环境冲突而解体的困局。

其实很多体系里的讨论很大胆，直接学习韩国换届经验，认为再过几任政治腐败就彻底解决了。

我倒是不认同，因为中国体量和韩国不同，每一届上台都把上一任领导抓起来，等于从董事会一直换到居委会，肯定动荡，尤其不利于技术经验的传递，总之造成的麻烦比好处多。

解决的办法是搞增量。不过目前，是调整存量。

调整存量不仅仅是在存的货币上做文章，更重要的是优化被分配的各阶层组织，把蛋糕分给有前途和最需要的，而把那些扯犊子只知道吃蛋糕的混蛋从体系中剥离。

股市的刀斧升降，反腐从老虎到苍蝇，这些都是把依附于政权的各种增熵财富消耗性人口从政权中剥离，保持政权的活力，弹性和健康发展的未来空间。

呵呵，哥发明了一个词，叫增熵财富消耗性人口，，，这个词大家很好理解，体质里有些结构就是增熵的，但是更重要的是把这些增上消耗性人口不断的代谢出去。
