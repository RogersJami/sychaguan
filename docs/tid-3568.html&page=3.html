目前，一般的理解里，最大的组织单位是国家，而集体很多年前的话语了很模糊。如果一定细分定位一下，集体更多的可以理解成国家下面的商业组织--企业、一些社会组织等。在人成为明确的人力资源以后，已经不可能回到几十年前的集体主义理念体系去了。改开前的企业所有权虽然是虚位的，逻辑上可以是人人所有，主人翁才能成立。在企业层面讲团队合作更合适些。
如果国家和集体概念不能同一，逻辑上很大的可能会强化小团体性。
作为一个国家绝大多数人民来说，国家是提供最后安全保障的单位，就是再有想法，也不希望他乱。以国家为单位定义更合适些。
