jolery 发表于 2014-8-31 18:57:19

作为一个吃货，回想了一下几十年的饮食历程，发现不喜欢吃的东西不算多，大概有纯肥肉（太油腻）、羊肉（不喜欢那个膻味）、生的葱蒜（熟的会吃）、姜丝（调味能接受，但不吃），鱼腥草（实在很腥啊），还有我不怎么吃野味，觉得不够卫生（曾经在不知情的情况下吃过穿山甲:L）。

我十几岁就在外上学，本来挑食的人不得不把温饱放在第一，慢慢地小时候不吃的很多东西都吃了，其中茄子、黄鳝、肥肠、香菜现在还很爱吃。

前面茶友们提到的一些不能接受的食物，有不少是我的心头好，比如说臭豆腐（大爱:lol）、臭豆腐乳（早晨吃稀饭常备）、榴莲（这个因家人反感吃的不多）、芥末（少少来一点很过瘾呢）。

总的来说我喜欢吃的东西多，不喜欢的少，还有一些就一般般，关键看做法了。
