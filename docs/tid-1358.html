找到了，在这里
现在满大街都是大忽悠小忽悠，搞教育的无良之人实在不少。

所以，父母们更需要培养自己的教育学素养。

系统的培训来不及了，快速的，我建议看这几本书，能武装一下，提高判断力。

1、陈鹤琴的《家庭教育》，虽然年代是早的，但依然适用。这是把人当人来教育的教育观念。值得借鉴。
2、《给教师的100条建议》，苏霍姆林斯基的。能帮助家长了解什么才是好的学校教育，什么才是好的教师。
3、《爱的艺术》弗洛姆。里面谈到父爱和母爱，观点不一定全对，但有助于对爱的理解和思考，而爱是教育的核心和基本。
4、《斯宾塞的快乐教育》。个人非常非常喜欢。但要看原文译本，不要看专家解读过的二手读本。

将这几本精读之后，抗忽悠能力会提到一个层次的。

这么说吧，评价一个儿童的课程好不好，要看：

1、你的目的：为什么要学这个？

2、谁在教？教师是关键，课程内容只是个工具和载体。设计得再高级的课程，换个烂教师，也能给变成烂课。再普通不过的课程，换个好教师，照样能给变得高级。

3、孩子在其中的感受如何？这个是硬指标。你家长再喜欢再欣赏，孩子根本不喜欢，就没必要勉强。

这几个方面考虑过了，基本判断就有了，我想一般人都能掌握的。而内行跟外行的区别只不过在速度跟程度方面的差异罢了，比如我去现场，可能在前3分钟就能做出判断，而你，可能需要花几天时间各方面都接触调查了解摸查后再做判断，结论应该差异不大。

我女儿曾经上过一个学期的蒙校。感觉还是不错的。如果不是因为搬家，我们还会上下去。

它那套理论我了解不多，但从我女儿的表现上看，蒙校有几个优点。

1. 让孩子以自己的节奏发展。有时候上课就是安安静静的，每个小孩自己拿本书看。不同的孩子看的书的难易可以差别很大。    那段时间，我女儿对看书比较感兴趣，根本不需要我们推。我很惊讶。
2. 教数学结合玩具，学生理解得快。
3. 比较尊重孩子。讲究让孩子的天性自然发展，老师会努力发现孩子的优点并加以引导。对孩子的有些不足不太在意，认为以后随身体发育会自然改过来。
4. 老师水平普遍比较高，谈吐雅致，要么资深，要么硕士。。有的对孩子充满了热爱。

谷歌的两位创始人，当年从斯坦福退学搞企业的那两个，就是蒙校出来的。据说他们都很喜欢。
