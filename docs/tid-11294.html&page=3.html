我不开日本车，但是我觉得从各方面数据来说，日本车的优点还是有的。偷工减料么，反正日本车的事故伤亡也 ...
摸过不少车，日本车韩国车在城市里开开还行，一上高速，致命性缺点就暴露无疑：

1，刹车反应时间慢、刹车距离长，踩下刹车踏板，车还在往前窜，刹不住；

2，抓地力差，像坐着船在水面上晃；

3，底盘太软，桥因为打桩下沉少，而路面下沉多，桥和路面之间有个高度差，如果下桥速度过快，车的前半部自重大，车头有栽下去磕到路面的危险，翻车。

4，加油口在驾驶座同一侧，油箱离驾驶座离得太近，如果着火，容许的反应时间短。

所以要么不出事，一出事就是大事故。我把车看作代步工具，看重安全。


国产车和日韩车的内饰一个德性，花里胡哨。
