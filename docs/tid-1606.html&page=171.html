多谢，我的膝盖也不是很好，我要试个更简单的，在脚脖子处把两条腿绑在一起，用条绳子或皮带就可以了....

当时医生说我膝盖受伤后一直不运动，导致膝盖周围肌肉力量不够，走路时关节受力太大，所以打针贴膏药啥的都没明显治疗效果。
他让从最简单的动作锻炼起：我坐在床沿上，绷直腿、并拢抬起，他握着脚脖轻轻向下按，自己用力保持双腿并拢、悬空、绷直。肌肉力量不够的话，坚持不了多长时间。此外就是绷直腿向下稍用力、医生握着脚脖向上扳；绷腿伸直试图分开、并拢，医生握着脚脖向相反方向并拢、分开。
这个动作很简单，主要目的都是为了从左右上下方向让膝盖周围肌肉受力。锻炼的时候，腿要保持并拢、悬空、绷直，而且不需要多大的力量，只是为了让膝盖得到适当的锻炼。
如果没人帮忙的话，可以坐在床沿上、腿并拢绷紧，脚脖伸进小凳子里，试图用力把小凳子抬起，这相当于有人按着脚脖；还可以夹凳子、压凳子等。绷直腿，很小的力量就能让膝盖充分锻炼，贵在经常做。
另一个动作类似这个里面的单腿下蹲：http://bbs.dzwww.com/thread-24609875-1-1.html。
更复杂的锻炼动作也可以看这个http://wenku.baidu.com/view/6b52 ... astweight=2&count=5
对于经常不运动的，经常做这些简单的活动可以锻炼关节肌肉，减少关节的磨损也减少疼痛。
