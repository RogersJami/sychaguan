如果我读懂了你的意思的话， 你的暗示很无聊－－这是客气的说法。

前年北美大雪， 有些人在机场被困了一个星期， 也没见有人冲跑道。冰岛火山爆发的时候欧洲被困机场的也差不多严重， 也没见有人冲跑道。
还是中国人民剽悍。 不过呢，羊毛出在羊身上， 航空公司不是活雷锋， 这种事情多了的话，等着机票涨价吧， 良民们为刁民买单。 涨价还是小事， 其实这里面的安全隐患大了， 难道要出了机毁人亡的事情， 才会有人后知后觉遗憾机场管理居然如此软弱，养虎为患。

以下是一个专业修飞机朋友的评论
－－－－－－－－－－－－－－－－－－－－
冲撞跑道是有很大危险性的。
其一，被飞机轮子压着，肯定是把自己被压死，同时飞机被倾覆的可能性极大。因为飞机起飞时，襟翼和升降舵都是计算机控制配合动作的，突然那么颠一下，平衡瞬间被破坏，飞行计算机怎么反应，实在是说不清楚。
其二，冲撞跑道有很大的可能是冲撞到发动机里去。从绞肉机里出来，就没有什么大块的东西了，这是冲撞者的结果。对飞机的结果是，发动机推力瞬间下降，最坏的情况，该飞起来的时候飞不起来，刹车中断起飞已经来不及，冲出跑道，也是机毁人亡。
其三，飞行员全神贯注操纵起飞，或降落时，别说是人冲过来，就是一只鸟，一只野兔冲过来，说必定都能吓个半死，结果也会很吓人。
其四，塔台上的空管可能因为看热闹分神，造成空中的两架飞机相撞……
随便举几种可能性。但是可以确定的是，这种行为绝对影响航空安全了。
