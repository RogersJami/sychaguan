自制芒果冰激凌

上次看到海之沙的冰激淋，自己也蠢蠢欲动。因为奶油之类比较不方便购买，自己也就经常diy芒果冰激淋。之前一致用盒装蒙牛，但自从蒙牛出事之后也就没有买过，也未做过。这次用巴氏牛奶用之前的懒人方法就不太合适，容易结块，也不知道是什么原因。以下做法也非常简单。

主要材料：冰糖少许（因为牛奶不会很甜），芒果2个（差不多一斤了），屋顶牛奶一盒

做法：1. 少许水将冰糖融化
           2. 倒入芒果粒，边煮边搅，直至浓稠。其实还可以收干一点。
           3. 冷却后倒入牛奶中搅拌。 （这个时候，里面可以放煮好的西米，直接就是芒果西米露。不能吃牛奶的可以换椰奶。）
           4. 放入冰箱冷冻。半小时搅拌一次，原则上是次数越多口感越细腻。按照海之沙的流程，8次，那天因为办事，没有时间多搅几次就冻了。后来成品的口感就不够滑。当然，如果有电动的打蛋机就更好了。比较容易达成效果。

最后一张是冻了一晚上的图。之前只搅了4次。吃起来的口感比较像冰沙。另外还有一种雪芭，方法类似，不过少了牛奶跟水。也是需要好体力来搅拌的。
