至于人工饲养就不存在虐。这是你的逻辑。我不接受。是生命就存在虐。试验室用动物，有其不可替代性。但是，就是试验室，如果不以实验为目的虐杀生命也是虐待。相应的，熊胆之用于药，是具有替代性的。人可以利用动物，但要取之有道。在对动物的害和对人类的利之间取一个平衡。
