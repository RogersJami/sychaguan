葡萄的4条消息：
1. 黄金暴跌是系统性风险的真实反映，不是宋鸿兵所说的国际大投行的佯动。
2. 系统性风险在即，来不及无差别放水了。
3. 若是无差别放水，我们会被美国打爆。
4. 不出意外我们经济刺激计划不远了。
然后前天网上出了一条消息，央企被要求现金为王。

对于黄金这点，争议还比较大。@美国李山泉 昨天还在怀疑宋的说法，今天又似乎在认可宋的说法了。实物黄金在中国被疯抢是事实，符合宋的说法，黄金的暴跌是打压后回吸，还是有人在出逃持现，还是要等等看。
但不论黄金会怎样，葡萄的观点是“（世界性的）系统性风险正在或者将要发生”，因为金价不是中国决定的，如果仅仅是中国的系统性风险，他就不该和金价联系起来。

不会无差别放水，有两种解读。一是不会放水，二是只会定向放水。
我们已经知道，四万亿是无差别放水，结果是便宜了地产党，并且形成严重的经济问题。如果继续无差别放水，那么房价必然会再来一波，不用美国动手，我们大概自己都会爆掉。对于葡萄的第三点，大家都没有争议。
所以问题是，我们会紧缩，还是会定向放水？如果是定向放水，水会不会跑冒滴漏？
另一个问题是，为什么世界性的系统性风险，会和人行的无差别放水关联起来？或者说，中央一直有意无差别放水？或者说地产党和地方政府一直在对中央施压无差别放水，而现在这个压力被另一股压力对顶掉了？

如果有新一波经济刺激计划的话，那么我的理解是：
1. 国际经济要出大问题，所以有人在抛黄金换现金。非常大的问题，才能被称为系统性风险，是南欧要脱离欧元？日本的贬值计划失败，然后经济崩溃？应该不是美元崩溃或者美国的社会保障破产，那个还要再等十年。
2. 出口要出问题，所以要依赖内需（经济刺激计划）来解决。本来在经济结构调整期，根本就不该出什么经济刺激计划。
3. 国内经济内在的问题还没解决，所以只能定向放水，不能无差别放水。如果地产党的问题被消化了，倒是可以无差别放水。
4. 定向必然是工业党和高铁，产业升级是这届政府的任务，这点没有疑义。
5. 央企的现金为王，这和葡萄所说的“不要追求杠杆和收益，以活下来为第一目标”相一致。这说明软妹币的货币市场，会有一波银根紧缩。紧缩的来源，一是国际环境（如08年外资回国救主），二是国内债务链崩溃导致货币乘数变小，而不会是人行（不然就不需要讨论放水问题，而是讨论紧缩问题）。
6. 小虎对地方政府债的意见是，这只是钱从左口袋放去右口袋的问题。那么这个债务链崩溃应该是指信托，即政府决心让信托崩溃的可能性在增加。我们都知道信托一直撑着是因为政府还没下决心放手不管，但是放手是迟早的事。会影响一部份的银行坏帐，特别是这两年大量基层内外勾结，把钱放出去然后转成高利贷。

基于以上理解，我们有一些新的问题：
1. 定向放水+经济刺激，是不是会在“国际系统性风险+国内信托崩盘”之后才出？并且在额度上会尽可能地小，只是起一个部份的对冲？因为之前四万亿还没消化掉，不可能再主动出击。在货币市场上的表现，就是紧-紧-紧-开始慢慢松，非死掉一批投机资本和信托这样大杠杆的资金，不能指望脖子上的绳子会松开。紧会持续多久，就要看杠杆资金死得多快。
2. 跑冒滴漏会有多厉害？我不认为会超过之前的表现，甚至可能会更紧，因为之前银行还有钱被转放出去做高利贷，现在存贷比都到顶了，没有无差别放水，意味着存贷比不会放松。而表外正是逃离的时候，前几年的表外开始出问题的时候应该没有人会冲进去解救别人。
3. 逃命行情是不是已经结束了？还是有最后一波？昨天的大涨，今天的大跌，说明市场上争议还很大。小虎在三一11块时满仓，现在他是在死抗，不过我觉得茅台还是远远高过了三公新政出台后的fair value。如果信托崩盘，中小银行股也是高估的，包括民生。如果银根变紧，利率上扬，估值中心（平均PE）必然要下降。
