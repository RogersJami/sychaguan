夜深人静的感慨，只是觉得忙总“【我的理想】最简单的要求”贴子，能加上一抹理想的色彩更好了。
每个人的人生有很多选择，如同有人爱在天涯扯淡，有人爱在西西河里战斗，有人爱在茶馆里喝茶一样。忙总是让我们高山仰止的人，他的人生经历、学识和思想的穿透力，不用我们替他选择。他的很多选择，值得我们思考和学习。
我们先人的故事传说，绝大多数都是美满的结局加上良好的祝愿。没有人的人生是完美无缺的，人的想法也会随着时间会变化。我的老师也是我非常尊敬的人，她的思想的形成，有早年成长的心理影响，有她先生类似忙总的漂泊和事业追求，也有现在的些许遗憾。
从《食客的意境》来看，忙总早年也不是打定主意要丁克到底的人呵。人生有时插曲变成主题曲，主题曲变成插曲的很多。但我想我们内心深处固有的传统的东西，总会不时泛上心头。你如果在国内，这种善意的劝说还会不绝于耳的。
我的话和茶馆童鞋们的一样，=“祝忙总幸福，祝康总幸福，祝忙总和康总幸福”。
