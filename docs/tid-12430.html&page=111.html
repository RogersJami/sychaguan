还漏了一个曙光股份也是小仓位试盘

这里策略是每个第一次试盘的票除非是自己很熟悉的老票仓位不超过百分之五，然后在突破位置启动后出现第一个百分之4且尾盘站稳百分之四，加倍加仓。如果第二次启动冲板不动。如果第二个百分之4触及再加倍。（讯飞就是基于这个标准第二次加到半仓）然后仓位锁定吃复利。等一轮走完各个不同板块与概念票梳理一下为下一轮做准备。

我这里做票大致特点是，半仓一个票做主力，辅助主力的票是主力仓位一半或者三分之一确保主力仓位波动区间波动可以被脉冲覆盖且做好主力仓位波段做完换入准备，机动仓位是个下一个下个半仓与辅助仓位票。

小小心得，各位参考。
