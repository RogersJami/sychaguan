从根子上讲，29年后，世界绝大部分工业集中在四个强力集团手中，美工业最强、英法集团、德国、苏联与其相差不大。受经济危机影响，除苏联外各国经济均受重大影响。为了争夺市场、争夺势力范围，各大国之间关系急剧恶化。

英法维护殖民利益，美国争取世界经济主导权要求放开殖民地经济、德国要扩张势力范围争夺资源和土地、苏联天然挑战旧殖民秩序和市场经济秩序。其余日本、意大利等国家，工业能力太差，在全球竞争根本不是四大的对手。一句话，英法守擂美德苏挑战，三个挑战者的利益又无法共存。

这种实力下的博弈局势，摆明了只要有人动手挑战则英法必然失去全球秩序主导权，但先挑战者是螳螂，后发动者是黄雀，最后登场的是猎人。所以美国、德国兴登堡政府都是宁可内伤也忍着不动，苏联增长速度快所以努力维持和平号多些时间增长实力，英法借挑战者之间的相互忌惮而暂时保有主导权。

但德国先熬不住了，希特勒上台，开始军国化了，军国化后除了开战就是经济崩溃，除非它能先后打败英法苏联和美国，否则战略上已经死定了，唯一的悬念是它会拖着谁一起死。当德国完成不可逆转的军备后，美国隔岸观火准备摘桃子，英法期望德国拖着苏联一起死然后全力压制美国以维持旧秩序，苏联期望德国打英法给自己创造时间和机会。所以有了绥靖政策和苏德互不侵犯条约。

德国选择了先挑战英法，获得了出乎所有人意料的快速胜利，也给自己赢得了一线生机。全世界都没想到战争进程这么快，苏联预计43年才能完成军队机械化武装，美国甚至还没开始全面武装，都无力主动进攻德国。只要德国能利用时间差在43年前搞定英国，四强变成了两大一小的三强，美苏都不敢先动手，德国就安全了。这个方案可行性很高，德国如能有效统和法国的产能，有很大把握用空军控制英吉利海峡的制海权，进而登陆干掉英国（插一句，站在英国的角度，这时是绝不可能和德国媾和的，想想和与不和的下场就知道了）。
