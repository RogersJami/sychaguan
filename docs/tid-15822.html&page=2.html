大师兄感觉敏锐，俩补充思路正确，通过脑电变化研究思维过程是最早被研究者意识到的物理工具。【事件相关电位】是一种有近五十年历史的研究方法。以下是我摘抄的部分介绍。

脑电图（EEG，electroencephalogram）是通过在头皮表面记录大脑内部的电活动情况而获得的。大脑内部活动的非常微小的电活动能够被置于头皮表面的电极记录到。这些变化可以通过示波器与阴极射线管显示出来。EEG的关键问题是脑电活动的自发性和背景活动会阻碍对刺激引起的信息加工活动的记录。

针对这一问题的解决方式是多次呈现相同的刺激。随后，每一刺激呈现后的EEG片段被抽取出来根据刺激发生的时间序列加以排列。把这些EEG片段叠加后平均就获得一个单一波形。这样一来，我们通过这种技术就从脑电图记录中获得了事件相关电位（ERP，event-related potential），从而允许我们把刺激的效应从背景中分离出来。

评论（我做了些删减）：事件相关电位并不能提供脑功能定位方面的精确信息。事件相关电位只有在刺激非常简单且所给任务只涉及基本任务时才有说服力。由于这些限制，事件相关电位不适宜研究复杂的认知过程，如问题解决或推理。
现代研究活动分工日益细化，但心理及行为学独立性不够，这是外行可以参与重要原因，高工科素养是可以快速进入的。如研究数理统计的质疑fMRI工具的统计方法有问题，虽然质疑有些外行但把存在的问题突出了。再比如脑神经研究中用到的仪器，研究者会用，对原理、维护方面并不精通。这个行业不少是搞计算机的，搞统计的，搞应用数学的。论文分两类，一类是实质性贡献的，如张益唐的孪生素数上确界结论，另外一类是添砖加瓦的，如普通博士论文和《Nature》被反驳的那些文章。

脑神经科学假说多模型多，某个方向的最终结论出来前容易迷路，学习者研究者也会怀疑这些模型的真理性。像颞叶的听力与记忆的关系，很多年前就有的结论，但我怀疑其真理性，直到发现与自己的记忆模式有关联才确信它。

师兄的两条很赞同。直觉好、能打破框架就够了。只要说的话发的文能够引起重视，逻辑没有明显漏洞，自然有人会帮忙验证的。例如希格斯波色子假说存在了三十年还是四十年，欧洲核子研究中心的强子对撞机找到现实证据诺贝尔奖立马颁给他老人家。
