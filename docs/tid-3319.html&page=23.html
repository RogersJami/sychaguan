美元收缩进一步明确
奥巴马敲定下任美国财长人选
美国总统奥巴马当地时间1月10日正式提名现任白宫办公厅主任雅各布·卢接替蒂莫西·盖特纳，担任下任（第76任）财政部长。

　　这一提名符合各界预期（详见东方早报1月10日A38版《美国下任财长最快本周揭晓》）。不过，从最新报道来看，奥巴马接下来还要面临多重挑战：一，上述提名能否获得美国参议院批准；二，现年57岁的雅各布·卢能否以自己的实力回击各方指责奥巴马从亲信圈子中选人；三，雅各布·卢除了丰富的预算经验，在金融监管和国际沟通方面能否令奥巴马安心。



“预算专家”

　　从最新的报道来看，各方普遍质疑，奥巴马此次是在亲信中选任。

　　对此，奥巴马在提名仪式上称，雅各布·卢是个“低调的人，更喜欢被政策专家而不是被摄像机包围”，并称他热衷于公共服务。

　　“经过多年，他获得了政策大师的声望，能与两党成员共事，促成原则上的妥协。”奥巴马说。

　　此外，奥巴马再次重点强调雅各布·卢在处理财政赤字上的能力。奥巴马说，“人们所说的削减赤字以确保收支平衡，这个人做到了，而且有三次。”

　　在雅各布·卢担任前总统克林顿的预算主管期间，政府在1998-2000年连续实现预算盈余。

任大嘴那句话怎么说来着：当地产商任志强被问及‘道德血液’论时,即毫不客气地回讥:“他欠缺什么 ...

其实任志强这几年的言论，大多最后验证是正确的，忠言逆耳。听了牛刀之流的空军，肠子都悔青了吧。

元旦前后十来天，我跑来跑去看了十好几个楼盘，总的来说，各楼盘的销售情况还算不错，没见到有降价的压力。

北上广的情况特殊一些，不去讨论。一般的省会城市和大城市现在房价恐怕很难跌下来，虽然房子的供应一直在增长，但是城镇化带来的外来人口也一直在增加。就拿我来说，就是大学毕业留下然后买房的，以后父母年纪大了也要来，还需要一套改善型房子呢。只要经济能力允许，这也算是刚需，其他的投资渠道也太少。身边的同学大学毕业留在城市的很多，和我情况类似的也非常多。

而在老家县城的人口并没有减少，更多的人源源不断的从农村进入大城市和县城。老家的二手房很大一部分是被刚进入县城的农村人口购买了。

十八大开完以后房价就开始涨，至少成交量放大。这里面说明消费者对房价的预期是涨的，就连我父亲都说收入要倍增，房价肯定要上涨。我一个专门从事炒房的长辈说，去年7，8月间是市场最清淡的时候，之后市场就转暖成交量放大了，房价也开始坚挺起来了。
