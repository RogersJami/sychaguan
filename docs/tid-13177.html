农村那点事

1、邻居张二叔

我们村有好几大姓，基本以生产队为单位，我们队以我的本家户祖为主。隔壁的张二叔家在我们村是孤门独户，虽然村里还有其他姓张的，但是和他家没有任何关系。张二叔娶了个非常厉害的老婆。张二婶是从邻县的大山里来的，我们那个地方虽然落后，但是和邻县的大山里相比，还算发达地区，因此张二叔家虽然穷，张二婶却颇有姿色。印象里张二婶特别地能战斗，先是把张大叔打出家门、另立门户，又把张爷爷打得喝了敌敌畏。张爷爷死后，张二婶又接着跟张奶奶打。印象中张奶奶活得挺长，真觉得老太太挺不容易的。张二叔则是窝窝囊囊的气管炎像。

张二叔家给我印象最深的是“跑计划生育”。张二叔家生了两个女儿后，就开始“跑计划生育”。张二婶怀上老三后，就不知道躲到哪儿了。为了确保生儿子，张二婶做了B超(当然这是违法的)，B超结果是女孩，于是张二叔和张二婶毅然决然地决定把孩子流掉。具体方法是往胎儿的脑子上打一针，胎儿死后自然会流下来。结果做B超的和做流产的都是二把刀：首先孩子是男孩，其次流产针打到了腿上。结果张二婶回来的时候，带了个瘸儿子。于是接着跑计划生育，终于生了个老四是儿子。

张二叔生孩子的时候，正是计划生育搞得最狠的时候。那时候经常能看到一些杀气腾腾的标语，比如“一胎不扎，墙倒屋塌”，“宁让家破，不让国亡”等等。于是张二叔家一次接一次地被抄，用家徒四壁来形容，非常确切。我们村在解放前是堡垒村；文革期间，地委副书记下放做我们公社的书记，工作就是以我们村为中心开展。我们村这种先进村，执行D的政策一直比较自觉，像张二叔那个年龄段的，大都是一、俩孩子，三个孩子的极少。因此我们村计划生育压力不大，否则像张二叔家那种情况，房子早就被拆了。

2、

我家兄妹俩早就出来了，我先是在北京，后来在国外。我妹妹呢，则是在市郊区政府工作。我家那种情况，在农村非常尴尬：首先老人身边没有孩子，容易被欺负。我父亲的两个姐姐都嫁到临县县城，我父亲在村里没有亲兄弟姐妹，更加剧了这种情况；其次，我兄妹俩比较成功，村民都觉得老人比较有钱，很多人惦记。我家被偷过，更不要提老有人上门借钱了。

我父母在那里生活，唯一的选择就是低调+好施。乡里乡亲的上门，尽量帮个三块两块，过年的时候，别管是本家、亲友还是邻居只要上门，压岁钱给的比一般人多点，这样下来，名声颇好，加上我们家是大姓，倒也没有人好意思欺负他们。

俗话说，远亲不如近邻。隔壁的邻居，自然要好好处。

我在国内的时候，张二叔家和我们家关系挺好。他家跑计划生育，一些简陋的家具和粮食就放在我家，这样回家的时候不至于没得用。我记得不止一次，张二叔或者张二婶听到风声以后，把钥匙往我家院子里一扔，撒腿就跑。我家要好吃的，也会给他家孩子留点。我和我妹妹回家，也会给他家带点东西，客客气气叫个二叔。直到我出国之前，关系一直是很好的。

我出国后，有一年，突然听到消息，说父母要卖房。而卖房的原因，则是被张二叔欺负的。

说句实在话，在我的印象里，张二叔一直是个窝囊废，只有被欺负的份。无论实在专政机器面前，还是张二婶面前，他都是逆来顺受。他如何欺负我父母，为何欺负我父母，我实在想不出来。

3、

赶忙跟妹夫和老家的堂兄弟打电话，问问到底是怎么回事。得知事情的来龙去脉以后，不仅暗叹一句“可怜之人必有可恨之处”。

张二婶生出了一个健康的儿子后，就不生了。该结扎结扎，该罚钱罚钱，没钱可以欠着，慢慢交。反正孩子已经生出来了，政府不能把他掐死。虽然作为反面典型，时不时还会被抄家，但是稳定多了。随着时间推移、孩子长大、计划生育政策不再那么极端，渐渐不用跑了。生活稳定下来后省吃俭用，开始能攒点钱。

农村孩子结婚早，尤其是女孩，早的16、7岁就开始说亲、相亲、定亲(我上高中的时候就不时听到有同学相亲)。前文说过，张二婶颇有姿色，两个女儿都长得还行，尤其是大女儿，按农村标准，蛮漂亮。张二叔嫁两个女儿收了彩礼，自然要为儿子打算。他家的房子家徒四壁不用说，而且破烂不堪。每到下雨的时候，都是外面大雨，里面小雨，不先把房子翻盖了，托人说亲都不好意思开口。

嫁两个女儿收的彩礼，加上省吃俭用攒下的一点钱，张二叔开始盘算翻盖房子。张二叔家房子不大。前面说过，张二婶特别地能战斗，先是把张大叔打出家门、另立门户，又把张爷爷打得喝了敌敌畏。张爷爷死后，张二婶又接着跟张奶奶打，就是这个原因。现如今张二叔有俩儿子，而且其中一个残疾，仅仅是拆了老房子，盖新房子，解决不了问题。

4、(带孩子出去学这学那，匆匆发了一段)

以张二叔家的实力，重划/买一块新地建房是不可能的，只能把原来的房子扒了重盖。为了尽量把新房盖大点，张二叔跟我家老人提了一个非常过分的要求：他家新房直接贴着我家屋山墙盖。大家都知道，山墙下边是地基，地基要比山墙宽，他的新房直接贴着我家山墙盖，就是直接盖在我家地基上，安全问题不讲，如此一来，两个房子就成了一个房子了。农村盖房引起的纠纷不少，两个房子中间留多大的间隙，前房为后院留多大的“滴水”，你比我高、我比你低的，花样不少，因盖房而吵架的不少，而像张二叔要求这样无理的还很少见。

张二叔之所以敢提这种要求，估计有两个原因：一是这些年他从我家占了不少便宜，给了他一个我家人不在乎钱的印象；二是农村房子不值钱，我和妹妹也都不会回去住，既然对我家人无用，就接着帮帮他家吧。故此，他对我父母答应他的要求有侥幸心理。

虽说老家的房子我们不在乎，对于老人来说那可是相当的重要。对于张二叔的要求断然拒绝。

张二叔最开始的时候来软的(套近乎、装可怜)，后来来硬的(动手他不敢，孤门独户的，敢动手的话，那是活腻歪了)，对于这个原则问题，我父母软硬不吃，坚决不同意。并找村领导出面调解。

张二叔眼看软硬都不行，最后扔下一句狠话：“别管你同意不同意，我到时就这样盖......”

5、

张二叔家开始盖房后，老人就在旁边看着。不仅自己看着，还准备好烟好茶请村干部来我家一起看着。干活的泥瓦工不傻，自然不会配合张二叔玩硬耍横，更何况村干部还在旁边，哪会为了一个孤门独户得罪大多数啊。最后，张二叔的便宜没有占成。

张二叔“应占的便宜”没占到，便露出弱势者丑恶的一面。自那以后，我家老人只要在家，他就在自家院里骂街，今天骂鸡，明天骂狗，后天骂孩子。老人如果接招，正中他下怀；不接招，又觉得窝囊。老人躲到别处去，则家里的东西准丢。最后老人把院里所有的东西都收到屋里藏起来，出门回来后发现放在茅厕里的敌敌畏(夏天放在喷雾器里打蚊子)被偷了。不仅丢东西，院子里的葡萄树、枣树、蔬菜花草也会遭到破坏......

您可能问我，你们家不是大姓吗？为何被一个孤门独户如此欺负。其实我本家的青壮年早就在那里喊打喊杀了，一直被我家老人拦着。老人说，他们打完了走了，还得留下我们跟他做邻居。再说了，万一打出毛病，还得花钱给他们看。尤其是他那个瘸儿子，打架的时候挥着拐杖狠打；被打倒了就躺在哪儿装死......

整个过程中，张二婶倒是表现的比较通情达理，试图劝张二叔；不过张二叔突然雄起，一改气管炎的形象，张二婶根本压不住他......

我家有个宅基地，是一个大池塘，因为是祖上继承的，所以一直是我家的。有个村民家里儿子多，没有宅基地，就用田地跟我家置换。但画好 了置换多大尺寸。结果，他家盖好房子以后，非说山墙东面也是他家的。于是，产生纠纷，巧的是，我家把换地的协议弄丢了。当时，村书记，村长都出面证明，说那东面的地方是我家的，他们都不听，就摆出架势要干架，镇上来人调节也不行。我家在村上是大姓，但是碰上这种不讲道理的，也是苍蝇含在嘴里，膈应。我父亲到镇上找律师，还被讹了300元，也没起作用。
为此，父母当个心事。我想出面，父母不干，说我出面丢身份。当时，我想，他们既然认为协议丢了，就不认账。那干脆我们也认账。既然他们说地方是他们的，让他们拿出证据，因为那个地方是我们的宅基地，全村公认的，打官司也不输理。我就把这个想法跟父母说了。后来，事情解决了，他们认了。至于怎么认的，父母说他们既然承认了，就不要再追问细节了。
