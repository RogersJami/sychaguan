楼主对石材行业前景怎么看？



不知你想问什么，如果是生产商，主要看矿山资源质量，花色，成材率，运输条件等等，如果矿山资源质量不好，天王老子也没办法，照样亏损。

如果是经销商，主要靠关系网。其实国内大的石材销售商的加工技术和设备都差不多，进口货源也一样，经常组团进口，拼盘分货，竞争的主要是请客送礼能力，也就是能不能把各省的管招标、基建和代建的领导请出来，把礼送出去。


进口石材现在市场基本还能维持，花色品种不同，毛利率差别很大，所以大家现在只能吃楼堂馆所这种毛利率大的客户，平均毛利率30%左右，还没什么坏账。一般客户基本做不下来了，因为应收账款太要命。

如果有关系，做起来很简单，投资几千万就可以。
