流浪汉的投机实验之四

先后在网上写了近一百篇不成样子的文字，前段时间准备搜集整理一下，却发现大多数已经找不着了。于是花了一个多月的时间，反思了近几年股市投机的艰辛历程，写了一篇“流浪汉关于股市投机的自言自语”发在几个常去的论坛。
“投机实验”已经写了三篇，都是在极为困顿之时。生活稍微稳定，就顾着贪玩去了；若是过于劳碌，也没有时间静下心来思考。苦难成就了一个人的深度，此言非谬。
通常，我都是想得太多，做的太少甚至没去做。有过实际体会的知识和单纯书本的理论，应该是截然不同的，尽管文字含义可能相差不大。
虽然我曾在大学中文系混过几年，但是文字写作真没认练过，通常是想到哪写到哪，毫无条理章法。前几天在茶馆写了点东西，被人批驳得十分狼狈，这回吸取教训，一次只写一个小题目，争取条理清晰。

juyern 发表于 2012-11-1 01:10:59

从公司的控制和管理者而言，他的这些想法比较传统，无可厚非。但是一旦进入二级市场，环境发生改变，规则也随之变化，再想坚持这些想法也就很难了。试想一下，股东衡量管理层最直接的指标，不就是近期股价的涨跌吗？你可以有长期的战略规划，但是你想要执行下去，得顶着现阶段的巨大压力，无论是时间还是风险都不匹配，因而这只能是教科书式的理念。

谈话中他提到，该集团准备将下属一家生态农业公司在主板上市，但是其资产规模不够，现已与券商和三师多次接触。他认为，公司应该在主板上市，创业板市盈率过高，存在泡沫，是投机。我觉得这些想法过于教条，近乎荒谬，理由如下：

1.对于市场的参与者而言，投机与投资，本身并无区别，只是我们人为地将其框限区隔起来。我们参与市场的目的就是赚钱，其他的功能应该让别人去操心；

2.对买方不利，则对卖方有利。别忘了你处于卖方，创业板市盈率高，这不正好可以利用创业板二级市场里众多赌徒的贪婪和愚蠢吗？同样是卖东西，自然是谁给的价钱高就给谁。

3.所谓三高一新公司的股权大多是画饼，只要我在招股书里详细和坦率地说明各种风险，你还要争抢着来买，我并无诱导或者误导之意，无论从道德和法律上都无可厚非。别忘了股权可是一经售出概不退换的。

juyern 发表于 2012-11-1 01:52:50

从另外一个角度来讲，作为二级市场的参与者而言，如果你的资金在数百万以下，那么你很难支付克服信息不对称的而付出的成本。市场就是这样，你的位置一旦确定，那么你的优势和劣势也就很明显，短期内很难改变。这本身就是一个不公平的游戏，一旦你参与的话。

从一个交易系统而言，只有买入，而没有卖出，这就是不完整的。由于分红极为有限，必然要求在股权出售中实现绝大部分利润。只强调价值和长线持有，这是极为荒谬的，无论多么好的股票，都有下跌和消失的那一天。永久持有不过是自欺欺人。

这里还有两个问题，一个是自我实现，一个是波动率，先说第一个。如果你买如一个公司的股权，清理管理层中的垃圾，提高公司的运营效率，那么你所做的就远离了一般的投资范畴，而是通过行动去促成自己的预言。顺带说一句，鼓吹长期持有好的有价值的公司，即使不去干涉公司运作管理，同样也是一种自我实现式的预言。很简单，你的买入，促成了股价上涨，公司融入更多的资金，扩大受益，股价由此上涨，这是一个正向的循环。这里只是简要地描述一下，里面有太多值得研究的东西。







juyern 发表于 2012-11-1 02:14:45

以前已经说过，我偏好市值小、股价低、跌幅大的目标。股价的波动可能是无序的，但是我们既然参与其中，就不得不从中摸索和分析一些在某短时间内多次重复出现的现象。价值是跌出来的，只要你跌得够多，以后总会涨起来，只要不退市。你的买入价比大多数人低，那么面对市场的波动，你就更为从容不迫。

试想一下，三个序列，分别如下，哪个价值更大，如果是股价的话：

1.    10——20——10——5——10

2.    10——10——10——10——10

3.    10——15——10——8——10

juyern 发表于 2012-11-1 02:55:43

这样就更容易理解了，一般情况，波幅更大的目标，蕴含的机会就更大；在同等波动幅度的情况下，波动的次数越多，则机会越多。如果每天的价格都是不变的，那你放在银行好了，几乎没有价值。如果说在商品领域，交易产生价值的话，在市场里，波动产生价值。这样股票对应的公司的情况却反而显得不是那么重要了，好像有些本末倒置。

基于上述原因，就能很好地理解我的选股标准了。一句话就是，质量轻，波动大。

不过，这还远远不够。

二、指数单向的运动，幅度和持续时间将会有改变：牛市的涨幅变小，时间变长；熊市时间缩短，跌幅变小，但是短期下跌速度可能变快。牛短熊长可能变化。
（这可以算是股指期货和融券做空的直接效果，也就是逐渐消灭死多头。关键在于，增加市场的供给，让更多的股票IPO上市。做空使得股价更为真实和健康。我以前多次写点小文章批驳那些价值投资专家以及价值明星股，我认为这是一种多层面的、立体化的市场操纵。）
三、先灭多头，再灭空头。长期的思维惯性，做多的普遍较多，多头不死，下跌不止；一旦杀完多头，培养了一批死空头，又该反转刀口杀空头了。
（历史原因，死多头太多，而且过于夸张。）
四、股指不变，个股大动。这有较大的概率出现。股指波动幅度小，而个股互有涨跌，甚至幅度不小。这叫小空间办大事。
（这是一种现实的可能性，或许是唯一的可能性）
