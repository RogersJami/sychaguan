你用股市作为论证大数据的例子，我在你的一个主题贴中已经回复了。你在这层楼中的“我去年尝试过用大数据管理流程梳理股票筛选工作”一语，很明显的说明了“大数据”是工具性质。

股市是一个博弈的场所。大数据增加了信息不对称状态，那些拥有大数据工具的人，自然会在信息不对称的博弈中得优。如果所有的参与者都拥有相同的大数据时呢？还会有这样的数倍赢利优势吗？所以，淘汰是信息不对称的博弈的结果，大数据加速了这种过程。

数据资产化，在法律层面有障碍。处理不好，有可能要违背当代公平公正平等的法律思想。比如，我的数据，你有什么权力无偿获得；我购买商品是我相应劳动的交换，不是我的数据交换。等等。如果遵循市场原则，资产可以等价交换，那么整个社会的交换成本就会大幅度提升，直到这个社会的运作崩溃。如果按照权力原则，那么暴力就是最后的保障。所以，有限的数据资产化，只能是权力、资本和数据操作者单方面的游戏。

新技术抛开许多人，这是社会资源有限造成的。在市场条件下，还可以通过人为制造有限状态而形成。而资本正是这方面的高手。逐利，可以通过薄利多销得到，也可以通过有限供给而形成“厚脂利润”得到。信息技术的物化成本已经很低了，所以，造成大数据抛开人的原因，不是大数据本身。

当一个社会运动，发展到一定的程度，它的对立面一定会形成，如果优势一方不共享，那么，对抗就会产生。最终这个社会运动解体或消亡。

事物的规律，是相对条件而言。就像股票一样，有短期趋势，中期趋势，长期趋势。规律，也有空间和时间中的转换。当条件不具备的时候，冲锋者自然是浪花在礁石上拍得粉碎。当条件具备时，观望者就会“一步落后，步步落后”。而人们的选择，有多种理由，而背后只有两道源泉，利他，或利己。

     对资产的界定不是以劳动为确认标准的。数据很久以前就作为无形资产来处理了。葡萄提出的作为固定资产才有有意思的地方，因为做固定资产处置的时候隐含着一个条件，那就是要可辨认。
      这个可辨认，我认为可以由两种方法获得，一种是虚拟现实。另一种是对目前软件产品固定资产化的延伸——如果软件必须附着在某固定资产上才能保持完整性（不可独立计量），则整个系统作为固定资产处理。只是不知道“数据附着在固定资产上不可单独计价（这种形态其实就是物联网）”，是否就是他所说的数据固定资产化。
      虚拟现实其实就是消灭中层，因为办公室虚拟化，所以决策层其实是直接控制到基层的，比如此次营改增后的增值税管理系统就可以看做是二维化的虚拟办公室。而物联网在解释资源分配问题的时候体现的是按需分配精神（完全需求拉动），这样就挤压到部分高层的资源分配权了。葡萄说过这样做（对数据固定资产化进行立法）会压缩高层。今天的新闻是《周鸿祎谈“互联网+”》我估计是物联网要进入立法程序了。
