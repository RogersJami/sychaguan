白沙 发表于 2015-9-30 23:09:18

我也是自建的房子，有些经验可以跟楼主交流一下。

一、风水这个东西不好说，尤其是平原地带，从你这个图上看不出特别的地形地貌，所以不好轻易评判。但是风水学上一些基本的原则是可以参考的，比如大门不要正对马路，认为马路是杀气的来源，不利。如果因地形限制非要正对马路，也要用屏障（比如风水墙、玄关等遮挡）。要避免污秽的事物（如别人的厕所、烟囱、屋角等正对你的房子），在目力所及的范围内，如果轻易看到这类事物是不利的。还有就是背面的所谓“靠”，从你的图片来看，屋后是一片竹林，这个比较阴，对气血弱的妇女、儿童及老人不利，建议清除部分竹子。正面的情况，建议朝东，因为即使朝南，你也基本不可能感受到前面的水源。风水学上认为水是有灵气的，但是流动不明显的水不是特别有利。如果很讲究，考虑大门的朝向有必要。即使总体朝东，也要选一个角度，这是风水师的活，可以花几百块请个风水师看一看。

以上基本是瞎掰，我自己是无神论者，我盖房子没有这些啰嗦思考，大门就横在马路边，中规中矩的位置选择。

二、建设过程中，工地一定要有正规管理。可以考虑花点钱买些彩钢瓦，象正规工地那样围起来，只留下建筑物料进出的口子。要有专人管理工地，最好是自家的老人，有建房经验，充当工地管理员、工程监理师及物料管理员。建筑物料一定要码放有序。

三、建议用钢管搭设脚手架，毛竹脚手架有太多施工安全隐患。物料提升要用钢管搭设的大井字架，农村简易井字架安全隐患台大，台风天容易倒伏。

四、水泥泵车能进的话，一定要用预拌混凝土，现在建材便宜，差不了多少钱，浇筑的砼体更结实，养护周期短。每层楼面浇筑结束以后，一定要把浆头处理干净，不然以后装修会麻烦。要用新的模板，混凝土浆的跑冒滴漏会少很多。

五、根据你的描述，周边房子层高均不高，建议你不要建得太高（这样也不会激化周边邻居的矛盾）。农村简易别墅的层高控制在2.5--3层左右，太高了不协调，房子肯定不会美观。

大致想到这么多，以后再想起什么来再交流。

白沙 发表于 2015-9-30 23:16:07

还有水电预埋的事情想起来了。

这个事情你首先要有今后装修的大致思考，线路走向、开关插座的位置安排等要心里有数，要请有经验的师傅做水电工。

强电、宽带、安防、监控以及有线电视等线路要考虑清楚，基本的原则是强电弱电不同管，线路要预设到位。比如今后安装监控，你要考虑是集中供电、单独供电的差别，当然还有插卡式的探头，这些因素都要考虑清楚。可视门铃、遥控卷闸门、可控锁等都要想清楚。

刹那芳华 发表于 2015-9-30 23:53:21

不知道你家和1号家的关系如何，现在批给你的地你家建好后他家基本就成了洼地，因为周围都是2层或3层，他家就1层，他家阳光受到影响吗？

我觉得向东更好，但是还是要听听老人的意见，因为各地风俗不一样，而风俗是有一定道理的。根据各地气候、地形、雨水不同，房屋朝向不同地方是不一样的。

自建房就是下水要注意下，否则自己臭到自己了。最好多花点钱在地基和下水道的铺设上，前面考虑多放点余量，后面生活方便很多，否则无法改动。

卫生间一定要明的，就是要有朝外的窗户，无论大小，通风对于卫生间实在很重要的。

另外电线插座一定要多留，特别是厨房，避免以后到处拉拖线板。。。

电线不要裸铺在墙里，要有穿线管，没有多加多点钱，安全系数高很多，还方便以后检修。

最好自己有空气制动多路开关，这样可避免跳闸伤电器，还可以方便维修，换电器等。
