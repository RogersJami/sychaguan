DF31 发表于 2012-6-20 02:16:49

逃逸塔的作用是，在火箭点火时，或者上升段时，火箭发生故障，逃逸塔会紧急启动，带着飞船与火箭飞离，上升到一定高度后，飞船返回舱再与其他舱段分离，按着陆程序开伞，在离发射场数公里外软着陆。

1983年9月26日，联盟T-10-1号飞船发射时，火箭在发射台上发生爆炸，逃逸系统发挥作用，将飞船带出，最终在4公里外降落，两名航天员全部生还。
