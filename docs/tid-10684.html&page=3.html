早年大家的利益意识还没开发、为国奉献的觉悟意识还没消退的时候，搞的一些政府公共项目还敢说没有 ...
根据我在国外客串拆迁{:soso_e113:}{:soso_e113:}{:soso_e113:}{:soso_e113:}{:soso_e113:}{:soso_e113:}的经历看，单个人与有组织的博弈要注意以下4点：

1、小便宜不占白不占。经办人权限范围内能给你的，绝大多数情况下都会给你。

2、大便宜占不到不要勉强。钉子户提出的经办人权限范围外的、过分的要求，经办人既无权给你，(一般)也不会帮你争取。钉子户一般有这种心理：对于xxx(公司或者机构)来说，这点钱不算事。的确，与重工业项目(我是搞重工业的)相比，民用那点钱毛都不算----但是这不是钱的问题，是没有人愿意坏规矩。

3、不要试图拿住对方。遇到不讲理的钉子户，经办人在组织内部是被理解的，没有压力(所以拿不住)。进一步说，公司或者政府部门，面对不讲理的钉子户，在舆论上、以及在相关主管部门那里，是被理解的(有理由拖着不做)。很多情况下，你钉就钉着吧。

4、单个人和有组织的博弈，不是对等的。对于经办人来说，与钉子户周旋，不过是养家糊口的一份工作。下了班就把你忘了，或者把你当笑话讲。摊上钉子户，不能接新活，说不定工作更轻松。而对于钉子户来说，付出的时间、金钱、精力都是自己的。而且情绪时时刻刻受到影响，实在不上算。
