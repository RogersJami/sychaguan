说起陈玄风，让我想起第一次读《超限战》时的感想了。

因为之前读过乔良的《末日之门》，末日之门的主题就是恐怖主义作战，所以超限战给我的第一感觉就是，这是小国弱国对大国强国作战用的战略战术。

大国对小国的主要军事优势就是完整的战争系统，美军面对伊拉克伊郎朝鲜时，在军队人数上倒并不占优，另外除了美国外，其它大国对小国的陆军装备优势也不是那么大。

所以，当时我的第一个想法就是，越南虽小，但是只要它有心，一定可以找到一招制伏中国的地方，遏阻中国侵略的意图，比如三峡大坝。

更深层次的不对称作战，是破坏对方社会的稳定结构，迫使对方社会陷入崩溃边缘。如王XX写的《黄祸》，如果东北下岗工人真得大面积揭不开锅的，演变成大规模流民，并不是没有可能的。

昨天读京虎子的《微生物的八卦》，讲到班图人成为黑非洲的主体，是因为镰刀状贫血的生存优势，结合了恶性疟疾这个生化武器的扩张。这样说来，恒河的微生物也是印度在生物武器作战时的优势:lol
