申氏说的，你给我一条路，我放你一码。是你好我好大家好。和气生财。

从国家队的操作来看，恐怕不像， ...

我一直不明白期指这种虚标的杠杆工具怎么会被管理层接受。买入优质上市公司的股票就算高一点，至少可以靠公司盈利和增长慢慢弥补，期指这个标的到了交割期就烟消云散，完全是靠高杠杆来以小博大。

期指市场是本身不创造任何价值的零和博弈，一帮砖家还拼命宣传期指卖空可以对冲股票下跌的损失。
第一，股票市值的下跌是浮盈，在高点市场中本来就没有足够的资金让大量筹码兑现。股票浮盈和期指套保就是空对空。
第二，期指市场做空能赚钱就意味着做多要相应赔钱，谁来当这个交割的冤大头？谁又能当得起这个承担机构万亿筹码下跌损失的冤大头？股票浮盈通过期指套保变成真金白银的盈利根本就是抢劫。
本来期指的设计就是基于场内个体互相残杀的，可是当系统性风险来临，已经没有足够的韭菜可以割了。
