《黑客帝国》我非常喜欢，我的电脑里一直有三部曲，存了n年了。《十三》用一部电影的时间讲述了两部《黑》的内容，所以我说伏笔深。

斯密斯特工控制人脑，尼奥战胜乌贼，第三部眼睛瞎了看见所谓真实的世界，这三个点已经被讨论无数遍了，所以早有一种观点说锡安也是虚拟空间的一部分。但有人指出导演兄弟接受采访的时候解释，绿色色调的空间是虚拟的，而破败地球的空间是真实的，如果导演真说过这样的话，粉丝的解读就过度了。第一部很禅，后两部有点说不通。

除了情节线索，还有地名人名。zion我没记错是来自圣经，指犹太人的救赎之地。morpheus有两个意思，一是希腊神话的睡梦之神，二是吗啡类药物，吗啡海洛因一类毒品是靠替代脑内神经递质而成瘾的，而人类自然生成的减缓痛苦麻痹神经与吗啡化学性质类似的神经递质被称为内部吗啡或内部morpheus，即内啡肽（做一些更正）。neo倒过来读是one，意指救世主。trinity可能是三位一体，具体是什么我不清楚。oracle指祭师，具体含义不明。

记得大航海时代第一次金融革命，是复式记账让财务管理可以跨过人的生活圈，越洋过海。当然那时候还是


热那亚人和威尼斯人博弈的地中海。第二次，是英国人通过劳埃德完成了从国王到平民，通过保险的方式把


高风险高收益的远洋博弈变成了可持续经营的生意。随后一次又一次对外扩张中把这个思路演变为通过英格


兰银行获得全国分肥的方式走上霸权之路。然后结果金融与经济组织改革的是接过有限责任公司的德国。是


把有限责任公司扩张为上市公司的美国。而美国在完成工业革命早期积累后。划时代的金融变革源自美林把


上市公司股权拆细黑崛起的中产阶级，就比开启了海盗时代的美国股市黄金篇章。


我上面的表达，无非在说每个技术革命改变生产关系及其组织关系的时代势必产生新的分配方式。而这些新


分配当时能获得成功的共性特点就是向更多人分配红利获得支持。并依托共同获利的群体形成对其他竞争者


的局部优势最后变成压倒性的全面优势。


这就是时代，或者说我们老祖宗的大白话：上下同欲者胜。普通人参与政治博弈，自己斟酌，普通人可以做


的最直接的看谁给的实惠就跟。仅此而已，千百年来没有例外过。
